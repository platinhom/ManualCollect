Click this text with one of your mouse buttons to return a number.

1 = The left mouse button
2 = The right mouse button
4 = The middle mouse button
8 = The fourth mouse button (typically the "Browser Back" button)
16 = The fifth mouse button (typically the "Browser Forward" button)

Tip: The return values are combined if you press two buttons at once.

Note: The buttons property is currently only supported in Firefox and IE9 and newer versions.
