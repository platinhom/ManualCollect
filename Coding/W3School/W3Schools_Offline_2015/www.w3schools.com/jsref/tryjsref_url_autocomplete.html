Homepage:

Click the "Try it" button to set autocomplete in the URL field to "off".

Note: If you are new to the autocomplete attribute, fill in a URL, starting with "http://", and submit the form, then reload the page, start to fill in the same values in the URL field again - and see how autocomplete works. Then, click the "Try it" button to set autocomplete to "off" - if you type something in the URL field now, the browser will not suggest values based on values that you have entered before.

Note: elements with type="url" are not supported in IE 9 (and earlier), or Safari.
