E-mail:

Click the "Try it" button to find out whether the email field has autocomplete set to "on" or "off".

Note: If you are new to the autocomplete attribute, fill in an email and submit the form, then reload the page, start to fill in the same values in the email field again - the browser will suggest values based on the values that you entered before.

Note: input elements with type="email" are not supported in IE 9 (and earlier), or Safari.
