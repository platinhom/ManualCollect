This example shows the difference between the seeking event and seeked event.

The seeking event occurs everytime the user STARTS moving/skipping to a new position in the audio/video.

The seeked event occurs when the user is FINISHED moving/skipping to a new position in the audio/video

Move to a new position in the video. Tip: Try to hold down the mouse button and seek back and forth in the video playback.
