WMLScript abs() 函数

返回 WMLScript Lang 库参考手册

abs() 函数返回一个数的绝对值。

语法

n = Lang.abs(x)
成分 描述
n 从函数返回的绝对值。
x 计算这个数的绝对值。

例子

var a = Lang.abs(-5);
var b = Lang.abs(-3.4);

结果

a = 5
b = 3.4
