.NET 软件

.NET 是技术、标准以及开发工具的混合体。

Windows.NET

今天，Windows 2000 和 Windows XP 组成了 .NET 的主干。

未来，.NET 基础构架会被整合到所有的微软操作系统中，包括桌面以及服务器产品。

Windows.NET 是下一代的 Windows。它将提供对所有 .NET 构建模块和 .NET 数字媒体的支持。当用户需要它时，Windows.NET 会经由 Internet 进行升级，以便进行自行维护。

Office.NET

新版本的微软 office - Office.NET - 将拥有新的基于因特网客户端以及 Web Services 的 .NET 基础构架。

通过 Office.NET 进行的浏览、通信、文档处理以及创作将被整合到一个基于 XML 的环境中，这个环境允许用户在因特网存储文档。

ASP.NET

ASP.NET 是最新版本的 ASP。它包括供连接使用 HTTP、HTML、XML 以及 SOAP 的应用程序、服务以及设备的 Web Services。

ASP.NET 的新特性:

  • 新的语言支持
  • 可编程的控件
  • 事件驱动的编程
  • 基于 XML 的组件
  • 用户身份验证
  • 用户帐户和角色
  • 高伸缩性
  • 已编译的代码
  • 易配置
  • 易部署
  • 非 ASP 程序的兼容性
  • 包含 ADO.NET

您可以在我们的 ASP.NET 教程中阅读更多关于 ASP.NET 和 ADO.NET 的信息。

Visual Studio.NET

Visual Studio 的最新版本 - Visual Studio.NET - 整合了 ASP.NET、ADO.NET、Web Services、Web 窗体以及对 Visual Basic 的语言革新。开发工具拥有对 XML 深层的支持、基于 XML 的编程模型、新的面向对象的编程性能。

Visual Basic.NET

微软对该语言进行了改进，使其成为完全面向对象（object-oriented）的编程语言。

SQL Server 2000

SQL Server 2000 是充分支持 web（fully web-enabled）的数据库。

SQL Server 2000 对 XML 和 HTTP 有着强有力的支持，这两者恰恰又是 .NET 的主要基础构架技术。

新的 SQL Server 的某些最重要的特性是：可从浏览器对数据的直接访问、以 XML 来返回对关系型数据查询的结果、使用关系型格式存储 XML。

Internet Information Services 6.0

IIS 6.0 对更多在服务器上发生的编程拥有强有力的支持，允许新的 web 应用程序运行于任何平台上的任何浏览器中。
