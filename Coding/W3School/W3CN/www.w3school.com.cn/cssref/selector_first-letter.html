CSS :first-letter 选择器

CSS 选择器参考手册

实例

选择每个 <p> 元素的首字母，并为其设置样式：

p:first-letter
{ 
font-size:200%;
color:#8A2BE2;
}

亲自试一试

浏览器支持

IE Firefox Chrome Safari Opera

所有主流浏览器都支持 :first-letter 选择器。

定义和用法

:first-letter 选择器用于选取指定选择器的首字母。

注释：以下属性可与 :first-letter 使用：

  • 字体属性
  • 颜色属性
  • 背景属性
  • 外边距属性
  • 内边距属性
  • 边框属性
  • text-decoration
  • vertical-align（只有在 float 为 'none' 时）
  • text-transform
  • line-height
  • float
  • clear
