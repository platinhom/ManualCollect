浏览器对 E4X 的有限支持

目前没有一款主流的浏览器支持 E4X。

Mozilla 引擎 (1.8) 的 beta 版拥有对 E4X 的有限支持。Firefox 1.1 被认为是第一款支持 E4X 的浏览器。

希望今后的 Internet Explorer 版本会支持 E4X。

Firefox 1.1

在 Firefox 的下个版本中，Firefox 1.1 被认为对 E4X 的支持相对较好。

在下一节中我们提供了一系列的例子，描绘了在 Firefox 1.1 中您可以通过 E4x 做的事情。

Internet Explorer 7

在 Internet Explorer 7 中，迄今为止依然没有支持 E4X 的迹象。

微软似乎打算支持其自己的 AJAX 方案。

AJAX 是一种使用 JavaScript, XML 以及 CSS 来创建交互 web 应用程序的新 web 开发技术。

您可以在 W3School 提供的 AJAX 教程 中学习更多内容。
