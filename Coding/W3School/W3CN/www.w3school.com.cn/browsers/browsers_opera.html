关于 Opera

Opera 是作为挪威电信公司 Telenor 的一个研究项目于 1994 年启动的，并于 1995 发展为一个独立的开发公司，Opera Software ASA。

Opera Software ASA 研发了 Opera web 浏览器，是桌面和设备市场的 web 浏览器开发领域的业界领导者。

Opera 浏览器（被称为继 IE 和 firefox 之后的“第三选择”），由于其较之其他浏览器具有更快、更小且标准兼容性性更强的优点，已经得到来自终端用户和业界的国际性赞誉。

Opera 不仅免费而且符合 Web 标准

Opera 浏览器是免费的！早期的旗帜广告和许可费制度已经被撤销了！

如果您的网站在 Opera 中运行良好，那么可以确定它符合标准！只需要用符合标准的代码来编写您的页面，就可使确保您的网站中运行良好，不论是在所有主流浏览器中，还是在所有主流平台以及操作系统上。

Opera 支持所有在用的 Web 标准，诸如 CSS, HTML, XHTML, HTTP, DOM, XML, XSL, ECMAScript (JavaScript), PNG, WML, SVG, Unicode, the Unicode Bidirectional Algorithm 等等。

Opera 10.0

Opera 10.0 于 2009 年 9 月发布。

Opera 10 中的特性：

高效

小带宽加速利器 Opera Turbo

Opera Turbo 使用服务器压缩技术，提升受限连接下的浏览速度。比如在使用咖啡馆拥挤的 WiFi 连接，或是调制解调器拨号上网。

自动崩溃报告

Opera 的设计是，崩溃后自动重启恢复您崩溃时的页面。也可以使用崩溃报告帮助不断改进 Opera。

Opera Unite

Opera Unite 是全新技术平台，您可直接与朋友共享内容，无需上传到网站。您可建立流媒体音乐电台，展示相册，共享文件和目录，甚至用浏览器作为主机运行你的网站。

随心邮件

Opera 邮件功能可以发现富格式文本，包括图像、风格文本、连接、自定义 HTML 等。您可以选择接受邮件后具体几天从 POP 服务器删除，保证网络空间整洁。

Opera 蜻蜓展翅

Opera 蜻蜓可以让网络开发者调试 JavaScript，检视 CSS、 DOM 和 HTTP header，整合在 Opera 浏览器。

Opera 邮件功能

Opera 浏览器提供内置邮件客户端，优化您的日常邮件需要。它被称为 Opera Mail，它可组织、检索并分类您的所有信息；您可尽情享受网络。最好的一点是，它完美无缝的整合于您的浏览器。您可以使用 Opera Mail 收取各大邮件服务提供商的邮件，如 Gmail 和 163，只需它们支持 IMAP 或 POP 协议。

Opera Link

Opera Link 让您自由选择同步线上数据，甚至可以在不同手机和电脑上同步。尽可同步您的快速拨号、书签、定制搜索、历史和笔记；将书签随手机携带，和家中电脑共享。

内置拼写检查

在您朋友的 Facebook wall、博客和网络邮件上自由留言。Opera 会标出拼写错误，您可尽情表达不收拘束。利用流行的 Hunspell 词典格式，您总是拥有所有语言不断更新的最新词典。

外观

全新可视化标签和清爽设计

Opera 一直是浏览器标签创新的先锋。现在，Opera 引入了可调整大小标签栏。拖动标签下手柄可以看到打开网页的缩略图。此外，强大的功能亦需完美包装，设计师 Jon Hicks 为 Opera 操刀打造了全新外观。

随心定制快速拨号

用快速拨号，更快访问最爱网站。频繁输入网址十分低效，单击直达才是王道。快速拨号是您在新开标签时显示的一组可视化书签，它是您在线生活的操控板。

要添加新页面，只需点击空白快速拨号项，Opera 将基于您的浏览历史和书签提出建议。您可以随需定制快速拨号。点击配置快速拨号按钮，选择范围从 4 到 25 个拨号项。您也可以添加定制背景，个性化您自己的浏览器。

内置网络搜索

可调整大小的搜索栏可以直达您常用搜索引擎，还可以在地址栏和弹出菜单进行搜索。您可以添加任何搜索引擎，只需在搜索页面搜索框上右击，选择创建搜索。

定制您的浏览器

Opera 定制性强。在上百皮肤和配置中选择，添加/删除按钮、工具栏，或者改变浏览器布局。改变和撤销都非常简单。

缩放

轻松阅读小字体网站，或缩小以概览网页全貌。Opera 有快速真实的页面缩放功能，缩放所有网页元素，不仅仅是文本。

工具

快捷可定制网络搜索

快捷访问 Google, 百度等搜索引擎，直接整合搜索到右上角搜索栏。或者使用地址栏快捷键直接搜索关键词。如地址栏输入 "g Opera" 将直接调用 google 搜索 Opera 关键词。您也可以在网上创建您的搜索加入搜索栏，只需右击页面搜索框选择“创建搜索”。

内置网络搜索

可调整大小的搜索栏可以直达您常用搜索引擎，还可以在地址栏和弹出菜单进行搜索。您可以添加任何搜索引擎，只需在搜索页面搜索框上右击，选择创建搜索。

极速搜索

是否想找到不久前看到的一篇文章？ Opera 可以记住您访问站点的真实内容，您只需在地址栏输入关键字即可找到曾经访问的页面。

内容拦截

阻止您不想看到的图像、弹出窗口和插件。右击页面选择“内容拦截”，点击即可组织内容。要使网页加速载入，或者拦截不希望得到的内容，可以暂时关闭载入图片，点击 Opera 中图像按钮即可。Opera 已经默认开启智能弹出窗口拦截。

支持 BT 下载的下载管理器

用 Opera 下载更快。Opera 在您保存文件时自动开始下载，绝不浪费时间。点击按钮即可暂停/续传下载，也可选择多个文件同时下载。Opera 也内置了社会性文件传输协议，BitTorrent，下载 BT 无需再开其它程序。

为您的生活添加 Widget

Widget 是小型在线应用（多媒体、新闻网摘，游戏或者更多），这些应用让您的桌面体验更加有趣使用。使用饰件菜单，访问您的最爱。访问 widgets.opera.com 了解更多。

安全性

加密

Opera 支持 Secure Socket Layer (SSL) version 3 以及 TLS。Opera 提供自动 256 位加密，网络浏览器目前最高的安全性。

自动更新

Opera 让保持使用最新版更容易。使用自动升级，您可选择通知更新或是自动下载。使用 Opera，完全由您掌控。

删除私人数据

Opera 可以配置为退出时清除历史和缓存，保护您的隐私。任何私人数据可以被快捷清除。

安全栏

Opera 在地址栏显示加密信息。点击黄色安全栏，可访问当前证书有效性的更多信息。

欺诈防护和扩展验证保证安全

并非所有网页都是所声称的身份。 Opera 默认打开欺诈防护，自动检测并警告在线欺诈。欺诈防护由 Netcraft 和 PhishTank 反钓鱼信息支持。为了防护恶意软件，Opera 支持扩展证书验证技术 (EV)。这为安全网站加入更多保障。

Cookie 控制

Opera 为您接受拒绝 cookie 提供细节控制，比如允许为不同服务器单独设置 cookie 策略。
