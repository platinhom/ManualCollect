Babel interface

VMD can use the program Babel, if installed, to translate a wide variety of different molecular data files into the PDB format. Not all of these have been tested for use with VMD, so your results may vary. VMD only uses Babel to read files and does not allow the use of Babel to save files to other formats. The VMDBABELBIN environment variable is used to specify the absolute path to the the Babel executable (including the executable name). For more information about Babel, see http://smog.com/chem/babel/. VMD currently supports version 1.6 of Babel.
