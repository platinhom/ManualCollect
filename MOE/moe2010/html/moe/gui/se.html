Introduction

In MOE, a molecular system consists of one or more chains, and each chain consists of zero or more residues. A residue consists of zero or more atoms. The Sequence Editor allows you to organize and modify molecules at the chain and residue levels. It also serves as the primary launch point for MOE's protein modeling and protein structure analysis commands.

The Sequence Editor displays the current molecular system as a residue schematic, where each chain is displayed as a list of residues. For instance, the figure below shows a residue schematic of $MOE/sample/mol/serine_prot.moe, a collection of serine proteases.

To open the Sequence Editor from the MOE Window, select Sequence Editor from the Window menu. Alternatively, press the SEQ button in the upper right corner of the MOE Window.

Chain numbering is reflected in the chain display order in the Sequence Editor. Chain numbers are shown in the column labeled Chain on the left. You can enable (or disable) display elements such as the compound name or hydrogen bonds by turning on (or off) toggles in the Display menu. Operations on chains are performed using the mouse, the menu bar commands, the Chain popup or the Selected Chains popup.

Residues composing the chains are drawn with either 1- or 3-letter residue codes, depending on the selected display mode. The horizontal alignment ruler in the Sequence Editor indicates the alignment position of residues. A given alignment position in the residue schematic is referred to as a residue column. Operations on residues are performed using the mouse, the menu bar commands, or the popup menus.

To close the Sequence Editor, choose SE | File | Quit. The Cancel menu, in the top right corner of the window, allows you to cancel tasks running in MOE. Note that the MOE Window and Database Viewer also provide Cancel menus. Next to the Cancel menu, the SVL and MOE buttons raise the SVL Commands Window and MOE Window, respectively.

Command Line

The Command Line Interface (CLI) is where SVL commands are typed and issued. While the Sequence Editor has the focus, all keystrokes are automatically directed to the CLI regardless of where the mouse is positioned within the window.

The command line area is also used as a prompt area for several MOE functions. This prompt area may be for entering data or simply for displaying messages. In either case, to cancel the function which is displaying the prompt, press <Esc>. While the Sequence Editor's CLI is occupied by a prompt, other CLIs in MOE function as usual.

For more information on the Command Line Interface see the SVL Commands Window.

Chain Numbers

The chain numbers are the boxes to the left of the residue display area. Chains are always numbered sequentially, and each chain number is colored by its chain color. If you color the atoms by chain color (in the MOE Window), this is the color which will be used. Chain color can be set using the Chain Color command in the Chain popup. Chains can also be colored by tag using SE | Display | Color Chains | By Tag. A selected chain has its number displayed inset.

The area labeled Chain above the chain numbers is called the chain label and when pressed, selects or deselects all chains. A popup menu which acts upon the selected chains can be accessed here using the mouse popup button.


Residue Display Area

The residue display area comprises the column ruler, the residues themselves and the empty space between the drawn residues. The popup menus Residue Column, Residue, and Selected Residues are located in these areas.


Footer

The bottom part of the window is referred to as the footer and is used to turn on or off the display of scalar plots. The Sequence Editor can plot scalar values assigned to residues (see rSetScalar) in a graph which is aligned with the residues. The scalar plot displays values in the range -1 to 1; values outside this range are clamped (i.e. the graphs are clipped).

The Measure menu provides several functions which use these plots. See Measuring Residue Properties and Stereochemistry for more information.

Familiarity with these popups will give you a much greater control over the display of atoms in the MOE Window and of residues in the Sequence Editor. The popup menus in the Sequence Editor allow you to select, view, hide or show atoms of a chain, a residue, a group of selected residues, or a residue column (based on alignment). Editing Molecular Objects

Moving Residues

Residues are moved with mouse drag movements within the residue display area. They can be moved to the right or left, but not beyond the start of the alignment ruler.

Action Description
To move one residue Drag the residue in the Sequence Editor using the middle mouse button. A red border highlights the residue being moved.

The residue is moved right or left (changing its alignment position). Any residues encountered will be pushed along, like beads on a string. Gaps between residues are not preserved.
To move several residues Select the residues to move (residues can belong to more than one chain and do not have to be consecutive), and press Alt while dragging the middle mouse button.

Selected residues are moved right or left. Any residues encountered will be pushed along, like beads on a string. Selected residues maintain their relative alignment positions while moving.
To push residues in a chain Press Shift while dragging the middle mouse button over a residue in the chain.

All residues to the right or left of the dragged residue are pushed to the right or left along with it, while maintaining gaps.

To slide all residues in an entire chain Press Ctrl and drag the middle mouse button over any residue in the chain. Gaps are maintained.

Reordering Chains

You can reorder the chains in the system by moving one or more chains using commands in the Edit | Move Chains submenu or in the Chain and Selected Chains popup menus.

When using the Edit menu, you can move several chains at once, but you must first select the chains you wish to move. This applies also to the Selected Chains popup menu.

The equivalent Chain popup commands move a single chain at a time.

SE | Edit
Command Action
Move Chains | Top Move the selected chains to the top of the chain list.
Move Chains | Bottom Move the selected chains to the bottom of the chain list.
Move Chains | Before Move the selected chains before a given chain. You will be prompted to select the chain before which the selected chains will be moved.
Sort Chains | by Name Order the chains alphabetically by chain name. The name can be displayed in the Sequence Editor using SE | Display | Compound Name.
Sort Chains | by Tag Order the chains alphabetically by chain tag. Chain tags can be edited using SE | Edit | Chain Tag or with the Chain and Selected Chains popup menus.

Joining and Splitting Chains

To join two or more chains:

  1. Select the chains to join.
  2. Choose SE | Edit | Join Chains or choose Join Chains from the Selected Chains popup.

The residues of the selected chains are re-parented to the lowest numbered chain.

To split a chain at a single residue, open the residue popup over the residue and select Split Chain.

To split the system at several residues:

  1. Select the residues that will mark the beginning of the new chains.
  2. Choose SE | Edit | Split Chains. The selected residues each become the first residue of a new chain. Each new chain appears directly below the chain from which it was split.

Deleting Chains and Residues

To delete a single chain:
  1. Position the cursor over the number of the chain to delete.
  2. Display the Chain popup and choose Delete this Chain.

To delete two or more chains at a time:

  1. Select the chains to delete.
  2. Display the Chain or Selected Chains popup and choose Delete Selected Chains. The same functionality is found in SE | Edit | Delete Selected Chains.

Deleting residues is accomplished similarly. To delete a single residue, you can use the Delete this Residue option from the Residue popup. To delete one or more residues, select the residues to delete, then use either the Delete Selected Residues option in the either the Selected Residues or Residue popup, or the SE | Edit | Delete Selected Residues menu command. To delete all residues in a residue column, use the Delete this Column option in the Residue Column popup. After the residue column is deleted, the alignment is "packed" (all residues to the right of the deleted column are moved left by one column to close the gap).

When deleting chains or residues, MOE displays the total number of chains, residues and atoms that will be deleted, and asks you to confirm or cancel the operation. You also have the option to "pack" the alignment; i.e. eliminate empty residue columns that may result from the deletion operation.

Coloring Molecular Objects

Coloring Residues

Residues can be colored in the Sequence Editor using one of the following:

  • The Residues | Color menu item from the Chain and Selected Chains popup menus in the Sequence Editor
  • The Color menu item from the Residue, Selected Residues, and Residue Column popup menus in the Sequence Editor
  • The Color Residues panel (SE | Display | Color Residues)

The Sequence Editor popup menus allow you to color individual or multiple residues. The colors are selected from a pre-defined palette which is accessible as a submenu of colors.

The Color Residues panel defines a group of color-coded amino acid equivalence classes. Choosing a color scheme and pressing Apply will color all the residues currently in MOE. When residues are created or mutated they are not automatically colored. In this case, you must press Apply in the panel to update the residue coloring.

The MOE Window includes the rendering option MOE | Render | Atoms | Color: Residue to automatically color atoms according to the color assigned to their parent residue.

Coloring Chains by Chain Number and Tag

Chain colors can be assigned according to chain number and chain tag using the SE | Display | Color Chains menu commands. The chain color is the color used to display the chain name in the Sequence Editor. Atoms can be colored by chain color using the MOE | Render | Atoms | Color: Chain command.

SE | Display | Color Chains
Command Action
By Number Color each chain a unique color.
By Tag Color chains of a common tag by the same color. Chain tags can be edited using SE | Edit | Set Chain Tag or similar commands in the Chain and Selected Chains popup menus.

Coloring Atoms of Selected Chains in the MOE Window

From the Selection menu, choose the SE | Selection | Chains | Color Separate Selected Chains command. This command is greyed out if no chains are selected. Atoms of selected chains appear in one color in the MOE Window, whereas atoms of all other chains are a different color. (To otherwise modify the coloring scheme of atoms in the MOE Window, use the MOE | Render | Atoms panel.)

Coloring the Backbone

The display of the backbone ribbon in the MOE Window can be controlled from within the Sequence Editor. Each of the Sequence Editor popup menus contains a submenu called Ribbon that allows you to render the backbone ribbon of specific residues in various styles or to change the color of that portion of the backbone to one of the pre-defined colors found in the Ribbon submenu. Note that changing the color will not automatically turn on the backbone display, so to see the color, first draw the backbone. To see the different backbone drawing styles, see Rendering the Backbone.

Measuring Residue Properties and Stereochemistry

The Measure menu offers various residue property and stereochemistry evaluation tools:

  • Residue scalar plots displayed in the Sequence Editor
  • Protein stereochemistry evaluation tools and plots

Residue scalar plots measure and display residue attributes such as:

  • Hydrophobicity
  • Residue charge
  • Solvent accessibility
  • Predicted solvent accessibility
  • Predicted secondary structure

MOE displays each numerical attribute as a scalar plot, which is represented by a bar graph aligned with residue sequences. Up to three scalar values -- or attributes -- may be plotted at the same time. Plotted values range from -1 to 1, with a horizontal black line marking 0. Plotted values exceeding this range are "clamped"; i.e. MOE clips the graph. Colors assigned to the plots can be configured in the Sequence Editor Colors page of the Configuration Options panel.

The Plot footer contains checkboxes corresponding to each of the scalar plots. Toggling one of these hides/displays the corresponding plot. When a particular residue attribute is calculated and assigned to a given plot, the name of the corresponding checkbox in the Plot footer changes to reflect the attribute that is plotted. For example, in the figure above, Hydrophobicity is assigned to Scalar 1, Charge to Scalar 2, and Accessible Surface to Scalar 3.

The following are the predefined attributes that can be plotted. To plot any one of them, select the appropriate command from the Measure menu and specify which scalar plot to use: Scalar1, Scalar2, or Scalar3.

SE | Measure
Command Description
Hydrophobicity Plot the amino acid hydrophobicity values of Eisenberg et al. [Eisenberg 1984].
Residue Charge Plot the charge of a residue, which is the sum of the formal charges of its constituent atoms.
Solvent Accessibility Calculate the solvent accessible surface areas of all amino acid residues in the system and plot the ratios of these values to pre-calculated values for each amino acid type in a GLY-X-GLY tri-peptide in extended conformation. The following reference values are used: 
ALA:119.77 ARG:242.85 ASN:161.57 ASP:155.38 CYS:145.01 
GLN:172.92 GLU:174.09 GLY: 86.63 HIS:193.65 ILE:187.39 
LEU:189.51 LYS:208.26 MET:195.41 PHE:213.44 PRO:145.71 
SER:130.84 THR:154.64 TRP:254.40 TYR:229.11 VAL:167.65 
The midline represents exactly 0.25 of the standard exposed surface area for each residue type. Bars above the midline (pointing up) indicate buried residues, below the midline, exposed residues.
Predicted Solvent Accessibility Plot the probabilities that residues are buried based on the residue sequence of each chain.
Predicted Secondary Structure Plot the probabilities that residues have a helix or strand secondary structure based on the residue sequence of each chain. The Predicted Secondary Structure command populates all three scalar plots.

The Display menu controls various aspects of the display within the residue area of the Sequence Editor.

The standard default coloring corresponds to the coloring of the backbone ribbon in the MOE Window; by default, helix is red, strand is yellow, turns are blue.
Predicted Secondary Structure Display the predicted secondary structure as calculated from the Homology | Predict Secondary Structure command. The secondary structure is shown as color-coded bars below the residues.
Hydrogen Bonds Display the protein backbone hydrogen bonds used by the DSSP secondary structure assignment algorithm to identify 3-turns, 4-turns and 5-turns. Bonds are drawn as arcs connecting the donor and acceptor residues.
Single Letter Residues Display single-letter abbreviations of amino and nucleic acids (nucleic acids are shown in lower case). Non-standard amino acids and nucleic acids are displayed using an X. Other residues, such as waters and moieties, are displayed as empty boxes. In single-letter mode, dots are used to represent alignment gaps. For the complete list of residue abbreviations, see Building Proteins and DNA.
Residue Indices Display the ordinal numbers of residues within chains. Choosing this option precludes the display of residue UID (described below).
Residue UID Display residue UID numbers. Choosing this option precludes the display of residue indices.
Color Residues Open the Color Residues panel, for coloring amino acid residues by class.
Color Chains Assign chain colors. If By Number is chosen, then each chain is assigned a unique chain color. If By Tag is chosen, then chains having the same tag are assigned the same color. Chain colors can also be set using the Color Chain command in the Chains and Selected Chains popup menus.

The default values of all the above options can be configured in the Configuration Options panel.
