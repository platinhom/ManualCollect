Database Viewer Open/Close Functions

Syntax

    dbkey  = dbv_OpenFile [ 'filename', read_only_flag ]
    dbkey  = dbv_Open [ 'filename', read_only_flag ]
    dbkeys = dbv_KeyList []
    dbkeys  = dbv_Key dbkeys

    dbv_Close dbkey

Obsolete

dbkeys = dbv_ViewKeys []

dbv_ViewKeys has been replaced with dbv_KeyList.

Description

A Database Viewer is a window that gives a view onto a MOE molecular database file. As changes are made to the database, they are displayed in the viewer window. Once opened, a Database Viewer and its key remain valid until the Database Viewer is closed. 

dbkey = dbv_OpenFile [ 'filename', read_only_flag ]

Opens a Database Viewer onto the specified database file, and returns a key for use in subsequent Database Viewer calls. If the file does not exist (or is []), an error is generated. If a Database Viewer is already open on the specified database file, the viewer is simply raised to the forefront.

To open the viewer in read-only mode, set read_only_flag to 1. Otherwise, if read_only_flag is 0 or unspecified, the database is opened according to its current file permissions. 

dbkey = dbv_Open [ 'filename', read_only_flag ]

dbv_Open is nearly identical in behavior to dbv_OpenFile. The only difference is that, when the argument is a null vector [], it opens the Open Database Viewer file selection box allowing the user to select the database file to open.

To open the viewer in read-only mode, set read_only_flag to 1. Otherwise, if read_only_flag is 0 or unspecified, the database is opened according to its current file permissions. 

dbkeys = dbv_KeyList []

Returns a vector of keys of all currently open Database Viewers. [] is returned if no Database Viewers are open. 

dbkeys = dbv_Key dbkeys

For each key in dbkeys return the key if it corresponds to a valid open database key. Return 0 otherwise. 

dbv_Close dbkey

Closes the specified Database Viewer. An error occurs if the viewer specified by dbkey does not exist.
