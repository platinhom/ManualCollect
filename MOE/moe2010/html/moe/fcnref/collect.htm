Molecular Object Collection Functions

Syntax

    name_tokens = oCollections [];
    object_keys = oGetCollection name_token
    oSetCollection [name_token, object_keys]

Description

In MOE, molecular data can be managed as named sets or collections. A collection comprises any mixture of atom, residue, and chain keys, and has a unique associated name. 

name_tokens = oCollections [];

Returns the names of all molecular object collections currently defined in MOE. 

object_keys = oGetCollection name_token

Returns the object keys of the specified collection. 

oSetCollection [name_token, object_keys]

Defines a new named collection or redefines a previous collection of the same name. The object keys, given as a flat vector, may be a mixture of atom, residue, and chain keys. The name associated with the collection is used with oGetCollection to recover the object keys. Setting a collection to the empty set deletes the collection.
