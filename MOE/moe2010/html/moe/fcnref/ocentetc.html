MOE 2010.10

Molecular Object Functions

Syntax

    [x,y,z] = oCenterOfMass keys 
    [x,y,z] = oCentroid keys

Description

[x,y,z] = oCenterOfMass keys

For each object specified in keys, oCenterOfMass returns the center of mass of all its atoms. 

[x,y,z] = oCentroid keys

For each object specified in keys, oCentroid returns the centroid (mean position) of all its atoms.
