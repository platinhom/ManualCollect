MOE 2010.10

Atom Functions

Syntax

    Protonate atom_keys 
    Deprotonate atom_keys

Description

Protonate Adds a hydrogen to each atom specified by atom_keys. Protonating a hydrogen has no effect. Protonation increases an atom's ionization level by 1.
Deprotonate Strips a hydrogen (if present) off each atom specified by atom_keys. Deprotonating an atom that has no bonded hydrogens has no effect. Deprotonation decreases an atom's ionization level by 1.
