MOE 2010.10

Database Viewer Molecule Drawing

Syntax

    old_state = dbv_MolElements [ mdb, new_state ]
    old_state = dbv_MolHydrogens [ mdb, new_state ]
    old_state = dbv_MolMultipleBonds [ mdb, new_state ]
    old_state = dbv_MolProteinTrace [ mdb, new_state ]
    old_state = dbv_MolUnifiedView [ mdb, new_state ]

Description

These functions return and optionally set molecule drawing attributes of all molecule fields in the database viewer mdb specified by either a database key or the database filename. The old drawing state is always returned. The state is set (enabled) if new_state is specified (i.e. not []) and is not equal to 0. The state is disabled if new_state is 0.

old_state = dbv_MolElements [ mdb, new_state ]

Controls element name drawing. When enabled, element names are displayed in a given molecule drawing only when the drawing is sufficiently large for the element names to be rendered clearly. When the drawing is too small, element names are hidden, regardless of the state.

old_state = dbv_MolHydrogens [ mdb, new_state ]

Controls drawing of hydrogen atoms. When disabled, hydrogen atoms are hidden.

old_state = dbv_MolMultipleBonds [ mdb, new_state ]

Controls drawing of multiple bonds. When disabled, all bonds are rendered as single lines. Note that calculation of bond orders can be a lengthy process on large databases.

old_state = dbv_MolProteinTrace [ mdb, new_state ]

Controls drawing of the alpha-carbon protein trace. When the trace is on, the display of atoms, bonds, and element names is disabled. The setting has no effect on non-proteins.

old_state = dbv_MolUnifiedView [ mdb, new_state ]

When enabled, all molecules in the database are displayed using the same view matrix. The result is that when one molecule is rotated and/or zoomed, all other molecules in the database move in unison with it. When unified viewing is disabled, each molecule's view can be individually manipulated. Unified viewing is useful, for instance, for examining a database containing conformations of the same molecule, such as those output from a dynamics run.
