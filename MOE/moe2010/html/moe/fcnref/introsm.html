Introduction to Molecule Input Language Functions

MOE uses an implementation of SMILES (Simplified Molecular Line Entry System) for molecule building, and a subset of the SMARTS language for substructure matching. SMARTS is essentially an extension of SMILES. The basic syntax is described in the entry for sm_Build, with additional matching expressions described in the entry for sm_Match. The string extractor sm_Extract provides a simple means for generating the SMILES strings describing molecules. Two syntax verifiers, one for building, the other for matching, are available for checking string validity. For matching, MOE supports the use of dictionaries of molecules specified as match strings.
