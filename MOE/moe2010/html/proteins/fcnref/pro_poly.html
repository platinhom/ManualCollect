MOE 2010.10

Protein Functions

Syntax

    pro_Polymerize residue_keys
    pro_Polymerize [ n_term_residue_keys, c_term_residue_keys ]

Description

Construct peptide polymer bonds between residues. If passed an argument of exactly length 2, each residue in n_term_residue_keys is bonded to the corresponding one in c_term_residue_keys. The two vectors of keys must be of the same length. The positions of oxygen atoms (and hydrogens, if they exist) are adjusted to conform to standard peptide geometry.

If passed an argument of length longer than 2, the residues are polymerized in sequence, in the order they appear.

If the argument is of length less than 2, the call has no effect.

N-terminal residues to be polymerized must contain CA, C, and O backbone atoms, and C-terminal residues must contain N and CA backbone atoms; the function call has no effect on those residues missing any of the requisite backbone atoms. 

    svl> pro_Polymerize Residues []
