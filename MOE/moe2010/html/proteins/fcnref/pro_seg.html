Protein Functions

Syntax

    pro_SegmentToDihedrals res_keys

Description

Sets the backbone dihedral angles of the residues specified by res_keys to standard angles associated with the current values of rSegment res_keys. For the backbone angles to be set, the segment type must be one of the known types, and the residues must have complete backbones and must be polymerized.
