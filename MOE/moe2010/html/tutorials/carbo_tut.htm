Building Carbohydrates in MOE

This tutorial introduces the Carbohydrate Builder and presents some fundamental concepts relative to building polysaccharides using a library of common residues. The tutorial is broken down as follows:

  • Exercise 1: Building Sucrose
  • Exercise 2: Building Cellulose Oligosaccharide
  • Exercise 3: Building an N-linked Oligosaccharide
  • Exercise 4: Editing a Carbohydrate Structure
    • Inverting a Center
    • Mutating a Residue
    • Inverting a Residue
  • References

Conventions Used in this Tutorial

MOE MOE Window
SE Sequence Editor
DBV Database Viewer

This tutorial assumes that you are using a three-button mouse. For information on using a two-button mouse see Using the Mouse.

Exercise 1: Building Sucrose

Common table sugar, or sucrose, is a 1-2 linked gluco-fructose disaccharide. Fructose exists as a five-membered ring in sucrose.

a-D-Glcp(1-2)b-D-Fruf
  1. To open the Carbohydrate Builder, choose:

    MOE | Edit | Build | Carbohydrate

  2. Change the Library option to pyranose.
    Change the Unit option to aDGlcp.
    Turn on the Auto-Select Anomeric checkbox.

  3. Click the Build button to instantiate the first saccharide unit.

    Note that the oxygen of the anomeric -OH is automatically selected, which is the appropriate junction for the next step.

  4. Change the Library option to furanose.

    Change the Unit option to bDFruf.

    Turn off the Auto-Select Anomeric checkbox.

    Click on the button labeled O2. This is the hydroxyl unit which will be attached to the currently selected position, and is important for linking the second unit to the first. The graphic is updated accordingly to highlight the selected position.

  5. Click Build to add the second unit.

    Tip The Carbohydrate Builder does not explicitly set chirality constraints, although the geometry is reproduced faithfully from the diagram. To constrain the atom centers which have known and non-fluxional chirality, select the non-anomeric atoms in the two rings, and use the Molecule Builder (press Builder in the MOE Window Right Button Bar, then press the G Stereochemistry button) or the MOE | Edit | Chirality | Geometry menu action to set the constraint to the current geometry.

  6. Choose an appropriate forcefield potential for minimization. The current forcefield setting is shown on the left of the MOE Footer:

    To change the forcefield, click the button showing the forcefield name, or use MOE | Window | Potential Setup

    1. Load the PEF95SAC Forcefield
    2. Press Fix Charges
    3. Close the Potential Setup panel

  7. To minimize the structure, choose:

    MOE | Compute | Energy Minimize

    Without changing the default values in the Energy Minimization panel, press OK. MOE will proceed to minimize the structure.

  8. Press View in the Carbohydrate Builder to maximize the view of the current molecule so that it fills the MOE Window. (You can also choose MOE | Render | Center from the MOE Window.) Change the rendering with MOE | Popup | Atoms and click on the ball and stick icon.

    The following picture represents the Sucrose (table sugar) molecule rendered in Ball-and-Stick mode:

    Notice the intramolecular hydrogen bond which is displayed. MOE annotates these interactions by default. If you whish to turn them off, open the Contacts panel from MOE Footer | Contacts. Then in the popup panel go to the interaction matrix for the hydrogen bonds and click on the Lig/Lig button.

  9. You can rotate, translate and zoom in on the molecule in the MOE Window:
    • One way to rotate around or translate along one of the XYZ coordinate axes is by using the trackball at the bottom of the RHS. Zooming toward or away from the molecule is done with the Zoom wheel.
    • A second way to rotate the molecule is to click and drag the middle mouse button over it in the MOE Window. To translate the molecule in any direction, press Shift and drag the middle mouse button. Zooming is done using the Ctrl key and the middle mouse button.
  10. If you wish to save the molecule to a file, choose:

    MOE | File | Save | sucrose.moe

  11. Close the molecular system:

    MOE | File | Close

Exercise 2: Building Cellulose Oligosaccharide

This exercise illustrates the ease with which regular oligosaccharides can be constructed using the Carbohydrate Builder. In it, you will build cellulose, a polysaccharide of beta-D-glucopyranose connected by 1-4 links.

[(1->4)bDGlcp]n

  1. Begin the tutorial with a blank MOE session, by using MOE | File | Close if necessary.

  2. Change the Library option to pyranose.
    Change the Unit option to bDGlcp.
    Turn on the Auto-Select Anomeric checkbox.

  3. Click the Build button to instantiate the first saccharide unit.

  4. Press the View button to maximize the view of the molecule in the MOE Window.

    The Carbohydrate Builder automatically selects the anomeric O of the newly added residue, which is the O1 atom. Have a look at the MOE Footer that shows the name or number of the selected atom's chain and residue. The atom name O1 is shown in the text box while the element is shown to the right in square brackets. By holding the mouse pointer over the displayed text, more information about the atom is shown, while holding the mouse pointer to the right of the text shows information about the number and properties of the selected atoms.

    Recall that glycosidic linkages are 1-4 in cellulose. The next step is to link the bDGlcp residue to the 4 position.

  5. To the right of the graphical representation, select O4.
  6. Set the phi and psi Torsion Angles to 35 and -28 degrees respectively. These values are reasonable for cellulose chains [Geßler 1994]. The omega torsion angle need not be specified as it is not defined for the 1-4 link in cellulose.

  7. Press the Build button once again.

    This connects the 4 position of the bDGlcp unit to the anomeric position of the first residue. Because the Auto-Select Anomeric O of Added Residue option is still active, the anomeric oxygen (position 1) of the newly added residue is automatically selected.

  8. At this point, examine the 3D structure of the bDGlcp(1-4)bDGlcp disaccharide by rotating the molecule or increasing the zoom perspective.
  9. The link point and torsion settings allow you to add more bDGlcp units by repeatedly pressing the Build button. The Carbohydrate Builder will extend the chain one residue at a time.

    Note: You can open the Sequence Editor (by pressing SEQ in the top right corner of the MOE Window) to keep track of residues as the chain lengthens.

  10. If you wish to save the molecule to a file, choose:

    MOE | File | Save | cellulose.moe

  11. Close the molecular system:

    MOE | File | Close

Exercise 3: Building an N-linked Oligosaccharide

The carbohydrate structure below is an example of a high-mannose [Robyt 1998] N-linked oligosaccharide. This oligosaccharide is linked to the side-chain N of the amino acid L-asparagine (ASN).

To build this structure:

  1. Use the Protein Builder to create the ASN amino acid in the system:

    MOE | Edit | Build | Protein

    Click on the ASN button in the Residue section of the Protein Builder and press Close.

  2. In the MOE Window, select either the side-chain amide N atom or a hydrogen atom attached to it.

  3. You will now attach the selected amide N to position 1 of the first residue to add:
    1. Open the Carbohydrate Builder if it is not already open.
    2. Select the amino-pyranose library.
    3. Select the bDGlcNAc unit.
    4. Select O1.
    5. Press the Build button.
  4. Select either the hydroxyl oxygen or hydrogen atom at position 4 (O4) on the newly added bDGlcNAc residue.

    5. Tip  Have a look at the MOE Footer while selecting different atoms of different residues. The annotation in the footer will change to display the residue and atom name of the currently selected atom.

  5. This selected atom at position 4 will link to the next bDGlcNAc residue:
    1. Make sure bDGlcNAc and O1 are still selected.
    2. Press Build to add this residue to position 4 of the first bDGlcNAc residue.
  6. Select either the hydroxyl oxygen or the hydrogen atom at position 4 (O4) on the newly added residue. Add the bDManp residue:
    1. Select the pyranose library, and the bDManp unit.
    2. Select O1.
    3. Press the Build button.
  7. Select the hydroxyl oxygen or hydrogen atom at position 3 (O3) of the newly added bDManp residue. Add an alpha-mannose residue to position 3 of the branching beta-mannose unit:
    1. Make sure aDManp and O1 are selected.
    2. Press the Build button.
  8. To add the alpha-mannose to position 6 (O6) of the beta-mannose unit, select the hydroxyl H or O atom at position 6 of the bDManp unit. You will now add the second alpha-mannose unit to the branching beta-mannose unit:
    1. Make sure aDManp and O1 are still selected.
    2. Press the Build button.

    You have now finished building the core-pentasaccharide [Montreuil 1975] of an N-linked oligosaccharide.

  9. Save this structure as it will be used in the next exercise:

    MOE | File | Save | oligosac.moe

Exercise 4: Editing a Carbohydrate Structure

The Carbohydrate Builder allows you to edit existing carbohydrate structures. Before starting this exercise, retrieve or build the pentasaccharide created in the previous exercise. The various edit functions will be described by performing actions on this carbohydrate structure.

Open the Sequence Editor to see the pentasaccharide sequence. Choose MOE | Window | Sequence Editor. The Sequence Editor will be used to highlight and select residues. Turn on SE | Selection | Synchronize to allow the atom selection in the MOE Window to reflect the residue selection in the Sequence Editor.

Inverting a Center

Suppose you want to change the bDManp unit to aDManp. The difference between the a and b forms of DManp is the chirality around the anomeric carbon atom. The b form of DMan can be converted to the a form by inverting the anomeric carbon.

  1. Deselect any selected atoms and select the anomeric carbon of the central bDManp unit (C1), from Step 6 of the previous exercise.
  2. In the Carbohydrate Builder, press the Invert (i) button to invert the center. The bDManp residue becomes an aDManp residue.

Note: Invert will not allow you to invert a bridgehead atom. Any attempt to do so will result in an error message.

Mutating a Residue

Suppose you want to mutate the tri-substituted mannose unit to a beta-D-galactopyranose unit. Because all the linkages out of this residue are glycosidic, and beta-D-galactose can form glycosidic links at the same positions as mannopyranose, the residue can be replaced directly.

  1. Make sure that no atoms are selected in the MOE Window.
  2. Select all the atoms in the mannose unit by pressing Ctrl as you click on one of them. Alternately, you can select all the atoms of the mannose unit by using the Residue popup menu in the Sequence Editor.
  3. Select the bDGalp unit from the pyranose library and click Build. The Carbohydrate Builder replaces the mannose unit with the bDGalp unit. The linkage points of the original mannose unit are preserved and applied to the galactose.

Mutation of the oligosaccharide is complete.

Notes on mutations:

  • Mutations performed with this operation only work if the new residue can form glycosidic links at the same positions as the original residue. For example, attempting to mutate the newly added galactose unit to a bDXylp by following the above steps will result in the error message "Cannot Replace Residue: Inappropriate Links". This is because bDXylp cannot form a 6-1 link, making it an inappropriate residue to replace galactose, which forms a 6-1 link to an alpha-mannose.
  • Only residues linked through O-glycosidic bonds can be mutated by this procedure. For example, the b-DGlcpNAc residue attached to the ASN amino acid is linked through an N-glycosidic bond. This residue cannot be mutated using the pushbuttons. If the selected atoms are not appropriate for a residue mutation, residue pushbuttons are grayed out.

Inverting a Residue

Suppose you want to invert the newly mutated bDGalp residue from the D to the L form. All linkage points to this residue are O-glycosidic bonds, thus enabling the inversion of the residue using the Invert button.

  1. Deselect all atoms in the system and select all the atoms in bDGalp (the residue you want to invert).
  2. Press Invert to invert the residue from D to L.

The galactose has now been inverted to the L form. Note that only free (unbonded) residues and residues linked through O-glycosidic bonds can be inverted using this procedure. The bDGlcNAc residue attached to ASN through a non O-glycosidic bond cannot be inverted using this method. If the selected atoms are not appropriate for an inversion, the Invert button will be disabled.
