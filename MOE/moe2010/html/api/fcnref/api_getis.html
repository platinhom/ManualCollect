Description

Each of the svl_GetIs... methods checks an SVL variable for a specific type of data. If the data item is of the requested type the function returns 1; otherwise, the function returns 0. The direct methods apply to the pos-th element of the variable v, or the entire v if pos is 0. The context methods work apply to the element at the current position of the current Get-variable.
