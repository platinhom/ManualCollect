API Get Functions

Syntax

    int svl_GetLength(
	svl_var v, 
	unsigned pos
    );

    svl_error svl_GetLLength(
        svl_var v, 
	unsigned pos, 
	int *len, 
	unsigned range
    );

    svl_error svl_GetCheckLength(
        svl_var v, 
	unsigned pos, 
	int len1, 
	int len2
    );

    svl_error svl_GetCheckLLength(
	svl_var v, 
	unsigned pos, 
	int len1, 
	int len2, 
	unsigned range
    );

    int svl_GetCvLength(void);
    svl_error svl_GetCvLLength(int *len, unsigned range);
    svl_error svl_GetCvCheckLength(int len1, int len2);
    svl_error svl_GetCvCheckLLength(int len1, int len2, unsigned range);

Description

These functions return or check either the length or the laminated length of an element of an SVL variable. The direct versions of the methods apply to the pos-th element of variable . If pos is 0, the methods apply to v itself. The context Cv methods apply to the element at the current Get-position of the current Get-variable.

svl_GetLength and svl_GetCvLength return the number of elements of the data item.

svl_GetLLength and svl_GetCvLLength return the length of the transpose, or the laminated length of the data item. A vector that is composed of sub-vectors can be visualized as a collection of sheets or laminae conformable to one another. The sub-vectors (laminae) unit-extend to the longest sub-vector of the data item or of a sequence of data items.

If range is 0, the data item is either v(pos), or the current element of the current Get-variable. (If pos=0, the data item is v itself.)

If both pos and range are non-zero then the data item is 

    [v(pos), ..., v(pos + range - 1)]

If pos = 0 and range is non-zero then the data item is 

    [v, ..., v]
(Note that the laminated length of [v, ..., v] is equal to the length of v.) The context methods behave identically to the direct methods, except that v is substituted by the current Get- variable and pos is substituted by the current Get- position.

The length is returned in *len. The starting contents of *len must be either 1 or the expected length of the incoming vector.

svl_GetCheckLength checks to see if the length of the data item (vector) at index pos in the SVL variable referenced by v is either len1 or len2. If it is neither, then return an error, otherwise return NULL. If pos is 0, the length of the SVL variable itself is returned.

svl_GetCheckLLength checks whether the laminated length (see svl_GetLLength) of the data items, as specified by pos and range, of the SVL vector v, are of either length len1 or len2. If they are not, an error is returned, otherwise NULL is returned.
