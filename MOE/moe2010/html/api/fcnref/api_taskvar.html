MOE 2010.10

API Task Functions

Syntax

    svl_var svl_TaskVar(svl_task task);
    int svl_TaskProcall(svl_task task);
    int svl_TaskId(svl_task task);

Description

svl_var svl_TaskVar(svl_task task);

Return the handle of the SVL variable associated with the given task. This SVL variable is used for passing both the input argument and the output result.

int svl_TaskProcall(svl_task task);

Return 0 if the associated SVL function was invoked as a function and is expected to return a result value. Return 1 if the associated SVL function was called as a procedure and its result value will be discarded.

Note: The API function svl_TaskProcall is equivalent to the SVL built-in function call[] (with null argument). 

int svl_TaskId(svl_task task);

Return the task ID of the current task.

Note: The API function svl_TaskId is equivalent to the SVL built-in function task_id[] (with null argument).
