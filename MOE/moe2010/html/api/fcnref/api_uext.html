API Functions

Syntax

    svl_error svl_Uext(int len, int *n);

Description

Checks if a vector of length *n can be unit-extended to a vector of length len or vice versa. If the answer is negative, the function returns an error. If the answer is positive, the contents of *n is reset to the length of the unit-extended vector.

Note: The contents of *n change only if it is equal to 1.
