Clustering the Database Using Values

The value-based cluster method is a clustering scheme based on the exact values in the database. Thus, two molecules are considered to belong to the same cluster if they have identical values in each of the selected fields.

Open a database in a Database Viewer and choose DBV | Compute | Descriptors | Value Clusters. The Value Cluster panel appears:

Database Path and filename of the database to cluster.

If Selected Entries Only is turned on, then only entries that are selected in the Database Viewer will be used in the calculation (the rest will be ignored).

Fields List of all fields in the database. Select the fields to be used in the cluster analysis. If no fields are selected in the panel, then all fields will be used.
Cluster Code If turned on, then an integer field, using the name specified in Cluster Field, will be created that will contain the cluster numbers of each entry used in the calculation.
Cluster Weight If turned on, then an integer field, using the name specified in Weight Field, will be created that will contain the weight value for each entry in the calculation. This value is equal to the inverse of the number of members in the cluster of each entry. For example, if a particular entry is assigned a code that is identical to three other entries, then a value of 0.25 will be written.

To start the calculation press OK. Once the calculation is complete, you can sort the database on the output cluster codes to group clustered entries together.

SVL Commands

db_ValueCluster
