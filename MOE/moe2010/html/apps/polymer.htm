The Polymer Properties application in MOE implements Jozef Bicerano's methodology described in [Bicerano 1996]. The properties predicted rely on the specification of only a monomer repeat unit of a polymer along with its atomic coordinates in an extended conformation. This repeat unit specification is analyzed and its polymer properties are predicted using connectivity indices and specialized structural fragment descriptors. In MOE, a monomer repeat unit is a connected molecule (in a single chain) with two named hydrogen atoms indicating polymerization connection points. The chain name is used as the polymer name. The hydrogen at the tail of the chain must be named A0 and the hydrogen at the head of the chain must be named A1. For example, the polystyrene monomer unit is represented with:

Select the indicated hydrogen atoms and use the MOE Footer to name these atoms. Alternatively, MOE | Edit | Build | Polymer can be used to specify the attachment points and name the polymer chain. An alternating polymer –(A–B)n is specified by creating the repeat unit for (–A–B)n.

Running the Calculation

Use MOE | Compute | Simulations | Polymer Properties to open the Polymer Properties window. Press Predict to calculate polymer properties for the repeat units currently loaded in MOE. If successful, the window will contain a list of polymer properties:

If more than one polymer is loaded, choose the polymer of interest in the Polymer option menu to display its respective properties and values in the list. MOE predicts over 50 polymer properties. They are divided into two groups: Scalar Properties and plottable properties, listed in the Property option menu. The following Scalar Properties are calculated:
