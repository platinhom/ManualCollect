This chapter describes how to specify repeating units (monomers) and to combine monomers to create polymeric molecules.

Building Polymers

The Polymer Builder is used to build polymer molecules composed of either a single repeating unit (homopolymers) or two or more repeating units (copolymers).

To open the Polymer Builder, choose MOE | Edit | Build | Polymer. The following panel appears:

Polymer Builder Panel Reference

Polymer Name Set the name of the resulting polymer chain. Residue names in the polymer chain will be set to the first three letters of the monomer chain names, or the first three letters of the result chain name if the former is null.
Sequence Choose how to combine the repeating units. Homo builds a homopolymer using a single repeating unit, Random builds a copolymer by randomly selecting from the input repeating units, Alternating will alternate between all input repeating units, and Block will create a block of n repeats of each unit in turn, where n is specified in the Multiplicity column. Only the first repeat unit menu will be active in Homo Sequence mode. In Alternating mode, the lowest number in the Multiplicity column determines the number of repeats each unit.
Dihedral Mode Set the backbone angles of the polymer. In Angle mode, each bond between monomers is set to the specified angle. In Helix mode, all bonds between backbone atoms are set to the specified angle. In Random mode, each bond between monomers is set to a random angle.
Repeat Units Select which repeat units are to be combined in the final polymer, the number of times each unit will appear (Multiplicity) and their Tacticity, which is the relative arrangement of the monomer sidechains with respect to polymer backbone. Chains will be listed as a potential monomer if they contain a single connected molecule which has head and tail atoms defined, i.e. hydrogen atoms named A1 and A0 respectively. Setting a tacticity of 0 (zero) means that all the sidechains will be on the same side of the polymer backbone (isotactic). Setting tacticity to 1 will create a syndiotactic polymer, where the sidechains have alternate from one side of the backbone to the other along the chain. Non-integral tacticity values are interpreted as the probability of the sidechain appearing on a particular side of the backbone.
Database A database of 253 standard repeating units is supplied with MOE in the file $MOE/lib/polymer.mdb. Clicking Open will open this file in a Database Viewer, and Close will close it again. The checkboxes can be used to restrict the visible entries to those that contain a particular chemical group; if multiple checkboxes are checked then all the specified groups must be contained in the repeating unit. If the desired monomer is not available in the database, a new monomer can be defined using the Define Repeat Unit tool, as described below.

Press the OK button to create the polymer.

Creating New Repeat Units

If the desired monomer is not already defined in MOE's database of standard repeat units (contained in $MOE/lib/polymer.mdb), a molecule can be marked as a monomer for use with the Polymer builder using the Define Repeat Unit application. To launch the application, choose MOE | Edit | Build | Repeat Unit.

To define a new repeat unit, first build a molecule in MOE using the Builder, for example. Select a hydrogen atom and click Define Tail to set the tail of the monomer, select a second hydrogen atom and click Define Head to set the head, then select the heavy atoms that link the head and tail and click the Set Backbone button. The name of the monomer can be set by entering it into the Name text box and clicking Apply. The monomer can now be used with the Polymer Builder application.

Example: Building polyvinyl chloride

Polyvinyl chloride (PVC) is formed by a free-radical mechanism, and as such it is an atactic polymer with an amorphous structure. To build a PVC molecule using the Polymer Builder in MOE:

  1. Launch the Polymer Builder.

    MOE | Edit | Build | Polymer

  2. Select the monomer unit.
    In the Polymer Builder panel:
    1. Press Open to open the database of standard repeat units $MOE/lib/polymer.mdb into a Database Viewer.
    2. Turn on the Halide box. In the Database Viewer, only those repeat units that contain halogen atoms will be shown.
    3. The polyvinyl chloride unit is entry 209 and has the name "poly(vinyl chloride)". Load this entry into MOE:

      DBV | Molecule Cell | Popup | Send to MOE.

    4. In the Polymer Builder panel, set the repeat unit name by entering polyvinyl chloride into the Polymer Name box.

  3. Set the polymer properties.
    1. Choose Random from the Dihedral Mode menu.
    2. Set the Multiplicity to 20 and the Tacticity to 0.5.

    3. Build the polymer.
      Press OK in the Polymer Builder panel. Use MOE | Render | Center to center the the resulting PVC polymer in the MOE Window.
