Wheel Widget

Description

A graphical dial for entering numeric data. As the wheel is turned (by dragging the mouse cursor over it), its associated numeric value is increased/decreased. If minimum and maximum values are specified (if one is specified, the other must be too), and if min < max, the dial's range of rotation is limited. Otherwise, the Wheel can spin continuously. In all cases, one complete rotation of the Wheel corresponds to 2*PI radians. In continuous-rotation mode, values are limited to the range [0..2*PI ]; i.e. the values wrap. In specified-range mode, the initial value of the wheel is the middle of the range. In continuous-range mode, the initial value is zero.

Attributes

Wheel inherits all core attributes and defines the following attributes:

Attribute Description Values Default
min Minimum value number 0
max Maximum value number 0
resetButton If 0, then the reset button will not be drawn. 0/1 1
vertical Specifies orientation of wheel 0=horizontal 1=vertical 0
width Size of wheel in character widths 1 to 255 20

Trigger

Mouse drag movement on the wheel.

Data Value

The number corresponding to the graphic dial position. The value is limited by the max and min values specified. If unspecified, the wheel has no range limit. The default value is 0 in this case.
