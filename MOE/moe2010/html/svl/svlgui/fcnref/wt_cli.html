CLI Widget

Description

The CLI widget is a text field that operates like a CLI. It will automatically perform as all other CLI's in the system and is linked in to the CLI history buffer.

Attributes

CLI inherits all core attributes and defines no other attributes.

Trigger

None.

Data Value

None.
