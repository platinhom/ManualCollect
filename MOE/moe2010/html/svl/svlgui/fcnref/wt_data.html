Window Functions

Syntax

    name_value_pairs = WindowGetData [ win, widget_names ]
    WindowSetData [ win, widget_name_value_pairs ]

    name_value_pairs = WindowValues wkey

Description

name_value_pairs = WindowGetData [ win, widget_names ]

Gets the values of the widgets (specified by name tokens in the vector widget_names) that belong to the specified window. The window must have already been created.

WindowSetData [ win, widget_name_value_pairs ]

Sets the values of the widgets that belong to the specified window. widget_name_value_pairs is a tagged vector of name-value pairs. The window must have already been created.

win can be either the window key (returned by WindowCreate) or the name of the window (if it is not a prompter) as set by the windowName attribute.

name_value_pairs = WindowValues wkey

Returns a tagged vector of names and values of all named widgets in the specified window.
