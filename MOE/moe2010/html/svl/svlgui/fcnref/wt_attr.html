Window Functions

Syntax

    WindowSetAttr [ wkey, attribute_value_pairs ]

Description

Sets widget attributes of the window specified after the window has been created. The window must have already been created. The attributes are passed as vectors of attribute-value pairs, tagged by widget name.

Here is an example of usage:

    function main[]
        local wkey = WindowCreate [
            title : 'Test Window',
            Text : [
                name : 'textfield',
                title : 'Enter a number:'
            ],
            Radio : [
                name : 'button',
                title : 'Radio:',
                text : ['A','B','C','D','E','F','G']
            ]
	];

        WindowSetAttr [ wkey, [ 
            textfield : [
		len : 20,
		onTrigger : 'return'
	    ],
            button : [
		columns : 3
	    ]
        ]];
        ...
    endfunction
