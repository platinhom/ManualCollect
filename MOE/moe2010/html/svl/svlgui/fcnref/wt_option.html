Option Menu Widget

Description

A pull-down menu of options of which exactly one is selected at any given time. The name of the selected option appears in the widget.

Attributes

Option inherits all core attributes and defines the following attributes:

Attribute Description Values Default
text String labels Vector of tokens []
columns Number of columns for layout 1 to 127 1
type Return value data type 'char', 'int' 'char'
minWidth Minimum width (in characters) of each of the menu items 0 to 127 0
foreground Option label text color WINDOW_COLORS 'black'
background Option label background color WINDOW_COLORS 'background'
font Option label text font WINDOW_FONTS 'medium'
graphics A vector of graphics object keys (see gr_create) or graphics content (see gr_data). The number of elements in the vector defines the number of buttons, and must be consistent with the text attribute. Vector of keys or content. 0
grPosition Layout of graphic object with respect to text, when both are present 'left', 'right', 'above', 'below', 'overlay' 'left'
grScale Scaling policy for the graphic object 'none', 'anisotropic', 'isotropic', 'isotropic_shrink' 'isotropic'

Trigger

A change in the option menu displayed item. Selecting the same item will not cause a trigger.

Data Value

The value of the option menu is returned as a token or one-based index (depending on the state of the type attribute) denoting the currently selected menu item. The default value is the first item.
