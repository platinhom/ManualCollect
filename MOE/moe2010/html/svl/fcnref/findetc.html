Pattern Matching Functions

Syntax

    matching_tokens = findmatch [ ['pat1', 'pat2',...], tokens ]
    match_mask = m_findmatch [ ['pat1', 'pat2',...], tokens ]
    match_indices = x_findmatch [ ['pat1', 'pat2',...], tokens ]

Description

matching_tokens = findmatch [ ['pat1', 'pat2',...], tokens ]

Returns the elements of tokens that match at least one of the given patterns ['pat1', 'pat2',...]. The input arguments must be flat vectors. 

match_mask = m_findmatch [['pat1', 'pat2',...], tokens]

Returns a mask which, when applied to vector tokens (e.g. using mget), yields those elements (if any) that match any of the specified patterns. 

match_indices = x_findmatch [['pat1', 'pat2',...], tokens]

Returns the indices of elements of vector tokens that match any of the given patterns.

Match Patterns

The match pattern is a sequence of match commands. The entire pattern must be matched, or else the match fails. Match commands are as follows:

* matches 0 or more of any character
? matches any single character
@ matches any character in [a..z, A..Z, _ ]
# matches any digit [0..9]
[...] set of matchable characters, one and only one match must succeed; ranges (using - symbol) allowed
[^...] set of characters which must not be matched

Note: Special characters (* ? @ #) must be put in square brackets. 

    svl> findmatch [ 'x*', ['xyz','wxyz'] ] 
    'xyz' 
    svl> findmatch [ ['x*','*b'], ['xab','xyz','lab'] ] 
    [ 'xab','xyz', 'lab']
    svl> findmatch [ '?x?', ['wxy','xyz','xy'] ]
    'wxy'
    svl> findmatch [ '@@', ['ab','a9'] ]
    'ab' 
    svl> findmatch [ '##', ['12','a9'] ]
    '12'
    svl> findmatch [ '[#]#', ['#9', '10', '#4'] ]
    ['#9', '#4'] 
    svl> findmatch [ '[abc]x', ['ax','bx','xx'] ]
    ['ax','bx']
    svl> findmatch [ '[^abc]x', ['ax','bx','xx'] ] 
    'xx' 
    svl> m_findmatch [ 'x*', ['xab','xyz','bc'] ] 
    [1,1,0]
    svl> x_findmatch [ ['x*'], ['xab','xyz','bc'] ]
    [1,2]
