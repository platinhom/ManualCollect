MOE 2010.10

Special Functions

Syntax

    val = erf x
    val = erfc x
    val = igammap [z, x]
    val = igammaq [z, x]
    val = logbeta [z, w]
    val = loggamma z

Description

SVL provides a number of special functions that appear often in physics and statistics applications but are difficult to evaluate accurately or have no closed form formula.

Function Value Description
erf x The error function.
erfc x The complementary error function.
igammap [z,x] The integral of the left tail of the Gamma probability density function (the incomplete gamma function).
igammaq [z,x] The integral of the right tail of the Gamma probability density function (the complementary incomplete gamma function).
logbeta [z,w] The natural logarithm of the beta function.
loggamma z The natural logarithm of the gamma function.

The unary operators (like erf, erfc and loggamma) apply to all scalar values in the argument vector (no matter how deeply nested). The binary operators (like igammap and logbeta) match elements of the arguments in pair no matter how deeply nested. Unit extension applies.
