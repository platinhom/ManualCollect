MOE 2010.10

Logical Functions

Syntax

    val = andE [a,b]
    val = andE [a,b,c,...] // reduction

    val = orE [a,b]
    val = orE [a,b,c,...] // reduction

    val = xorE [a,b]
    val = xorE [a,b,c,...] // reduction

    val = a and b // infix notation
    val = a or b // infix notation

Description

Element-wise AND, OR and XOR. Arguments must be of the same shape (allowing for unit extension), and contain only numerical data. The infix notation for andE is and, whereas the infix notation for orE is or. xorE has no infix form.

Note: All operands are evaluated before application of the logical operator. These are not "short-circuited" operations.

    svl> 1 and 2
    1
    svl> 0 and 2
    0
    svl> [1,0] and [2,2]
    [1,0]
    svl> orE [ 0, [2,0] ] // unit extension
    [1,0]
    svl> xorE [3,4]
    0 
    svl> xorE [3,0]
    1 
    svl> xorE [3,3,4] // xorE [ 3, xorE [3,4] ]
    1 
    svl> xorE [0,0] 
    0
