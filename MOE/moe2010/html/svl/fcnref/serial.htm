MOE 2010.10

Serial Numbers

Syntax

    key = serial n

Description

key = serial n
Returns a unique serial number (numeric key). Each number is a positive integer with its lowest n bits set to 0, where 0 ≤ n ≤ 20. The returned serial number is guaranteed to be unique for the lifetime of MOE across all hosts and all values of n. Within the same value of n, the returned serial numbers will also be ascending.

The result of the function is intended to be used as access handles to user data. The lowest n may be used to store additional, application-specific information, such as data type or hash code.

The function accepts (and returns) a vector argument of any shape.

Note: Since the number of bits available to store unique serial numbers is limited, the function is intended to generate only a moderate total amount of serial numbers. For large n, the rate of generating new serial numbers may become limited. It is recommended not to exceed the average rate of k new serial numbers per second, where k = bitshl [1, 20 - n ].
