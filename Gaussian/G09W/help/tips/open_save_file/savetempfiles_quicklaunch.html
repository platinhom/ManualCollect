Saving Temporary Quick Launch Files

When a Gaussian Quick Launch calculation is done using the default "Temp File" option, the Gaussian job file (*.gjf or *.com) and the corresponding result files (*.log, *.chk, etc.) are stored in a temporary scratch directory with a unique, arbitrary base name.   These files are deleted when the last view window corresponding to one or more of the files is closed or when GaussView is exited.

To save such temporary Gaussian Quick Launch files to a non-scratch directory for later use, select "Save Temp Files" from the "File" menu.
