Mouse and Keyboard Bindings for View Windows

A molecule in a view window can be zoomed into and out of the screen Z-axis using any of the following methods:

  • Scroll with the mouse wheel
     
  • Move the mouse pointer along the screen Y-axis with the right mouse button down
     
  • Move the mouse pointer along the screen Y-axis with the left mouse button down and the Ctrl key pressed
     
  • Use the up and down arrow keys with the Ctrl key pressed
