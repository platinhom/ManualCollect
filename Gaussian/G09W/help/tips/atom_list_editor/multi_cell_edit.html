Multi-Cell Editing in an Atom List Editor

Most columns in an Atom List Editor support multi-row editing.

  1. Select the rows (atoms) you want to edit.
     
  2. Edit a cell which is both in the desired column (e.g., "Symbol") and in one of the selected rows.
     
  3. Each cell in the column of the edited cell whose row is selected will be changed to the value of the edited cell.
