Use a View Window to Select a Row in an Atom List Editor and / or a Peak in an NMR Spectrum.

Right-clicking on an atom in a View Window with the Shift key down will:

  • Cause the View Window's Atom List Editor (if showing) to scroll to that atom's row and select it.
  • Cause the View Window's NMR spectra (if available and showing) to select the peak corresponding to the atom.
