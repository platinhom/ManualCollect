Copy Summary Dialog Information to Clipboard

When a molecule's summary dialog is displayed, pressing Ctrl+C will copy any user selected items to the systems clipboard.  To select items:

  • Pressing Ctrl+A will select all items in the dialog
     
  • An item in the dialog may be selected by clicking it with the left mouse button
     
  • A range of items in the dialog may be selected pressing and holding the left mouse button
     
  • Multiple sets of items may be selected by pressing the Ctrl key while selecting using the left mouse button
