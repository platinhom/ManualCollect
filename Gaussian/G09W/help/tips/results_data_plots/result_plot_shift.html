Shifting Result Spectra and Data Plots

For result spectra (NMR, UV/Vis, etc.) or result data plots (IRC, Scan, etc.), the spectra/plot may be shifted in the the horizontal and/or vertical directions.  For a set of spectra/plots, each can be adjusted independently.  For a few types, shifting in the vertical direction (y-axis) is not allowed.  To set the shift:

  • On the spectrum/plot of interest, click the right mouse button to bring up its context menu
     
  • From the context menu, select Properties... at the bottom to bring up the Plot Properties Dialog
     
  • In either the Y-Axis (top) box or the X-Axis (bottom) box:

type in the desired value in the field next to origin =

                       or

select the current button to shift the plot relative to the current active point

  • To accept the change and exit the Plot Properties Dialog, click the left mouse button on the okay button
