NoDensityFit

DESCRIPTION

Turns off use of density fitting in pure DFT calculations by default. Density fitting is not on by default in the program, so this keyword is primarily used to override density fitting options set in your Default.Route file which includes the DensityFit keyword.

NoDenFit is a synonym for NoDensityFit.

AVAILABILITY

Applies only to DFT calculations using pure (non-hybrid) functionals.

RELATED KEYWORDS

DensityFit
