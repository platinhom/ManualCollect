Databases Distributed with MOE

A number of useful databases are distributed with MOE. Some are designed for use with specific MOE applications while others are for more general use.

Small Molecule Databases

Leadlike Conformer Database. The MOE Conformation Database set, found on the DVD (leadlike_conf_###.mdb.gz), contains 3D conformations in MOE Molecular Database format for approximately 650,000 (Oprea) lead-like chemical structures. These are ready to be used for pharmacophore searching, fingerprint analysis, and so on. The structures were obtained from the following chemical suppliers' public catalogs:

The conformations were generated with the MMFF94x forcefield using Conformation Import in MOE. A limit of 4.5 kcal/mol strain energy was imposed with a 500 conformation limit for each molecule. Duplicate conformations were removed with a heavy atom RMSD tolerance of 0.6 angstroms (0.75 angstroms for conformations with strain greater than 3.5 kcal/mol).

Linker Database. The MOE Linker Database contains conformations for approximately 750,000 fragments suitable for scaffold replacement methodologies. The fragments were generated using MOE's SD Pipeline Command Tools from 4.3 million unique structures culled from medicinal chemistry literature and various vendor catalogs. On average there are 24 conformations per molecule. The database is available on the DVD (linker_2010_10.mdb).

RECAP Fragments. recaplib.mdb is the output of RECAP analysis on a collection of ~750,000 diverse lead- and drug-like chemical structures from vendor catalogs and medicinal chemistry journals. It can be used with the RECAP Panel.

Protein Databases

Remediated PDB. A complete copy of the PDB database as of September 21, 2010 can be found on the DVD (pdb_2010_09_21.mdb). Each record has been processed and converted into MOE format, and where possible ligands corrected according to the CIF dictionary. Key fields such as COMPND, HET and KEYWDS have been extracted, to facilitate searching the database. This database may be searched with the Protein Database Panel.

Non-redundant PDB. $MOE/lib/pdb.mdb is a database of non-redundant chains derived from the PDB. Chains were clustered such that all pairs of chains within a cluster had 90% pairwise sequence identity as well as similar structure. One representative was chosen from each cluster based on resolution, R free, and number of missing residues not at the ends of chains. Note that ligands and solvent are not included in this database, only the protein chains are stored. This database may be searched with the Protein Database Panel and also serves as a set of possible templates for use with Homology Modeling.

Protein Families. The non-redundant set of PDB chains is further clustered structurally to form families. These structure-based sequence alignments are stored in the $MOE/lib/pdb.fam file in plain text. Small families are enhanced with additional sequences from UniRef 50. This file can be used during Homology Searching to identify possible templates to a query.

Antibody Families. Similarly to the Protein Families, the antibody sequences in the PDB have been grouped into families and can be used when making homology models of antibodies. $MOE/lib/fab.fam contains all antibody chains clustered by organism and type. $MOE/lib/fab_ref.fam contains two reference alignments, one for heavy and one for light chains, and is used to classify antibody chain types. $MOE/lib/fab_germ.fam contains several VL-kappa, VL-lambda and T Cell Receptor sequence families which can be used to identify other members of these sequence families.

Rotamer Library. The Rotamer Library in MOE is a collection of all-atom sidechain conformations derived from high quality x-ray crystal structures. These conformations differ from traditional rotamer libraries in that bond length and angle variations are permitted. The intent is to provide adequate coverage of the conformational space available to a sidechain and not just encode dihedral preferences. The rotamers are located in $MOE/lib/amino.mdb in a packed conformation field associated with each sidechain.

Protein-Ligand Complex Database. $MOE/sample/mol/complex.mdb contains a set of structures from the PDB with experimental binding affinities. For each entry, ligand stores the ligand for which binding was measured, and receptor, the nearby chains. image contains symmetry-related molecules which are near the ligand, if any, and water contains any water chains. dG contains the experimental binding affinity of the ligand in kcal/mol, obtained from various sources in the literature. degeneracy indicates the number of copies of the ligand found in the PDB file; each copy will have its own row in the database. The other fields contain summary information about the structure from the PDB file. This database can be used for validation studies.
