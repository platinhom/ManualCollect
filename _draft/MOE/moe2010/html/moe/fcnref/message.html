MOE 2010.10

Miscellaneous Functions

Syntax

    msg_key = Message [msg_key, 'message']

Description

Handles the creation, modification and deletion of messages that are displayed in the MOE Window. The behavior of the function is summarized as follows:

  'message' == [] 'message' <> []
msg_key == 0 Does nothing Creates new message
msg_key <> 0 Destroys message Modifies message

Whenever Message is called with 'message'<>[], 'message' is displayed in the MOE Window. The message is removed as soon as the task that created the message is completed. As a consequence, a call to Message at the CLI on its own will result in the associated message token being put up and removed in such rapid succession that the message will not appear to have been displayed in the MOE window at all.

Messages are meant to be used within a function that wants to display progress messages while a long operation is being performed. The key of the message is returned if the message survived the operation (the message was not destroyed).

The following function illustrates the use of the Message function: 

    global function message_example []
        local msgkey = Message [ 0, 'Performing a long operation' ];

	    // msgkey is destroyed when the task is finished. 
	    // Sleep here for 5 seconds in order to see the
	    // message in the MOE Window.

	sleep 5;
    endfunction
