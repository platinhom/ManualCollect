Pharmacophore Functions

Syntax

    [washed_keys, del_smi] = ph4_Wash [ atom_keys, options ]
    ph4_WashMDB [ mdb, 'mol_field', 'dest_field', options ]

    options = [
        one_component   : flag,
        delete_atoms    : flag,
        coord_scale     : flag,
        set_fc          : flag,
        acid_base       : flag,
	add_h           : flag
    ]

Description

These functions are used to "wash" molecules. This procedure converts a molecule from its neutral gas phase to its solvated state in aqueous solution. The functions optionally delete all solvent molecules. 

[washed_keys, del_smi] = ph4_Wash [ atom_keys, options ]

Returns the atom keys of the solute molecule(s) in their washed state. All atoms which were deleted or which were marked for deletion (but were not deleted due to the delete_atoms flag being 0) are returned as a molecule input language string expression del_smi.

If options (described below) is not set, the default value for each of the options is 1. 

ph4_WashMDB [ mdb, 'mol_field', 'dest_field', options ]

Washes all of the molecules in the molecular database. The database is specified either by database key or by name token. The molecules are taken from the specified molecule field, and the washed molecules are written to the specified destination field, which can be the same as the source molecule field.

If options (described below) is not set, the default value for each of the options is 1.

Both functions support the following options:

one_component:flag
If flag is non-zero, then ph4_Wash performs a series of filtering operations on the data after which, if more than one connected component remains, the component with the most heavy atoms will be retained. For more information on the filters, see Washing Molecules.

Note: Atoms will only be deleted if the delete_atoms flag is set.

delete_atoms:flag
If flag is non-zero, atoms will be deleted from the system. All molecules recognized to be part of the solvent medium -- water molecules, counter-ions, simple acids and bases, common solvents, common salts -- are deleted. If the one_component flag is set, then all but one of the remaining molecules are also deleted. If flag is 0, all atoms are preserved.

coord_scale:flag
If flag is non-zero, the coordinates of all non-solvent atoms whose coordinates do not appear to be correctly scaled will be adjusted. This option is sometimes needed to correct for improper scaling by molecule drawing packages that do not set the 3D coordinates correctly. If flag is 0, coordinates are not adjusted.

set_fc:flag
If flag is non-zero, solute atom ionizations will be set to their formal charges. If flag is non-zero, the original ionizations are preserved.

acid_base:flag
If flag is non-zero, acids will be deprotonated and bases will be protonated. This applies to solute atoms only. If flag is 0, no protonation/deprotonation occurs.

add_h:flag
If flag is non-zero, the number of hydrogens on all solute molecules will be adjusted to satisfy full valence requirements. If flag is 0, no adjustment of hydrogens is made.
