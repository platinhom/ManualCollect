Database Viewer Entry Functions

Syntax

    entry_keys	= dbv_Entries mdb

    nb_selected	= dbv_nSelectedEntries mdb
    selection	= dbv_SelectedEntries mdb
    states	= dbv_EntrySelected [ mdb, ekeys ]
    dbv_EntrySetSelected [ mdb, ekeys, states ]

    nb_visible	= dbv_nVisibleEntries mdb
    visibles	= dbv_VisibleEntries mdb
    states	= dbv_EntryVisible [ mdb, ekeys ]
    dbv_EntrySetVisible [ mdb, ekeys, states ]

    old_height	= dbv_EntryHeight [ mdb, new_height ]

Obsolete

    old_state = dbv_EntryHide [ mdb, new_state ]
    old_state = dbv_EntrySelection [ mdb, new_state ]
    old_state = dbv_EntrySelect [ mdb, entkey, new_state ]

Use dbv_EntryVisible, dbv_EntrySetVisible, dbv_EntrySelected and dbv_EntrySetSelected.

Description

These functions return and optionally set attributes which control the display of database entries in a Database Viewer. It is important to note that MOE does not save these attributes to disk. Hence, these functions do not modify the actual contents of the database file in any manner. Should you modify certain attributes (e.g. the selection of certain entries in the database) and close the Database Viewer, the database will appear unchanged when you reopen it.

mdb is a database identifier. All functions accept either a database key or the name of the database. The database must be open in a Database Viewer. 

entry_keys = dbv_Entries mdb

Returns a vector of all database entry keys. This function is identical to db_Entries except that it is faster because it does not need to go to disk. It requires that the database be open in a Database Viewer. 

nb_selected = dbv_nSelectedEntries mdb
nb_visible  = dbv_nVisibleEntries mdb

Respectively returns the number of selected or visible entries in the specified database. If the database is not currently open in a Database Viewer, both functions return 0. 

selection = dbv_SelectedEntries mdb
visibles  = dbv_VisibleEntries mdb

Respectively returns a vector of selected or visible database entry keys. It requires that the database be open in a Database Viewer. 

states = dbv_EntrySelected [ mdb, ekeys ]
states = dbv_EntryVisible [ mdb, ekeys ]

For each entry key in the argument, these functions returns a value of 1 if the entry is selected/visible in the Database Viewer, and zero otherwise. These functions can be used on arbitrarily nested vectors. 

dbv_EntrySetSelected [ mdb, ekeys, states ]
dbv_EntrySetVisible [ mdb, ekeys, states ]

Respectively sets the selection/visibility state for each given entry key. These functions can be used on arbitrarily nested vectors. 

old_height = dbv_EntryHeight [ mdb, new_height ]

Returns and optionally sets the height (in characters) of all entries in the specified Database Viewer. The old height is always returned. The height is set if new_height is specified (i.e. not []).

Obsolete

old_state = dbv_EntryHide [ mdb, new_state ]
old_state = dbv_EntrySelection [mdb, new_state]
old_state = dbv_EntrySelect [ mdb, entkey, new_state ]

dbv_EntryHide returns and optionally sets the hidden state of all entries in the database (1=hidden, 0=visible). The old state is always returned. The state is set if new_state is specified (i.e. not []). Hiding a selected entry deselects it.

dbv_EntrySelection returns and optionally sets the selection state of all entries in the database (1=selected, 0=unselected). The old selection state is always returned. The state is set if new_state is specified (i.e. not []). A hidden entry cannot be selected. The database must be open in a Database Viewer for this function to be meaningful.

dbv_EntrySelect returns and optionally sets the selection state of a single database entry (1=selected, 0=unselected). The old selection state is always returned. entkey is the key of an entry in the specified database. The state is set if new_state is specified (i.e. not []). A hidden entry cannot be selected. The database must be open in a Database Viewer for this function to be meaningful.
