MOE Database State Functions

Syntax

    stamp = db_DataState mdb
    stamp = db_EntriesState mdb
    stamp = db_FieldsState mdb
    stamp = db_State mdb

Description

Functions allowing the caller to determine whether a change has occurred in a MOE database. The stamp returned is a numeric value. If this value has changed since the previous call to the State function then a change has occurred and an update may be required. Changes which can be monitored through these functions are the selection and visibility status of the entries and fields.

mdb specifies the MOE database and may be either a key or a filename. 

stamp = db_DataState mdb

Returns a stamp used to monitor the values in the specified database. The data state of the database will change if the content of any entry is updated or if any non empty field or entry is deleted. 

stamp = db_EntriesState mdb

Returns a stamp used to monitor the entry keys of the specified database. The entries state of the database will change if any entries are created, deleted or reordered. 

stamp = db_FieldsState mdb

Returns a stamp used to monitor the fields of the specified database. The fields state of the database will change if any fields are created, deleted, renamed or reordered. 

stamp = db_State mdb

Returns a stamp used to monitor activities on the specified database. The state of the database will change if any one of the data, entries or fields states change.
