MOE 2010.10

DNA_Create

Syntax

    chain_keys = DNA_Create [ "nucleosides", 'conformation' ]

Description

Creates a double-stranded DNA molecule in the specified conformation and returns the keys of the two newly created chains. The nucleoside composition of the first chain will be as specified by DNA_Create while the second chain will be the complement chain as dictated by Watson-Crick base-pairing rules. In the Sequence Editor, both the sequence of the main chain and the sequence of the Watson-Crick base-pairing complement chain are written in the 3' to 5' direction.

nucleosides
String of one-letter codes specifying the nucleoside composition:

1-Letter Code Nucleic Acid
A Adenosine
C Cytidine
G Guanosine
T Thymidine

One-letter codes are not case sensitive.

conformation
One of the standard DNA conformations expressed as a token (case sensitive):
  • 'A'
  • 'B'
  • 'C'

The default is 'B'.

For example, the following function: 

    svl> DNA_Create [ "GTA", 'A' ]

returns the keys of both chains. The Sequence Editor will show the GTA sequence as well as the complement sequence TAC.

References

For A and B conformations: S. Arnott & D.W.L. Hukins. Biochem. Biophys. Res. Comm. 47, 1504 (1972).

For C conformations: S. Arnott & E. Selsing. Journal of Molecular Biology. 98, 265 (1975).
