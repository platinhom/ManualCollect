Database Viewer Defaults

Syntax

    dbkey = dbv_DefaultView []
    entry_key = dbv_DefaultEntry []
    'field' = dbv_DefaultField []

Description

These functions are used to determine from which Database Viewer window, entry, or field the current task was launched. For example, Database Viewer menu commands can use dbv_DefaultView to determine on which Database Viewer window to operate. Similarly, popup menu commands are able to determine in which cell/field/entry of a Database Viewer they were invoked.

dbkey = dbv_DefaultView []

Returns the key of the Database Viewer which launched the task of the calling function. If the function was not invoked from within a Database Viewer (i.e. it was not invoked from the Database Viewer command line, menu bar or popup menu, nor by double-clicking in a cell), dbv_DefaultView will raise a Select Database Viewer prompt. If no Database Viewers are open, it will raise the Open Database Viewer file selection box.

entry_key = dbv_DefaultEntry []

Returns the key of the Database Viewer entry from which the task was launched. If the task was not launched from within a Database Viewer entry (i.e. it was not launched by selecting an item from an entry index popup menu or cell popup menu, nor by double-clicking in a cell), [] is returned.

'field' = dbv_DefaultField []

Returns the name of the Database Viewer field from which the task was launched. If the task was not launched from within a Database Viewer field (i.e. it was not launched by selecting an item from a field title popup menu or cell popup menu, nor by double-clicking in a cell), [] is returned.
