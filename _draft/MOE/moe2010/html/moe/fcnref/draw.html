Object Rendering Functions

Syntax

    Drawfcn flag 
    old_flag = Drawfcn flag
    flag = Drawfcn []

Description

Each of the following functions turns on or off a display attribute of graphics in the MOE rendering window. Drawing is enabled if flag is a non-zero value, and disabled if flag == 0.

If passed a null argument, functions return the current drawing state: 1 if drawing is enabled, 0 if disabled. In the SVL Commands window, on is printed if drawing is enabled, off if disabled. If flag is not [], and the return value is used in an assignment, or forced by the presence of enclosing parentheses, the old value of flag is returned.

Function Description
DrawAxes Turns on or off the x-y-z coordinate axes. 
    DrawAxes flag 
    old_flag = DrawAxes flag
    flag = DrawAxes []
DrawConstr Turns on or off the display of created restraint meters. (See Restraints.) 
    DrawConstr flag 
    old_flag = DrawConstr flag
    flag = DrawConstr []
DrawDipole Turns on or off the dipole moment. 
    DrawDipole flag 
    old_flag = DrawDipole flag
    flag = DrawDipole []
DrawHBonds Turns on or off the display of hydrogen bonds as dashed lines. 
    DrawHBonds flag 
    old_flag = DrawHBonds flag
    flag = DrawHBonds []
DrawMeters Turns on or off the display of created measurement meters. (See Meters.) 
    DrawMeters flag 
    old_flag = DrawMeters flag
    flag = DrawMeters []
DrawVDWContacts Turns on or off the display of VDW contacts as yellow dotted lines. 
    DrawVDWContacts flag 
    old_flag = DrawVDWContacts flag
    flag = DrawVDWContacts []
