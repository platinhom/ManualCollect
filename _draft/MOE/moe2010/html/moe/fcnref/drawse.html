Sequence Editor Drawing Functions

Syntax

    DrawSEfcn flag
    old_flag = DrawSEfcn flag
    flag = DrawSEfcn []

    old_flag = DrawSEScalar [index, flag]
    old_flag = DrawSEScalar index

Description

Each of the following functions turns on or off a display attribute of the Sequence Editor. Drawing is enabled if flag is a non-zero value, and disabled if flag == 0. If passed a null argument, functions return the current drawing state.

Function Description
DrawSE2Struct Turns on or off the actual secondary structure labels drawn as color-coded bars above the residues. 
DrawSE2Struct flag
old_flag = DrawSE2Struct flag
flag = DrawSE2Struct []
DrawSECompound Turns on or off the compound name. 
DrawSECompound flag 
old_flag = DrawSECompound flag
flag = DrawSECompound []
DrawSEChainTag Turns on or off the chain tag. 
DrawSEChainTag flag 
old_flag = DrawSEChainTag flag
flag = DrawSEChainTag []
DrawSEHBonds Turns on or off hydrogen bonds drawn as arcs above residues. 
DrawSEHBonds flag 
old_flag = DrawSEHBonds flag 
flag = DrawSEHBonds []
DrawSEPred2Struct Turns on or off the predicted secondary structure labels drawn as color-coded bars below the residues. 
DrawSEPred2Struct flag
old_flag = DrawSEPred2Struct flag
flag = DrawSEPred2Struct []
DrawSEResIdx Turns on or off the residue index. This option is mutually exclusive of DrawSEResUID. 
DrawSEResIdx flag 
old_flag = DrawSEResIdx flag 
flag = DrawSEResIdx []
DrawSEResUID Turns on or off the residue UID number. This option is mutually exclusive of DrawSEResIdx. 
DrawSEResUID flag 
old_flag = DrawSEResUID flag 
flag = DrawSEResUID []
DrawSEScalar Turns on or off the residue scalar plots below residues. index indicates which of the three residue scalars to plot (see rSetScalar). 
old_flag = DrawSEScalar [index, flag]
old_flag = DrawSEScalar index

The following enables the display of the plot of residue scalar 2: 

    svl> DrawSEScalar [2,1];
DrawSESingleLetter Turns on or off the single-letter display of residue codes. When off, residue codes have three letters. 
DrawSESingleLetter flag 
old_flag = DrawSESingleLetter flag
flag = DrawSESingleLetter []
