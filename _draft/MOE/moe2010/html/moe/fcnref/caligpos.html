MOE 2010.10

Sequence Functions

Syntax

    residue_position_lists = cAlignmentPositions chain_keys

Description

For every residue in each chain specified in chain_keys, cAlignmentPositions returns its horizontal position in the residue schematic. One vector of residue positions is returned per chain. This function is typically used after an alignment operation.
