MOE 2010.10

Potential Energy Functions

Syntax

    partial_charges = PEOE [ damp, damp_scale ]
    partial_charges = PEOE []

Description

PEOE is an implementation of the Gasteiger and Marsili Partial Equalization of Orbital Electronegativities method of calculating partial charges.

damp
Damping factor (specified as a real) used to guarantee convergence.

damp_scale
Scaling factor (specified as a real) used to scale the damping factor with each successive iteration of the algorithm.

PEOE can be called with either or both arguments null, in which case default values as found in file peoe.svl are used (0.5, 0.5).

The function calculates and returns the partial charge for every atom in the system but does not however actually set the atom charges. To do so, use: 

    aSetCharge [ Atoms [], PEOE [] ]
