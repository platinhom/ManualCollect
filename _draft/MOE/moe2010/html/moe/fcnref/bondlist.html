MOE 2010.10

Atom Functions

Syntax

    [A_keys, B_keys] = BondList atom_keys
    [A_keys, B_keys] = BondListExclusive atom_keys
    nbrlist = BondGraph atom_keys
    rlist = SmallestRings atom_keys
    atom_keys = ConnectingAtoms atom_keys

Description

[A_keys, B_keys] = BondList atom_keys

Returns all bonds in which the atoms in atom_keys participate. Each bond is specified by the atoms it connects: each atom in A_keys is bonded to the corresponding atom in B_keys. A_keys contains only the keys from atom_keys that are bonded to other atoms. If an atom has two bonds emanating from it, it will appear twice in A_keys. A_keys and B_keys are both flat vectors.

[A_keys, B_keys] = BondListExclusive atom_keys

Returns all bonds between the atoms in atom_keys only. Both ends of the bond must be in atom_keys for the bond to be reported. 

nbrlist = BondGraph atom_keys

Returns the neighbor list representation of the molecular graph of given atoms. For each input atom, the function returns the indices of all input atoms that are bonded to that atom. Bonds outside the set of input atoms are ignored. 

rlist = SmallestRings atom_keys

Returns the list of all smallest rings in the molecular graph of given atoms. A smallest ring must contain a bond such that the ring is one of at most two smallest rings containing that bond. 

atom_keys = ConnectingAtoms atom_keys

Given a set of atoms, the function returns the connecting atoms, as follows:

  • A: Atoms of the input set.
  • B: Atoms on a shortest path between any two input atoms (from set A).
  • C: Atoms on a (smallest) ring that contain at least two atoms from set B.
  • D: Atoms connected to atoms of set B or C by a double bond or a coordination bond (but not a ring bond).
the function returns the union of sets A, B, C and D.
