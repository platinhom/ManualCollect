Graphics Object Manager

The Graphics Object Manager lists the graphical entities currently created in the MOE system. Typical graphical objects are surfaces, Ligand Interactions bond vectors, and user-built objects (constructed using SVL graphics functions). The Graphics Object Manager allows you to hide, show, delete and rename such objects.

Important! The Graphics Object Manager does not control the display of atoms, bonds, meters, the coordinate axis, the periodic box, or backbone ribbons.

To open the Graphics Object Manager, select Window | Graphics Objects from either the MOE Window, the Sequence Editor or a Database Viewer. The following panel appears:

The list updates dynamically and indicates whether current objects are shown or hidden.

Tip   A shortcut to opening the Graphics Object Manager is to use the keystroke combination <ctrl>-g while the focus is in one of the MOE, Sequence Editor or Database Viewer windows.

One or more graphic objects can be selected. To select multiple objects, use <ctrl>-click or <shift>-click to add to the current selection set. Select one or more objects followed by the action to perform:

Tip   Double-clicking on a selected entry in the list will show the selected object and hide all other objects, enabling you to single out an object of interest.

Hide The selected graphics objects are no longer drawn in the MOE rendering area. The word [Hidden] appears beside the entries in the panel.
Show The selected graphics objects are drawn in the MOE rendering area. The word [Shown] appears beside the entries in the panel.
Toggle Toggles the visibility of the selected graphics objects. A hidden object is shown, and a shown object is hidden.
Delete Deletes the selected graphics objects.
Name The name of the last selected graphic object is displayed in the Name text field below the list. If you wish to change the name of the selected object(s), type the new name into the text field and hit Return or press Apply.
Transparency The transparency levels of the last selected graphic object are displayed in the Transparency subpanel. When the Faces option is set to Front And Back, the transparency can be adjusted for both the front face and the back face of the surface. When the option set to Identical, the front face and the back face transparency levels are set to the same value. To set the transparency level to zero (i.e. none), press 0. Any change will be applied to the entire selection.

Material The material properties of the last selected graphic object are displayed in the Material subpanel. When the Faces option is set to Front And Back, the material properties can be adjusted for both the front face and the back face. When the option set to Identical, the front face and the back face materials are set to the same value. Values for Shading, Gloss and Brightness (S, G and B) can be entered manually (click Apply for the changes to take effect), or various preset materials can be chosen from the drop-down menu. (These controls are explained in Visualization Setup.) Any change will be applied to the entire selection.

Note: MOE treats graphic objects differently from molecular objects. For instance, when a file is closed using File | Close, MOE displays a dialog in which the option of deleting graphic objects is selected by default. If the option is turned off, graphic objects remain in the system although the molecular system is deleted.
