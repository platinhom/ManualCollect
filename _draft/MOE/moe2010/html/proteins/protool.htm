Creating Proteins

The MOE Protein Builder is used to construct polypeptides from the standard 20 amino acids, plus acetyl and amide end-caps, as well as three variants of histidine and one of proline. It is intended to be used when proteins with atomic data are desired. The Create Sequence command (see next section) is used when sequence only data is required.

To open the Protein Builder, choose SE | Edit | Protein Builder in the Sequence Editor (or MOE | Edit | Build | Protein). The following panel appears:

Press one of the buttons in the Residue bank to append the selected amino acid to the carboxyl terminal of the chain being built. The Geometry button bank can be used to select the backbone dihedral angles of any subsequent residue.

Press New Chain to create the new chain. Note that when New Chain is pressed, any previously created chain can no longer be edited using the Protein Builder.

Protein Builder Panel Reference

Residue Appends the selected amino acid to the carboxyl terminal of the chain being built.

Acetylate N-Term creates an ACE N-terminus, and Amidate C-Term, an NME C-terminus. Once an NME has been attached to a protein, no further residues can be appended: pressing buttons from the Residue bank will have no effect. The ACE end-cap can be prepended at any time; it is the only residue that can be joined into the sequence out of order.

Creating Sequences

To open the Create Sequence panel, choose SE | Edit | Create Sequence in the Sequence Editor. This panel is used to create polypeptide and DNA fragments from a string of single-letter identifiers.

The Create Sequence input line recognizes single-letter codes of residues (see below). Sequence Type options allow you to create a sequence of empty residues (i.e. without atoms) or a fully-populated peptide chain in either extended or alpha-helix conformation. The DNA options create fully-populated strands in A, B or C conformation [Arnott 1975].

The name, tag and header attributes of the chain can be set in this panel. Press OK to create the sequence. Press Apply to create the sequence without closing the panel.

Tip   To cut and paste a sequence into the text field use the keyboard cut and paste facilities. For more information on the text field see GUI Basics.

Editing Proteins

The SE | Edit | Protein submenu in the Sequence Editor contains commands to amidate or acetylate protein chains, as well as to manipulate protein secondary structure.

Amidate C Terminus

Choose SE | Edit | Protein | Amidate_C_Terminus. This command allows you to append the NME C-terminal residue to any protein chain. You will be prompted in the Sequence Editor for the chain to amidate.

The command will remain active until it is cancelled with Esc.

Acetylate N Terminus

Choose SE | Edit | Protein | Acetylate_N_Terminus. This command allows you to prepend any protein chain with the ACE N-terminal residue provided that the first residue of the chain has a backbone nitrogen and alpha-carbon. You will be prompted in the Sequence Editor for the chain to acetylate.

The command will remain active until it is cancelled with Esc.

Secondary Structure

The SE | Edit | Protein | Structure command allows you to edit the user-assigned secondary structure labels of selected residues. You can set them to one of the legal segment types (see R_SEGMENT_TYPES), or you can copy the actual secondary structure labels - as determined by the protein conformation - to the user-assigned labels.

You can also adjust the backbone dihedral angles of the protein to values which correspond to the assigned segment labels with the Edit | Protein | Structure | To_Actual command.

See pro_SegmentToDihedrals for a table of the backbone dihedral angles which will be used for each segment type.

Note: This function will have no effect where bonds are not rotatable (e.g. if there is a bond cycle due to the existence of disulfide bonds).
