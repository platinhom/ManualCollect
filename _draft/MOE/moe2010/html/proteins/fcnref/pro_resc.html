MOE 2010.10

Protein Functions

Syntax

    pro_ResCopy [ src_residue_keys, dst_residue_keys ]

Description

Destroys all atoms in the destination residues, given as keys in dst_residue_keys, then copies all backbone atoms in the source (including the beta carbons in non-glycines) to the destination. Note that the resulting destination residues will not be polymerized and will be missing all sidechain atoms except beta carbons.
