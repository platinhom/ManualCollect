MOE 2010.10

Protein Functions

Syntax

    pro_AtomCreate [ res_keys, mol_data ]

Description

Creates and/or modifies atoms in the residues specified as keys in res_keys to conform to the protein given as mol_data, a MOE molecule format data structure, as detailed in the entry for mol_Extract. The length of res_keys must equal the number of residues in the protein. The result is to make the atoms in the specified residues identical in all ways (name, element, hybridization, etc.) to those in mol_data. The residue properties remain unchanged.
