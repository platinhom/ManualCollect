API Context Functions

Syntax

    int svl_GetCvPrev(void);
    int svl_GetCvNext(void);
    int svl_GetCvPos(unsigned k);

    int svl_PutCvPrev(void);
    int svl_PutCvNext(void);
    int svl_PutCvPos(unsigned k);

Description

svl_GetCvPrev
svl_GetCvNext
svl_GetCvPos 		Advance the current Get-position to the previous, next, or given (k'th) position, respectively. They return the old current Get-position.

svl_PutCvPrev
svl_PutCvNext
svl_PutCvPos 		Advance the current Put-position to the previous, next, or given (k'th) position, respectively. They return the old current Put-position.
