MOE 2010.10

API Context Functions

Syntax

    svl_error svl_GetCvCheckEOV(void);
    int svl_GetCvIsEOV(void);

Description

Checks that the end of vector has been reached (i.e. that all elements have been read.) svl_GetCvIsEOV returns zero if the end has not been reached. svl_GetCvCheckEOV returns an error if the end has not been reached.
