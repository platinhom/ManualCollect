Edit Widget

Description

The Edit widget is a scrollable multi-line text editor.

Attributes

Edit inherits all core attributes and defines the following attributes:

Attribute Description Values Default
len Number of lines displayed 1 to 255 25
width Number of characters displayed per line 1 to 255 80

Trigger

None.

Data Value

The editor's text is returned as a token.
