FSB, FSBText Widgets

Description

FSB provides a file selection box containing two lists:

  • List of directories
  • List of files in the currently searched directory

Also provided is a Path text input field for entering a directory search path or filename or filename filter; press Tab for name completion of partially entered file or directory names.

The Path text field supports a recent directories popup menu, which can be pre-loaded using the recentDirs attribute. The recent directories are common to all file selection box widgets (anywhere within the entire application).

By default, FSB is set to trigger when a user double-clicks on a filename from the file list, or when the user activates the text field by typing a filename and pressing Return.

Successful selection of a file guarantees that the file exists; an error will be flagged if it does not. In 'saveAs' mode, if the specified file already exists, the user will be prompted for overwrite confirmation.

The 'none' mode specifies that neither file-existence validation nor write confirmation will be performed.

FSBText provides a file selection box text field. File name completion is activated with the Tab key. The directory in which the file name completion searches is specified with the directory attribute. If no directory is specified then the current directory is used. Unlike the FSB path text field, there is no recent directories popup menu, however, a shortcut pulldown list is available.

Attributes

FSB and FSBText inherit all core attributes and define the following attributes:

FSBText further defines:

Attribute Description Values Default
allowBlank If true then an empty file name will be allowed 0/1 1
directory Directory which the text field will use for file name completion token []
len Length of the text field Maximum of 255 40
mode File validation mode WINDOW_FSB_MODES 'none'
shortcut List of tokens to appear as a pulldown list located in the rightmost position of the text field vector of tokens []
shortcutTitle Title to appear on the pulldown list. Will not be displayed if shortcut is empty. token ''

fsb further defines:

Attribute Description Values Default
len Number of rows displayed in each file list 1 to 127 15
mode File validation mode WINDOW_FSB_MODES 'none'
width Number of characters displayed in each list 0 to 127 20
recentDirs List of recent directories to be loaded into the recent directories popup, provided as tokens. The new directories will be added to the current list. Duplicate entries will be ignored. 'dir1;dir2;dir3...' []

Trigger

Return pressed in the file text entry field or a file double-clicked on in the file list.

If a shortcut list is present and an item is selected from the list then the text is replaced with the selected text and the widget triggers.

Data Value

The full directory path and name of the file; '' if invalid or none.
