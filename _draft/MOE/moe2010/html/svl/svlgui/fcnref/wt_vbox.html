Manager Widgets

Description

Manager objects are containers for other widgets. The children of a manager are specified as attributes of the manager.

Object Description
Mbox Manager that arranges its children in a geometric grid.
Vbox Mbox with only 1 column.
Hbox Mbox with only 1 row.

Attributes

Vbox, Mbox, and Hbox inherit all core attributes and define the following attributes

Attribute Description Values Default
spacingH Amount of horizontal space between the widget's children WINDOW_SPACING 'none'
spacingV Amount of vertical space between the widget's children WINDOW_SPACING 'none'

Mbox further defines:

Attribute Description Values Default
columnMajor Traverse manager's children by rows or columns 0 = by rows
1 = by column		 0
columns Number of columns 1 to 127 1
uniformRows Uniform height sets all rows to have height equal to the tallest widget in Mbox. Non-uniform height sets the height of a given row to the height of the tallest widget in that row. 1 = uniform
0 = non-uniform		 0
uniformCols Uniform width sets all columns to have width equal to the widest widget in Mbox. Non-uniform width sets the width of a given column to the width of the widest widget in that column. 1 = uniform
0 = non-uniform		 0

Trigger

None.

Data Value

A vector of the values of all the manager's children that return values. The values appear in the child creation order.
