Label Widget

Description

A set of one or more non-editable labels. A label may contain single or multi-line text or a bitmap. The number of labels created is determined by the length of the text attribute vector.

Attributes

Label inherits all core attributes and defines the following attributes:

Attribute Description Values Default
text Label Vector of tokens []
minWidth Minimum label width (in characters) 0 to 127 0
foreground Text color WINDOW_COLORS 'black'
background Text background color WINDOW_COLORS 'background'
font Text font WINDOW_FONTS 'mediumFixed'
graphics A graphics object key (see gr_create) or graphics content (see gr_data) Scalar key, or vector content 0
grPosition Layout of graphic object with respect to text, when both are present 'left', 'right', 'above', 'below', 'overlay' 'above'
grScale Scaling policy for the graphic object 'none', 'anisotropic', 'isotropic', 'isotropic_shrink' 'isotropic'

Trigger

None.

Data Value

Label defines no data value. If the data value for a Label widget is requested through a WindowGetData function call, the return value will be [].
