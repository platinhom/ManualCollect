Structural Functions

Syntax

    reversed_v = reverse v 

    result = shift [ v, k ]
    shifted_left = shiftl v 
    shifted_right = shiftr v

Description

reversed_v = reverse v

Reverses the order of the top level elements of v.

    svl> reverse [1,2,3,4,5] 
    [5,4,3,2,1] 

    svl> reverse "palindrome" 
    "emordnilap" 


result = shift [ v, k ]
shifted_left = shiftl v 
shifted_right = shiftr v

If k is positive, all elements are shifted to the right by k positions. The first element is repeated k times while the last k elements are lost. If k is negative, all elements are shifted to the left by abs k positions. The last element is repeated abs k times while the first abs k elements are lost.

Note that: 

    shiftr v === shift [v, 1] 
    shiftl v === shift [v, -1]

Examples: 

    svl> shift [igen 5, 2]
    [1,1,1,2,3]

    svl> shift [igen 8, -3]
    [4,5,6,7,8,8,8,8]
    
    svl> shiftr [1,2,3,4,5]
    [1,1,2,3,4] 

    svl> shiftl [ 2, 3, [4,5] ]
    [ 3, [4,5], [4,5] ]
