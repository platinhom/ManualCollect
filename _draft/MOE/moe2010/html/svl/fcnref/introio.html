SVL I/O Functions

SVL formatted input and output (I/O) is controlled using formatting instructions written in the format language. The language is similar for both input and output. For input, however, it is used to specify how the input data are to be interpreted, i.e., it specifies the conversion to perform, whereas for output it is used to specify the manner in which the output data are to appear. For more information on SVL I/O please see the SVL manual chapter Input and Output.

Various file manipulation functions exist. Both temporary and regular files can be written, and different functions exist for different handlings of creation and errors.
