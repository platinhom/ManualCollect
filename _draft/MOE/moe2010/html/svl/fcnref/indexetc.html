MOE 2010.10

Find Positions Functions

Syntax

    indices = indexof [ v, w ]
    indices = indicesof [ v, w]

Description

indices = indexof [v, w]

Returns, for each element of v, the position of its first occurrence in w. If an element of v is not in w, the return value is 0. 

    svl> indexof ["abc", "xxcaa"]
    [4,0,3]

    svl> indexof [ [1,2,3,4,5], [2,2,3,3] ] 
    [0,1,3,0,0] 

    svl> indexof [ [2,2,3,3], [1,2,3,4,5] ] 
    [2,2,3,3]


indices = indicesof [v, w]

Returns, for each element of v, the vector of positions of all its occurrences in w. If an element of v is not in w, the return value is the null vector, i.e. []. 

    svl> indicesof ["abc", "xxcaa"]
    [[4,5], [], 3]

    svl> indicesof [ [1,2,3,4,5], [2,2,3,3] ] 
    [ [], [1,2], [3,4], [], [] ] 

    svl> indicesof [ [2,2,3,3], [1,2,3,4,5] ] 
    [2,2,3,3]
