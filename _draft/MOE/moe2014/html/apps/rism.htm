Introduction

The MOE Solvent Analysis application uses the 3D-RISM (three-dimensional reference interaction site model) method to analyze the role of solvent in macromolecular systems. The application computes a time-averaged distribution of water H and O densities, along with free-energy maps for analyzing solvent stability and solvation contributions to binding free-energy. A unique feature of 3D-RISM is its ability to include various concentrations of salt or hydrophobic molecules as part of the solvent, thereby producing density maps for ions and hydrophobic groups as well as for water. The Solvent Analysis tool can be applied to various water-related tasks such as water placement in low resolution X-ray protein structures, estimating solvent effects in protein-ligand binding, and designing ligands to exploit solvation effects such as water displacement.

Solvent Distribution Density

The structure of water or any solvent can be described by measuring or calculating the time-averaged (or ensemble averaged) spatial distributions of the solvent atoms. These distributions are often computed by processing the output of molecular dynamics (MD) or Monte-Carlo (MC) simulations to produce probability densities of finding solvent particles (such as oxygen or hydrogen atoms) at any given point in space. The particle densities reflect the spatial distribution of solvent atoms, which in turn characterizes the organization of solvent around a solute in a thermodynamic fashion. The solute can be an apo protein, a ligand, or a bound complex. Although solvent densities are typically calculated by analyzing the results of long MD or MC simulations, the 3D-RISM approach computes solvent particle density directly by using a system of self-consistent equations to iteratively converge density values. The advantage is that no simulations are required, reducing run-times dramatically and rendering simulation options (such as run-length, clustering options etc.) a non-issue. The results obtained with 3D-RISM are in qualitative and often even quantitative agreement with MD for less than 0.5 % of the computer work. More information on the origins of 3D-RISM and its computation can be found elsewhere [Beglov 1997, Kovalenko 1999, Kovalenko 2000].

The following examples of solvent densities calculated using 3D-RISM demonstrate how these densities reflect solvent organization. In Figure 1, the 3D-RISM water O and H densities are plotted around a single water molecule. The oxygen density is in solid blue and the hydrogen density is in white mesh. Solvent Stability and Free Energy

Solvation free energy is also called the vacuum-to-water transfer free energy since it corresponds to the reversible work needed to transfer 1 mol of solute into pure water at infinite dilution. Statistical mechanics gives a few rigorous ways of calculating this quantity; one of them is the thermodynamic integration technique (TI) that gives a very general formulation to compute free energies:

\begin{equation} \Delta G_{solv} = k_{B} T \int_{0}^{1} d \lambda \left< \frac{\partial U_{\lambda}}{\partial \lambda} \right>_{\lambda} \label{eqn:1} \end{equation}

where \(k_{B}\) is the Boltzmann constant, \(T\) the temperature, \(U\) the potential energy and \(\lambda\) a coupling parameter. The \(\lambda\) parameter provides a means of gradually increasing the interaction between a solute molecule and water. The angle brackets represent ensemble averages at a given value of \(\lambda\) (at constant temperature and for a given solvent bulk density).

Typically, calculating the free energy is computationally intensive. For example, with MD or MC, one needs to converge approximately 20 different extensive simulations. However, the 3D-RISM theoretical framework is able to use a much smaller computation for evaluating the solvation free energy by recasting the problem in an elegant manner. Using the single solution for the fully coupled system (\(\lambda = 1\)), Eq. \eqref{eqn:1} becomes

\begin{equation} \begin{aligned} \Delta G_{solv} & = k_{B} T \sum_{\alpha}^{H,O} \int_{0}^{1} d \lambda \int_{V} \rho_{\alpha} \left( \vec{r}; \lambda \right) \frac{\partial U_{\alpha} \left( \vec{r}; \lambda \right)}{\partial \lambda} d V \\ & = \int_{V} \Delta G_{solv} \left( \vec{r} \right) d V \end{aligned} \label{eqn:2} \end{equation}

where \(\rho_{\alpha} \left( r; \lambda \right) \) is the number of particles, and \(\alpha\), the water Oxygen or water Hydrogen per volume unit at a given point \(r\). This is the density distribution function discussed previously in units of particles per cubic angstrom. The potential energy between the solute and the solvent average particles is computed using a two-body additive forcefield such as the AMBER or the MMFF94 forcefields. The key point is that the integral of Eq. \eqref{eqn:2} submits to analytic solution when the Hypernatted Chain or the Kovalenko-Hirata closure relationships are used.

The consequence of the second equality in Eq. \eqref{eqn:2} is that contributions to the total solvation free energy can be evaluated as a local function whose value can be computed at each point on a grid. This grid is referred to as a solvent free energy map, and can be used to compute the free energy of solvation for a given solute. Following the arrows of the ligand binding thermodynamic cycle in Figure 5, one obtains

\begin{equation} \Delta G_{solv}^{bind} = \Delta G_{solv}^{complex} - \Delta G_{solv}^{receptor} - \Delta G_{solv}^{ligand} + \Delta G_{gas}^{bind} \label{eqn:3} \end{equation}

Figure 5. Binding Thermodynamic Cycle: The solvent contribution to ligand binding can be decomposed into three vacuum-to-solution transfer free energies: \(\Delta G_{complex}\), \(\Delta G_{receptor}\) (prot) and \(\Delta G_{ligand}\) (lig).

In 3D-RISM calculations, the last term (free energy of binding in the gas phase), which accounts for protein-ligand interactions, will be ignored since the focus is only on solvent effects. Substituting Eq. \eqref{eqn:2} into Eq. \eqref{eqn:3} yields:

\begin{equation} \begin{aligned} \Delta G_{solv}^{bind} & = \Delta G_{solv}^{complex} - \Delta G_{solv}^{receptor} - \Delta G_{solv}^{ligand} \\ & = \int_{V} dV \left[ \Delta G_{solv}^{complex} \left(\vec{r}\right) - \Delta G_{solv}^{receptor} \left(\vec{r}\right) - \Delta G_{solv}^{ligand} \left(\vec{r}\right) \right] \\ & = \int_{V} d V \Delta G_{solv}^{bind} \left(\vec{r}\right) \end{aligned} \label{eqn:4} \end{equation} The consequence of Eq. \eqref{eqn:4} is that solvation contributions to the binding free energy are distributed over a grid, henceforth referred to as the solvent binding free energy map. Eq. \eqref{eqn:4} also leads to interpretations for each of the individual solvation free energy maps. Explicit vs. Implicit Solvent: The 3D-RISM Perspective

3D-RISM does not use explicit solvent in its calculations. However, an examination of the potential of mean force calculation under the 3D-RISM paradigm reveals that, although solvent is treated implicitly in 3D-RISM, it is still possible to draw inferences about explicit solvent behaviour. This distinguishes 3D-RISM from other implicit solvent models, in particular Poisson-Boltzmann.

The potential of mean force (PMF) calculation computes the reversible work required to bring two molecular entities to within a certain distance of each other in solvent. When PMF is plotted against distance, explicit solvent behaviour manifests as maxima and minima at distances consistent with solvent shells. In the figure below, the potential of mean force is calculated, within the 3D-RISM framework, for a guanidinium and an acetate molecule being brought to within a certain O..H H-bond distance in pure water.

Potential of Mean Force along the Guanidinium-Acetate H-bond

When 3D-RISM results are compared to Poisson-Boltzmann and molecular dynamics [Rozanska 2000] the first thing to note is the similarity between the 3D-RISM and MD curves. Although shifted, the 3D-RISM curves exhibit the same kind of explicit solvent behavior as the MD ones. Indeed, running MD simulations at different H-bond lengths reveals that the first maximum is due to the cost of forming a cavity in water as the volume of the complex increases. Eventually, a single water layer comes between the two moieties to bridge optimally at the second minima. Two very stable water arrangements observed in an MD run are shown below at the second MD H-bond minima. This trend is repeated until the second layer of bridging waters appears. Poisson-Boltzmann completely misses these explicit solvent effects.

Two Stable Bridging Water Configurations at the MD Second PMF Minima

The following images show how 3D-RISM is able to express this explicit behavior of water while not performing a simulation. Below, the water oxygen density is rendered at positions indicated by circles on the potential of mean force plot. The first maximum corresponds to a large cavitation penalty. At the second minimum, both the middle bridging water and the two bridging water arrangements as seen in MD simulations are reproduced. As the two ions separate further, a second minimum forms, indicating that a second water molecule layer can fit.

Examples of 3D-RISM Applications

The following examples demonstrate how 3D-RISM density and energy maps can be used to analyze solvent effects in protein binding pockets.

Application: Water Placement

3D-RISM can be used to analyze water placement in a binding pocket. Figure 6 compares the positions of crystallographic waters (yellow) with the 3D-RISM water oxygen density plotted at two iso-levels (solid = 3, mesh = 6). The high density regions correspond to regions of localized water. The higher density mesh contours are in close agreement with the X-ray water positions. 3D-RISM does not always give a full water position, which in a way may better reflect room temperature and non-crystalline water arrangements. The density peaks can be used to infer water to ligand-receptor bridging.

Figure 6. 3D-RISM and Water Placement: Crystallographic waters (yellow) with the 3D-RISM water oxygen density plotted at two iso-levels (solid = 3, mesh = 6).

Application: Differentiate H-Bond Desolvation Penalty

In Figure 7, the 3D-RISM computed \(\Delta G_{binding}\) is plotted at two iso-levels (PDB code 3FKT). These are positive \(\Delta G_{binding}\) contributions which indicate regions where desolvation is detrimental to binding. The black mesh iso-level is 8 times larger (more positive) than the red iso-level. The iso-levels indicate that there is a much larger penalty for desolvating H-bond partners in the Phe128 region than in the Gln93 region. The lower desolvation penalty in the Gln93 region suggests that forming an H-bond here may have a higher impact on affinity because less of the binding energy goes to compensate for the desolvation penalty.

Figure 7. H-Bond Desolvation Penalties: The 3D-RISM computed \(\Delta G_{binding}\) plotted at two iso-levels. The black mesh iso-level is 8 times larger (more positive) than the red iso-level.

Application: Characterizing Binding Sites

In Figure 8(A), the 3D-RISM water O density (blue-mesh) and the positive (red) and negative (green) contributions to \(\Delta G_{complex}\) are plotted in the S3 subpocket of renin (PDB code 2IKO). This subpocket is left empty upon ligand binding, leaving room for water molecules. The water O density shows some water localization in this subpocket, but the overall positive value of \(\Delta G_{complex}\) indicates that the water localization is not particularly stable. In Figure 8(B), the 3D-RISM water O density (solid-blue) and hydrophobe (mesh-orange) densities are compared in the same renin S3 subpocket. The fact that the hydrophobe distribution (orange mesh) fills the S3 subpocket more completely than the water O density, and that the positive \(\Delta G_{complex}\) indicates unstable ordered waters in this pocket suggests that filling the pocket with a hydrophobic group would be preferred. Furthermore, such a group could be decorated with H-bonding groups to take advantage of backbone donors responsible for the two small negative \(\Delta G_{complex}\) regions (shown in green in Figure 8(A)).

Figure 8. Subpocket Characterization: S3 subpocket of renin (PDB code 2IKO): (A) The 3D-RISM water O density (blue-mesh) and the positive (red) and negative (green) contributions to the \(\Delta G_{complex}\) are shown; (B) The 3D-RISM water O (solid-blue) and hydrophobe (mesh-orange) densities are shown in the same renin S3 subpocket.

Application: Highlighting Hydrophobic Sites

In Figure 9, the receptor water O density (blue solid surface), the positive \(\Delta G_{receptor}\) (solid red), and the negative \(\Delta G_{binding}\) (green mesh) are shown in a hydrophobic pocket of BACE (PDB code 3FKT). The positive \(\Delta G_{receptor}\) around the localized water density indicates that this water can be favorably displaced by a ligand. The negative \(\Delta G_{binding}\) further confirms this by indicating that desolvation will result in free energy gains.

Figure 9. Highlighting Hydrophobic Sites: The receptor water O density (blue solid surface), the positive \(\Delta G_{receptor}\) (solid red), and the negative \(\Delta G_{binding}\) (green mesh) are shown in a hydrophobic pocket of BACE

Application: Exploring Displaceable Waters

In Figure 10, the 3D-RISM water O density (solid blue) has a peak in a non-obvious region. However, the \(\Delta G_{complex}\) solvation free energy is close to zero at the location of this peak, suggesting it might be interesting to try to displace a water in this region using a small greasy substituent. The green R-vector highlighted in this picture shows a potential point of substitution on the ligand. This specific strategy has been shown to increase significantly the activity of the parent structure (shown) [McGhaughey 2011].

Advanced Considerations when Using 3D-RISM

There are several considerations that should be taken into account when interpreting 3D-RISM results. Firstly, the equations are solved with respect to a fixed geometry of the solute. It is possible, for example, for a ligand to suffer from an intra-molecular desolvation penalty if it is in a ligand bioactive conformation. This situation cannot be detected by 3D-RISM. The same is true for the receptor that can often adopt a different arrangement in a bound state.

The orientation of the polar hydrogen atoms of both the ligand and the receptor may have an impact on the calculated solvent distributions. When in doubt, one may want to repeat the 3D-RISM calculation with different geometries.

In analyzing the results, it is important to always account for the receptor-ligand interactions that are in direct competition with the desolvation effects. 3D-RISM does not account for any direct receptor-ligand interactions. H-bonds are obvious cases where the desolvation penalty is large and compensated for by a strong polar interaction energy. One can also find: water proton-\(\pi\), cation-\(\pi\), attractive non-polar vdW interactions, salt bridges, etc.

Sometimes one may find strange water hydrogen density in places it should, in principle, not be. This is due to actual limitations of the method. The effect can be exacerbated close to negatively charged groups found in aspartate or glutamate residues for instance.

The quantitative aspect was deliberately left out of the current 3D-RISM implementation. Although we have found ways to improve 3D-RISM in this regard, more research is needed.

Running Solvent Analysis

To use the Solvent Analysis application, load an all-atom macromolecular structure and select MOE | Compute | Solvent Analysis. The following panel will appear:

Binding
Solute		 Specify the type of calculation to be performed. Use Binding to calculate 3D solvent occupancies and solvation free energy maps in the context of binding two molecules (the Receptor and the Ligand). Use Solute to calculate 3D solvent occupancies and free energy of solvation maps of a single collection of molecules (the Solute) with no distinction made between ligands and receptors.
Receptor In the Binding calculation, Receptor specifies the set of atoms to use as the receptor. Press ? to briefly select the atoms specified.
Ligand In the Binding calculation, Ligand specifies the set of atoms to use as the ligand. Press ? to briefly select the atoms specified.
Atoms In the Solute calculation, Atoms specifies the set of atoms to use in the calculation. Press ? to briefly select the atoms specified.
Near
Within		 In the Solute calculation, Near specifies the set of atoms to use to localize the calculation. Those residues with atoms within the specified distance of the Near set will be used in the calculation. This narrowing of the calculation to a subset of the system not only can lead to more precise results but can also significantly reduce the computational cost in both time and memory.
Contour in
MOE Window		 Highlight the atoms specified by Receptor/Ligand/Atoms in the MOE Window with a contour graphic. In the Binding calculation, the Ligand and Receptor will be colored differently.
Filename The base filename used to name the various output files generated by the calculation. By default, rism3d is specified. In this case, for a Binding calculation, the files rism3d_input.rism, rism3d_complex.rism, rism3d_receptor.rism, rism3d_ligand.rism or rism3d_rism.svl would be created. In the case of a Solute calculation, the complex, receptor and ligand output files are not created, and the file rism3d_solute is created instead. Press the Browse button to select an output file using a file browser. This is mostly used to change the output path of the file. If the selected file is a *.rism file, the rism specific postfix is trimmed.
Salt The concentration of salt (NaCl ions) in the solvent. The presence of salt reduces the effects of formally charged systems and adds robustness to the results. The calculation time and memory requirements will increase with the use of salt. Separate anion and cation 3D density maps will be generated.
Hydrophobe The concentration of a given hydrophobe in the solvent (a Lennard-Jones particle). The hydrophobe will be correlated with solvent and salt atoms. The calculation time and memory requirements will increase with the use of a hydrophobe. A separate hydrophobic 3D density map will be generated. Note that in this map the hydrophobe distribution does not represent hydrophobic regions, but rather where a probe of the size of a neutral Cl atom could better interact with the solute.
3D-RISM Press 3D-RISM to change various options that affect the underlying 3D-RISM calculation. Normally, the defaults should be used; however, changes are reasonable if memory requirements are prohibitively large or if very approximate (fast) calculations are desired.

  • Grid Spacing. The dimensions of each grid cell (\(\mathring A\)). Smaller values provide better results but use more memory.

  • Buffer. The distance (\(\mathring A\)) by which the bounding box of atoms is extended. Two solvation shells is sufficient.

  • Convergence. The precision at which the 3D-RISM equations are solved: Loose (approximate), Medium or Tight (very accurate).

  • NDIIS. In the DIIS (Direct Inversion in the Iterative Subspace) method, the number of copies (N) of the various 3D grids retained in memory for accelerating convergence by extrapolation. Smaller values use less memory but slow down the convergence rate.

Options Options to control various aspects of the Solvent Analysis calculation:

  • Calculate Partial Charges. If on, then the AM1-BCC partial charges will be used on the small molecule ligand and AMBER10 on the rest of the system. This does not change the current partial charges of the system. Otherwise, the existing partial charges will be used as-is.

  • Simultaneous Run. If on, then the various grids for the complex, receptor and ligand will be calculated in parallel; however, more computer resources will be required. This option is only applicable to a Binding calculation.

  • Launch Analysis. If on, then upon the successful completion of the calculation an analyzer panel will be raised to visualize the results.

Message The text line at the bottom gives an estimate of the memory that will be used at the peak of the calculation. When performing a simultaneous run on a binding calculation, the estimated memory sums the requirements of the three calculations. Many fields affect the memory: the system size Within, the presence or not of Salt and Hydrophobe and all the 3D-RISM options except the Convergence criteria. The text also reports the stage of the calculation when creating the input file; the input file is created when Run or Batch is pressed.

Press Run to begin the calculation. Progress will be displayed in the MOE Window. The panel will close. The calculation time depends on the size and complexity of the system as well as whether Binding or Solute is selected. Typically, several minutes are required for a protein:ligand complex. Once the calculations are ongoing, they can be terminated using MOE | Cancel | rism3d. To defer the run, press Batch. A file named using the file prefix specified in the Filename textfield, e.g. rism3d_rism.svl if rism3d is specified, is written containing all the instructions and data, including the input file, needed for a solvent analysis calculation. This file is standalone and can be run using moebatch. Running instructions are printed on the CLI. Press Close to close the panel.

The batch file can be run either inside MOE or independently in a shell. In MOE, the file is run from a MOE CLI by typing 

    run 'rism3d_rism.svl'

In a shell, the file is run using: 

    % moebatch -run rism3d_rism.svl -p outputpath -m [0|1]
where the -p option specifies the directory (path) in which to write the output file. This is useful if the the output file is to be saved to a directory different than the current working directory or "working folder" (default). The -m option specifies if the calculations are to be simultaneous (-m 0) or not (-m 1). When not specified, the default running mode is determined by the state of the Simultaneous Run checkbox when the batch file as created.

Analyzing the Results

To analyze the Solvent Analysis results, Use MOE | File | Open and select one of the output *_solute.rism, *_ligand.rism, *_receptor.rism or *_complex.rism files. The appropriate multiple files will be automatically loaded. The *_input.rism file can be selected as well. Alternatively, turn Launch Analysis on before pressing Run. The analysis panel will open automatically. The following panel will appear:

Binding		 The 3D grid data contours are controlled by a matrix of buttons. Each row of the matrix corresponds to Complex, Receptor, Ligand or Binding data. Each column of the matrix contains a single type of grid: \(\Delta G\), Water Occupancy, Salt Occupancy or Hydrophobe Occupancy. If a matrix element is not associated with data from the .rism files then it is disabled; otherwise it appears as dark gray (for hidden) or colored (for shown).
