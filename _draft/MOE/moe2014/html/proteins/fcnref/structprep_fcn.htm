Syntax

    issues = StructurePreparation options

    options = [
	cmd		  : 'command',
	reportName	  : 'filename',
	batch_protonate3D : flag,
	disableTypes	  : types
    ];

Description

    issues = StructurePreparation options

If run with default values will automatically correct all structural items present.

issues contents depends on the option for cmd passed to the function.

options is an optional list of parameters, supplied as a tagged vector and described as follows:

cmd:'command name'
The cmd specifies the action of the application. The legal values are:

cli Automatically correct all diagnosed structural items found in the current MOE system. When finished returns the summary of all remain structural items present. This is used by default.
summary Return only the summary of structural items.
items Return a laminated vector of diagnosed structural items. Where item(1) is the residue key, if 0 it applied to all residues, item(2) is the item type and item(3) is the text found in the details section.

reportName:'filename'
If a reportName is specified then a report "filename.log" will be written after correcting all diagnosed structural items. By default no report is written. This option is only available when cmd:'cli' is used.

batch_protonate3d:flag
If batch_protonate3d is non-zero then Protonate3D will be run after correcting all diagnosed structural items. By default this flag is set to 1. This option is only available when cmd:'cli' is used.

disableTypes:
disableTypes is a vector of token representing the types to be disabled when diagnosing or correcting structural items. disableTypes:'HName' is used by default.
