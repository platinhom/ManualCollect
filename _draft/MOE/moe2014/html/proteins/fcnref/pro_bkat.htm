Syntax

    [N,CA,C,O] = pro_BackboneAtoms residue_keys

Description

For each residue specified in residue_keys, returns the keys of the main chain nitrogen, alpha-carbon, carbonyl carbon, and carbonyl oxygen atoms. Each of the vectors N, CA, C and O are of the same length as residue_keys. Zeros are returned where backbone atoms are missing. 

    svl> chain = pro_ResAppend [0, ['ACE','ALA','NME']]
    svl> [n,ca,c,o] = pro_BackboneAtoms cat oChildren chain
    svl> n   // the n-terminal ACE residue has no mainchain nitrogen atom
    [0,272683152,272643648]
