Syntax

    [name, hyd_val] = pro_ResHydrophobicity res_names
    pro_ResHydrophobicity []

Description

For each unique amino acid residue in res_names, return the residue name and the hydrophobicity value as stored in the residue database. The names and their corresponding hydrophobicity values are returned sorted by residue name. The residue names in res_names must be specified using the standard 3-letter amino acid codes (tokens), in uppercase. No values are returned for unrecognized names.

If called with no arguments (i.e., []), the names and hydrophobicity values of all residues in the residue database are returned.

Note: The same hydrophobicity value is returned for all protonation states of histidine. NME and ACE are returned with hydrophobicity values of 0.

The hydrophobicity scale is that developed by Eisenberg et al. (Journal of Molecular Biology, 179, No. 1, 125-142 (1984)).

Example:

    svl> pro_ResHydrophobicity ['X','ACE','GLY','ASP','ASP']
    [ ['ACE','ASP','GLY'], [0,-0.9,0.48] ]
