Syntax

    flag = pro_Acetylate residue_keys
    flag = pro_Amidate residue_keys

Description

key = pro_Acetylate residue_keys

This function prepends an ACE residue to the residues defined by residue_keys. The function will only acetylate a residue if it is capable of creating a new amide bond.

A value of 1 is returned if the function was able to acetylate the residue, otherwise 0. 

    svl> c = pro_ResAppend [0, ['GLY', 'TRP', 'GLY']];
    svl> pro_Acetylate first Residues []
    1
    svl> rName cat oChildren c
    ['ACE','GLY','TRP','GLY']
    svl> pro_Acetylate first Residues []
    0


key = pro_Amidate residue_keys

This function appends an NME residue to the residues defined by residue_keys. The function will only amidate a residue if it is capable of creating a new amide bond.

The flag of 1 is returned if the function was able to amidate the residue, otherwise 0. 

    svl> c = pro_ResAppend [0, ['GLY', 'TRP', 'GLY']];
    svl> pro_Acetylate first Residues []
    1
    svl> rName cat oChildren c
    ['ACE','GLY','TRP','GLY']
    svl> pro_Acetylate first Residues []
    0
