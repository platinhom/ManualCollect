Syntax

    res_names = Seq_LetterToName letter_codes
    letter_codes = Seq_NameToLetter res_names

Description

These functions convert between the three-letter names and the single-letter abbreviations for the standard amino acids.

Seq_LetterToName Takes a single string or a vector of single-letter tokens and returns a vector of the corresponding 3-letter residue tokens.
Seq_NameToLetter Takes a vector of 3-letter residue tokens and returns the single letter equivalents in a vector of tokens.

Example: 

    svl> c = pro_ResAppend [0, ['GLY', 'TRP', 'GLY']]
    svl> rn = rName cat cResidues c
    svl> rn
    ['GLY','TRP','GLY']
    svl> rl = Seq_NameToLetter rn
    svl> rl
    ['G','W','G']
    svl> Seq_LetterToName rl
    ['GLY','TRP','GLY']
    svl> rl = app string rl
    "GWG"
    svl> Seq_LetterToName rl
    ['GLY','TRP','GLY']
