Syntax

    data = pro_Predict2Struct chain_keys
    data = pro_PredictAccessibility chain_keys

Description

Each of these functions uses a Bayesian statistical approach to calculate the probabilities of residues within a sequence taking on a particular secondary structure, or belonging to a particular solvent accessibility class.

Each function returns a vector of tagged vectors, equal in length to chain_keys. Each tag points to a flat vector of probabilities, equal in length to the number of residues constituting the chain for which the predictions were calculated. 

data = pro_Predict2Struct chain_keys

Calculates the probabilities of residues assuming helical conformation, strand conformation or neither. The corresponding tags in the return vector are 'helix', 'strand', and 'none'.

In the training set, residues were considered helical if their DSSP assignment was 'helix' or 'pi-helix'. They were classified as strand if their DSSP assignment was 'strand' or 'anti-strand'.

Example: 

    svl> Close []
    svl> pdb_Open '1A1S'
    svl> data = pro_Predict2Struct Chains []


data = pro_PredictAccessibility chain_keys

Calculates the probabilities of residues being buried or exposed. The corresponding tags in the return vector are 'buried' and 'exposed'.

In the training set, residues were defined as buried if their solvent accessible surface area was less than 9 percent of their exposed area in a GLY-X-GLY tripeptide in extended conformation, where X is a amino acid of the same type.

Example: 

    svl> Close []
    svl> pdb_Open '1A1S'
    svl> data = pro_PredictAccessibility Chains []
