Syntax

    pro_Mutate [ residue_keys, residue_names ]

Description

The residues specified in residue_keys are assigned the corresponding names in residue_names and then standardized (see pro_Standardize). Chi angles are preserved, where appropriate. The residue names are supplied as a vector of 3-letter tokens.

The residues to be mutated must contain a minimum of four atoms whose names are 'N', 'CA', 'C' and 'O'; i.e. the four backbone atoms. Otherwise, the call has no effect. For instance, ACE and NME residues cannot be mutated using pro_Mutate. The call also has no effect in the case where the original name of the residue and the name of the residue into which to mutate are the same; in such a case, use pro_Standardize instead. 

    svl> pro_Mutate [first Residues[], 'ALA']
