Syntax

    [codes, chain_ids, headers] = pdb_List []
    [codes, chain_ids, headers] = pdb_List codes
    chain_keys                  = pdb_Open codes
    mol_data_list               = pdb_Read [ codes, res_indices ]

Description

The MOE Protein Database is a molecular database file containing a non-redundant selection of high-resolution protein chain models from the Protein Data Bank. MOE functions which access this database accept PDB accession codes or chain identifiers as arguments. Codes must be tokens. 

[codes, chain_ids, headers] = pdb_List []
[codes, chain_ids, headers] = pdb_List codes

Returns the codes and annotations of the corresponding entries in the MOE protein database.

If given an empty argument, pdb_List will return the codes and annotations for all proteins stored in the database. Otherwise, if passed specific codes, it will return a vector containing the query codes from the argument list which match entries in the MOE database, along with the chain identifiers and annotations.

Note that the part of codes before the dot ('.') is case-insensitive, while the chain letter after the dot is case-sensitive. The case-insensitive portion of codes is converted to upper-case in the returned vector.

Note also that if a given code is not found then it will be ignored. Thus, the length of the input vector may not match the length of the returned vector. 

chain_keys = pdb_Open codes

Reads one or more protein chains specified by codes tokens into MOE. Passing an invalid code to pdb_Open is an error. 

mol_data_list = pdb_Read [ codes, res_indices ]

Returns a vector containing the molecular data associated with a particular code or codes. Optional residue indices allow you to extract information on specific residues in a given protein. For the format of the return data, see mol_Extract. 

    data = first pdb_Read '1A0F.A'

	// get data for just residues 10, 11, and 12

    data = first pdb_Read [ '1A0F.A', [[10,11,12]] ]
