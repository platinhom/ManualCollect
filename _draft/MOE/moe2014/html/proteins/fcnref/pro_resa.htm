Syntax

    pro_Builder []

    key = pro_ResAppend [ key, res_name_tokens, dihedral_angles ]
    key = pro_ResPrepend [ key, res_name_tokens, dihedral_angles ]

Description

pro_Builder []

Opens the Protein Builder panel with integrated Rotamer Explorer for interactive building and manipulation of polypeptides. 

key = pro_ResAppend [ key, res_name_tokens, dihedral_angles ]
key = pro_ResPrepend [ key, res_name_tokens, dihedral_angles ]

Both functions either extend an existing chain, specified by key, or build a new chain, if key is 0. pro_ResAppend builds new residues onto the C-terminal of the chain whereas pro_ResPrepend builds new residues onto the N-terminal. The residues are supplied as a vector of 3-letter name tokens. Optional dihedral angles are supplied as a 3-vector [phi,psi,omega], where each of phi, psi, and omega are vectors of angles given in radians. The length of each of these angle vectors must be equal to the length of the residue sequence.

The key of the chain is always returned. 

    svl> c = pro_ResAppend [0, ['GLY', 'TRP', 'GLY']];
    svl> pro_ResPrepend [c, 'ACE'];
    svl> rName cat oChildren c
    ['ACE','GLY','TRP','GLY']
