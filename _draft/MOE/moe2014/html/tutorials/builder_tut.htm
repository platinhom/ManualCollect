This tutorial introduces fundamental concepts for building molecules using the Molecule Builder in MOE. A variety of editing, selection and rendering commands will also be introduced.

This tutorial includes the following exercises:

  • Exercise 1: Molecule Builder
  • Exercise 2: Molecule Builder E/Z

Conventions Used in this Tutorial

MOE MOE Window
MOE | RHS Right hand side button bar
MOE | Footer Footer Area (MOE Window)

This tutorial assumes that you are using a three-button mouse. For more information on using the mouse in MOE, click MOE | Help | Mouse or see Using the Mouse.

Exercise 1: Molecule Builder

In this exercise, you will use the Molecule Builder to construct a 3D model of the following sulfonamide carbonic anhydrase inhibitor:

dorzolamide

The diagram above shows an arbitrary numbering scheme for the ring atoms, which will be used as necessary throughout this section of the tutorial.

The Molecule Builder allows construction of molecules by either adding new molecular fragments to the system (if no atoms are selected) or by substituting fragments onto selected atoms. The Molecule Builder can also be used to change properties of selected atoms (element, bond type, ionization, etc.) as well as to edit stereochemistry or to choose alternative tautomers. You can use the MOE Footer editing area to edit bond lengths, angles and torsions.

  1. Begin the tutorial with a blank MOE session, by using MOE | RHS | Close if necessary.

  2. Open the Molecule Builder panel using:

    MOE | RHS | Builder

    3. Tip!   Press the Attach button to attach the Builder to the left side of the MOE window.

  3. Create the first fragment by clicking the saturated 6-membered ring icon button:

    Tip!   You can use: MOE | Edit | Undo (or Ctrl+Z) and MOE | Edit | Redo (or Ctrl+Y) to undo or redo your changes if you make any mistakes while building the molecule.

    The MOE main window will now show cyclohexane. Click on the MOE window background to deselect the atom and render the molecule in stick mode using:

    MOE | Popup | Atoms.

  4. In the main MOE window, use the left mouse button and click on the bond between atoms 5 and 9, according to the diagram of dorzolamide shown at the beginning of this section, to select the bond. Click the double-bond icon button:

    The molecule will now reflect the changed bond order. Notice that there are fewer hydrogen atoms, and the alkene hydrogens have been repositioned.

  5. With atoms 5 and 9 still selected, click on the unsaturated 5-membered ring icon button:

    An adjacent fragment of a cyclopentadiene ring is created.

    Note that odd-membered unsaturated cycles must assign a methylene carbon at one position. To the extent possible, the newly alternating bond orders are most compatible with existing bonding patterns.

  6. Two of the carbon atoms in the fused ring system must be converted to sulfur. Select atoms 1 and 8, then click the element S button:

    The two selected carbon atoms are changed to sulfur. Since they both have two carbon neighbors and have no formal charge, the hydrogens are removed.

  7. Select only the sulfur atom at position 1. Oxidize it fully by clicking on the oxygen ylide button (=O) twice:

    Two oxygen atoms are added.

    Note that the oxidation state of both atoms is adjusted each time. After the first click, the sulfur is assigned a charge of +1 and the new oxygen atom -1. After the second click, the sulfur is further ionized to +2 and both the oxygen atoms are -1. Internally, the geometry of the sulfur and oxygen atoms is considered to be sp3.

  8. Select one of the hydrogen atoms attached to the methylene carbon at atom 2, then click the element C button:

    The hydrogen substituent is changed to carbon. Now atom 2 in the ring system has become a chiral center. Select atom 2, and notice the stereochemistry buttons have become active:

    The correct stereochemistry for this molecule is S. Depending on which of the methylene hydrogens you selected, the chiral center will be either R or S, and one of the two will be highlighted in green. The green coloration implies that the selection includes a stereocenter which is not constrained. To properly constrain the chiral center, click the S button in the stereochemistry button group.

    If the enantiomer was already S according to geometry, the structure will not change, but the button will turn blue which indicates that the constraint has been applied, and that the geometry + constraint are in accordance. If the enantiomer was R according to geometry, the chiral center will be inverted immediately prior to applying the constraint.

  9. Select one of the methylene hydrogens attached to atom 4. Click on the element N button, which will convert the hydrogen into an -NH2 functional group. Note that one of the hydrogens attached to the nitrogen is automatically selected, which makes the building of short chains quite convenient. Click the element C button twice to convert two more hydrogens into carbon atoms, which has the effect of creating a chain of NH-CH2-CH3:

    Once the chain is created, select atom 4 and click on the S button in the stereochemistry button group, to ensure that the chiral center is correctly constrained. The substituent will be reoriented if necessary.

  10. The last step is to create the sulfonamide functional group off of the thiophene ring. Select the hydrogen atom attached to atom 7. Click on the common functional group button labeled as -SO2:

    A sulfonyl functional group will be created, -SO2H. For convenience, the hydrogen atom is selected automatically. Click on the element N button, which will complete the functional group.

  11. Press the Minimize button in the Builder to refine the coordinates.

  12. The tutorial example is now complete. Optionally save the MOE molecule using MOE | File | Save | dorzolamide.moe, then clear the system with MOE | File | Close.

Exercise 2: Molecule Builder E/Z

In this exercise, you will use the Molecule Builder to construct a 3D model of the following benzylidene quinuclidine nicotinic agonists:

(3E)-3-phenethylidenequinuclidine

(3Z)-3-phenethylidenequinuclidine

  1. Begin the tutorial with a blank MOE session, by using MOE | RHS | Close if necessary.

  2. Open the Molecule Builder using MOE | RHS | Builder

  3. Create the first fragment by clicking the saturated 6-membered ring icon button:

    The MOE main window will now show cyclohexane. The molecule has been rendered in stick mode using MOE | Popup | Atoms.

  4. In the main MOE window, use the left mouse button to select a carbon atom, then click the element N button in the Molecule Builder. Select the hydrogen atom attached to the nitrogen atom and click the element C button.

  5. Select the hydrogen atom attached to the carbon atom on the opposite side of the nitrogen atom (the hydrogen atom which is on the same side of the ring as the newly added carbon atom, see picture below). Then click on the element C button.

  6. Now select the two newly-added carbon atoms. Use the left mouse button to select the first atom, do likewise for the second whilst holding down the Shift key.

    Click the single bond icon button:

    A branching single bond will be added between the two carbon atoms.

  7. Press the Minimize button in the Builder to relax the structure.

    Select the hydrogen atom as shown in the picture below.

  8. Click the element C button. Then click on the newly created bond to select the two carbon atoms of the bond and click on the double bond icon button.

    The molecule will now reflect the adjusted bond order.

  9. Select one of the hydrogen atoms attached to the double bonded carbon, then click the unsaturated 6-membered ring icon button:

    A phenyl group is attached to the double bond. Note that the relative orientation of the phenyl and quinuclidine groups around the double bond can lead to alternative E and Z structures. Select the stereo double bond, and notice the E/Z stereochemistry buttons have become active:

    The more potent form of this molecule is E. Depending on which of the hydrogens you selected, the stereo bond configuration will be either E or Z, and one of the two will be highlighted in green. The green coloration implies that the stereo bond is not constrained. To properly constrain the configuration, click the E button in the stereochemistry button group.

    If the stereo bond configuration was already E according to geometry, the structure will not change, but the button will turn blue which indicates that the constraint has been applied. If the configuration was Z according to geometry, the configuration of the bond will be inverted prior to applying the constraint.

  10. Press the Minimize button in the Builder to refine the coordinates.

  11. Save the E configuration of the molecule as a MOE file using MOE | File | Save | BQNE.moe.

  12. Now we change the configuration of the bond to Z. Select the double bond if it is no longer selected and click the Z button in the stereochemistry button group. Press the Minimize button in the Builder to refine the coordinates. Save the Z configuration of the molecule as MOE file using MOE | File | Save | BQNZ.moe.

  13. The example is now complete. Clear the system with MOE | File | Close.
