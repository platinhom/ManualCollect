Syntax

    stamp = dbv_SelectedEntriesState dbv 
    stamp = dbv_VisibleEntriesState dbv
    stamp = dbv_SelectedFieldsState dbv
    stamp = dbv_VisibleFieldsState dbv
    stamp = dbv_State dbv

Description

Functions allowing the caller to determine whether a change has occurred in the Database Viewer. The stamp returned is a numeric value. If this value has changed since the previous call to the State function then a change has occurred and an update may be required. Changes which can be monitored through these functions are the selection and visibility status of the entries and fields.

dbv specifies the Database Viewer and may be either a key or a filename. 

stamp = dbv_SelectedEntriesState dbv

Returns a stamp used to monitor the selection state of the specified Database Viewer entries. Any change in the selection of entries in the Database Viewer will change the value of the stamp.

Note: hiding a selected entry deselects it and therefore causes the selected entries state to change. 

stamp = dbv_VisibleEntriesState dbv

Returns a stamp used to monitor the visibility state of the specified Database Viewer entries. Any change in the visibility of entries in the Database Viewer will change the value of the stamp. 

stamp = dbv_SelectedFieldsState dbv

Returns a stamp used to monitor the selection state of the specified Database Viewer fields. Any change in the selection of fields in the Database Viewer will change the value of the stamp.

Note: hiding a selected field deselects it and therefore causes the selected fields state to change. 

stamp = dbv_VisibleFieldsState dbv

Returns a stamp used to monitor the visibility state of the specified Database Viewer fields. Any change in the visibility of fields in the Database Viewer will change the value of the stamp. 

stamp = dbv_State dbv

Returns a stamp used to monitor the visibility and selection state of both the entries and fields of the specified Database Viewer. The stamp will change for any visibility or selection changes made in the Database Viewer.
