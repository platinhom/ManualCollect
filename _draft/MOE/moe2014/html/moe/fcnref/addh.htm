Syntax

    Add_H atoms_keys
    Add_HLP atom_keys

Description

Add_H atom_keys

Adds or deletes valence hydrogens to give each atom specified in atom_keys the appropriate number of valence hydrogens for its hybridization state. 

Add_HLP atom_keys

Adds or deletes valence hydrogen and lone pair pseudo-atoms for each atom specified in atom_keys.
