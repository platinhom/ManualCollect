Syntax

    Undo []
    Redo []
    UndoSnapshot []
    UndoReset []

Description

The global undo system in MOE applies to the MOE molecular system and to graphic objects. Note that changes to parameters in panels are not undoable.

The undo system is based on a stack of snapshots of the molecular system in MOE. Successive undo operations traverse the stack backwards in time, and successive redo operations traverse the stack forwards in time. If a modification to the system takes place when the undo stack is partially traversed, the more recent elements of the stack are cleared, and are superseded by a snapshot corresponding to the most recent change. 

Undo []

Undoes the last change to the molecular system in MOE up to the nearest call to UndoSnapshot. 

Redo []

Reverts the state of the MOE molecular system to its condition prior to the most recent undo action. 

UndoSnapshot []

Adds a frame to the undo stack which corresponds to the most recent changes to the molecular system in MOE. 

UndoReset []

Clears the entire undo stack; subsequently, there will be no possible undo and redo actions until UndoSnapshot is called.
