Syntax

    old_state = dbv_ShowPlot [ mdb, new_state ]
    old_state = dbv_PlotLine [ mdb, new_state ]
    old_state = dbv_PlotMarker [ mdb, new_state ]
    old_range = dbv_PlotRange [ mdb, new_range ]

Description

These functions return and optionally set attributes which control the plot display in a Database Viewer. mdb is a database identifier. All functions accept either a database key or the name of the database. The database must be open in a Database Viewer. 

old_state = dbv_ShowPlot [ mdb, new_state ]

Returns and optionally sets the state of the plot display in the specified Database Viewer (1=visible, 0=hidden). The old state is always returned. The state is set to a new value if new_state is specified (i.e. not []). 

old_state = dbv_PlotLine [ mdb, new_state ]

Returns and optionally sets the state of the lines connecting data points in the plot display of the specified Database Viewer (1=draw lines, 0=no lines). The old state is always returned. The state is set to a new value if new_state is specified (i. e., not []). 

old_state = dbv_PlotMarker [ mdb, new_state ]

Returns and optionally sets the drawing state of the data point markers in the plot display of the specified Database Viewer (1=draw markers, 0=no markers). The old state is always returned. The state is set to a new value if new_state is specified (i.e. not []). 

old_range = dbv_PlotRange [ mdb, new_range ]

Returns and optionally sets the range of the data currently displayed in the plot of the specified Database Viewer. The old range is always returned. A new value of range is set if new_range is specified (i.e. not []). Ranges are specified as [minimum_value, maximum_value].
