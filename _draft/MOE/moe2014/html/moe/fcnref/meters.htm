Syntax

    [meter_keys, types, atom_vectors] = Meters []
    meter_keys = MeterCreate [ types, atom_vectors ]
    serial = MeterChanged []
    MeterDestroy meter_keys

Description

[meter_keys, types, atom_vectors] = Meters []

Returns information on all meters in the system:

meter_keys
Identifying key of meter. Returns a vector of meter_keys if there is more than one meter in the system.

types
Type of meter:
  • 'distance'
  • 'angle'
  • 'dihedral'

Returns a vector of types if there is more than one meter in the system.

atom_vectors
Vector of atoms involved in meter measurement. There are as many atom vectors as there are meters in the system. The number of atoms reflects the type of meter: 4 atoms for dihedral angle measurements, 3 for angles, and 2 for distances. 

meter_keys = MeterCreate [ types, atom_vectors ]

Creates as many meters as there are vectors of atoms in atom_vectors and returns the key of each meter.

Arguments used in this function are as described above in Meters.

Lengths of types and atom_vectors must match; unit extension applies.

For example, to create an angle meter to measure the angle between atoms a1, a2 and a3, use: 

    svl> MeterCreate ['angle', [ [a1,a2,a3] ] ]


serial = MeterChanged []

Returns the serial number (an identifier internal to the MOE system) of the last modification to a meter or a restraint. For instance, two calls to MeterChanged which return the same serial number indicates that no changes have occurred during the time between the calls. 

MeterDestroy meter_keys

Destroys all meters specified by keys in meter_keys.
