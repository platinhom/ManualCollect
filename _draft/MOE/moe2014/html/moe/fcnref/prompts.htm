Syntax

    AtomPrompt 'prompt_string'
    AtomPrompt []

    ResiduePrompt 'prompt_string'
    ResiduePrompt []

    ChainPrompt 'prompt_string'
    ChainPrompt []

Description

AtomPrompt 'prompt_string'
AtomPrompt []

Prompts the user to select an atom in the MOE Window. The prompt_string is displayed in the prompt area of the MOE window. If no prompt string is supplied to the function, the default string 'Pick an atom:' is used. 

ResiduePrompt 'prompt_string'
ResiduePrompt []

Prompts the user to select a residue from the Sequence Editor. The prompt_string will be displayed in the prompt area of the Sequence Editor. If no prompt string is supplied to the function, the default string 'Pick a residue:' is used. 

ChainPrompt 'prompt_string'
ChainPrompt []

Prompts the user to select a chain from the Sequence Editor. The prompt_string will be displayed in the prompt area of the Sequence Editor. If no prompt string is supplied to the function, the default string 'Pick a chain:' is used.

In all cases, selecting the object causes the prompt string to disappear and returns the key of the selected object. The selected object is not added to the selection set, which is preserved. Prompting can be canceled by typing Esc in the window displaying the prompt.
