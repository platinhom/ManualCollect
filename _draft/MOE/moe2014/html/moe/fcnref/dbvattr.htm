Syntax

    dbv_SetDatabase [ dbvkey, 'mdb_filename' ]
    old_state = dbv_ReadOnly [ dbvkey, new_state ]

Description

dbv_SetDatabase [ dbvkey, 'mdb_filename' ]

Sets the Database Viewer specified by dbvkey (returned by dbv_Open) to view the database 'mdb_filename'. It is an error to attempt to use this function to view a database that is already being viewed by another Database Viewer at the time of the call. 

old_state = dbv_ReadOnly [ dbvkey, new_state ]

Returns and optionally sets the read-only status of the specified Database Viewer. The old state is always returned (1 = read-only, 0 = read-write). The new state is set if new_state is specified (i.e. is not []).
