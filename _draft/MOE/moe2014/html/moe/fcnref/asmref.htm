Syntax

    [slogp, smref] = aSlogPMRef atom_keys
    slogp          = aSlogP     atom_keys
    smref          = aSMRef     atom_keys

Description

[slogp, smref] = aSlogPMRef atom_keys
Returns the log of the octanol-water partition coefficient and the molar refractivity contribution of each atom in the argument. The values are returned as two vectors, each of which is the same length as atom_keys.

Both the log octanol-water partition coefficient and the molar refractivity are calculated using an atomic contribution model [Crippen 1999] that takes into account implicit hydrogens. The model assumes correct protonation states (washed structures). 

slogp = aSlogP atom_keys
Returns the log octanol-water partition coefficient of each atom in the argument. 

smref = aSMRef atom_keys
Returns the molar refractivity contribution of each atom in the argument.

References

[Crippen 1999] Wildman, S.A., Crippen, G.M.; Prediction of Physicochemical Parameters by Atomic Contributions; J. Chem. Inf. Comput. Sci. 39 (1999) 868–873.
