Syntax

    atom_lists = cAtoms chain_keys 
    residue_lists = cResidues chain_keys

Description

atom_lists = cAtoms chain_keys

Returns a list of all the atom keys that are the grandchildren of each chain specified in chain_keys. 

residue_lists = cResidues chain_keys

Returns a list of all the residue keys that are the children of each chain specified in chain_keys.

Note: In both cases, the length of the return vector is always equal to the length of the argument vector.
