Syntax

    SaveSelection []
    RestoreSelection []

Description

SaveSelection Saves the current atom selection set.
RestoreSelection Restores the saved atom selection sets in reverse-chronological order.

Selections are saved onto a stack (last-in-first-out structure) of size 16. A saved selection is popped off the stack (and hence cleared from memory) with each call to RestoreSelection, i.e. a particular selection may be restored only once.
