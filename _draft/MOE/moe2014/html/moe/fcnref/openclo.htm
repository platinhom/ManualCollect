Syntax

    Open 'filename'
    Open []

    Close options
    Close []

Description

Open 'filename'
Open []

Reads the file 'filename', which can include a directory path. The format of the file is automatically detected. If called with a null argument, Open[] displays the Open Panel and returns. See the Open Panel documentation for a list of recognized formats. 

    SVL> Open 'crambin.pdb' // PDB format
    SVL> Open 'aspirin.moe' // MOE format


Close options
Close []

Removes all graphical and molecular objects from the system, i.e. clears the workspace. If the current molecular data has not been saved, and the function call was issued with a null argument, a confirmation prompt appears. options is a tagged vector that can contain any number of the following name-value pairs:

force:flag
A non-zero value of flag will force a close without confirmation; a zero value (default) will cause prompting to occur.
viewreset:flag
A non-zero value of flag (default) will cause the view to reset; a zero value will keep the current view.
delgobj:flag
A non-zero value of flag (default) will cause all graphic objects to be deleted; a zero value will not.
