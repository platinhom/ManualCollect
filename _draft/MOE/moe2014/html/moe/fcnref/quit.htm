Syntax

    Quit []
    quit []

Description

Quit []

Terminates the MOE session. If the molecular data currently displayed in the MOE Window has been modified since the last save operation, you will be queried as to whether to discard the data and leave, or to stop the quitting process.

Also terminates a moebatch session.

quit []

Unconditional halt of MOE. No warnings are given for unsaved data.
