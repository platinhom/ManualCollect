Syntax

    Save []

    SaveAs 'filename'
    SaveAs []

    oldname = DataFilename 'filename'
    filename = DataFilename []

    oldflag = DataModified flag
    flag = DataModified []

Description

Save []

Writes the current molecular data to the current data file. If the current data file has not been set, then Save opens the Write Molecule window. Save ignores any argument passed to it.

SaveAs 'filename'
SaveAs []

Writes the current molecular data to the file filename. The current data filename will be set to filename. If called with a null argument, SaveAs opens the Write Molecule panel.

In general, the filename you provide in the Write Molecule panel should have the extension appropriate to the write format, e.g. .moe or .pdb. Recognized formats and their file extensions include:

MOE .moe
PDB .pdb .ent
TriposMOL2 .mol2
MDLMOL .mol
PIR .pir
GDE .gde
Fasta .fst .fasta
Clustal .aln
MSF .msf
Phylip .phy

For a complete list see the File Format option menu in the Write Molecule panel. The default format for unrecognized extensions is the MOE format.

oldname = DataFilename 'filename'
filename = DataFilename []

Sets the current data file to filename. The current data file is the file into which the current molecular data is written on a save operation. DataFileName always returns the old value of the data filename.

When called with a null argument, DataFilename returns the name of the current data file without modification.

For example, to set the file trial3.moe to be the current data file, you would type:

    SVL> DataFilename 'trial3.moe'


oldflag = DataModified flag
flag = DataModified []

Sets the data-modified flag to flag. Any change in the molecular system sets the data-modified flag to 1 (true). This state may be overridden using DataModified. A zero flag sets the data-modified state of the system to unmodified. The return value is always the status of the prior data-modified flag.

When called with a null argument, DataModified returns the current data-modified flag without changing it.

Note that DataModified returns nothing if called as a procedure.
