Syntax

    old_num = AntiAlias num_samples

    old_flag = HighQualityRotation flag
    old_flag = FullScreen flag
    old_flag = FatLines flag

Obsolete

    old_flag = PolyhedralAtoms flag
    old_flag = PolyhedralBonds flag
    old_flag = WireframeRotation flag

Description

old_num = AntiAlias num_samples
Set high quality anti-aliasing mode. The value num_samples specifies the number of samples used for high quality anti-aliasing. The higher the number the better the quality (but also the slower the rendering). A value of 1 will switch to line-only anti-aliasing; 0 will turn anti-aliasing off. If [] is passed, the current value is left unchanged and the value returned. 
    svl> AntiAlias 0; // turn off anti-aliasing 
    svl> AntiAlias 4; // turn on high quality anti-aliasing with 4 samples
    svl> AntiAlias []
    4 


old_flag = HighQualityRotation flag
old_flag = FullScreen flag
old_flag = FatLines flag
These functions turn on (flag is <>0) or off (flag is 0) the current rendering mode of the specified function. If [] is passed, the current value is left unchanged and the value returned. 
    svl> FullScreen 0; // turn off fullscreen mode
    svl> FullScreen []
    0
