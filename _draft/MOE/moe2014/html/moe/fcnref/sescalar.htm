Syntax

    value = SEScalarPlotName index
    old_name = SEScalarPlotName [ index, 'name' ]

    [old_ridx, old_cidx] = SEDisplayPos [ ridx, cidx ]

    [nres, nchains] = SEDisplaySize []

Description

value = SEScalarPlotName index
old_name = SEScalarPlotName [index, 'name']

Reports/sets the name of the Sequence Editor scalar plot indicated by index. Index values begin at 1.

When passed only an index, returns the name of the associated plot. When passed both a name token and an index, the name of the scalar plot is set to name; the pre-modified value is returned.

[old_ridx, old_cidx] = SEDisplayPos [ ridx, cidx ]

Reports/sets the indices of the residue and chain to appear in the top left corner of the Sequence Editor drawing area. The pre-modified display positions are always returned. Indices start at 1. Out-of-range values are clamped. If SEDisplayPos is passed [], the current top left residue and chain indices are returned without modification to their values.

[nres, nchains] = SEDisplaySize []

Reports the size of the Sequence Editor display in units of current residue width (which varies depending on whether single-letter display is enabled) and current chain height. In other words, a maximum of nres residues and nchains chains are visible in the Sequence Editor at a time.
