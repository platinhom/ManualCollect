Syntax

    LicenseInfo []
    [feature, version, expiry_date, key, server, num_tokens] = LicenseInfo []

Description

Returns information about the license file. The license file limits the usage of the licensed application, specifying the highest version of the software that can be run within the duration of the license, and the maximum number of copies that can be run simultaneously.

The returned information is as follows:

feature Name of the licensed application.
version Highest numbered version allowed.
expiry_date Expiry date, given as dd-mmm-yyyy. If year is 0000, the feature never expires.
key Hexadecimal authentication key used to verify that the license has not been modified.
server Hostname or IP address of the computer acting as license server.
num_tokens Total number of tokens allocated by the license. A value of 0 means unlimited.
For example: 
    svl> x = LicenseInfo []
    svl> x
    [ 'moe', '2003.05', '31-may-2003', '4E763DB0C667', 'vali.chemcomp.com', 3 ]

When the return value of LicenseInfo is not assigned to a variable, thefunction prints a report about the license. 

    svl> LicenseInfo []
     Feature: moe
     Version: 2003.05
     Expires: 31-may-2003
         Key: 4E763DB0C667
      Server: octane.chemcomp.com
      Tokens: 3
