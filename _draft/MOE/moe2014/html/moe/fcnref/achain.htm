Syntax

    chain_keys = aChain atom_keys
    residue_keys = aResidue atom_keys

Description

chain_keys = aChain atom_keys

Returns the key of the grandparent chain for every atom specified in atom_keys. 

residue_keys = aResidue atom_keys

Returns the key of the parent residue for every atom specified in atom_keys.
