Syntax

    atom_lists = rAtoms residue_keys 
    chain_keys = rChain residue_keys

Description

atom_lists = rAtoms residue_keys

Returns a list of child atom keys for each residue specified by a key in residue_keys. 

chain_keys = rChain residue_keys

Returns the parent chain key for each residue in residue_keys. The length of the return vector is always the same as the length of the argument vector.
