MOE/web SOAP Service

The MOE/web server can be used as a webservice provider. It offers a framework for building lightweight browser-based web applications, and function calls via the Simple Object Access Protocol (SOAP). This protocol can be used to interface MOE to third party software, such as KNIME and Pipeline Pilot.

SOAP Functions

The http://{host}:{port}/soap.html URL produces a directory page that lists the SOAP functions currently being hosted by the server. Clicking on the name of a SOAP function on this page will provide a basic overview of the function, as well as an inteface to test its SOAP functionality.

More detailed information is provided here for these SOAP functions:

  • AntibodyModeler SOAP function
  • Gaussian SOAP function
  • HomologyModeler SOAP function
  • Ph4Search SOAP function
