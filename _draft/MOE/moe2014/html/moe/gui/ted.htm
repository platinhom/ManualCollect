Introduction

The SVL Text Editor is used to create and edit ASCII files. It is especially convenient for creating and modifying SVL code since the syntax in files with the .svl extension is automatically colored in the editor. Colors can be configured in the user configuration file (moe-rc or .moe-rc). In MOE, the SVL Text Editor is often used from within an SVL application to display a series of results. Such is the case for the binary classification tree and QuaSAR-Model, for instance.

The SVL Text Editor can be opened from the MOE Window or the Sequence Editor by choosing File | New | Text Editor. Any number of MOE SVL editors can be opened at the same time; however, it is not possible to open a given file in different ones. Currently open editors are listed under the Window | Text Editors menu, and files that have not been saved appear as <Untitled>. Choosing a file from the list raises that Text Editor.

When first opened, the Text Editor is empty. You can invoke commands using the menu bar at the top of the Text Editor or a popup menu. The popup is displayed by pressing the mouse popup button over the text display area. It includes the same commands as the menu bar.

To open an existing file, choose Open from the File menu in the editor's menu bar. To create a file in the SVL Text Editor, choose the Save As command. The following snapshot depicts the SVL file verlet.svl:

You can use the scroll bars to move vertically or horizontally through the text, or you can use the options in the popup menu by clicking the mouse popup button over the scroll bar area.

Loading and Saving Files

File Menu Keystrokes Action
Open Ctrl+O Displays the Open Text Editor File panel. This file selection box allows you to open an existing file.
New --- Opens a new Text Editor window. Equivalent to File | New | Text Editor in the main window.
Save Ctrl+S Saves the contents of the Text Editor providing that a save file is currently specified (e.g., the contents have already been saved using the Save As command).
Save As --- Displays the Save Text Editor File As panel. This file selection box allows you to save the contents of the Text Editor under a new name.
Clear --- Clears the contents of the Text Editor, providing that the contents have already been saved.

If the contents have not been saved, you will be asked whether you wish to discard the unsaved changes. Choosing No cancels the Clear operation, enabling you to save. Choosing Yes deletes the contents of the editor.
Include --- Displays the Include Text Editor File panel which allows you to merge the contents of another file into the current Text Editor, at the cursor position.
Print --- Displays the SVL Text Editor Print Preview panel which allows you to send the content of the Text Editor to a printer.
Quit --- Closes the Text Editor. You will be asked whether you wish to save any unsaved changes.

Editing Files

Edit Menu Keystrokes Action
Undo Ctrl+Z Cancels the last action that was performed in the Text Editor such as the replacement or deletion of a string. It is possible to undo several operations by repeatedly selecting Undo.
Redo Ctrl+Y Redoes the last action that was "undone" (with the Undo command). This command only becomes available after an Undo is performed.
Cut Ctrl+X
or
Shift+Delete		 Cuts the highlighted text to the clipboard.
Copy Ctrl+C
or
Ctrl+Insert		 Copies the highlighted text to the clipboard.
Paste Ctrl+V
or
Shift+Insert		 Pastes the contents of the clipboard at the cursor position.

Finding and Replacing Strings

Find Menu Keystrokes Action
Find and Replace Ctrl+F Displays the Find and Replace footer in the Text Editor. To find a string, enter the search text in the text field next to the Find button and press the Find button or Enter. The matched text will be highlighted in the editor.

To close the Find and Replace footer, press Esc.
Find Next Ctrl+N Finds the next match of the search text. Pressing the Find button produces the same result.
Replace Then Find -- Replaces the matched text with the replacement text entered in the Replace field, which appears in the footer of the editor. The next match is highlighted in the editor.
Replace All --- Replaces all matched text with the replacement text entered in the Replace field, which appears in the footer of the editor. Pressing the Replace All button produces the same result.

Note: In order for the Find Next, Replace Then Find and Replace All commands to be available, you must first have specified the search text using the Find and Replace command.

Modifying the Display

Display Menu Keystrokes Action
Goto Line Ctrl+G Allows you to go to a specific line in the file. Enter the line number in the Goto Line text field in the footer of the panel and press Enter. The specified line is highlighted in the editor.
Small Font --- Displays the contents of the file in small size font.
Normal Font --- Displays the contents of the file in normal size font.
Large Font --- Displays the contents of the file in large size font.

Note: Font size, whether it be set to Small, Normal or Large, is machine dependent.

Loading and Running SVL Files

SVL Menu Keystrokes Action
Save and Load Ctrl+L Saves the editor contents to a file, then compiles and loads all functions found in the file. If a syntax error is encountered, the file is not loaded. Note that the executable code is kept in memory until the program is unloaded. Loaded programs appear in the SVL Modules and Tasks panel.
Save and Run --- Saves the editor contents to a file and runs the program. If a syntax error is encountered, the file is not run The program is in memory only for the duration of execution and is subsequently freed.

If an error message appears stating Module Entry Point Not Found the file requires either a main function or #set main 'fcn name' directive.
Execute Command Ctrl+E Opens a temporary SVL command line in the footer of the Text Editor. As with all CLIs, the output of the executed command is displayed in the SVL Commands window.
Edit Function --- Allows you to find the code of an SVL function. Highlight the name of the SVL function in the SVL Text Editor and choose Edit Function. This will open the file containing the code of the SVL function in a new Text Editor window.

If this operation is unsuccessful, this means that the function is not an SVL function or the file in which the function is defined is not currently loaded in the system.
