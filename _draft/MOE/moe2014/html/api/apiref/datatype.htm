Syntax

    int svl_DataType(svl_var v);

Description

Returns the data storage type used in variable v. This is required for efficient processing by a C switch-statement, typically in the callback function defining a generalized unary or binary operation.

If the variable is a flat vector of scalar of C storage type: 

    svlChar
    svlInt
    svlReal
    svlTok
the function return value is: 
    SVL_C_
    SVL_I_
    SVL_R_
    SVL_T_

If the variable is not a flat vector of scalars or its storage type is not one of the above, the function returns a value other than one of the predefined constants.
