Syntax

    void svl_MustMakeConst_I(char *name, svlInt val);
    void svl_MustMakeConst_R(char *name, svlReal val);
    void svl_MustMakeConst_string(char *name, char *val);
    void svl_MustMakeConst_strings(char *name, char **val);
    void svl_MustMakeConst_nstrings(char *name, char **val, int n);
    void svl_MustMakeConst_V(char *name, svl_var val);

Description

The MustMakeConst functions create constants available within SVL. name is the symbol used in SVL, and val is the value associated with name.

svl_MustMakeConst_I
svl_MustMakeConst_R
svl_MustMakeConst_string 		Create a scalar constant named name of type integer, floating-point, and token, respectively. The constant's value is set to val.

svl_MustMakeConst_strings Creates a constant named name, containing a flat list of tokens that is initialized to the contents of val, a NULL-terminated table of C strings.

svl_MustMakeConst_nstrings Creates a constant named name, containing a flat list of n tokens that is initialized to the contents of the first n elements of the table of C strings, val.

svl_MustMakeConst_V Creates a constant named name, initialized by the SVL variable val.
