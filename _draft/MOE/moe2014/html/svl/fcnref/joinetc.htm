Syntax

    common_values = join [v, w] 

    join_mask = m_join [v, w]

    join_indices = x_join [v, w]

Description

Set intersection functions. 

common_values = join [v, w]

Returns all elements of v that also occur in w, preserving ordering and duplicates in v. 

    svl> join [ "afg", "cfigg" ]
    "fg"
    svl> join [ "cfigg", "afg" ] 
    "fgg" 


join_mask = m_join [v, w]

Returns mask join_mask such that: 

    join [v, w] === mget [v, join_mask]

    svl> m_join [ "cfigg", "afg" ]
    [0,1,0,1,1]


join_indices = x_join [v, w]

Returns index list join_indices such that: 

    join [v, w] === get [v, join_indices] 

    svl> x_join [ "cfigg", "afg" ]
    [2,4,5]
