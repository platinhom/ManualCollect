Syntax

    n = length v
    m = l_length v

Description

n = length v
Returns the number of top-level elements of v. Note that: 
    v === v [ igen length v ]

length v is equivalent to last x_id v. 

    svl> length [ [2,3], [4,5,6], [7,8] ] 
    3 
    svl> length "abc"
    3 
    svl> length ["abc", "def"]
    2 


m = l_length v
Returns the laminated length of v, defined as the length of tr v: 
    l_length v === length tr v

For example: 

    svl> length [[1,2,3],[4,5,6]]
    2
    svl> l_length [[1,2,3],[4,5,6]]
    3
    svl> l_length [[1,2,3],[4,5]]
    error
