Syntax

    print expression;
    pr val

Description

print expression;

print is an SVL keyword that causes the value of expression to be echoed to the CLI. print is not an SVL function, i.e.

  • The expression does not have to be parenthesized
  • The keyword cannot be used inside an expression 
    svl> print a = "enterprise";
    "enterprise"

    svl> print 5 + 6;
    11


pr val

pr is an SVL function that returns the value of its argument with the side-effect of printing the result at the SVL Commands window. pr is useful for printing intermediate results when debugging programs. Unlike the keyword print, function pr can be used inside an expression. 

    svl> a = pr sort [7,3,5,2] 
    [2,3,5,7] 

    svl> b = first pr a;
    [2,3,5,7]

    svl> pr 5 + 6;
    5
