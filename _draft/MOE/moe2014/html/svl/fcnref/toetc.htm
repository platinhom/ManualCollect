Syntax

    val = tolower v 
    val = toupper v 
    val = tofupper v

Description

Changes the case of letters in the character or token vector v. Non-letter characters are left unchanged. tofupper converts the first letter to uppercase and the remainder to lowercase. 

    svl> toupper "3aB ." 
    "3AB ." 
    svl> toupper ["3aB .", 'y-'] 
    ["3AB .", 'Y-'] 

    svl> tofupper 'hElLo WoRlD'
    'Hello world'

    svl> tolower "3aB ." 
    "3ab ."
