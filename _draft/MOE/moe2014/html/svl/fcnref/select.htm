Syntax

    result = select [v,w,mask]

Description

Selection by mask. For each element of mask, one element is returned, either from v or from w. A non-zero value in mask results in the corresponding element in v being returned. A zero value result in the corresponding element of w being returned. mask must be a flat numerical vector. All 3 arguments must unit-extend to the same length.

    svl> select [[1,2,3,4], [10,20,30,40], [1,1,0,0]] 
    [1,2,30,40]
