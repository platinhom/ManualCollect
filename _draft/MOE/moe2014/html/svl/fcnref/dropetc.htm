Syntax

    w = drop [v, val]
    w = drop [v, val1, val2]

    w = keep [v, val]
    w = keep [v, val1, val2, ...]

    all_but_first = dropfirst v 
    all_but_last = droplast v

    mask = m_dropfirst v
    indices = x_dropfirst v 
    mask = m_droplast v
    indices = x_droplast v

    first_element = first v
    second_element = second v
    third_element = third v
    last_element = last v

    mask = m_first v 
    mask = m_last v

Description

w = drop [v, val]
w = drop [v, val1, val2]

If val is a positive number, drop drops the first val elements of v, and returns the remaining elements. Otherwise, if val is negative, drop drops the last val elements. 

    svl> a = [1,2,3,4,5]
    svl> drop [a, 3] 
    [4,5] 
    svl> drop [a, -3] 
    [1,2]
    svl> drop [a, 3, -1] // drop[ drop[a,3], -1 ] 
    4


w = keep [v, val]
w = keep [v, val1, val2, ...]

If val is a positive number, keep returns the first val elements of v. Otherwise, if val is negative, keep returns the last val elements of v.

    svl> keep [a, 3]
    [1,2,3] 
    svl> keep [a, -3] 
    [3,4,5] 
    svl> keep [a, -4, 3] // keep[ keep[a,-4], 3 ]
    [2,3,4]


all_but_first = dropfirst v
all_but_last = droplast v

dropfirst returns all but the first element of v, and droplast returns all but the last. 

    svl> v = [1,2,3,4,5]
    svl> dropfirst v 
    [2,3,4,5]
    svl> droplast v 
    [1,2,3,4]


mask = m_dropfirst v
indices = x_dropfirst v
mask = m_droplast v
indices = x_droplast v

m_dropfirst and m_droplast each returns a mask whose length is the same as that of v. In both cases, the returned mask has only one zero value. m_dropfirst has a zero as its first element, and m_droplast has a zero as its last. 

    svl> m_droplast v 
    [1,1,1,1,0]

x_dropfirst returns a vector of all indices of v except the first. This is equivalent to dropfirst x_id v. 

    svl> v = [9,6,3,7,5]
    svl> x_dropfirst v 
    [2,3,4,5]

x_droplast returns all indices but the last. This is equivalent to droplast x_id v. 

    svl> v = [9,6,3,7,5] 
    svl> x_droplast v
    [1,2,3,4]


first_element = first v
second_element = second v
third_element = third v
last_element = last v

first returns the first element of v; second, the second element; third, the third element; and last, the last element. 

    svl> v = [1,2,3,4,5]
    svl> first v
    1 
    svl> second v
    2
    svl> third v
    3 
    svl> last v
    5


mask = m_first v 
mask = m_last v

m_first and m_last each returns a mask the same length as vector v. In each case, the mask has only one non-zero element. m_first has 1 as its first element, and m_last has 1 as its last. 

    svl> m_first v 
    [1,0,0,0,0]
    svl> m_last v 
    [0,0,0,0,1]
