Syntax

    v = tag [ attributes, vals ]
    [attributes, values] = untag v 
    tag_vector = tags v 
    val_vector = tagvals v

Description

v = tag [ attributes, vals ]

Given a vector of tag tokens attributes and a vector of tag values vals, combine them into attribute-value pairs to form a tagged vector v.

[attributes, values] = untag v

Returns the tag tokens of tagged vector v and their corresponding values, separated into 2 vectors. To strip out the tags from a tagged vector v, use: first untag v 

tag_vector = tags v

Returns the tags of a tagged vector. tags v is equivalent to first untag v.

Examples: 

    svl> v = [ a:3, b:'four' ]; 
    svl> untag v 
    [ ['a','b'], [ 3, 'four' ] ] 
    svl> tag untag v 
    [ a:3, b:'four' ] 
    svl> v = [ a:3, b:[ c:4, d:5 ] ]; 
    svl> v.b.d 
    5 
    svl> tags v
    ['a','b'] 
    svl> tags v.b
    ['c','d'] 


val_vector = tagvals v

Returns the values of a tagged vector. tagvals v is equivalent to second untag v.

Examples: 

    svl> v = [ a:3, b:'four' ]; 
    svl> tagvals v
    [3, 'four']
