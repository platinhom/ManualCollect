Syntax

    val = funcL [v1, v2] 
    val = funcL [v1, v2, v3, ...] 

    val = a === b    // infix notation for eqL

Description

Relational lexicographic functions. Arguments being compared need not be conformable, i.e. need not be of same shape or type (although comparison may be meaningless if types are different). Comparisons are performed at the top level of the argument vector. Each function returns 1 if the comparison is true, 0 otherwise. Tokens are compared lexicographically by their contents.

When more than two arguments are passed, reduction takes place, i.e.: 

    funcL [v1, v2, v3, ...]

is equivalent to: 

    funcL [v1, v2] and funcL [v2, v3, ...]

Note: Only eqL has an infix form.

Function Description
cmpL Lexicographic comparison.
This function returns:
-1 if ltL [v1, v2, ...]
0 if eqL [v1, v2, ...]
1 if gtL [v1, v2, ...] 
    svl> cmpL [ 'b', 'a' ]		// 'b' > 'a'
    1
    svl> cmpL [ [4,5,6], 5 ]		// 4 < 5
    -1 
    svl> cmpL [ "ab", "ab" ]	// "ab" == "ab"
    0
eqL Shorthand notation: ===
Returns 1 if the top level arguments are identical. 
    svl> eqL [1,1,1]
    1 
    svl> eqL [ [1,2], [1,3] ]
    0 
    svl> [ 0.3, 'cart' ] === [ 0.3, 'cat' ]
    0
neL Equivalent to not eqL.
Returns 1 if the top level arguments are not identical.
geL Lexicographic greater than or equal to comparison.
Returns 1 if the top level arguments are sorted in decreasing order from left to right. 
    svl> geL [3,2,1]
    1 
    svl> geL [ 3, 2, [1,4] ]
    1 
    svl> geL ['a','c','b']
    0 
    svl> geL [3,2,2,1]
    1
gtL Lexicographic greater than comparison.
Returns 1 if the top level arguments are sorted in strictly decreasing order from left to right. 
    svl> gtL [3,2,1]
    1 
    svl> gtL [ 3, 2, [1,4] ]
    1 
    svl> gtL ['a','c','b']
    0 
    svl> gtL [3,2,2,1]
    0
leL Lexicographic less than or equal to comparison.
Returns 1 if the top level arguments are sorted in increasing order from left to right. 
    svl> leL [1,2,3]
    1 
    svl> leL ['a','c','b']
    0 
    svl> leL [1,1,2]
    1
ltL Lexicographic less than comparison.
Returns 1 if the top level arguments are sorted in strictly increasing order from left to right. 
    svl> ltL [1,2,3]
    1 
    svl> ltL ['a','c','b']
    0 
    svl> ltL [1,1,2]
    0
