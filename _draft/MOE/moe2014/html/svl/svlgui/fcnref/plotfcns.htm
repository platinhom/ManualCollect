Syntax

    wkey = Plot  [[x1,...,xn],[x1,...xn],...,[x1,...,xn]];
    wkey = PlotS [[[x1,...xn],[y1...yn]],..., [[x1,...xn],[y1...yn]]];
    wkey = PlotC [matrix of data];
    wkey = PlotH [[x1,...,xn],[x1,...xn],...,[x1,...,xn]];

    PlotSet [wkey, attributes_value_pairs];

Description

The Plot functions are essentially convenience routines that plot data in a variety of manners such as in scatter and contour plots. Each function opens a window containing a plot widget and returns the window key.

PlotSet is used to set plot attributes.

Function Description
Plot 1D function plot. Plots the values for each series of data against sample numbers that have no unit. 
svl> t = 0.01 * igen 628
svl> wkey = Plot [ sin t ]
PlotS 2D scatter plot. Plots y against x, where x represents the horizontal axis and y, the vertical. Both x and y must be flat vectors.

svl> t = 0.01 * PI * igen 360
svl> PlotS [ [ t * (180/PI), sin t ] ] // plot in degrees
PlotC 3D contour plot. Plots iso-level contours of the z values. z is a matrix of values of a function f(x, y) sampled on a rectangular m×n grid, and is given row-wise as a vector of length m (number of rows), each element of which is of length n (number of columns). The distance between contours cannot be modified. Only one series may be viewed at a time. 
svl> t = 0.01 * igen 314
svl> wkey = PlotC [ [sin t]*cos t ]
PlotH Histogram plot. Displays a bar plot for each series in the given data. 
svl> wkey = PlotH [ igen 10];
PlotSet Sets the attribute information on the given plot window. Plot attributes are summarized in Plot widget. 
wkey =  Plot [igen 10]
PlotSet [wkey, 
    [plotTitle: 'This is a test', seriesColor: ['green']]
];
