Description

A numeric text input field accompanied by a slider. The precision of the numeric value is determined by incr in the range attribute. Scale behaves identically to the Spin button except that a slider exists instead of the two arrow buttons.

Attributes

Scale inherits all core attributes and defines the following attributes:

Attribute Description Values Default
width Size of the slider in character widths 1 to 255 10
vertical Orientation of slider 1 = vertical
0 = horizontal		 0
len Number of characters displayed in the text field 1 to 255 10
range Data increment and range multipliers: [minfac,maxfac,incr]

The actual data range is given by [minfac*incr,maxfac*incr]; the end values are included in the range

 [minfac, maxfac, increment] [INT_MIN, INT_MAX, 1]

Trigger

Mouse drag on the slider or Return pressed within the scale's text field.

Data Value

The floating point number corresponding to the slider position.
