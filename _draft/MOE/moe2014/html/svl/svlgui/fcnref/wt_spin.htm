Description

The spin button is a numeric text field accompanied by arrow buttons for incrementing/decrementing the object's numeric value. The precision of the numeric value is determined by incr in the range attribute. Spin behaves identically to Scale except that, instead of a slider, it has arrow buttons.

Attributes

Spin inherits all core attributes and defines the following attributes:

Attribute Description Values Default
vertical Orientation of arrow buttons 1 = vertical layout
0 = horizontal layout		 1
len Number of characters displayed in the text field 1 to 127 10
range Data increment and range multipliers: [minfac,maxfac,incr]

The actual data range is given by [minfac*incr,maxfac*incr]; the end values are included in the range

 [minfac, maxfac, incr] [INT_MIN, INT_MAX, 1]

Trigger

Mouse press on either arrow button or Return pressed within the spin text field.

Data Value

The number corresponding to the number entered in the text field.
