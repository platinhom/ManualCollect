Description

The Plot widget is a display area used to plot data and set plot-related attributes.

Attributes

Plot inherits all core attributes and defines the following attributes:

General Plot Attributes

Attribute Description Values Default
plotTitle Title to display at top of plot token ''
plotType Type of plot 'contour', 'function', 'histogram', 'scatter' 'function'
series Data for each series Vector of series data []
showAxes Turn on/off display of axes 0,1 1
showGridlines Turn on/off display of grid 0,1 1
showLegend Turn on/off display of legend at top of plot 0,1 1
showTicks Turn on/off display of axis ticks 0,1 1
showXAxisLabels Turn on/off display of labels on X axis 0,1 1
showYAxisLabels Turn on/off display of labels on Y axis 0,1 1
XAxisRange Lower and upper bounds for X axis Vector of 2 numbers Calculated automatically
YAxisRange Lower and upper bounds for Y axis Vector of 2 numbers Calculated automatically
ZAxisRange Lower and upper bounds for Z axis Vector of 2 numbers Calculated automatically
XAxisTickSpacing Spacing on X axis real Calculated automatically
YAxisTickSpacing Spacing on Y axis real Calculated automatically
ZAxisTickSpacing Spacing on Z axis real Calculated automatically
XAxisTitle Title to display on X axis token ''
YAxisTitle Title to display on Y axis token ''
ZAxisTitle Title to display on Z axis token ''

Series-Specific Attributes

Attribute Description Values Default
seriesColor Colors of each series in plot Vector of PLOT_COLORS 'black'
seriesLineStyle Line styles for each series in plot Vector of PLOT_LINE_STYLES 'solid'
seriesMarker Marker shape - see note below Vector of PLOT_MARKERS 'none'
seriesMarkerStyle Marker style - see note below Vector of PLOT_MARKER_STYLES 'none'
seriesTitle Title for each series in the plot Vector of tokens ''

Note: The marker and markerStyle attributes both set the shape of plot marks. The marks shaped by marker intersect the actual plot point, whereas those determined by markerStyle encircle the point.

Constants used for the plot attributes are as follows:

Constant Values
PLOT_COLORS 'background', 'black', 'blue', 'brown', 'cyan', 'darkGray', 'darkGreen', 'darkShadow', 'gray', 'grayed', 'green', 'highlight', 'lightGray', 'lightShadow', 'magenta', 'orange', 'purple', 'red', 'selected', 'white', 'yellow'
PLOT_LINE_STYLES 'dashed', 'dotted', 'dotted-light', 'none', 'solid'
PLOT_MARKERS 'cross', 'dot', 'fine-dot', 'none'
PLOT_MARKER_STYLES 'filled-circle', 'filled-diamond', 'filled-square', 'filled-triangle', 'hollow-circle', 'hollow-diamond', 'hollow-square', 'hollow-triangle', 'none'

Trigger

A selection box is drawn by dragging and releasing the left mouse button over the Plot area. Releasing the mouse button is the trigger.

Data Value

The selection box is returned as the data value. The plot widget returns [[x1,y1],[x2,y2]], where [x1,y1] are the coordinates of the top left corner of the selection box and [x2,y2] are the coordinates of the bottom right corner of the selection box.
