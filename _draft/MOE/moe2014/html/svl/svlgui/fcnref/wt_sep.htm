Description

An object that draws a line to separate other objects visually. By default the line spans its parent lengthwise if the parent is a Vbox, or widthwise, if its parent is an Hbox. The default for a parent Mbox is a horizontally-oriented line. In all cases, either orientation may be explicitly specified.

If the flushLeft and/or flushTop attributes are not set, separators are subject to alignment points, and may not span the full width or height of their parents.

Attributes

Separator inherits all core attributes and defines the following attributes:

Attribute Description Values Default
vertical Orientation 1 = vertical
0 = horizontal		 see below

Notes

The shadow attribute defaults to 'etched-in'.

The orientation defaults to vertical if the separator is a child of an Hbox manager, horizontal otherwise.

Trigger

None.

Data Value

None.
