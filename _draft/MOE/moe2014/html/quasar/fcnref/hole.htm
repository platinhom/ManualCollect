Syntax

    QuaSAR_HoleFillerMDB [
	'output_database',
 	source_database_tokens,
	descriptor_tokens, 
      	quasar_cluster_options
    ]

    QuaSAR_HoleFiller []

Description

QuaSAR-HoleFillerMDB is used to generate a new database of compounds that spans the diversity of molecules in multiple source databases. The new database will contain all the molecules from the source databases which have unique cluster codes. Only one representative molecule is kept from a set of molecules with identical cluster codes. The cluster codes themselves will depend upon which descriptors are chosen for the calculation.

Supported options are:

'output_database'
Name of the output database, enclosed in single quotes.

source_database_tokens
Vector of source database filenames enclosed in single quotes, e.g. 
    [
	'database1.mdb',
	'database2.mdb',
	'database3.mdb',
	'database4.mdb'
    ]

A maximum of four source databases may be specified.

descriptor_tokens
Vector of descriptor tokens used by the diversity calculation. The descriptor names are enclosed in single quotes, e.g. 
    [ 'a_aro', 'a_count', 'chi0']

There is no limit to the number of descriptors allowed in a QuaSAR-HoleFiller calculation.

quasar_cluster_options
Tagged vector of options suitable for use by QuaSAR_ClusterMDB.

If any calculation option is omitted from the options vector, QuaSAR_HoleFillerMDB uses the default value for the missing option. For example, the command: 

    QuaSAR_HoleFillerMDB[
	'output.mdb', 
	['database1.mdb','database2.mdb'],
	['a_aro', 'chi0']
    ]

will initiate QuaSAR_HoleFiller with all cluster calculation options at their default values. The a_aro and chi0 descriptors will be used in the calculation.

Running QuaSAR_HoleFillerMDB[] without any options will generate an error. The output database filename, the filename of at least one source database and at least one descriptor must be specified in a QuaSAR_HoleFillerMDB[] command.

QuaSAR_HoleFiller[] opens the QuaSAR_HoleFiller panel. The user may then specify options in the HoleFiller panel and press OK to initiate a HoleFiller calculation.
