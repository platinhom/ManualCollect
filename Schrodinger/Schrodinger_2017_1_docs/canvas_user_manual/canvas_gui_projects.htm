Canvas Projects

Canvas stores all of its data in a project, which is simply a directory that contains a number of subdirectories and files. A Canvas project directory has the extension .cnv. Canvas does not have scratch projects, so you must create a project before you can import structures. Information on Canvas projects is given in the topics listed below.
