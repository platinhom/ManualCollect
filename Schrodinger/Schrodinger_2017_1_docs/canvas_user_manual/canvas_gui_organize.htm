Organizing the Canvas Spreadsheet Data

Canvas provides various ways of organizing the rows in the spreadsheet. Some of them, such as clustering, involve running an application. These means of organizing data are treated in Running Applications from Canvas. The simplest ways of organizing the data are sorting and partitioning, which are described in the first two subsections below. Another way of highlighting the data values is by coloring the table cells using a heat map.

The topics listed below describe ways in which you can organize the spreadsheet data.
