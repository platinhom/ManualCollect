Importing the Receptor

1. Choose File → Import Structures.
2. In the Import panel, select the file IFD_receptor.mae.

Navigate to your working directory if necessary.

3. Click Options.

The import options are displayed in the panel.

4. Ensure that Replace Workspace is selected.
5. Click Open.

The receptor-ligand complex appears in the Workspace, as shown below. The ligand is a nonstandard residue and is therefore colored differently from the receptor. You must have the receptor displayed in the Workspace to set up the job.
