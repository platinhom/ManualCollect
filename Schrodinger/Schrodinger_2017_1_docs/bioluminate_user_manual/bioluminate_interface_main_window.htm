The Maestro BioLuminate Main Window

The Maestro BioLuminate main window opens with the following features displayed by default:

  • Menu bar. This is at the top of the window on Linux and Windows, and is the menu bar on the Mac.

  • Main toolbar. This toolbar has tools for selecting and styling atoms, building structures, and access to jobs and tasks. See Selection Toolbar for more information on these tools. For BioLuminate, the ligand items are not present, and there is a quick select item for protein-protein interfaces ("I" button).

  • Favorites toolbar. This toolbar has favorite actions, which you can add or delete from the Task Tool. By default, it provides access to the Multiple Sequence Viewer and the Protein Preparation Wizard, among other actions.

  • Workspace Navigator. This dockable panel is displayed on the left side of the main window. See Workspace Navigator or Workspace Navigator Panel for more information on this panel.

  • Toggle Table. This dockable panel is displayed on the right side of the main window. You can undock it from the main window and redock it with the docking button.

    Many other panels are also dockable. You can change the docking behavior in the Preferences panel, which you open by choosing Edit → Preferences, or pressing Ctrl+, (⌘,) .

  • Workspace. This is the large black area that occupies the main part of the main window. It is where structures are displayed, along with any associated objects such as surfaces and text labels.

  • Footer toolbar. This toolbar is below the Workspace, and contains the Status Bar and the Workspace Configuration Toolbar. When the pointer is not over an atom in the Workspace, the Status bar gives information on the contents of the Workspace. When the pointer is over an atom, the status bar gives information on the identity of the atom.

  • Sequence Viewer. This is at the bottom of the main window and contains the sequences of the structures that are in the Workspace.

There are several other components of the main window that can be displayed when needed, from the Windowmenu. These components include the Find toolbar (also opened with Ctrl+F, ⌘F), which you can use to find structural components in the Workspace, like chains or residues; and the Clipping Planes window, which shows a top view of the Workspace and the planes where the structures in the Workspace are clipped for display.
