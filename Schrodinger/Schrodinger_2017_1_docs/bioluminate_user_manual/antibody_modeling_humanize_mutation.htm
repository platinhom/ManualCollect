Humanizing Antibodies by Residue Mutation

If you have an antibody model that is not based on a human species, you can humanize it by mutating residues to create a more human-compatible protein. Appropriate sites for mutation can be found by comparing the antibody to homologs and identifying residues that satisfy various criteria, such as solvent accessibility and maximum number of side-chain interactions with the antigen or with the antibody itself. Multiple mutations can be done at each site, but only one site is mutated in each mutant structure.

The mutations can be set up and run in the Antibody Humanization: Residue Mutation panel, which you open by choosing Tasks → Biologics → Humanization → Residue Mutation in the Task Tool. The panel has two tabs, one for setting up the criteria for choosing residues to mutate, and one for selecting the residues and defining the mutants. When both these tasks are done, you can start the job to mutate the residues.
