Minimizing the Structure Energy from the Toggle Table

You can minimize the energy of the Workspace or the selected atoms using the OPLS_2005 force field, by choosing Clean from the Action menu. The current selection is updated (or a new selection created) to refer to the minimized atoms. To avoid starting a lengthy calculation that is better performed in the background, Clean is limited to structures of fewer than 1000 atoms.
