Importing Homologs for Residue Scanning

If you already have a set of homologs for a structure that you want to use to identify potential mutation sites, or if you have other structures that you want to use as homologs, you can import them.

  • If you have a set of homologs in a file, click the Import homologs button and choose From file. A file selector opens so you can locate and import the file.

  • If you want to manually import structures from the PDB, click Import and choose From PDB ID, then specify the PDB ID in the dialog box that opens.

  • If the structures are in the Workspace, click Import and choose From Workspace.
