Trimming the 1orq Template Structure

Some further trimming down of the structure will be done before using the structure as a template.

1. In the main window, click the Define button on the Select toolbar.

The Atom Selection dialog box opens.

2. In the Residue tab, choose Residue number from the list.
3. Enter 1-104 in the Residue number text box and click Add.
4. In the Chain tab, select chain K.
5. Click Subtract, then click OK.
6. Press the Del key, or choose Edit → Delete Atoms.

The structure is now ready for use as a template. If you examine the status bar below the Workspace, it should report that there are 4178 atoms, 550 residues, 5 chains, and 10 molecules.
