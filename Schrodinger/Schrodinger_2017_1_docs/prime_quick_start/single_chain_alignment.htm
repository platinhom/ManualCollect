Saving the Alignment

Before making changes to the initial BLAST alignment, it is a good idea to save the results so that you can return to them later if you need to start over with the alignment. To do this, you will save the current run and give the run you are going to work in a new name.

1. From the File menu, choose Rename.
2. Enter Blast_Alignment in the dialog box, then click OK.
3. From the File menu, choose Save As.
4. Type New_Alignment in the text box and click OK.

The New_Alignment run is the one that is now open. The Blast_Alignment run has been closed, but can be reopened at any time.
