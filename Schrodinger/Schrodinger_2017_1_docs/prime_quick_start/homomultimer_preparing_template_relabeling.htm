Assigning a Chain Name in the 1orq Template

The next step is to assign the potassium ions to a separate chain.

  1. In the Structure Hierarachy in the main window, open the tree for the 1orq structure to Other → Metals / Ions.

  2. Select the six potassium ions, which are labeled K1 through K6.

    These ions are selected in the Workspace.

  3. In the 3D Builder in the main window, click Other Edits and choose Change Other Properties.

  4. Choose Residue / Chain Name from the Property option menu.

  5. Enter K in the Chain name text box.

  6. Click Apply, then click Close.

    The potassium ions are now assigned to chain K.

The status bar below the Workspace should report “Chn:5”.
