Homology Modeling of a Single Chain

This chapter provides an exercise for building and refining a model of a query sequence for which a sequence homolog can be identified using BLAST. While the tutorial is self-contained, you may find it useful to refer to the Prime User Manual or the online help for more detailed information on the panels or procedures.
