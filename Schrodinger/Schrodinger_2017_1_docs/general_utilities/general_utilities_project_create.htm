Creating a Maestro Project from Files: project_create

This utility creates a project using specified structure files. The syntax is:

$SCHRODINGER/run project_create structure-file [-t type] project.prj 
$SCHRODINGER/run project_create -f file-list [-t type] project.prj
$SCHRODINGER/run project_create -d dir [-l type] project.prj

The project name must include the .prj extension. For a description of the options, run the command with the -h option.
