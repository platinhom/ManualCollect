Reporting the Maestro and Release Version of a Project: project_version

This utility lists the Maestro version and release that could have produced a project. The syntax is:

project_version project1 [project2 ...]

Each project is listed with the Maestro version and the release name in parentheses. If the project has ben converted from one format to another, the version for the latest format is reported.
