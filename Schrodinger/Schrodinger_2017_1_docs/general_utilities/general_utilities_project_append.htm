Appending Structures to a Maestro Project: project_append

This utility appends the structures in the specified structure files to a project. The syntax is:

$SCHRODINGER/run project_append structure-file [-t type] project.prj 
$SCHRODINGER/run project_append -f file-list [-t type] project.prj
$SCHRODINGER/run project_append -d dir [-l type] project.prj

The project name must include the .prj extension. For a description of the options, run the command with the -h option.
