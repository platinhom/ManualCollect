Conversion Error Codes for pdbconvert

When running pdbconvert from UNIX, you will receive numerical messages (2, 1, 0) indicating the status of an attempted PDB conversion. These messages are equivalent to Maestro’s dialog box warnings. The message numbers are defined as follows:

2 (ERROR):

A fatal error occurred, and no Maestro file was generated. Check disk permissions and disk space. This failure can also occur if the PDB file has varying number of atoms in different MODELs.

1 (WARNING):

A Maestro file was created, but an error at or above the base error level was returned. At the default base error level, this value is returned when red or blue atoms are present, and for orange atoms if the duplicate CONECT record convention is not followed. See PDB Conversion and Converters for a description of the atom colors.

0 (OK):

A Maestro file was generated without errors at or above the base error level.
