Structure Extraction and Filtering Utilities

This section describes utilities for extracting a subset of structures from a structure file or a database. The first two utilities, maesubset and sdsubset, extract structures by index or title. The third utility, getpdb, extracts a structure or a chain from the PDB database.The fourth utility, ligfilter, extracts structures from a Maestro-formatted file based on values of properties.

Extracting Structures from a Maestro File: maesubset

Extracting Structures from an SD File: sdsubset

Extracting Proteins from the PDB: getpdb

Filtering Structures by Property: ligfilter

Removing Duplicate Structures: merge_duplicates
