CDR Grafting - Advanced Options Dialog Box

Set options for additional regions to graft and for customizing the CDR loop region.

To open this dialog box, click the Advanced Options button in the Antibody Humanization: CDR Grafting panel.

CDR Grafting - Advanced Options Dialog Box Features

  • Numbering scheme text
  • Additional regions to graft section
    • Vernier zone option
    • Canonical structural residues option
    • Interface residues option
    • User-defined option and text area
  • Use custom CDR loop region option
  • CDR loop table
  • Reset button
Numbering scheme text

The numbering scheme selected in the main panel is reported here. Residue designations in this dialog box are given in this numbering scheme.

Additional regions to graft section

Select additional regions to graft.

Vernier zone option

Perform grafting for the Vernier zone, which is a layer of residues that lies between the CDR and FR. The residues affected are listed in the text box below.

Canonical structural residues option

Perform grafting for the canonical structural residues, which are residues that are important in determining CDR conformations. The residues affected are listed in the text box below.

Interface residues option

Perform grafting for the interface residues, which are important residues at the heavy chain-light chain interface. The residues affected are listed in the text box below.

User-defined option and text area

Specify your own selection of residues to graft, in a comma-separated list. Residues are specified by L:number or H:number for the light or heavy regions.

Use custom CDR loop region option

Use a custom definition of the CDR loop region, which overrides the default automatic definitions.

CDR loop table

This table lists the CDR loops and the start and end residue for each. You can change the loop definitions by editing the table cells.

Reset button

Reset all the settings in this dialog box to their defaults.
