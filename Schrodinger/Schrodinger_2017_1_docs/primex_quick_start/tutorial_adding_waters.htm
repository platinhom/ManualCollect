Adding Waters to p38 Kinase

Now that the protein and ligand have been fairly well refined, the next step is to add waters.

1. Click Water Placement in the PrimeX panel.

The Water Placement panel opens. The defaults can be used for this job.

2. Start the job with an appropriate entry title.
3. Run a reciprocal-space coordinate minimization, following the instructions in Reciprocal-Space Minimization for p38 Kinase.
4. Run a B-factor refinement with a B-factor restraint weighting factor of 0.2, following the instructions in B-Factor Refinement for p38 Kinase.
