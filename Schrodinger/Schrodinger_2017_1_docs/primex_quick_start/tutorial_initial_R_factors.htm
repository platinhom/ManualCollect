Generating Initial R-Factors for p38 Kinase

1. In the PrimeX panel, click R-Factor Calculation.

The R-Factor Calculation panel opens. Since the calculation settings have already been made, there is no need to change them. There is also no need to make job settings as the default settings replace the structure in the Project Table, and run the job locally.

2. Click Run.

This job takes a few minutes or less. When the job finishes, the structure in the Project Table is replaced. Use the horizontal scroll bar to scroll to the values of R and Rfree.

You might want to enlarge the Project Table panel (by dragging the panel border). You might also want to adjust the column widths, which you can do by choosing Fit to Data from the Columns button menu on the Project Table toolbar (or use Properties → Columns → Fit to Data).

3. Close the R-Factor Calculation panel.
