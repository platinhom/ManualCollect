Deleting the Loop from Residues 30 to 38

Finally, the loop spanned by residues 30–38 should be deleted. The electron density is not yet accurate enough to place these residues with Loop Refinement. Before you delete the residues, the entry will be duplicated so that the results of the minimization are preserved.

1. Ensure that the new loop entry with the lowest Ref Score is selected, and type CTRL+D (⌘D).

The entry is duplicated, selected, and included in the Workspace.

2. Change the title for the duplicated entry to indicate its origin.
3. From the Delete button menu on the Edit toolbar, choose Select.

The Atom Selection dialog box opens.

4. In the Residue tab, choose Residue number from the list.
5. Enter 30-38 in the Residue number text box.
6. Click Add, then click OK.

The Atom Selection dialog box closes, and the residues are deleted from the structure.
