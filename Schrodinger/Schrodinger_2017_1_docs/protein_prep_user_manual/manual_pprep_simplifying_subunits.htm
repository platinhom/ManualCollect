Removing Unwanted Subunits from a Protein Complex

If the protein complex structure is a multimer with duplicate binding sites, it can be truncated by deleting all but a single ligand binding site and the associated receptor subunits.

  1. Choose Molecules from the Set selection scope button on the toolbar:

  2. Pick an atom from the desired ligand in the Workspace.

    The ligand is selected.

  3. Click the Define button on the Select toolbar.

    The Atom Selection dialog box opens.

  4. Click Selection.

  5. Click Proximity. In the Proximity dialog box:

    1. Select Within, enter a distance in the text box, and select Angstroms.

    2. Under Fill, select Molecules and deselect Exclude source if it is selected.

      This selects the ligand and any molecules that are within the specified distance. As the protein chain is a single molecule, it is selected along with any waters or cofactors.

    3. Click OK to exit the Proximity dialog box.

  6. Click Invert.

    The chains and other molecules that are not near the ligand are selected.

  7. Click OK

  8. Press the Del key.

    The selected chains and other molecules are deleted, leaving only the desired ligand, protein chains, and nearby molecules.
