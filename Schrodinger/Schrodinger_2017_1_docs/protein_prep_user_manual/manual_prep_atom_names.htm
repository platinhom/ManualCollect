Correcting PDB Atom Names

Prime requires that the PDB atom names of all atoms are correct. In the imported structure, residues that have incorrect PDB atom names are colored red or blue. You can select these residues using the procedure given in Selecting Residues by PDB Conversion Status.

To correct PDB atom names:

  1. Choose PDB Atom Namefrom the Apply Labelsmenu in the Style toolbox.

  2. Choose Change Other Properties from the Other Edits menu in the 3D Builder panel.

  3. Choose PDB Atom Name from the Property option menu.

  4. Ensure that Apply PDB atom name is selected.

  5. For each incorrectly-named atom, pick the atom in the Workspace, enter the correct PDB atom name in the PDB atom name text box, then click Apply.

    You may need to zoom in to pick the atom, which you can do with the mouse wheel.
