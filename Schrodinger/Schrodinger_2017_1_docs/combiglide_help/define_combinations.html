CombiGlide — Define Combinations Step

In this step you select a molecule that is used as a "core" structure and choose the reagents that are used to vary the side chains of the core structure.

Using the Define Combinations Step

To perform combinatorial screening, you must have a core molecule whose side chains, or attachments, you want to replace, and a set of reagent files that define the replacements. For information on preparing reagent files, see the Reagent Preparation Panel topic. The core structure must be a 3D, all-atom, properly minimized structure.

When you first enter the Define Combinations step, Pick molecule is selected in the Core structure section, and most other controls are unavailable. After you have defined a core structure, the other controls become available.

Once you have defined a core structure and its attachments, click Next to continue to the next step.

To define the core structure:

  1. Ensure that the structure is displayed in the Workspace.

  2. Click on an atom in the core structure.

After you pick the core structure, Pick molecule is deselected, and the Pick option is selected in the Pick to define an attachment section. The core molecule is now defined, and you can define the positions of the attachments.

To define an attachment:

  1. Pick two atoms in the core molecule that define the position of an attachment, the atom that is kept first, then the atom that is deleted.

    A turquoise arrow is displayed over the bond, and the Select Reagent File dialog box opens.

  2. Select the appropriate reagent file, and click OK

    The reagent files have a .bld extension: they are Maestro-format files that have special information for the Maestro builder, which does the library enumeration. The dialog box closes, and the attachment is listed in the Attachments table. The size of the library with the current set of attachments is given in the lower portion of the panel. The arrowhead changes to a sphere and the color changes to gold.

If you choose the wrong location for an attachment, you can delete it by selecting it in the table and clicking Delete Attachment. If you simply attached the wrong file, you can select the attachment in the table, and click the Browse button below the table to select the correct file.

For each attachment a minimal capping group is automatically defined. This minimal capping group is attached to the core at its attachment point in place of what is in the actual core structure when the core structure is docked. You can view the minimally capped core by selecting Minimally capped core from the View as options. To return to the original core structure, select User-capped core.

Define Combinations Step Features

  • Core structure section
  • Library definition section

Core structure section

This section provides controls for selecting and viewing the core molecule.

  • Pick molecule option
  • Display Core Structure button
  • View as options

Pick molecule option

When selected, this option allows you to pick a molecule in the Workspace for the core structure. The molecule you pick should be a 3D, minimized structure. Picking a core molecule deletes any attachment data. This option is selected by default when you open the panel and no core molecule has been picked. After picking, the Workspace is cleared and the picked molecule is displayed. This molecule is copied into the CombiGlide run. Any changes you make to the structure of the molecule are not saved in the run. If you make changes to the structure, you must pick the molecule again to make it the core structure.

Display Core Structure button

Redisplays the core structure in the Workspace.

View as options

These options control the display of the core structure. The options are Minimally capped core and Original core, which have the obvious actions.

Library definition section

This section provides controls for the definition of attachment points and selection of reagent files that will be used to enumerate the library.

  • Attachments table
  • Pick to define an attachment controls
  • Reagent file buttons
  • Delete Attachment button
  • Import Definition button
  • Export Definition button

Attachments table

Lists the attachments that have been defined. The table columns are defined below. You can select multiple rows in the table. Right-clicking in a table row displays a menu with four items: Select Reagent File, Clear Reagent File, Reagent File Statistics, and Delete Attachment. The actions are applied to all selected rows. The Reagent File Statistics item displays statistics on the selected reagent files. The other items perform the same actions as the Browse, Clear, and Delete Attachment buttons.

Name   Label for the position of the attachment. Default is 1, 2, 3, etc. Editable.
Reagent File   Name of the file containing the reagents, minus the .bld extension. The tooltip for this table cell displays the full path to the file.
Functional Group   Name of the functional group that defines the reagents.
Reagents   Number of reagents in the file
Pick to define an attachment controls

Pick option and menu: The option menu has two choices: Atoms and Bonds. When the Pick option is selected, you can pick either one bond or two atoms to define an attachment position, depending on the menu choice. If you pick a bond, you should pick the half that is furthest from the core. After picking, the Select Reagent File file selector opens, and you can browse to the desired reagent file. After selecting the file, the attachment is listed in the Attachments table.

Markers option: When selected, the grow bond for each attachment is displayed in the Workspace. The bond is a turquoise arrow before the reagent file is selected, and changes to gold with a spherical head after the reagent file is selected. The marker for the attachment selected in the Attachments table is turquoise; markers for unselected attachments are gold.

Reagent file buttons

Browse button: Opens the Select Reagent File file selector, in which you can browse to a reagent file. Properly prepared reagent files have the suffix .bld.

Clear button: Clears the table row of its reagent file and associated data. The reagent file is no longer associated with the attachment, and you must click Browse to select a new reagent file for the attachment.

Delete Attachment button

Deletes the attachments that are selected in the Attachments table.

Import Definition button

Import a previously-defined library definition, including the core structure and the attachments. Opens a file browser with a specified filter so that only valid library definitions are displayed by default.

Export definition button

Export the current library definition to a Maestro file, including the core structure and the attachments. Opens a file browser. The standard suffix is automatically added to the file name when the library is exported.
