Features

3D Minimization Dialog Box

In this dialog box you can run an energy minimization on a set of structures to determine their 3D geometry. The minimization is done with a force field, using LigPrep. If the stereochemistry of chiral centers is not specified, LigPrep chooses a stereochemistry that is consistent with a database of natural products.

You must have a LigPrep license to run this application.

3D Minimization Dialog Box Features

  • Data set option menu
  • Save results as text box
  • Force field option menu
  • Incorporation of minimized structures option menu
  • Append _min to each minimized structure name option
  • Export as option, menu and text box
  • Minimize button
Data set option menuSelect the source of the data. The data source can involve both row and column selection. The choices are: Selected rows—Use the rows that are selected in the master view. You must select the rows before opening the dialog box. Visible rows—Use the rows that are displayed (visible) in the master view. You must ensure that the required rows are visible before opening the dialog box. Saved view—Use the rows and columns in a chosen saved view. When you select this item, an option menu that lists the saved views in the project is displayed to the right. You can choose the desired view from this option menu. A second option menu, labeled Domain, allows you to choose whether to restrict the rows or the columns, or both, to those of the saved view. Otherwise the visible rows or columns of the Master View are used. Original data set for jobname—Use the rows and columns that were used for the specified job. This item is only available when you are cloning a job or viewing the results of a job.
Save results as text box

Assign a name to the results. This name is listed under 3D Minimization in the Project View window.

Force field option menu

Choose the force field for the minimization from this option menu. The choices are MMFFs and OPLS2005.

Incorporation of minimized structures option menu

Choose whether the minimized structures overwrite the existing structures (Overwrite) or are appended to the project as new structures (Append).

Append "_min" to each minimized structure name option

When the minimized structures are appended to the project, add _min to the names of the structures, to distinguish them from the source structures.

Export as option, menu and text box

Export the minimized structures to a Maestro or SD file. Choose the file type from the option menu. Enter the file name in the text box, or click Browse and navigate to the location and name the file. The extension you give to the file name determines whether it is compressed or not. If no extension is given, the files are compressed, with extension .maegz for Maestro format and .sdf.gz for SD format.

Minimize button

Run the minimization. Opens the Start Application dialog box, in which you can choose a host to run the minimization, and specify the number of processors for distributing the job.
