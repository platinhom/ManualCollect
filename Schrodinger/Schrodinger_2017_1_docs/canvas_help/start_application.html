Features

Start Application Dialog Box

This dialog box is used to control where an application is run and how many CPUs it uses. The job is run under the Job Control facility. For information on this facility, see the Job Control Guide.

Start Application Dialog Box Features

  • Host option menu
  • CPUs box
Host option menu

Choose a host on which to run the job from this option menu. The list of hosts is read from the schrodinger.hosts file, and can include queues on a host as well as actual hosts. For information on setting up the list of hosts and configuring access to remote hosts, see Preparing for Remote Job Submission.

CPUs box

Set the number of CPUs in this box, either by typing in the number or using the arrow buttons. The value in this box can only be set if the application can distribute its work over multiple CPUs. You should check that the host you choose has the required number of CPUs. If it doesn't, then some or all of the subjobs will run serially rather than in parallel.
