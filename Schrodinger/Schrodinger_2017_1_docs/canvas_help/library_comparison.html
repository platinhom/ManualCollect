Library Comparison Dialog Box

Set up and run a job to find nearest-neighbor similarities between two collections of compounds or within one collection.

Using the Library Comparison Dialog Box

The library comparison works by finding the nearest neighbor in the reference library for each compound in the query library, using fingerprint similarity based on a chosen metric. The nearest-neighbor similarities are returned in a column named MaxSim.

The results of the library comparison can be used to create a histogram from the nearest neighbor similarities, to assess the similarity of the two libraries (if comparing two libraries), or the diversity of a single library (if comparing to itself).

Save results as text box

Assign a name to the results. This name is listed under Library Comparison in the Project View window.

Reference library option menu

Choose the option for the rows to use for the reference library. If you want to use the selected rows, you must select them before opening the dialog box.

Compare to option menu

Choose the source of library that you want to compare to the reference library. The choices are Saved view, which uses a saved view to define the library, and Self, which compares the reference library to itself.

View name option menu

Choose the view that defines the library that you want to compare to the reference library.

Fingerprint column option menu

Choose the fingerprint column to use to calculate the nearest neighbor distances. You must generate fingerprints prior to the calculation, and incorporate them into the project. The available fingerprints are those that are listed under Binary Fingerprints in the Project View window and marked as incorporated.

Similarity metric option menu

Choose the metric used to calculate the similarity matrix.

Formula Key button

Open a panel that displays the definitions of the terms in the Description text box.

Description button

Noneditable text box that displays the definition of the metric.

Tversky alpha and beta text boxes

Specify the α and β parameters for the Tversky metric.

Compare Libraries button

Click this button to run the job that compares the libraries.
