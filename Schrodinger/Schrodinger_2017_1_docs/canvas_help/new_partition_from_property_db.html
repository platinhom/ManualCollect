New Partition from Property Dialog Box

Create a partition by assigning classes from the values of a property. You can split a property into classes of widths that you determine, with multiple property values included in each class, or you can create classes for each property value.

New Partition from Property Dialog Box Features

  • Property option menu
  • Name text box
  • Description text box
  • Split property into bins option and Bins button
  • Update button
  • Class values as options
  • Reset button
  • Classes table
Property option menu

Choose the property that you want to use for the partition from this option menu.

Name text box

Enter the name for the partition in this text box. A default name is assigned, based on the property name, which can be edited.

Description text box

Enter the description for the partition in this text box. The description can include any information — for example, if you split the property into ranges, you might want to store the minimum, maximum, and width values for reference.

Split property into bins option and Bins button

Create classes that cover a range of property values. The widths of the classes are determined by clicking Bins and setting the bin widths in the New Partition from Property - Bins dialog box. The class includes the lower boundary of the range.

Min and Max text boxes

Specify the minimum and maximum property values to use when dividing the property into ranges. The range between the minimum and maximum is split into classes of equal width. Property values that are less than the minimum value are placed in the first class, and property values that are greater than the maximum value are placed in the last class. The difference between the minimum and maximum values must be a multiple of the width.

Update button

Click this button to assign the rows to classes after making changes.

Class values as options

Choose an option for the representation of the class value.

  • Integers—Use the bin index for the class value.
  • Ranges—Represent the class value as a string m–n that indicates the range of property values in the class.
  • Midpoints—Represent the class value by the midpoint of the range.
Reset button

When class values have been edited, click this button to restore them to the default values, as defined by the Class values as option selected.

Classes table

This table has three columns:

  • Value—Value for the class. You can edit this column to change the class values.
  • Count—Number of members in the class. This column is noneditable.
  • Color—Color used to color the table cells for members of the class. You can change the color for the class by right-clicking in a cell and choosing Change Color. A color selector opens, in which you can choose the color you want to use.
