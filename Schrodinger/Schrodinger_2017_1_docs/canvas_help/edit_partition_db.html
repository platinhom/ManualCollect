Edit Partition Dialog Box

In this dialog box you can change the name and description of a partition, and edit the class values and the colors used for each class.

Edit Partition Dialog Box Features

  • Name text box
  • Description text box
  • Classes table
Name text box

Name of the partition. You can change the name of the partition by editing the text in this text box.

Description text box

Description of the partition. You can change the description by editing the text in this text box.

Classes table

This table has three columns:

  • Value—Value for the class. For values assigned from splitting a property or assigned randomly, these values are a sequence of integers by default. You can change the class value by editing the table cell.
  • Count—Number of members in the class. This column is noneditable.
  • Color—Color used to color the table cells for members of the class. You can change the color for the class by right-clicking in a cell and choosing Change Color. A color selector opens, in which you can choose the color you want to use.
