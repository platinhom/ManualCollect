Filter all rows in the current view by conditions on the values of properties.

Use existing filter option menu

Choose an existing filter to apply to the data. Filters can be created in this dialog box using the Save button.

Import button

Import a previously exported property filter. Opens a file selector, in which you can navigate to and choose the filter file (.flt)

Return non-matches option

Apply the inverse of the filter: return the rows that do not match rather than the rows that match

Store results in options

Choose the view in which to store the results of the sort. If you choose Current View, the current view is updated with the new ordering of the rows. If you choose choose New View, the Save Custom View dialog box opens after the sort is done, and you can name, describe, and save the sort results as a new view.

Show rows where controls

Set up a condition for the filter using the option menus and the text box. The first option menu lists properties, the second option menu lists relational operators, and the text box allows you to enter a value or a string, depending on the relational operator. The operators include arithemtic relations, string matching, and counts or percentages relative to the highest or lowest end of the property range.

And and Or options

These options allow you to select the Boolean operator that relates the current set of conditions in the Filter String text box with the newly defined condition when you click Add.

Add button

Adds the condition defined with the Show rows where controls to the set of conditions in the Filter String text box, connecting them with the selected Boolean operator.

Undo button

Undo the last change made to the filter string.

Clear button

Clear the Filter String text area.

Filter String text area

Displays the current filter, which is a set of conditions connected by Boolean operators. The string is editable.

Save button

Save the filter for later use. Opens a dialog box in which you can name the filter. The filter is then listed on the Use existing filter option menu.

Export button

Export the filter to a file. Opens a file selector, in which you can navigate to a location and name the filter file (.flt).

Run button

Run the filtering job. When the job finishes, the view is updated to display only the rows that pass the filter.
