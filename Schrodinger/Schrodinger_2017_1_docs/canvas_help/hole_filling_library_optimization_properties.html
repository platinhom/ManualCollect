Hole-Filling and Library Optimization - Property Filter Dialog Box

In this dialog box you can select the properties to optimize and set the desired range of property values.

Using the Hole-Filling and Library Optimization - Property Filter Dialog Box

To specify the properties to optimize, select them in the Available properties list and click Add to add them to the Selected properties table. You can remove properties from the Selected properties table by selecting them and clicking Remove.

To set the desired property ranges, you can edit the cells in the Min and Max columns. If you want to return to the original value, select the cell and click Reset.

To save a set of properties with their desired ranges as a filter, click Export Filter, then navigate to a location and name the file in the file selector that opens. You can then retrieve this filter in another job with the Import button. If you want to use a filter from another run, you can choose the run from the Use existing filter option menu.

Hole-Filling and Library Optimization - Property Filter Dialog Box Features

  • Use existing filter option menu
  • Import button
  • Available properties list
  • Search text box
  • Add and Remove buttons
  • Reset button
  • Selected properties table
  • Export Filter button
Use existing filter option menu

Choose a filter from another hole-filling job from this option menu.

Import button

Import a property filter (.hflt) from a file. Opens a file selector, in which you can navigate to and select the filter file.

Available properties list

List of properties that are available for selection. You can select multiple properties with shift-click and control-click. When properties are transferred to the Selected properties table (by clicking Add), they are removed from the list.

Search text box

Use this text box to filter the Available properties list. When you type text in this box, the list of properties is restricted to those properties whose names contain the text. To return to the full list, clear the text box.

Add and Remove buttons

Use these buttons to add properties to or remove properties from the Selected properties table. To add properties, select them in the Available properties list and click Add. To remove properties, select them in the Selected properties table and click Remove.

Reset button

Reset the value of the selected cell in the Selected properties table to its input value, which is the minimum or maximum value in the set of input structures.

Selected properties table

List of properties that have been selected for optimization in the hole-filling and library optimization procedure. You can select rows in the table to remove from the list, by clicking Remove. You can edit the Min and Max table cells to change the minimum and maximum allowed values of the properties. You can reset the value in a table cell to its input value by selecting it and clicking Reset.

Export filter button

Export a property filter to a file (.hflt). Opens a file selector, in which you can navigate to a location and name the filter file.
