Features

Details Dialog Box

This dialog box provides information on the number of rows and columns in the view, and the times it was created, modified, and saved. It also allows you to change the description of the view.

Details Dialog Box Features

  • Name text box
  • Description text area
  • Number of rows text box
  • Number of columns text box
  • Created text box
  • Modified text box
  • Modifications saved text box
Name text box

Name of the view. Not editable.

Description text area

Description of the view. You can change the text in this text area, and click Apply or OK to apply the change.

Number of rows text box

Number of rows in the view. Not editable.

Number of columns text box

Number of columns in the view. Not editable.

Created text box

Date and time when the view was created. Not editable.

Modified text box

Date and time when the view was last modified. If this time is later than the Modifications saved time, then there are unsaved modifications in the view. You can save the modifications by choosing View > Save in the view window. Not editable.

Modifications saved text box

Date and time when the modifications were last saved. Not editable.
