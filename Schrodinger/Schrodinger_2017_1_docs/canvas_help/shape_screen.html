Shape Screen Dialog Box

Set up and run a job to search for molecules by the similarity of their shapes with the shape of a query molecule. The results are returned as a property, Shape Sim.

This application requires a Phase license.

Shape Screen Dialog Box Features

  • Data set option menu
  • Save results as text box
  • Use shape query from option menu
  • Shape query file text box
  • Generate conformers option
  • Advanced button
  • Incorporation of aligned structures option menu
  • Append _aligned to each structure name option
Data set option menuSelect the source of the data. The data source can involve both row and column selection. The choices are: Selected rows—Use the rows that are selected in the master view. You must select the rows before opening the dialog box. Visible rows—Use the rows that are displayed (visible) in the master view. You must ensure that the required rows are visible before opening the dialog box. Saved view—Use the rows and columns in a chosen saved view. When you select this item, an option menu that lists the saved views in the project is displayed to the right. You can choose the desired view from this option menu. A second option menu, labeled Domain, allows you to choose whether to restrict the rows or the columns, or both, to those of the saved view. Otherwise the visible rows or columns of the Master View are used. Original data set for jobname—Use the rows and columns that were used for the specified job. This item is only available when you are cloning a job or viewing the results of a job.
Save results as text box

Assign a name to the results. This name is listed under Shape Screen in the Project View window.

Use shape query from option menu

Choose the source of the shape query.

  • PyMOL viewer—Take the structure from the active PyMOL session.
  • External file—Read the query structure from a file.

The query must be a single molecule: if it is not, a warning is posted.

Shape query file text box

If you chose External file for the source of the shape query, enter the file name in the text box, or click Browse and navigate to the file. You can read Maestro files (.mae) or SD files (.sdf).

Generate conformers option

Run a conformational search for the molecules being screened, using ConfGen.

Advanced button

Open the Shape Screen - Advanced Options dialog box to specify more parameters for the shape search and conformer generation.

Incorporation of aligned structures option menu

Choose how the structures that are aligned to the query are incorporated into the project: overwrite the existing structures, or append the structures to the project as new structures.

Append "_aligned" to each structure name option

Identify the aligned structures by appending the suffix _append to the structure names.
