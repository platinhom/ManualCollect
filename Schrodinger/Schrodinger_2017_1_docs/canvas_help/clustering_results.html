Clustering Results Panel

This panel displays clustering results for K-means and leader-follower clustering in a limited custom view, in which you can display structures for each cluster.

Clustering Results Panel Features

  • Menu bar
  • View cluster box
  • Spreadsheet
Menu bar

The menu bar has a reduced set of menus and items from the custom views menu bar:

  • File—Export results to a file or to Maestro.
  • View—Save the view with a name, apply it to the Master View, or manage the properties that are displayed in the view.
  • Structure—Show or hide the 2D structure and apply scaling to the structure.
View cluster box

Select the cluster to view, by entering the cluster index in the box, or use the arrows to step through clusters.

Spreadsheet

Shows the structures in the selected cluster, along with any properties that you choose to show. Structures can be selected as in any view.
