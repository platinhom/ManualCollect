Data set option menuSelect the source of the data. The data source can involve both row and column selection. The choices are: Selected rows—Use the rows that are selected in the master view. You must select the rows before opening the dialog box. Visible rows—Use the rows that are displayed (visible) in the master view. You must ensure that the required rows are visible before opening the dialog box. Saved view—Use the rows and columns in a chosen saved view. When you select this item, an option menu that lists the saved views in the project is displayed to the right. You can choose the desired view from this option menu. A second option menu, labeled Domain, allows you to choose whether to restrict the rows or the columns, or both, to those of the saved view. Otherwise the visible rows or columns of the Master View are used. Original data set for jobname—Use the rows and columns that were used for the specified job. This item is only available when you are cloning a job or viewing the results of a job.
Save results as text box

Assign a name to the results. This name is listed in the Project View window under SOM for numeric variables and under SOMBits for fingerprints.

Map generation options

Select the option for the action to perform.

  • Build new map—Build a new map.
  • Apply saved map—Apply the map selected from the option menu. Only maps that match the X variables option are shown.
  • Apply imported map—Import and apply a map. Click Import and navigate to the map file.

X variables option menu

Choose the type of input X variables you want to use: From properties or From fingerprints. The tools for the chosen type of input are displayed below this option menu.

Property selection tools

Use these property selection tools to select the properties to use for the X variables.

When applying a map, lists the source properties used to generate the map.

Fingerprint column option menu

Choose the fingerprint column to use for the X variables from this option menu.

Lattice options section

Set the options for the lattice. Not available when Apply saved map or Apply imported map is selected.

Dimensions text boxes

Specify the height and the width of the lattice (number of nodes in each direction).

Topology options

Select the option for the lattice topology, from Rectangular or Hexagonal.

Use periodic boundaries option

Select this option to make the lattice topology periodic. This eliminates edge effects when updating the neighbors of the winning node.

Training options section

Set options for the training of the model. Not available when Apply saved map or Apply imported map is selected.

Maximum number of cycles box

Specify the maximum number of training cycles to use.

Stop if RMSE increases option

Stop the training if the RMSE increases. Incompatible with the threshold set by Stop when RMSE reaches.

Stop when RMSE reaches option

Stop the training when the RMSE falls below the specified value. Incompatible with Stop if RMSE increases.

Decay options

Specify the type of decay used to decrease the gain term as the training proceeds.

Random seed text box

Specify the random seed for initialization of the map.

Create Map button

Run the job to create the map. Opens the Start Application dialog box, in which you can choose a host for the job.

Apply Map button

Apply the map to the data set. Opens the Self-Organizing Map Viewer panel. Not available when Build new map is selected.

View Map button

Display the map on a grid, with cells colored by distance, population, or property. Opens the Self-Organizing Map Viewer panel. Not available when Build new map is selected.

Export Map button

Export the map to a file. Opens a file chooser, in which you can navigate to a location and name the file. Not available when Build new map is selected.
