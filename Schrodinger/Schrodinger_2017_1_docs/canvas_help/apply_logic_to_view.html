Apply Logic to View Dialog Box

In this dialog box, you can perform a logical AND, OR, NOT, or XOR operation on the current view and another view, and update the current view with the results, or create a new view.

To open the Apply Logic to View dialog box, choose View > Logic in the view window.

The logical operations can be applied to either the rows or the columns. To apply operations only to the rows, select Retain columns in current view. To apply operations only to the columns, select Retain rows in current view.

To keep only the rows (or columns) that are common to both views:

  1. Select And.
  2. Choose the other view from the option menu.
  3. Click Apply or OK.

To keep only the rows (or columns) that are in either view:

  1. Select Or.
  2. Choose the other view from the option menu.
  3. Click Apply or OK.

To keep only the rows (or columns) that are in either view but not in both:

  1. Select Exclusive Or.
  2. Choose the other view from the option menu.
  3. Click Apply or OK.

To keep only the rows (or columns) that are not in the other view:

  1. Select Not.
  2. Choose the other view from the option menu.
  3. Click Apply or OK.

If you want to create a new view, select New view. When you click Apply or OK, the Save Custom View dialog box opens, in which you can name, describe, and save the view.

If you want to store the results in an existing view, select Other view and choose the view from the option menu. When you click Apply or OK, the view that you chose is overwritten with the results of the logical operation.

To apply multiple logical operations, click Apply when you have made a selection, then make a new selection and click Apply again. You can repeat this process as many times as you want.
