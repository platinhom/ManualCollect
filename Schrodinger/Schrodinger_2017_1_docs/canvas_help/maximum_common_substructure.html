Maximum Common Substructure Dialog Box

Determine the maximum common substructure for a number of structures within a set between a minimum and maximum number. The results include the substructure and the list of molecules that match the substructure.

Maximum Common Substructure Dialog Box Features

  • Data set option menu
  • Save results as text box
  • Minimum number of compounds to match box
  • Maximum number of compounds to match box
  • Enforce minimum substructure size option
  • Restrict compound membership to at most a single MCS group option
  • Consider only the largest fragment from each structure option
  • Add ring designation to all SMARTS outputs option
  • Add hydrogen count to all SMARTS outputs option
  • Atom/bond typing list
  • Import Custom Types button
  • Imported custom types file text box
  • Compute button
Data set option menuSelect the source of the data. The data source can involve both row and column selection. The choices are: Selected rows—Use the rows that are selected in the master view. You must select the rows before opening the dialog box. Visible rows—Use the rows that are displayed (visible) in the master view. You must ensure that the required rows are visible before opening the dialog box. Saved view—Use the rows and columns in a chosen saved view. When you select this item, an option menu that lists the saved views in the project is displayed to the right. You can choose the desired view from this option menu. A second option menu, labeled Domain, allows you to choose whether to restrict the rows or the columns, or both, to those of the saved view. Otherwise the visible rows or columns of the Master View are used. Original data set for jobname—Use the rows and columns that were used for the specified job. This item is only available when you are cloning a job or viewing the results of a job.
Save results as text box

Assign a name to the results. This name is listed under MCS in the Project View window.

Minimum number of compounds to match box

Minimum number of structures that must match the maximum common substructure. If this number exceeds the number of input molecules it is interpreted as requiring all to match.

Maximum number of compounds to match box

Maximum number of molecules that must match the maximum common substructure. When the maximum is greater than the minimum, a series of common substructures spanning this range is produced, with at least one substructure for each number of molecules in the range.

Enforce minimum substructure size option

Select this option to stop processing when the count of atoms and bonds in the maximum common substructure is less than the specified value. When you select this option, a box is displayed in which you can set the value.

Restrict compound membership to at most a single MCS group option

If a structure has more than one maximum common substructure, add it to the set for the largest number of structures and the largest substructure.

Consider only the largest fragment from each structure option

If the structure contains more than one molecule or fragment, ignore all but the largest fragment when analyzing the structure.

Add ring designation to all SMARTS outputs option

In the output MCS SMARTS patterns, mark each atom as cyclic (R) or acyclic (R0). By default this designation is not added.

Add hydrogen count to all SMARTS outputs option

In the output MCS SMARTS patterns, include hydrogen counts for each atom. These counts are not included by default.

Atom/bond typing list

Select the atom and bond typing scheme to be used to classify atoms and bonds in the structures. If you choose Custom, click Import Custom Types to locate the file that contains the custom atom and bond types. The file must contain SMARTS patterns for these types, one per line.

Import Custom Types button

Import a custom set of atom and bond types. Opens a file chooser, in which you can navigate to and select the file.

Imported custom types file text box

Displays the name of the custom atom/bond types file that was imported. You can also enter the file name in this text box to open it.

Compute button

Run the job to generate the maximum common substructures.
