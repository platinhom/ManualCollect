Distance Matrix from Properties Dialog Box

Calculate a distance matrix for selected structures from selected properties using a chosen metric.

Distance Matrix from Properties Dialog Box Features

  • Data set option menu
  • Options tab
  • Results tab
  • Compute button
  • Export button
Data set option menuSelect the source of the data. The data source can involve both row and column selection. The choices are: Selected rows—Use the rows that are selected in the master view. You must select the rows before opening the dialog box. Visible rows—Use the rows that are displayed (visible) in the master view. You must ensure that the required rows are visible before opening the dialog box. Saved view—Use the rows and columns in a chosen saved view. When you select this item, an option menu that lists the saved views in the project is displayed to the right. You can choose the desired view from this option menu. A second option menu, labeled Domain, allows you to choose whether to restrict the rows or the columns, or both, to those of the saved view. Otherwise the visible rows or columns of the Master View are used. Original data set for jobname—Use the rows and columns that were used for the specified job. This item is only available when you are cloning a job or viewing the results of a job.
Options tab

In this tab, you can set up the input and options for the calculation.

  • Source properties section
  • Distance metric option menu
  • p-norm power box
Source properties section

Select the properties to be used for the distance calculation using the property selection tools.

Distance metric option menu

Choose the distance metric from this option menu. The two available metrics are:

  • p-norm— Choose this option to use a p-norm to calculate the distances. The p-norm is defined by (Σxp)1/p. When this option is chosen, the p-norm power box is displayed.

  • Mahalanobis— Generate a Mahalanobis distance matrix using the covariance matrix to correct non-orthogonality of the data. For background information, see http://en.wikipedia.org/wiki/Mahalanobis_distance

p-norm power box

Specify the power p used in the p-norm. Only present if you choose p-norm from the Distance metric option menu

Results tab

This tab displays the distance matrix as a table, labeled with the structure names.

  • Structure tooltips option
  • Color by value option
Structure tooltips option

Show the 2D structures for a table cell in a tool tip when the pointer pauses over the table cell.

Color by value option

Color the results table cells by value, using a linear interpolation between red (highest) and blue (lowest) in RGB color space.

Compute button

Calculate the matrix. A dialog box opens, in which you can choose to display the matrix as a table in the results area or export the matrix directly to a CSV file. If you choose to export to file, click Browse to navigate to the location and name the file, or enter the file name in the Matrix file text box.

Export button

Export the distance matrix to a CSV file. Opens a file selector, in which you can navigate to a location and name the file.
