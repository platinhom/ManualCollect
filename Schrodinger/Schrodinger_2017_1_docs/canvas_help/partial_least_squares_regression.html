Partial Least-Squares Regression Dialog Box

Build a multiple linear regression model based on variables derived from a partial least squares (PLS) analysis of a selection of properties for a data set of structures, and apply the model to other data sets.

This method is useful for large numbers of X variables, with both underdetermined and overdetermined systems. PLS is not available with fingerprints as X variables (due to legal reasons). PLS "factors" are constructed as linear combinations of the input X variables by correlating them with the Y variable. If poor test set results are obtained, an alternative is Principal Components Regression.

Data set option menuSelect the source of the data. The data source can involve both row and column selection. The choices are: Selected rows—Use the rows that are selected in the master view. You must select the rows before opening the dialog box. Visible rows—Use the rows that are displayed (visible) in the master view. You must ensure that the required rows are visible before opening the dialog box. Saved view—Use the rows and columns in a chosen saved view. When you select this item, an option menu that lists the saved views in the project is displayed to the right. You can choose the desired view from this option menu. A second option menu, labeled Domain, allows you to choose whether to restrict the rows or the columns, or both, to those of the saved view. Otherwise the visible rows or columns of the Master View are used. Original data set for jobname—Use the rows and columns that were used for the specified job. This item is only available when you are cloning a job or viewing the results of a job.
Save results as text box

Assign a name to the results. This name is listed under PLS in the Project View window.

Model optionsSelect the option for the action to perform.Build new model—Build a new regression model.Apply saved model—Apply the model selected from the option menu.Apply imported model—Import and apply a regression model. Click Import and navigate to the regression model file.
Y variable option menu and search box

Select the Y variable to use in the partial least-squares regression. You can type the name in the search box to restrict the menu to the matching properties. The values of the Y variable are displayed in the data set structures table.

X variables property selection tools

Select the properties to use as the X variables in the multiple linear regresssion, using the property selection tools.

Maximum number of PLS factors box

Specify the maximum number of PLS factors to use in the regression model. Regression models are built for increasing numbers of PLS factors up to this number. It is rarely useful to build models with more than a few PLS factors, as such models tend to be overfit.

Stop adding PLS factors when SD drops to N option and text box

Select this option to stop adding PLS factors when the standard deviation of the regression drops below the value specified in the text box. Using this option could result in fewer PLS factors than the number specified in the Maximum number of PLS factors box.

Autoscale X variables option

Scale the X variables by dividing the values of each property by the standard deviation in the value of the property.

Eliminate X variables with [t-value] < N option and text box

Eliminate X variables whose t-value is less than the value given in the text box. The t-value is the ratio of the coefficient of the variable in the fitted model to the standard error of the model. Small t values indicate that the variable is not contributing significantly to the model.

Training Set/Results tab

This tab is initially labeled Training Set, and is relabeled Results after a model is built. It contains controls for selecting the training set and test set, and displays the results.

  • Training set section
  • Data set structure table
  • Training tab
  • Test tab
  • Factor loadings tab
Training set section

Assign the training set and the test set using an automated procedure, using the controls in this section, described below.

Assign randomly option and box

Select this option to assign the training set randomly, and specify the percentage of the rows to include in the training set. The rest are assigned to the test set.

Seed text box

Specify the seed for generating random numbers, used to assign the training set.

Generate Seed button

Click this button to generate a seed. The seed is displayed in the Seed text box.

Assign from partition option and menu

Select this option to assign the training set and the test set from a partition. One class value can be chosen for each set. The rows in the remaining classes are not assigned to any set.

Training set value option menu

Choose the partition class value from this option menu to assign the members of the class to the training set. The percentage of the total number of rows in this class is shown in the text box to the right.

Test set value option menu

Choose the partition class value from this option menu to assign the members of the class to the test set. The percentage of the number of rows in this class is shown in the text box to the right.

Apply button

Apply the choices in this section to generate the training set and the test set.

Dataset structure table

Table of structures in the dataset. The Structure column lists the title of each structure. Pausing the pointer in this column displays the 2D structure in a tooltip. The Model Set column indicates whether a structure is in the training set (training), the test set (test), or neither (none). When you select the Y variable, its values are added to the table.

When Build new model is selected, you can assign the training and test sets manually by changing the value in the Model Set column. If you double-click in a cell, an option menu is displayed, from which you can choose the set. To assign the set for multiple rows, select the rows, then right-click in a cell and choose the set from the Change Selected submenu of the shortcut menu.

When a model is applied, a column is added to the table for the predicted property value and for the error in the predicted value.

To export the table data, select the rows you want to export, then right-click and choose Export to File or Export to Spreadsheet. For export to a file, data in all columns for the selected rows are written to a CSV file with column headings. Exporting to the spreadsheet adds the predicted values to the spreadsheet with a prefix that you can change.

Training tab

Displays statistics of the regression for the training set.

Test tab

Displays statistics for the test set.

Factor loadings tab

Lists the loadings for each PLS factor for each of the X variables.

Export Model button

Export the model to a file. Opens a file chooser, in which you can navigate to a location and name the file.

Scatter Plot button

Display a scatter plot of predictions of the Y variable property against the property for the model that is selected in the results tabs.

Plot Loadings button

Display a plot of the factor loadings. Opens a standard scatter plot panel, in which you can choose which factors to plot.

Build Model button

Run the job to build the regression model.

Apply Model button

Apply the selected model to the dataset. Not available when Build new model is selected.
