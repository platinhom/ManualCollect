Find Panel

In this panel you can search for a text string or a regular expression, and select the matches to the search string.

Find what text box

Enter the text to find in this text box (the "search string").

Find button

Click this button to perform the search. All instances of the search string are located with a single click. You can navigate through the matches with the navigation buttons.

Literal string option

Treat the text to find exactly as it appears in the text box.

Allow wildcards option

Allow the use of the wildchard characters *, for any number of any character, and ?, for any single character.

Interpret as regular expression option

Interpret the text to find as a regular expression.

Interpret as floating point value option

Interpret the text to find as a floating point value, which is matched to the precision given in each numeric cell.

Match entire cell contents option

When searching, the entire cell contents must match the search string. By default, if a part of the contents matches the search string, the string is regarded as found.

Match case option

Match the case of the search string.

Within option menu

Define the scope of the search. The options are Spreadsheet, Selected cells, and Column Names. The last of these allows you to locate and select specific columns. The default is to search the entire spreadsheet.

Search option menu

Define the search order: By rows or By columns. The default is to search row by row.

Match navigation buttons

Navigate through the matches. The double-arrow buttons |<< and >>| select the first and last matches; the single-arrow buttons select the next and the previous match.

Select All Matches button

Select all cells that match the search string.
