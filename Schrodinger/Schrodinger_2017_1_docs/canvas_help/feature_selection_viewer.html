Feature Selection Viewer Panel

Using the Feature Selection Viewer Panel

The initial set of properties displayed in the panel is selected from the hierarchical clustering results, with Kelley criteria to choose the default number of clusters.

To determine how well the selected properties (reduced list) represents the orginal properties, multiple linear regression is performed for each property to calculate an R^2 value for the fit of the property by the reduced list. First, a principal components analysis is done on the reduced list and principal components are selected to account for 95% of the variance. The X variables in the PLS model are taken from the projection of the principal components onto the property data from the reduced list, and the Y variable is the original property. These R^2 values are reported in the second column of the Original properties table.

You can change the number of properties in the reduced list in several ways:

  • Select Number of clusters and change the number of clusters, then click Update Properties.

  • Select Maximum correlation and change the value of the maximum allowed correlation between properties on the reduced list, then click Update Properties.

  • Select a property in either of the lists and click Add or Remove, then click Update R^2.

You might want to change the property manually if you have a preference for a particular property. The property chosen from each cluster is the one that is closest to the centroid. You might want to replace it with another property from the same cluster. To do this, it is useful to sort the list of original properties by the clustering order (right-click and choose Sort by Cluster), so that properties in the same cluster are adjacent. It might also be useful to view the correlation matrix (click Correlation Matrix) for the original properties, ordered by cluster, so that you can identify the properties that are in the same cluster as the one you want to replace. With cluster ordering, the heat map should have red diagonal blocks that mark the clusters.

You can also sort the lists, to order the structures alphabetically or by the order produced by clustering, or in ascending or descending magnitudes of R^2. To sort the lists, right click on the column heading and choose the desired item from the shortcut menu.

Selection criterion options

These options allow you to specify the reduced list by different criteria. After you have made a change, you should click Update Properties to update the property lists and the R^2 values.

Number of clusters option and box

Select this option to take the reduced properties from the clusters. Specify the number of clusters in the box. This is also the number of properties in the reduced list. Each property in the reduced list is the property that is closest to the centroid of the cluster. The default number of clusters is chosen by Kelley criteria. The maximum correlation between the reduced properties for the chosen number of clusters is shown in the Maximum correlation text box.

Maximum correlationoption and text box

Select this option to choose the reduced list by the degree of correlation between the properties, and enter a value in the text box for the maximum absolute value of the correlation matrix between the properties. The value in this text box is updated to the nearest value that corresponds to a particular number of clusters. It is also updated when you change the number of clusters.

Update Properties button

Update the reduced list of properties after changing the selection criterion, and recalculate the R^2 values.

Original properties table

This table lists the properties that were selected for reduction in the Feature Selection dialog box. It also shows the R^2 value derived from a MLR fit of each property by the reduced list of properties.

If you chose to remove properties from the list for which a specified percentage of the values were identical, these properties are dimmed in the list, and the cells in the R^2 column are colored yellow and contain text describing the elimination criterion rather than a numeric value. Likewise, if you chose to remove properties that did not meet the Shannon entropy cutoff, the cells are colored magenta and appropriate text is used instead of a value. If you did not choose to remove such properties, the cells in the R^2 column in which all values of the property are identical are colored red and contain the text "No Variance".

Properties on this list that are also on the reduced list are dimmed, and have a line of - characters instead of a numeric value in the R^2 column.

You can select a property in the list and click Add to transfer it to the reduced list. If the property was originally on the reduced list, the cyan coloring is removed. If not, the property is colored cyan in both lists.

You can sort the table columns by right clicking on the heading and choosing a sort option. The R^2 values can be sorted in ascending or descending order. The property names can be sorted in alphabetical order (actually ASCII sort order) or in cluster order (as determined by the hierarchical clustering).

Reduced list table

Reduced list of properties that is intended to be representative of the original list. You can select a property in the list and click Remove to transfer it to the original properties list. If the property was manually added (colored cyan), the coloring is removed when it is transferred back to the original properties list. If it was not, the property is colored cyan in the original list.

Correlation Matrix buttons

There is one button for each list. Clicking the button displays the correlation matrix for the list as a heat map. Opens the Correlation Matrix dialog box.

Add and Remove buttons

Use these buttons to add properties to the reduced list or remove them from the reduced list. The Add button is active only when properties are selected in the Original properties list; likewise the Remove button is active only when properties are selected in the Reduced list.

Update R^2 button

Recalculate the R^2 values after manually adding or removing a property from the reduced list.

Default button

Reset the selection criteria and the reduced list of properties to the defaults.

Variance Plot button

Plot the cumulative variance from a principal components analysis of the reduced variable list and the original variable list.

Filter Master View button

In the master view, keep the properties in the reduced list visible, and hide the remaining properties on the original list. This action has no effect on properties that were not in the list selected for reduction.

Save Custom View button

Save the reduced list of properties as a custom view.
