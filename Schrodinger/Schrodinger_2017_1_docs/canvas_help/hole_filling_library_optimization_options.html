Hole-Filling and Library Optimization - Advanced Options Dialog Box

In this dialog box, you can set options to control the optimization of the library when doing hole filling and property optimization.

Hole-Filling and Library Optimization - Advanced Options Dialog Box Features

The command-line options for the canvasLibOpt program are given in the description of the features.

  • Number of optimization cycles box
  • Stop optimization when total score changes by less than N for M consecutive cycles boxes
  • Maximum run time option and text box
  • Monte Carlo score threshold text box
  • Random seed text box
  • Generate Seed button
Number of optimization cycles box

Specify the number of optimization cycles performed. Each cycle consists of making a pass through the selected subset for each structure, with an attempt to replace the poorest scoring structure at each pass. (Option -cycle)

Stop optimization when total score changes by less than N for M consecutive cycles boxes

Specify criteria for stopping the optimization if the score is not changing significantly. The optimization stops if the total score does not change by more than the specified amount for the specified number of consecutive cycles. (Option -stop)

Maximum run time option and text box

Select this option to limit the amount of CPU time for which the optimization can run, and se the limit in the text box. (Option -tmax)

Monte Carlo score threshold text box

Specify a threshold on the increase in the similarity and property scores, which is used in a simulated annealing process to determine if a structure is accepted when the score does not improve. The scheduling in this process is set so that the probability is 0.5 in the first optimization cycle and 0.01 in the last cycle. A larger threshold means that larger increases in the score will be accepted when the score does not improve. (Option -mc)

Random seed text box

Specify the random seed used wherever a seed is needed in the optimization process, including selection of the initial set of structures. The seed is an integer. (Option -rand)

Generate Seed button

Click this button to generate a random seed, which is placed in the Random seed text box.
