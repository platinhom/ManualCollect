Working with Excluded Volumes

In this exercise we will add excluded volumes to an existing hypothesis, then manipulate the excluded volume site locations and scaling, and selectively delete individual excluded volume sites.

  1. Choose File → Import Structures.

    The Import dialog box opens.

  2. From the Files of type option menu, ensure that Maestro is selected.

  3. If the options are not displayed, click Options.

  4. Ensure that Import all structures and Replace Workspace are selected.

  5. Select the file factorXa_inplace_phypo.mae.gz.

    This file contains multiple hypotheses from a develop common pharmacophore hypotheses experiment using prealigned ligands.

  6. Click Open.

    The Import dialog box closes. The first imported hypothesis, AADHHR_1, is displayed in the Workspace.

  7. Following the same procedure, import the file 2ei6_prepped.maegz.

    This file contains the 2EI6 PDB protein-ligand complex that is already aligned to the ligands used to create the in-place hypotheses imported in step 5.

  8. View both AADHHR_1 hypothesis and the 2ei6 entry in the Workspace by control-clicking (command-clicking) the AADHHR_1 hypothesis entry and the 2ei6 entry.

  9. Right click the blue H button on the AADHHR_1 entry and select Manage excluded volumes, to open the Manage Excluded Volumes panel.

    Initially it will be blank as the hypothesis was generated without excluded volumes.

  10. Select non-hydrogen atoms on the protein that define the binding site.

    1. Select Binding Sites from the More Objects button menu.

    2. Click the Remove from selection (minus) button.

    3. Select Hydrogens → All Hydrogen Atoms from the More Objects button menu to remove hydrogen atoms from the selection.

  11. In the Manage Excluded Volumes panel, change the Radius to 0.8.

  12. Click the Selected Atoms button to create excluded volume spheres with an 0.8 Å radius for all selected atoms in the Workspace.

    Each excluded volume is shown as a blue sphere in the Workspace and a row in the table in the Manage Excluded Volumes panel.

    Selecting rows in the excluded volumes table highlight in blue the corresponding excluded volumes in the Workspace.

  13. Select the first few excluded volumes by shift-clicking.

    The selected excluded volumes can be moved by dragging them with the right mouse button to a new location. This is the usual "translate" action.

  14. Delete the selected excluded volumes by clicking the Delete button in the Manage Excluded Volumes panel.

Excluded volumes they can be selected and moved directly from the Workspace.

  1. Select Pick sphere to translate in the Manage Excluded Volumes panel.

  2. Pick the excluded volume for the hydroxyl atom of residue TYR 99.

  3. Add to the selection by shift-clicking the adjacent aromatic carbon atom.

    Both excluded volumes are highlighted in blue, and are also selected in the excluded volumes table. Once selected they can be moved by dragging with the right mouse button (the usual translate action).

    Radii and XYZ coordinates for any excluded volume can be adjusted manually in the table of excluded volumes by double-clicking in the cell.

  4. Select the first excluded volume in the list and change its radius from 0.8 to 3.0.

    Note that the excluded volume shown in the Workspace now has a significantly larger volume than the other excluded volumes.

  5. Click Close in the Manage Excluded Volumes panel to save the excluded volume changes for the hypothesis.
