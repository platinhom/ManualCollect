Creating a Hypothesis with e-Pharmacophores from a Protein-Ligand Complex

In this exercise we will create a pharmacophore hypothesis from a ligand in the Maestro Workspace with perception of pharmacophore features coming from recognition of key protein-ligand interactions based on geometry.

  1. Choose File → Import Structures.

    The Import dialog box opens.

  2. From the Files of type option menu, ensure that Maestro is selected.

  3. If the options are not displayed, click Options.

  4. Ensure that Import all structures and Replace Workspace are selected.

  5. Select the file 3el8_prepped_complex.maegz.

  6. Click Open.

    The Import dialog box closes. The imported ligand is displayed in the Workspace.

  7. Click the Tasks button, type “phase” and press Enter.

  8. Select Develop Pharmacophore Hypothesis from the results list.

    The Develop Pharmacophore Model panel opens.

  9. Choose Receptor-ligand complex (Workspace) from the Create pharmacophore model using option menu.

  10. For Method, select Auto (E-Pharmacophore).

  11. Ensure ligand A:PD5 (601) is shown in the Choose ligand list.

    There is only one ligand in the complex so only one selection is available in the Choose ligand list.

  12. Click Hypothesis Settings to open the Hypothesis Settingsdialog box.

  13. Under Definitions to use, select Donors as projected points.

    The other options can be left at their defaults. In this tab there are options to control the maximum number of features in the returned hypothesis, enforce geometric separation between features returned, and the ability to import and employ a non-default feature definition file.

  14. In the Excluded Volumes tab, ensure that Create receptor-based excluded volume shell is selected.

    This option adds excluded volume to the pharmacophore hypothesis for receptor atoms close to the ligand. You can control the radii size and scaling of excluded volumes as well as the minimum and maximum distances from the ligand where excluded volumes can be located.

  15. Click Save to close the Hypothesis Settings panel.

  16. Change the Hypothesis name to 3el8_complex_epharm.

  17. Click Create to run the e-pharmacophores calculation.

    This calculation takes less than five minutes on a modern computer and returns a hypothesis to the Project Table.

The hypothesis created has 6 points, ADDRRR. Note the use of projected points for donors D5 and D7, which do not have vector characteristics and are located at the position of the protein atom making the hydrogen bond. The use of projected points for acceptor and donor features can be very useful when the location of a particular interaction is known but the directionality of the interaction is not.

Note the excluded volumes for protein atoms between 2 and 5 Å from the receptor that are scaled by their vdW radii.
