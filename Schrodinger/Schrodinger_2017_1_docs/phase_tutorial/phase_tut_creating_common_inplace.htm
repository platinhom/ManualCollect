Creating a Common Hypothesis from Prealigned Ligands

In this exercise we will create a pharmacophore hypothesis from a collection of pre-aligned factor Xa ligands. For this example a diverse set of factorXa protein-ligand complexes were aligned by binding site to create the ligand alignments.

  1. Choose File → Import Structures.

    The Import dialog box opens.

  2. From the Files of type option menu, ensure that Maestro is selected.

  3. If the options are not displayed, click Options.

  4. Ensure that Import all structures and Replace Workspace are selected.

  5. Select the file factorXa_dcph_inplace.maegz.

  6. Click Open.

    The Import dialog box closes. The imported ligand is displayed in the Workspace.

  7. View all newly imported ligands in the group factorXa_dcph_inplace, by including the first ligand in the group in the Workspace and then shift-clicking to include the last ligand in the group.

    All ligands between the first and last ligand in the group are included in the Workspace. You can see they are all aligned.

  8. Click the Tasks button, type “phase” and press Enter.

  9. Select Develop Pharmacophore Hypothesis from the results list.

    The Develop Pharmacophore Model panel opens.

  10. Set Create pharmacophore model using to Multiple ligands (selected entries).

  11. Ensure that all entries in the factorXa_dcph_inplace group are selected, by clicking in the row column of the factorXa_dcph_inplace group entry.

    Pharmacophore features should now be shown on ligands in the Workspace.

    It can be difficult to view the potential pharmacophore sites when large numbers of ligands are in the Workspace. Using a tile view can make this far easier.

  12. Press Ctrl+L (⌘L) .

    All ligands in the Workspace are shown in individual tiles. They are in the same frame of reference, and rotation by default works on all tiles. You can scroll up and down through the list using the slider to the right of the Workspace.

  13. Press Ctrl+L (⌘L) again to turn off tiling and revert to the single view of the Workspace.

    If you want more control over the tiles, click this icon on the lower right side of the main window.

    The Workspace Configuration panel opens, and you can change the tiling settings.

  14. Select Use prealigned ligands (consensus model) for the Pharmacophore method.

    All 37 ligands are treated as actives in this exercise. The common pharmacophore perception algorithm will look for common pharmacophores that retain the conformation and coordinates of the input structures. Note that the Generate conformers option is not inactivated.

  15. Click Hypothesis Settings to open the Hypothesis Settings panel.

    The Features tab is displayed by default and shows options relevant to the selection of features in the common pharmacophore. We are interested in a diversity of potential hypothesis, so we want to require only a fraction of the active ligands to match any given hypothesis, and we want to allow flexibility in the number of features in any given hypothesis.

  16. Set Hypothesis should match at least N% of actives to 15%.

  17. Set Number of features in the hypothesis to range from 4 to 7.

  18. Select Preferred minimum number of features and set the value to 5.

    Pharmacophore perception in this case will start by looking for 7 point hypothesis, then 6 point hypothesis, 5 point hypothesis, etc. When a hypothesis that finds the criteria is found the algorithm stops searching. When the Preferred minimum no. of features option is active and set to 5, if a 7 point hypothesis is found the algorithm will continue looking and will stop only after checking for 5 point hypotheses. You can leave the other options at their default settings.

  19. Switch to the Scoring tab of the Hypothesis Settings panel.

    The default scoring function for ranking hypothesis in common pharmacophore perception is the PhaseHypoScore. This metric is a linear combination of the Survival score and the BEDROC(ɑ=20) enrichment performance on a small-scale virtual screen automatically set up and executed by the common pharmacophore perception process. Each hypothesis is evaluated in its ability to rank actives used in common pharmacophore perception against the 1000 compound Glide decoy set. The Glide decoy set was created by selecting 1000 ligands from a one million compound library that were chosen to exhibit "drug-like" properties.

  20. To speed the current calculation change the Scoring Function to Custom (fast scoring).

    This option avoids execution of the small-scale virtual screen and the returned hypothesis are rank ordered by their survival scores. You can leave the coefficients for the Survival score at their default values.

  21. In the Excluded Volumes tab, ensure that Create excluded volume shell is not checked.

    We will not generate excluded volumes for this calculation.

  22. Click Save to close the Hypothesis Settings panel.

  23. Change the Job name to "factorXa_inplace" and click Run to start the calculation.

    This job takes only a few seconds on a single CPU. Upon completion results are incorporated into the Project Table in a group named the same as the job name, i.e. factorXa_inplace.

  24. In the Workspace Navigator the hypotheses are each shown as a group that contains the hypothesis entry and a directory with the aligned active ligand structures. The top-ranked hypothesis was AADHHR_1 which contains two acceptors, one donor, two hydrophobes and one ring feature. Double clicking the blue H button for any hypothesis displays it in the Workspace. Right clicking the blue H button for that entry shows options that are available to visualize and modify the hypothesis. These options are described in more detail in Changing and Viewing Hypothesis Attributes. Right clicking the green H button on the factorXa_inplace group entry enables visualization of multiple binding modes and when PhaseHypoScore is employed the enrichment viewer. These are described in more detail in Identifying Multiple Binding Modes from a Set of Hypotheses and Viewing Enrichment for a Set of Hypotheses.
