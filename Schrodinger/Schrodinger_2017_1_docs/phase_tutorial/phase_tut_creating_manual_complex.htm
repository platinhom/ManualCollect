Creating a Hypothesis Manually from a Protein-Ligand Complex

In this exercise we will create a pharmacophore hypothesis from a ligand in the Maestro Workspace with perception of pharmacophore features coming from recognition of key protein-ligand interactions based on geometry.

  1. Choose File → Import Structures.

    The Import dialog box opens.

  2. From the Files of type option menu, ensure that Maestro is selected.

  3. If the options are not displayed, click Options.

  4. Ensure that Import all structures and Replace Workspace are selected.

  5. Select the file 1f0r_prepped_complex.maegz.

  6. Click Open.

    The Import dialog box closes. The imported ligand is displayed in the Workspace.

  7. Click the Tasks button, type “phase” and press Enter.

  8. Select Develop Pharmacophore Hypothesis from the results list.

    The Develop Pharmacophore Model panel opens.

  9. Choose Receptor-ligand complex (Workspace) from the Create pharmacophore model using option menu.

  10. Select the Manual method option.

    After a few seconds proposed features are shown for ligand A:815 in the Workspace.

  11. Move the Show features slider to the right, adding feature D6 to the previous set of A4, D7, P9, and R11 features.

    This slider softens or hardens geometric criteria to recognize a protein-ligand interaction. When moved to the right the slider shows the most features, which for the 1f0r ligand includes six features. The R11 and R10 features arise from pi-pi stacking while the A and D features arise from neutral-neutral, charge-neutral and charge-charge protein-ligand hydrogen bonds.

  12. Ensure that Pick feature in Workspace is selected and pick the A4, D7, P9, and R11 features directly in the Workspace.

  13. Create excluded volumes:

    1. Click Hypothesis Settings.

    2. Select Create receptor-based excluded volume shell.

      While default settings will be used here, it should be noted that you have considerable control over scaling and radii of the excluded volumes as well as control over the thickness of the shell and distance from the ligand to which excluded volumes will be added.

    3. Click Save to close the Hypothesis Settings panel.

  14. Change the hypothesis name to "1f0r_complex" and click Create to create the requested 4-point hypothesis.

    By default, excluded volumes are shown as blue spheres
