Creating a Phase Database

 

  1. Click the Tasks button, type “phase” and press Enter.

  2. Select Create Database from the results list.

    The Create Phase Database panel opens.

  3. We will be creating a new Phase database so ensure that Create new database is selected.

    You can also add compounds to an existing database with the Append to existing database option.

  4. Click the Database path Browse button, navigate to the desired directory for the new database to be created, and enter the name tutorial_database in the File name text box.

  5. Ensure that Use ligands from is set to File.

  6. Click the Input structure file Browse button and select the 50ligs.mae.gz file.

  7. Click Open to select the file.

  8. Ensure that Generate ligand conformers is selected (the default), as this file has only one conformer per ligand.

  9. Retain default values in the Ligand Preparation tab to ensure low energy ionization/tautomeric states are sampled by Epik.

  10. In the Ligand Filtering tab, select Generate QikProp properties, Prefilter by Lipinski’s Rule, and Skip ligands with reactive functional groups.

    This will result in a Phase database that contains compounds with desirable properties. Significantly more control over filtering is available by generating LigFilter properties and using them to filter via a filter criteria file.

  11. Set the Job name to "phasedb_50ligs" and click Run to run the calculation to create a Phase database that is ready for use in screening.

    Note that no licenses are checked out during the creation of a Phase database so you can distribute the calculation over as many CPUs as you have available without running out of licenses.

    The calculation takes less than 10 minutes on modern CPUs and results in the creation of a Phase database named tutorial_database.phdb.
