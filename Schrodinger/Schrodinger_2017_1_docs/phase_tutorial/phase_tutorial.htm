Pharmacophore Modeling with Phase

These exercises are designed to help you quickly become familiar with performing pharmacophore modeling with Phase. Basic familiarity with Maestro is assumed. In these exercises you will learn to create several pharmacophore hypotheses (models) using different approaches suited to available data, learn how to change attributes of the hypothesis including location and orientation of features, how to manipulate and add excluded volumes, and how to control query conditions of hypothesis, how to screen (query) ligands against pharmacophore hypotheses, and how to analyze hypotheses for enrichment and multiple binding modes. Finally you will learn how to create and modify a Phase database for use in screening.

These exercises can be run independently, and are divided into these categories:

  • Creating Hypotheses
  • Modifying Hypothesis
  • Screening Ligands Against Hypotheses
  • Analyzing Hypotheses
  • Creating and Modifying Phase Databases

If you need help with a Phase task, click the Help button.
