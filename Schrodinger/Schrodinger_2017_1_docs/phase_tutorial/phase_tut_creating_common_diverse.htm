Identifying Common Pharmacophore Hypotheses from Diverse Ligands for Virtual Screening

In this exercise we will create a pharmacophore hypothesis from a collection of known actives to estrogen receptor beta. For this example a diverse set of known estrogen receptor active ligands is used to recognize common pharmacophore hypotheses. The resulting hypotheses are recommended for application in virtual screening experiments.

  1. Choose File → Import Structures.

    The Import dialog box opens.

  2. From the Files of type option menu, ensure that Maestro is selected.

  3. If the options are not displayed, click Options.

  4. Ensure that Import all structures and Replace Workspace are selected.

  5. Select the file erb_train.maegz.

  6. Click Open.

    The Import dialog box closes. The imported ligand is displayed in the Workspace.

  7. View 2D structures of all ligands in the erb_train group, by clicking on the group entry in the Row column to ensure all 50 entries are selected then choosing Window → 2D Viewer.

    Note the diversity of this set of ligands and the lack of obvious congeneric series.

  8. Close the 2D viewer to continue.

  9. Click the Tasks button, type “phase” and press Enter.

  10. Select Develop Pharmacophore Hypothesis from the results list.

    The Develop Pharmacophore Model panel opens.

  11. Set Create pharmacophore model using to Multiple ligands (selected entries).

  12. Ensure all entries in the erb_train group are selected.

    All 50 ligands will be treated as actives in this exercise.

  13. For the Pharmacophore method, select Find best alignment and common features.

  14. Open the Generate Conformers Options panel by clicking Options.

  15. Select Minimize output conformers.

    This option requests minimization of generated conformers with the OPLS3 force field.

  16. Close the panel by clicking the Save button.

  17. Open the Hypothesis Settings panel by clicking Hypothesis Settings.

    Because we are looking to recognize pharmacophore hypotheses in a diverse ligand set we will set some nondefault options.

  18. Change Hypothesis should match at least  to 25% of actives.

  19. Change Number of features in the hypothesis to range from 4 to 7 features.

  20. Select Preferred minimum number of features and set the value to 5.

    You can keep all other settings at their defaults. In this panel you can set the min/max number of features of each type allowed per hypothesis, the tolerance (radii in angstroms) per feature, the number of hypotheses returned per number of features, and feature definition specification.

    We will not be generating excluded volumes or changing from the default PhaseHypoScore scoring function in this exercise so there is no need to adjust default options on the Scoring or Excluded Volumes tabs.

  21. Click Save to close the Hypothesis Settings panel.

  22. Change the Job name to erb_common.

It is not necessary to run this calculation, which will take several hours on one CPU or <1 hour on 8 CPUs. If you run the calculation resulting, the hypothesis will be automatically incorporated into the Workspace. If you prefer not to run the calculation, import the results manually by choosing File → Import Structures and importing the file erb_common_phypo.mae.gz.

In the Workspace Navigator the hypotheses are each shown as a group that contains the hypothesis entry and a directory with the aligned active ligand structures. The top-ranked hypothesis was APRR_1 which contains one acceptors, one positive ionic, and two ring features. Right clicking the H button for that entry shows options that are available to visualize and modify the hypothesis. These options are described in more detail in Changing and Viewing Hypothesis Attributes. Right clicking the green H button on the erb_common group entry enables visualization of multiple binding modes and when PhaseHypoScore is employed the enrichment viewer. These are described in more detail in Identifying Multiple Binding Modes from a Set of Hypotheses and Viewing Enrichment for a Set of Hypotheses.
