Examining the Results

When the job finishes, the Residue Scanning Viewer panel opens, displaying changes in properties for each mutation, and a graph of one of the properties.

1. Choose D Affinity from the Graph property option menu.

The graph shows the change in affinity of the mutated chain for the other chain on mutation, as a function of the row number in the table. Note that some mutations improve the affinity between the chains, while others destabilize the complex.
