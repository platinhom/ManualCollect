Preparing the Protein for Consensus Water Display

1. Click Protein Preparation Wizard on the Favorites toolbar.

The Protein Preparation Wizard panel opens.

2. Enter 2pcy in the PDB text box, and click Import.

The 2PCY structure is imported into the project and displayed in the Workspace.

3. Deselect Delete waters beyond N Å from het groups.

This exercise uses the waters, so they should not be deleted.

4. Click Preprocess.

The structure is processed, and a new structure is added to the project and displayed in the Workspace, replacing the imported structure.

5. Close the Protein Preparation Wizard panel.
