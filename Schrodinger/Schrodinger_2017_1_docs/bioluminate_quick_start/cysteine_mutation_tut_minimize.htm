Minimizing the Mutated 1vhu Structure

After mutating the structure, the protein should be refined for further use. This can be done in the Protein Preparation Wizard panel, which should still be open from the preparation exercises. If not, open it.

1. In the Protein Preparation Wizard panel, click Minimize in the Refine tab.
2. When the job finishes, close the Protein Preparation Wizard panel.

The mutated protein is ready for the rest of the exercise.
