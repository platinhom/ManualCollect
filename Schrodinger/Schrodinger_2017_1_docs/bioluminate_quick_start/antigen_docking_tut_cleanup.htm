Cleaning Up after the Antibody-Antigen Docking Exercise

To clean up after the exercise:

1. Close the Protein-Protein Docking panel, if it is still open.
2. Choose File → Close Project, and click Discard in the Save Scratch Project dialog box.

The project is closed.
