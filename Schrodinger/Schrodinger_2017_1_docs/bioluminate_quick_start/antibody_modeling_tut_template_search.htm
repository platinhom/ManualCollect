Searching for Framework Templates for 1fsk

The next step is to search for a template for the framework region.

1. Click Search.

When the search finishes, the table is filled in with the results. 1fsk is not in the list, because it was removed by the filter.

1. Select the row containing 1i3g.

This template is among the highest-scoring results and will be used for this exercise.

2. Click Accept.

The template for the selected row in the results table is accepted as the template, and is imported into the project. After a short while, the Basic Loop Model tab is displayed.
