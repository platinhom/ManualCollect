workspacescopeview

Determine which things are visible inside and outside of the scope. Only used when the scope is enabled.

Syntax: workspacescopeview molecules= inside | outside | everywhere ribbons= inside | outside | everywhere surfaces= inside | outside | everywhere volumes= inside | outside | everywhere

Options:

molecules

View molecules nowhere, inside the scope, outside the scope or everywhere (inside and outside).

Valid values: nowhere
inside
outside
everywhere
Default value: everywhere

ribbons

View ribbons nowhere, inside the scope, outside the scope or everywhere (inside and outside).

Valid values: nowhere
inside
outside
everywhere
Default value: everywhere

surfaces

View surfaces nowhere, inside the scope, outside the scope or everywhere (inside and outside).

Valid values: nowhere
inside
outside
everywhere
Default value: everywhere

volumes

View volumes nowhere, inside the scope, outside the scope or everywhere (inside and outside).

Valid values: nowhere
inside
outside
everywhere
Default value: everywhere
