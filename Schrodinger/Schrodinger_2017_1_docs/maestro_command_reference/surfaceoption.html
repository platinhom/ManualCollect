surfaceoption

Set surface options.

Syntax: surfaceoption cavitydepth= yes | no cavityiterations= <n> cavitylimit= <x> defaulttransparency= <n> defaulttransparencyback= <n> defaulttransparencytogether= yes | no extentstep= <x> fastrender= yes | no keepcenter= yes | no lowqualitytransparency= always | never maxmeshwidth= <x> meshwidth= <n> minmeshwidth= <x> pairedbluecolor1= <n> pairedbluecolor2= <n> pairedcolor1= gray | dark_blue | blue | light_blue | aquamarine | turquoise | spring_green | dark_green | green | lime_green | yellow_green | yellow | orange | maroon | red | pink | plum | magenta | blue_violet | white pairedcolor2= gray | dark_blue | blue | light_blue | aquamarine | turquoise | spring_green | dark_green | green | lime_green | yellow_green | yellow | orange | maroon | red | pink | plum | magenta | blue_violet | white pairedgreencolor1= <n> pairedgreencolor2= <n> pairedredcolor1= <n> pairedredcolor2= <n> randomstipple= always | never scalemeshwidth= yes | no showtable= yes | no singlebluecolor= <n> singlecolor= gray | dark_blue | blue | light_blue | aquamarine | turquoise | spring_green | dark_green | green | lime_green | yellow_green | yellow | orange | maroon | red | pink | plum | magenta | blue_violet | white singlegreencolor= <n> singlelayertransparency= yes | no singleredcolor= <n> smoothing= <n> style= mesh | dots usecheesy= yes | no

Options:

cavitydepth

Enables weighting surface colors by cavity depth.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: true

cavityiterations

Sets number of iterations of cavity depth coloring algorithm.

Valid values: integers
Default value: 50
Minimum: 5
Minimum: 200

cavitylimit

Sets a darkening limit for the cavity depth algorithm.

Valid values: reals
Default value: 0.5
Minimum: 0.0
Minimum: 1.0

defaulttransparency

Sets default front surface transparency.

Valid values: integers
Default value: 0
Minimum: 0
Minimum: 100

defaulttransparencyback

Sets default back surface transparency.

Valid values: integers
Default value: 0
Minimum: 0
Minimum: 100

defaulttransparencytogether

Set the flag of adjusting default front and back surfaces together.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: true

extentstep

Step size of increasing or decreasing density map extent.

Valid values: reals
Default value: 1
Minimum: 0.1

fastrender

Enables optimized rendering of transparent surfaces.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: true

keepcenter

Set the flag of keeping electron density map center while translation is performed.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: false

lowqualitytransparency

Specifies drawing quality for transparent surfaces when rotating. Valid values are `When rotating', `Always', and `Never'.

Valid values: rotating
always
never
Default value: never

maxmeshwidth

Maximum thickness of scaled mesh lines.

Valid values: reals
Default value: 3.5
Minimum: 2.0
Minimum: 50.0

meshwidth

Thickness of mesh lines.

Valid values: integers
Default value: 1
Minimum: 1
Minimum: 20

minmeshwidth

Minimum thickness of scaled mesh lines.

Valid values: reals
Default value: 1
Minimum: 0.1
Minimum: 2.0

usecheesy

Off (the default) uses the high-quality but possibly slower-to-draw transparency. On uses the lower-quality cheesy translucency, but in some cases this will be significantly faster to draw. Cheesy translucency mimics transparency by mapping level of transparency to amount of holes in the surface.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: false
