tablealigncolumn

Set the alignment of the specified column in the current table.

Syntax: tablealigncolumn alignment= center | right [<columnname> <alignment> ]

Options:

alignment

The alignment to be set for the column. Valid values are "left", "center", or "right".

Valid values: left
center
right
Default value: left

Operands:

<columnname> <alignment>

The name of the column to align. This is the name displayed in the column header.
