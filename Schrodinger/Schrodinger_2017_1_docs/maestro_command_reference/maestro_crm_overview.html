Maestro Command Reference Overview

The Maestro Command Reference Manual provides information on Maestro's internal languages and commands.

  • The Atom Specification Language defines the language used to specify a set of atoms for some action.

  • The mmsubs Substructure Notation defines a linear substructure notation used for matching sets of atoms with a given connectivity.

  • The Entry Specification Language defines a language for selecting entries by their properties.

  • The Maestro Command Language defines the commands used :by Maestro to perform actions.
