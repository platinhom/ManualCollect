varytorsion

Defines a rotatable bond to be used in a conformational search.

Syntax: varytorsion maximum= <x> minimum= <x> [<atom1> <atom2> ]

Options:

maximum

The maximum value for the torsional rotation.

Valid values: reals
Default value: 180
Minimum: 0.0

minimum

The minimum value for the torsional rotation.

Valid values: reals
Default value: 0
Minimum: 0.0

Operands:

<atom1> <atom2>

The two atom numbers which define a bond to be rotated in a conformational search. These two atoms must have a bond (usually single) between them. Note that specifying a-b is the same as specifying b-a.
