entrygroupungroup

Ungroups the entries in the given group.

Syntax: entrygroupungroup isexp= yes | no [<group_name> ]

Options:

isexp

Specifies whether the <group_name> is actual group name or it is expression such as 'selected'.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: false

Operands:

<group_name>

<group_name> Remove the group and move all the entries in that group to the end of ungrouped section. In project table, all the ungrouped entries (if any) will be present at the top i.e. before all the groups.
