errorcheck

Specifies optional error checking to be performed.

Syntax: errorcheck ct= yes | no

Options:

ct

An option which determines whether mmct error checking is enabled during certain (primarily graphical) operations.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: false
