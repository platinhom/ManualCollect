psp_pairwise

Specifies a pair of atoms to define a Prime refinement pairwise constraint.

Syntax: psp_pairwise distance= <x> forceconstant= <n> [<atom1> <atom2> ]

Options:

distance

The distance from atom1 to atom2 of the pairwise constraint.

Valid values: reals
Default value: 0
Minimum: 0.0

forceconstant

The force constant for the pairwise constraint.

Valid values: integers
Default value: 350
Minimum: 0

Operands:

<atom1> <atom2>

The two atoms that define a Prime refinement pairwise constraint.
