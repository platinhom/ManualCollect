Implicit Operators in ASL

When no operator is specified, the following operations are assumed:

  • Specifications within a property list have a Boolean or relationship. The following expression returns atoms with PDB names CA or CB.

    atom.ptype CA,CB

  • When the classes are in order of decreasing priority in the class hierarchy, the and operator is assumed. For example, the following expressions are equivalent, and both return the alpha carbons in chain A of molecule 1.

    mol. 1 chain. A atom.ptype CA
    mol. 1 and chain. A and atom.ptype CA
