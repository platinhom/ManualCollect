volumerename

Renames the given volume.

Syntax: volumerename entry= <text> newname= <text> [<volume> ]

Options:

entry

The entry that the volume belongs to.

Valid values: text strings
Default value:  

newname

The new name for the volume.

Valid values: text strings
Default value:  

Operands:

<volume>

The name of the volume to rename.
