plotxyseriesdelete

Deletes the given XY data series.

Syntax: plotxyseriesdelete plot= <text> [<seriesname> ]

Options:

plot

This option is the name of the plot containing the series.

Valid values: text strings
Default value:  

Operands:

<seriesname>

The name of the series to delete.
