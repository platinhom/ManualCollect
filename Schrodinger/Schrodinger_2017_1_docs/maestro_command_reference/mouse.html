mouse

Specify how mouse movements are to be interpreted

Syntax: mouse rotate= x | y | z | adjust | trackball translate= x | y | z [There are no operands for this command. ]

Options:

rotate

How mouse movements are interpreted for rotation. Default is rotate in xy.

Valid values: xy
x
y
z
adjust
trackball
Default value: xy

translate

How mouse movements are interpreted for translation. Default is translate in xy.

Valid values: xy
x
y
z
Default value: xy

Operands:

There are no operands for this command.
