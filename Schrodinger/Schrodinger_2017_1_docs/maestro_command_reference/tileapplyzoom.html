tileapplyzoom

Set the zoom for all tiles from that of the specified Workspace tile.

Syntax: tileapplyzoom [<tile_name> ]

Operands:

<tile_name>

The name which identifies the contents of the tile. This name will be the index of an entry group (greater than 0), or 0: followed by the entry ID for un ungrouped entry (or Scratch entry). A tile named 0: (with no entry ID) does not belong to any entry or entry group. In tile by surface mode, the entry ID will be followed by the name of the special surface for that tile.
