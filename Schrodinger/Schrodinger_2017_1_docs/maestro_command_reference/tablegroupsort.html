tablegroupsort

Reorder the groups in the specified table.

Syntax: tablegroupsort field= <text> order= descending source= firstentry [<table> all|<group name list> ]

Options:

field

The name of the group/entry property to be used as sort-key attribute. Use grouptitle to reorder groups based on group's title. Use property name to reorder groups based on first entry's property value.

Valid values: text strings
Default value: grouptitle

order

This option sets order of sorted values to be either ascending or descending.

Valid values: ascending
descending
Default value: ascending

source

This option determines whether the 'field' option takes 'group property name' or 'entry property name'. Valid values are group or firstentry .

Valid values: group
firstentry
Default value: group

Operands:

<table> all|<group name list>

The name of the table to sort. Comma separated list of group names. Or 'all' to sort all groups.
