createcustompropertygroup

This command creates a custom property group for the given space separated property names.

Syntax: createcustompropertygroup [<propertygroupname> <propertynames> ]

Operands:

<propertygroupname> <propertynames>

The name of the custom property group. The space separated property names use to create the custom property group.
