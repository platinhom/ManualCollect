tablesort

Sort current entry selection in the specified table.

Syntax: tablesort field= <text> order= descending [<table> ]

Options:

field

The name of the property to be sorted on. This option has been deprecated in favor of the tablesortfields command. This option does nothing now.

Valid values: text strings
Default value: s_m_entry_name

order

This option sets order of sorted values to be either ascending or descending. This option has been deprecated in favor of the tablesortfields command. This option does nothing now.

Valid values: ascending
descending
Default value: ascending

Operands:

<table>

The name of the table to sort.
