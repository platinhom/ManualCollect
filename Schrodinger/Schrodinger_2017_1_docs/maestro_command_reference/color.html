color

Specifies a color or set color options.

Syntax: color alpha= <n> blue= <n> green= <n> red= <n>

Options:

alpha

Alpha color component.

Valid values: integers
Default value: 128
Minimum: 0
Minimum: 255

blue

Blue color component.

Valid values: integers
Default value: 128
Minimum: 0
Minimum: 255

green

Green color component.

Valid values: integers
Default value: 128
Minimum: 0
Minimum: 255

red

Red color component.

Valid values: integers
Default value: 128
Minimum: 0
Minimum: 255
