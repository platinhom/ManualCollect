entrygroupcreatewithselected

Creates a group with selected groups and entries.

Syntax: entrygroupcreatewithselected autodetectparentgroup= yes | no parentgroupid= <text> [<group_title> ]

Options:

autodetectparentgroup

If autodetectparentgroup is true, then parentgroup option is ignored.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: true

parentgroupid

Parent group of the new group to be added i.e. nested group, it will be empty if the group is not nested group.

Valid values: text strings
Default value:  

Operands:

<group_title>

Title of the new group to be created.
