Creating New Atom Sets from Existing Ones in ASL

Maestro supports a mechanism for defining and naming atom sets via the Sets Panel. The names of existing sets may be used in expressions if they are prefixed with the word set. For example, if there are two sets defined as:

set S1 mol. 1 (molecule number 1)
set S2 atom.ptype C,O,N,CA (all backbone atoms)

the following are valid atom specifications, giving the described result:

  • set S1 and set S2, all backbone atoms in molecule 1
  • set S1 or set S2, all backbone atoms or atoms in molecule 1
  • within 5.0 set S1, all atoms within 5.0 Å of molecule 1.
