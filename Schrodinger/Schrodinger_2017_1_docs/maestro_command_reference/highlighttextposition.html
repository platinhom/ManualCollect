highlighttextposition

Position of the optional text for a highlight. If there is text an associated, an arrow is rendered. Its tail is always attached to the text. The head position is controlled by the headx, heady, headz option values. These are automatically set and updated based on the ASL definition of the highlight. If the text is empty, no arrow is drawn.

Syntax: highlighttextposition headx= <x> heady= <x> headz= <x> x= <x> y= <x>

Options:

headx

3D X position of the arrow head

Valid values: reals
Default value: 0

heady

3D Y position of the arrow head

Valid values: reals
Default value: 0

headz

3D Z position of the arrow head

Valid values: reals
Default value: 0

x

Fractional X window position of the optional highlight text.

Valid values: reals
Default value: 0.75
Minimum: 0.0
Minimum: 1.0

y

Fractional Y window position of the optional highlight text.

Valid values: reals
Default value: 0.5
Minimum: 0.0
Minimum: 1.0
