workspaceselectionmatch

An atom is selected if its value for the specified attribute matches that of any currently selected atom.

Syntax: workspaceselectionmatch [<attribute> ]

Operands:

<attribute>

The attribute (color, representation, element, entry, or residue_type) for which matches are to be found.
