ASL Matching of Missing or Added Atoms

When an ASL expression refers to atoms that are not present (or not yet present) in the Workspace, these atoms are ignored, and the expression matches the atoms that are present. If atoms are subsequently added to the Workspace and one or more of these atoms are matched by the expression, they are added to the set of atoms defined previously by the expression.

For example, if a structure in the Workspace has only 100 atoms when the ASL definition: atom.num 1,8,44,101,103 is applied, Maestro simply matches the atoms numbered 1,8 and 44. If three more atoms are subsequently added to the structure, the atoms numbered 101 and 103 are added to the previously defined set.
