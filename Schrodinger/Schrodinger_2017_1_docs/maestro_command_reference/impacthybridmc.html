impacthybridmc

Settings associated with hybrid Monte Carlo simulations in Impact.

Syntax: impacthybridmc ncycles= <n> nmdmc= <n> timestep= <x>

Options:

ncycles

The total number of HMC cycles to be performed during an Impact hybrid Monte Carlo simulation.

Valid values: integers
Default value: 100
Minimum: 1

nmdmc

The number of MD steps per HMC cycle to be performed during an Impact hybrid Monte Carlo simulation.

Valid values: integers
Default value: 4
Minimum: 1

timestep

The time step to be used (in ps) during an Impact hybrid Monte Carlo simulation

Valid values: reals
Default value: 0.001
Minimum: 0.0000001
