exportrotatablegroups

Exports the receptor plus optional poses, setting the position of the receptor's movable atoms from the current pose.

Syntax: exportrotatablegroups poses= current | matching [<file name> ]

Options:

poses

Determine what to export.

Valid values: none
current
matching
Default value: none

Operands:

<file name>

The file name to export the receptor and poses to.
