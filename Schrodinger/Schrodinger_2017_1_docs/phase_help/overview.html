Phase Overview

Phase is a versatile product for developing a pharmacophore model from a set of ligands, creating a 3D QSAR (quantitative structure-activity relationship) model, and searching a set of structures for matches to the pharmacophore model. Phase can be used for lead discovery, optimization, and expansion. It can also be used for SAR development and alignments for third party QSAR models.

Phase Tasks

The tasks that you can carry out in Phase are:

  • Building a pharmacophore model and an optional QSAR model from a set of ligands
  • Creating a pharmacophore hypothesis from a single ligand
  • Preparing a 3D database that includes pharmacophore information
  • Searching the database for matches

Each of these tasks is supported by a Maestro panel.

Building a pharmacophore model involves the following steps:

  • Preparing the ligands, including 2D-3D conversion and the generation of ligand conformations
  • Defining and identifying the pharmacophore sites in the ligands
  • Creating hypotheses by finding common pharmacophores
  • Scoring the hypotheses
  • Building a 3D QSAR model

Building a database involves preparing the ligands, including 2D-3D conversion, and can include the generation of ligand conformations and the definition and identification of the pharmacophore sites in the ligands. Searching for matches involves conformer generation and identification of the pharmacophore sites in the ligands, if not done previously; locating matches to a hypotheses; application of filters, including excluded volumes; and prediction of the activity of the matches using the QSAR model.

Phase Interface

The Maestro interface for building a pharmacophore mode is wizard-like, and takes you through each step of the process. The interface is more flexible than a wizard, because it allows you to exit at any step and resume the process later at the same stepâ€”or a different step, provided that you have the data to support that step. Default options are provided that should give good results, but you can also choose from a range of other options to suit your purposes. The interface for each of the other three tasks is a single panel.

Running Phase Jobs

All Phase jobs can be started from Maestro. Many of these jobs generate a large amount of data, and most of them can be distributed across multiple processors. In some panels, you can choose Job Settings from the Settings button menu,

set the job parameters in the Job Settings dialog box, and click Run to run the job. In other panels, when you click a Start button or an action button that starts a job, a panel is displayed that allows you to make job settings.
