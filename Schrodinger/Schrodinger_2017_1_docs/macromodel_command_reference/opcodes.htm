MacroModel Operation Codes

This chapter provides reference information for using the operation codes in the MacroModel command file. Unless otherwise specified, a zero in any opcode argument specifies the default value, if there is a default. Also, unless otherwise specified, the default is a reasonable value to choose if you do not have a good reason to select a different value.
