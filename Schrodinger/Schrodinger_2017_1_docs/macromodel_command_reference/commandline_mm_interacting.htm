Interacting With a Running MacroModel Job

The Schrödinger Job Control facility allows you to interact with running MacroModel jobs. Because all jobs submitted, either from Maestro or the command line, are controlled by the Job Control facility, all jobs can be monitored using the Monitor panel in Maestro or using the jobcontrol script from the command line. For specific instructions on using the Monitor panel or the jobcontrol script, see Managing Jobs.
