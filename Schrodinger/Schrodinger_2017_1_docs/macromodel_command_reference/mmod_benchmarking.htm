MacroModel Benchmark Suites

A benchmark suite is included in the MacroModel distribution for use in comparing the relative speed of different releases or of the same release on different platforms. The benchmark suite contains a number of individual tests which collectively exercise MacroModel calculations involving minimizations and conformational searches on small and protein-sized molecules.

To run the benchmark copy the directory:

macromodel-vversion/test/benchmarks

from the installation directory to a suitable location. In the new benchmarks directory type:

./do > run_log &

At the end of the calculation a file called Summary.machine_name is written that contains the times for the various tests performed. There is also a README file in the benchmark directory that provides a brief description of the various benchmarks.
