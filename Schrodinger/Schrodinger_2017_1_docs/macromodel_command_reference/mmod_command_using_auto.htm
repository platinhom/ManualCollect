Automatic Setup (AUTO) of Comparison Atoms for Redundant Conformer Elimination

You can use automatic setup for minimizations of a collection of conformers with redundant conformer elimination. Automatic setup is enabled by adding the AUTO opcode to the command file for a calculation. You should place the AUTO opcode, with appropriate arguments, just before the MINI opcode in the command file. You must also add a MULT opcode to the command file, as seen in the example in MacroModel Example: Multiple Conformer Minimization With Automatic Redundant Conformer Elimination. AUTO is available from Maestro only for conformational searches. Automatic setup should not be used with minimizations of single structures or minimizations with multiple non-conformers as input.

AUTO selects comparison atoms (COMP), so that redundant conformers can be identified and removed without explicitly designating comparison atoms in the command file or from Maestro. In addition, chiral atoms (CHIG ) and torsion checks (TORC) are automatically identified by AUTO . AUTO applies the desired structural criteria to the minimized conformers as if the CHIG and TORC opcodes were present in the command file. AUTO has options to adjust or turn off these features. See the AUTO opcode description for more information.

In addition, AUTO uses the information in any substructure specified for the computation, including information on fixed or frozen atoms. The substructure must be specified in a substructure file and read in with a SUBS 0 command, and cannot be specified in the command file. Likewise, fixed or frozen atoms must be specified in the substructure and cannot be specified independently.

Finally, if only automatic comparison atom setup is required for a multi-conformer minimization with redundant conformer elimination, then COMP arg1=0 may also be used in the command file. Examples are given in MacroModel Example: Multiple Conformer Minimization With Automatic Redundant Conformer Elimination.

AUTO is not currently compatible with LOOP, and must be used carefully with Embrace. For more information on the parameters that AUTO is assembling, use DEBG 520 or 521.
