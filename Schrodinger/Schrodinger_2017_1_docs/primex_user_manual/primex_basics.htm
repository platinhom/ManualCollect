PrimeX and Maestro Basics

The graphical interface for all Schrödinger products, including PrimeX, is Maestro. Maestro provides many features for storing, managing, manipulating, creating, and modifying structures, surfaces and associated data. Information on the general capabilities of Maestro can be found in the Maestro Quick Reference Guide. The topics listed below describe the Maestro features that are particularly relevant to PrimeX, in the context of performing common tasks.
