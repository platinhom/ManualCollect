R-Factor Calculations

Most PrimeX tasks include an R-factor calculation, so there is usually no need for a separate calculation of R factors. At the beginning of a refinement, however, the R factors are not likely to have been calculated. You can calculate them with the R-Factor Calculation task. The R-Factor Calculation panel contains a Calculation Settings button, which opens the Calculation Settings dialog box (see Specifying Calculation Settings for PrimeX). If you are satisfied with the settings, click Run. The job takes only a few minutes.
