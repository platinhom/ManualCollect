Setting Up a PrimeX Refinement

The first step in the PrimeX refinement process is to specify the source of the model structure, the sequence, the reflection data, optional maps, and set the space group and unit cell parameters. These tasks are performed in the Input Data dialog box, and are described in the topic listed below.
