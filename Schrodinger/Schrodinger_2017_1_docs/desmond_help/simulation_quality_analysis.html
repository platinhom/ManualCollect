Using Features Links

Simulation Quality Analysis Panel

This panel displays various measures of the quality of a simulation, and allows you to plot various quantities as a function of simulation time.

To open this panel, choose Tasks → Classical Simulation → Molecular Dynamics Analysis → Simulation Quality Analysis in the Task Tool.

Using the Simulation Quality Analysis Panel

To analyze the quality of a simulation, you must load an Energy Sequence (.ene) file, then click Analyze to analyze the simulation results.

Simulation Quality Analysis Panel Features

  • Job base name text box and Browse button
  • Analyze button
  • Plot button
  • Block length for averaging text box
  • Simulation summary section
  • Properties section
Job base name text box and Browse button

Enter the name of the simulation job including the path in this text box, or click Browse and navigate to the job; the appropriate file to select should have a .ene extension.

Analyze button

Analyze the simulation results. You must have write permission to the directory that you loaded the .ene file from. This operation takes a minute or so. When it finishes, the Simulation summary and Properties sections are filled in.

Plot button

Plot thermodynamic properties as a function of simulation time. Opens a panel that displays the plot.

Block length for averaging text box

Enter the block length for averaging of the simulation results to reduce the noise and eliminate the correlation between consecutive reports.

Simulation summary section

This section displays a summary of the simulation settings and features.

Properties section

This section displays a set of thermodynamic properties extracted from the simulation results.
