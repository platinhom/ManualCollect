Radial Distribution Function Panel

Calculate and plot radial distribution functions from a molecular dynamics simulation.

To open this panel, choose Applications → Desmond → Radial Distribution Function in the Task Tool.

Using the Radial Distribution Function Panel

The radial distribution function (RDF) gives the probability of finding a particle at a distance r from another particle. It is also known as the pair distribution function. The function is calculated from a trajectory as a histogram, in which bins are created for each distance range, and the count of interparticle distances that lie in the range for each bin are accumulated from each frame of the trajectory. The results are then normalized to give a probability.

Radial distribution functions are usually zero at small distances because of interatomic repulsion at these distances. There are often sharp peaks at distances that correspond to structure in the system: for example, the first solvation shell of a molecule in solution. The integral of the radial distribution function can provide information about average coordination numbers.

This panel allows you to calculate the radial distribution function for a set of identical atoms, or between two sets of atoms. The atoms are specified using the usual picking tools.

The calculation requires a trajectory from a simulation. Once you have loaded the simulation file, you can specify the two atom sets and calculate the RDF. You can restrict the range of frames used from the trajectory (Zoom), set the bin size and maximum distance (Options), use periodic boundary conditions (PBC), and make some choices for the appearance of the plot.

When the job has run, you can import the results and display a plot of the radial distribution function. You can export an image of the plot, or export the data (function and integral) to a text file.

You can also generate radial distribution functions from the command line—see Other Desmond Scripts.

Run Calculation tab

Specify the simulation for which you want to calculate a radial distribution function and set up the calculation. You can run the job using the Job toolbar.

File name text box

This noneditable text box shows the name of the CMS file for the simulation that you chose for calculation of the radial distribution function.

Load button

Load the CMS file (-out.cms) for the simulation that you want to use for the radial distribution function. Opens a file selector, in which you can navigate to and select the file.

Zoom button

Zoom in on a specific portion of the trajectory to use when calculating the radial distribution function. Opens the Trajectory Filter dialog box, in which you can change the range of frames used in the calculation.

Options button

Set the maximum distance (Max r) used and the bin width (delta r) for accumulating distance values. The calculation time increases as you increase the maximum distance and decrease the bin width. Opens the RDF Options dialog box, in which you can make these settings.

Atom selection section

In this section you can specify the atoms for which you want to calculate the radial distribution function and perform the calculations.

Selection 1 atoms picking tools

Pick the atoms whose radial distribution function you want to calculate with these standard picking tools. If you want to calculate the distribution between two sets of atoms, specify the first set of atoms. The ASL expression for the atoms is given (or can be entered) in the text box.

Selection 2 option and text box

If you want to calculate the radial distribution (correlation) function for two different kinds of atoms, select this option and use the picking tools for the second set of atoms to pick the atoms. The ASL expression for the atoms is given (or can be entered) in the text box.

CoM options

This option is available for both Selection 1 and Selection 2. When you select this option, the center of mass of the selected atoms in each molecule is calculated and the pair correlation function is calculated between the centers of mass instead of the individual atoms. For example, if the ASL expression for Selection 1 is water, the pair correlation function between the centers of mass of all water molecules is calculated.

PBC option

Use periodic boundary conditions when evaluating the radial distribution function. This reduces artifacts arising from the box boundary, but also introduces artificial long-range correlations that would not be present in a real system. You should ensure that the maximum distance you use for the radial distribution function is smaller than the box dimensions to avoid artificial structure in the plot.

View Results tab

View the radial distribution function and export it to a file.

Import Results File button

Import the results of a radial distribution function calculation. The file that is selected is the log file (.log) for the calculation. Opens a file selector so you can navigate to and select the file.

Export button

Export the radial distribution function to a plain text file or as an image in PNG, JPEG, or SVG format.

Show integral option

Plot the integral of the radial distribution function on the plot as well as the function itself. The integral is evaluated at the same time as the function, so it is displayed immediately. You can use the integral to infer average coordination numbers.

Show grid option

Display a grid of dotted lines that are aligned to the axis tick marks.

Show legend option

Show the ASL expression and the line color at the top of the display area.

Display area

This area displays the function and the legend. When the integral is displayed, the axis on the right side is used to mark the integral values.

Job toolbar

Manage job submission. See Job Toolbar for a description of this toolbar.

Status bar

The status bar displays information about the current job settings for the panel. This includes the job name, task name and task settings (if any), number of subjobs (if any) and the host name and job incorporation setting. It also displays a message for a short time when you start a new job.

The status bar also contains the Help button, which opens the help topic for the panel.
