Running Jaguar pKa Calculations from the Command Line

To submit a pKa job for a single molecule using the command line, use the following command:

jaguar pka [options] [-csrch] {jobname.in|acid-and-base-files}

The options are the same as for the jaguar run command (see Running Jaguar Jobs from the Command Line). If the acid and base conformations are similar, you need only specify one input file, jobname.in, which can contain a structure for an acid or a base. The output Maestro file contains the optimized geometry of the input structure.
