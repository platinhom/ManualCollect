Running Complex Jaguar Jobs with Python Scripts

The Python package provided with Schrödinger software provides Python language extensions to analyze and manipulate structures, including reading and writing a variety of file formats; interfaces to Maestro, including support for creating customized user interfaces; and functions to interface with the Schrödinger Job Control facility. For more information, see the Python API web page.

The Jaguar software distribution includes several Python scripts, which are described in the sections below. Some of these scripts also have Maestro interfaces for setting up the job.

counterpoise.py: Counterpoise-Corrected Binding of Two Molecules

hydrogen_bond.py: Binding Energy of a Hydrogen-Bonded Complex

h_abstraction.py: Homolytic Breaking of a Bond to Hydrogen

AutoTS.py: Automated Transition State Search

distributed_scan.py: Jaguar Scan Distributed over Processors

fukui.py: Calculate Fukui Functions

vcd.py: Vibrational Circular Dichroism Spectra

canonical.py: Enforce Use of the Same Number of Canonical Orbitals

jreactions.py: Reaction Energy, Enthalpy, Free Energy
