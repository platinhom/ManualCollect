The Jaguar Configuration File

Jaguar provides a way of defining sets of gen section keywords that can be used to "configure" a calculation. These configurations, or macros, are stored in the file jaguar.config, which you place in your Schrödinger user resources directory. The file consists of a set of macros, in the following format:

MACRO1
keyword1=value1
keyword2=value2
...

Each macro has a name, which can be used in the gen section of the input file. The name must be on a separate line. It is followed by a set of lines with keyword=value settings, one per line. Any line that does not have an = is considered to be the name of a new macro. The macro name is case-sensitive; we recommend using upper case to distinguish macros from regular keywords in the input file. Blank lines are ignored.
