Jaguar Reactions Output

The output from the Jaguar - Reactions panel (or the jreactions.py script) that gives the reaction results is contained in the file jobname_rxn.out. A typical example is given below.

Date and time:          14 January 2014, 15:21:50
 
Reactant input files:
CH3CN.in
H2O.in
 
Product input files:
CH3CONH2.in
 
Reaction output file: hydrolysis_rxn.out
 
Reaction energetics in kcal/mol:
----------------------------------------
Delta E (gas phase)      = -33.35
Delta E (solution phase) = -30.66
Delta H (solution phase) = -26.99
Delta G (solution phase) = -14.56
