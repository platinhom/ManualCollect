Generating a Small Set of Low-risk Variations with LigPrep

To generate a few low-risk variations on the input structures:

ligprep -r 1 -s 8 -t 4 -i 2 -W i,-pht,1.0 ‑imae input.mae -omae output.mae

For each structure in the input structure file, LigPrep generates an extended conformation containing only the lowest energy ring conformation for any rings present, up to 8 stereoisomers, up to 4 tautomers, and ionization states highly populated in the pH range 6.0 to 8.0 (7.0 ± 1.0). Typically, far fewer than 8 stereoisomers and 4 tautomers would be produced.

This command assumes that the available information on the stereoisomers is recorded as R/S properties in the input structure file. If the input structures are 3D and have the correct chiralities, you could use the -g option to instruct LigPrep to respect the chiralities present in the 3D geometry.
