Tools Used by LigPrep

This chapter describes the tools used by LigPrep. These tools exist as libraries, which LigPrep uses directly, and as utilities, which can be run independently. A description of the purpose and the utility syntax is provided for tools that are unique to LigPrep, and a reference to the full description is provided for common utilities. A description of the options can be obtained by running the command with the -h option. The options for the tools cannot be used directly as options for ligprep. The ligprep options can be used to control the behavior or the tools, and for some tools, options can be passed to the tool with the ligprep -W option. The descriptions are provided so that you can understand how these tools function in the context of LigPrep, and so that you can use these tools separately from LigPrep.

LigPrep can use a separate product, Epik, instead of the ionizer and the tautomerizer to adjust protonation states. For more information on Epik, see the Epik User Manual.

In the command syntax summaries given in the topics listed below, the path to the command is not given. On Linux and Mac, you may need to prefix these command with $SCHRODINGER/utilities/. On Windows, the commands are already in the path in the Schrodinger shells. See Running Schrödinger Software for more information. For detailed command syntax, run the command with the -h option.
