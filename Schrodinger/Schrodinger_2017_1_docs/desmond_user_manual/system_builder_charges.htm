Using Custom Charges in the Desmond System

If you want to use partial charges from a source other than the force field, you can do so by selecting Use custom charges in the Solvation tab. You can then choose to use the partial charges from the structure, or enter the name of the property that defines the custom charges in the Custom text box. The property name is the internal name, which should start with r_ (i.e. a real-valued property). For example, the property r_j_ESP_Charges selects Jaguar-generated ESP charges.

When you have selected the property, click Select to select the atoms for which these charges are to be used. There is no default. The selection is made in the Atom Selection dialog box, which is described in detail in Atom Selection Dialog Box. If the property you chose has values only for some of the atoms (e.g. the ligand), you can select these atoms by specifying the entire range of values. Atoms that do not have a value for the property will not be selected.
