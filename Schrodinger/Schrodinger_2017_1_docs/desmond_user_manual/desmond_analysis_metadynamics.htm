Metadynamics Analysis

The Metadynamics Analysis panel analyzes a metadynamics calculation and displays a plot of the free energy as a function of the collective variables (CVs) used for the calculation. You can analyze calculations with one or two collective variables. The free energy value can be plotted relative to the minimum value or as an absolute value. The plots can be saved and loaded. To open this panel, choose Applications → Desmond → Metadynamics Analysis in the Task Tool.
