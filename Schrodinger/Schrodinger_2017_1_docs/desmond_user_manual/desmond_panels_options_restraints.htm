Restraints for Desmond Simulations

In the Restraints tab you can specify restraints on atom positions. A restraint is defined by a set of atoms and a force constant. The restraints are listed in the restraints table. The Atoms column is filled in automatically when you click Select and use the Atom Selection dialog box to specify the atoms. You must enter the force constant in the table manually, by editing the table cell.

To manage the restraints, you can use the buttons beside the table:

  • Select—Opens the Atom Selection dialog box to specify the atoms for the selected restraint. Only available if a single row is selected in the table.

  • Add—Adds a row to the restraints table so that a new restraint can be defined.

  • Delete—Deletes the selected restraints.

  • Reset—Resets the table to its default state.
