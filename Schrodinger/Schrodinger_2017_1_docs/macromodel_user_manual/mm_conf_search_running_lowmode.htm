Low-Mode Parameters for Conformational Searches

In the lower portion of the Customize the search section are three text areas that are relevant only to low-mode searches. These are active only when any of the four methods involving Low-Mode Conformational Searching is the current search method.

Probability of a torsion rotation/molecule translation

This control is available only with the mixed methods. This text box allows you to set a probability that any defined torsion rotations and molecule translations will be made during the search. The value should be a number from 0.0 to 1.0.

Minimum distance for low-mode move
Maximum distance for low-mode move

Used for setting the minimum and maximum distance for a low-mode move. During a search, the fastest moving atom is moved at randomly generated distances that are between the minimum and maximum values specified in these text boxes.
