MacroModel General Settings

All Maestro energetic calculation panels have the same basic setup process:

1. Specify the structure input source from the Use structures from option menu (see Specifying the MacroModel Structure Input Source below).
2. Specify settings in the tabs:
  • The MacroModel Potential Tab
  • The MacroModel Constraints Tab
  • The MacroModel Substructure Tab

For tabs specific to the energetic calculation, see the appropriate chapter.

3. Click Run to start the job with the current job settings, click the Settings button to set up and launch the job, or choose Write from the Settings button menu to write the files for future use (see Setting Up, Running, And Monitoring MacroModel Jobs).
