Analyzing a Set of Structures for R-Groups

The analysis and display of R groups is done in the R-Group Analysis panel. The panel is available from Maestro, from Canvas, and from the command line.

To open the R-Group Analysis panel:

  • from Maestro, choose Tasks → Structure Analysis → Small Molecule Analysis → R-group Analysis in the Task Tool.

  • from Canvas, choose Applications → R-Group Analysis in the master view.

  • from the command line, type the following command in a terminal window:

    $SCHRODINGER/run r_group_analysis.py [input-file]
