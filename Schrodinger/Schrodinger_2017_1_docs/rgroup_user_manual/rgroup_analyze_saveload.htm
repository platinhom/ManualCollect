Saving and Loading Results of an R-Group Analysis

If you want to save the results of the analysis, click Save Results, and navigate to a location and save the results (as a zip file) in the file selector that opens. You can then read the results of this analysis back in at a later time, by selecting Read results from previous run, then clicking Browse and browsing to the zip file. When the results are read, you can click View Results to view the results of the analysis, as explained below. The analysis controls are dimmed when you select Read results from previous run.
