Running the R-Group Analysis

When you have selected the input structures and chosen a method for defining the core, click Start to run the analysis. Once the analysis finishes, the R-Group Viewer panel opens. In this panel you can view the groups, export them, and open other panels that display various analyses of the properties. These panels are described in the following sections.
