Getting Help

Welcome to the Maestro Help System! We have a variety of material and features designed to help you learn about Maestro and find the information you need.

To get started with Maestro, you can use our Online Workshop Series. Here you will find introductory tutorials and videos covering the essentials of using Maestro. This page also provides access to a wider range of training material in webinars and walk-throughs, a quick reference card, and a reference guide.

For the latest documentation, go to the Documentation page on our web site.

Inside Maestro you will find information in a number of places:

  • Information on tasks, such as notification of the success of an operation, is displayed in banners at the top of the Workspace. Banners that are used for tasks give brief information on what to do to complete the task.

  • There are tooltips (balloon help) in most places throughout Maestro. To display the tooltip, pause the pointer over the GUI element. The tooltip should appear within a few seconds near the pointer location. If it doesn't appear, check the General – Appearance section of the Preferences Panel to ensure that Show tooltips is selected.

  • The help facility contains more detailed information. For information about a panel or the tab that is displayed in a panel, click the Help button in the panel,

    or press F1. The relevant help topic is displayed in this panel.

  • You can search the entire documentation set using the Search box in the header of any page. If you include special characters, make sure you enclose the search text in double quotes, e.g. "&gen". The results are displayed in a page that has links to the topics in which the text was found. When you click a link to display a topic, the search text is highlighted in the topic. You can restrict the search using the dropdown list at the right of the search bar.

    To search within a topic, press Ctrl+F (⌘F). A search bar is displayed at the bottom of the panel, where you can enter search text and step through the matches to the text.

  • The Help menu provides access to the help facility and a range of other types of documentation:

    • To work through tutorial exercises, you can use the Tutorials panel to copy the tutorial files and open the tutorial documentation.

    • For information on keyboard shortucts, choose Help → Keyboard Shortcuts.

    • The Knowledge Base, which you can search for information that isn't in the help facility or manuals. It has over 1000 short articles on a wide range of topics.

    • The New Features web page lists the major features that have been added to each software release for the last few years.

    • The Known Issues pages lists any serious issues that have been found in each software release for the last few years.

    • If you need to contact technical support, choose Help → Contact Technical Support.

  • The manuals provide more detailed and background information on our products. To open a manual, you can use the navigation frame to the left. Manuals are classified as Tutorials, User Manuals (which contain information on using the product from Maestro), and Command Reference Manuals. To search the manuals, use the Search tool at the top right of the page. The manuals and the help are integrated into a single, searchable system.
