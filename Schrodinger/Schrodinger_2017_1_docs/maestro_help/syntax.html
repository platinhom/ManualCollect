Maestro Command Language Syntax

Maestro commands are one-line commands. They consist of a keyword and zero or more options and zero or more operands. Options are of the form name=value, while operands are names. The syntax of a Maestro command is:

keyword [options] [operands]

The keyword is always first on the command line and it must match a known keyword completely (i.e. to all characters). There is an alias facility that allows you to map keywords to shorter names. These substitutions are made before the command is processed. See the topic Command Alias Panel for details.

The options can consist of any number of name=value pairs. Options can be of type Boolean (yes/no;y/n;t/f;true/false are valid values), string, integer or real. Some integer options may have "lookup" tables. For example in the adjustdistance command below, the move option can have three values, which are stored internally as integers.

Options are used to set values that generally correspond to states within the program. Once an option is set its value persists for all subsequent commands until it is specified again. Commands that only contain a keyword and options will only set the states corresponding to those option values and not perform any other action. For example the command:

adjustdistance distance=2.0

does not change any distances, but just sets the value of the distance. A subsequent command

adjustdistance 5 7

adjusts the distance between atoms 5 and 7 to the value specified previously.

The operands are simply anything left over once the keyword and options are removed. The nature of the operands depends on specific commands. It could be a file name, a list of atom numbers, or a string in Maestro Atom Specification Language (ASL) that is used to describe arbitrary sets of atoms based on certain properties.

Some examples of commands are given in the following table.

Command Action/Description
fit The 'fit' command has no options and no operands. It fits all displayed structures to the 3D window.
displaypose pose-name The displaypose command has one operand--a name. The command displays the pose a hold box named pose in the Workspace.
adjustdistance distance=x move={attached|terminal|single} atom1atom2 The adjustdistance command has two options, distance and move, and two operands, atom1 and atom2. The command moves atom2 so that the distance between it and atom1 is the value given by x specified atoms, sets the force constant to the value specified in force_constant_value, and sets the width to the value of halfwidth.

All graphical actions in the GUI cause one or more commands to be recorded by the Maestro GUI. The commands can be saved and played back at a later time or during a later session. See the Command Script Editor Panel topic for more information.
