mtzprint

This utility reads a binary .mtz (CCP4) file and writes out a text version of the file. The command syntax is as follows:

mtzprint -i input-file [-o output-file] [-nref number-of-reflections]

If the output file is not specified, the file is written to standard output. The default number of reflections is all reflections. This utility may be of use for interpreting the column labels: when reading an .mtz file, an assignment of the Fobs and Sigma F properties is required.
