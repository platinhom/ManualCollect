refconvert

This utility converts between various reflection file formats. The command syntax is:

refconvert [options] [MTZ-options] -iinformat input-file -ooutformat output-file

The format string defines the formats. There must be no spaces between the format string and the -i or -o option. For information on the options, run the command with the -h option.
