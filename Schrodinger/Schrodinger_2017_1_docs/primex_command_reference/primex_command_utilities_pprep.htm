Protein Preparation Command for PrimeX

To run the preparation steps that are necessary before a structure can be used by PrimeX, you can use the prepwizard utility, as follows:

$SCHRODINGER/utilities/prepwizard -keepfarwat -nohtreat

These options perform the same treatment as is done from the Input Data panel when you select Assign bond orders. The -keepfarwat option ensures that waters are kept, and -nohtreat ensures that hydrogen atoms are not added.

For more information on this utility, see The prepwizard Utility.
