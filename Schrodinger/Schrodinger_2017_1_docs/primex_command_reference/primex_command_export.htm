PrimeX Keywords for Exporting a Structure to a File

The following keywords are needed to export a structure to a PDB file.

  • PRIMEX_TYPE PDB_EXPORT
  • Calculation settings keywords
  • Reflection data keywords
  • Export keywords, as listed in Table 1

 

Table 1. Export keywords

Keyword

Description

SEQ_FILE

Name of file containing the complete sequence in FASTA format.

OUTPUT_NAME

File name of file to be exported. Must be in Maestro format. Either a relative path or an absolute path can be specified.
