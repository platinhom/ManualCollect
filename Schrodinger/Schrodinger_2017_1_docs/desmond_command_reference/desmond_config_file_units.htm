Units in the Desmond config File

The units used in the configuration file are:

  • time: ps
  • distance: angstroms
  • energy: kcal/mol
  • pressure: bar
  • surface tension: bar angstroms
  • temperature: kelvin

Time can be specified as a positive real number or as the string inf, meaning infinity, or “never”.
