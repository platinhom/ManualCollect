Desmond-Specific Common multisim Keywords

The stages that run Desmond directly (minimize, replica_exchange, lambda_hopping, and simulate) accept a common set of keywords in addition to keywords specific to the Desmond task. Keywords that are supported in the front-end config file (see The Desmond Configuration File) can be used directly in the corresponding stage.
