Assigning Partial Charges

To assign partial charges to the structures in the Workspace based on the current OPLS force field, choose Edit → Assign → Partial Charges. If the structure does not have explicit hydrogens, the partial charges for the implicit hydrogens are added to the partial charge for the heavy atom to which they are implicitly attached.

These partial charges can be used for many purposes, such as displaying the dipole moment of the molecule (from the Workspace Configuration toolbox).
