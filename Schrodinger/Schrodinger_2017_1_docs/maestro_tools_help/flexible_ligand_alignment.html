Flexible Ligand Alignment Panel

This panel allows you to perform a flexible alignment of the entries that are selected in the Project Table. The first selected entry is used as a template, and is kept as is. The remaining ligands are then aligned to this template. The conformation of the structure changes, so you should ensure that you copy the original structures if you want to keep them.

It is also advisable to use well-minimized structures as input to flexible ligand alignment. The structures must include hydrogens—no implicit hydrogens are permitted. The alignment can take 1–2 seconds per entry.

Using the Flexible Ligand Alignment Panel

There are two methods you can choose for the alignment:

  • Flexible shape-based alignment—a ligand torsional search is performed with ConfGen. The conformers generated by ConfGen are then aligned in turn to the reference ligand, and the conformer that has the best overlap with the reference ligand is chosen.

  • Common scaffold alignment—common scaffolds among the ligands are detected, and the ligands are aligned on the scaffold, with conformational variation of the side chains. This option requires a Phase license, as the alignment is performed with the phase_align_core utility. Ligands with no scaffold in common with the template are not aligned. There are three options for the common scaffold, described below.

To align the ligands click Align Selected Entries.

Flexible shape-based alignment option

Perform a ligand torsional search with ConfGen and align the conformers in turn to the reference ligand. The conformer that has the best overlap with the reference ligand is chosen.

Common scaffold alignment option

Locate common scaffolds by one of three methods, listed below, and align the ligands on the scaffold, with conformational variation of the side chains.

Largest common Bemis-Murcko scaffold option

Use the largest common scaffold that contains complete ring systems and the linkers that join them.

Use fuzzy matching option

Treat all non-hydrogen atoms as equivalent, but distinguish bond types (single, double, triple, aromatic).

Maximum common substructure option

Use the largest common substructure found in all the ligands. This is more general than the Bemis-Murcko scaffold, as it is not restricted to rings and linkers.

SMARTS option

Use a specified SMARTS pattern. You can type or paste the SMARTS pattern into the text box, or generate a SMARTS pattern from the Workspace atom selection by clicking Get from Selection.

Get from Selection button

Create a SMARTS pattern from the selected atoms in the Workspace. The pattern replaces the text in the SMARTS text box. You can set preferences for inclusion of various notational elements in the generated SMARTS pattern in the Preferences panel, under SMARTS generation.

Align Selected Entries button

Align the selected entries in the Project Table using the chosen alignment method.
