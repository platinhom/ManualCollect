Setting Glide Redocking Parameters for QPLD

In the Redocking tab, you can choose the docking precision (SP or XP) for redocking of the ligands with updated QM charges, set the van der Waals scaling, set criteria for the elimination of duplicate poses, and enter the maximum number of poses to keep for each redocked ligand. These controls are the same as in the Initial docking tab. The remaining Glide docking options are set to their default values, which includes post-docking minimization.
