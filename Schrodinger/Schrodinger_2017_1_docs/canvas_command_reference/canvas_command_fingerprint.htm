2D Fingerprint Programs

The programs available for 2D fingerprints are listed in Table 1. For a description of the arguments and options, run the command for the program with the -h option.

Table 1. Tools for 2D fingerprints

Tool

Description

canvasFPGen

Generate fingerprints for a set of molecules. Can be distributed over multiple processors.

canvasFPCombine

Combine fingerprints for distinct or overlapping sets of molecules.When a molecule appears more than once, the bit sets are combined using a logical OR. If fingerprints were created using the -scaling option of canvasFPGen, real-valued data for duplicate molecules are averaged.

canvasFPBinary2CSV

Extract fingerprints to a CSV file.

canvasCSV2FPBinary

Convert a CSV file to canvas binary fingerprint file.
