canvasSearch

This program searches a list of target molecules against a set of queries, composed of either molecules or partial structures. This program can also be used to filter a target list based on either standard REOS rules or a user-defined file containing SMILES queries and minimum and maximum number of times a query should be matched. Filtering and searching can be performed separately or sequentially (filtering first).
