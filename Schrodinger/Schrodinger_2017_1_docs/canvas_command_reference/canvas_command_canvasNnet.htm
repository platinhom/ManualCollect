canvasNnet

This program builds and tests an ensemble model of neural networks with data supplied in a CSV (delimited) file. The network has three layers, one input layer, one output layer, and one hidden layer. Networks are trained using a BFGS algorithm. In build mode, a model is developed from a training set, with built-in cross-validation. Optionally, this model can be applied to a test set in the same file as the training set. In test mode an existing model is applied to a test set in its own file.
