Setting Charge and Multiplicity in Jaguar Calculations

Apart from the geometry, the main setting that you might want to make in an otherwise default calculation is to set the molecular charge and the spin multiplicity. You can set these quantities in the Input tab. The default molecular charge is determined by the formal charges on the atoms in the Workspace. The default spin multiplicity is set to 1 (singlet). You can change the charge by entering a value in the Charge column,1Keyword molchg in the gen section. and you can change the spin multiplicity by entering a value in the Spin Mult. column.2Keyword multip in the gen section. If the molecular charge and spin multiplicity settings you make do not agree with your molecular input—for instance, if your molecule has an odd number of electrons and you set the spin multiplicity to 1—Maestro warns you of the inconsistency by default, and you must choose consistent values to submit a job. The charge and multiplicity are added as properties to the output Maestro file and are added to the Project Table when the job is incorporated.
