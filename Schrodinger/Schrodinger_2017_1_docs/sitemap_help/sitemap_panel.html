SiteMap Panel

SiteMap is a tool for finding, visualizing and characterizing protein binding sites. Sites are represented as collections of site points at or near the surface of a protein that are contiguous or are separated in solvent-exposed region by short gaps that could plausibly be spanned by ligand functionality. The visualization uses a grid of points to identify potential hydrophobic and hydrophilic regions; the hydrophilic regions are further classified into hydrogen-bond donor, hydrogen-bond acceptor, and metal-binding regions, and the surface of the protein is contoured. These regions can be displayed in the Workspace.

Key properties, including an overall SiteScore that gauges the propensity of the site to bind ligands tightly, are returned to the Project Table. Other properties measure the exposure of the site to solvent, its degree of enclosure by the protein, the tightness with which the site points interact with the protein ("contact"), the degree of hydrophobicity of the site, the degree of hydrophilicity, the balance between the two, and the extent to which a ligand would be likely to donate, rather than accept, hydrogen bonds ("don/acc"). SiteScore and the contact, phobic, and philic properties are expressed relative to the average value found for a large number of protein sites that bind ligands tightly.

To open this panel, choose Tasks → Structure Analysis → Binding Site Detection in the Task Tool.

Using the SiteMap Panel

To set up a SiteMap calculation:

  1. Display the protein in the Workspace. If you want to find all sites, ensure that only the protein is displayed, without cofactors, waters, or ligands. If you want to map a particular site that is defined by a molecule, include that molecule in the Workspace along with the protein.

  2. Open the SiteMap panel.

  3. Select the task in the Specify task section. If you want to characterize a single region, you must also select a molecule or entry to be used to define the region to be characterized. The atoms of this molecule or entry must not be part of the protein.

  4. Make any settings in the Settings section to define the sites and the output.

  5. Choose Job Settings from the Settings button menu,

    set the job parameters in the Job Settings dialog box, and click Run to run the job.

When the job is finished, the site maps are incorporated and displayed in the Workspace. The Manage Surfaces panel is also opened so that you can manage the display of the site maps.

Task Section

In this section you can choose whether to examine the entire protein for potential binding sites, or restrict the search to a given region.

  • Identify top-ranked potential receptor binding sites option
  • Evaluate a single binding site region option
Identify top-ranked potential receptor binding sites option

Identify potential binding sites on the surface of the receptor, rank them and return the top-ranking sites. The atoms in the Workspace are taken to be the receptor, so you should ensure that only the receptor is displayed. This is the default option.

Evaluate a single binding site region option

This option allows you to evaluate the region of the receptor that is within a specified distance of selected non-receptor atoms, such as a ligand. You can set the distance in the Region about selected atoms plus N Å buffer will be examined text box. The atoms can be selected in the Select non-receptor atoms defining region to evaluate section. You can pick an entry or a molecule using the Pick menu. The identity of the structure you pick is shown in the ASL text box. The X button clears the ASL expression. If you have Show markers selected, the picked atoms are marked in the Workspace. You must have only the receptor and the picked structure in the Workspace, as everything that is not the picked structure is taken to be the receptor.

Settings Section

This section provides basic controls for the site search and display of the site maps.

  • Require at least n site points per reported site text box
  • Report up to n sites text box
  • Use definition of hydrophobicity option menu
  • Use grid option menu
  • Crop SiteMaps at d Å from nearest site point text box
  • Detect shallow binding sites option
Require at least n site points per reported site text box

Set the minimum number of site point required to define a site. The default value is 15 site points; with default settings the average number in examined sites that bind ligands tightly is 150. This minimum applies to the second and subsequent sites; SiteMap always reports at least one site that contains the minimum number of points required to recognize a site, which is 3. This setting is not relevant when you select Characterize a single binding site region.

Report up to n sites text box

Specify the number of sites to report in the output. Sites are ordered by the number of points they contain. The default is 5 sites. This option is not available if Characterize a single binding site region is selected.

Use definition of hydrophobicity option menu

Choose from more restrictive and less restrictive. The more restrictive option excludes points that border on too many assigned philic points, that have too few neighbors that are classified as phobic or inside the protein, or that border on too many points that occupy free space. In addition, regions that are exposed to solvent are assigned reduced hydrophobicity. The less restrictive option includes all points that are initially classified as phobic and does not apply these exclusions and modifications.

Use grid option menu

Set the size of the grid to use in computing the displayed site maps. The choices of coarse, standard, and fine correspond to grid spacings of 1.0 Å, 0.7 Å, and 0.35 Å. This choice has no effect on the grids used for locating and analyzing sites.

Crop SiteMaps at d Å from nearest site point text box

The default distance from the nearest site point at which to crop the individual site maps for display. The default choice is 4 Å. This option does not affect the data, only the display. A different value can be specified later in the Manage Surfaces panel.

Detect shallow binding sites option

Detect shallow binding sites suitable for protein-protein interactions, by decreasing the amount of enclosure and the threshold for van der Waals interactions.

Job toolbar

Manage job submission. See Job Toolbar for a description of this toolbar.

Status bar

The status bar displays information about the current job settings for the panel. This includes the job name, task name and task settings (if any), number of subjobs (if any) and the host name and job incorporation setting. It also displays a message for a short time when you start a new job.

The status bar also contains the Help button, which opens the help topic for the panel.
