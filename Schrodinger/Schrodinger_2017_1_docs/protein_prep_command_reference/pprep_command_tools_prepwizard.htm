The prepwizard Utility

If you know what treatment is needed in advance, you can run the Protein Preparation Wizard from the command line, using the utility prepwizard. The syntax of the command is as follows:

prepwizard [options] [job-options] input-file output-file

The -NJOBS n, option can be used to distribute preparation of multiple proteins. The number of jobs should not be more than the number of proteins in the input file.

The input file can be a single-structure or multi-structure Maestro file (.mae, .maegz, or .mae.gz) or a single-structure PDB file (.pdb or .ent); the output file should be in Maestro format. If you supply a multi-structure Maestro file as input, the script processes all structures in the file.

Note: If you run the Protein Preparation Wizard from the command line, it is especially important that you verify the correctness of the results. Information on the changes made is given in the log file.
