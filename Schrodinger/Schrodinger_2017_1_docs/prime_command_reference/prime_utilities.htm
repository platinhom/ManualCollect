Prime Command-Line Utilities

In addition to the command-line programs described in Prime Command Syntax, Prime provides a number of utilities, which are described in this chapter. These utilities are installed in the utilities directory in your Schrödinger software installation.

To run these utilities:

  • On Linux and Mac, open a terminal window, set the SCHRODINGER environment variable, and prefix the command with $SCHRODINGER/utilities/.

  • On Windows, open a command prompt window with Start → All Programs → Schrodinger-2017-1 → Schrodinger Command Prompt, and enter the command without any prefix.

A summary of the use of these utilities is given in the topics linked below. For a description of the options of any utility, run the utility with the -h option.
