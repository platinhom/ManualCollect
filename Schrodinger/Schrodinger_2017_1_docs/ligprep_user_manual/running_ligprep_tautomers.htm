Generating Tautomers

For some ligand molecules, a tautomer of the commonly-given structure binds to the active site. To generate up to 8 tautomers per input structure, select Generate Tautomers. LigPrep performs keto-enol tautomerization and the analogous sulfur and nitrogen tautomerizations, and histidine and DNA base tautomerizations. For more information, see The tautomerizer Utility .
