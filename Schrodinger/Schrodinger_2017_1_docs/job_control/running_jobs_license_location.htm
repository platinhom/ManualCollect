License File Location

When an executable starts or the queueing software checks the availability of licenses, the following locations are searched, in order, for the license file; the first one found is used.

1. $SCHROD_LICENSE_FILE
2. $LM_LICENSE_FILE
3. $SCHRODINGER/licenses/
4. $SCHRODINGER/license or $SCHRODINGER/license.txt.
5. release-independent-location/licenses/
6. release-independent-location/license or release-independent-location/license.txt

where release-independent-location is a predefined license file location, given below fore each platform. Having a license location that is not inside a software installation allows easier use of licenses that are valid for multiple releases, and is the preferred location for licenses. The location is:

Mac:

/Library/Application Support/Schrodinger/

Windows:

C:\ProgramData\Schrodinger

Linux

/opt/schrodinger
