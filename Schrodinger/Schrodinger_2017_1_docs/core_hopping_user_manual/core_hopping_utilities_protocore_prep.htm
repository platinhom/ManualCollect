protocore_prep: Protocore Preparation

The protocore_prep utility prepares a set of protocores by adding linkers and marking all possible attachment bonds on the protocores. The syntax of the command is as follows:

$SCHRODINGER/utilities/protocore_prep [options] input-file output-file

Both the input file and the output file must be Maestro files. For information on the command options, run the command with the -h option.
