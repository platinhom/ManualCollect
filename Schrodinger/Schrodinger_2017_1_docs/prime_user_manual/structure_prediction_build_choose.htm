Choosing the Model Type for the Protein Structure in PSP

If you are using multiple templates, you must choose the type of model you are building: composite, consensus, homo-multimer, or hetero-multimer. For composite models, you must select the regions from each template that are used to build the structure. The following sections provide more detail on the choices to be made.

If you are using a single template, these choices are not available, and you can proceed to the setting of other options.
