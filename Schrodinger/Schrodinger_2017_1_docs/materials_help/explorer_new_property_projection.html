Materials Explorer — New Property Projection Dialog Box

In this dialog box you can project a property onto a particular clustering of the structures.

New Property Projection Dialog Box Settings

  • Map option menu
  • Property list
  • Property filter text box
  • Statistic option menu
  • Calculate Projection button
Map option menu

Choose the self-organizing map (structure clustering) that you want to project a property onto.

Property list

Choose a property to project onto the self-organizing map.

Property filter text box

Enter a text string to filter the properties by. The property list is restricted to the properties that have a match to the string anywhere in the property name.

Statistic option menu

Choose a statistic to calculate for the structures in each cell, as a representation of the property.

Calculate Projection button

Calculate the projection of the property onto the map, and add it to the projections in the Materials Explorer panel. After the projection is generated, the dialog box closes.
