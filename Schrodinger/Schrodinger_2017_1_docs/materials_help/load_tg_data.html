Load Tg Data Panel

Load data from a Thermophysical Properties run so that it can be analyzed.

To open this panel, choose Tasks → Materials → Classical Mechanics → Thermophysical Properties → Load Tg Data in the Task Tool.

Using the Load Tg Data Panel

This panel is primarily intended to load data for calculation of Tg when for some reason there is no structure available, or the job failed or other results have been lost. The data can then be analyzed to obtain a value for Tg. It can also be used to merge results from multiple Tg runs on the same structure and then load them for analysis.

To load data, it must be associated with a structure. As the structure has no influence on the data or its analysis, it does not matter which structure is used. The procedure is as follows:

  1. Display a structure in the Workspace (any structure will do, but a relevant structure is preferable).

  2. (Optional) Change the density convergence threshold for importing data points.

  3. Click Select Tg Job Directory and navigate to and select the job directory that contains the data.

  4. Click Run.

  5. When the data has been loaded, open the Thermophysical Properties Analysis panel to analyze the data.

Load Tg Data Panel Features

  • Density convergence text box
  • Select Tg Job Directory button
  • Reset button
  • Run button
Density convergence text box

Specify the density convergence cutoff for loading data points. Any data point for which the density convergence does not meet this criterion (i.e. is greater than this value) is not read in.

Select Tg Job Directory button

Select the job directory that contains the Tg data you want to load. Opens a file browser so you can navigate to and select the appropriate job directory.

Reset button

Reset the panel data to its defaults.

Run button

Run the data loading process. When it finishes, a dialog box informs you of the completion of the process, and some standard Tg properties are added to the Project Table. Any problems loading the data are also reported.
