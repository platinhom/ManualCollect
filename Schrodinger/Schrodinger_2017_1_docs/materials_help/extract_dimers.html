Features

Extract Dimers Panel

Extract dimers from a set of molecules that are displayed in the Workspace, and write them to a file or add them to the Maestro project. The set of molecules must be part of a Desmond model system.

To open this panel, choose Tasks → Materials → Build → Extract Dimers in the Task Tool.

Extract Dimers Panel Features

  • Dimer selection options
  • Number of pairs text box
  • Maximum intermolecular distance for pairs text box
  • Store new pairs in options
  • Reset button
  • Run button
Dimer selection options

Select an option for the dimers to extract. Random extracts the number of pairs specified in the Number of pairs text box, chosen at random. Exhaustive extracts all pairs of molecules.

Number of pairs text box

Specify the number of pairs of molecules to extract at random. The total number of pairs is reported to the right.

Maximum intermolecular distance for pairs text box

Specify the maximum distance between two molecules for them to be considered as pairs for extraction. The distance is the smallest distance between any atom in one molecule and any atom in the other molecule. Any pair of molecules for which this distance is larger than the value specified in the text box is not considered when defining the pairs for extraction.

Store new pairs in options

Select an option for storing the pairs. Each pair is stored as a separate entry, with the title taken from the title of the source entry with _dimer_n appended.

  • Project Table—Add the pairs to a new entry group in the Project Table. The group title is the title of the source entry with -dimers appended. The first entry in the group is the full system, which is copied from the source entry. The remaining entries are the dimers.

  • File—Write the pairs to a file. When you click Run, a file selector opens so you can navigate to a location and name the file. The first structure written is the source structure, the remaining structures are the dimers.

Reset button

Reset the panel settings to the default values.

Run button

Run the extraction, and place the results in the selected location.
