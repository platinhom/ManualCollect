Using Features

Restart Workflow Panel

Restart one of the restartable Materials Science workflows, chosen by job type. Restarting can be done on jobs that fail due to problems such as resource, hardware, or network issues.

To open this panel, choose Tasks → Materials → Restart Workflows in the Task Tool.

Using the Restart Workflow Panel

To restart a job:

  1. Choose the type of job from the Job type option menu.

  2. Choose an option for locating the job:

    • Load from job database allows you to choose from the job database.
    • Manually locate job directory allows you to browse the file system for the directory containing the job files.

  3. Select the job you want to restart:

    • If you chose Load from job database, choose the job from the table below.
    • If you chose Manually locate job directory, enter the full path of the job directory in the Path text box, or click Browse and navigate to the job directory.

  4. Click Run to restart the job, or click the Job Settings button to change the job host or settings and restart the job.

Restart Workflow Panel Features

  • Job type option menu
  • Load from job database option
  • Job table
  • Manually locate job directory option
  • Path text box and Browse button
  • Job toolbar
  • Status bar
Job type option menu

Choose the type of job (workflow) that you want to restart.

Load from job database option

Choose the job to restart from the list in the job table below, which is loaded from the job database.

Job table

Select the row for the job you want to restart from this table. Only one row can be selected, as each job must be started individually.

Manually locate job directory option

Choose the job to restart by specifying the job directory.

Path text box and Browse button

Enter the full path to the job directory of the job you want to restart in this text box, or click Browse and navigate to the job directory. The job directory name is usually the same as the job name. If the job directory is incompatible with the chosen job type, a warning is displayed when you try to restart the job.

Job toolbar

Manage job submission. See Job Toolbar for a description of this toolbar.

Status bar

The status bar displays information about the current job settings for the panel. This includes the job name, task name and task settings (if any), number of subjobs (if any) and the host name and job incorporation setting. It also displays a message for a short time when you start a new job.

The status bar also contains the Help button, which opens the help topic for the panel.
