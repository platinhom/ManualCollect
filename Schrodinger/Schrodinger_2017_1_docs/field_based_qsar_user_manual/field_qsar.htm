Field-Based QSAR

Field-Based QSAR allows you to build 3D QSAR models based on fields, such as electrostatic, hydrophobic, or steric fields, for a set of aligned ligands. The models can be applied to other ligands, or stored with a pharmacophore hypothesis, or exported for later use. For a tutorial introduction, see Field-Based QSAR Tutorial.

A related method is atom-based QSAR, which allows you to build 3D QSAR models based on the spatial distribution of atom types or pharmacophore types. See Building QSAR Models from Ligands for more information on these models.
