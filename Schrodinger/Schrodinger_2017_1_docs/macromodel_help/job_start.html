MacroModel — Start Panel

The Start panel is used to start MacroModel calculations using command and structure files that were prepared previously.

To open this panel, choose Applications → MacroModel → Start Job From File in the Task Tool.

Using the Start Panel

The Start panel can be used to run jobs from previously created command (.com) and structure (.dat) files.

To start a job, select it in the Available jobs in current project list and click Start. The Monitor panel opens, and the job's progress is displayed.

Start Panel Features

  • Use structure from option menu
  • Available jobs in current project
  • Update List
  • Job toolbar
  • Status bar
Use structures from option menu

Choose the source of the structures for the current task.

  • Project Table (selected entry)—Use the entry that is currently selected in the Project Table. Only one entry must be selected.
  • Workspace (included entry)—Use the entry that is currently included in the Workspace. Only one entry must be included in the Workspace.
Available jobs in current project

A list of accessible jobs is displayed in the Available jobs in current project text area. The list includes all .com/.dat file sets in the current directory that do not belong to jobs which have yet to be incorporated into the project they were started from.

Update List

This button revises the Available jobs in current project list to include recently created .com files pairs, files from a job that was running when the panel was first opened, or files in a new directory.

Job toolbar

Manage job submission. See Job Toolbar for a description of this toolbar.

Status bar

The status bar displays information about the current job settings for the panel. This includes the job name, task name and task settings (if any), number of subjobs (if any) and the host name and job incorporation setting. It also displays a message for a short time when you start a new job.

The status bar also contains the Help button, which opens the help topic for the panel.
