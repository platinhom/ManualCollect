MacroModel Panel

This topic provides an overview of the MacroModel panels, as well as how the panels function. Each of these MacroModel panels is focused on a particular task or type of calculation. All MacroModel panels have common features, which are described here.

Using the MacroModel Panel

Molecular mechanics calculations in Maestro can be carried out by MacroModel. The calculations are set up and run from various panels that can be opened from the MacroModel submenu of the Applications menu.

Note: Some types of jobs (such as minimization and coordinate scans) are restricted to taking input from a single entry. Make sure that only one entry is displayed in the Workspace or selected in the Project Table and then start the job.

After writing out and editing job input, you can run the jobs by choosing Applications → MacroModel → Start Job From File. Select a job in the list and click Run, or make job settings and start the job (see above). This mechanism can be used to create job files that can be edited to take advantage of MacroModel features that are not accessible from Maestro.

Only one MacroModel panel can be active at any one time. If you choose an item from the MacroModel submenu of the Applications menu while another MacroModel panel is active, the existing panel is dismissed before the new one is displayed.

MacroModel Panel Features

  • Use structures from option menu
  • Tabs
  • Job toolbar
  • Status bar
Use structures from option menu

Choose the source of the structures for the current task. The items depend on whether the panel takes a single structure or multiple structures as input.

  • Project Table (selected entries)—Use the entries that are currently selected in the Project Table.
  • Workspace (included entries)—Use the entries that are currently included in the Workspace, treated as separate structures. The Workspace should not contain a scratch entry.
  • Workspace (included entry)—Use the entry that is currently included in the Workspace. Only one entry must be included in the Workspace.
Tabs

Controls that are specific to a MacroModel panel are contained in a series of tabs. Each tab contains settings relevant to the task that the panel performs. The Potential tab defines the potential energy functions to be used (force field, solvation model, etc.). There are also tabs for setting positional constraints on atoms (Constraints) and geometric parameters (Substructure). Finally, there are tabs that are specific to the panel itself. These tabs are dsecribed in topics for each panel.

Job toolbar

Manage job submission. See Job Toolbar for a description of this toolbar.

Status bar

The status bar displays information about the current job settings for the panel. This includes the job name, task name and task settings (if any), number of subjobs (if any) and the host name and job incorporation setting. It also displays a message for a short time when you start a new job.

The status bar also contains the Help button, which opens the help topic for the panel.
