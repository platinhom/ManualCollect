New Caption/Edit Caption Dialog Box

In this dialog box you can add a caption to the Workspace, and place it where you want. The same dialog box is used to edit captions, with the name Edit Caption. For information on using captions, see the topic Captions.

To open this dialog box, choose Workspace → Add Caption.

New Caption/Edit Caption Dialog Box Features

  • Workspace caption text area
  • Background option menu
  • Font option menus and buttons
  • Color patch and Edit button
Workspace caption text area

Enter the text of the caption in this text area. Line breaks in the caption text are preserved when the caption is displayed.

Background option menu

Choose an option for the background.

  • Automatic—set the background color automatically, based on the color of the text.
  • No fill—make the background transparent.
Font option menus and buttons

Choose the font, point size, and style for the text. The B and I buttons change the text to bold face and italic, respectively.

Color patch and Edit button

Choose a color for the caption by clicking Edit. The color is displayed in the patch to the left of the Edit button.
