SiteMap Hydrophilic Map

SiteMap constructs a measure of hydrophilicity by adding an “electric-field reward” term to the van der Waals energy:

Grid_philic = vdW_energy + oriented-dipole_energy

where the oriented-dipole energy is necessarily negative. Hydrophilic regions are those within which the sum of the two terms is more negative than a given threshold, which by default is −8 kcal/mol.
