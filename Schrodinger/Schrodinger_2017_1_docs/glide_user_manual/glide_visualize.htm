Visualizing Glide Docking Results

Several tools are provided to help visualize the results of Glide docking runs. The View Poses facility in the Project Table panel enables you to display the ligand poses with the receptor in the Workspace, along with hydrogen bonds, bad and ugly contacts, and per-residue interaction information. This facility is described in Viewing Poses.

For Glide SP and XP docking runs, you can visualize the contributions to the XP docking score, provided that descriptor information was requested in the docking run. Visualization in the Workspace is done using the Glide XP Visualizer panel, which is described in The Glide XP Visualizer.

See Examining Poses for 1fjs for a tutorial introduction to visualizing Glide docking results.
