Selection of Receptor Partial Charges

You can use partial charges from the input structure, rather than from the force field, by selecting Use input partial charges. This option is useful if, for example, you have obtained improved partial charges around the active site, such as those from a QSite calculation or a QM-Polarized Ligand Docking calculation.
