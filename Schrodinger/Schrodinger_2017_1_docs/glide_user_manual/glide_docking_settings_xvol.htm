Applying Excluded Volumes to Ligand Docking

For a tutorial example, see Using Excluded Volumes in Ligand Docking.

If the grid that you chose contains excluded volume data, you can choose to apply the excluded volumes when docking the ligands, and set the penalty for occupying the excluded volume. To do so, select the option Apply size excluded volume penalties, and choose a size for the penalty. The sizes available are large, medium, and small, which correspond to particular choices of the penalty value. The penalty ramps up linearly from zero at the edge of the sphere to the value specified, over a shell of width 10% of the sphere radius, and remains at that value over the rest of the sphere. Each atom that occupies the excluded volume incurs the penalty, and the penalty is applied in both the rough scoring stage and the final docking. You can also view the excluded volumes in the Workspace, by selecting Show excluded volumes. The excluded volumes controls are not available if the grid does not have excluded volumes.
