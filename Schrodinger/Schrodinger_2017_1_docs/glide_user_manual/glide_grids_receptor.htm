Setting Receptor Parameters for Grid Generation

In the Receptor tab you define the part of the Workspace system for which receptor grids should be calculated. You can also scale receptor atom van der Waals radii in this tab, and choose whether to use partial charges from the force field or from the input structure. The tab has three sections, Define receptor, Van der Waals radii scaling, and an unlabeled section with several options in it. Some more advanced settings that are not commonly needed are in a dialog box, which you open by clicking Advanced Settings.
