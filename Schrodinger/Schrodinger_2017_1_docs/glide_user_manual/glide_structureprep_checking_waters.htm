Checking the Orientation of Water Molecules

You only need to perform this step if you kept some structural waters. Reorienting the hydrogens is not strictly necessary, as their orientation should have been changed during refinement, but it is useful to check that the orientation is correct.

If the orientation is incorrect, reorient the molecules by using the procedure outlined in Correcting Orientations.

When you have corrected the orientation of the retained water molecules, you should run a refinement on the adjusted protein-ligand complex.
