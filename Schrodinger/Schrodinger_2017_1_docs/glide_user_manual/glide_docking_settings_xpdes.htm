Writing XP Descriptor Information

If you want to visualize the various scoring terms from the XP scoring function in the XP Visualizer panel, select Write XP descriptor information. These terms are written to a file that can be read by the XP Visualizer. You can write these terms only for XP docking runs.

Note: To generate XP descriptor information, you must have a license for this feature.
