Making Basic Settings For Glide Docking

For a tutorial example, see Setting Basic Docking Options.

The Settings tab defines the basic options for docking ligands: selecting the precision, setting flexibility options. You can also make settings for the selection of initial poses and for the energy minimization of the poses that pass the initial selection in the Settings - Advanced Settings dialog box. To open this dialog box, click Advanced Settings.
