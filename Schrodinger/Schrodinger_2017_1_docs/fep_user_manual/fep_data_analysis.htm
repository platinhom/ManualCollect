FEP Data Analysis

Once problematic edges have been removed in the FEP Mapper Analysis Panel, we would like to understand why a compound is predicted to be more or less potent. There are several ways of doing this.

Visualizing Trajectories from the Simulations

  1. Navigate to the folder jobname_6/trajectories_files.

  2. Untar the tarball for the edge of interest. The trajectories in each folder are:

    • Lambda0 for the complex: jobnamecomplex_10_replica0-out.cms
    • Lambda11 for the complex: jobnamecomplex_10_replica11-out.cms
    • Lambda0 for the solvent: jobnamesolvent_11_replica0-out.cms
    • Lambda11 for the solvent: jobnamesolvent_11_replica11-out.cms

  3. Import the -out.cms files for lambda 0 and lambda 11 (solvent or complex) into Maestro.

  4. In the Entry List panel or the Project Table panel, double-click the blue T button to open the trajectory viewer.

  5. To center the trajectory, you have the following options:

    • Under Positioning, select one of the following:

      • Superimpose on Frame, enter 0
      • Center molecules

    • Under Pick atoms for frame position, do one of:

      • Select the ligand
      • Select the ligand and residues within 5 Å

    • Use the Atoms to display in each frame to allow you to focus on the ligand in the binding site, e.g.

      1. Select the ligand and residues within 5 Å.
      2. Select Display only specified atoms.

    See Trajectory Panel for more information on setting up and playing the trajectory.

  6. Look at the extent of sampling, e.g. is a ring flipping, are the correct contacts being observed?

  7. You can also go back to your WaterMap results and see if the potency changes can be explained for your ligand series.

  8. You can also compare the WaterMap water positions to those in the FEP MD trajectory.

Using Simulation Interaction Diagrams for Edge End-Point Analysis

You can check the stability of the ligand and the protein during the simulations with the help of simulation interaction diagrams. These diagrams are available for each edge in the map, and show an analysis of the end points of the perturbation.

  1. Open the FEP Mapper Analysis Panel. (You can do this from the command line with $SCHRODINGER/run -FROM scisol fep_leadopt_review.py.)
  2. Import a filename_out.fmp file into the panel.
  3. Right click on an edge of interest.
  4. Select Analyze, to open the Analysis: ligand1 <--> ligand2 Panel.
  5. Scroll through the summary, protein-ligand interactions, ligand details and protein details.
  6. Save a PDF for the details of this edge if you wish.
