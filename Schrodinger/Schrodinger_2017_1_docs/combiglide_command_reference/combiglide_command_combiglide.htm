Combinatorial Screening

You can run combinatorial screening from the command line with the combiglide command. The syntax is:

$SCHRODINGER/combiglide [options] jobname

The best way to run combinatorial screening manually is to create input files using Maestro, modify them as needed, and use the above syntax to launch a job.
