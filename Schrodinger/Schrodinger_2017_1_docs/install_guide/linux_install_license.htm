Licensing the Software on Linux

Installation of the license can be done with the Configure Schrodinger Software application. You can copy an existing valid license, install a new license, or request a new license with this application. If you don’t have a license already, or have not been sent a license key, go to http://www.schrodinger.com/licensing for more information.

The license must be made available on each machine on which you want to run the software. This can be done by installing the license on individual machines, or installing the license on a license server machine, then configuring access to the license server for each machine used to run the software. Either way, you will need to run the Configure Schrodinger Software application from each installation to set up licensing.1If you want to install the licenses from the command line, you can do so with the licadmin command.

To start the Configure Schrodinger Software application, enter the following command:

$SCHRODINGER/utilities/configure

You will need to set the SCHRODINGER environment variable first (see Setting the Environment Variables on Linux). When the application starts, you can use the guidance given in the panel, or click the Help button for instructions.
