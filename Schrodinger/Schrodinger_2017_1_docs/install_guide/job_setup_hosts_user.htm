The user Setting in the Hosts File

If you have different user names on the submission and execution hosts, you can include a user setting for the execution host in the hosts file on the submission host. The user setting should never be added to entries in the hosts file in the installation directory, because this would prevent other users from using those entries. If a user setting is required, the user should copy the hosts file to $HOME/.schrodinger (Linux or Mac) or %USERPROFILE%\Schrodinger (Windows) on the job submission hosts and add the user settings to this copy.
