Linux Requirements

Basic requirements:

  • If using NFS, file locking must be enabled.

RedHat/CentOS requirements:

  • The nfs-utils package must be installed.

    yum install nfs-utils

Ubuntu requirements:

  • The mesa-utils and lsb-core packages must be installed:

    sudo apt-get install mesa-utils
sudo apt-get install lsb-core

    If you are serving Schrödinger licenses from an Ubuntu machine, you must install the lsb-core package on that machine, even if you do not have any Schrödinger software installed on it.

  • Ubuntu Server also requires some X11 libraries:

    sudo apt-get install xserver-xorg libgl1-mesa-glx libfontconfig1
