OpenMPI Setup for LSF

Loose integration for LSF should work without change. Tight integration is supported by LSF from version 7.0.2 on. It is generally recommended to use a single node.

Here is an example of a hosts file entry:

# Request %NPROC% processors on a single node
Name: label
Queue: LSF
Qargs: -q LSF-queue -n %NPROC% -R \"span[hosts=1]\"
Host: submit-node
Processors: processors-in-queue
