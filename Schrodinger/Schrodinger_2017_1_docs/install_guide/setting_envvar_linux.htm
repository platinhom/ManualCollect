Linux

To set an environment variable on Linux, enter the following command at a shell prompt:

csh/tcsh:

setenv variablevalue

bash/ksh:

export variable=value

where variable is the name of the environment variable (such as SCHRODINGER) and value is the value you want to assign to the variable, (such as /opt/schrodinger2017-1).
