Setting Up the Display for 1bel

With a large number of atoms in the Workspace, it is difficult to see specific changes. To best view the changes in the orientation of Tyr 115, the display needs to be set up. Tyr 73 is close to Tyr 115, and starts with a hydrogen bond to it which is broken and briefly reformed during the simulation. You might not see this hydrogen bond when you are setting up the display.

  1. Include the 1bel_md2-out entry, using the Entry List panel.

  2. Click the Define button on the Select toolbar.

    The Atom Selection dialog box opens.

  3. In the Residues tab, choose Residue number from the list.

    Tools for selecting by residue number are displayed in the center of the panel.

  4. Enter 115 in the Residue number text box and click Add.

  5. Choose Residue type from the list.

  6. Choose TYR from the Residue type list and click Intersect, then click OK.

    The Atom Selection dialog box closes, and Tyr 115 is the only residue selected in the Workspace.

  7. In the Style toolbox, click the Ball & Stick button.

  8. Click the Expandbutton on the Selection toolbar, and choose Use Custom Radius.

  9. In the dialog box that is displayed, enter 10.0 and click OK.

    The selection expands to include all residues within this distance from Tyr 115.

  10. Click the Remove from Selection button.

  11. Click the Choose item to select button on the Select toolbar and choose Waters.

    The waters are removed from the selection.

  12. Click the Display selected atoms only button.

    Tyr 115 and the residues around it are displayed, but not the waters.

  13. Click the Remove from Selection button again, to return to normal selection.

  14. Click the Set selection scope button and choose Residues.

  15. Pick an atom in Tyr 73.

    The residue name and number are displayed in the Status bar when you pause the cursor over an atom, so you can check that you have the correct residue before you pick. You could also use the Find tool or the sequence viewer to find Tyr 73.

  16. In the Style toolbox click the Ball & Stick button.

  17. Move the cursor to the Interactions button on the Workspace Configuration toolbar, and click the button that appears above it.

    The Interactions toolbox opens.

  18. Click the Noncovalent bonds button, then choose Other from the menu to the right.

    The Edit H-Bonds Definition dialog box opens.

  19. In the Atom set 1 section, choose Residues from the Pick menu, and pick an atom in Tyr 115.

  20. In the Atom set 2 section, choose Residues from the Pick menu, and pick an atom in Tyr 73.

  21. Click OK.

    The hydrogen bond (if it is present) is displayed as a yellow dashed line.

  22. Click the Fit to Workspace button on the Fittoolbar.

    The view zooms out so that the displayed residues occupy most of the Workspace. You might also want to rotate the structure to get a better view of Tyr 115. Note that any changes you make to the display are applied throughout the playing of the trajectory.

  23. Repeat the above instructions for the other trajectory, 1bel_md1-out.
