Importing Results

In the next two exercises, you will examine the results of the calculation. If you decided not to run the job, you can import the results from your working directory, as follows:

1. Choose File → Import Structures.

The Import panel opens.

2. Choose Maestro from the Files of type option menu.
3. Select the file qsite-1tpb.01.mae.
4. Click Open.
