Defining the Core Poses

The final setup task is to determine how the core poses will be generated in the first stage of the docking step. The core poses can be generated with any molecule that includes the core, not just the structure you selected to define the core in the Define Combinations step. This task is carried out in the Define Core Poses step, and involves selecting a docking mode and specifying the structures that will be docked.
