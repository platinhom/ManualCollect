Enumerating the Library

When you have defined a combinatorial library, you can then proceed to enumerate it. When you click Enumerate, the Start dialog box opens, and you can set job options and start the enumeration job.

The results can be appended to the Project Table or written to the current directory. The default is to append the results to the Project Table.
