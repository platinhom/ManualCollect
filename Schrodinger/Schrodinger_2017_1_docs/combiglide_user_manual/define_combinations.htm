Combinatorial Screening: Defining the Core and Its Attachments

The combinatorial screening process begins with the definition of the core and its attachments. The core-containing structure must be a 3D, all-atom, minimized structure. Likewise, the reagents used for the attachments must be 3D, all-atom, minimized structures, and also include information that defines the side chain to be attached to the core. Preparation of the core-containing molecule and the reagents is described in Combinatorial Screening: Preparing Structures. The tasks of selecting the core-containing molecule, defining the attachment points, and associating a set of reagents with each attachment point are carried out in the Define Combinations step of the Combinatorial Screening panel.
