Combinatorial Screening: Preparing Structures

When you have decided on the biological target and the synthetic route to the library you want to generate, and obtained structure files for the reagents and the receptor, you can proceed to the preparation of the structures for CombiGlide. Both the protein structure and the reagent structures must be properly prepared. You can perform these tasks in the Protein Preparation Wizard and Reagent Preparation panels. Following protein preparation, you must set up grids for Glide docking, which you can do in the Receptor Grid Generation panel.
