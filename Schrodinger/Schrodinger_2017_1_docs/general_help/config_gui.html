Configure Schrodinger Software Panel

Configure access to Schrödinger software: install and set up access to licenses, start and stop the license server.

This panel is intended for use before you start Maestro or run jobs. To open this panel, do the following:

  • Linux: Enter $SCHRODINGER/utilities/configure in a terminal window.

  • Windows: Choose Start → All Programs → Schrodingerrelease → Configure, or enter configure in a Schrodinger Command Prompt window.

  • Mac: Choose Applications → Configure.app, or enter $SCHRODINGER/utilities/configure in a terminal window.

You can also open this panel from Maestro, by choosing Help → Configure Licensing and Hosts.

Setting up new licenses on a single computer

To set up the licensing, you will need the key which was sent to you by email. If you did not receive a key, contact your sales representative. You will need access to the web so that the license corresponding to your key can be downloaded.

Note: this computer is set up as a license server. If you are only using this computer, it is its own license server.

  1. Check that I have a key is selected for Add Licenses.

  2. Copy the key and paste it in the text box below.

  3. Click Install License.

    The Install Licenses on Host dialog box opens, with information on the current host. You do not usually need to change anything.

  4. Click OK.

    The license is downloaded and installed. The License Successfully Installed dialog box opens.

  5. Ensure that Start the license server process now is selected.

  6. Click OK.

    The license server is started on the local host, and the licenses are ready to use.

Setting up new licenses on a license server and the computers it serves

First, you will need to install the license on the license server, then add access to licenses from the other computers.

  1. Log on to the computer that you want to use as a license server, and open this panel.

  2. Follow the instructions for installing a license on a single computer, making a note of the LMGRD Port value in the Install Licenses on Host dialog box. GO

For each computer that you want to run software on:

  1. Log on to the computer and open this panel.

  2. For Add Licenses, choose I can identify my license server.

  3. Enter the host name of the license server (the fully qualified domain name) in the Host Name text box, e.g. myhost.mycompany.com

  4. Enter the LMGRD Port value from the license server installation process into the Port text box.

    If you didn't make a note of the port, you can probably leave this box empty and it will use the default value.

  5. Click Save Server.

    A client license is created and installed, and a dialog box opens informing you of the success of the procedure.

To set up the licensing, you will need the key which was sent to you by email. If you did not receive a key, contact your sales representative. You will need access to the web so that the license corresponding to your key can be downloaded.

Note: the license can only be installed on the specific computers for which they were obtained.

  1. Check that I have a key is selected for Add Licenses.

  2. Copy the key and paste it in the text box below.

  3. Click Install License.

    The Install Licenses on Host dialog box opens with information about the licenses.

  4. Click OK.

    The license is downloaded and installed. The License Successfully Installed dialog box opens.

  5. Click OK.

    The license is installed and ready to use.

Using existing licenses

On Mac computers, an existing license is detected automatically, so you should not have to do anything. The instructions below apply to either a server or a client computer.

  1. For Add Licenses, choose I have a license file.

  2. Enter the path to the license file in the text box, or click Browse to locate it.

    The license file for a previous release is by default at the top level of the installation directory. This location is detected automatically on Windows.

  3. Click Install License.

    The Install License File on Host dialog box opens. If the computer is a server machine (or single computer), information on the server is displayed. If the computer is a client machine, the server location is displayed.

  4. Click OK.

    The License File Successfully Installed dialog box opens.

  5. If the computer is a server, ensure that Start the license server process now is selected.

  6. Click OK.

    The license is installed and ready to use. If the computer is a server, the server is restarted.

Current Status section

When the panel opens, a check is run to identify the licenses you have access to from the machine you opened the panel on (the current host). A report is displayed in the Current Status section of the tab. This report tells you whether you have licenses installed, what kind of licenses they are, if the license server is running (if applicable). There are also buttons to view details of the licenses and to start and stop the license server, which are displayed when appropriate.

Add Licenses option menu

The tools in the lower half of the panel allow you to set up licensing. There are several ways of adding licenses to a machine, depending on the situation. The options for adding licenses are available from the Add Licenses option menu:

  • I have a key

    This is the default option if you don't have valid licenses or don't have licenses at all and need to install new ones. When you purchase licenses or start an evaluation, you are sent a key by email. Copy and paste the key into the text box below, then click Install License.

    Note: The current host is set up as a license server when you use this method of installation, unless you have licenses for specified machines (node-locked licenses). This is the normal scenario if you are doing an installation on a single machine, and don't have a special license server host. However, if you do have a license server host, you should ensure that you open this panel on that host to do the license installation.

    The panel that opens when you click Install License gives information about your computer name, address, and the license server ports used. You do not need to do anything with this information unless you need to change the ports; this is usually only necessary if you are managing licenses on a license server host. Click OK in this panel to install the licenses. The licenses are downloaded from the web and license files are created and installed.

  • I have a license file

    This option can be used if you already have valid licenses from a previous release and want to use them for a new release. On Mac computers, the license is in a standard location and is automatically detected.

    To use the previous license, enter the name of the license file in the text box, or click Browse to navigate to the license file. On Windows, the license file from the most recent previous release is detected. Click Install License to install the license file as the license for the current release.

  • I can identify my license server

    This option can be used if you have installed licenses on the server and now want to make them available on the computers that are served by the server (the clients). It does not matter whether the licenses are new or from a previous release.

    Enter the host name of the license server in the text box, e.g. myserver.mycompany.com. If you know the port number (this is the LMGRD port shown when you installed the licenses on the server) you can enter it, otherwise the port will be detected when connecting to the server. Click Save Server to save the server information, which is placed in a "stub" license file.

  • I need to request a license

    Use this option if you don't have a license key or a license file at all. Information on obtaining a license is displayed, with a link to the licensing web page. Usually, requesting a license is done by contacting a sales representative.

Once the licenses are installed, you may need to start (or restart) the license server process if you are installing the licenses on a server. Ensure that Start the license server process now is checked in the License Successfully Installed dialog box, and click OK. If you uncheck this box, you can start the license server process by clicking Start Process in the Current Status section.

License source specification section

This section displays the tools for adding licenses from different sources. The tools depend on the choice made from the Add Licenses option menu, and are described above.

Tech Support button

Open the Contact Support web page, where you can submit a request for technical support. If you need to attach diagnostic information, you can generate the information in the Diagnostics panel.
