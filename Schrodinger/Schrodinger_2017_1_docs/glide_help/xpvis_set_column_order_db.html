Set Column Order Dialog Box

This dialog box allows you to set the order of the columns in the Ligands table of the Glide XP Visualizer, either manually or in order of variance. The order of the columns is changed when you click OK.

To open this dialog box, click Set Column Order in the XP Visualizer panel.

Set Column Order Dialog Box Features

  • Column list
  • Move Selected Columns buttons
  • Reset to Original Order button
  • Set Order to Variance button
Column list

List of columns in the table, with the variance of the data in the columns. You can select multiple entries in this list for reordering.

Move Selected Columns buttons

The four buttons move the columns that are selected in the list as follows:

  • Top—Move the selection to the top of the list.
  • Up—Move the selection up one place in the list for each click. If a discontiguous range has been selected, the spacing between the parts of the range is preserved.
  • Down—Move the selection down one place in the list for each click. If a discontiguous range has been selected, the spacing between the parts of the range is preserved.
  • Bottom—Move the selection to the bottom of the list.
Reset to Original Order button

Reset the list to the original order, before any changes in order were made.

Set Order to Variance button

Order the list by decreasing values of the variance.
