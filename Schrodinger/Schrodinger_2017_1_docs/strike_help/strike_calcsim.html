Using Features Links

Calculate Similarity Panel

Use this panel to determine similarities in descriptor or 2D-structure space. Several distance-based descriptor similarity measures are available. Similarity in 2D-structure space is determined using an atom-pair-based approach.

To open this panel, choose Tasks → Discovery Informatics and QSAR → Similarity in the Task Tool.

Using the Calculate Similarity Panel

Similarity, either atom-pair or descriptor, is calculated with respect to probe molecules. Specify probe molecules by including them in the Workspace. In the Project Table, select (highlight) the entries you want to calculate similarity for. At least one molecule must be included in the Workspace, and at least one entry must be selected in the Project Table, before similarity calculations can proceed. Choose options for the calculation, then click Run to run the job.

Calculate Similarity Panel Features

  • Similarity will be calculated for the N selected entries
  • Calculate options
  • Use Descriptors table
  • Adjust Weight button
  • Job toolbar
  • Status bar

Similarity will be calculated for the N selected entries

The number of entries selected in the Project Table is displayed. Similarity will be calculated for the selected entries only.

Calculate options

The options for similarity calculations are:

Atom pair similarities

A similarity property will be created for each selected entry in the Project Table.

Descriptor similarities

Similarity in terms of properties will be calculated for the selected Project Table entries in terms of the properties chosen from the Use Descriptors scrolling list.

Use Descriptors table

This list is available when descriptor similarities are selected to be calculated. Choose the properties you want to include in the similarity calculation.

Adjust Weight button

Opens the Adjust Weight dialog box, in which you can set the weights for the selected descriptors. Only available if you select Descriptor similarities.

Job toolbar

Manage job submission. See Job Toolbar for a description of this toolbar.

Status bar

The status bar displays information about the current job settings for the panel. This includes the job name, task name and task settings (if any), number of subjobs (if any) and the host name and job incorporation setting. It also displays a message for a short time when you start a new job.

The status bar also contains the Help button, which opens the help topic for the panel.
