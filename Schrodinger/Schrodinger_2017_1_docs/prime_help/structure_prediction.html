Using Features

Structure Prediction Panel

Predict the structure of a protein from its sequence by homology modeling.

To open this panel, choose Tasks → Protein Preparation and Refinement → Homology Modeling in the Task Tool.

Using the Structure Prediction Panel

The Structure Prediction workflow is presented as a sequence of steps, each of which provides controls for a particular task in the sequence: importing sequences, finding homologs, performing alignments, and finally building a structure.

The worflow allows you to try out many choices of templates, options, alignments. Each such choice can be stored in a Prime run. For more information on runs, see Using Prime Runs.

Different choices can be made by changing the settings in the current step or navigating back to a previous step. If you want to preserve the results at the current point, you can save the run then make the changes.

If you have gone back to a previous step and changed your selections, when you proceed to the next step you are presented with three choices about what to do with the changes:

  • If you want to proceed with the new selections and clear the original data in the subsequent steps of the current run, click Overwrite.
  • If you want to discard the changes you made in the previous step, and continue to the next step without changing the original data, click Ignore Changes.
  • If you want to go back to the previous step and not proceed forward, click Cancel.

Structure Prediction Panel Features

Each step in the Structure Prediction panel has a common set of features as well as features that are unique to the step. In the upper part of the panel are the menu bar, the toolbar, and the Prime sequence viewer. The center of the panel contains the Step View, which includes the step name and a brief description of the step, followed by some buttons, and data relevant to the step. Below the Step View are the navigation controls for navigating between steps.

Step title

Title of the step, followed by a brief description of the step function.

Job Options button

Set the host for the jobs that are run from the workflow. Opens the Structure Prediction - Job Options dialog box.

Job status button

The job status icon to the right of the Job Options button spins when you launch a job. When the icon stops spinning, the job is finished. Click the icon to open the Monitor panel to monitor the job.

Step View

The step view contains buttons for performing step actions, such as importing files, running jobs, setting options.

The data relevant to the step is usually shown in table format. You can view the complete text of a truncated table entry as a tool tip by pausing the pointer over the text. To resize a table column, drag a column border. You can sort the tables by clicking the column heading, once for ascending order, twice for descending order. You can also export data from any table in CSV or HTML format from the table shortcut menu.

Navigation controls

Navigation between steps can be performed with the Back and Next buttons, or with the Guide.

The Guide is a diagram of the Prime workflow as a series of steps, each step represented by a button which you can click to go to that step. The current step is highlighted in the Guide, and all previously accessed steps, forward or backward, are available. A forward step is available if the criteria for using it have been met. Steps that are unavailable are dimmed.

The Guide is hidden by default. You can display it from the Step menu.

Note: You cannot launch searches and jobs by clicking the steps in the Guide. To launch actions pertaining to a step, click the appropriate buttons within the step view.
