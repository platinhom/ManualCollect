Aligning Grids for VSW

If you are generating grids, you can align the receptors before the grids are generated, using the Protein Structure Alignment tool. To do so, select Align receptors. This option does not align receptors for pregenerated grids, only those that are generated by the current VSW run. The receptors are aligned to the frame of reference of the first receptor in the table.
