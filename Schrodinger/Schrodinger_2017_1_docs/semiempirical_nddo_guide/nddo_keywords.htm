Keywords Supported in the Semiempirical NDDO Module

The keywords that are supported in the Semiempirical NDDO module for use with the -keywords command option are given in this section. Keywords are not case-sensitive. The keywords listed below are the only keywords that are supported with this module. Any other keywords that exist in the original code might work, but you use them at your own risk, as they might not give the results you expect.

For a full list of supported keywords for MOPAC2012, go to http://openmopac.net/manual/allkeys.html.
