Run Input Files Panel

In this panel you can select a number of Jaguar input files and run them as a batch job.

To open this panel, choose Applications → Jaguar → Run Input Files in the Task Tool.

Using the Run Input Files Panel

To select an input file, click Browse and choose a file in the file selector that is displayed. Repeat this process for as many input files as you want to use. The files are listed in the Input files list. If you want to delete a file, select it in the list and click Delete.

When you have finished selecting files, click Run, or choose Job Settings from the Settings button menu to open the standard Maestro Job Settings dialog box. The incorporation options are not available because the input files are not necessarily associated with a project.

Run Input Files Panel Features

  • Input files list
  • Browse button
  • Delete button
  • Job toolbar
  • Status bar
Input files list

Displays the list of input files selected. You can select a file in this list and delete it with the Delete button.

Browse button

Opens a file selector in which you can select an input file to add to the list.

Delete button

Deletes the selected file from the list

Job toolbar

Manage job submission. See Job Toolbar for a description of this toolbar.

Status bar

The status bar displays information about the current job settings for the panel. This includes the job name, task name and task settings (if any), number of subjobs (if any) and the host name and job incorporation setting. It also displays a message for a short time when you start a new job.

The status bar also contains the Help button, which opens the help topic for the panel.
