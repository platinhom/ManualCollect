Importing and Displaying the 1fjs Receptor and Ligand

The complex for this exercise is actually in two files, one containing the receptor and one containing the ligand.

1. Choose File → Import Structures.

The Import dialog box opens.

2. From the Files of type option menu, ensure that Maestro is chosen.
3. If the options are not displayed, click Options.
4. Ensure that Import all structures and Replace Workspace are selected.
5. Select the files 1fjs_prep_recep.mae.gz and 1fjs_prep_lig.mae.gz.
6. Click Open.

The Import dialog box closes. Only one of the structures is displayed in the Workspace.

7. In the Entry List , control-click the check box for the entry that is not in the Workspace.

Now both structures are displayed. The protein structure is displayed in ribbon representation with the backbone shown. The protein structure entry includes solvent molecules (glycerine) and ions (Ca2+ and Cl−). The ligand is displayed in tube representation.
