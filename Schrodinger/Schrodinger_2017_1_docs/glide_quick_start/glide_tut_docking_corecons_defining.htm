Defining the Core Constraint

The core constraint is defined in the following steps.

1. In the Core tab, select Use core pattern comparison.

2. Select Restrict docking to reference position.

The first few controls in the Define core section become available.

3. Enter 1.5 in the Tolerance text box.
4. Select Pick core-containing molecule.
5. Click an atom in the reference ligand (in the Workspace).

The ligand is marked with purple markers, and the remaining controls in the Define core section become available.

6. Under Core atoms, select SMARTS pattern.
7. Rotate the structure so that you can clearly see the three six-membered rings.
8. Ensure that the Select button is selected and the selection scope button shows an A (for picking atoms).
9. Select the three six-membered rings with their ether linkages, and the amidine group on the terminal ring.

Do not include the carboxyl on the middle ring or the hydroxyl on the terminal ring.

You can drag to make the first selection, then click the Add to selection (+) button and click or drag to add atoms to the selection.

10. In the Ligand Docking panel, click Get From Selection.

The Smarts pattern text box is filled in with the pattern that corresponds to the atoms selected in the Workspace. The markers on the Workspace selection turn green.
