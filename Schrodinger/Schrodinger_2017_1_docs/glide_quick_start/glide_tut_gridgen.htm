Generating Receptor Grids for 1fjs

This set of exercises demonstrates how to use the Receptor Grid Generation panel to set up and start a grid file calculation job. Grid files represent physical properties of a volume of the receptor (specifically the active site) that are searched when attempting to dock a ligand. You will use the grid files calculated in this chapter to dock ligands in later Glide exercises.
