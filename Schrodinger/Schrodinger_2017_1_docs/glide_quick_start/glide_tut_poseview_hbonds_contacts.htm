Visualizing Hydrogen Bonds and Contacts in 1fjs

In this exercise, you will display hydrogen bonds and contacts between the ligand and the receptor, and generate counts of contacts.

  1. In the Pose Viewer panel, select Display H-bonds & halogen bonds in the Non-bonded interactions section, and deselect Halogen bonds.

  2. Hydrogen bonds to the currently displayed pose are displayed as yellow dashed lines.

    If you want to change the cutoffs for defining hydrogen bonds you can do so in the Preferences panel, under Non-bonded interactions – Criteria. 

    If you want a count of hydrogen bonds between the ligands and the receptor click Create Property in the Non-bonded interactions section of the Pose Viewer panel, select H-bonds and halogen bonds and clear other options in the Create Properties dialog box. The count may take a few seconds to finish. When it does, an HBond property is added to the Project Table. 

  3. Use the RIGHT ARROW and LEFT ARROW keys to step through the poses.

    The hydrogen bonds are displayed as each pose is included in the Workspace. Note the difference in hydrogen bonding patterns between the ligands.

  4. In the Pose Viewer panel, select Display contacts.

    The contacts are shown as dashed lines connecting Workspace atoms. The default is to display only Bad or Ugly contacts between the ligand and the receptor. Ugly contacts are shown in red and Bad contacts are shown in orange. By default, atoms that are hydrogen bonded are not considered to have bad or ugly contacts.

    If you want to change the distance cutoff criteria for Good, Bad, and Ugly contacts you can do so in the Preferences panel, under Non-bonded interactions – Criteria. 

  5. Click Create Property in the Non-bonded interactions section.

    The Create Properties dialog box opens.

  6. Select Contacts and clear any other options. The count may take a few seconds to finish, and three new properties are added to the Project Table: Good vdW, Bad vdW, and Ugly vdW.

  7. Click OK.

    The count may take a few seconds to finish, and three new properties are added to the Project Table: Good, Bad, and Ugly.

  8. If the Project Table panel is not open, open it.

    You can do this by choosing Window → Project Table.

  9. Scroll the Project Table all the way to the right to view the contact count properties.

    If they are not displayed, choose Property → Show → Primary Only.

  10. Scroll down to view the properties for each pose.

    Even the worst ligand pose has many more good contacts than bad or ugly ones, and the best has some bad contacts.
