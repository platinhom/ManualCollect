Importing the Reference Ligand for Core Constraints

The first task is to import the reference ligand.

1. Choose File → Import Structures.

The Import dialog box opens.

2. From the Files of type menu, ensure that Common is chosen.
3. Ensure that Import all structures and Replace Workspace are selected.
4. Select the file sar_reference.mae.gz.
5. Click Open.

The reference ligand is displayed in the Workspace.

6. In the Style toolbox in the main window, click the Show only polar hydrogens button.
