Docking to a Site Found by SiteMap

If you do not have information on the active site of a protein, you can use SiteMap to locate possible active sites, and use the results to identify the active site for docking. In these exercises, you will use the site map for 1ke8, for which a site and a ligand are known, to illustrate how well the use of a site map performs.

  • If you have not started Maestro, start it now (see Starting Maestro for instructions).

  • If you have not copied the tutorial files, do so now (see Copying the Glide Tutorial Files).

  • If you have already created the site map by running the SiteMap Tutorial Introduction, you can skip to Generating a Grid from a Site Map, but first you should change your working directory to the directory that contains the SiteMap results (File → Change Working Directory).

  • Otherwise, start the exercises with Finding Sites with SiteMap.
