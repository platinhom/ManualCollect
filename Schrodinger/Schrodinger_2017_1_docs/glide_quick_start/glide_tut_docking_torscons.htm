Docking Ligands Using Torsional Constraints

If you have flexible ligands and want to constrain some of their torsions, or you want to enforce a particular orientation of a group in the ligands, you can use torsional constraints. The constraints are applied to any ligand that matches a supplied SMARTS pattern; ligands that do not match are docked without applying the torsional constraints. You can define multiple patterns, and for each pattern you can define multiple torsional constraints.

In this exercise, you will only define one pattern and one torsion.

First, any settings from the previous exercise are reset for the current exercise.

1. In the Ligands tab under Use ligands from, select Workspace.
2. In the Settings tab of the Ligand Docking panel, select SP (standard precision).
3. In the Core tab, deselect Use core pattern comparison.

In this exercise you will dock the native ligand for 1fjs, but with the imidazole ring flipped by applying a torsional constraint to the flipped geometry.
