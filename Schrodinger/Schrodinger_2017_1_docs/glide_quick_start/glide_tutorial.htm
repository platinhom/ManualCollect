Docking Ligands to a Target with Glide

These exercises are designed to help you quickly become familiar with the functionality of the ligand docking program, Glide, using the Maestro interface. Protein preparation is not covered here: see the Protein Preparation Guide for details of this task. Basic familiarity with Maestro is assumed.

  • Generating Receptor Grids for 1fjs has exercises for the generation of grids from a prepared protein to represent the receptor for docking.

  • Docking Ligands to 1fjs has exercises for docking and scoring a set of ligands with various settings.

  • Examining Poses for 1fjs has exercises for examining the ligand poses and information on them.

  • Docking to a Site Found by SiteMap has exercises for locating a potential active site on a protein using SiteMap and docking ligands to this site.

The exercises are intended to be run sequentially.

If you need help with a Glide task, click the Help button or see the Glide User Manual.
