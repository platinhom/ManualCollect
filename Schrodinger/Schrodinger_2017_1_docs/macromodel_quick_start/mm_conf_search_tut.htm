Conformational Search Exercises

The goal of conformational searching is to locate the low-energy configurations of a molecular structure. MacroModel includes a number of conformational searching algorithms as well as mixed methods. This exercise first explores three standard conformational searches, then explores searches with the ligand/protein system prepared earlier. The final search is a large-scale low-mode search with another protein. The exercises are listed below.
