Building Structures with the Materials Science Suite

The Materials Science suite offers several specialized tools for building different types of structures:

  • Organometallic complexes (Building Organometallic Complexes, Build Single Complex Panel, Build Multiple Complexes Panel)
  • Structure libraries (Building Structure Libraries by Adding to a Scaffold)
  • Element variations (Varying the Elemental Composition in a Structure, Elemental Enumeration Panel)
  • Nanotubes and nanosheets (Building Nanostructures, Nanostructure Builder Panel)
  • Nanoparticles (Building Nanoparticles, Nanoparticle Builder Panel)
  • Disordered molecular systems (Building a Disordered Molecular Mixture, Disordered System Builder Panel)
  • Crystal structures (Building a Crystal Structure, New Crystal Structure Panel)
  • Slabs and interfaces (Build Slabs and Interfaces Panel)
  • Polymers and amorphous polymer cells (Building Polymers, Polymer Builder Panel, Crosslink Polymers Panel)

For information on general structure-building and adjustment tools, the following topics may be helpful:

  • Maestro Quick Reference Guide
  • Updating Coordinates
  • Symmetrize Workspace Dialog Box
  • Align Atoms Panel
