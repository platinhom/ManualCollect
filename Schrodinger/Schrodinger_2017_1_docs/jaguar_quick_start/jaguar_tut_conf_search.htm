Conformational Search, Refinement, and Analysis Exercises

This set of exercises covers the use of Jaguar to refine the results of a MacroModel conformational search, and the analysis of the resulting conformations. The analysis involves several tasks: calculating relative QM energies, creating a sequential index for the optimized structures, coloring, superimposing, and labeling the structures, plotting the energies, and calculating RMSD and maximum deviation.

You must have a MacroModel license as well as a Jaguar license to run these exercises.
