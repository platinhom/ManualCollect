Running a Jaguar Coordinate Scan for Chloromethyl Fluoromethyl Ether

Coordinate scans involve energy calculations at a series of points along one or more coordinates, to generate a grid representation of a potential energy surface. This exercises selects two coordinates and sets the values of the coordinates at which the energy calculations (or rather, geometry optimizations) will be done.

  1. Import the file clme_fme_ether.maegz from your tutorial directory (File → Import Structures).

    The chloromethyl fluoromethyl ether molecule is imported and displayed in the Workspace.

  2. Open the Jaguar - Relaxed Coordinate Scan panel.

    How do I do this?

    1. Click the Tasks button.

      A menu is displayed. To the left there should be a pane with a TASKS heading and an APPLICATIONS heading. If it is not displayed, click Browse on the menu.

    2. Under TASKS, choose Structure Analysis.

      The contents of the pane are replaced with the tasks available for Structure Analysis.

    3. Under Coordinate Scan, click QM Relaxed.

    The relaxed scan optimizes the geometry for all the coordinates other than the scan coordinates. You can also do rigid scans in which the geometry is not relaxed. These are faster, but do not give as good a picture of the minimum energy surface.

  3. In the Scan tab, choose Dihedral from the Type option menu of the Add new coordinate section.

  4. Pick the Cl atom, then the C, the O and the C of the ether linkage.

    When you have finished picking, the dihedral is added to the Defined coordinates table.

  5. In the Selected coordinate section, enter -180 in the Starting value text box, 180 in the Final value text box, and 60 in the Increment text box.

    When you press Enter, the Number of steps text field is updated, and the value is entered into the table.

  6. Now pick the F atom, then the C, the O and the C of the ether linkage.

    A new coordinate is added, and the values in the Selected coordinate section are set to the same values as for the previous coordinate. If for some reason this does not happen, enter -180 in the Starting value text box, 180 in the Final value text box, and 60 in the Increment text box.

  7. Set the job name to ether_scan.

  8. (Optional) If you have multiple processors available, you can distribute the job by opening the Job Settings dialog box (click the gear icon) and entering the number of processors in the CPUs text box.

    As the number of points for each coordinate is 7, a suitable number of processors for even load balancing would be 7, but you can choose any number you like. Each point in the scan takes about 3 minutes.

  9. Click Run to start the job.
