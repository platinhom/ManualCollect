Geometry Optimization with DFT

In this exercise, you will optimize the geometry of the output structure from the previous exercise, at the BLYP/6-31G** level.

  1. Ensure that the entry from the previous exercise (Single Point Jaguar Energy Calculation) is included in the Workspace.

    Starting here?

    Import the file benzene.maegz from your tutorial directory (File → Import Structures).

  2. Open the Jaguar - Optimization panel.

    How do I do this?

    1. Click the Tasks button.

      A menu is displayed. To the left there should be a pane with a TASKS heading and an APPLICATIONS heading. If it is not displayed, click Browse on the menu.

    2. Under TASKS, choose Quantum Mechanics.

      The contents of the pane are replaced with the tasks available for Quantum Mechanics.

    3. Click Optimization.

  3. In the Theory tab, for Recommended, choose Gradient-corrected, then select BLYP.

    You can view the input file contents by choosing Edit from the Settings button menu. Notice the addition of keywords in the gen section. A BLYP calculation is specified by dftname=blyp and a geometry optimization is specified by igeopt=1. Close the panel by clicking Cancel.

  4. Type benzene_dft in the Job name text box, and click Run.

    This job takes a few minutes to finish. When it has finished, a new entry is added to the project and displayed in the Entry List.

  5. Close the Jaguar - Optimization panel.
