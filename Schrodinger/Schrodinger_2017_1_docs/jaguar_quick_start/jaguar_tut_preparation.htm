Preparation for Jaguar Exercises

This manual contains exercises designed to help you learn the basic tasks you will need to prepare and initiate Jaguar calculations from Maestro. For more information about Jaguar features, see the Jaguar User Manual.

This section contains instructions for setting up the tutorial.

  • Running Basic Jaguar Calculations contains exercises on running various basic Jaguar calculations: single-point energies with both DFT and LMP2 theory, geometry optimizations, and orbital plotting.

  • Calculating and Displaying Properties contains exercises that show you how to predict and display properties, covering pKa prediction, potential energy surfaces from coordinate scans, mapping a 3D property (ESP) onto a surface, noncovalent interactions, and calculating VCD spectra.

  • Conformational Search, Refinement, and Analysis Exercises contains exercises on using MacroModel and Jaguar together for conformational searches.

  • Transition State Search and Analysis Exercises contains a set of exercises for transition state searches, intrinsic reaction coordinate calculations, frequency calculations.

To navigate through a set of exercises, you can click the links in the bar at the top or bottom of a topic. Topics that begin a section usually have a list of topics at the end of the text.

The preparations that must be made for the exercises are to create a working directory to keep all your input and output files, start Maestro, ensure that this directory is Maestro’s current working directory, and create a project. These tasks are described in the following topics.

It is assumed that you are using the default Maestro profile. If you have made changes, the instructions might not apply to your modified profile. To return to the default profile, you can zip the contents of the Maestro folder under maestro111/profiles in your Schrödinger user resources directory, and then remove the folder. When you have finished the tutorial, remove the Maestro folder created for the tutorial and unzip your original Maestro folder.
