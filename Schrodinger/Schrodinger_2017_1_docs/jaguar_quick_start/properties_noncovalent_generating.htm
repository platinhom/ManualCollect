Generating Noncovalent Interaction Surfaces for 3-hydroxypropanal

In this exercise, you will run a calculation to generate the surfaces needed to display noncovalent interactions.

  1. Import the file 3-hydroxypropanal.maegz from your tutorial directory (File → Import Structures).

  2. Open the Jaguar - Optimization panel.

    How do I do this?

    1. Click the Tasks button.

      A menu is displayed. To the left there should be a pane with a TASKS heading and an APPLICATIONS heading. If it is not displayed, click Browse on the menu.

    2. Under TASKS, choose Quantum Mechanics.

      The contents of the pane are replaced with the tasks available for Quantum Mechanics.

    3. Click Optimization.

  3. If you have run other Jaguar calculations in this Maestro session, choose Reset Panel from the Settings button menu to reset the options, and confirm the reset.

  4. In the Properties tab, select the row for Surfaces and check the box.

    The Surfaces options are displayed below the Properties table.

  5. Select Noncovalent interactions, and clear the options for all other surface types.

  6. Name the job 3-hydroxypropanal, and click Run.

The job takes a few minutes. When it finishes, three surfaces are incorporated: the reduced density gradient, the interaction strength, and the density Laplacian. By default the reduced density gradient isosurface is displayed.

In addition to the surfaces, the bond critical point of the noncovalent interaction is displayed as a dummy atom (not visible in wire frame representation) with the bond paths represented by zero-order bonds (blue lines). You can see that the critical point is in the middle of one of the surfaces.
