Displaying Orbitals in Maestro

In this exercise you will experiment with displaying orbitals in Maestro. If you have not done the previous exercise (Generating Orbitals for Display in Maestro), please do so now.

Since both the HOMO and the LUMO are degenerate orbitals, the orbital displayed could be either of the degenerate pair. The row in the Entry List has an S button on the right, indicating that the entry has surfaces associated with it.

  1. In the Entry List panel, click the S button for the benzene_mo entry.

    The Manage Surfaces panel opens. The table shows the surfaces generated by the job.

  2. Click the In column for HOMO-1.

    The HOMO-1 orbital replaces the HOMO orbital in the Workspace.

  3. Move the Isovalue slider.

    The volume occupied by the orbital changes with the isovalue.

  4. Reset the isovalue to −0.05.

  5. Click Display Options.

    The Surface Display Options dialog box opens. This dialog box has controls for the style, transparency, and color scheme.

  6. Experiment with changing the transparency, the style, and the colors of the positive and negative lobes.

    You can also make some of these changes from the table shortcut menu in the Manage Surfaces panel.

  7. Close the Surface Display Options dialog box.

  8. Click the In column for the first orbital, and shift-click the In column for the last orbital.

    All four HOMO and LUMO orbitals are displayed in the Workspace, but they are superimposed.

  9. Open the Workspace Configuration toolbox. How? Click the + button in the bottom right of the Workspace. 

  10. Under WORKSPACE LAYOUT, click the Tile button, then for Tile by choose Surface.

    The orbitals are displayed in separate areas of the Workspace. Each is in its own mini-Workspace, which can be manipulated as usual. By default all tiles are rotated or translated together, but you can perform actions on a single tile by clicking the Select Tile button and then selecting the tile to operate on.

  11. Open the toolbox again and click the Tile button to exit Tile mode.

    The orbitals are again superimposed.

To save images of the Workspace like those shown above, set up the color of the Workspace contents (Surface Display Options) and background (Workspace → Appearance), choose Workspace → Save Image As, then make settings and supply a file name in the Save Image panel.
