Coloring and Labeling the Conformers

In this exercise, the entries will be colored and labeled to distinguish them from each other. Coloring by entry allows you to distinguish them easily, but the color does not indicate which entry is which. Adding a label identifies the entries. The label used here is the sequential index, which was created in Setting Up Properties to Facilitate Comparisons. This index will also be used for plotting.

If you have not done the previous exercise (Superimposing the Conformers), please do so now.

  1. Ensure that the QM-optimized conformers are included in the Workspace.

  2. Select all the atoms in the Workspace. How?Press CTRL+A (⌘A) or drag out a rectangle over all the atoms.

  3. Click the Style button.

    The Style toolbox opens.

  4. Click the arrow next to Color Atoms, and choose Entry from the menu that is displayed.

    Each entry is colored with its own color.

Next, one of the fluorine atoms will be labeled with the sequential index.

  1. Click the Define button on the Selection toolbar.

    The Atom Selection dialog box opens.

  2. In the Atom tab, choose Atom name from the list on the left.

    A list of atom names is displayed in the center.

  3. Choose F19 from the Atom name list.

  4. Click Add, then click OK.

  5. The Atom Labels panel opens. By default it is docked into the main window.

  6. Click the Style button.

    The Style toolbox opens.

  7. Click the arrow next to Apply Labels, and choose Edit Custom Label from the menu that is displayed.

    The Edit Custom Label dialog box opens.

  8. Click Add.

    The Add Label Fields dialog box opens at the Atom Properties tab.

  9. Select Sequential index in the Entry properties tab, and click OK.

    If you don’t see Sequential index, choose All properties from the Subset option menu.

  10. In the Edit Custom Label dialog box, select Sequential index in the list and deselect the rest.

  11. Click OK.

    The atom named F19 (a fluorine at the end of the side chain) is labeled with the sequential index.

The structures are now clearly distinguished, and it can be seen that the three pairs are structures 1 and 4, 3 and 6, and 5 and 7. Conformer 2 does not have a matching structure with the opposite orientation of the hydroxyl, but this may have been filtered out in the conformational search.
