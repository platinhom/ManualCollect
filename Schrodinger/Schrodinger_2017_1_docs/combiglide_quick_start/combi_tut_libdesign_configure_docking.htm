Configuring the Glide Docking Calculations for the Pyrazole Library

The next step is to configure the CombiGlide docking calculations. In a real application, you would select a receptor and generate the grids before starting the combinatorial screening process. You would then set up the parameters for the docking process in this step. For this tutorial, grids have already been generated.

1. In the Settings tab, click Browse in the Receptor grid section.
2. Select receptor_grid.zip, and click Open.

The path to the grid files is displayed in the Receptor grid base name text box. We will be using the default docking settings in the tutorial, so no further settings need to be made.

3. Click Next.

The Define Core Poses step is displayed.
