Importing the Core-Containing Molecule for the Pyrazole Library

To run CombiGlide, you must supply a molecule that contains the core structure. This molecule must be an all-atom, 3D structure that has a reasonable representation of the experimental geometry of the core structure. Ordinarily you would have to build or obtain this structure and minimize it using MacroModel or LigPrep, for example. For this tutorial, the core has already been built and minimized, and you only need to import it.

1. Choose File → Import Structures.

The Import panel opens.

2. Ensure that Common (or Maestro) is chosen from the Files of type option menu.
3. Select core_tutorial.mae and click Open.

The Import panel closes and the 3D, minimized core-containing molecule is imported into the Project Table and displayed in the Workspace.
