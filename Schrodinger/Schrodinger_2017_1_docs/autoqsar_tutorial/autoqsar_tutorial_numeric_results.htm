Analyzing Predictions from a QSAR Model for Binding Affinity and Identifying Compounds Outside the Domain of Model Applicability

In this exercise predictions made by AutoQSAR will be analyzed to examine prediction accuracy and identify compounds that fall outside the applicability domain of the QSAR model. These are steps that should be taken when evaluating performance of a QSAR model for an external validation set of compounds for model validation. For prospective application of the QSAR model, no comparison to experimental properties is possible.

  1. Choose File → Import Structures.

    The Import dialog box opens.

  2. From the Files of type option menu, ensure that Maestro is selected.

  3. If the options are not displayed, click Options.

  4. Ensure that Import all structures and Replace Workspace are selected.

  5. If calculations from the preceding exercise were performed, select the file fxa_prediction/fxa_prediction-out.mae.gz. If running this exercise on its own, import the provided file, fxa_prediction-out.mae.gz.

  6. Click Open.

    The Import dialog box closes. Only one of the imported structures is displayed in the Workspace.

  7. Open the Project Table panel by choosing Window → Project Table .

  8. Ensure all ligand entries in the fxa_prediction-out group are selected.

  9. Click the Plot button to open the Manage Plots panel.

  10. Click New Scatter Plot to open a Scatter Plot panel.

  11. In the Scatter Plot panel set the X-Axis property to Binding Affinity and the Y-Axis property to the Pred fxa affinity (these are the AutoQSAR predicted affinities).

  12. Set the Color by property to fxa affinity Domain Alert and the Color map to Winter(blue-green).

  13. Select the Best fit line option.

    This plot shows the consensus QSAR model predictions in the Pred fxa affinity property reproduce experimental binding affinities in the Binding Affinity property well with an R2 of 0.60. Compounds with a Domain alert are colored in Green. These are compounds that fall outside the applicability domain of training set compounds where predictions aren’t expected to be accurate. Three of these outliers were added from a different factorXa congeneric series with a central beta-lactam ring. The fourth compound is structurally related but falls outside the 95% confidence interval used to set the applicability domain alerts and so is flagged as an outlier. Visual inspection shows it is part of the same congeneric series as the training set and is not an outlier.

  14. Remove the three outliers added from a congeneric series for the 1f0r protein structure.

    1. In the Project Table deselect the three compounds labeled 160-86 1f0r series, 16131 1f0r series, and 16180 1f0r series.
    2. Regenerate the plot following instructions 10 through 13. Note that the correlation coefficient is now significantly larger.
