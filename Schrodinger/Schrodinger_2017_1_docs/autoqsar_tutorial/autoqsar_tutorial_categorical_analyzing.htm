Analyzing a Categorical QSAR Model for Mutagenicity

In this exercise we will analyze the categorical QSAR Models created by AutoQSAR to predict mutagenicity.

  1. Open the Task Tool, type autoqsar in the Search box, and press Enter.

  2. Choose Other Applications > AutoQSAR to open the AutoQSAR panel.

  3. Select View model and make prediction.

  4. From the option menu choose mutagen_model.qzip.

    If the previous exercise to create the QSAR model was not performed in the current Maestro session that option will not be available. You can instead import mutagen_model.qzip into the AutoQSAR panel by clicking Browse and importing it manually.

  5. In the Model Report section click the + button to display more information about the QSAR models that were created. Click Show More to show all key information about the models.

  6. Select the top-ranked model from the first row, bayes_dendritic_15 and click Report Details

    A panel opens, showing a detailed report showing performance of the model on the learning set.

    The Training and Test set confusion matrices in this report can be used to assess the accuracy, sensitivity and specificity of the categorical QSAR model as shown below. The prediction accuracy for training and test set in this example is excellent.

    Accuracy = (108+141)/254 = 98%
    Sensitivity = 141/(141+0) = 100%
    Specificity = 108/(108+5) = 96%

    Accuracy = (29+42)/83 = 86%
    Sensitivity = 42/(42+4) = 91%
    Specificity = 29/(29+8) = 78%
