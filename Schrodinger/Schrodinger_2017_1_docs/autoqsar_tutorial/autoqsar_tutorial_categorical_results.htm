Analyzing Predictions from a Categorical QSAR Model for Mutagenicity

In this exercise predictions made by AutoQSAR will be analyzed to examine prediction accuracy. These are steps that should be taken when evaluating performance of a categorical QSAR model for an external validation set of compounds for model validation. For prospective application of the QSAR model, no comparison to experimental properties is possible.

  1. Choose File → Import Structures.

    The Import dialog box opens.

  2. From the Files of type option menu, ensure that Maestro is selected.

  3. If the options are not displayed, click Options.

  4. Ensure that Import all structures and Replace Workspace are selected.

  5. If calculations from the preceding exercise were performed, select the file mutagen_prediction/mutagen_prediction-out.mae.gz. If running this exercise on its own, import the provided file, mutagen_prediction-out.mae.gz.

  6. Click Open.

    The Import dialog box closes. Only one of the imported structures is displayed in the Workspace.

  7. Open the Project Table panel by choosing Window → Project Table .

  8. Ensure all 702 ligand entries in the mutagen_prediction-out group are selected.

  9. Sort compounds by the predicted mutagenicity classes in the Pred mutagen Class and then the experimental mutagenicity in the Ames Category property.

    This groups the true predictions and false predictions in a way that is easy to manually count.

  10. Count the following:

    • the number of compounds correctly predicted to be nonmutagens, i.e. Ames category=0 and pred mutagen Class=”<0.50”
    • the number of compounds predicted to be mutagens that are actually nonmutagens, i.e. Ames category=0 and pred mutagen Class=”>=0.50”,
    • the number of compounds predicted to be nonmutagens that are experimentally mutagens, i.e. Ames category=1 and pred mutagen Class=”<0.50”,
    • the number of compounds correctly predicted to be mutagens, i.e. Ames category=1 and pred mutagen Class=”>=0.50”.

    From these values the following confusion matrix can be created that shows good predictive ability:

    Accuracy = (233+363)/702 = 85%
    Sensitivity = 363/(363+42) = 90%
    Specificity = 233/(233+64) = 78%
