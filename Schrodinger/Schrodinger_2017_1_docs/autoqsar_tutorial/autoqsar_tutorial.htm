Creating and Applying Predictive QSAR Models with AutoQSAR

These exercises are designed to help you quickly become familiar with the functionality of AutoQSAR as run from Maestro. Basic familiarity with Maestro is assumed.

  • Creating a Numeric QSAR Model for Binding Affinity has exercises to create and apply QSAR models to predict protein-ligand binding affinities, including examples of the use of alerts to flag compounds that are structurally distinct from the training set.
  • Creating a Categorical QSAR Model for Mutagenicity has exercises to create and apply global QSAR models including how to create and interpret confusion tables.

These exercises can be run independently. Links to the exercises are given below.

If you need help with an AutoQSAR task, click the Help button.
