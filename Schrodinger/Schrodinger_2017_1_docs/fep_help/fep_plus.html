FEP+ Panel

Set up, run, and analyze FEP calculations for the relative binding affinities of a set of congeneric ligands to a target.

To open this panel, choose Tasks → Free Energy Perturbation → FEP+ in the Task Tool browser.

Setup panel

In this panel you import the structures or perturbation map and make some settings for the review panel.

Import structures or perturbation map from option menu

Import the receptor and ligands or import an existing perturbation scheme, which contains the receptor and ligands, into the workflw. The receptor and ligands must be imported together; you cannot do multiple imports to add a receptor or ligands. The menu has two items:

  • Project Table (n selected entries)—Import the entries that are currently selected in the Project Table. The number of entries selected is shown on the menu item. An icon is displayed to the right which you can click to open the Project Table and select entries.
  • File—Import the receptor and the ligands from a Maestro file, or import a perturbation scheme, which includes the receptor, ligands, and graph data, from a file. When this option is selected, the File name text box and Browse button are displayed. Click Browse and navigate to the file you want to use. The file name is displayed in the text box when you click Open in the file selector. You can also enter the file name in the text box.
Open Project Table button

Open the Project Table panel, so you can select or include the entries for the source of the structures.

Import button

Import the selected structures from the Project Table into the workflow.

File name text box and Browse button

If you chose File from the Import structures or perturbation map from option menu, enter the file name in this text box, or click Browse and navigate to the file. The name of the file you selected is displayed in the text box.

Data Views options

Select one or more of these options to display the corresponding data view tab in the review panel.

Receptor and ligand information

Displays the receptor title and the total number of ligands, with an indication of the status. The tooltip gives a text description of the status. If potential problems are detected, the status is a clickable button that opens a panel. For the receptor, it opens the Protein Reliability Report Panel. For ligands, it opens the Force Field Builder Panel, so you can generate force-field parameters for missing torsions.

Next button

Go to the review panel. The setup panel closes.

Review panel

In this panel you can review the ligand data and the perturbation map, add ligands and modify data, start an FEP job with the current map, and view analyses of the results of the perturbation calculations. The main part of the panel consists of a set of tabs, which are described in separate topics:

  • FEP+ Panel — Overview Tab
  • FEP+ Panel — Map Tab
  • FEP+ Panel — Activity Cliffs Tab
  • FEP+ Panel — Analysis Tab

The remaining controls are described below.

Ligands information

This report shows the number of ligands, the number shown, and the number selected.

Search text box

Search for ligands by title, property, or by SMARTS or SMILES matches. The tabs are updated to show data only for the ligands that match, and the ligands information is updated. The search can be done with a plain text string, which is used to find matches to the title or to properties in the table in the current tab, or a SMARTS or SMILES pattern, which is used to find matches to the structure. When you clear the search text box, the full set of ligands is displayed.

Apply filter option menu and settings button

Filter ligands by their properties. You can choose a filter from this option menu, or choose Create new to open the FEP+ — Edit Filter Dialog Box and create a new filter. To edit or delete the current filter, click the Settings button (gear icon) and choose the appropriate item.

Add New Ligands option menu

Choose an item to add new ligands to the FEP calculation.

  • From File—import the new ligands from a file. All ligands in the file are imported. Opens a dialog box in which you can specify the file name or browse to the file.

  • From 2D Sketcher—create new ligands by sketching them in 2D. Opens the Add Ligand from Sketcher panel, where you can sketch the ligand using the 2D sketching tools - see 2D Workspace Panel for more information on the tools. This custom version has an option menu, Choose template ligand, which allows you to import one of the existing ligands into the sketcher to use for sketching the new ligand. The menu item tooltip shows the 2D structure of the ligand. When you have the desired structure, click Save as New to provide a title and add the ligand to the set. You can then continue to sketch other ligands and save them. When you have finished adding new ligands, click Close to close the sketcher.

Export option menu

Export structures and data to external files. The items available depend on the tabs that are present in the panel. Structures are exported to Maestro files, plots or maps are exported as images, with a choice of formats, and tables are exported to CSV files with headings or an Excel spreadsheet.

Add Affinity Data button

Add binding affinity data to the ligands. The data must be present in the structure source. Opens the Choose Affinity Property dialog box, where you can choose the property and set the affinity units.

Back button

Go to the Setup panel. The review panel closes.

Job toolbar

Manage job submission. See Job Toolbar for a description of this toolbar.

The Settings button menu provides access to advanced options for the MD simulations.

Status bar

The status bar displays information about the current job settings for the panel. This includes the job name, task name and task settings (if any), number of subjobs (if any) and the host name and job incorporation setting. It also displays a message for a short time when you start a new job.

The status bar also contains the Help button, which opens the help topic for the panel.
