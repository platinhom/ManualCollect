Delete Properties Panel

The Delete Properties panel can be used to delete properties from the Project Table. The properties to be deleted must be selected from the Properties list, which is a standard property selection tool that includes filtering by subset or by text matching.

Selection of properties can be done with the usual shift-click and control-click actions in the list. You can also select properties with the Select properties buttons.

  • All—Select all the properties that are shown in the list. Properties that are not shown are deselected.

  • All Empty—Select the properties for which none of the entries have a value. All of the empty properties are selected, including any that are not shown in the list.

  • None—Clear the selection of properties.

When you are satisfied with your selection, click Delete Selected Properties to delete the properties. You can repeat the selection and deletion as many times as you like.

You can also delete properties from the Property Tree panel, by right-clicking on a property and choosing Delete Property.
