Color by Property Panel

Color the atoms in the included or selected entries depending on the values of the selected entry property.

To open this panel, choose Tasks → Workspace and Project Table Operations → Project Table Operations → Color by Entry Property.

Using the Color by Property Panel

To color atoms by the values of an entry property:

  1. Select or include the entries whose atoms you want to color.

  2. Choose the desired property from the Property to color by option menu.

    The range of values for this property in the entire project is displayed below the option menu.

  3. Set the number of property ranges ("bins") that you want to divide the property into, in the Number of desired bins text box.

  4. For each bin, specify the upper limit of the property value in the text box and click the color button to set the color.

  5. If you have a receptor in the Workspace or the selection and don't want to color receptor atoms, select Skip receptor entries.

  6. Choose the atoms you want to apply the color scheme to, with the Apply color scheme to atoms picking tools.

    If you are using the Workspace entries, you can pick the atoms. For more options, click Select and use the Atom Selection Dialog Box to choose the atoms.

  7. Click Included Entries or Selected Entries to apply the color scheme.

Property to color by option menu

Choose the property to use for coloring the atoms. The menu is populated from the properties that are currently in the Project Table, and is refreshed when you click Reset.

Range of values text

This text shows the range of values for the property that is observed for all entries in the Project Table that have values for the property.

Bins section

In this section, you specify the number of bins (property ranges) to use for coloring and set the colors.

Number of desired bins text box

Specify the number of bins to partition the property into. As you change this value, sets of controls for each bin are added or removed.

Bin n color button and text box

Specify the color for a bin and the range of values to which it applies. Click the color button to set the color, and enter the upper limit for the value in the text box. The first bin includes all values less than or equal to the value specified in the text box; the last one includes all values greater than the value specified for the previous bin. The range covered includes the upper value but excludes the lower value.

Skip receptor entries option

If any of the entries has more than 300 atoms, treat it as a receptor, and do not color these entries.

Apply to buttons

Apply the color scheme to the Included Entries or the Selected Entries, by clicking the appropriate button.

Apply color scheme to atoms picking tools

Specify the atoms that you want to color in the selected or included entries, either by picking or by using the Atom Selection Dialog Box (click Select). Picking can be used to apply the color to the included entries. The Atom Selection dialog box can be used for either the selected or included entries.

Reset button

Clear all settings and reset the panel to its initial state. This includes updating the Property to color by option menu to reflect the current set of properties in the Project Table.
