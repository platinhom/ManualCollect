Trajectory Panel

In this panel you can play through trajectories from Desmond simulations, examine individual frames, and export trajectory data in a variety of forms.

Using the Trajectory Panel

The speed at which a trajectory is played depends on the number of operations that are performed on each frame. For faster play, options like Update secondary structure and Also display residues within should not be selected, as they involve a fair amount of time to evaluate. However, if you are creating a movie, the speed of play in the Workspace does not matter, and including these options is not an issue.

You can have other entries in the Workspace while playing through a trajectory. This can be useful for viewing the trajectory against some static objects. It also allows you to set up measurement markers between the trajectory and the other entries, which are preserved during play. When the trajectory is loaded, you are prompted to choose whether to keep these objects in the Workspace or remove them.

Setting up superposition on an external structure must be done with care. The superposition is done with an ASL expression, and if the order of the atoms in the trajectory frames is not the same as in the external structure, you may get unexpected results. It is best to use ASL expressions that do not translate to atom numbers, like mol.entry 1, for the first molecule in the entry, or protein. The atom selection button contains a list of choices of this kind, like Protein or Ligands. If the atom ordering is a problem, you could align the external structure to a particular frame, and then play the trajectory by superimposing on that frame.

Toolbar and Menu Bar

The toolbar contains a standard set of controls for playing through the trajectory frames, which are listed below. The menu bar has one item, Play, which contains items that correspond to the toolbar buttons.

Go to start
Display the first frame.
Previous
Display the previous frame.
Play backward
Display the frames in sequence, moving toward the first.
Stop
Stop playing through the frames.
Play forward
Display the frames in sequence, moving toward the last.
Next
Display the next frame.
Go to end
Display the last frame.
Loop
Choose an option for repeating the display of the frames. Single direction displays frames in a single direction, then repeats. Oscillate reverses direction each time the beginning or end of the list is reached.

Frame control section

This section has controls for the display of individual frames and for playing through the frames.

Start text box

Starting frame number. Play starts from this frame when played forward from the beginning. Frames are numbered from 0.

End text box

Ending frame number. Play ends at this frame when played forward.

Frame slider and frame text box

Selects the frame to view. The frame number is displayed in the text box below the slider, which you can also use to select a frame. The total number of frames is also displayed in a noneditable text box.

Step text box

Set the number of frames to step when playing through frames. This value does not affect the Frame slider. The frames that are selected for play can be exported as a selection of frames, using the output buttons.

Time text boxes

Displays the time for the current frame and the total time for the trajectory. You can enter a time in the text box to select a frame.

Speed slider

Set the speed at which the frames are played.

Hide clipping planes during continuous play option

This option hides the clipping planes window while the trajectory is being played, and redisplays it when play stops. Hiding the clipping planes window speeds up playing the trajectory by about 50%.

Display section

This section controls what is displayed when playing the trajectory.

Use lower quality drawing to speed up play option

Selecting this option uses a lower quality representation of objects (tubes, spheres) in the Workspace to speed up play. It has no effect on wire frame representation.

Update secondary structure

Update the secondary structure assignment for each frame during play.

Show simulation box option

Show the edges of the simulation box (in purple) when this option is selected.

Show axes option

Show the coordinate axes (in green) when this option is selected.

Replicate system text boxes

Enter the number of replicas of the system to display in each of the three directions. This enables you to visualize the movement across the simulation box boundaries.

Trajectory smoothing text box

Smooth the trajectory by averaging the coordinates over the specified number of frames.

Positioning options

These options control the positioning of each frame during play.

  • Do not adjust—Do not adjust the positioning of each frame.
  • Superimpose on frame N—Superimpose selected atoms in each frame on the corresponding atoms in the frame number given in the text box. Use the Pick atoms for frame positioning picking controls to make the atom selection.
  • Superimpose on Workspace structure—Superimpose selected atoms in each frame on the corresponding atoms in the Workspace structure. This allows you to view the trajectory against some static objects in the Workspace. Use the Pick atoms for frame positioning picking controls to make the atom selection, which must match both the Workspace entries and the trajectory structure. See Using the Trajectory Panel for information on using this feature.
  • Center molecules—Center the selected molecules in the viewing area. Use the Pick atoms for frame positioning picking controls to select the molecules.
Pick atoms for frame positioning picking controls

Pick the atoms that are used to position each frame, using the standard picking controls in this box. You should consider picking atoms that do not change their position much during the simulation.

Atoms to display in each frame section

This section controls which atoms are displayed in each frame.

Always display these atoms picking controls

Pick the atoms that should be always visible in each frame — for example, the ligand in a ligand-receptor complex.

Also display residues within option and text box

Display entire residues that have any atoms within the specified distance of the atoms that are always displayed. This expression is evaluated dynamically for each frame. As a consequence, a particular residue might not be displayed in all frames — for example, a water molecule might move in and out of the specified distance.

Display only specified atoms option

Turn on or off the display of the specified atoms. If this option is selected, only those atoms that are specified with the tools in this section are displayed. If this option is not selected, all atoms are displayed.

Output buttons

These buttons allow you to export the trajectory data in various forms. You can export individual frames, all frames, or the selection of frames defined by using the Start, End, and Step text boxes.

Structure

Save structures from the trajectory to a file or create project entries from the structures. Opens the Export Structure dialog box, in which you can specify where the structures will go, which structures to export, and which atoms to export.

Image

Create an image of the Workspace with the current frame displayed. Opens the Save Image dialog box.

Movie

Save a movie of the trajectory in MPEG format. Opens the Export Movie dialog box, in which you can select the format and the frames to be exported.
