Using the Record Movie Dialog Box

To record a movie, first select the entries in the Project table that you want to use for the movie, and set up the Workspace display the way you want it. Then open the dialog box and record the movie. If you want to set the resolution and the speed, click Options and make the settings in the Movie - Options dialog box. The maximum speed is 25 frames per second.

Record Movie Dialog Box Features

  • Filename text box and Browse button
  • Options button
Filename text box and Browse button

Enter the file name for the movie in this text box, or click Browse and navigate to the location to save the file. The extension .mpeg is added automatically if it is not present.

Options button

Opens the Movie - Options dialog box, in which you can set the speed of play, select the movie resolution and quality, and choose to add interpolated frames between entries.
