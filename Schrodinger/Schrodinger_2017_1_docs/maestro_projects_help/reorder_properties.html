Reorder Properties Dialog Box

In this dialog box you can reorder the properties in the Project Table. Tools are provided for manual reordering and for reordering by family or alphabetically. You can reorder just the shown properties, or all properties. You can also change which properties are shown in the Project Table at the same time.

To open this panel, choose Property → Columns → Reorder

Reorder Properties Dialog Box Features

  • Properties options and list
  • Buttons for moving properties
  • Show properties buttons
  • Create Family button
  • Select section
  • Order all option menu
  • Apply button
Properties options and list

The Properties list displays the properties in the scrolling section of the Project Table that can be reordered manually. The options above the list allow you to choose whether to list all properties (All) or only the properties that are shown in the Project Table (Shown). If you use the Show properties buttons, then the list reflects the use of these buttons to change which properties will be shown in the Project Table when you click OK rather than the properties that are currently shown in the Project Table.

You can select multiple properties in the list with the usual shift-click and control-click actions. When you have a selection in the list, you can drag the selected properties to a new location in the list, or act on them with the buttons that are described below.

If the selection is discontiguous when you move it to a new location, the properties are moved so that they are contiguous in the new location.

If the selection range includes hidden properties, these properties are moved if you select All for the properties to list. In this case, the hidden properties are set in italics. If you selected Shown, the hidden properties are treated as though they are not selected, and so are not moved with the selected properties.

If hidden properties are not listed, properties are always moved to a point above any hidden properties in the new location.

Buttons for moving properties

The four buttons to the right of the list allow you to move the selected properties to the top or the bottom of the list, or to move them up or down by one place. If the selection is discontiguous, the properties are first moved to a contiguous block below the first selected property, then moved to the new location.

Show properties buttons

These buttons can be used to change whether properties are shown or hidden in the Project Table. The change in visibility is not made until you click OK, but any changes affect the properties that are marked as hidden in the Properties list.

Show Only button

Show only the properties that are selected in the list.

Show button

Show the hidden properties that are selected in the list.

Hide button

Hide the shown properties that are selected in the list.

Create Family button

Create a custom property family from the properties that are selected in the list. Opens the Create Custom Property Family dialog box, in which you can name the new custom family.

Select section

This section provides tools for selecting properties in the Properties list.

All button

Select all properties in the list.

None button

Clear the selection of properties, so that no properties are selected.

Invert button

Invert the selection of properties in the list: the selected properties are deselected, and the unselected properties are selected.

Select by options

Select properties on the basis of their family membership or by filtering their names.

  • Family—Select the members of the property family that you choose from the option menu.
  • Filter—Select properties whose names match the filter given in the text box. The filter can include wildcard characters: * for any number of any character, ? for any single character, and square brackets to enclose a choice of characters. There is an implicit * at the beginning and end of the expression. The filter is case-insensitive.
Add and Replace buttons

These buttons apply the selection made under Select by to the list. Add adds the selection to what is already selected in the list; Replace replaces the selection in the list with the selection made under Select by.

Order all option menu

Order all properties (shown and hidden) according to the option chosen from the option menu. The choices are Alphabetically, By standard families, By custom families, and By all families.

When ordering by family, the families are ordered alphabetically for each family type. Custom families are placed before standard families. If a property belongs to more than one family, it is ordered according to the first family it belongs to, and ignored for the other families. Within each family, the properties are ordered according to their current order in the list. This allows you to order the families, but not change the order within the family. Any properties that are not in a family are effectively moved to the end, in the current relative order.

Apply button

Apply the ordering chosen from the Order all option menu.
