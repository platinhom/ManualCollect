New Property Dialog Box

In this dialog box, you can specify property types and set the number of decimal places for real properties when you import properties from a spreadsheet.

The New Property dialog box opens when you click OK in the Choose Import Key dialog box, and one or more of the properties does not already exist in the Project Table.

  • To set the property type, select the properties in the table (use shift-click and control-click for multiple properties), choose the property type from the Type option menu, and click Set.

  • To set the number of decimal places for real properties, select the properties in the table, enter the number of decimal places to display in the Decimal places text box, and click Set.

    You can set the property type for a real property and the number of decimal places at the same time. The default number of decimal places is set in the Project - Project table tab of the Preferences panel.
