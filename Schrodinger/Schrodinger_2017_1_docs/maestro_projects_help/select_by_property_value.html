Select by Property Value Dialog Box

Select project entries based on the values of a property.

To open this dialog box, choose Select by Property Value from the Property shortcut menu in the Project Table panel.

Select by Property Value Dialog Box Features

  • From options
  • Value condition option menu and text boxes
  • Property statistics button
  • Stars property checkboxes
  • Also save as filter option and text box
  • Selection buttons
From options

Select an option for the set of entries from which the selection by property is made:

  • Selected entries—choose entries by property value from those that are already selected. This is useful for narrowing the selection from a selection made by other means, such as selecting an entry group.
  • All entries—choose entries by property value from all entries.

The way in which the chosen entries are used depends on the button you click to apply the choice.

The option you select also determines the entries for which statistics are calculated for use in property selection.

Value condition option menu and text boxes

Set the condition on the properties that is used to choose entries by property, and enter the relevant values in the text boxes:

  • Values between—choose entries that have values between those in the text boxes.
  • Exact value—choose entries that have the value specified in the text box.
  • Values <—choose entries that have values strictly less than the value specified in the text box
  • Values <=—choose entries that have values less than or equal to the value specified in the text box
  • Values >—choose entries that have values strictly greater than the value specified in the text box
  • Values >=—choose entries that have values greater than or equal to the value specified in the text box
  • No value—choose entries that do not have a value for the property.
  • Any value—choose entries that have a value for the property, regardless of the value itself.

The number of entries that match the condition is reported below these controls.

Property statistics button

Set the values in the text boxes for the value condition from statistics of the property values. The statistics are calculated for the entries specified with the From option, and are reported below the selection report. This button has a menu from which you can choose the statistics to use. For example, you can use this button to choose entries whose values are less than the mean.

Stars property checkboxes

Check the boxes for the numbers of stars to select entries with those numbers of stars. These controls are only present for the Stars property, and the other controls for setting values are absent.

Also save as filter option and text box

Save the set of entries chosen by properties as an entry filter, and enter the name of the filter in the text box. The set is then added to the Filter submenu of the Select menu, and in other places where filters are available.

Selection buttons

Apply the choice of entries by property to the Project Table selection:

  • Select Only—Replace the selection in the Project Table with the set of entries chosen by property.

  • Select Also—Add the set of entries chosen by property to the selection in the Project Table.

  • Deselect—Deselect any entries in the Project Table that are in the set of entries chosen by property.
