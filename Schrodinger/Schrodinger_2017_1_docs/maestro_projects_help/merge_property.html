Merge Property Panel

In this panel, you can merge the values of several properties to form strings that are appended to a destination property or are used to define the values of a new property.

Using the Merge Property Panel

Properties are merged by constructing a string from the source properties, with a separator for the values. The string is appended to the destination property, if an existing property is chosen; otherwise it is used for the new property value, if you choose <Add New Property>. The source properties are not modified in the merge operation.

To merge properties:

  1. Choose source properties in the Available properties list.

  2. Click the right arrow button to the right of the list to move them to the central list.

    This is the list of properties that will be used. The properties are merged in the order that they appear in this list.

  3. Repeat the process to add more properties, or select properties in the central list and click the left arrow to move them back to the Available properties list.

  4. When you have selected all the properties you want to use, select a destination property from the To list, or choose <Add New Property>

  5. Specify a separator in the Separator text box.

  6. If you are creating a new property, enter the name of the property in the New property name text box.

  7. Click All Rows to merge the properties for all rows in the Project Table, or click Selected Rows to merg the properties only for the rows that are selected in the Project Table.

Merge Property Panel Features

  • Available properties property selector
  • Properties to merge list
  • Destination property property selector
  • Separator text box
  • New property name text box
  • Merge values for buttons
Available properties property selector

This property selector lists the properties that are available as source properties. You can select multiple properties and click the right arrow to transfer them to the central property list. To restrict the properties that are displayed, you can choose a subset or filter the properties. You can also sort the properties.

Properties to merge list

This list displays the properties that will be merged. You can select properties and transfer them back to the Available properties list if you don't want them to be merged, or you want to reorder the properties.

Destination property property selector

This property selector lists the properties that are available as destination properties. You can select a single property from this list, or select <Add New Property> to create a new property. To restrict the properties that are displayed, you can choose a subset or filter the properties. You can also sort the properties.

Separator text box

Enter the string that is placed between the property values when they are merged.

New property name text box

If you select <Add New Property> in the To list, this text box becomes available and you can specify the name of the new property.

Merge values for buttons

Click All Rows to merge the properties for all rows in the Project table, or click Selected Rows to merg the properties only for the rows that are selected in the Project Table.
