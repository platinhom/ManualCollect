Choose Entry Dialog Box

The Choose Entry dialog box is used whenever it is necessary to choose an entry from the Project Table, and is opened by a Choose button. The name of the dialog box varies according the purpose for which the entry is being chosen.

Entries can be chosen from three categories: selected entries, included entries, or all entries. You can choose the category from the Choose entry from option menu.

To help you choose an entry, you can select the properties to display in the entry table. Click Show Property, and select the desired properties from the list in the Show Property dialog box, and click OK. The properties are then displayed in the entry table. By default, only the Title and Entry ID are shown.

You can sort the entries into the order shown in the Project Table by clicking Sort by Project Table Order.

To choose the entry, simply select it in the entry table, then click OK.
