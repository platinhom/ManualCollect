Save State

You can store a copy of the project in its current state within the project. To save the state of the project, choose File → Save Checkpoint. This may take some time if the project is large, and Maestro will be unavailable until the state is saved.

This facility provides a single-level version history for a project. Once the state is saved, you can discard the current version of the project and revert to the snapshot version by choosing File → Revert to Checkpoint.
