Delete Atoms

Delete atoms from a structure file, based on a Maestro ASL expression.

Backend implementation

ScriptCenter/CommandLineScripts/deleteatoms.py
The script deleteatoms.py is used to implement this node.

Dialog Options

Output Column Structure
Choose the output columns:

Input plus Output - The input columns are included in the output.

Output replaces Input - The output columns are the same as the input, with the output structures replacing the input column selected

Output only - Only Output structures

ASL Expression
Maestro ASL expression to delete atoms

Ports

Input Ports
0 Molecules in Maestro, SD, or PDB format
Output Ports
0 Molecules without deleted atoms in Maestro format plus input molecules (if specified)

Views

Std output/error of Desalter
Std output/error of Delete Atoms
