View CSV

View a data table with a spreadsheet program, such as one from OpenOffice or MS Office

Backend implementation

This node uses the command line that is specified in Preferences -> KNIME -> Schrodinger under the field "Open Office Spreadsheet Command Line", which defaults to oocalc.

Dialog Options

Include Column Headers
Default is to include the column headers.
Include Row IDs
Default is to include the row IDs.

Ports

Input Ports
0 Input data table
