Ligparse

Cull or filter a collection of structures based on structural composition and structure component counts, retaining the matches. A user-defined criteria file is required.

Backend implementation

utilities/ligparse
The command line tool ligparse (job type 4) is used to implement this node.

Dialog Options

Column containing structure
Select column in the data input table containing the structures
Structure Keys File
Defines descriptors using patterns. (optional, defaults exist)
Composite Types File
Defines combinations of descriptors (optional, defaults exist)
Criteria File
User-defined LigParse filter file. Retain matches. (required)
For additional information see the README file located at 
$SCHRODINGER/macromodel-v*/data/README.ligparse
Output Column Structure
Choose the output columns:

Output replaced within Input columns - The output columns are the same as the input, with the output structures replacing the input column selected

Output only - Only Output structures
