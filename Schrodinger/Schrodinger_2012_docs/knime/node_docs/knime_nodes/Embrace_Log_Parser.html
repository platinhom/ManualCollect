Embrace Log Parser

The Embrace Log Parser inputs a column that has Embrace logs and extracts values into the output. Values include:

Ligand Name
Total Energy
Stretch Energy
Bend Energy
Proper Torsion Energy
Out-of-Plane Energy
Electrostatic Energy
Van-der-Waals Energy
Solvation-SA Energy
Solvation-GB Energy
Fixed-atom Constraint Energy
Non-bonded Energy

Dialog Options

Column containing input
Choose the input column for the external program
Include Input
All columns in the input table are included in the output

Ports

Input Ports
0 Any number of columns, with one Embrace Log column
Output Ports
0 The Extracted Values from the Embrace Log
