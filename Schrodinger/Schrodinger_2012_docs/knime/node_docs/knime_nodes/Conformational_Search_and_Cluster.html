Conformational Search and Cluster

A script for running a quality MacroModel conformation search, then Cluster to reduce the output to high value members.

Platform Limitations: Linux only

Backend implementation

ScriptCenter/MaestroPythonScripts/mmod_ligsearch_cluster.py
The script mmod_ligsearch_cluster.py is used to implement this node.

Dialog Options

Output Column Structure
Choose the output columns:

Input plus Output - The input columns are included in the output.

Output replaces Input - The output columns are the same as the input, with the output structures replacing the input column selected

Output only - Only Output structures
