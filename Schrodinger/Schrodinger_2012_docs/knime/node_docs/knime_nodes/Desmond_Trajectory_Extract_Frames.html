Desmond Trajectory Extract Frames

The trajectory_extract_frame.py script extracts selected frames from a Desmond trajectory into a table of output structures.

Platform Limitations: Linux only

Backend implementation

run -FROM desmond trajectory_extract_frame.py
The script trajectory_extract_frame.py is used to implement this node.

Dialog Options

Column containing Desmond Trajectory
Select column in the data input table containing the Desmond Trajectory
ASL
Enter the ASL expression to specify the part of the structure to be extracted. The default is all. Not valid with pdb or cms output types.
Frames to extract
The list of frames to extract from the trajectory. The list should be comma-separated and can include ranges specified as n:m. Leaving this field blank will allow extracting of all the frames.
Frame frequency
Specify frequency of frames to extract. For example, a value of 2 will extract every other frame in the specified range.
Output type
Select the output structure type: mae, pdb or cms
Location of output CMS files
This is where the output CMS files will be stored if output type chosen is CMS.

Ports

Input Ports
0 Input data table containing Desmond Trajectory
Output Ports
0 Structures in Maestro, PDB or CMS format

Views
