Prime Energy Filter

Sorts and filters complexes according to Prime Energy.

Backend implementation

scripts/ifd/sort-filterCLI.py
The script sort-filterCLI.py is used to implement this node.

Dialog Options

Keep poses
Threshold on property for filtering poses. The syntax is as follows: n% - Keep the n% of poses with the lowest prime energy values n# - Keep the n poses with the lowest prime energy values n - Keep poses with prime energy values within n of the lowest value.

Ports

Input Ports
0 Complex structures in Maestro format
Output Ports
0 Sorted and filtered complex structures in Maestro format
