SD-to-Smiles

Converts each structure in the SD input into SMILES.

Backend implementation

utilities/canvasConvert
canvasConvert is used to convert the SD input to SMILES output.

Dialog Options

Include Input
All columns in the input table are included in the output
Produce unique, canonical SMILES strings
Produce unique, canonical SMILES strings.
Produce kekulized SMILES strings
Produce kekulized SMILES strings of aromatic atoms and bonds.
SD field used to overwrite the name of a molecule
Field in a SD file that can be used to overwrite the name of a molecule (usually the top line). e.g. "ID" or "library-ID" or any existing field in a SD file.

Ports

Input Ports
0 Molecules in SD Format
Output Ports
0 Molecules in Smiles format (2nd column is Molecule title)

Views

Std output/error of SD-to-Smiles
Std output/error of SD-to-Smiles
