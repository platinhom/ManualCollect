XP Visualizer

Runs the XP Visualizer by starting Maestro

Backend implementation

maestro
The maestro command with arguments is used to implement this node.
Note: This node uses temporary files in the Schrodinger temporary directory that start with "internal_". If these files are deleted, or the temporary directory is changed, the node will need to be re-run for it to work properly.

Dialog Options

Column containing input structure
Select the column in the input data table that contains the input pose viewer molecules.

Ports

Input Ports
0 Receptor and ligand molecules. Note: This node will not configure properly unless this input column has the XP Visualization Information (or is configured to have it once the workflow has been executed)
