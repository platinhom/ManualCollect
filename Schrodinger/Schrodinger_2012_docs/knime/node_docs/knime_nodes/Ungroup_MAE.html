Ungroup MAE

Maestro cells in KNIME can contain more than object, whether it is a molecule (i.e., CT), Sequence, or Alignment. This MAE Expander node expands such Maestro cells so that for each one of these objects, a copy of the row is generated. Every data value in the other columns besides the Maestro cell that is expanded is copied from the original row into each of the expanded rows. If there is more than one column with Maestro cells, the configure dialog allows the selection of which column should be expanded.

Ports

Input Ports
0 This table includes a column, which can be a Sequence, Alignment, or Maestro format
Output Ports
0 This table includes the input that is expanded on the selected column
