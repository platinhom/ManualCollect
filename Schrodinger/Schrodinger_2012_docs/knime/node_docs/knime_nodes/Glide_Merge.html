Glide Merge

Merge a set of sorted Glide pose files into a single sorted Glide pose file. This assumes that the input files have already been sorted.

Backend implementation

utilities/glide_merge
The command line tool glide_merge is used to implement this node.

Ports

Input Ports
0 Pose viewer 1 in Maestro format
1 Pose viewer 2 in Maestro format
Output Ports
0 Pose viewer file in Maestro format

Views

Std output/error of glide_merge
Std output/error of glide_merge
