Convert Matrix to Table

Converts a binary Canvas Matrix into the output table

Backend implementation

utilities/canvasPW2CSV
canvasPW2CSV is used to implement this node.

Ports

Input Ports
0 Binary Canvas Matrix (a binary pairwise similarity or distance matrix)
Output Ports
0 A table consisting of the values in the input Matrix. The first column and the column headers are the molecule names.

Views

Std output/error of Matrix-to-CSV
Std output/error of Matrix-to-CSV
