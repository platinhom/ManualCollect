Package graphics3d

Various classes for drawing 3D objects in the Maestro workspace via OpenGL.

Can be used to draw 3D objects into the current OpenGL context. Also can be used to draw into Maestro's Workspace by registering drawing functions. The registration functions are located in the schrodinger.maestro.maestro module
