Schrödinger Suite 2012 Update 2 Documentation


A complete list of Schrödinger documentation for the latest release is given in the side bar. The documentation is installed in the directory $SCHRODINGER/docs. The documentation is either in PDF format or in HTML format.

Installation and configuration instructions for Schrödinger products can be found in the Installation Guide. Information on managing jobs can be found in the Job Control Guide. These documents are included in the General Documentation set.

For legal information on Schrödinger software and documentation, see the Legal Notice. The Schrödinger software distribution also includes third-party products. See the Legal Notices for Third-Party Software for full details of third-party software copyrights, terms, and conditions.

The most recent documentation is posted on the Support Center of the Schrödinger web site. If you have problems that are not addressed in the manuals or the Maestro online help, try the Knowledge Base, or the Known Issues pages on the Support Center. You should always check the Support Center for the latest information.

Note: If your browser does not open the PDF documents using a plug-in PDF reader, the live links between documents might not work. To avoid this problem, open the document directly in your PDF reader, or open the PDF version of the documentation index, Manuals_Index.pdf directly in your PDF reader, and use the links in that file to open the documentation. You can open this file from the Maestro Help menu.
