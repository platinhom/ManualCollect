New Features in Canvas 1.5

    Spreadsheet

  • The Chemistry menu has been renamed Structure. With it, the Chemistry Filter dialog box has been renamed Structure Filter and has been moved from the Data menu to the Structure menu. Some menu items have been renamed as a consequence of the change.

  • Structure names can be replaced with the values of a selected property.

  • Structures that are found to be invalid on import are marked as invalid in the Structure column. You can select these structures with View → Select Rows with Invalid Structures. (Update 2)

  • Hydrogen atoms in the 2D structures can be shown or hidden with a new item on the Structure menu.

  • The Property Calculator panel has been renamed Calculator.

  • Properties are exported in the order in which they apppear in the spreadsheet (or view).

  • You can restrict the list of properties available for plotting in a chart by selecting the desired properties in the current view before plotting.

  • You can add an empty row or an empty column to the spreadsheet from the Edit menu or using the toolbar.

  • You can select the columns to show in the view in the Manage Properties dialog box, which you open from the View menu.

  • You can copy the selection from the spreadsheet to the clipboard. A rectangular region containing the selection is what is copied.

  • Class values in partitions can be edited.

  • An Apply button has been added to the Partition Filter dialog box, so you can apply a succession of filters.

  • The 2D structures of the matches can be aligned to a reference structure after a substructure query. The aligned atoms are those in the substructure.

  • The number of decimal places displayed in the spreadsheet can be set as a preference. (Update 2)

    • Applications

  • The layout of many of the model-building dialog boxes has been improved, by creating tabs for the job setup and the results rather than displaying them in a single window. (Update 2)

  • Filling in holes in an existing library along with property optimization can be done with a new application, canvasLibOpt, and the corresponding panel on the Applications menu, Hole-Filling and Library Optimization.

    In Update 2, you can use existing property filters.

  • A set of structures can be decomposed into its unique (ring-containing) scaffolds with a new application, canvasScaffold, and the corresponding panel on the Applications menu, Scaffold Decomposition. Once scaffolds are located, a this set can be matched against a set of structures to locate the structures that match each scaffold.

  • QikProp descriptors have been added to the Molecular Descriptors panel. (Update 2)

  • Kernel-based partial least squares regression is now available as a new application on the Applications menu. Several improvements have been made to the Kernel-Based Partial Least Squares application in Update 2: uncertainty column in results, color scale limits, sorting, layout.

  • Shape screening now uses pharmacophore types by default. This is a new option, set in the Advanced Options dialog box. (Update 2)

  • When using a view for input to an application, you can choose to use the rows or the columns from the view, or both. (Update 2)

  • The output rows from canvasFPMatrix can be sorted, and only the top rows returned. For each row, the top columns can be returned.

  • The Shannon entropy can be used to remove properties in the Feature Selection dialog box, as well as or instead of the percent identity. (Update 2)

  • In a self-organizing map, the value used to determine the color of a cell can be displayed in the cell.

  • You can save an image of a self-organizing map.

  • In the hierarchical clustering dendrogram, you can color the leaf nodes by cluster membership or by the value of a property.

  • When exporting clusters, you can export just the centroids to a file or a view.

  • You can save an image of the hierarchical clustering dendrogram.

  • The data set selection and property selection tools have been improved.

  • You can set a preference to incorporate jobs automatically, rather than incorporate each job manually. (Update 2)

    • Projects

  • You can export a saved view from a Canvas project from the command line, with canvasProjectDB.

  • Files in SD format can be imported into Canvas when starting it from the command line.

  • When exporting properties, you can choose to autoscale them so the mean is zero and the standard deviation is 1. (Update 2)
