Getting Help

Canvas Online Help

The Canvas online help facility contains information about the panels in Canvas and how to use them. In this beta release of Canvas, the online help is opened from the Help menu.

The browser window is divided into two frames, the navigation frame and the content frame. The content frame is used to display help topics. The navigation frame has two tabs:

  • Topics—A list of topic categories is displayed by default. This list has links to a list of topics in each category, which replaces the topic list when you click the link.
  • Search—The search facility allows you to search for words in various combinations. The search results have links to the topics found, and the search words are highlighted in these topics when you click these links to open them.

Other Sources of Information

If you don't find the information you need in the online help, try the following sources:

  • The user manual. To open the manual, choose Help → User Manual.

  • The Knowledge Base, at http://www.schrodinger.com/kb. You can go to the Knowledge Base from the Help menu.

  • The Known Issues page on the Support Center.

All the documentation for all products, both PDF and HTML, is available from the HTML manuals index

Generally, e-mail correspondence is best because you can send machine output, if necessary. When sending e-mail messages, please include the following information:

  • A description of the problem and the operation being performed when it occurred
  • All relevant user input and machine output
  • Product purchaser (company, research institution, or individual)
  • Primary Product user
  • Computer platform type
  • Operating system with version number
  • Product version number
  • Maestro version number
  • mmshare version number

The best way of providing the information is to open the Diagnostics panel from the Help menu and click Technical Support when the diagnostics have run. This creates a file, which you should attach to your email message. The location of the file is given in a dialog box.
