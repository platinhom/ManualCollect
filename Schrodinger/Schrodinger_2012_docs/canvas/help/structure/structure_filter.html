Summary

This dialog box allows you to filter the structures on counts of SMARTS patterns.

Opening the Structure Filter Dialog Box

To open the Structure Filter dialog box:

  • Choose Data > Structure Filter in the main window.

Using the Structure Filter Dialog Box

The filters in this dialog box are specified in the rows of the table. In the SMARTS column you can enter a SMARTS pattern to filter on, and in the Min and Max columns, you can enter the minimum and maximum number of occurrences of the SMARTS pattern in a structure that are required for the structure to pass through the filter. The comments column allows you to annotate the SMARTS pattern. Complex filters can be set up by using multiple rows. The structures that pass all the filters are returned when you run the job.

Structure Filter Dialog Box Features

  • Use existing filter option menu
  • Import button
  • Invert (retain failures) option
  • Store results in options
  • Filter table
  • Insert and Delete buttons
  • Run button
  • Save button
  • Export button
Use existing filter option menu

Choose an existing filter to apply to the data. Filters can be created in this dialog box using the Save button.

Import button

Import a previously exported chemistry filter or import structures that are used to generate SMARTS patterns. Opens a file selector, in which you can navigate to and choose the filter file (.flt), or a structure file in Maestro (.mae, .mae.gz, .maegz), SD (.sd, .sdf, .sd.gz, .sdf.gz), or SMILES only (.smi) format. When structures are imported, they are converted to SMARTS patterns and added to the table, one row per structure in the file.

Invert (retain failures) option

Invert the filter, so that the structures that do not pass the filter are returned, rather than the structures that do pass.

Store results in options

Choose the view to store the results in. If you choose New View, you are prompted for the name of the view when you run the job.

Filter table

Table that defines the filter to be applied. The table rows are described below.

SMARTS SMARTS pattern to be matched in each structure.
Min Minimum number of occurrences of the SMARTS pattern in a structure for the structure to pass through the filter. If this column is left blank, no minimum is applied, and you must set a maximum.
Max Maximum number of occurrences of the SMARTS pattern in a structure for the structure to pass through the filter. If this column is left blank, no maximum is applied, and you must specify a minimum.
Comments Enter comments, for example on the SMARTS pattern and its limits.
Insert and Delete buttons

Insert a row in the table above the selected row or delete the selected rows.

Run button

Run the filtering job.

Save button

Save the filter in the project for later use. Opens a dialog box in which you can name the filter. The filter is then listed on the Use existing filter option menu and under Chemistry Filters in the Project View window. If you want to store the filter outside the project, for example, to import into another project, use Export.

Export button

Export a chemistry filter to a file. Opens a file selector, in which you can navigate to a location and name the filter file (.flt).
