Data Menu

The Data menu allows you to perform various operations on the table data. The items on the Data menu are as follows.

Sort

Sort the table rows by the values in one of the columns. Opens the Sort dialog box.

Property Filter

Filter the data by a set of conditions on the properties. Opens the Property Filter dialog box.

Heat Map

Create a heat map from selected properties.

Statistics

Calculate univariate and bivariate statistics for properties in the current view. Opens the Statistics dialog box.

Calculator

Calculate new properties as a function of existing properties. Opens the Calculator dialog box.

Edit Spreadsheet

Edit table cells in the spreadsheet directly. Enters Edit mode, in which you can only edit the table cells. Cells containing structures and cells belonging to a partition cannot be edited in this mode. Changes are not saved until you choose Data → Save Spreadsheet Changes.

Undo Last Edit

Undo the last editing operation.
Keyboard shortcut: CTRL+Z (⌘Z).

Redo Last Edit

Redo the last editing operation that was undone.
Keyboard shortcut: CTRL+SHIFT+Z (⇧⌘Z).

Save Spreadsheet Changes

Save the changes made by editing the spreadsheet, and exit Edit mode.

Exit Without Saving Spreadsheet Changes

Discard the changes made by editing the spreadsheet and exit Edit mode. The spreadsheet is restored to the state it was in when you entered Edit mode.
