Summary

Perform a principal components analysis for a set of source properties.

Opening the Principal Components Analysis Dialog Box

To open the Principal Components Analysis dialog box:

  • Choose Applications → Principal Components Analysis in the main window.

Principal Components Analysis Dialog Box Features

  • Data set option menu
  • Save results as text box
  • Options tab
  • Scores tab
  • Principal Components tab
  • Action buttons
Data set option menu

Select the source of the data. The data source can involve both row and column selection. The choices are:

  • Selected rows—Use the rows that are selected in the master view. You must select the rows before opening the dialog box.

  • Visible rows—Use the rows that are displayed (visible) in the master view. You must ensure that the required rows are visible before opening the dialog box.

  • Saved view—Use the rows and columns in a chosen saved view. When you select this item, an option menu that lists the saved views in the project is displayed to the right. You can choose the desired view from this option menu. A second option menu, labeled Domain, allows you to choose whether to restrict the rows or the columns, or both, to those of the saved view. Otherwise the visible rows or columns of the Master View are used.

  • Original data set for jobname—Use the rows and columns that were used for the specified job. This item is only available when you are cloning a job or viewing the results of a job.

Save results as text box

Assign a name to the results. This name is listed under Principal Components Analysis in the Project View window.

Options tab

In this tab, you can set up the input and options for the job.

  • Number of PCs box
  • Autoscale when property variance > X option and text box
  • Source properties section
  • Property type option menu
  • Property selection tools
  • Fingerprint column option menu
  • Most informative bits box
Number of PCs box

Set the maximum number of principal components to generate.

Autoscale when property variance > X option and text box

Scale the property values by dividing by the standard deviation if the variance is greater than the specified value.

Source properties section

Select the properties or fingerprints that are used as the independent variables in this section

Property type option menu

Choose the type of source property you want to use: From properties or From fingerprints. The tools for the chosen type of source property are displayed below this option menu.

Property selection tools

Use these property selection tools to select the source properties.

Fingerprint column option menu

Choose the fingerprint column to use from this option menu.

Most informative bits box

Keep only the specified number of most informative bits, when a fingerprint is used for analysis.

Scores tab

This tab contains a table of principal component values for each structure in the dataset. To export the data to a file or the spreadsheet, select the desired columns and right-click in the heading of one of the columns. An Export Selection dialog box opens, in which you can choose the destination and specify a prefix for the property column in the spreadsheet. If you pause the pointer over the structure name, the 2D structure is displayed in a tool tip.

To export the table data, select the rows you want to export, then right-click and choose Export to File or Export to Spreadsheet. For export to a file, data in all columns for the selected rows are written to a CSV file with column headings. Exporting to the spreadsheet adds the predicted values to the spreadsheet with a prefix that you can change.

Principal Components tab

This tab contains a table of coefficients of the properties in each principal component.

Action buttons

  • Plot Loadings button
  • Compute button
  • Plot Scores button
Plot Loadings button

Display a scatter plot of the loadings of two to four principal components. The property name for each point on the plot is displayed in a tool tip.

Compute button

Run the job to generate the principal components and compute the scores.

Plot Scores button

Display a scatter plot of the scores for each molecule for two to four principal components. By default, the scores for the first two components are displayed. The structure and name of the molecule is displayed in a tool tip for each point.
