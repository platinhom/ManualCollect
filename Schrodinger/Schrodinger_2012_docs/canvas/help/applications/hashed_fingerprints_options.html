Summary

In this dialog box you can set the parameters used to generate hashed fingerprints.

Opening the Hashed Fingerprints - Advanced Options Dialog Box

To open the Hashed Fingerprints - Advanced Options dialog box:

  • Click Advanced in the Binary Fingerprints from Structures panel.

Hashed Fingerprints - Advanced Options Dialog Box Features

  • Atom/bond typing section
  • Use 3D distancesoption
  • Precision options
  • Minimum linear path box
  • Maximum linear path box
  • Maximum linear + ring closure path box
  • Radial iterations box
  • Discard bits that are ON in fewer than N % of molecules option and box
  • Discard bits that are ON in more than N % of molecules option and box
  • Scaling list
Atom/bond typing section

Select the atom and bond typing scheme to be used to classify atoms and bonds in the structures. If you choose Custom, click Import Custom Types to locate the file that contains the custom atom and bond types. The file must contain SMARTS patterns for the fingerprints, one per line.

Use 3D distancesoption

Use actual 3D coordinate distances instead of topological distances.

Precision options

Select an option for the precision of the fingerprints. Selecting 64-bit reduces collisions of "on" bits, but doubles the space required to store each key.

Minimum linear path box

Minimum path length for linear, dendritic, and molprint2D fingerprint types. Defaults: 0 (linear, dendritic), 2 (molprint2D).

Maximum linear path box

Maximum path length for linear, dendritic, and molprint2D fingerprint types. Defaults: 7 (linear), 5(dendritic), 2 (molprint2D).

Maximum linear + ring closure path box

Maximum linear ring path. Valid only with linear fingerprint type. 0 means no ring closure. Default: 14.

Radial iterations box

Number of iterations for radial fingerprints. Valid only with radial fingerprint type. Usually between 2 and 6. Default: 4.

Discard bits that are ON in fewer than N % of molecules option and box

Select this option to discard bits that are set by fewer than the specified percentage of molecules. Default: 5%.

Discard bits that are ON in more than N % of molecules option and box

Select this option to discard bits that are set by more than the specified percentage of molecules. Default: 95%.

Scaling list

Select a method from this list to scale the binary fingerprint data to real numbers. Default: No scaling.
