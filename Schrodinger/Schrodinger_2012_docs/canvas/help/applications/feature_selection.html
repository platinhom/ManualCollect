Summary

Identify a subset of properties that are representative of a larger set, using hierarchical clustering on the correlation matrix, followed by multiple linear regression on the subset. This procedure is useful for reducing a large number of related properties to a smaller set that has little linear dependence.

Opening the Feature Selection Dialog Box

To open the Feature Selection dialog box:

  • Choose Applications → Feature Selection in the main window.

Using the Feature Selection Dialog Box

To run the feature selection job, select the structure source, then select the properties that you want to reduce. You should check the list for properties that are not relevant (such as an entry ID) or came from other jobs.

When the job finishes, right-click on it in the Project View panel and choose View. The results are displayed in the Feature Selection Viewer panel.

The feature selection procedure uses hierarchical clustering to identify properties that are representative of a larger set. The distance between each pair of properties is one minus the absolute value of the correlation matrix element. The property that is closest to the centroid of each cluster is selected as the representative property.

Feature Selection Dialog Box Features

  • Data set option menu
  • Save results as text box
  • Properties section
  • Exclude properties whose values are identical in more than N % of molecules option and box
  • Exclude properties with Shannon entropy lower than X option and text box
Data set option menu

Select the source of the data. The data source can involve both row and column selection. The choices are:

  • Selected rows—Use the rows that are selected in the master view. You must select the rows before opening the dialog box.

  • Visible rows—Use the rows that are displayed (visible) in the master view. You must ensure that the required rows are visible before opening the dialog box.

  • Saved view—Use the rows and columns in a chosen saved view. When you select this item, an option menu that lists the saved views in the project is displayed to the right. You can choose the desired view from this option menu. A second option menu, labeled Domain, allows you to choose whether to restrict the rows or the columns, or both, to those of the saved view. Otherwise the visible rows or columns of the Master View are used.

  • Original data set for jobname—Use the rows and columns that were used for the specified job. This item is only available when you are cloning a job or viewing the results of a job.

Save results as text box

Assign a name to the results. This name is listed under Feature Selection in the Project View window.

Properties section

Select the properties that you want to reduce to a representative set, using the property selection tools.

Exclude properties whose values are identical in more than N % of molecules option and box

Select this option to remove from the calculation any properties whose values are identical in more than the specified percentage of molecules. As these properties have little variation in their values they are not likely to be useful in other applications.

Exclude properties with Shannon entropy lower than X option and text box

Select this option to remove from the calculation any properties whose Shannon entropy is lower than the specified cutoff value. The Shannon entropy is defined by

−Σi=1,n Pi / log10Pi

where n is the number of distinct values that the property has, and Pi is the fraction of properties that have a given value. As these properties have little variation in their values they are not likely to be useful in other applications.
