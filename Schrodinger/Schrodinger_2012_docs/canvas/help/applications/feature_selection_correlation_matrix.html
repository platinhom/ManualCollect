Summary

This dialog box shows the correlation matrix for either the original property set or the reduced property set from a feature selection job.

Opening the Correlation Matrix Dialog Box

To open the Correlation Matrix dialog box:

  • Click Correlation Matrix in the Feature Selection Viewer panel.

Correlation Matrix Dialog Box Features

  • Color by value option and controls
  • Order by cluster option
  • Correlation matrix
Color by value option and controls

Color the cells in the correlation matrix according to their value (as a heat map). The color represents the absolute value of the property on the range defined by the maximum and minimum values set in the boxes. Values above the maximum are set to the color for the maximum, shown next to the maximum value box. Likewise, values below the minimum are set to the color for the minimum, shown next to the minimum value box. You can use either the slider or the boxes to change the maximum and minimum values represented by the color gradient.

Order by cluster option

Sort the properties into the order determined by the hierarchical clustering. The default is to sort alphabetically (ASCII sort order). Ordering by the clustering order should place the largest matrix elements close to the diagonal. This makes it easier to detect larger elements between clusters.

Correlation matrix

The correlation matrix is displayed in a table, with cells colored according to the absolute value of the matrix element, as described above.
