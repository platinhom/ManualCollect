Summary

In this dialog box you can set up and run a job to create fingerprints from 3D structures using 3-point or 4-point pharmacophore features, with optional generation of conformers of the 3D structures. The structures must be all-atom structures: structures that do not have hydrogen atoms will not be processed.

Opening the 3D Pharmacophore Fingerprints Dialog Box

To open the 3D Pharmacophore Fingerprints dialog box:

  • Choose Applications → 3D Pharmacophore Fingerprints in the main window.

3D Pharmacophore Fingerprints Dialog Box Features

  • Data set option menu
  • New fingerprint column name text box
  • 3-point pharmacophores option
  • 4-point pharmacophore option
  • Use existing 3D structures option
  • Generate conformers option
  • Advanced button
Data set option menu

Select the source of the data. The data source can involve both row and column selection. The choices are:

  • Selected rows—Use the rows that are selected in the master view. You must select the rows before opening the dialog box.

  • Visible rows—Use the rows that are displayed (visible) in the master view. You must ensure that the required rows are visible before opening the dialog box.

  • Saved view—Use the rows and columns in a chosen saved view. When you select this item, an option menu that lists the saved views in the project is displayed to the right. You can choose the desired view from this option menu. A second option menu, labeled Domain, allows you to choose whether to restrict the rows or the columns, or both, to those of the saved view. Otherwise the visible rows or columns of the Master View are used.

  • Original data set for jobname—Use the rows and columns that were used for the specified job. This item is only available when you are cloning a job or viewing the results of a job.

New fingerprint column name text box

Enter a name for the new fingerprint column. This name is listed under 3D Pharmacophore Fingerprints in the Project View window, and is used for the column header when a column is added to the spreadsheet with the bit count for the fingerprint. You must provide a name before you can run the job. A default name is supplied.

3-point pharmacophores option

Select this option to use 3-point pharmacophores for the fingerprint generation. This option is selected by default.

4-point pharmacophore option

Select this option to use 4-point pharmacophores for the fingerprint generation. This option is not selected by default. 4-point pharmacophores usually set an order of magnitude more bits than 3-point pharmacophores, and do not provide much additional benefit.

Use existing 3D structures option

Use the existing 3D structures for pharmacophore models. No conformers of these structures are generated, and no detection of conformers is applied if the structures come from a file. This means that if the file does contain conformer sets, they will be considered as separate structures. Conformers are not stored in Canvas projects, so using a Canvas project for input also treats the structures as separate.

Generate conformers option

Generate conformers of the input structures, and use these conformers for pharmacophore models. The input structures are only used to seed the conformational search, and are not used directly for pharmacophore models. Once the conformers have been used to create pharmacophore models, they are discarded. Options for the conformational search can be set in the 3D Pharmacophore Fingerprints - Advanced Options dialog box.

Advanced button

Open the 3D Pharmacophore Fingerprints - Advanced Options dialog box, in which you can import feature definitions, set the fingerprint precision, modify the binning of inter-feature distances, set options and thresholds for conformer generation, and set options for filtering bits.
