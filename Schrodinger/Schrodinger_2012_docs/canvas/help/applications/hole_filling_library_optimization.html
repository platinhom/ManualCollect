Summary

This application allows you to select a subset of structures from a pool to minimize the similarity to other structures and optionally to minimize the number of properties whose values lie outside given ranges (property optimization). The structures can be chosen to fill holes in a library, and the similarity to the library is then taken into account in the minimization.

Opening the Hole-Filling and Library Optimization Dialog Box

To open the Hole-Filling and Library Optimization dialog box:

  • Choose Applications → Hole-Filling and Library Optimization in the main window.

Using the Hole-Filling and Library Optimization Dialog Box

If you are using this application to fill holes in a library, both the library and the source of structures to choose from to fill the holes must be in the project. The library must exist as a named view in the project. If you are importing the library from a file, select Store each input file as a view.

The similarity minimization is based on the Tanimoto similarity of a set of fingerprints for the structures. You must therefore generate fingerprints, and the fingerprints for the library and the structures used to fill the library must be of the same type.

Choose structures from option menu
Data set option menu

Select the source of the data. The data source can involve both row and column selection. The choices are:

  • Selected rows—Use the rows that are selected in the master view. You must select the rows before opening the dialog box.

  • Visible rows—Use the rows that are displayed (visible) in the master view. You must ensure that the required rows are visible before opening the dialog box.

  • Saved view—Use the rows and columns in a chosen saved view. When you select this item, an option menu that lists the saved views in the project is displayed to the right. You can choose the desired view from this option menu. A second option menu, labeled Domain, allows you to choose whether to restrict the rows or the columns, or both, to those of the saved view. Otherwise the visible rows or columns of the Master View are used.

  • Original data set for jobname—Use the rows and columns that were used for the specified job. This item is only available when you are cloning a job or viewing the results of a job.

Save results as text box

Assign a name to the results. This name is listed under Hole-Filling and Library Optimization in the Project View window.

Fingerprint column option menu

Choose the fingerprint column that is used to determine the similarities used in hole filling from this option menu. You must have fingerprints of the same type for the library and for the structures used to fill the library.

Number of structures to select box

Enter the number of structures that you want to select from the specified source. The computational effort scales as the square of the number of structures selected.

Hole-filling option

Select this option to fill holes in the existing library by choosing the structures with the lowest similarity to any other member of the selected set or the library.

Saved view containing library with holes option menu

Specify the library whose holes you want to fill, by choosing from among the saved views in the project. A saved view is automatically created when you select Store each input file as a view in the input options dialog box when you import the library from a file.

Optimize properties option and Properties button

Select this option to optimize the values of chosen properties. To choose the properties you want to optimize, click Properties, which opens the Hole-Filling and Library Optimization - Property Filter dialog box.

The properties are optimized on a simple score based on whether the property for a structure lies within the specified range or not. The score is the fraction of properties for which a value did not lie in the specified range (a "failure"); this fraction is averaged over all structures to define the score. The score is minimized in the optimization procedure.

Advanced Options button

Open the Hole-Filling and Library Optimization - Advanced Options dialog box, to set options that control the optimization process.
