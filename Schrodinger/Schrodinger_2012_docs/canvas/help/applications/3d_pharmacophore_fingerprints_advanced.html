Summary

In this dialog box you can set options and thresholds for pharmacophore feature definitions, inter-feature distance binning, conformer generation, and bit filtering. The options used for the canvasPharmFP command are indicated in the descriptions.

Opening the 3D Pharmacophore Fingerprints - Advanced Options Dialog Box

To open the 3D Pharmacophore Fingerprints - Advanced Options dialog box:

  • Click Advanced in the 3D Pharmacophore Fingerprints panel.

3D Pharmacophore Fingerprints - Advanced Options Dialog Box Features

  • Use custom feature definitions option and file controls
  • Precision options
  • Inter-feature distance bins section
  • Conformer generation section
  • Filter bits by section

Use custom feature definitions option and file controls

Select Use custom feature definitions if you want to specify a pharmacophore feature definition file. The file must be in the format used by Phase. The default feature definitions are the Phase defaults. (Sets the -fd command option.)

Precision options

Set the precision for the fingerprints. The default is 32-bit fingerprints. A larger number of bits reduces the incidence of collisions; 32 bits are enough for most purposes. (Sets the -xp command option.)

Inter-feature distance bins section

Specify the bins used for classifying the inter-feature distances. The fingerprint is set on the basis of the occupancy of these bins.

  • Lower limit text box
  • Upper limit text box
  • Bin width text box
  • Bin overlap text box
Lower limit text box

Minimum distance between pharmacophore features in angstroms. Features that are closer than this distance will not be used in the same pharmacophore. The default is 2.0 Å. (Sets the -dmin command option.)

Upper limit text box

Maximum distance between pharmacophore features in angstroms. Features that are further apart than this distance will not be used in the same pharmacophore. The default is 16.0 Å. (Sets the -dmax command option.)

Bin width text box

Distance bin width in angstroms. The default is 2.0 Å. (Sets the -width command option.)

Bin overlap text box

Set the bit for a neighboring bin if a distance is within the specified distance from that bin. The default is half the bin width. (Sets the -overlap command option.)

Conformer generation section

In this section you can set options for conformer generation. This section is available when you select Generate conformers in the 3d Pharmacophore Fingerprints panel. Conformers are discarded once they have been used for pharmacophore generation.

  • Sampling options
  • Amide bond torsions option menu
  • Number of conformers per rotatable bond box
  • Maximum number of conformers per structure box
  • Relative energy window text box
Sampling options

Set the sampling level for conformer generation. (Sets the -sample command option.) The options are:

  • Rapid: All the core conformations are generated, and the conformations of the peripheral (rotamer) groups are sampled one by one. This is the default.

  • Thorough: A complete set of conformations is generated for both the core and the peripheral groups.

Amide bond torsions option menu

Set options for sampling of amide bond torsions. (Sets the -amide command option.)

  • Vary—This option allows the amide dihedral angle to be varied. It can take any value, not just cis or trans.
  • Retain original—Retain the conformation that was in the input structure.
  • Set to trans—Set the conformation to trans for all amide bonds.
Number of conformers per rotatable bond box

Limit the number of conformers returned to the specified multiple of the number of rotatable bonds in the structure. This number is also capped by the maximum number specified below. (Sets the -bf command option.)

Maximum number of conformers per structure box

Limit the number of conformers returned per structure to the value given. If the structure has more conformers, a sample of the specified size is taken of the conformers. (Sets the -cmax command option.)

Relative energy window text box

Discard conformers that are more than the specified amount higher in energy than the lowest energy conformer. (Sets the -ewin command option.)

Filter bits by section

In this section you can choose how to filter the bits, to reduce the number of bits that are stored in the fingerprint.

  • ON/OFF frequency option
  • Discard bits that are on in fewer than N % of molecules option and box
  • Discard bits that are on in more than N % of molecules option and box
  • Information content option
  • Retain the N most informative bits option and box
ON/OFF frequency option

Select this option to discard bits that are on in most molecules or only a few molecules. These bits are not usually useful for distinguishing molecules. When you select this option, you can then choose whether to discard high-frequency or low-frequency bits and set a threshold for the percentage of molecules that set or do not set the bits.

Discard bits that are on in fewer than N % of molecules option and box

Omit bits that are set by less than the specified percentage of input structures. The default is to include all bits. (Sets the -fmin command option.)

Discard bits that are on in more than N % of molecules option and box

Omit bits that are set by more than the specified percentage of input structures. The default is to include all bits. (Sets the -fmax command option.)

Information content option

Set this option to filter bits by their information content.

Retain the N most informative bits option and box

Select this option to keep the most informative bits, and enter the number of bits to be kept in the text box. (Sets the -mostSig command option.)
