Summary

In this dialog box you can perform tasks to manage the scripts on the Scripts menu.

Opening the Manage Scripts Dialog Box

To open the Manage Scripts dialog box:

  • Choose Scripts → Manage Scripts in the main window.

Manage Scripts Dialog Box Features

  • Contents of the Scripts menu table
  • Delete button
  • Description button
  • Reload button
Contents of the Scripts menu table

This table lists all the scripts currently installed and visible in the Scripts menu. The Name field lists the name of the script as it appears in the Scripts menu. The Command field lists the command that is run when you choose the script from the Scripts menu. You can select a script and apply an action to it with the buttons described below.

Delete button

Select a script in the table and click this button to remove it from the Scripts menu.

Description button

Select a script in the table and click this button to display its description in a dialog box. The button is not available if the script does not have a description or if no script is selected.

Reload button

Select a script in the table and click this button to reload the script after it has been modified.
