Summary

In this dialog box you can set up the axes and their labels.

Opening the Axes Dialog Box

To open the Axes dialog box:

  • Choose Settings → Axes in the Scatter Plot panel.

Using the Axes Dialog Box

Axes Dialog Box Features

This dialog box has two tabs, one for each axis. The controls in each tab are identical, and are described below.

  • Axis limits section
  • Axis line section
  • Tick marks section
  • Tick mark labels section
  • Axis title section
  • Color buttons
Axis limits section

Set the minimum and maximum values on the axis, and set the value of the coordinate at which the other axis crosses.

Axis line section

Set the color and weight of the axis line, in points.

Tick marks section

Set the spacing of the tick marks and the coordinate value at which the marks should start; and set the color and weight of the marks. If you want to display grid lines in the plot at the position of the tick marks, select Grid lines.

Tick mark labels section

Set the spacing and starting value for the tick mark labels, the font size in points, the color, and the number of decimal places displayed. The spacing should be a multiple of the tick mark spacing.

Axis title section

Set the axis title, font size in points, and color of the text.

Color buttons

The Color buttons open a color chooser, in which you can select the desired color. The selected color is displayed to the right of the button.
