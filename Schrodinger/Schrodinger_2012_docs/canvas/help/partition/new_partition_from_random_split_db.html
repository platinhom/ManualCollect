Summary

In this dialog box you can create a partition in which the rows are distributed randomly between the classes. This feature can be used to creating training and test sets of structures, for example, for applications that do not offer an assignment choice.

Opening the New Partition from Random Split Dialog Box

To open the New Partition from Random Split dialog box:

  • Choose Partition > New > Random Split in the main window.

Create Random Partition Dialog Box Features

  • Name text box
  • Description text box
  • Number of classes box
  • Seed text box
  • Update button
  • Classes table
Name text box

Enter the name for the partition in this text box

Description text box

Enter the description for the partition in this text box. The description can include any information — for example, you might want to copy the seed used to generate the classes into this text box for reference.

Number of classes box

Enter the number of classes that you want in the partition in this text box. Tne default is 2. Rows are divided evenly between the classes.

Seed text box

Enter the seed for generating the random numbers used to assign the rows to classes in this text box.

Update button

Update the assignment of rows to classes after changing the number of classes. The Classes table is updated with the new partitioning.

Classes table

This table has three columns:

  • Value—Value for the class. The default assignment is a sequence of integers. You can change the values by editing the table cells. For example, you might want to use text labels, like "test set" and "training set".
  • Count—Number of members in the class. This column is noneditable.
  • Color—Color used to color the table cells for members of the class. You can change the color for the class by right-clicking in a cell and choosing Change Color. A color selector opens, in which you can choose the color you want to use.
