Project Menu

The Project menu gives you access to information about the Canvas project.

  • Project View—This option displays or hides the Project View panel. This panel displays a tree view of the project, which provides access to all the objects created in the project, such as views, jobs run, and so on. By default, it is docked into the main window. You can undock it and redock it by clicking the docking button.

  • Messages View—This option displays or hides the Messages View panel. This panel displays log and error messages from various operations that are done on the project. By default, it is docked into the main window. You can undock it and redock it by clicking the docking button.
