Summary

The ePlayer is a tool for displaying the selected entries in the current project one by one, with options to superimpose entries and run a script as each entry is displayed. It is operated from the Project Table panel.

Overview

The ePlayer controls are available on the ePlayer toolbar and from the ePlayer menu in the Project Table panel. The toolbar controls are like most other player controls: you can skip to either end, step in either direction one at a time (Step mode), or play in either direction (Continuous mode).

In addition to simply displaying each entry in the Workspace, you can perform a superposition or execute a script when each entry is displayed. The options for these actions are available in the ePlayer Options panel, which you open by choosing ePlayer → Options in the Project Table panel.

The ePlayer includes each of the selected entries in the Workspace without affecting any unselected entries. Atoms in the Workspace that do not belong to any of the selected entries are left in place during ePlayer operation, to provide context. So, for example, you could include a protein in the Workspace, then select a number of ligands and play through the ligands. Each would be displayed in turn with the protein.

Using the ePlayer in Step Mode

Step mode is a means of including entries in the Workspace consecutively. As each entry is included, superposition or script execution takes place if you have selected these options. Any changes to the project are made according to the synchronization options you have selected in the Preferences panel. In step mode, the ePlayer "remembers" the last entry that was displayed by the ePlayer. When you click the Previous or Next toolbar buttons, the entry that is before or after the last entry displayed becomes the new displayed entry.

Using the ePlayer in Continuous Mode

Continuous mode is a means of automatically displaying the selected entries in sequence. Unlike step mode, the synchronization preferences are ignored. Instead, there is an option in the ePlayer Options panel to save Workspace changes during continuous mode. You can also control the rate at which entries are displayed from this panel.

Continuous play mode has two options for looping, which you can select from the Loop submenu of the ePlayer menu or from the Loop toolbar button menu. Single direction plays through the entries in the chosen direction to the end of the selection, then starts again at the other end. Oscillate plays through the entries in the chosen direction to the end of the selection, then reverses direction and plays through the entries to the other end, and reverses direction again. You can stop play by pressing ESC.

Superimposing Entries

You can superimpose the entries as you step or play through them, by choosing an option in the ePlayer Options panel. The two options are to superimpose each entry on the last entry, or to superimpose each entry on a reference entry. The reference entry does not have to be one of the selected entries, and it does not have to be included in the Workspace. When a superposition is performed, the coordinates of the structure change. In Step mode, the change is saved according to your synchronization preferences. In Continuous mode, the change is saved if you have the relevant option set in the ePlayer Options panel.

Running a Script At Each Step

You can run a script when each entry is included in the Workspace during ePlayer operation, by choosing a script in the ePlayer Options panel. In Step mode, any changes are saved according to your synchronization preferences. In Continuous mode, changes are saved if you have the relevant option set in the ePlayer Options panel.
