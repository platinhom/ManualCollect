Summary

In this tab you choose a solvent from a predetermined set or enter parameters for a user-defined solvent. The solvent is included in the calculation as a dielectric continuum with a cavity for the molecule.

For information about the controls at the top and bottom of the panel, see the Jaguar Panel topic.

Solvation Tab Features

  • Solvent model option menu
  • Solvent option menu
  • Dielectric constant text box
  • Molecular weight text box
  • Density text box
  • Probe radius
  • Gas-phase reference energy controls

Solvent Model Option Menu

Choose the solvent model from this option menu. The choices are

  • None—do not include solvation, run a gas phase calculation only.
  • PBF—Use the standard Poisson-Boltzmann continuum solvation model
  • SM6—Use the Minnesota solvent model SM6. This model can only be used with water as the solvent.
  • SM6—Use the Minnesota solvent model SM8. This model can be used with all available solvents.

Apart from the Solvent option menu, the remaining controls in the panel are unavailable for the SM6 and SM8 models.

Solvent Option Menu

Choose the solvent from this option menu. The Dielectric constant, Molecular weight, and Density text boxes are filled in, and the Probe radius value is displayed. Choosing an item other than None sets isolv=2 in the gen section. The solvent choice sets epsout and radprb in the gen section.

If you edit any of the text boxes and press ENTER, the solvent choice changes to Other, and the solvent is taken to be a user-defined solvent.

If you choose Other, you must enter values in the Dielectric constant, Molecular weight, and Density text boxes to define a solvent. The values in the latter two text boxes are used to calculate the probe radius.

For pKa calculations, the only solvent choices are Water, DMSO, and Other. Water is the default for all except CH and NH acids, whose default is DMSO.

Dielectric Constant Text Box

Displays the dielectric constant for the chosen solvent. You can edit this value to define a new solvent. When you edit this value solvent choice changes to Other. You must also provide a molecular weight and a density to define a solvent. Sets epsout in the gen section.

Molecular Weight Text Box

Displays the molecular weight for the chosen solvent. You can edit this value to define a new solvent. When you edit this value solvent choice changes to Other. You must also provide a dielectric constant and a density to define a solvent. Sets radprb in the gen section.

Density Text Box

Displays the density for the chosen solvent. You can edit this value to define a new solvent. When you edit this value solvent choice changes to Other. You must also provide a dielectric constant and a molecular weight to define a solvent. Sets radprb in the gen section.

Probe Radius

Calculated from the molecular weight and density. Non-editable.

Gas-Phase Reference Energy Controls

Select an option for the gas-phase reference energy. This energy is used to calculate the solvation energy.

Optimized gas-phase structure

Optimize the geometry in the gas phase first, and use this energy as the reference energy. Sets nogas=0 in the gen section. This is the default.

Input structure

Use the energy of the input structure for the reference energy. Sets nogas=2 in the gen section.

Value

Use the value in the text box as the reference energy. Sets nogas=2 and sets esolv0 in the gen section.
