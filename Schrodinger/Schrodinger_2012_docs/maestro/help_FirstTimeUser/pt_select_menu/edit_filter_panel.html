Summary

The Edit Filter panel allows you to edit the filters that you use to select entries in the Project Table.

Opening the Edit Filter Panel

To open the Edit Filter panel:

  • Choose Select → Filter → Edit in the Project Table panel.

Using the Edit Filter Panel

To edit a filter, select the filter in the filter list, then make your changes. You can edit the ESL expression for the filter directly, in the ESL text box, or you can edit the filter in the Entry Selection dialog box, which you open with the Select button. You can also create new filters, rename filters, and delete filters in this panel, and apply the selected filter to the Project Table.

Edit Filter Panel Features

  • Filter list
  • New
  • Delete
  • Apply
  • Name
  • ESL
  • Clear
  • All
  • Select

Filter list

This text box contains the list of named filters. You can select a single filter by clicking on it.

New

This button opens the New Filter dialog box, in which you can create and name a new filter. Once you have created it, you can then specify the content of the filter in the ESL section.

Delete

This button deletes the selected filter.

Apply

This button applies the filter to the Project Table to select a set of entries.

Name

This text box displays the name of the selected filter. You can change the name of the filter using this text box.

ESL

The ESL text box displays the ESL expression that defines the filter. You can edit this expression directly.

Clear

This button clears the ESL expression.

All

This button makes the ESL expression 'all', which means all entries.

Select

This button opens the Entry Selection dialog box, in which you can create a filter by choosing properties and conditions on the properties.
