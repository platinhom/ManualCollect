Summary

In this panel, you can set up and run FEP calculations for molecule annihilation, which provides a measure of the total free energy of solvation or complexation. The panel is designed to make setting up FEP jobs as easy as possible: most of the computational details, such as setting the force field parameters, solvation, relaxation of the system, and setting the FEP simulations, are taken care of automatically.

Opening the Total Free Energy by FEP Panel

To open the Total Free Energy by FEP panel:

  • Choose Applications → Desmond → Total Free Energy by FEP.

Using the Total Free Energy by FEP Panel

Calculating the absolute free energy for a molecule involves running an "annihiliation" FEP simulations, in which the selected molecule is removed from the system. The simulations are automatically prepared, performed, and analyzed to yield the absolute free energy for the selected molecule.

To run an annihilation job:

  1. Include in the Maestro Workspace the system that contains the ligand molecule to be annnihilated, the receptor protein (optional), and any other key molecules of the system. (This step can be performed prior to opening the panel.)

  2. Select Pick the molecule, then pick the ligand (or molecule) in the Workspace.

    When the ligand is picked, it is colored with green carbons.

  3. In the Plan calculation tab, choose the calculation types.

    The choices will depend on whether you want to calculate binding free energies (complex and solvent) or solvation free energies (solvent and vacuum).

  4. Choose the FEP protocol you want to use for each calculation type from the FEP protocol option menu.

  5. Click Start, set the job parameters in the Start dialog box, and click Start in the dialog box to run the job.

Define Perturbation tab

This tab displays a series of steps to follow. Each step has text describing the task for the step.

Step 1 involves displaying the ligand, the receptor, and any other important molecules in the Workspace.

Step 2 involves picking the molecule for annihilation. It has a single option, Pick the molecule. When this option is selected, you can pick an atom on the ligand in the Workspace. When the ligand is picked, it is colored with green carbons. Once focus is returned to the panel, this option is deselected.

Action buttons

At the bottom left of the panel are three buttons that perform various actions:

  • Start—Opens the Start dialog box, in which you can set the job parameters such as job name, host, number of CPUs, and so on, and then start the job.

  • Write—Writes the input files for the job but does not start the job. Opens a dialog box in which you can specify a job name, which is used to name the files. This capability is useful if you want to change the FEP protocol.

  • Reset—Resets the entire panel to its initial state. This includes clearing the Workspace. If you want to use the same structures, you will have to redisplay them befor proceeding.
