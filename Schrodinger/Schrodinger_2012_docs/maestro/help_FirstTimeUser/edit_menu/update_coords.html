Updating Coordinates

Global rotation and translation of structures in the Workspace does not change the coordinates in the Project table, it merely changes the "camera angle", or view of the structures. If you want to change the coordinates, choose Update Coordinates from the Edit menu. The coordinates are updated to reflect the axis system of the current view.
