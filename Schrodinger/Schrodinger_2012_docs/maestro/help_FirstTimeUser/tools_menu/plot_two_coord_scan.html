Summary

The Plot of Two-Coordinate Scan panel is used to display a contour plot of the data from a two-dimensional coordinate scan, such as a MacroModel dihedral drive involving two angles or a Jaguar coordinate scan.

Opening the Plot of Two-Coordinate Scan Panel

To open the Plot of Two-Coordinate Scan panel:

  • Choose Tools → Plot Coordinate Scan → Two Coordinates.

Using the Plot of Two-Coordinate Scan Panel

After a 2D scan calculation has been successfully completed, the resulting .grd file can be displayed graphically using this panel. Click Open in the top left of the panel to open a file selector in which you can choose a .grd file. Only one .grd file can be displayed at any given time. Reading in a second file clears the data from the first.

The appearance of a displayed plot can be changed using the controls on the left side of the panel. The range of the axes can be changed by entering values the Minimum Energy and Maximum Energy fields.

Each time a dataset is read in, the corresponding structure file is read. When you click in the plot area or trace the plot with the middle mouse button, the structure in the Workspace is updated to reflect the energy and coordinate values that correspond to the pointer position on the plot (these energy and coordinate values are displayed in the Energy:, Coordinate 1:, and Coordinate 2 text areas located in the lower left corner).

Toolbar

The toolbar provides tools for manipulation of the plot and for saving an image of the plot. This is a generic toolbar, and some of the actions may not be useful in the current context. The toolbar buttons are described below.

Reset
Reset the plot to the original pan and zoom settings.
Back
Display the previous view of the plot in the view history.
Forward
Display the next view of the plot in the view history.
Pan/zoom
Pan the plot with the left mouse button, zoom with the right button.
Zoom to rectangle
Drag out a rectangle on the plot to zoom in to that rectangle.
Configure subplots
Configure the margins and spacing of each plot in the panel.
Edit axis and curve parameters
Make settings for the title, range, labeling, and scale of the axes; the color, style, and width of lines; and the color, style, and size of markers.
Save image
Save an image of the plot to file. Opens a file selector in which you can browse to a location, select the image format, and name the image.
Minimum energy text box

The value entered in the Minimum energy text box sets the lower limit of the mapped energy range. There is no contour line at this energy value. Contour lines are drawn at equal energies between the minimum and maximum values. The energy values for the contour lines are determined by dividing the energy range (specified in the Minimum energy and Maximum energy text boxes) by n+1, where n is the specified number of contour lines.

Maximum energy text box

The value entered in the Minimum energy text box sets the upper limit of the mapped energy range. There is no contour line at this energy value. Contour lines are drawn at equal energies between the minimum and maximum values. The energy values for the contour lines are determined by dividing the energy range (specified in the Minimum energy and Maximum energy text boxes) by n+1, where n is the specified number of contour lines.

Full Scale button

To restore a plot whose range has been altered using the energy text boxes to its full scale, click Full Scale.

Number of contours option menu

This option menu allows you to select the number of contour lines you wish to be used in the current plot. The fewer the number of lines, the larger the range each contour line represents. The available options are 2 through 10.

Contour width option menu

This option menu allows you to specify the thickness of the contour lines displayed on the current plot. The options are 1 (thinnest line) through 5 (thickest).

Negative dashed option

In some instances, the clarity of a plot can be improved if the contours for negative values are drawn with dashed lines. To do this, select Negative dashed.

Fill contours option

If this option is selected, the space between the contours is filled with the color of the next highest contour. The legend shows the color ranges.

Show legend option

If this option is selected, a legend showing the contour colors and values is displayed to the right of the plot area.

Label contours option

If this option is selected, the contours are labeled with the contour value. The labels are not displayed if Fill contours is selected.

Color map option menu

This option menu allows you to choose the range of colors used to represent the contours from a set of predefined color ranges.

Energy units options

The two options available for energy units are kJ/mol and kcal/mol. Energies are displayed using the selected option.

Energy Scale

The energy mapped in a 2D plot can be displayed with either an absolute or a relative value. The Relative option is useful for estimating barrier heights.

Decimal places text boxes

These text boxes set the number of decimal places in the values that are displayed on the x axis, the y axis, and the contour labels.

Labels text box

In these text boxes, you can set the labels that are displayed on the x and y axes and the title of the plot.

Open button

Open a .grd file that contains 2D plot data, such as a coordinate scan over two coordinates. This button opens a file selector from which a .grd file can be opened.
