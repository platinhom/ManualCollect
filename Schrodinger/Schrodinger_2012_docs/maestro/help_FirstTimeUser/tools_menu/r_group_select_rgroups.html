Summary

In this panel you can select the R groups for each position.

Opening the Select R Groups Panel

To open the Select R Groups panel:

  • Choose Restrict Structures from the shortcut menu in the R-group position importance section of the Heat Map panel.

Select R Groups Panel Features

  • Position option menu
  • R-group table
  • Select All R-Groups, All Positions button
Position option menu

Choose the position for whicy you want to select R groups. The R groups for the chosen position are displayed in the table.

R-group table

Displays the R groups for the chosen position. You can select R groups by using click, shift-click and control-click in the table cells. The heat map is updated immediately when you make a selection.

Select All R-Groups, All Positions button

Select all R groups at all positions. This button undoes any selection that you made.
