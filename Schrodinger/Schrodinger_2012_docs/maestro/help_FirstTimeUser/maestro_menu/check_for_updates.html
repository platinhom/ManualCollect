Checking for Software Updates

Maestro provides a facility that checks the Schrödinger web site for software updates. It compares the software version in your installation with that on the web site, and notifies you in your browser of whether there are updates available. The page that is displayed is a page from the Support Center, so you can easily download any available updates or patches.

To check for updates, choose Maestro → Check for Updates.

When you start Maestro, the Schrodinger Updates panel is displayed to remind you to check for updates. You can choose how often you want to be reminded from the Remind me to check for updates option menu. The panel lists the currently installed packages, and has a Check Now button so you can perform the check immediately. You can also display this panel by running $SCHRODINGER/utilities/checkupdates. The settings you make and the last check date is stored in $HOME/.schrodinger/checkupdates_suiteyear.prefs
