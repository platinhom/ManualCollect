Duplicating Project Entries

To duplicate the selected entries, choose Entry → Duplicate in the Project Table panel.

There is no panel or dialog box associated with this command.

Each new entry is placed immediately below its source entry. If any of the selected entries is grouped, the duplicated entry is added to the same group. The new entries retain the same entry name as the originals. The title is preserved when the copy is made.

If a single entry is selected and is also included in the Workspace, the source entry is excluded and the duplicated entry is included in the Workspace. When the entry is duplicated, synchronization of the Workspace with the entry is performed according to the synchronization preference (see the Workspace - Changes preferences section of the Preferences Panel topic). Any changes that were made to the Workspace that were not synchronized are transferred to the duplicated entry.
