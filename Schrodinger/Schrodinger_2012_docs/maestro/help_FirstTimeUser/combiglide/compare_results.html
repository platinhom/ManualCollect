Summary

The Compare Results panel enables you to compare the lists of reagents with their GlideScores from different library selections.

Opening the Compare Results Panel

To open the Compare Results panel

  • Click Compare in the Analyze Library step of the Combinatorial Screening panel.

Using the Compare Results Panel

To compare sets of results, select them in the Select results to compare list, up to a maximum of 6. The reagents for the position selected from the Position option menu are displayed in the panel for each set of results. When you select another position from the Position option menu, the reagents for that position are displayed.

You can also delete results, by selecting them in the Select results to compare list and clicking Delete. The result sets are deleted without further confirmation, so check your selection carefully.

Compare Results Panel Features

  • Position option menu
  • Select results to compare list
  • Delete button
  • Results tables

Position option menu

Select the position for which results are to be displayed. The results tables are updated to show reagents for the selected position.

Select results to compare list

List of results that have been saved in the Analyze Library step. To select or deselect a set of results, click the appropriate list item.

Delete button

Delete the results selected in the Select results to compare list. This action removes the results permanently from the run, and there is no confirmation of the action, so you should check your selection carefully before clicking this button.

Results tables

The list of reagents for the selected position are displayed in a table for each results set. The table contents is the same as in the Analyze Library step, except for the Workspace inclusion column.
