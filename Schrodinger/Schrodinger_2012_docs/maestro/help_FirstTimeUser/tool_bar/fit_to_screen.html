Fit To Workspace

This button is located on the View toolbar in the main window:

When you click this button, the selected atoms (or all atoms, if there is no selection) and their associated graphical objects are resized so that they occupy the entire Workspace. Enough space is left so that rotation can be done without any part of the structure going outside the boundaries of the Workspace.

Technically, this is done by calcluating a bounding box for the relevant atoms and associated graphical objects. The bounding box is then used to visually scale these items so that they all appear in the Workspace. The bounding box is also used to position the clipping planes (shown in the Clipping Planes window).

The Preferences panel has a number of preferences that affect fitting to the Workspace, under Workspace - Fitting.
