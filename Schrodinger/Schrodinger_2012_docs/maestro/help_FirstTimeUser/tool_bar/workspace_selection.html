Workspace Selection

Maestro offers two ways of selecting and acting upon structures in the Workspace:

  • Selection First: Select atoms in the Workspace and then choose an action.
  • Action First: Choose an action and then select an item.

Selection First

You can select items in the Workspace by choosing a structural unit from the Select button menu on the Edit toolbar, then click on an atom to select all the atoms in that structural unit.

For instance, click and hold the Select button and choose Residues. Then click on an atom in the Workspace to select all the atoms in that residue. Once you finish selecting items, you can choose an action to perform, e.g. atom coloring.

Note: You can select atoms by either clicking them or dragging to select them. See Mouse Interaction and Settings for a full description of using the mouse to select items.

You can also store the Workspace selection in a memory location—see Storing Atom Selections.

Action First

You can choose an action first by opening a panel or using a button such as Label picked atoms. Choose the desired action, then select objects in the Workspace to which to apply that action.
