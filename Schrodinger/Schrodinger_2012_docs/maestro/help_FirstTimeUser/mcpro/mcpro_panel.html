Summary

The MCPRO+ panels provide an interface for setting up and running MCPRO+ jobs. The controls in the panels are configured differently for each kind of MCPRO+ task.

For information on the controls in a tab, see the help topic for the tab, which you can open by clicking Help when the tab is displayed.

Opening the MCPRO+ Panels

To open the MCPRO+ panel for a particular task, choose the task from the MCPRO+ submenu of the Applications menu.

Using the MCPRO+ Panels

To run an MCPRO+ job, you first need to set up a model system. This task involves selecting the receptor and the ligand structures, depending on the task, and preparing them for the MCPRO+ calculation. For a receptor structure, this usually means truncating the structure and capping it with NMA or ACE, and neutralizing the protein or the complex by adding or removing protons from ionizable groups remote from the ligand binding site. Constraints can also be specified for both receptor and ligand. Solvent water molecules are added in the final part of the setup.

The receptor and ligand structures that you select must be 3D all-atom structures with correct charges and bond orders. To prepare the receptor structure, you can use the Protein Preparation Wizard. To prepare the ligand structures you can use LigPrep.

Each MCPRO+ task panel has a tab in which you can create a model system, which is required for the calculation. The tab is wizard-like, and takes you through a series of steps. Once the model system is defined, the other tabs become available, and you can set the rest of the parameters for the calculation.

MCPRO+ Panel Features

The MCPRO+ panels contain a set of tabs and some action buttons. Each panel contains the Create Model Systems tab and the Solvation tab, and the third tab is task-dependent. The tabs are described in separate topics:

  • Create Model System Tab
  • Solvation Tab
  • Minimization Tab
  • Free Energy Difference by FEP Tab
  • Relative Binding Affinity by FEP Tab
  • Monte Carlo Tab
  • Linear Response Tab

The action buttons consist of a Start button, which opens a Start dialog box for setting up and submitting the job, a Write button for writing out the input file, and a Reset button for clearing the settings in the tab and setting them to the defaults.
