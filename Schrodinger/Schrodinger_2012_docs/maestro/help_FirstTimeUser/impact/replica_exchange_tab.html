Summary

This tab is used for setting up replica temperature exchange MD simulations.

Replica Exchange Tab Features

  • Replica exchange MD option
  • Number of replicas text box
  • Number of MD steps between exchanges text box
  • Temperature Settings button
  • Restart a previous job option
  • Restart input file text box and Browse button
Replica exchange MD option

Select this option to enable the use of replica exchange. When you do so, the remaining controls in the tab become available.

Number of replicas text box

Enter the desired number of replicas in this text box. The maximum allowed is 100.

Number of MD steps between exchanges text box

Enter the number of MD steps to take between exchanges of replicas.

Temperature Settings button

Opens the Temperature Settings dialog box, in which you can set the target temperature for each replica and select the temperature of interest.

Restart a previous job option

Select this option to restart a calculation using the reference structure output from a previous replica exchange job. You can change parameters except for the temperature settings and the number of replicas.

Restart input file text box and Browse button

Enter the name of the reference structure output file you want to use to restart the job, or click Browse and navigate to the output file. The file should normally end in -out.maegz.
