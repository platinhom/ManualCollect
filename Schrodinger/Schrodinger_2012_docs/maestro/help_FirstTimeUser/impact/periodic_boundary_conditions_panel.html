Summary

Use this panel to specify settings for Impact calculations using periodic boundary conditions. When explicit solvent is used, the periodic boundary conditions technique is usually applied in order to avoid nonphysical "edge effects". The system of interest is defined to be in a box of a given size, images of which are replicated throughout space to form an infinite 3D lattice.

The options in this panel allow you to set the size of the simulation box and to select and specify parameters for the Ewald summation method, which is used to efficiently sum long-distance electrostatic interactions in periodic systems. (This option cannot be used if you have selected Use fast multipole method.)

Opening the Periodic Boundary Conditions Panel

To open the Periodic Boundary Conditions panel, select the Use periodic boundary conditions option in the Potential tab of the Impact Energy Minimization, Impact Dynamics, or Impact Hybrid Monte Carlo panel, then click the adjacent Settings button.

Periodic Boundary Conditions Panel Features

  • Box [X,Y,Z] length text boxes
  • Use Ewald long-range correction option
  • Maximum length of K-space vectors text box
  • Alpha text box
Box [X,Y,Z] length

These three text boxes set the size of the simulation box. The minimum size for any dimension that Maestro will use is 18.62 Å.

Use Ewald long-range correction

Select this option to use the Ewald summation method for efficiently summing long-distance electrostatic interactions in periodic systems. Unlike the Fast Multipole Method, Ewald summation does not require the net charge of the system to be zero. Because of their incompatible requirements, this option is not used if Use fast multipole method is selected.

Maximum length of K-space vectors

If Use Ewald long-range correction is selected, you can use this text box to specify the number of component terms to retain in the reciprocal-space part of the summation. The default value is 5. Larger values will yield increased accuracy but result in slower execution.

Alpha

If Use Ewald long-range correction is selected, you can use this text box to set the value of the a parameter. A reasonable value is 5.5/L, where L is the length of the cubic simulation box. The default value is 0.25.
