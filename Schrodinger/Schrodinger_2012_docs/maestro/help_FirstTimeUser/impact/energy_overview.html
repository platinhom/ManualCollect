Summary

This topic describes common features of Impact panels. Some features described here appear in every Impact panel; others are common to most, but not all, panels.

Common Features

Unlike many other panels in Maestro, only one Impact panel can be active at any one time. If you choose a new Impact panel while another Impact panel is already open, the existing panel is dismissed before the new one is displayed.

Controls that are specific to a panel are usually contained in a series of tabs. Each tab contains settings relevant to the task that the panel performs.

  • Use structures from option menu
  • Start button
  • Write button
  • Reset button
  • Close button
  • Help button
Use structures from

This option menu appears in the upper part of each Basic Impact panel. The options are:

Workspace (included entry)

This is the default setting. The structure that is currently included in the Workspace is used as input to the job. This includes whatever atoms or molecules are part of the structure, even those atoms that have been undisplayed. Performing a simulation with atom constraints (Frozen or Constrained atoms) requires that you use the Workspace structure to which the atom constraints have been applied.

Project Table (selected entry)

Select this option to use the entry that is currently selected in the Project Table. This may be different from the structure displayed in the Workspace. Because atom constraints are applied to the Workspace structure, they are ignored if an entry from the Project Table is used instead.

Only one entry from the Maestro Project Table can be displayed and used for each Impact job. If you want to run an Impact job on a system consisting of two entries, use the Merge option in the Project Table to create a combined entry, and use that as the basis for your simulation.

Start button

In the lower left corner of each panel is a Start button, which opens the Start dialog box. The features of this dialog box depend on the type of job that is being launched. See the Start dialog box topic for information about each feature.

Write button

The Write button writes out all the files required for the job, but the job is not actually started. Once the run files (an input file, jobname.inp, and an input structure file, jobname.mae) are written by Maestro, the job can be run from the command line in a terminal window using the syntax:

$SCHRODINGER/impact -i jobname.inp

where jobname.inp is the input file for the job in question. The log output is written to jobname.log by default; a different filename can be specified with -o othername.log.

Type $SCHRODINGER/impact -h for a usage summary of the impact command, or see your product User Manual or the Impact Command Reference Manual for a discussion of running Impact from the command line.

Reset

Resets the options in all the tabs to the default values.

Close button

The Close button, which is located on all Maestro panels, dismisses the current panel without starting the job or writing any files.

Help button

The Help button, also on all panels, opens a browser with an appropriate help topic displayed.
