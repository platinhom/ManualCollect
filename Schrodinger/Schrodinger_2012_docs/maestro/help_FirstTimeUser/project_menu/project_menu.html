Project Menu

This menu provides commands that perform actions on projects. At the end of the menu is a list of recent projects that can be opened by choosing the menu item for the project. The other menu items are as follows:

  • Show Table—Open the Project Table panel.

  • 2D Viewer—Open the 2D Viewer panel, to display 2D structures in a rectangular grid.

  • Summary—View and add summary information for the project. Opens the Project Summary panel, in which you can see information on project entries, runs, and jobs, and view, add, or delete annotations.

  • New—Create a new project. The current project is closed.

  • Open—Open a project. Displays a project selector in which you can navigate to the desired project. The current project is closed.

  • Close—Close the current project. A scratch project is created to replace it.

  • Save As—Save the project under a new name, and continue working in the new project. The old project remains as it was when you performed the save.

  • Delete—Delete the current project. Any uncompleted jobs started from this project will finish, but cannot be incorporated.

  • Publish—Create a web page (HTML file) with a relative link to the project, which is stored as a zip file (.prjzip). You can provide a summary of the project, which is included in the web page. The page and project can then be placed on a web site.

  • Change Directory—Change Maestro's working directory, using a directory chooser. You can also change the directory in the Commands text box with the changedirectory Maestro command.

  • Save Workspace Changes—Save changes in the Workspace to the project. This item is only available if you have your project synchronization preference set to Only when specifically requested (in the Project - Entry changes tab of the Preferences panel) and you have made changes to the Workspace.

  • Import Structures—Import structures into the project. Opens the Import panel.

  • Import from Project—Import entries from another project. Opens a project selector, then a dialog box to choose the entries.

  • Export Structures—Export project entries to files. Opens the Export panel.

  • Export Project—Export part or all of the project to a new project, and continue working in the current project. The project selector that opens offers options for inclusion of user-created files, for saving as a compressed file (.prjzip, the default), for copying only the selected or included entries, for inclusion of workflow data from CombiGlide, Phase, or Prime, and for saving in Suite 2012 project format. Plots are only copied if you copy all entries to the new project.

  • Get PDB—Import a structure from the PDB, either from a local installation or from the web. Opens the Get PDB File dialog box.

  • Merge—Merge another project into the current project. Opens a project selector, in which you can navigate to and select the project to merge.

  • Back Up—Make a backup copy of the project in a zip file (.prjzip); the copy is stored in the directory project.backup, where project.prj is the name of the project.

  • Restore from Backup—Restore the state of a project from a backup copy. The project is overwritten by the backup.

  • Save State—Make a copy of the project in its current state; the copy is stored inside the project.

  • Revert to Saved State—Revert the state of the project to the last saved state, discarding changes made since it was last saved.

  • Revert to Opening State—Discard all changes made since the project was opened. This action requires that the project was saved in compressed form when it was closed (see the Project - Closing tab section of the Preferences panel for more information).
