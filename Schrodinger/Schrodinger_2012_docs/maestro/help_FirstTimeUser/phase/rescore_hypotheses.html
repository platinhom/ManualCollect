Phase — Rescore Hypotheses Dialog Box

Summary

In the Rescore Hypotheses dialog box, you can adjust the weights of the hypothesis survival scoring function and adjust the weight of the inactive match score, to generate post-hoc scores.

The terms in the scoring function are the same as in the Score Actives dialog box, and the adjusted survival score is the same as in the Score Inactives dialog box
