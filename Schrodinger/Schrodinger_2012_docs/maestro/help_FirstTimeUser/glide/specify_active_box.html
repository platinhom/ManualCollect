Summary

In this dialog box you can pick residues in the active site, either to define the active site of a receptor for grid generation, or for the writing of per-residue interaction scores.

Opening the Active Site Residues/Per-Residue Interaction Scores Dialog Box

To open the Active Site Residues dialog box:

  • Click Specify Residue in the Site tab of the Receptor Grid Generation panel, after selecting Centroid of selected residues.

To open the Per-Residue Interaction Scores dialog box:

  • Click Specify Residue in the Output tab of the Ligand Docking panel, after selecting Write per-residue interaction scores and Pick residues to include.

Using the Active Site Residues/Per-Residue Interaction Scores Dialog Box

Choosing the set of residues Glide will use to define the active site can be more complicated than picking a single ligand molecule. A broad range of picking controls and selection features are provided in this panel so that you can make large and small adjustments to the set, return to the last defined set, or start over. You can begin by picking residues (or larger objects), or you can deselect the Pick option and use Workspace selection (lassoing) followed by adding the selection, if it is satisfactory, to the list. All residues in which you have selected one or more atoms will be added in their entirety. You cannot add atoms or partial residues to the set. However, you can choose to pick larger objects such as molecules or chains.

If you are using this dialog box to pick residues for per-residue interactions, you do not need to be concerned about defining the active site, though it is likely that the most important interactions are with residues in the active site.

For more information about picking controls and atom selection, see the Picking Controls and Atom Selection Dialog Box topics.

Active Site Residues/Per-Residue Interaction Scores Dialog Box Features

  • Set of selected residues text area
  • Define active site residues picking tools
  • Delete button
  • Delete All button
Set of selected residues text area

The text area in the upper part of the panel displays as a list the set of residues that are currently defined.

Define active site residues/Define residues for per-residue interaction scores picking tools

Use these picking tools to pick the residues to include in the residue set. The choices for picking are limited to Residues, Chains, Molecules, and Entries. Individual atoms cannot be added to the list. Instead, the entire residue in which you have picked an atom is added to the list.

Delete button

Remove the selected residues from the list.

Delete All button

Remove all residues from the list.
