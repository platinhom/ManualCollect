New Features in Desmond 3.1

Maestro panels

  • A panel has been added that allows you to examine the exchanges of the replicas graphically (Applications → Desmond → Replica Exchange Review).

Model Systems

  • DMPC is available as a predefined membrane model.

  • You can regenerate a valid model system from a full system structure, with a new menu item at Applications → Desmond → Model System Regeneration.

Multisim

  • New builder stages have been added, build_geometry, for building a system from components or from a full-system CT without asssigning the force field, and assign_forcefield, for assigning the force field, including those available with viparr.

Trajectories

  • Trajectories can be superimposed on an external structure, that is, one that is not part of the trajectory. This can be done by including the structure in the Workspace beforehand. The Trajectory panel has new controls for this feature.

  • The clipping planes window is hidden by default when playing trajectories. This speeds up play by about 50%. There is an option in the Trajectory panel to control the display of the clipping planes window during play.

  • The display state of atoms is exported when you export structures from the trajectory.

Replica Exchange

  • A panel has been added that allows you to examine the exchanges of the replicas graphically (Applications → Desmond → Replica Exchange Review).

Metadynamics

  • A Metadynamics Analysis panel has been added for analysis of one or two collective variables. It is available from Applications → Desmond.

  • The analysis is run automatically as part of the job if the multisim input contains an analysis stage. This stage is added when the job is started from Maestro.

  • RMSD collective variables can be used by adding the information to the multisim stage in the .msj file.

Other

  • Desmond can be run locally under Mac OS X.
