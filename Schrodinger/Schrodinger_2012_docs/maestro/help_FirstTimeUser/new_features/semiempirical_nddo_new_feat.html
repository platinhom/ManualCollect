New Features in Semiempirical NDDO Module 2012

New features and improvements in the semiempirical NDDO module package in Schrödinger Suite 2012 are listed below.

  • Analysis of the ESP, ALIE, and ALEA surfaces as mapped to the isodensity surface are reported in the output and recorded as Maestro properties. The analysis includes maximum, minimum, mean, and variance, among other statistics.

  • Calculation of surfaces for larger systems no longer fails with insufficient memory.
