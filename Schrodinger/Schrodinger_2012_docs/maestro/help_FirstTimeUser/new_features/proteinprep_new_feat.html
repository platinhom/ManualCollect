New Features in Protein Preparation 2012

  • The pKa values of the protein residues can be calculated with PROPKA, for use in the optimization of the H-bond network.

  • You can remove waters on the basis of the number of hydrogen bonds they form with atoms that are not in other waters.

  • The sampling algorithm for H-bond network optimization has been significantly improved. Sampling is done at a much higher level than the extended sampling in the previous release, so the controls for sampling have been removed.
