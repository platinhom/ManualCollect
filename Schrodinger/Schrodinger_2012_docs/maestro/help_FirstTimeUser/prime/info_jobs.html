Summary

Before launching a long Build job, you may want to choose which host the job will run on and check or modify job settings. When the series of jobs in a run is complete, you can incorporate the results into the Maestro Project Table and save them as a project.

The Job Options Dialog Box

The Structure Prediction - Job Options panel allows you to specify which machine host each type of Prime job will run on. Open the panel by choosing Job Options from the File menu. Specify the host machine and the user name for each type of job. Make sure your schrodinger.hosts file contains a list of available machines to run on. See the Installation Guide for information on the schrodinger.hosts file.

Setting Up and Launching Jobs

While the machines on which jobs are to run are specified in the Job Options panel, Prime job settings are selected, and jobs are launched, from the Prime step panels. Sometimes this is as simple as clicking an action button, such as Search. In other steps, the type of job is selected from a Task menu, and then the job is launched by clicking the Run button. Settings for step actions or tasks can be modified using the Options button. The residues or secondary structure features on which a refinement task operate are selected from lists, tables, and Atom Selection dialog boxes in the Refine Structure step panel; until a selection has been made, the Run button remains unavailable.

For more information on setting up and launching jobs, see the Prime help topics for specific steps.

Monitoring Jobs

When a job has been launched and is running, the job status icon on the button at the upper right of the step panel turns green and spins. To monitor the job using the Monitor panel, click the button. For more information about monitoring jobs, see the Monitor Panel topic.
