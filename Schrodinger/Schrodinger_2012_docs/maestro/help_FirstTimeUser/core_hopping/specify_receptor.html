Summary

In this box you can specify a receptor, to which hydrogen bonds can be required and steric clashes forbidden.

Opening the Specify Receptor Dialog Box

To open the Specify Receptor dialog box:

  • Click Specify Receptor in the Find New Cores tab of the Ligand-Based Core Hopping panel.

Specify Receptor Dialog Box Features

  • Receptor file name text box and Browse button
  • Minimum H-bonds required box
  • Preserve specific H-bonds option and section
  • Require new H-bonds option and section
  • Prevent heavy-atom steric clashes with receptor option and section
Receptor file name text box and Browse button

Click Browse to navigate to and select a Maestro file that contains the receptor. The file name is displayed in the text box, which is noneditable.

Minimum H-bonds required box

Specify the minimum number of hydrogen bonds to the receptor that are required for a new core.

Preserve specific H-bonds option and section

Select this option to require specific H-bonds to the receptor from a new core that exist between the template and the receptor.

Begin H-Bond Selection button

Click this button to begin picking H-bonds in the Workspace. The first click marks the H-bond as an optional match (blue), the second marks it as a required match (red), and the third returns it to the unmarked state (yellow). The button is renamed Stop H-Bond Selection after clicking. When you have finished selecting H-bonds, click this button. The count of required and optional matches is updated.

Require all N critical (red) H-bond matches text box

This noneditable text box displays the number of "critical" matches.

Require M of N optional (blue) H-bond matches option and boxes

Require a minimum number of matches to any of the optional matching H-bonds. The total number of such H-bonds that you selected is displayed in the noneditable text box. You can set the minimum number in the other box.

Require new H-bonds option and section

Select this option to require specific H-bonds to the receptor from a new core that do not exist between the template and the receptor.

Begin H-Bond Selection button

Click this button to begin picking H-bonds in the Workspace. The first click marks the H-bond as an optional match (blue), the second marks it as a required match (red), and the third returns it to the unmarked state (yellow). The button is renamed Stop H-Bond Selection after clicking. When you have finished selecting H-bonds, click this button. The count of required and optional matches is updated.

Require all N critical (red) H-bond matches text box

This noneditable text box displays the number of "critical" matches.

Require M of N optional (blue) H-bond matches option and boxes

Require a minimum number of matches to any of the optional matching H-bonds. The total number of such H-bonds that you selected is displayed in the noneditable text box. You can set the minimum number in the other box.

Prevent heavy-atom steric clashes with receptor option and section

Select this option to filter out cores in which too many heavy atoms clash with receptor heavy atoms, and set criteria for the number of atoms and the proximity that defines a clash.

Maximum clashing ligand atoms box

Specify the maximum number of ligand atoms that are permitted to clash with the receptor in this box.

Clash criterion: less than N Å box

Specify the distance inside which atoms are considered to clash.
