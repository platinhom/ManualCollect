Summary

The Ring Closures panel allows ring-opening locations for MCMM and SPMC calculations to be specified.

Opening the Ring Closures Panel

To open the Ring Closures panel:

  • In the CSearch tab of the Conformational Search panel, choose Ring Closures from the Search variables option menu, then click Edit.

Using the Ring Closures Panel

During a Monte Carlo-based conformational search (MCMM or SPMC) it is necessary to break rings open before varying their torsions. Then after random torsion variations have been evaluated, the ring must be re-closed. The appropriate location for ring closures needs to be defined before a calculation can be started. "Ring Closures" consist of four atoms that define the bond to be broken (and then re-closed) during a MCMM or SPMC step.

The simplest way to create ring closures is to click the Perform Automatic Setup button in the Conformational Search panel. MacroModel makes reasonable decisions about where a ring should be opened.

Ring Closures Panel Features

  • List of ring closures
  • Minimum distance text box
  • Maximum distance text box
  • Define ring closures section
  • Markers option
  • Delete button
  • Delete All button
List of ring closures

This list shows the sets of four atoms that define the ring closures to use in the calculations. You can select a line in the list to display or delete.

If you clicked Perform Automatic Setup in the Conformational Search panel, this list contains the results of the setup when you open the panel.

Minimum distance text box

The minimum acceptable distance between the second and third ring closure atoms must be specified. If the distance between the ring closure atoms is less than this minimum distance, then the ring is reopened and a different set of random variations is performed. The default value for this is 0.5 Å, and this value should suffice for most searches.

Maximum distance text box

This text box allows a value to be specified for the maximum acceptable distance between the second and third atoms of a re-closed ring. If the distance between the ring closure atoms is greater than this maximum distance, the ring is reopened and a different set of random variations is performed. The default value for this is 2.5 Å, and this value should suffice for most searches.

Define ring closures section

To manually define the location of a ring opening, define the ring closure by picking four atoms with the standard picking controls in this section. The actual bond breakage occurs between the second and third picked atoms. The first and fourth atoms should also be in the ring. As the first three atoms are picked, a purple box is placed around them. Once the fourth atom is picked, a new entry is displayed in the list.

A ring closure can be redefined by picking new atoms while the closure is selected in the list.

Markers option

If this option is selected, the four selected atoms are marked with a light green line. The line is solid between the first and second atoms and the third and fourth atoms. To indicate where the ring closure will occur, Maestro places a broken line and a lightning bolt between the second and third atoms. To distinguish the currently selected closure, a solid line is placed on either side of the broken one.

Delete button

To delete a defined ring closure, select it by clicking on it in the list, or re-pick the atoms that define it, then click Delete.

Delete All button

To delete all the defined ring closures, click this button.
