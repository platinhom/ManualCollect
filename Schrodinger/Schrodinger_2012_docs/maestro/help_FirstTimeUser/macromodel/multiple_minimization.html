Summary

The Multiple Minimization panel is used to set up energy minimization jobs on multiple structures. The structures may be a set of conformers from a previous conformational search, for example, or a group of unrelated molecules. The Multiple Minimization panel can also be used for the estimation of partition coefficients between solvents.

Opening the Multiple Minimization Panel

To open the Multiple Minimization panel:

  • Choose Applications → MacroModel → Multiple Minimization

Using the Multiple Minimization Panel

MacroModel panels allow you to choose structures from either the Workspace or the Project Table (see Energy Overview for details). The Multiple Minimization panel contains five tabs. For descriptions of the controls in the first four tabs, see Potential, Constraints, Substructure, and Mini. The fifth tab, Mult, contains the controls that are used to set up multiple minimizations. These are described below. Once you have finished configuring the settings on the panel, you can click Start to start the job or click Write to write the files without starting the job (see Energy Overview for details).

Mult Tab Features

  • Input file text box
  • Minimization mode options

Input File Text Box

Enter a file name of a Maestro-format file that contains one or more valid structures, or click Open to select a file. The structures in this file are usually the results from a previous conformational search, which are to be reminimized. If no file is entered, then the structural input is taken from the source indicated by the Use structures from option at the top of the panel.

Minimization mode options

There are three modes that you can select for multiple minimization calculations:

  • Minimization of non-conformers
  • Minimization of conformers
  • LogP estimation
Minimization of non-conformers

Select this option if you want to minimize a set of structures that are not conformers.

Minimization of conformers

Select this option if you want to minimize a set of conformers, such as the results of a prior conformational search. This option has a number of associated options and settings that can be made.

Comparison Atoms button

Click to open the Comparison Atoms panel and define the atoms that are used to test each structure to ensure that it is unique relative to all the other structures that have been already been found.

Eliminate redundant conformers using options and Cutoff text boxes

Select the distance mesasure for determining whether structures should be considered equivalent:

  • Maximum atom deviation—consider structures to be different if the maximum atom deviation for any pair of corresponding atoms exceeds the threshold given in the Cutoff text box.

  • RMSD—consider structures to be different if the RMS deviation for all compared atoms exceeds the threshold given in the Cutoff text box.

The default cutoff is 0.5 Å for both options.

Distinguish mirror-image conformations option

Select this option if you want the mirror image of a structure to be treated as a separate conformer.

Energy window for saving structures text box

Specify a value for the energy window. If a structure is minimized and found to be above the global minimum by more than the energy window value, it is rejected and not included in the output file.

Maximum number of structures to save option and text box

Select this option if you want to limit the number of structures to save, and enter the desired number in the text box. Structures are ordered by energy, so it is the lowest-energy structures that are kept.

LogP estimation

Select this option to calculate the partition coefficients between two solvents for the set of input structures. Choose a second solvent from the Secondary solvent option menu. The first solvent is specified in the Potential tab.
