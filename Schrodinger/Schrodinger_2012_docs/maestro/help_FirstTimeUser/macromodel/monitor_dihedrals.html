Summary

This panel is used to set up torsions to be monitored during a dynamics simulation. Results are saved in the .mmo file.

Opening the Monitored Torsions Panel

To open the Monitored Torsions panel:

  • Click the Torsions button in the Monitor tab of the Dynamics panel.

Using the Monitored Torsions Panel

Torsions to be monitored can be specified by picking the atoms that define the torsions, using the standard picking controls in the Define monitored torsions section, and clicking on four atoms. When the first three atoms are picked, Maestro places purple boxes around them. After the fourth atom is picked, the atoms appear in the list at the top of the panel. Maestro traces the torsion with a solid red line, draws a dotted red line through the torsion, and places a red "eye" icon beside the dotted line. To distinguish the currently selected torsion, the lines are drawn thicker.

To delete a torsion from the list, select it in the list and click Delete. To delete all monitored torsions, click Delete All.

Monitored Torsions Panel Features

  • Monitored torsions list
  • Define monitored torsions section
  • Delete button
  • Delete All button
Monitored torsions list

Lists the torsions to be monitored. After the fourth atom is picked, a new entry appears in the list.

Define monitored torsions section

The Define monitored torsions section contains the standard picking controls, which are used to select atoms that define a torsion. It also contains a Markers option, which you can use to show markers for the monitored torsions in the Workspace.

Delete button

To delete a monitored torsion, select it in the Monitored Torsions List, then click Delete.

Delete All button

Click this button to delete all monitored torsions.
