Summary

In this panel you can set up and run a job to calculate R-factors for the structure in the Workspace. The R values are returned as Maestro properties.

Opening the R-Factor Calculation Panel

To open the R-Factor Calculation panel, do one of the following:

  • Choose Applications → PrimeX → Task → R-Factor Calculation

  • Choose Tasks → Protein X-Ray Refinement → Task → R-Factor Calculation

  • Choose R-Factor Calculation from the Task button menu on the PrimeX toolbar.

R-Factor Calculation Panel Features

  • Calculation Settings button
  • Action buttons
Calculation Settings button

Open the Calculation Settings dialog box, so you can make any settings that affect the R-factor calculation.

Action buttons

The Start button opens the Start dialog box, in which you can make job settings and start the job. The Write button writes out the input file. You can then start the job from the command line.
