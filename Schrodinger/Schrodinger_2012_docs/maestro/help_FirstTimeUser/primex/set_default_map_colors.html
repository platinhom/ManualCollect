PrimeX — Set Default Map Colors Dialog Box

This dialog box allows you to set the default colors for the various types of maps. Each map has an option menu that offers a choice of colors. For the Fo-Fc map, there is an option menu for negative values of the difference density as well as for the positive values.

The color settings that you make here affect maps generated after making these settings. They do not affect existing maps. You can change the color for individual maps in the Surface Display Options dialog box of the Manage Surfaces panel or by using the shortcut menu in the Manage Surfaces panel.

You can also set the default map colors in the Preferences panel, under Surfaces - Appearance.

To open the Set Default Map Colors dialog box, click Set Default Map Colors in the Calculation Settings dialog box.
