Main Menu Bar

The menus on the main menu bar provide access to panels, allow you to enter commands, and control the appearance and contents of the Workspace. The menu bar has the following menus by default:

  • Maestro—change directories, check for software updates, set preferences, customize menus, set up Maestro command scripts and aliases, and quit Maestro.
    [Display topic list]

  • Project—show project entries and information, perform tasks on projects such as opening and closing, import and export structures, make backups.
    [Display topic list]

  • Edit—undo actions, copy and paste structures or images, build and modify structures, and find atoms in the Workspace.
    [Display topic list]

  • View—change the view of the Workspace (camera position and angle), save, restore and manage named Workspace views, set up 3D and stereo effects.
    [Display topic list]

  • Display—control the display of the contents of the Workspace, save Workspace images.
    [Display topic list]

  • Tools —group atoms; measure, align, minimize, and superimpose structures; assign structural features; and view and visualize data.
    [Display topic list]

  • Applications—set up, submit, and monitor jobs for Schrödinger's computational programs.
    [Display topic list]

  • Workflows—set up, run, and analyze jobs for workflows (also known as "solutions").
    [Display topic list]

  • Window—Open, close, and raise windows, arrange windows in a cascade, save and restore window layout.

  • Help —View various kinds of documentation, including online help, manuals, FAQs, information about Maestro and other Schrödinger products.
    [Display topic list]
