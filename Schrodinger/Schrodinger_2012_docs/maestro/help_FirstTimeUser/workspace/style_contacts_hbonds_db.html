Summary

In this dialog box you can set options for displaying hydrogen bonds and contacts with the Workspace style. For more information on hydrogen bonds and contacts, see the H-Bonds and Contacts Panel topic.

Opening the Contacts and H-Bonds Dialog Box

To open the Contacts and H-Bonds dialog box:

  • Click the Contacts button on the Style toolbar in the main window.

Contacts and H-Bonds Dialog Box Features

  • Display receptor-ligand hydrogen bonds option
  • Display intra-ligand contacts option
  • Display receptor-ligand contacts option
  • Mark contacts options
Display receptor-ligand hydrogen bonds option

Select this option to display hydrogen bonds between the receptor and the ligand.

Display intra-ligand contacts option

Select this option to display hydrogen bonds that are internal to the ligand.

Display receptor-ligand contacts option

Select this option to display contacts between the receptor and the ligand. The types of contacts displayed are selected under Mark contacts

Mark contacts options

Select one of these options for the types of contacts to display in the Workspace. The choices are Bad and Ugly, Bad, Ugly, and Good.
