Summary

In this dialog box you can define how the cells in property columns are colored using the numerical values of the property.

Opening the Color By Property Dialog Box

To open the Color By Property dialog box, choose Color By Property from the Entry menu in the Project Table panel.

Using the Color By Property Dialog Box

To apply a color scheme to a property column in the Project Table:

  1. Select the property from the Available properties list.

  2. Click Add.

    The property is transferred to the Colored properties list, selected, and default options and colors are set for this property.

  3. Select the desired options for coloring the property, and set the value thresholds and choose the colors.

  4. Click Apply to apply the scheme and continue on to color another property, or click OK to apply the scheme and close the dialog box.

To remove a color scheme:

  1. Select the property from the Colored properties list.

  2. Click Remove.

    The property is transferred to the Available properties list. The coloring is turned off, but the scheme is still available if you decide to reapply coloring.

Color By Property Dialog Box Features

  • Available properties property selector
  • Colored properties list
  • Add and Remove buttons
  • Property text box
  • Range display
  • Fill with Range Values button
  • Color below option and controls
  • Color above option and controls
  • Color between values option and controls
  • Color exact match option and controls
  • Save as Preferences button
Available properties property selector

Lists the properties that are available for coloring, with tools for restricting the range (Subset and Filter) and sorting (see the Property Selection topic). The list includes only properties that have real or integer values. You can select one property at a time to add to the Colored properties list.

Colored properties list

Lists the properties that have been selected for coloring. When a property is added to this list (with the Add button), the setting and colors are initialized for this property. If the property has previously had color settings, these settings are restored. Otherwise, standard default values are used.

Add and Remove buttons

The Add button transfers the property that is selected in the Available properties list to the Colored properties list. When it is transferred, it is selected in the new list and the color options are initialized.

The Remove button removes the property that is selected in the Colored properties list, and returns it to the Available properties list.

Property text box

Displays the name of the property that is selected in the Colored properties list. Noneditable.

Range display

Displays the range of values in the project for the property that is selected in the Colored properties list.

Set Limits to Range button

Set the values in the Color below and Color above text boxes to the minimum and maximum values of property that is selected in the Colored properties list, as shown in the Range display.

Color below option and controls

Select this option to color property values that are below the specified threshold. The threshold is a strict inequality (value < threshold) You can enter the threshold in the text box. To change the color, click the color button, and select a color in the color selector that is displayed.

Color above option and controls

Select this option to color property values that are above the specified threshold. The threshold is a strict inequality (value > threshold) You can enter the threshold in the text box. To change the color, click the color button, and select a color in the color selector that is displayed.

Color between values option and controls

Select this option to color property values that are between the Color below and Color above thresholds. There are two options:

Solid

Use a single color between the two thresholds. To change the color, click the color button, and select a color in the color selector that is displayed.

Interpolate colors in N steps

Interpolate the colors betweeen the Color below and Color above colors in the given number of steps. The range of values between the two thresholds is partitioned into steps of the same size.

Color exact match option and controls

Select this option to color property values that are equal to the specified value (value = target) You can enter the target value in the text box. To change the color, click the color button, and select a color in the color selector that is displayed. The color is shown on the color button.

Save as Preferences button

Save the property color schemes in the user preferences directory ($HOME/.schrodinger/maestrovv). These preferences are read and applied to properties in any project, not just the current project, and are used as the default colors. The property color schemes are stored in the project by default, and are used whenever the project is opened.
