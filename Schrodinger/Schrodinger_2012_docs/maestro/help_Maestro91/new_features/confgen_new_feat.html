New Features in ConfGen 2.3

Note: Confgen is a separately-licensed product, distributed along with MacroModel.

The following features have been added for the Suite 2012 release of ConfGen 2.3:

  • Two options have been added that reproduce the conformations generated by Phase. The Phase fast search without minimization option runs the default conformational search used in database generation. This option is supported from the command line by the -PHASE_FAST option to the confgen command. The Phase high quality search with minimization reproduces the default conformational search run in the Develop Common Pharmacophore Hypotheses workflow. This option is supported from the command line by the -PHASE_QUALITY option to the confgen command.

  • Command options for the Chen and Foloppe model (-FAST_CF) and the enhanced planarity parameters (-FORCE_PLANAR) have been added to the confgen command.

The following features were added for the Suite 2011 release of ConfGen 2.3:

  • Considerable improvements have been made in ring templating:

    • Custom location for additional templates
    • More templates, including macrocycles
    • Amide bonds can be mapped to double-bonded templates

  • The Enhance planarity of conjugated pi groups option has been added in both the ConfGen Standard and the ConfGen Advanced panels. This option promotes planar geometry for bonded sp2 atoms by increasing the torsional potential around the bond. It is supported from the command line by the -FORCE_PLANAR option to the confgen command.

  • The default conformation for esters has been set to trans, to avoid generating cis conformations when the trans conformation has bad contacts in the initial sampling.

  • The OPLS_2005 force field is used by default, which allows a greater range of elements an parameters (e.g. including B and Si).

  • An option to reward hydrogen bonds that complete 5- or 6-membered pseudo-rings has been added.

  • More efficient distributed processing is available with the confgen command.

  • A fast mode has been added that is based on the work described in I-J. Chen and N. Foloppe, J. Chem. Inf. Model. 2010, 50, 822. This mode is included in the Confgen Advanced panel as Fast (CF), and is supported by the -FAST_CF option to the confgen command.
