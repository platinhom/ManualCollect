New Features in Maestro 9.3


    Workspace and Toolbars

  • The Save Image panel is now a file selector with options for the image included in the panel.

  • Ribbon quality can be reduced when moving the ribbon, by setting a preference.

  • The Atom and Bond Coloring panel has been redesigned to simplify it. The Color Scheme tab is now a separate panel, and the controls to apply a color scheme are now in the Atom Color tab. The color selection tabs have been removed. Atom colors are set by opening a standard color selector; bond colors are still set with the palette, which you open by clicking the color button in the Bond Color tab.

  • The Atom Labels panel has been redesigned.

  • The fields in atom labels can be reordered, by dragging them in the Fields list. (Update 2)

  • The Display/Undisplay Atoms panel has been redesigned. (Update 2)

  • The Molecular Representation panel has been redesigned. All of the settings are now in the Preferences panel, and the panel itself is only for applying a representation to atoms or bonds. Ribbons can now be applied from a new Ribbons panel, and the Ribbon Options dialog box has been removed, as all the settings are now in the Preferences panel. (Update 2)

  • Measurement labels and atom labels can be dragged in the Workspace to reposition them. For atom labels, you can restore the default positions of all labels.

  • When selecting captions or highlights, you can add to the selection with shift-click.

  • Several new keyboard shortcuts have been introduced, for translation and rotation. These are implemented along with menu items on the View menu.

  • Toolbar buttons have been added for increasing and decreasing Workspace fonts.

  • A preference has been added to enhance fog when fitting to the Workspace.

  • The Fog subtab in the 3D tab of the Appearance panel has been moved up to become a tab in the panel itself.

  • Clipping planes can be adjusted to fit the selection, the ligand, or all atoms without doing zoom, from the View → Move Clipping Planes submenu.

  • Stereo images can be swapped in the 3D tab of the Appearance panel. This is useful for resolving synchronization issues with hardware stereo.

  • Perspective is enabled by default. (Update 2)

    • Menu Bar

  • Some of the Maestro menu items have been moved to other menus.

  • The View menu now has Rotate and Translate submenus, which offer rotation and translation by small increments, as well as the 90 degree rotations available previously. In addition, there is a Transform Mode submenu, which allows you to select a mode in which the "normal" mouse action (left mouse button) is used for rotation, translation, or zoom, and picking is suspended.

    • Projects and Project Table

  • The capability for automatic backups of projects has been added. Settings for backups can be made in a new Backups section in the Preferences panel. (Update 2)

  • The main toolbar has been split into two toolbars, the Project toolbar and the Property toolbar, with a number of new buttons.

  • Properties for each product (or "author") have been classified into "primary" and "secondary" properties. These classifications can be used to restrict what is shown in the Project Table and in property selectors.

  • The Sort Project Table panel has been redesigned, to support group sorting, grouping of entries before sorting, and selection of entries after sorting, as well as unlimited sort keys. The Sort Groups panel has been removed.

  • The colors and shading of the rows in the Project Table, and columns in the fixed area, can be set as preferences.

  • Numbers are now right-aligned, and text is left-aligned, subject to a preference.

  • A preference for showing the Stars column in new projects was added.

  • The number of chiral centers in a ligand-sized molecule can be calculated in the Calculate Properties panel, and chirality properties added to the chiral atoms.

  • More control over the recording of a movie has been added. You can specify the frame duration or the length of the movie in the Movie - Options dialog box, and you can choose to interpolate between frames to create a smoother movie.

    • Atom Selection

  • The atom selection tools that are available in panels where picking is done have been redesigned, to provide more options and to simplify the tools. See the Picking Controls topic for a description. (Update 2)

  • In the Sets panel, you can apply a chosen set to select atoms in the Workspace. (Update 2)

    • Building

  • In the 2D Sketcher, you can add or replace functional groups by typing in a standard name or a SMARTS pattern.

    • Import and Export

  • A Common file type has been added in the Import panel that shows Maestro, PDB, and SD files.

  • Compressed SMILES files, with the extension .smigz or .smi.gz, can be imported.

  • Compressed PDB files (.pdb.gz) can be imported and exported. (Update 2)

  • When writing PDB files, you can select an option to reorder the atoms by increasing residue number, so that atoms for each residue are contiguous. (Update 2)

    • Jobs and Monitoring

  • An option has been added to the Create Postmortem dialog box to alter path names to make them unrecognizable.

  • An Incorporate All button has been added to the Monitor panel so that you can incorporate all completed jobs in the current project in one action. (Update 2)

    • Surfaces

  • Electrostatic potential surfaces can be created, using the partial charges of the entry, in the Electrostatic Potential Surfaces panel.

  • Darkening colors by cavity depth can be applied to individual surfaces via an option in the Surface Display Options dialog box, as well as being set as a global preference.

  • An outline ("contour") can be added to the edges of surfaces when the edge is more than a given distance in front of the rest of the surface, to make the closer part of the surface stand out.

  • Angle-dependent transparency can be applied to a surface, in which the transparency depends on the surface normal: maximum when the normal is perpendicular to the screen, minimum when it is parallel to the screen.

  • Surfaces can be colored by the color of the nearest alpha carbon atom. (Update 2)

  • To display or undisplay a particular surface, you can use middle-click in the Surface table.

  • A preference has been added to set the default surface style.

  • A preference has been added to open the Manage Surfaces panel when surfaces are created. (Update 2)

  • A Preferences button has been added to the Manage Surfaces panel, which opens the Preferences panel at the preferences for surfaces.

    • Preferences and Customization

  • A Search text box has been added, which you can use to search for text in the panel (including tooltips).

  • The alignment of numbers and text (left, center, right) in the Project Table is controlled by a preference.

  • A preference for showing the Stars column in new projects has been added.

  • The background of measurement markers can be set with a preference.

  • The colors and shading of the rows in the Project Table, and columns in the fixed area, can be set as preferences.

  • Padding can be added to Project Table columns when fitting to data, by setting a preference.

  • The settings from the Molecular Representation panel have been moved to the Preferences panel. (Update 2)

  • Ribbon options are now in a new section in the Preferences panel.

  • Ribbon quality can be reduced when moving the ribbon, by setting a preference.

  • Reapplication of the Workspace Style when the Workspace changes can be turned off when a project is closed, by setting a preference.

  • A preference has been added to enhance fog when fitting to the Workspace.

  • Thick lines can be used for tubes instead of wire frame when using a simplified representation during rotation and translation, by setting a preference

  • A preference has been added to set the default surface style.

  • A preference has been added to open the Manage Surfaces panel when surfaces are created. (Update 2)

  • Dialog boxes can be centered on the current pointer position when they are opened, by setting a preference.

  • Job drivers can be run remotely by default, by setting a preference.

  • A PyMOL mode has been added to the standard list of mouse customizations in the Customize Mouse Actions panel.

  • A preference has been added for showing the index of the object being picked in the Workspace when the picking cursor is over an atom. The index is shown at the beginning of the status bar. This allows you to see the molecule number or entry number when picking.

    • Tools

  • The Protein Structure Alignment panel has been updated. It now provides a choice of the reference residues and the residues in the proteins that are to be aligned to the reference. It runs the structalign utility directly, which uses the optimized alignment. The option for this feature and the Options button have therefore been removed. You can also force an alignment for low similarity with a new option.

    • The background of measurement markers can be set with a preference.

  • The Measurements panel has been redesigned. Picking to delete measurements is no longer available, as you can do this in the table. The colors of the markers are now set in the Measurements section of the Preferences panel, and a Preferences button has been added to open the Preferences panel at this section. (Update 2)

  • For distance measurements, an Atoms & Surfaces item has been added to the Pick menu in the Distances tab of theMeasurements panel, which allows you to pick atoms or surfaces. The Atoms item is now strictly for picking atoms.

  • The H-Bonds and Contacts tabs in the Measurements panel have been moved into a separate H-Bonds and Contacts panel. The colors and cutoffs can be set in a new H-bonds and contacts section of the Preferences panel, and a Preferences button has been added to open the Preferences panel at this section. (Update 2)

  • 1-4 interactions can be excluded from consideration as contacts by setting an option in the Contacts tab of the H-Bonds and Contacts panel. (Update 2)

  • A Workspace selection section has been added to the Sets panel. This section contains three buttons, Replace, Add, and Subtract, for replacing the Workspace selection with the atoms that match the set, adding these atoms to the Workspace selection, or removing these atoms from the Workspace selection. (Update 2)

    • Help Facility

  • Tutorials can be set up in the Tutorials panel, which you open from the Help menu.

  • Due to updates in the generation of the PDF documentation, the default PDF reader has been changed on Linux to kpdf. This is because Adobe Reader on Linux does not handle links to other PDF files consistently with the newer PDF files.

    • Other New Features and Changes

  • Maestro preference sets can be saved in named profiles. Six profiles are provided by default; the three main profiles are for standard Maestro, BioLuminate, and Maestro Elements. Three other profiles customize Maestro as it appeared in Maestro 9.1, provide a somewhat easier interface for first-time users, and provide an interface that focuses on small molecule tools, particularly QM tools.

  • The Hetatom tab has been removed from the Atom Labels panel.

  • A utility has been added for validating Maestro files, named maevalidate.
