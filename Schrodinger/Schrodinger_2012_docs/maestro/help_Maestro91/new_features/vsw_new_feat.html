New Features in Virtual Screening Workflow 2012

  • Positional and NOE constraints can be added for a receptor when setting up a grid, and the ligand features for matching these constraints can be defined, in the Receptor Constraints dialog box.

  • The new Phase database format is now used for ligand input.

  • Input files can be in compressed or uncompressed format.

  • You can specify the hosts to use for ligand preparation (LigPrep) and for grid generation and docking (Glide). For each, you can specify the number of CPUs to use.

  • The selection of the number of subjobs for each docking stage can be done automatically, to ensure a run time between 1 and 10 hours for each subjob, by selecting the Adjust option in the Start dialog box.

  • The results of a docking job can be incorporated (appended) into the Maestro project by selecting an incorporation option in the Start dialog box.

  • You can use the enhanced planarity force field parameters by setting the FORCEPLANAR keyword to true in the input file.
