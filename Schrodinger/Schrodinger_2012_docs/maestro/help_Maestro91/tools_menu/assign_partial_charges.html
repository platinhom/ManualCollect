Assign Partial Charges

The Assign Partial Charges command on the Tools menu assigns partial charges to the structures in the Workspace, based on the OPLS_2005 force field. If the structure does not have explicit hydrogens, the partial charges for the implicit hydrogens are added to the partial charge for the heavy atom to which they are implicitly attached.

These partial charges can be used for many purposes, including displaying the dipole moment of the molecule (see the Workspace Menu topic for details).
