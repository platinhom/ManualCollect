Summary:

The Write Measurements panel allows you to write (save) measurements of distances, angles, dihedrals, H-Bonds or Contacts to a text (CSV) file.

Opening the Write Measurements Panel

The Write Measurements panel is a set of five similar panels: Write Distances, Write Angles, Write Dihedrals, Write H-Bonds, and Write Contacts. It can be opened by clicking the Write buttons in the Measurements panel or the H-Bonds and Contacts panel. The Write button in these panels is available only when there are some existing measurements.

Write Measurements Panel Features:

  • Delimiter options
  • File text box
  • Write button
  • Write To button

Delimiter options

These options set the column separator in the output file. Commonly used styles are "Comma" for a comma-separated value file or "Tab" for a tab-delimited file. User defined delimiter can be specified in the text edit box only when the User Defined option is selected.

File text box

The File text box sets the output file name.

Write Button

Write the measurements to the file in the File text box. The file format is one measurement per line where each line is formed by atom numbers, measurement values, and related information which are separated by comma, tab, or user defined delimiter. The Write button is available only when there is a pathname in the File text box.

Write To button

Open a file selection dialog box so you can navigate to a location and name the file.
