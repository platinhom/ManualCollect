Summary

In this dialog box you can add a membrane to the Desmond model system and position the membrane with respect to the solutes.

Opening the Set Up Membrane Dialog Box

To open the Set Up Membrane dialog box:

  • Click Add Membrane in the System Builder panel.

Set Up Membrane Dialog Box Features

  • Membrane model options
  • Transmembrane atoms ASL text box and buttons
  • Place Automatically button
  • Place on Prealigned Structure button
  • Adjust membrane position option
  • Save Membrane Position button
  • Load Membrane Position button
Membrane model options

There are two options:

  • Predefined—Select a predefined membrane. The available membranes are DPPC, POPC, POPE, and DMPC. The temperature at which the membrane patch was preequilibrated is given in parentheses after the membrane name. Because DPPC has a gel transition temperature around 313  K, the recommended minimum is also 325 K.
  • Custom—Read in a custom membrane from file. Click Browse to locate the file, or enter the file name in the text box.
Transmembrane atoms ASL text box and buttons

Select the atoms that are inside the membrane, by one of the following methods:

  • Click Select to make the atom selection in the Atom Selection dialog box. The ASL expression for these atoms is displayed in the text box when you click OK in this dialog box.
  • Click Set to Helices to select residues whose secondary structure assignment is a helix. This button enters the ASL expression res.sec helix in the ASL text box. If your protein does not have a secondary structure assignment, you can assign it with Tools → Assign Secondary Structure.
  • Enter an ASL expression directly into the text box.

Place Automatically button

Place the membrane in the default position and orientation. If you select transmembrane atoms, the membrane is placed so that these atoms lie inside the membrane. If you have a protein from the OPM database (http://opm.phar.umich.edu), the membrane is placed using the information provided with the protein.

If you click this button after adjusting the membrane, the membrane is returned to its default position and orientation. The membrane is represented as two red slabs, with a yellow line perpendicular to the slab planes. The actual membrane molecules are placed when the system builder job is run.

Place on Prealigned Structure button

Place the membrane according to the current coordinates of the structure in the Workspace. The membrane is placed symmetrically about and parallel to the xy plane, so that the surfaces are perpendicular to the z axis. This allows you to place a membrane on a prealigned protein.

Adjust membrane position option

Select this option to adjust the position of the membrane. You can change the orientation of the membrane using the middle mouse button (Workspace rotation).

Save Membrane Position button

Save the membrane position to the entries that are selected in the Project Table.

Load Membrane Position button

Load the membrane position from the entry that is selected in the Project Table.
