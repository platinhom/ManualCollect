Summary

In this panel, you can set up and run FEP calculations for the mutation of a ligand functional group. The panel is designed to make setting up FEP jobs as easy as possible: most of the computational details, such as setting the force field parameters, solvation, relaxation of the system, and setting the FEP simulations, are taken care of automatically.

Opening the Ligand Functional Group Mutation by FEP Panel

To open the Ligand Functional Group Mutation by FEP panel:

  • Choose Applications → Desmond → Ligand Functional Group Mutation by FEP.

Using the Ligand Functional Group Mutation by FEP Panel

In ligand mutation, a group on the ligand molecule, called the substitution group, is replaced with selected fragments. FEP simulations are automatically prepared, performed, and analyzed to yield the free energy difference between the mutant and the original ligand molecule for each fragment.

To run a ligand mutation job:

  1. Include in the Maestro Workspace the system that contains the ligand to be mutated, the receptor protein (optional), and any other key molecules of the system.

    You can use an existing model system that may, for example, include a membrane. If you do, note that the SPC water model is used in the FEP simulations, regardless of the choice made for your model system. This step can be performed prior to opening the panel.

  2. Select Pick the attachment bond, then pick the attachment bond in the Workspace, at the end toward the substitution group.

    This bond is the bond at which the substitution will take place. When the bond is picked, a green arrow is drawn over the bond, pointing to the substitution group, and the ligand core is colored with green carbons. If you clicked on the wrong end of the bond, click again.

  3. In the Fragment library table, select the fragments that you want to use to replace the original substitution group by selecting their table rows. Use shift-click and control-click to select multiple table rows.

    You can view the mutated ligand by clicking in the View column.

  4. (Optional) Display the mutated ligand and adjust the geometry of the fragment.

    It is possible that the fragment may have clashes with the protein in its default orientation. Adjusting the fragment will reduce the amount of minimization required prior to the FEP calculation, because the adjusted geometry is used as the starting point in the FEP simulations.

    Once the fragment is displayed, you can use the adjustment tools or the local transformation tools to adjust the fragment geometry. To reset to the original geometry, click the button in the Reset column.

  5. In the Plan Calculation tab, select the calculation types.

    The choices will depend on whether you want to calculate relative binding free energies (complex and solvent) or relative solvation free energies (solvent and vacuum).

  6. Choose the FEP protocol you want to use for each calculation type from the FEP protocol option menu.

  7. Set the buffer size for the periodic box. The buffer size is ignored if you use a prepared model system as input.

  8. Set the time used for the actual simulation (following any initial simulation) at each lambda point.

  9. Set the temperature for the simulation.

  10. Click Start, set the job parameters in the Start dialog box, and click Start in the dialog box to run the job.

Define Perturbation tab

This tab displays a series of steps to follow. Each step has text describing the task for the step.

Step 1 involves displaying the ligand, the receptor, and any other important molecules in the Workspace.

Step 2 involves picking the attachment bond for the functional group substitution. It has a single option, Pick the attachment bond. When this option is selected, you can pick a bond on the ligand in the Workspace. When the bond is picked, a green arrow is drawn over the bond, pointing to the substitution group, and the ligand is colored with green carbons. You therefore should click on the bond nearest to the substitution group. If you click on the wrong end, you can click again to rectify the error. Once focus is returned to the panel, this option is deselected.

Step 3 involves selecting the fragments that will replace the substitution group. This step is performed using the Fragment library table. The fragments that will be used are the fragments that you select in the table. To view the ligand with a particular fragment replacing the substitution group, click in the View column.

If you want to reorient the fragment, you can do so with the adjustment tools, which are available from the Adjust button on the Edit toolbar:

or with the local transformation tools, which are available from the Transform button on the Edit toolbar:

The reoriented fragment will be used in the FEP. You can revert to the original fragment by clicking the button in the Reset column. You should not make modifications to the ligand core or other molecules in the system.

Action buttons

At the bottom left of the panel are three buttons that perform various actions:

  • Start—Opens the Start dialog box, in which you can set the job parameters such as job name, host, number of CPUs, and so on, and then start the job.

  • Write—Writes the input files for the job but does not start the job. Opens a dialog box in which you can specify a job name, which is used to name the files. This capability is useful if you want to change the FEP protocol.

  • Reset—Resets the entire panel to its initial state. This includes clearing the Workspace. If you want to use the same structures, you will have to redisplay them befor proceeding.
