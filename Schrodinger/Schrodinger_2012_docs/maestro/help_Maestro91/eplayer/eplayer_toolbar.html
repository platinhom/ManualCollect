Project Table — ePlayer Toolbar

This toolbar has controls for playing through the selected entries in the Project Table, including them in the Workspace one by one.

Go to start
Display the first selected structure.
Menu equivalent: ePlayer → Go To Start.
Previous
Display the previous structure in the list of selected structures.
Menu equivalent: ePlayer → Previous.
Play backward
Display the selected structures in sequence, moving toward the first.
Menu equivalent: ePlayer → Play Backward.
Stop
Stop the ePlayer.
Menu equivalent: ePlayer → Stop.
Play forward
Display the selected structures in sequence, moving toward the last.
Menu equivalent: ePlayer → Play Forward.
Next
Display the next structure in the list of selected structures.
Menu equivalent: ePlayer → Next.
Go to end
Display the last selected structure.
Menu equivalent: ePlayer → Go To End.
Loop
Repeat the display of the structures. The mode is determined by the choice you make from the ePlayer → Loop submenu on the menu bar.
