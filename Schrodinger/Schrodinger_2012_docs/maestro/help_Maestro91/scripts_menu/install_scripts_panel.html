Summary

The Install Scripts panel allows you to install Python scripts so that they appear on the Scripts menu.

Opening the Install Scripts Panel

To open the Install Scripts panel:

  • Choose Scripts → Install Scripts in the main window.

  • Choose Tasks → Scripts → Install Scripts in the main window.
