Scratch Project

Maestro is always working in a project. If no explicit project is loaded, Maestro creates a scratch project. Scratch projects are just like regular projects EXCEPT for the following:

  • scratch projects are not permanent
  • all entries and work in a scratch project are destroyed when the project is closed, another project is opened or you quit from Maestro
  • closing a scratch project destroys all entries and work and is replaced by another, new scratch project

The scratch project is stored in a temporary location, which you can set in the General - Directories tab of the Preferences panel.
