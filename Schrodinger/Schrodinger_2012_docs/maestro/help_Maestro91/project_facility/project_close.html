Closing a Project

To close the current project, choose Project → Close in the main window. The current project is closed and all its view settings (rotation, translation, and zoom) are saved. User-defined sets of atoms are also saved with the project. When the project is opened again, the view settings and the user-defined sets are restored.

Once the current project is closed a new, empty scratch project is created.

There is no panel or dialog box associated with this command.
