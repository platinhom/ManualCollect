Summary

This topic provides an overview of the energy-related panels that are opened from the MacroModel menu, as well as how the panels function. Each of these MacroModel panels is focused on a particular task or type of energetic calculation. All MacroModel panels have common features, which are described here.

Common MacroModel Panel Features

Molecular mechanics calculations in Maestro can be carried out by MacroModel. The calculations are set up and run from various panels that can be opened from the MacroModel submenu of the Applications menu.

The structural input for a MacroModel job may come from two sources: the Workspace or the Project Table. Make your selection from the Use structures from menu. In general, the default setting is Workspace (included entries). This means that any entries currently included in the Workspace are used as input to the job.

The other option for structural input is the currently-selected entries in the Project Table. In this case what is displayed in the Workspace may be different from what is passed to MacroModel as input for the calculation.

Note: Some types of jobs (such as minimization and coordinate scans) are restricted to taking input from a single entry. The choices in the Use structures from menu change from plural to singular:

  • Project Table (selected entry)
  • Workspace (included entry)

Make sure that only one entry is displayed in the Workspace or selected in the Project Table and then start the job.

Controls that are specific to a MacroModel panel are contained in a series of tabs. Each tab contains settings relevant to the task that the panel performs. The Potential tab defines the potential energy functions to be used (force field, solvation model, etc.). There are also tabs for setting positional constraints on atoms (Constraints) and geometric parameters (Substructure). Finally, there are tabs that are specific to the panel itself.

In the lower left corner of each MacroModel panel is a Start button. Clicking this button opens the Start dialog box which allows you to configure settings and run the job.

The Write button writes out all the files required for a job, so you can start it later. To start the job, choose Start Job From File from the MacroModel submenu of the Applications menu. Select a job in the list and click Start. This mechanism can be used to create job files which can be edited to take advantage of MacroModel features that are not accessible from Maestro.

The Close button, which is located on all Maestro panels, dismisses the current panel without starting the job or writing any files. The Help button, also on all panels, opens the Help panel with an appropriate help topic displayed.

Only one MacroModel panel can be active at any one time. If you choose an item from the MacroModel submenu of the Applications menu while another MacroModel panel is active, the existing panel is dismissed before the new one is displayed.
