Summary

This panel displays the force-field file, in its entirety, from which the parameters shown on the individual "Stretch," "Bond Angle," etc. panels were drawn.

Opening the Force Field Panel

The FField panel can be opened from any Force Field Viewer panel. To open the FField panel, open one of the nine panels that are available from the Force Field Viewer panel ("Stretch", "Bond Angle", etc.), then click Show Force Field.

Force Field Panel Features

  • Force Field Display Box
  • File
  • Line
Force Field Display Box

All the text in a given force-field file is displayed in the text box at the top of the panel. This is a read-only field.

File

This text box displays the name and path of the force field file that is currently being displayed.

Line

This box displays the number of the line in the force-field file that contains data associated with the atom set that is currently selected in the active Force Field Viewer subpanel. For newer force fields whose parameters come from a server, the line number is set to 1.
