Summary

The Start panel is used to start MacroModel calculations using command and structure files that were prepared previously.

Opening the Start Panel

To open the Start panel:

  • Choose Applications → MacroModel → Start Job From File.

Using the Start Panel

The Start panel can be used to run jobs from previously created command (.com) and structure (.dat) files.

To start a job, select it in the Available jobs in current project list and click Start. The Monitor panel opens, and the job's progress is displayed.

Start Panel Features

  • Host
  • Username
  • Available jobs in current project
  • Write structure from Maestro
  • Start
  • Update List
Host

The Host menu displays the names of all hosts that are available for use. To run the selected job on a remote host, choose the host from this menu.

Username

To run jobs on remote machines, a user name may be required. The default value is the user name of the user who started Maestro. If this user name is not correct for the selected host, you can change it in this text box.

Available jobs in current project
A list of accessible jobs is displayed in the Available jobs in current project text area. The list includes all .com/.dat file sets in the current directory that do not belong to jobs which have yet to be incorporated into the project they were started from.

Write structure from Maestro

This option determines whether or not structures from Maestro are written as input to a MacroModel job. If it is selected the Source Of Job Input and Incorporate Output Into Project By sections become available. The options in these sections are common to all energy panels and are described elsewhere (see Energy Overview).

Start

This button starts the selected job.

Update List

This button revises the Available jobs in current project list to include recently created .com/.dat file pairs, files from a job that was running when the panel was first opened, or files in a new directory.
