Summary

In this panel you can select a number of Jaguar input files and run them as a batch job.

Opening the Jaguar - Run Input Files Panel

To open the Jaguar - Run Input Files panel:

  • Choose Applications → Jaguar → Run Input Files.

Using the Jaguar - Run Input Files Panel

To select an input file, click Browse and choose a file in the file selector that is displayed. Repeat this process for as many input files as you want to use. The files are listed in the Input files list. If you want to delete a file, select it in the list and click Delete.

When you have finished selecting files, click Start. The standard Maestro Start dialog box is displayed. The incorporation options are not available because the input files are not necessarily associated with a project.

Jaguar - Run Input Files Panel Features

  • Input files list
  • Browse button
  • Delete button
  • Start button

Input files list

Displays the list of input files selected. You can select a file in this list and delete it with the Delete button.

Browse button

Opens a file selector in which you can select an input file to add to the list.

Delete button

Deletes the selected file from the list

Start button

Opens the Start dialog box, in which you can set up the job parameters and start the job.
