Window Menu

The Window menu allows you to display components of the main window, manage the open panels and save or load panel layouts.

  • Status Bar—Show or hide the status bar.

  • Clipping Planes—Show or hide the clipping planes window.

  • Command Input Area—Show or hide the Commands text box, which is used for entering Maestro commands.

  • Sequence Viewer—Show or hide the sequence viewer.

  • Toolbars—Submenu to manage display of toolbars. The toolbars are
    Manager
    Project
    Edit
    Workspace
    View
    Style
    Saved Views
    Display Atoms
    Representation
    Labels
    Build
    Fragments
    PrimeX
    Find

  • Command Script Editor—Edit and replay a command script.

  • Python Shell—Open a panel that can be used to run Python commands.

  • Close All—Close all open windows.

  • Reopen—Reopen all panels that have been opened in the Maestro session and closed. Close closes all panels, Restore restores panels that were previously open; and

  • Cascade—Place panels in a cascade from the top left towards the bottom right of the screen, in order of opening.

  • Save Layout— Save the current sizes and locations of the panels that have been opened in the current Maestro session.

  • Restore Layout— Restores the panels to the locations and sizes that were last saved.

  • Raise All—Raise all open panels.

  • Panel names—Raise the named panel to the top.
