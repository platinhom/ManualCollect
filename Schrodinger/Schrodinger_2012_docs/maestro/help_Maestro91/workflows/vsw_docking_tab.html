Summary

In this tab and its dependent dialog box, you can set up parameters for the ligand docking, and choose the docking stages that are run.

Docking Tab Features

  • Force field option menu
  • Use Epik state penalties for docking option
  • Scaling of ligand van der Waals radii for nonpolar atoms text boxes
  • Dock with Glide HTVS option and Options button
  • Dock with Glide SP option and Options button
  • Dock with Glide XP option and Options button
  • Postprocess with Prime MM-GBSA option
Force field option menu

Select the force field to be used in the docking. The choices are OPLS_2001 and OPLS_2005. The default is OPLS_2005. OPLS_2005 provides access to a wider range of atom types than OPLS_2001, including boron. Because Glide is tuned to the OPLS_2001 force field, use of OPLS_2005 might result in degradation of some results. In general, OPLS_2005 produces results of very similar quality to OPLS_2001.

Use Epik state penalties for docking option

Apply Epik state penalties to the final docking score. This includes metal-binding penalties if they were calculated. This option is selected by default.

Scaling of ligand van der Waals radii for nonpolar atoms text boxes

Scale the van der Waals radii of ligand atoms by the factor given in the Scaling factor text box if the magnitude of the atomic partial charge is less than the value given in the Partial charge cutoff text box. The scaling simulates flexibility of nonpolar groups. For more information on the scaling, see the Glide Ligand Docking — Ligands Tab topic.

Dock with Glide HTVS option and Options button

Include a high-throughput virtual screening (HTVS) docking stage in the workflow. To change the default options for this stage, click Options, and make settings in the VSW - Docking Options dialog box.

Dock with Glide SP option and Options button

Include a standard precision docking stage in the workflow. To change the default options for this stage, click Options, and make settings in the VSW - Docking Options dialog box.

Dock with Glide XP option and Options button

Include an extra precision docking stage in the workflow. To change the default options for this stage, click Options, and make settings in the VSW - Docking Options dialog box.

Postprocess with Prime MM-GBSA option

Run Prime MM-GBSA for the final poses to obtain the binding energy of the ligand. This task is performed at the end of the calculation for whatever docking stages have been selected.
