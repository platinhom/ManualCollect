Summary

In this dialog box you can define how the cells in the XP terms table are colored using the numerical values of the term.

Opening the Edit Conditional Coloring Dialog Box

To open the Edit Conditional Coloring dialog box:

  • Click Edit in the Glide XP Visualizer panel.

Using the Edit Conditional Coloring Dialog Box

To set up or change a color scheme for one or more XP terms:

  1. Select the XP terms from the Colored XP terms list.

    If it is not already in the list, select the XP term from the Available XP terms list, and click Add.

  2. Select the desired Color options for coloring the XP terms, and set the value thresholds and choose the colors.

To remove a color scheme:

  1. Select the XP term from the Colored XP terms list.

  2. Click Remove.

    The XP term is transferred to the Available XP terms list. The coloring is turned off, but the scheme is still available if you decide to reapply coloring.

When you have finished making changes, click OK to apply the color schemes and close the dialog box.

Edit Conditional Coloring Dialog Box Features

  • Available XP terms list
  • Colored XP terms list
  • Add and Remove buttons
  • Reset to Default button
  • Save as Default button
  • Reset to Original button
  • XP Term text box
  • Min value and Max value text boxes
  • Color below option and controls
  • Color above option and controls
  • Color between values option and controls
  • Color exact match option and controls
Available XP terms list

Lists the XP terms that are available for coloring. You can select multiple XP terms using shift-click and control-click to add to the Colored XP terms list.

Colored XP terms list

Lists the XP terms that have been selected for coloring. When an XP term is added to this list, its setting and colors are initialized. If the XP term has previously had color settings, these settings are restored. Otherwise, standard default values are used.

Add and Remove buttons

The Add button transfers the XP terms that are selected in the Available XP terms list to the Colored XP terms list. When they are transferred, the color options are initialized.

The Remove button removes the XP terms that are selected in the Colored XP terms list, and returns them to the Available XP terms list. The color scheme is turned off for each removed term.

The Add All button adds all XP terms from the Available XP terms list to the Colored XP terms list, and initializes their color schemes.

The Remove All button removes all XP terms from the Colored XP terms list, returning them to the Available XP terms list and turning off their color schemes.

Reset to Default button

Resets the color schemes for all terms to their default values.

Save as Default button

Saves the color schemes for all terms as the default values.

Reset to Original button

Resets the color schemes for all terms to the original values prior to any customization ("factory defaults").

XP Term text box

Displays the name of the XP term that is selected in the Colored XP terms list.

Min value and Max value text boxes

These noneditable text boxes display the minimum and maximum values of the XP term that is selected in the Colored XP terms list.

Color below option and controls

Select this option to color XP term values that are below the specified threshold. The threshold is a strict inequality (value < threshold) You can enter the threshold in the text box. To change the color, click the colored button to the right of the text box, and select a color in the color chooser that is displayed.

Color above option and controls

Select this option to color XP term values that are above the specified threshold. The threshold is a strict inequality (value > threshold) You can enter the threshold in the text box. To change the color, click the colored button to the right of the text box, and select a color in the color chooser that is displayed.

Color between values option and controls

Select this option to color XP term values that are between the Color below and Color above thresholds. There are two options:

Solid

Use a single color between the two thresholds. To change the color, click the colored button to the right of the text box, and select a color in the color chooser that is displayed.

Interpolate below/above colors, steps

Interpolate the colors betweeen the Color below and Color above colors in the given number of steps. The range of values between the two thresholds is partitioned into steps of the same size.

Color exact match option and controls

Select this option to color XP term values that are equal to the specified value (value = target) You can enter the target value in the text box. To change the color, click the colored button to the right of the text box, and select a color in the color chooser that is displayed.
