Delete Properties Panel

The Delete Properties panel is used to delete properties from the Project Table. The properties to be deleted must be selected from the Properties list. You can select all properties with the Select All button, and clear the selection with the Deselect All button.

In the Select by property family section, you can select properties that come from a particular family (such as a product) from the option menu, and select or deselect them with the Select and Deselect buttons. These buttons do not affect the selection of properties other than thoes in the chosen family.

When you are satisfied with your selection, click Delete Selected Properties to delete the properties.
