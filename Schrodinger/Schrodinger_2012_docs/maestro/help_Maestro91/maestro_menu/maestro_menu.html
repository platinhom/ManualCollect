Maestro Menu

The Maestro menu provides tools for performing various general or global Maestro operations. The standard menu items are listed below.

  • Check for Updates—Check the Schrödinger web site for software updates. Opens a web page that displays information on the status of your software.

  • Preferences— Set preferences for Maestro appearance and behavior.

  • Customize—Submenu to customize various main window components: Menus, Toolbars, and Mouse Actions.

  • Command Aliases—Create and edit aliases for commonly used Maestro commands

  • Function Key Macros—Create and edit macros that can be bound to the Fn function keys.

  • Macros—Define and edit macros.

  • Quit—Quit Maestro.
