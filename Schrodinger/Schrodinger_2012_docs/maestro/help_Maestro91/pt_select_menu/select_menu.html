Project Table — Select Menu

The Select menu provides tools for selecting entries.

All   Select all entries in the Project Table.
None   Deselect all entries.
Invert   Invert the selection: select the unselected entries, and deselect the selected entries.
Only   Open the Entry Selection dialog box, so that you can select entries on the basis of their properties.
Add   Open the Entry Selection dialog box, so that you can add entries to the selection on the basis of their properties.
Deselect   Open the Entry Selection dialog box, so that you can remove entries from the selection on the basis of their properties.
By SMARTS   Open the Select by SMARTS dialog box to select entries that match one or more SMARTS patterns.
Random   Open the Random Selection dialog box to randomly select a percentage of the current selected or included entries.
Filter   From this submenu, choose a named filter that is then used to select entries, or edit named filters in the Edit Filter panel. Filters are defined in terms of ESL (Entry Selection Language) expressions, and do not constitute a fixed set of entries. Whenever the filter is used, the ESL expression is applied to the current contents of the Project Table.
