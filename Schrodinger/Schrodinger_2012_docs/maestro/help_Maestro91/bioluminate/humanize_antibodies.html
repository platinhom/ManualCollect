Summary

This panel allows you to humanize an antibody by mutating residues to create a more human-compatible protein. Appropriate sites for mutation can be found by comparing the antibody to homologs and identifying residues that satisfy various criteria, such as solvent accessibility, maximum number of side-chain interactions with either the antigen or with the antibody itself.

Opening the Humanize Antibodies Panel

To open the Humanize Antibodies panel:

  • Choose Applications → BioLuminate → Antibody Modeling → Humanize in the main window.

Using the Humanize Antibodies Panel

The basic workflow in this panel is:

  1. Display the structure in the Workspace and analyze it, to identify residues and antibody features (Analyze Workspace).

  2. Load an antibody database that is used to search for homologs.

  3. Run the search (Search Antibody Database for Homologs) and align the homologs (Align Homologs).

  4. (Optional) Select an option for the chain to show, and examine the alignment for that chain.

  5. Choose the regions that you want to substitute residues in.

  6. Specify the criteria for automatic selection of residues, in the Selection by homology criteria and Parent structure 3D criteria sections.

  7. Click Select in the Residues Tab.

  8. In the Residues tab, make any changes to the residues marked for mutation.

  9. Choose the mutations for these residues by clicking in the Mutations column of the table and selecting from the option menu that is displayed. A default selection is included for the automatically selected residues that is based on the variations seen in the homologs.

  10. Click Start, make job settings, and start the job.

    The mutated structures are incorporated into the project as new entries, and the Humanize Antibody Viewer opens.

Only one site is mutated at a time to produce a mutated structure: multiple simultaneous mutations are not performed. The number of structures is linear in both the number of sites mutated and the number of mutations at each site (it is in fact the sum of the number of mutations at each site). The number of mutation sites and the total number of mutations is reported below the table.

Workspace analysis controls

These controls allow you to analyze the structure in the Workspace to locate the antibody regions and calculate solvent-accessible surface areas (SASA). By default, the entire Workspace is analyzed, but if you want to limit the analysis, you can select Analyze only selected Workspace residues to limit the analysis to the Workspace selection.

Humanization Criteria tab

In this tab you can set various criteria for the selection of residues to humanize.

Show in viewer options

Select which chain to show in the sequence viewer.

Find pattern text box and buttons

Find a pattern in the sequences, using PROSITE pattern syntax. Click the buttons to step through the occurrences of the pattern.

Sequence viewer

The sequences of the selected chain and its homologs are shown in this area, colored by residue type. The secondary structure assignments and disulfide bond linkages are also displayed.

Antibody database text box and Browse button

Specify the antibody database to use by clicking Browse and navigating to the database. The path to the database is displayed in the (noneditable) text box.

Search Antibody Database for Homologs button

Click this button to search the selected antibody database for homologs.

Align Homologs button

Perform a multiple sequence alignment of the homologs to the query (parent).

Substitute residues in options

Choose the region in which you want to perform residue substitution:

Pick Individual Regions button
Pick Regions by Group button

This button changes between these two alternatives when you click it. Using this button allows you to choose the regions to substitute residues in either by group or by individual region. When you click Pick Individual Regions, options for each region in both chains are displayed in the panel and the options for region groups are dimmed. When you click Pick Regions by Group, the options listed by chain are hidden again.

Selection by homology criteria options

Set criteria for the selection of residues to mutate based on the homologs found. The variations found are used as a basis for choosing default mutations.

  • Variability at position >/< N %—Filter residues based on the percentage variability at the residue position. Choose whether to apply a minimum or a maximum variability from the option menu, and specify the percentage threshold in the text box.
  • Variability at position >/< N residue types—Filter residues based on the residue type variability at the residue position. Choose whether to apply a minimum or a maximum variability from the option menu, and specify the threshold for the number of residue types that can vary in the text box.
  • Ignore positions with group conservations—Ignore (do not select) residues that are strongly conserved or that are either strongly or weakly conserved. Select the appropriate option for strong conservation or both strong and weak conservation.
    Strongly conserved groups: STA, NEQK, NHQK, NDEQ, QHRK, MILV, MILF, HY, FYW.
    Weakly conserved groups: CSA, ATV, SAG, STNK, STPA, SGND, SNDEQK, NDEQHK, NEQHRK, FVLIM, HFY.
  • Ignore parent sequence in applying above criteria—Ignore the parent (query) sequence when applying the variability or conservation criteria.
  • Parent residue different than N % homologs at position—Select residues for which the parent residue is different from more than the specified percentage of homologs at the residue position.

Parent structure 3d criteria options

Set criteria for selection of residues based on the 3D structure of the parent (the structure that was analyzed). Interactions are determined by a distance cutoff: any residue that has atoms within 4 Å of the side chain is considered to interact.

  • Solvent accessible surface area—Select this option to filter residues by their solvent-accessible surface area (SASA) relative to an isolated residue of the same type, and set a threshold for the maximum or minimum allowed relative SASA. This option is useful for locating surface residues.
  • Residue side chain makes no more than N interactions with protein—Select this option to filter out residues whose side chains interact with the protein, and set the maximum number of such interactions.
  • Residue side chain interacts/does not interact with molecule N—Select this option to filter residues by their interaction with a selected molecule. Choose whether to allow or disallow the interaction from the option menu, and specify the molecule in the text box, or select Pick and pick the molecule in the Workspace.

Select in the Residues Tab button

Select residues in the list of all Workspace residues in the Residues tab according to the criteria selected in this tab. Clicking this button also makes the Residues tab available.

Residues tab

In this tab you select the residues for mutation and choose the method for refinement of the mutated structure.

  • Residues in the Workspace table
  • Display options
  • For all selected rows, set mutations to option menu and Apply button
  • Check Mutation Box for Selected Residues button
  • Number of concurrent mutations text box
  • Refinement option menu
  • Calculate binding affinity option and menu
Residues in the Workspace table

Table containing a list of all the residues in the Workspace. This table is used to select the residues for mutation, and lists the mutations that will be applied.

Mutate Check box for mutation of the residue. When you click Start, the residues for which this box is checked will be mutated.
Residue Chain, residue number and insertion code, and 3-letter residue name.
Mutations This column lists the mutations that will be applied to the residue. If you click in the column, an option menu is displayed, from which you can choose the mutations.
Display options

Select an option for the type of residue to display in the table, from Polar, Neutral, or All.

For all selected rows, set mutations to option menu and Apply button

Select a set of mutations from the option menu, and click Apply to apply these mutations to the residues that are selected in the table.

Check Mutation Box for Selected Residues button

Check the check box in the Mutate column of the table for all residue rows that are selected in the table.

Refinement cutoff text box

Specify the cutoff distance for including residues in the refinement that are near to the mutated residues. Any residue that has an atom within the specified distance of a hypothetical Arg residue at the mutation site is included in the refinement. A hypothetical Arg residue is used to ensure that the set of residues refined is identical regardless of the initial or mutated residue identities.

Refinement option menu

Choose the method for refinement of the mutated (and other) residues.

  • Gas-phase minimization—Minimize the energy of the residues selected for refinement using the internal force-field minimizer. This is the fastest method.
  • Implicit solvent minimization—Run a Prime minimization with an implicit (continuum) solvation model.
  • Side-chain prediction—Run a Prime side-chain prediction, adjusting only the side chains.
  • Side-chain prediction (cbeta)—Run a Prime side-chain prediction that includes variation of the CA-CB direction.
  • Side-chain prediction (bb)—Run a Prime side-chain prediction that includes backbone adjustments.

Calculate binding affinity option and menu

Calculate the binding affinity of the selected chains to the rest of the Workspace structure. The chains are treated as the ligand, which can be a structure of any size, such as an antibody. By default, the heavy and light chains of the antibody are selected, so the binding affinity of the antibody is calculated.

Action buttons

  • Start—Click this button to make job settings and start the job. Opens a dialog box in which you can name the job, choose a host and the number of processors to use, and start the job.

  • Reset—Clear all data from the panel and reset all settings to their defaults.
