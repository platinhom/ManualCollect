Ligand-Based Core Hopping — Define Template Tab

    Summary

    In this tab, you define a template for the replacement of the core. This involves selecting the template molecule and defining its core.

    Define Template Tab Features

    • Input Structure button
    • Start Defining Template button
    • Require H-bond donor/acceptor matches option
    • Start Donor/Acceptor Selection button
    • Require all N critical (red) matches text box
    • Require M of N optional (blue) matches option and boxes
    • Export Template button
    Input Structure button

    Import a structure into Maestro and display it in the Workspace. Opens a file selector, in which you can choose a Maestro file or a combinatorial definition file.

    Start Defining Template button

    When you have a structure in the Workspace, click the Start Defining Template button to start defining the template for core hopping. The button text changes to Stop Defining Template.

    Click the bonds to the side chains that you want to keep as part of the new structures. The bonds are marked with an arrow pointing towards the side chain. Multiple clicks on the first bond change the direction of the arrow, then clear the bond selection, then select it again. For the subsequent bonds, multiple clicks select and deselect the bond.

    When you have finished selecting bonds, click Stop Defining Template to exit template definition mode.

    Require H-bond donor/acceptor matches option

    Require that new cores have donors or acceptors within 0.75 Å of the positions in the template core. When you select this option, the possible locations of donors and acceptors are marked in the Workspace with yellow spheres.

    Start Donor/Acceptor Selection button

    Start the selection of donor and acceptor sites that new cores are required to match. After clicking this button, you can pick spheres in the Workspace. The first click marks the site as an optional match (blue), the second marks it as a required match (red), and the third returns it to the unmarked state (yellow). The button is renamed Stop Donor/Acceptor Selection after clicking. When you have finished selecting sites, click this button. The count of required and optional matches is updated.

    Require all N critical (red) matches text box

    This noneditable text box displays the number of "critical" matches.

    Require M of N optional (blue) matches option and boxes

    Require a minimum number of matches to any of the optional matching sites. The total number of such sites that you selected is displayed in the noneditable text box. You can set the minimum number in the other box.

    Export Template button

    Export the template to a Maestro file. The file includes the core definition information.
