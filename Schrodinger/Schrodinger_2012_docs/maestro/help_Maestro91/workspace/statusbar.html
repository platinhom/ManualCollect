Status Bar

The status bar is located along the lower edge of the Workspace in the main window. It shows information about the current contents of the Workspace.

To hide the status bar, right-click in the status bar and choose Hide from the shortcut menu.

To show or hide the status bar, choose Window → Status Bar.

When the pointer is on an atom in the Workspace, the following information is displayed:

atom:chain : residue number : (element) PDB : charge : "userchoice"

  • atom = atom number in the Workspace
  • chain = chain name
  • residue = residue name
  • number = residue number + insertion code
  • element = element of the atom
  • PDB = PDB atom name
  • charge = formal charge of the atom
  • userchoice = choice of entry property. This field is set in the Preferences panel.

When the cursor is on empty space the following is displayed:

Atoms:a1/a2/a3 Entries:S/e1/e2 Res:r1 Chn:c1 Mol:m1 Chg:ch1

  • a1 = number of atoms selected in the Workspace
  • a2 = number of atoms displayed in the Workspace
  • a3 = total number of atoms in the Workspace.
  • S = indicates there is a scratch entry in the Workspace
  • e1 = number of project entries in the Workspace
  • e2 = total number of entries in current project
  • r1 = total number of residues in the Workspace
  • c1 = total number of chains in the Workspace
  • m1 = total number of molecules in the Workspace
  • ch1 = total formal charge of structures in the Workspace

If you are picking an object in the Workspace and you have the preference set, the index of the object is displayed at the beginning of the status bar, in the format Pick: index. For example, if you are picking molecules, the molecule number is displayed.

You can also show job status information in the Workspace. This information is shown by a button on the left end of the status bar. The button shows the number of active jobs in the project and the total number of active jobs. The tooltip for the button gives information on various job categories (running, failed, finished), listing up to three in each category. Clicking the button opens the Monitor panel. You can control whether the button is displayed and how often the information is updated in the Preferences panel.
