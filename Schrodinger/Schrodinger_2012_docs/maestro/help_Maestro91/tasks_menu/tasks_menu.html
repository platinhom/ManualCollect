Tasks Menu

The Tasks menu provides access to applications, workflows, and tools organized by task rather than by application. To display this menu, choose Applications → Tasks View. The Applications and Workflows menus are replaced with the Tasks menu. To change back, choose Tasks → Applications View.

The Tasks menu contains items that are on the Tools menu, the Workflows menu, and the Applications menu, but it does not contain the full range of items on these menus.

  • Ligand Preparation—Prepare ligand-sized molecules for use in modeling, with LigPrep. Includes 2D→3D conversion, adjustment or expansion of ionization states, tautomers, ring conformations, stereochemistry, and minimization.

  • Ligand Properties—Generate properties for ligand molecules that can be used as descriptors in a QSAR or QSPR model. (See Ligand and Stucture-Based Descriptors.)

  • Library Design—Prepare a library of molecules from a core structure by addition of fragments at designated attachment points (see Combinatorial Library Enumeration).

  • pKa Prediction—Predict the pKa of ligand-sized molecules. Two methods are available: Empirical, which uses Hammett and Taft methodology (Epik), and Quantum Mechanical, which uses ab initio methods and continuum solvation, with an empirical correction term (Jaguar).

  • Protein Preparation—Prepare proteins for use in modeling, with the Protein Preparation Wizard.

  • Protein Analysis—Perform various kinds of protein analyses. The submenu offers the following choices:

    • Binding Site Detection—Find potential binding sites on a protein, with SiteMap.
    • Protein Reports—Display reports on protein properties, such as geometric deviations from ideal values, G-factors, B-factors, stereochemistry, planarity, missing atoms, in the Protein Reports panel.
    • Ramachandran Plot—Display Ramachandran plot of protein dihedrals.

  • Homology Modeling—Predict protein structure from a protein sequence using information from homologous proteins, with Prime Structure Prediction.

  • Minimization—Minimize the energy of a structure with respect to geometry changes. The submenu offers the following choices:

    • Force-Field—Minimize the force-field energy with MacroModel.

    • Semiempirical QM—Minimize the semiempirical NDDO energy.

    • Ab Initio QM—Minimize the ab initio quantum mechanical energy with Jaguar.

    • QM/MM—Minimize the mixed quantum-mechanical/molecular mechanics energy with QSite.

  • Quantum Mechanics—Run electronic structure calculations with Jaguar.

  • Conformational Search—Perform a search for the conformations of one or more molecules. The submenu offers the following choices:

    • Bioactive Search—Search for ligand conformations that are likely to be bioactive, by varying only the torsions, with ConfGen.
    • Advanced Conformational Search—Perform a complete search for ligand conformations with extensive control over the search parameters, using MacroModel.
    • Monte Carlo—Search for low-energy conformations of proteins, ligands, or protein-ligand complexes using a Monte Carlo algorithm, with MCPRO+.

    • Ligand-Receptor Complex—Perform a conformational search on a ligand-receptor complex and its components, with MacroModel Embrace.

    • Macrocycle Sampling—Run a hybrid simulated annealing/large-scale low-mode sampling job to sample the conformational space of macrocycles.

  • Protein Refinement—Refine the structure of a protein, by predicting loops and side chains, and minimizing the energy, using Prime Refinement.

  • Protein X-Ray Refinement—Refine the X-ray structure of a protein using PrimeX.

  • Docking—Dock potential ligands to a receptor . The submenu offers the following tasks:

    • Grid Generation
    • Glide Docking
    • Interactive (enumeration and docking, combiglide)
    • Combinatorial Screening
    • Covalent Docking
    • Visualize XP Interactions
    • Virtual Screening Workflow
    • Induced-Fit Docking
    • QM-Polarized Ligand Docking

  • Molecular Dynamics—Run molecular dynamics simulations on a periodic system that can include a protein, solvent, salt, and a membrane, using Desmond. The submenu offers the following choices:

    • System Setup
    • Molecular Dynamics
    • Simulated Annealing
    • Replica Exchange
    • Replica Exchange Review
    • Simulation Quality Analysis
    • Metadynamics
    • Metadynamics Analysis

  • Binding Energy Estimation—Estimate the binding energy of a ligand to a protein. The submenu offers the following choices:

    • MM-GBSA—Calculate ligand binding energies using Prime MM-GBSA.
    • Linear Interaction Model—Generate a model for predicting the binding free energy of new ligands using a model based on known binding energies and simulations, using Liaison.
    • Ligand Ring Atom Mutation—Calculate relative binding energies by FEP for mutation of ligand ring atoms, with Desmond or MCPRO+.
    • Ligand Functional Group Mutation—Calculate relative binding energies by FEP for mutation of ligand functional groups, with Desmond or MCPRO+.
    • Protein Residue Mutation—Calculate relative binding energies by FEP for mutation of protein residues, with Desmond.

  • Coordinate Scan—Calculate the energy of a structure as a function of one or more coordinates at defined values of the coordinates. The submenu offers the following items:

    • Force-Field—Scan one or two coordinates with MacroModel.
    • QM Relaxed—Scan up to 5 coordinates with Jaguar, optimizing the remaining geometric degrees of freedom.
    • QM Rigid—Scan up to 5 coordinates with Jaguar, keeping the remaining geometric degrees of freedom fixed.

  • Core Hopping—Substitute the core of a ligand for other cores, retaining the side chains. The submenu offers the following choices:

    • Receptor-Based—Substitute cores in the presence of a receptor, using Glide for docking of the new ligands.
    • Ligand-Based—Substitute cores using alignment to find similar cores, based only on the ligand.
    • Isosteric Matching—Substitute cores using shape overlap to find similar cores, based only on the ligand.

  • Shape Screening—Screen a set of molecules to find those that have the same shape as a query molecule (or shape), using Phase Shape Screening.

  • Pharmacophore Modeling—Develop pharmacophore models of ligands (hypotheses) and use them to screen a database for matches; create and manage a database for screening. Uses Phase.

  • ADME—Calculate ADME properties with QikProp.

  • P450 Site of Metabolism—Predict the likely sites of metabolism of ligands in various P450 sites.

  • QSAR—Build and apply QSAR models. The submenu offers the following choices:

    • Atom-Based—Build and apply a QSAR model based on atom types, using Phase.
    • Field-Based—Build and apply a QSAR model based on molecular fields (like CoMFA or CoMSIA).
    • Property-Based—Build and apply a QSAR model based on molecular properties, with Strike.

  • Align Structures—Align structures to a reference structure. This submenu duplicates items on the Tools menu:

    • Superposition
    • Flexible Ligand Alignment
    • Protein Structure Alignment
    • Align Binding Sites

  • KNIME—Run KNIME workflows from Maestro.

  • Monitor Jobs—Open the Monitor panel, to monitor the progress of jobs.
