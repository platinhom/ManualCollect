Summary

The Clean Structures dialog box sets up a LigPrep job that performs a number of tasks to clean up and prepare ligand structures for use:

  • Convert structures from 2D to 3D
  • Add hydrogen atoms to ensure that the structure is an all-atom structure
  • Remove counter ions and water molecules
  • Add or remove protons to produce the most probable ionization state at the target pH (optional)
  • Generate stereoisomers (optional)
  • Remove "bad" structures
  • Perform an energy minimization (geometry optimization)

The options available in the Clean Structures dialog box are limited. If you need more flexibility, you can run LigPrep directly, from the LigPrep Panel For more detailed information about the process, see the LigPrep User Manual.

Opening the Clean Structures Dialog Box

To open the Clean Structures dialog box:

  • Click Clean Structures in the Prepare Ligands step of the Develop Common Pharmacophore Hypotheses panel.

Using the Clean Structures Dialog Box

To develop a pharmacophore model, the ligand structures must be three-dimensional, all-atom structures that closely represent the experimental structures. Structure databases can be represented in 2D form, without explicit hydrogen atoms, or with counter ions and solvent molecules. If any of these is the case for your database, you must run a Clean Structures job to obtain structures that are suitable for model development. The 2D-3D conversion, removal of counter ions and solvent molecules, and energy minimization are performed by default as part of the process.

If any of the structures do not have complete chirality information, you should generate the combinations of chiralities to ensure that you have covered the possibilities. Otherwise you can retain the specified chiralities or determine the chiralities from the 3D structure. If you choose either of these two options and the chiralities are not fully determined, the possible stereoisomers will be generated.

When you vary the stereochemistry, you can limit the number of stereoisomers generated. The process starts with the configuration at all chiral atoms set to R, and systematically varies the configuration. If you select fewer stereoisomers than the maximum, there is a chance that you might not generate the most important isomers.

In the generation of the 3D database, stereoisomers are treated as separate ligands. In the development of the pharmacophore model, you can choose to treat stereoisomers as separate ligands or as conformers of the same ligand.

You can ensure that any acidic or basic groups in the structures are appropriately ionized (protonated) for a selected pH value, which by default is 7, or you can neutralize the molecule or retain the input ionization states. For databases, you can also generate the most probable tautomers at pH 7. These tasks are performed with the ionizer and tautomerizer programs from LigPrep. If you want to use Epik for these tasks, you must do so independently, and then provide the structures to Phase.

Once you have selected options for cleaning up structures, click Start to run the cleanup job.

Clean Structures Dialog Box Features

  • Stereoisomers section
  • Ionization section
Stereoisomers section

In the Stereoisomers section you can choose from three options:

  • Retain specified chiralities—If the ligand has chirality information, this information is retained and used to ensure that the chiral atoms all have the correct chiralities. Chirality information includes parities and bond directions from SD files and the chirality property from Maestro files. If the configuration or chirality of a chiral center is not specified, the two possible chiralities are generated.

  • Determine chiralities from 3D structures— This option discards any information from the input file and determines the chirality from the 3D geometry. These chiralities are held fixed. For centers whose chirality is indeterminate, the two possible chiralities are generated.

  • Generate all combinations— This option discards chirality information and generates all possible configurations that result from the combination of chiralities on each chiral center.

You can set the maximum number of stereoisomers to generate in the Maximum stereoisomers per ligand text box.

Ionization section

In the Ionization section you can choose from three options:

  • Retain original states— This option bypasses the generation of ionization states. If the ligands all have the correct ionization state for acidic and basic groups, choose this option.

  • Neutralize— This option converts all acidic and basic groups into their neutral form. For example, zwitterion groups are converted from a carboxylate and an ammonium to a carboxylic acid and an amine.

  • Ionize at target pH— This option generates the most probable ionization state at the given target pH, for which the default value is 7.

The ionization states are generated by the ionizer. If you want to use Epik, you must do so independently and then import the structures into Phase.
