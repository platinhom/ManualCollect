Summary

In the Score Actives dialog box, you can adjust thresholds for keeping hypotheses during the scoring and adjust the hypothesis survival scoring function.

Opening the Score Actives Dialog Box

To open the Score Actives dialog box:

  • Click Score Actives in the Score Hypotheses step of the Build Common Pharmacophore Hypotheses panel.

Score Actives Dialog Box Features

  • Vector and site filtering section
  • Survival score formula section

Vector and Site Filtering Section

In this section, you can set the thresholds for filtering out hypotheses with low alignment and vector scores, limit the number of hypotheses returned, and set tolerances for matching features.

  • Alignment filters
  • Numerical filters
  • Feature matching tolerances

Alignment filters

The site alignment and vector scores are used to filter the hypotheses. You can set the following thresholds for the filtering:

Keep those with RMSD below threshold Å

Specify the threshold for RMS deviation of the intersite distances of any contributing ligand from those of the reference ligand.

Keep those with vector scores above threshold

Specify the threshold for the variation in the alignment of vectors between any contributing ligand and the reference ligand. The maximum is 1.0, the minimum is -1.0 (which would perform no filtering on vector alignment).

Numerical filters

These filters keep hypotheses based on their score.

Keep the top n %

Limit on the percentage of hypotheses to keep, according to their survival score.

Keep at least n and at most m

Lower and upper limits for the number of hypotheses to keep, in order of combined alignment score. If the percentage of hypotheses kept is lower or higher than these limits, these limits override the percentage limit.

Feature matching tolerances

In addition to using the RMSD to filter matches, you can set matching tolerances on individual features. Features are considered to match if the site points are within the specified tolerance. This feature is useful if the RMSD matching is satisfied, but one or more features does not match well enough.

Use feature matching tolerances option

If selected, matching of individual features will be used in filtering.

Feature table

This table lists all features (whether or not they are in the hypotheses) by their single-letter type. To set a tolerance for a feature, edit the table cell in the Tolerance column. All tolerances are applied: if you want to disable matching for a particular feature, set the tolerance to a large value.

Survival Score Formula Section

This section defines the survival score of the hypotheses, which is reported in the Hypotheses table of the Score Hypotheses step along with the individual scores that make up the survival score. The possible range for each score is given below.

vector score

Possible range is from the cutoff specified in the Vector and site filtering section to 1.0 (perfect alignment).

site score

Possible range is 0.0 (alignment at RMSD threshold) to 1.0 (perfect alignment).

volume score

Possible range is 0.0 (no volume overlap) to 1.0 (perfect volume overlap). This score is only included when its check box is selected.

reference ligand relative conformational energy

Possible range is 0.0 upward. 0.0 means that the reference ligand is the lowest-energy conformation. The energy is given in kcal/mol. The default coefficient is zero. This score is only included when the volume score is also included.

selectivity score

Possible range is from 0.0 upward. A score of 2.0 means 1 in 100 molecules would match. Selectivity score is excluded by default because it might eliminate useful hypotheses. This score is only included when its check box is selected.

number of matches

Possible range is 0 to the number of ligands in the active set. The contribution to the score is the value in the text box raised to the power of the number of matches minus 1. A value of 1.0 does not discriminate on the basis of number of matches. Useful when the required minimum number of actives is smaller than the total number of actives. Use with caution: even a value of 2.0 can give large variations in survival scores.

reference ligand activity

The range of ligand activities in the entire ligand set is indicated in the panel text. The default coefficient is zero.
