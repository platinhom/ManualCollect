Summary

In the Create Sites step, you can modify the pharmacophore features used to define the model or add new features, and then you create pharmacophore sites (labeled 3D location) for the features in each conformer of each ligand.

Working in the Create Sites Step

The basic task of this step is to create pharmacophore sites for the ligands you selected to develop a pharmacophore model. Phase supplies a set of six built-in features:

  • Hydrogen bond acceptor (A)
  • Hydrogen bond donor (D)
  • Hydrophobic group (H)
  • Negatively charged group (N)
  • Positively charged group (P)
  • Aromatic ring (R)

To use these features, you can click Create Sites and continue to the next step when the job is finished.

If you want to supplement the built-in features, you can add SMARTS patterns to define new groups that contribute to a feature, ignore patterns in the built-in feature definitions, or mark patterns that define groups that are to be excluded from the feature. To make these modifications, click Edit Features, and make the changes in the Edit Features dialog box. This dialog box also allows you to view the individual patterns for the displayed ligands.

You can also view the features in the Workspace for any of the ligands in the Ligands table. To do this, click the diamond in the In column for the ligand, select Mark selected features, and select the features you want to display from the list below. The appearance of the features is described in the Mark selected features option and list section below.

If you know that certain features on one or more of the active ligands cannot contribute to a pharmacophore model, you can mark these features as excluded, and they will not be used in the search for common pharmacophores. To do this, place the ligand in the Workspace, and select the features to exclude, either by selecting the relevant rows in the Excluded features list or by picking the features in the Workspace. (Repeated picks select and deselect the feature). You can include more than one ligand in the Workspace, to pick features for multiple ligands at the same time. This might be useful if the ligands are aligned and have the same feature that should be excluded. The ligands are identified in the Excluded features list by their row number from the Ligands table.

In addition to the main tasks in this step, you can also perform tasks from the Prepare Ligands step: assign the ligands to a pharm set either manually or by activity value, group the actives or require matches to certain actives. See the Prepare Ligands Step topic for more information.

Create Sites Step Features

The controls and features of the Create Sites step are described here.

  • Edit Features button
  • Create Sites button
  • Ligands table
  • Show markers for selected features option and list
  • Excluded features list
  • Activity Thresholds button

Edit Features button

This button opens the Edit Features dialog box, in which you can define your own features, or modify the standard features.

Create Sites Button

This button runs the job to create pharmacophore sites for all conformers of all ligands. If the sites already exist for a conformer set, a link is made to this set instead of running the job.

Ligands Table

The Ligands table is similar to the table in the Prepare Ligands step. The first four columns are the same, and have the same behavior; the selection behavior of the rows is the same, and the right-click menu is the same. See the Phase — Prepare Ligands Step topic for more information.

In place of the Conformations column is a series of columns, one for each pharmacophore feature, with the single-letter designation of the feature in the column header. These columns are populated with feature counts (the number of times a feature is present in a ligand).

Show Markers For Selected Features Option and List

This option displays the selected features on the ligands that are included in the Workspace. You can select multiple features from the list. The default features are displayed as follows:

  • Acceptor (A): Light red sphere centered on the atom with the lone pair, with arrows pointing in the direction of the lone pairs
  • Donor (D): Light blue sphere centered on the H atom, with an arrow pointing in the direction of the potential H-bond.
  • Hydrophobic (H): Green sphere
  • Negative (N): Red sphere
  • Positive (P): Blue sphere
  • Aromatic Ring (R): Orange torus in the plane of the ring
  • Custom features: Colored sphere, with a unique color. Sphere includes arrows if the feature is a vector feature.

Excluded Features List

This is a list of the features for the ligands that are included in the Workspace. The feature is represented by the row number in the Ligands table and the feature identifier (type and index). It is populated when you include ligands in the Workspace. Selecting features in this list, or picking them in the Workspace, excludes them from consideration in the search for common pharmacophores. The feature labels in the Workspace for excluded features are colored red.

Activity Thresholds Button

The Activity Thresholds button opens the Activity Thresholds dialog box, in which you can set a cutoff value for the active and the inactive ligands. Ligands with activity greater than the active threshold are included in the active set; ligands with activity less than the inactive threshold are included in the inactive set; ligands with activity between the thresholds are included in neither set (no set membership). This is the same dialog box as opened in the Prepare Ligands step.
