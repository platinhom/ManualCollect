New Features in QPLD 2012

  • The output structures from a QPLD job can be appended to the current project, by making a selection from the option menu in the Output section of the Start dialog box.

  • When running a job from the QPLD panel, the driver job runs on the local host. If you want to run the driver job on a remote host, you can write out the input files and submit the job from the command line.
