New Features in LigPrep 2.5

The following features have been added to LigPrep 2.5 in Suite 2012:

  • LigPrep command input files can be read and written from the LigPrep panel. Using the Read button reads an input file and changes the settings in the panel to match those in the file. Using the Write button writes out the panel settings to a LigPrep command input file.

  • The command options used with the ligprep command can be written to a file specified with the -writeinp option. If you also want to run the LigPrep job, you can use the -runinp option together with the -writeinp option.

  • The function of para_ligprep has been merged into the ligprep command, and para_ligprep is now merely a synonym for ligprep

  • Options have been added to the ligprep command to specify the target pH (-ph) and its range (-pht).
