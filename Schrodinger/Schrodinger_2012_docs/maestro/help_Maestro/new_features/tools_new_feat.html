Tools New Features

New features of the graphical tools that are distributed with Schrödinger Suite 2012 and are used in multiple products are listed below.

    2D Sketcher

  • Functional groups can be added to a structure by typing in the functional group name.

  • Cleanup can be applied to an atom selection.

  • Structures that originated from a project entry in Maestro can be saved to an existing project entry, or replace in the Workspace as a scratch entry.

  • Image resolution can be set when saving an image.

    • Multiple Sequence Viewer

  • Finding patterns can include PDB residue numbers with the syntax @number. This does not include insertion codes.

  • Finding patterns now uses an extended syntax in which restrictions can be placed on particular residues, e.g. Ae means Ala in an extended region. As a consequence each element (a sequence of characters that matches a single residue) must be separated from others by a - character. The simple syntax in which just the residue letters are entered is still accepted.

    Simple syntax: ACE Extended syntax: A-C-E

  • Repeat counts can be specified when finding patterns by placing the number of repeats in parentheses, e.g. A(3) for 3 consecutive alanines. A variable number is expressed as (m,n) with m<n, e.g. A(2,4) means from 2 to 4 alanines, and is matched in the sequence by AA, AAA, or AAAA.

  • You can create a similarity matrix from an alignment and use it in subsequent alignments, in the Pairwise Alignment Settings dialog box.

  • You can align sequences so that residues with identical residue numbers (and insertion codes) are aligned.

  • You can renumber residues, either by simply specifying the base and increment, or by copying the numbering from a template to which the sequence is aligned. (Update 2)

  • An Analyze Binding Site tool has been added for calculating sequence identity in an alignment for residues within a range of distances from a specified binding site.

  • Building a homology model incorporates new features from Prime, including choice of the model-building method and some new options.

  • The default color scheme has been changed to Side-chain chemistry. (Update 2)

  • You can undo and redo color scheme and annotation changes.

  • An annotation for antibody CDRs has been added.

  • You can add your own custom annotation to sequences, which you can edit in Edit mode. You can also mark rectangular regions across sequences as an annotation. (Update 2)

  • The New item from the File menu is now on the Sequences menu as Clear All. This item performs essentially the same function.

    • The MSV project is stored by default inside the Maestro project.

  • The query management items have been moved from the Sequences menu to the File menu.

  • Fasta files with residue numbers can be imported. (Update 2)

    • R-Group Analysis

  • In the R-Group Analysis panel, you can now save the results of an analysis and reload them later on for viewing.

  • You can export the table data in most panels as an image or as a CSV file. The CSV file includes structures as SMILES strings, with the attachment points marked with Xe and Kr as substitutes for the capping groups.

  • Cores are now included in activity and importance analysis.

  • You can export the histogram in the Activity Cliffs panel as an image file.

  • Controls have been added for resizing the 2D structures in most panels.

    • Scaffold Decomposition

  • A new tool for analyzing structures into possible scaffolds has been added, the Scaffold Decomposition panel.

    • Diagnostics

  • A graphical tool (Diagnostics panel) has been added for running tests and reporting on licensing and host-related conditions. The tool can be run from the command line as $SCHRODINGER/diagnostics, from the Help menu in Maestro, or as an application on Windows or Mac. This tool should be used to gather information to send to technical support.
