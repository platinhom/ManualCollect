Saved Views Toolbar

The Saved Views toolbar provides tools for saving, managing, and displaying views of the Workspace.

Save view
Save the current view as a new named view. Opens the Save View dialog box, in which you can name the view, if Prompt for name is selected in the Manage Views panel.
Previous view
Go to the previous view in the list of views.
Next view
Go to the next view in the list of views.

Go to the first view in the list. The other four numbered buttons go to the view number. These buttons are always assigned to the first five views, and the assignment changes if you reorder the list in the Manage Views panel. If there are fewer than five views, the buttons that do not have a corresponding view are not available.

Open the Manage Views panel, in which you can do various tasks on the views.
