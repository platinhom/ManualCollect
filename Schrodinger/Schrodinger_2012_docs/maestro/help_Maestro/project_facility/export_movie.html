Summary

In this panel you can export a selection of frames from a trajectory generated by Desmond or Impact as a movie.

Opening the Export Movie Dialog Box

To open the Export Movie dialog box:

  • Click Movie in the Trajectory panel.

Export Movie Dialog Box Features

  • Format options
  • Frames options
  • Options button
Format options

Select a movie format. The only choice at present is MPEG.

Frames options

Select an option for the frames to export. The options are:

  • Selected—Export the frames that are selected by the Start, End, and Step values in the Frame control section of the Trajectory panel.
  • All—Export all frames.

The number of frames is indicated after the option text.

Options button

Set options for movie speed and resolution, in the Movie - Options dialog box. The maximum speed is 25 frames per second.
