Opening a Project

To open a recently used project, choose the project from the Project menu. The number of projects displayed can be set in the General - History tab of the Preferences panel.

To open any existing project, choose Project → Open. The Open Project project selector opens, in which you can navigate to and select the project. When you click Open, the current project is closed and the selected project is opened. This button is not active until you have provided a valid project name.
