Saving a Project with a New Name

Projects are automatically saved to disk, so you never need to explicitly save a project. However, you can save a copy of the current project with a new name.

To save a project with a new name, choose Project → Save As The Save Project As project selector opens. Specify the desired location and name of the project, and click Save. This button is not active until you have provided a unique project name. Maestro adds a .prj extension to the project name if you did not supply one. When you click Save, the project is copied, including all user-created files, and the copy becomes the current project.
