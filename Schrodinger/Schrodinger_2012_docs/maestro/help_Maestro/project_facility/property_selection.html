Property Selection

Several panels have tools that can be used to select properties. In some panels, you can select a single property; in others, you can select multiple properties.

For both single and multiple property selection, there is a Subset option menu and a Filter text box below the list of properties. For single property selection, you simply choose a property from the list. For multiple property selection, you choose multiple properties from the available properties list and transfer them to the list of properties to which the designated action will be applied.

The tools are described in some more detail below.

Property list (single selection)

List of properties for selection for the designated action. The list can be restricted by using the Subset option menu or by filtering, or both. You can select a single item in this list.

Available properties list (multiple selection)

List of properties that are available for the desired action. The list can be restricted by using the Subset option menu or by filtering, or both. You can select items with click, shift-click and control-click. You can select all items with Ctrl+A (⌘A).

Subset option menu

Restricts the range of properties shown in the Available properties list for multiple property selection, or the property list for single property selection, to those belonging to the chosen subset. This option menu lists the defined property families and two other items, All properties, and Properties shown in Project Table.

Filter text box

Filter the contents of the Available properties list so that only the items that match the expression in the text box are displayed. You can use the wildcard characters * for zero or more characters and ? for a single character, and square brackets to enclose a range of characters. The filtering is done as soon as you type in the text box. To remove the filter, clear the text box.

Sort option menu

Sort the property list according to the option selected from the menu: Alphabetically or By project table order.

Arrow buttons (multiple selection)

These buttons are used to transfer properties between the source list (available properties) and the destination list (properties for action). The upper button (pointing right) moves the properties that are selected in the source list to the destination list; the lower button (pointing left) moves the properties that are selected in the destination list to the source list. You can also use the left and right arrow keys to transfer the properties between the lists.

Properties for action list (multiple selection)

List of properties to which the designated action will be applied. You can select items with click, shift-click, and control-click. You can select all items with Ctrl+A (⌘A).
