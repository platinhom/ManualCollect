Summary

In this panel, you can download workflows from the Workflows Center on the Schrödinger web site.

Opening the Update Workflows From Website Panel

To open the Update Workflows From Website panel:

  • Choose Workflows → KNIME → Update.

Update Workflows From Website Panel Features

  • Location section
  • Filter section
  • Workflows table
  • Status display
  • Install button

Location section

In this section you can choose the location for the workflows, from one of two locations that are searched by Maestro. The two choices are:

  • User Area—Install workflows for your own use in your home directory. There are separate locations for Maestro workflows and for command-line workflows, which are listed below the Install workflows in options.

  • Common Area—Install workflows for the use of all users in the current installation ($SCHRODINGER). There are separate locations for Maestro workflows and for command-line workflows, which are listed below the Install workflows in options. You must have permission to write to these directories to install workflows in them.

Filter section

In this section you can filter the workflows that are shown in the workflows table.

Show options

Select an option to filter what is listed in the table. The options are:

  • All available workflows—Show all available workflows of the chosen type in the chosen category.
  • Only workflows updated or added within N weeks—Show workflows of the chosen type in the chosen category that have been updated or added within a specified number of weeks. Enter the number of weeks in the text box.
  • New versions of installed workflows—Show only new versions of workflows of the chosen type in the chosen category that are already installed in the selected location.
  • Workflows not currently installed—Show only workflows of the chosen type in the chosen category that are not currently installed in the selected location.
Category option menu

Choose a workflow category from this option menu to display only workflows from that category. To display all workflows, choose All.

Type option menu

Choose a workflow type from this option menu, or choose All to show all workflows. There are three types:

  • Command-line—Workflows that are to be used only from the command line. Can include shell scripts and Python scripts. These workflows will be installed in the command-line location given in the Location section.
  • Maestro Command—Workflows consisting of Maestro commands. These workflows will be installed in the Maestro location given in the Location section.
  • Maestro/Python—Python scripts for use from Maestro, including Python panels. These workflows will be installed in the Maestro location given in the Location section.

Workflows table

This table lists the available workflows that satisfy your filtering choices. For each workflow, the title, workflow name, category, and last update date are given. When you pause the cursor over the title, a description of the workflow is displayed in a tooltip.

You can select multiple workflows using shift-click and control-click. The workflows you select are the ones that will be installed when you click Install.

Status display

This line, below the workflows table, shows the number of workflows displayed out of the total number available.

Install button

Click this button to install the workflows that are selected in the table. The workflows are downloaded from the Schrödinger web site. If you are not logged on, you will be prompted for your logon ID (email address) and password. Once you are logged on, the workflows will be downloaded and installed.
