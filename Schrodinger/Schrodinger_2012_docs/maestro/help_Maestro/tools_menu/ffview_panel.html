Summary

The Force Field Viewer panel facilitates detailed examination of all the force-field interactions that are present in a given MacroModel calculation. In order to use the Force Field Viewer, a MacroModel energy operation must have been performed with a Complete listing (see the Energy Calculation panel topic).

Opening the Force Field Viewer Panel

To open the Force Field Viewer panel:

  • Choose Tools → Force Field Viewer.

Using the Force Field Viewer Panel

Data displayed by the Force Field Viewer is located in .mmo files, which are created when energy operations are performed using Complete energy listings. To read an .mmo file, click Open. A file selector opens, in which the desired file can be located.

At the top of the Force Field Viewer panel, a number of buttons appear that represent the various types of interactions that can be present in a molecular mechanics calculation (stretch, bond angle etc.). After a .mmo file has been read into the Force Field Viewer, any buttons that correspond to types of interactions present in that particular file become active. Each button, when selected, displays a corresponding panel. These panels are described in detail elsewhere (see the list of related topics below).

Open

This button opens the Read .mmo File panel that allows you to navigate to the desired file.

MMO File Name

This text box displays the name, path included, of the .mmo file that is currently selected. If you know the name of the file you wish to work with, you may open it directly by entering the filename in this text box.

Stretch

This button opens the Stretch panel, which is used to display all the stretching interactions for a given calculation. When this panel is opened, the first entry in the Stretch Interactions List is selected. The location in the displayed structure that corresponds to the selected entry is marked with a dashed yellow line and magnifying glass icon.

Bond Angle

This button opens the Bend panel, which is used to display all the bending interactions for a given calculation. When this panel is opened, the first entry in the Bend Interactions List is selected. The location in the displayed structure corresponding to the selected entry is marked with a dashed red line and magnifying glass icon.

Torsion Angle

This button opens the Torsion Angle panel, which displays all the torsion interactions for a calculation. When this panel is opened, the first entry in the Torsion Interactions List is selected. The location in the displayed structure corresponding to the selected entry is marked with a pale green dashed line and magnifying glass icon.

Improper Torsion

This button opens the Improper Torsion panel, which displays the improper torsion interactions associated with a particular calculation. can be viewed. When this panel is opened, the first entry in the Improper Torsion Interactions List is selected. The corresponding location in the displayed structure is marked with a gold-colored asterisk and magnifying glass icon.

Note that the Wilson Angle button and the Improper Torsion buttons are not active simultaneously, since they perform similar operations, but are associated with different force fields. Wilson angles are used in MMFF and MMFFs force field calculations, and improper torsions are used in the others.

Wilson Angle

This button opens the Wilson Angle panel, which displays all Wilson angle interactions present in the current .mmo file. By default, the first entry in the Wilson Angle Interactions List is selected, and the corresponding location in the displayed structure is marked with a peach-colored asterisk and magnifying glass icon.

Note that the Wilson Angle button and the Improper Torsion buttons are not active simultaneously, since they perform similar operations, but are associated with different force fields. Wilson angles are used in MMFF and MMFFs force field calculations, and improper torsions are used in the others.

GB Solvation

This button opens the GB Solvation panel, which can be used to view the Generalized Born (GB) portion of the solvation interactions for a given calculation. The first entry in the GB Interactions List on the GB Solvation panel is selected by default when the panel is first opened, and the corresponding atom in the displayed structure is marked with a yellow asterisk and magnifying glass icon.

SA Solvation

This button opens the SA Solvation panel, which can be used to view the Surface Area (SA) solvation interaction components for a given calculation. When this panel is opened, the location in the on-screen structure corresponding to the selected entry in the SA Solvation List is marked with a peach-colored line and magnifying glass icon.

Van der Waals
This button opens the Van der Waals panel, which can be used to view all van der Waals interactions for a given calculation. When this panel is opened, the location in the displayed structure corresponding to the selected entry in the Van der Waals Interactions List is marked with a dashed orange line and magnifying glass icon.

Electrostatic

This button opens the Electrostatic panel, which can be used to view all electrostatic interactions for a given calculation. When this panel is opened, the location in the displayed structure corresponding to the selected entry in the Electrostatic Interactions List is marked with a dashed purple line and magnifying glass icon.
