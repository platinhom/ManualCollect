Project Table — Table Menu

The Table menu provides access to tools that operate on the contents of the table or configure the panel.

Find/Replace   Show or hide the Find toolbar to find and replace text in any table cell except the row number. This toolbar can also be displayed from the Project toolbar.
Sort   Sort project entries or entry groups. Opens the Sort Project Table panel to sort entries by property values. This panel can also be opened from the Project toolbar.
Preferences   >Open the Preferences panel at the Project Table - Properties tab, so you can set project-related preferences.
Customize   Customize the menus or toolbars. Submenu provides selection of Menus or Toolbars to open the Customize Menus panel or Customize Toolbars panel. Note: if you customize menus or toolbars, some of the information about them in the help topics or manuals will no longer apply.
Toolbars   Show or hide the toolbars. Submenu provides selection of the toolbars to show. See the topic Project Table Toolbars for information on the toolbars.
Import   Import structures or data from files. Submenu provides selection of structure, spreadsheet, vibrational data, or surfaces for import.
Export   Export structures or data to files or to Canvas. Submenu provides selection of structure or spreadsheet for export format, and an item for exporting to Canvas. The submenu items open file selectors or dialog boxes to perform the actions.
Manage Plots   Open the Manage Plots panel to plot property data for entries. This panel can also be opened from the Project toolbar.
2D Viewer   Open the 2D Viewer panel to display 2D structures for project entries in a grid. This panel can also be opened from the Property toolbar.
Record Movie   Record an MPEG movie of the selected entries in the Project Table. Opens the Record Movie dialog box, in which you can specify the file name or browse to the location, and set options for resolution and speed in an Options dialog box. The preference for fitting to screen when the Workspace changes is honored when the movie is recorded, so it is possible to have a blank frame in the movie.
Single-Entry Feedback   Display or undisplay entry information in the Workspace for a single "target" entry, subject to the related preference.
