Summary

In this panel, you can set up and run FEP calculations for the mutation of a ring atom in a ligand. The panel is designed to make setting up FEP jobs as easy as possible: most of the computational details, such as setting the force field parameters, solvation, relaxation of the system, and setting the FEP simulations, are taken care of automatically.

Opening the Ring Atom Mutation by FEP Panel

To open the Ring Atom Mutation by FEP panel:

  • Choose Applications → Desmond → Ring Atom Mutation by FEP.

Using the Ring Atom Mutation by FEP Panel

Ring atom mutation changes a selected ring atom to a different atom type while preserving the hybridization of the atom. FEP simulations are automatically prepared, performed, and analyzed to yield the free energy difference between the mutant and the original molecule for each of the selected atom types.

To run a ring atom mutation job:

  1. Include in the Maestro Workspace the system that contains the ligand to be mutated, the receptor protein (optional), and any other key molecules of the system.

    You can use an existing model system that may, for example, include a membrane. If you do, note that the SPC water model is used in the FEP simulations, regardless of the choice made for your model system. This step can be performed prior to opening the panel.

  2. Select Pick the ring atom, then pick the ligand atom to be mutated in the Workspace.

    When the atom is picked, it is rendered in ball-and-stick, and the ligand is colored with green carbons.

  3. In the Mutation targets table, select the targets that you want to use to replace the original ring atom by selecting their table rows.

    Use shift-click and control-click to select multiple table rows. You can view the mutated ligand by clicking in the View column.

  4. (Optional) If you want to adjust the structure or add a functional group to the mutant, use the adjustment and build tools to make the desired changes to the mutated ligand structure.

  5. In the Plan calculation tab, choose the calculation types.

    The choices will depend on whether you want to calculate relative binding free energies (complex and solvent) or relative solvation free energies (solvent and vacuum).

  6. Choose the FEP protocol you want to use for each calculation type from the FEP protocol option menu.

  7. Set the buffer size for the periodic box. The buffer size is ignored if you use a prepared model system as input.

  8. Set the time used for the actual simulation (following any initial simulation) at each lambda point.

  9. Set the temperature for the simulation.

  10. Click Start, set the job parameters in the Start dialog box, and click Start in the dialog box to run the job.

Define Perturbation tab

This tab displays a series of steps to follow. Each step has text describing the task for the step.

Step 1 involves displaying the ligand, the receptor, and any other important molecules in the Workspace.

Step 2 involves picking the ring atom for the functional group substitution. It has a single option, Pick the ring atom. When this option is selected, you can pick an atom on the ligand in the Workspace. When the atom is picked, it is renedered in ball-and-stick, and the ligand is colored with green carbons. Once focus is returned to the panel, this option is deselected. You cannot pick substituted ring atoms.

Step 3 involves selecting the targets that will replace the ring atom. This step is performed using the Fragment library table. The targets that will be used are the targets that you select in the table. To view the ligand with a particular target replacing the substitution group, click in the View column.

The button in the Reset column resets the substituted ligand to its initial geometry, if you have done any adjustments. You should not make modifications to the ligand core or other molecules in the system.

Action buttons

At the bottom left of the panel are three buttons that perform various actions:

  • Start—Opens the Start dialog box, in which you can set the job parameters such as job name, host, number of CPUs, and so on, and then start the job.

  • Write—Writes the input files for the job but does not start the job. Opens a dialog box in which you can specify a job name, which is used to name the files. This capability is useful if you want to change the FEP protocol.

  • Reset—Resets the entire panel to its initial state. This includes clearing the Workspace. If you want to use the same structures, you will have to redisplay them befor proceeding.
