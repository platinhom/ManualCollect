Summary

In this tab you choose the types of calculation that will be performed and the FEP protocol for each type, and set some options for the simulation. Normally you will choose only two types of calculation to obtain the desired relative free energy.

Plan Calculation Tab Features

This tab has three sections, each of which consists of an option for enabling the calculation type, some text, a FEP protocol option menu, and some simulation settings. Below these sections is a text box for setting the temperature.

  • In complex section
  • In pure solvent section
  • In vacuum section
  • FEP protocol option menu
  • Buffer size text box
  • Production simulation time text box
  • Use lambda hopping to enhance sampling option
In complex section

Select the In complex option if you want to perform FEP calculations for the solvated protein-ligand complex, and choose a protocol from the FEP protocol option menu, and make any simulation settings.

In pure solvent section

Select the In pure solvent option if you want to perform FEP calculations for the solvated ligand, choose a protocol from the FEP protocol option menu, and make any simulation settings.

In vacuum section

Select the In vacuum option if you want to perform FEP calculations for the free (unsolvated) ligand, choose a protocol from the FEP protocol option menu, and make any simulation settings. This option is deselected by default.

FEP protocol option menu

The FEP protocol option menu offers five choices for calculations in the complex or in pure solvent:

  • Desmond NPT—Use the NPT ensemble for the calculation
  • Desmond NVT—Use the NVT ensemble for the calculation
  • Desmond NPT (quick relaxation)—Use the NPT ensemble with a short relaxation time for the calculation
  • Desmond NVT (quick relaxation)—Use the NVT ensemble with a short relaxation time for the calculation
  • User defined—This option displays a text box and Browse button, which allow you to navigate to a protocol file (.msj). You can set up a protocol using the Desmond panel.

For calculations in vacuum, the choices are:

  • Desmond vacuum—Perform the calculation on the free system (no solvent).
  • Desmond vacuum (quick relaxation)—Perform the calculation on the free system with a short relaxation time (no solvent).
  • User defined—This option displays a text box and Browse button, which allow you to navigate to a protocol file (.msj). You can set up a protocol using the Desmond panel.
Buffer size text box

Specify the size of the buffer zone in angstroms between the solute and the box boundary. The buffer size is ignored if you use an existing model system as input.

Production simulation time text box

Specify the simulation time in ns for the production simulation (after any initial simulations) for each lambda value. The default value is 0.6 ns for relative free energies (mutation FEP calculations), and 2 ns for calculations in vacuum. The longer time for total free energies is recommended for accurate results.

Use lambda hopping to enhance sampling option

Use Hamiltonian replica exchange (lambda hopping) to enhance the sampling. Lambda hopping is useful when the endpoints have preferred rotational states that are different or when the system has conformational flexibility. This option is not available for total free energy by FEP.
