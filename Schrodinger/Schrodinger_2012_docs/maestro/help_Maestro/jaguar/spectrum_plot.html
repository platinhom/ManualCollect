Summary

In this panel you can plot IR, UV/visible and VCD spectra from data generated by Jaguar. You can plot multiple spectra in stacked or overlaid format, and plot weighted averages of multiple spectra.

Opening the Spectrum Plot Panel

To open the Spectrum Plot panel:

  • Choose Applications → Jaguar → Plot Spectra in the main window.

Using the Spectrum Plot Panel

The Spectrum Plot panel plots spectra that are generated from Jaguar calculations. Jaguar produces transition frequencies and transition probabilities, but does not provide any information on line widths. Default values of the widths are provided, but you can change them. Multiple spectra can be displayed on separate plots, stacked vertically, or on a single plot, either overlaid or as a weighted average.

To generate spectral data for plotting:

  • For UV/Vis spectra, run a CIS or TDDFT calculation (select Excited State in the Theory tab of the Jaguar panel).

  • For IR spectra, run a frequency calculation (select Vibrational frequencies in the Properties tab of the Jaguar panel).

  • For VCD spectra, run a vibrational circular dichroism calculation (select Vibrational circular dichroism in the Properties tab of the Jaguar panel)

To display spectra in the Spectrum Plot panel:

  1. Select the entries in the Project Table for which you want to plot spectra.

    These entries must have been incorporated (or imported) from a Jaguar job that generated spectral data.

  2. Choose Applications → Jaguar → Spectrum Plot.

  3. Choose the spectrum type from the Type option menu.

  4. Optional: Choose the line shape and units, if available.

  5. If plotting multiple spectra, choose the plot format from the Multiple spectra option menu.

    If you chose Average weighted by property, click Choose Property to select an entry property to use for the weights. Suitable properties might be concentrations or Boltzmann weights. If you do not choose a property, equal weights are used.

  6. Click Plot Spectra for Selected Entries.

    A file selector opens for each entry, in sequence.

  7. Navigate to the spectrum file for each entry, select it, and click Open.

    When you open the last file, the spectra are plotted.

Once you have displayed the spectra, you can make various changes.

  • To change the color, line style, or line width for a spectrum, choose it from the Spectrum option menu, then click the Options toolbar button and make the changes in the Figure options dialog box

  • To change the frequency or wavelength scale or the half-bandwidth for all spectra, select Share scale factors and bandwidths, then make settings in the Scale factor and Half-bandwidth text boxes (or with the sliders).

    To change the frequency or wavelength scale or the half-bandwidth for each spectrum independently, firs deselect Share scale factors and bandwidths, then choose the spectrum from the Spectrum option menu and make settings in the Scale factor and Half-bandwidth text boxes (or with the sliders). For a stick spectrum, set the half-bandwidth to 0.

  • To label peaks in the spectrum with the frequency and intensity, select the lines in the table, then click Label Selected Frequencies.

  • To display a legend that shows the color, style, and width of the lines used for each spectrum, select Legend.

If you have multiple spectra, you can switch between overlaid, stacked, and averaged by selecting from the Multiple spectra option menu. You can change the weighting property also.

To save an image of the spectrum click the Save image button on the toolbar.

You can then select an image format, navigate to a location and save the image file.

If you choose a different spectrum type from the Type option menu, you must click Plot Spectra for Selected Entries again to load spectrum files for the new type and plot them.

Toolbar

The toolbar provides tools for manipulation of the plot and for saving an image of the plot. This is a generic toolbar, and some of the actions may not be useful in the current context. The toolbar buttons are described below.

Reset
Reset the plot to the original pan and zoom settings.
Back
Display the previous view of the plot in the view history.
Forward
Display the next view of the plot in the view history.
Pan/zoom
Pan the plot with the left mouse button, zoom with the right button.
Zoom to rectangle
Drag out a rectangle on the plot to zoom in to that rectangle.
Configure subplots
Configure the margins and spacing of each plot in the panel.
Edit axis and curve parameters
Make settings for the title, range, labeling, and scale of the axes; the color, style, and width of lines; and the color, style, and size of markers.
Save image
Save an image of the plot to file. Opens a file selector in which you can browse to a location, select the image format, and name the image.
Plot area

This area displays the spectra. You can use the toolbar to change the overall positioning and zoom level of the spectra.

Type option menu

Choose the type of spectrum from this option menu.

Line shape option menu

Choose the line shape used for the spectrum. The available shapes are Lorentzian and Gaussian.

Units option menu

Choose the units for the display of the energy, from nm or wavenumbers. Only available for UV/Vis spectra.

Legend option

Select this option to display a legend for the spectra in the top right corner of the plot area. The legend displays the line type, the entry title and entry ID. You can drag the legend anywhere in the plot area.

Multiple spectra option menu

Choose how to display multiple spectra.

  • Overlaid—The spectra are plotted on top of each other on the same plot. There is only one Y axis.
  • Stacked—Each spectrum is plotted separately (with its own Y axis), in a vertical stack. Zooming and panning works on all spectra simultaneously.
Share scale factors and bandwidths option

This option links the scale factors and band widths for all the spectra. Deselect it if you want to make changes to individual spectra.

Averaged weighted by property option

Sum the spectra with weights specified by a property chosen by clicking Choose Property.

Choose Property button

Choose a property to use for weighting the spectra. Opens the Property for Averaging dialog box, from which you can choose an entry property. The property name is displayed to the right of the button. Suitable properties would be ones that represented the concentration of the different molecules, or their Boltzmann weights.

Show components option

Show the components of the averaged spectrum (i.e. the source spectra for the averaging). The components are stacked or overlaid, depending on the choice from the Multiple spectra option menu.

Spectrum option menu

Choose the spectrum for which you want to change the parameters.

Scale factor text box and slider

Scale the frequencies of the spectral lines by the specified factor.

Half-bandwidth text box and slider

Specify the half-bandwidth (full width at half maximum) for the spectral lines.

Color option menu

Choose a color for the spectrum, from a selection of 9 colors.

Style option menu

Choose a line style for the spectrum, from Solid, Dashed, Dot, or Dash-Dot.

Width option menu

Choose a width for the lines in the spectrum.

Remove button

Remove the spectrum from the plot area.

Spectral line table

This table lists the frequency and the intensity of the spectral lines in the spectrum that is selected from the Spectrum option menu. You can select multiple table rows with shift-click and control-click.

Label Selected Frequencies button

Label the lines that are selected in the table with the frequency value and the intensity value.

Clear Labels button

Remove the labels from the spectrum.

Frequency only option

Label peaks with the frequency only, instead of with the frequency and the intensity.

Plot Spectra for Selected Entries button

Read the spectrum files (.spm) of the chosen spectrum type for the entries that are selected in the Project Table and and plot the spectra. A series of file selectors opens, so that you can select the correct spectrum file. The file selector title includes the entry title and entry ID. When a spectrum file has been opened for each entry, the spectra are plotted according to the settings made in the panel. Colors and line styles are assigned automatically, but you can change them if you want.

Read Experimental Spectrum File button

Read an experimental spectrum and plot it. An experimental spectrum must be in a plain text file (.txt), consisting of pairs of frequency and intensity values, one per line, with the values separated by spaces. This allows you to compare experimental spectra with predicted spectra. The experimental spectra are always displayed, and are not included in any averaging. Experimental spectra have no data on the positions or intensities of lines, so the Frequency table is empty, and controls for the line shape or scaling factor are not available.

Export Data button

Export the spectra to a tab-delimited file. The file contains two columns for each spectrum, with the x (frequency) and y (intensity) data. They are labeled with the spectrum name as name-X and name-Y. The data exported include the line broadening. This feature should be useful for importing spectra into other plotting programs.
