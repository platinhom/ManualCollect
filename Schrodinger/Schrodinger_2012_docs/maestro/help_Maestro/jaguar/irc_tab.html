Summary

In this tab you specify the reactant, product, and transition state for the IRC (intrinsic reaction coordinate) scan, and set parameters for the scan.

For information about IRC scans, see the Jaguar Intrinsic Reaction Coordinate Scans topic.

For information about the controls at the top and bottom of the panel, see the Jaguar Panel topic.

IRC Tab Features

  • Direction option menu
  • Structures section
  • IRC scan controls

Direction option menu

Choose the direction in which the scan proceeds from the transition state. menu. Sets ircmode in the gen section. All options except Downhill require an initial Hessian. Forward means toward the product; reverse means toward the reactant.

Forward/Reverse

Calculate points in both forward and reverse directions. This is the default. Sets ircmode to both.

Forward

Calculate points in the forward direction. Sets ircmode to forward.

Reverse

Calculate points in the reverse direction. Sets ircmode to reverse.

Downhill

Calculate points in the downhill direction from the provided non-transition-state structure. Sets ircmode to downhill.

Structures Section

Select the transition state, reactant, and product geometries in this section. The selections in this section add zmat2 and zmat3 sections to the input file. The controls in this section are the same as for the Transition State tab. If you already specified structures in that tab, they are specified here by default.

You must choose entries from the Project Table for these structures. If the atom numbering or labeling scheme is incompatible, a warning is displayed.

  • Inclusion state button
  • Entry name text box
  • Choose button

Inclusion state button

When you have chosen a structure for a given structure type, this button becomes available. You can click it to display the structure in the Workspace, and click again to remove the structure from the Workspace. These buttons do not behave like the In column of the Project Table: they only affect the inclusion state of the corresponding entry. The inclusion state is reflected in the Project Table.

Entry name text box

Displays the entry name for the structure type. You can enter an entry name in this text box to select it for the structure type.

Choose button

Click this button to open an entry chooser, in which you can choose an entry for the structure type.

IRC Scan Controls

Set the parameters that control the scan.

  • Number of IRC points text box
  • IRC step size text box
  • Maximum optimization steps per point text box
  • Use mass-weighted coordinates option

Number of IRC points text box

Set the number of IRC points to be located in each direction. Sets ircmax in the gen section.

IRC step size text box

Set the size of the step along the reaction coordinate to be taken for each point. Sets ircstep in the gen section.

Maximum optimization steps per point text box

Set the maximum number of steps in the optimization of the geometry perpendicular to the reaction path. Sets ircmxcyc in the gen section.

Use mass-weighted coordinates option

Use mass-weighted coordinates for the reaction path. Selected by default. Sets irc=2 in the gen section. Use of mass-weighted coordinates defines an IRC path. If you deselect this option (sets irc=1), the path is a minimum energy path (MEP).
