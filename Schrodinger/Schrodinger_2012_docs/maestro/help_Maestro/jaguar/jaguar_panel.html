Summary

The Jaguar panel provides an interface for setting up and running Jaguar jobs. The controls in the panel are configured differently for each kind of Jaguar task.

For information on the controls in a tab, see the help topic for the tab, which you can open by clicking Help when the tab is displayed. For information on the tasks, see the help topic for the task. Links to these topics are given in the Related Topics section.

Opening the Jaguar Panel

To open the Jaguar panel for a particular task, choose the task from Applications → Jaguar or Tasks → Quantum Mechanics.

Jaguar Panel Features

The Jaguar panel has some controls at the top for input selection, a set of tabs in the center, and a row of buttons at the bottom of the panel.

  • Input selection
  • Tabs
  • Buttons

Input selection

  • Use structures from option menu
  • Files text box
  • Browse button

Use structures from option menu

Choose the source of the structures for the Jaguar job from this menu. If you choose Selected structure files, the File text box and the Browse button become available.

Files text box

Enter the names of the structure files you want to use in this text box, separated by commas. You can scroll the text box horizontally by dragging.

Browse button

Click this button to open the Select Structure File file selector, in which you can select one or more structure files. When you click Open, the file selector closes and the files you select are listed in the Files text box.

Buttons

  • Start
  • Read
  • Write
  • Edit
  • Reset
  • Close
  • Help

Start

Opens the Start dialog box, in which you can make selections for running the job and incorporating the output, and then start the job.

Read

Opens the Jaguar Read file selector in which you can select a Jaguar input file to read.

Note: Maestro discards @, *, and # symbols when an input file is read. This means that any counterpoise atoms, atoms for Hessian refinement, and constraints that were set in the input file are no longer set.

Write

Opens the Jaguar Write file selector in which you can write out the input file.

Edit

Opens the Edit Job dialog box, in which you can edit the entire input file or just the geometry (structure).

Reset

Resets the options in all the tabs to the default values.

Close

Closes the Jaguar panel.

Help

Opens your browser with the help topic for the current tab displayed.
