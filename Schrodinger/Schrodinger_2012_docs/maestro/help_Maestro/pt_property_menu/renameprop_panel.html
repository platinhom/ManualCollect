Summary

The Rename Property panel allows you to rename a property.

Opening the Rename Property Panel

To open the Rename Property dialog box:

  • Right-click on the property heading and choose Column → Rename.

Using the Rename Property Panel

To rename a property, choose the property from the Property list, enter the new name in the the Name text box, then click the Set.

If you want to change the internal name as well, select Change internal name. The internal name is given below theName text box.

You can rename a property by clicking in the property column heading, editing the name, and pressing Enter.
