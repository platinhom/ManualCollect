Summary

The ConfGen Advanced panel is used to set up conformation generation starting from the structure in the Workspace or from selected entries in the Project Table. The ConfGen Advanced utility is used to perform very efficient searches that rapidly sample diverse conformations of small ligand-like molecules.

Note: Input structures must be all-atom, 3D structures. If your structures are not 3D structures with hydrogens, you can use LigPrep to convert them.

Opening the ConfGen Advanced Panel

To open the ConfGen Advanced panel:

  • Choose Applications → ConfGen → Advanced in the main window.

ConfGen Advanced Panel Features

The ConfGen Advanced panel contains three tabs, Potential, Mini, and ConfGen.

The Potential tab is essentially the same as the Potential tab in the MacroModel panels. The differences are:

  • The H-bond cutoff is not present; it is replaced by the Suppress hydrogen-bonding electrostatics option, which is available regardless of the Cutoff option menu choice.

  • The Debugging options text box is not present

  • There is another option, Enhance planarity of conjugated pi groups, which can be used to increase the torsional potential around bonds between atoms whose geometry should be planar (i.e. sp2 atoms). This option should make aromatic rings, amides, esters, and so on, less likely to adopt a nonplanar geometry.

For information on this tab, see the MacroModel Potential Energy Description topic.

The Mini tab is the same as the Mini tab in the MacroModel panels, except that the Maximum iterations text box is replaced by a Maximum number of iterations section with two text boxes:

Preminimization of input structures

Enter the number of cycles of minimization of the input structures to be performed prior to the conformational search. The default is 100.

Postminimization of generated structures

Enter the number of cycles of minimization of the generated structures in this text box. The default is 0.

For information on this tab, see the MacroModel — Minimization Panel topic.

The third tab, ConfGen, is described in the next section.

Search Parameters

There are several ways of limiting the number of conformations returned by the search. The number of conformations actually returned by the search is the minimum of the number of conformations ConfGen can generate for the molecule, the specified maximum number of conformers generated, and the specified multiple of the number of rotatable bonds. These limits are specified under the Maximum number of conformations returned heading.

Maximum number of conformations returned text boxes

Specify the maximum number of conformations to be returned by the search. This number is then subject to other filters, which are set up with other options in this tab. There are two controls for specifying the maximum number:

  • Total text box—Specify the maximum number of conformations in total to be returned by the search. If ConfGen finds more conformations than this for a given ligand, it uniformly selects conformations from this collection to ensure broad coverage of conformational space. If ConfGen finds fewer conformations than this for a given ligand, only that many conformations are returned.

  • Number of rotatable bonds times option and text box—Specify the maximum number of conformations to return from the search, as a multiple of the number of rotatable bonds in the molecule. With this option, the number of conformations to be returned is limited to the specified number times the number of rotatable bonds in the molecule. Sampling is done if more conformations than the limit are found. The number specified in the Total text box overrides this limit if the latter is larger.

Save at most n conformations per ligand option and text box

Specify the maximum number of structures to save from each search. If this number is less than the number of conformations returned from the search, the conformations are selected by energy, after eliminating redundant conformers and applying other filters. A zero value means "save all unique structures", which is the same as deselecting this option.

Retain mirror-image conformations option

Treat mirror-image structures as separate conformers, and retain both. Distinguishing mirror images is important for ligand-protein interactions, where it is usually only one stereoisomer that is active.

Search mode option

Choose the type of search:

  • Rapid: Only one peripheral group's orientation is varied at a time.
  • Thorough: Samples all combinations of orientations of the peripheral groups (default).

Structure Options

Amide bonds option menu

Choose how amide bonds are treated:

  • Vary conformation (default)
  • Retain original conformation
  • Set conformation to trans
Sample rings option and Maximum ring conformations text box

Select this option to sample ring conformations, and enter a value in the Maximum ring conformations text box. The default is 16 and the value must be greater than 0.

Conformation Comparison and Filtering

Energy window for saving conformations text box

Specify the threshold value for comparison of trial structures. New structures are kept only if their energy is less than this value above the current global minimum. Lowering this value results in fewer structures saved. The default value is 100 kJ/mol.

Compare conformers by option menu

Choose the class of atoms by which to compare conformers:

  • None
  • Heavy atoms
  • Heavy atoms plus polar hydrogens (default)
Eliminate redundant conformers using options and Cutoff text boxes

Select the distance mesasure for determining whether structures should be considered equivalent:

  • Maximum atom deviation—consider structures to be different if the maximum atom deviation for any pair of corresponding atoms exceeds the threshold given in the Cutoff text box. The default value is 1.5 Å.

  • RMSD—consider structures to be different if the RMS deviation for all compared atoms exceeds the threshold given in the Cutoff text box. The default value is 1.0 Å.

These options are available only if Heavy atoms or Heavy atoms plus polar hydrogens is chosen from the Compare conformers by option menu.

Maximum torsional angle difference for polar H's

This option is only enabled if Heavy atoms plus polar hydrogens is selected.
