Summary

Run a hybrid simulated annealing/large-scale low-mode sampling job to sample the conformational space of macrocycles.

Opening the Macrocycle Conformational Sampling Panel

To open the Macrocycle Conformational Sampling panel, do one of the following:

  • Choose Applications → MacroModel → Macrocycle Conformational Sampling in the main window.

  • Choose Tasks → Conformational Search → Macrocycle Sampling in the main window.

Using the Macrocycle Conformational Sampling Panel

This panel provides access to a protocol for conformational sampling of macrocycles that is more suited to their features than the small-molecule conformational searches or protein conformational searches. Candidates generated by iterative simulated annealing runs are further sampled with a large-scale low-mode search. By default, broad large-scale low-mode moves are allowed and the eigenvalues are calculated more frequently, which may be useful for macrocycle systems.

The options in this panel are a subset of the options available from the command line: for example, the initial temperature, the high-temperature simulation length, the cooling phase length, and the simulation time step can be set from the command line. The defaults have been chosen on the basis of many tests, so there should be little need to modify them, but you can do so by running the macrocycle_conformational_sampling.py script.

Use structures from option menu

Specify the source of the input structures. You can use the selected entries in the Project Table or a file. If you choose File, specify the file with the Input file text box and Browse button.

Input file text box and Browse button

If the source of input structures is a file, specify the file by entering the path to the file in the text box, or click Browse and navigate to the file.

Force field option menu

Choose the force field for the simulations and conformational search. The usual range of force fields is supported.

Electrostatic treatment options

Select an electrostatic treatment for the environment of the macrocycle. The options are to use the GB/SA continuum solvation model with water as the solvent, or a distance-dependent dielectric constant.

Energy window for saving structures text box

Specify the energy window relative to the global minimum for saving structures. Structures whose energy is more than the specified amount above the global minimum are discarded.

Eliminate redundant conformers using RMSD text box

Specify the threshold for eliminating redundant conformers. Any conformer whose RMSD from another conformer is within this threshold is considered redundant.

Simulation cycles text box

Specify the number of simulated annealing iterations. To disable the simulated annealing stage, enter 0 in the text box.

LLMOD search steps text box

Specify the number of large-scale low-mode search moves. To disable this stage of the calculation, enter 0 in the text box.

Eigenvectors determined for options

Select an option for when the eigenvectors of the Hessian are recalculated:

  • Original structure only—Calculate the eigenvectors for the input structure only. This option saves time, but the results may be of lower quality.

  • Each new global minimum—Calculate the eigenvectors each time a new global minimum is found.

  • Each search step—Calculate the eigenvectors for every new conformer that is found.

Torsion sampling options option menu

This option menu allows you to control the automatic setup of torsional sampling around some bonds that generally have high barriers. These bonds are the C-N and C-O single bonds in groups like amides and esters, and C=N and N=N double bonds. The four levels of sampling are as follows:

Restricted   Do not sample any of these bonds
Intermediate   Sample C-N and C-O single bonds other than in standard amides and esters
Enhanced   Sample all C-N and C-O single bonds
Extended   Sample all C-N and C-O single bonds and C=N and N=N double bonds
Action buttons

  • Start—Open the Start dialog box to set job options and run the job.
  • Reset—Reset all settings in the panel to their defaults.
