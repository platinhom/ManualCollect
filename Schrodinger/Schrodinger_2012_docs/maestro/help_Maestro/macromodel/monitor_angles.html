Summary

This panel is used to set up angles to be monitored during a dynamics simulation. Results are saved in the .mmo file.

Opening the Monitored Angles Panel

To open the Monitored Angles panel, click the Angles button in the Monitor tab of the Dynamics panel.

Using the Monitored Angles Panel

Angles to be monitored can be specified by picking the atoms that define the angles, using the standard picking controls in the Define monitored angles section, and clicking on three atoms. When the first two atoms are picked, Maestro places purple boxes around them. After the third atom is picked, the atoms appear in the list at the top of the panel. Maestro traces the angle with a solid green line, draws a dotted green line through the angle, and places a green "eye" icon beside the dotted line. To distinguish the currently selected angle, the lines are drawn thicker.

To delete an angle from the list, select it in the list and click Delete. To delete all monitored angles, click Delete All.

Monitored Angles Panel Features

  • Monitored angles list
  • Define monitored angles section
  • Delete button
  • Delete All button
Monitored angles list

Lists the angles to be monitored. After the third atom is picked, a new entry appears in the list.

Define monitored angles section

The Define monitored angles section contains the standard picking controls, which are used to select atoms that define an angle. It also contains a Markers option, which you can use to show markers for the monitored angles in the Workspace.

Delete button

To delete a monitored angle, select it in the Monitored Angles List, then click Delete.

Delete All button

Click this button to delete all monitored angles.
