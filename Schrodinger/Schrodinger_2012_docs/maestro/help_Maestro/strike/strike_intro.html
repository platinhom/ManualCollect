Summary

Strike is a collection of statistical tools for examining correlations within a dataset. It can develop and employ QSAR/QSPR models using a number of linear approaches, generate univariate and bivariate statistics, and perform similarity/diversity analysis in descriptor and 2D-structure space.

Using Strike

Strike is a chemically-aware statistical package which is integrated with Maestro to provide a flexible, intuitive interface. (Additional features are available from the command line. For more information, see the Strike User Manual.) Using molecular data generated by Schrödinger software or other sources such as third-party software or experimental data, Strike can be used to generate basic univariate and bivariate statistics such as mean, median, mode, averages, covariance, and correlation; generate structure-activity relationship hypotheses using rigorous statistical methods; run validation tools to assess the validity and predictive power of generated QSAR/QSPR models; employ such models as filters and predictive tools; and generate similarity analyses.

Before using Strike, molecular data (in this document, referred to as "descriptor data") should be obtained and imported into the Maestro Project Table. This data can be generated by running QikProp, for example, or by using the Ligand & Structure-Based Descriptors panel to run a variety of Schrödinger programs. Alternatively, descriptors generated by external programs or sources may be imported using standard comma-separated value (csv) format files.

Once the data is incorporated in the Project Table, you can perform statistical analyses and create and use QSAR models using the Strike panels described in the next section.

Strike Features

The Strike interface in Maestro consists of three panels:

  • The Build QSAR Model panel
  • The Predict panel
  • The Similarity panel
Build QSAR Model Panel

Use this panel to generate a QSAR model using a training set of molecules selected from the Maestro Project Table, a set of independent descriptors, and a dependent descriptor chosen from those available in the data.

Predict Based on QSAR Model Panel

Use this panel to import a model or select one from the table of generated models, then perform property predictions for molecules that were not part of the training set. Results can be viewed and unsatisfactory models can be deleted from the table.

Similarity Panel

Use this panel to determine similarities in descriptor or 2D-structure space. Several distance-based descriptor similarity measures are available. Similarity in 2D-structure space is determined using an atom-pair-based approach.
