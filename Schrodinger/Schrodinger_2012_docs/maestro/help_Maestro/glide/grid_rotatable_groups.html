Summary

In this tab you can choose to treat receptor hydroxyl and thiol groups as flexible, and select the groups that are actually treated flexibly. H atoms in flexible receptor groups cannot be used for constraints.

Treating hydroxyl and thiol groups flexibly takes longer in the docking run, but the extra time is much less than taken by generating grids for each combination of flexible group orientations and docking to each of these grids.

Rotatable Groups Tab Features

  • Allow rotation of receptor hydroxyl and thiol groups option
  • Available groups table
  • Pick groups option
  • Use Existing Definitions button
Allow rotation of receptor hydroxyl and thiol groups option

Select this option to allow rotation of the selected receptor hydroxyl and thiol groups during docking. When you select the option, the table is loaded with a list of all rotatable groups within the enclosing box.

Available groups table

This table lists the available rotatable groups (which are all such groups in the grid box). The group is identified by the hydrogen atom number, the chain name, residue name and residue number. You can select or deselect a group for use as a rotatable group by clicking in the Use column. When you do so, the rotatable group is marked in the Workspace, and a checked check box is displayed in the column.

Pick groups option

Pick the hydrogen atom in a hydroxyl or thiol group to select or deselect it for use. The rotatable group is marked in the Workspace, and the group is checked in the Use column of the Available groups table.

Use Existing Definitions button

Load the selection of rotatable groups to use that was defined previously for the receptor. These groups are added to the Maestro files produced by Glide that contain the receptor, so if you use one of these files, the rotatable groups in the file can be loaded with this button.
