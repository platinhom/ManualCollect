Summary

In this tab you choose an FEP protocol and specify the perturbation pathways.

Plan Calculation Tab Features

  • FEP protocol option menu
  • Reference fragment option menu
  • Autoset option
  • Pathways table
FEP protocol option menu

This option menu allows you to select a protocol for performing the FEP. There is one preset protocol, MCPRO+, which defines all the parameters for the simulation and the system setup. You can also choose User defined, and read a protocol from file, by entering the file name in the text box or navigating to the file.

Reference fragment option menu

The perturbations are defined with respect to a reference fragment, which you can choose from this option menu.

Autoset option

Choose the reference fragment that minimizes the number of perturbation steps needed.

Pathways table

Lists each ligand mutation that will be performed, in the FEP column, and for each mutation, lists the steps that are taken to transform the initial structure to the final structure. Each of these steps is performed as a separate FEP calculation, but FEP calculations that are common to several mutations are only performed once.
