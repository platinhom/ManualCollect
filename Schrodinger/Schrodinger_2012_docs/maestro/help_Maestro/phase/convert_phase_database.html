Summary

This panel can be used to convert a Phase database from the earlier format in which the database was marked by a file whose name ends in _phasedb to the current format, in which the database is a directory with the extension .phdb.

Opening the Convert Phase Database Panel

To open the Convert Phase Database panel:

  • Choose Applications → Phase → Convert Phase Database in the main window.

Using the Convert Phase Database Panel

The Convert Phase Database panel allows you to convert a database from the earlier format, in which the database was marked by a file whose name ends in _phasedb, to the current format, in which the database is a directory with the extension .phdb. The conversion is done by the phase_database application, which is run as a job under Job Control. A new database is created in the same location as the old database, without removing the old database.

To convert a database:

  1. Enter the path to the database in the Open database text box, or click Browse and navigate to the database. You must enter the name of a file that ends in _phasedb

  2. Click Convert.

    The status of the conversion is reported below the Convert button as "Status: Converting the database". The database conversion job is started, and the Monitor panel opens, showing the progress of the conversion. The Convert Phase Database panel is not available while the job is running.

  3. Choose whether to open the database or not.

    When the conversion finishes, you are prompted to open the database. If you click Open, the Manage Phase Database panel opens with the database loaded. If you click Cancel, no action is taken on the database (it remains where it was, and is not opened).

    After dismissing this dialog box, the status is reported as "Status: Conversion is complete" in the panel.

If the database conversion fails, you will need to determine why it failed, which you can do by examining the log file in the Monitor panel. If the source database was incomplete for some reason, the database cannot be converted, and you should consider removing the newly created database. If the conversion failed because of resource issues, you can restart the database conversion from the command line.

Convert Phase Database Panel Features

  • Open database text box and Browse button
  • Convert button
  • Status area
  • Reset button
Open database text box and Browse button

Enter the path to the Phase database in the old format, or click Browse and navigate to the database. You must enter the name of a file that ends in _phasedb.

Convert button

Click this button to start the job that converts the chosen database.

Status area

This area displays the status of the database conversion.

Reset button

Clear the panel data and reset it to its initial state.
