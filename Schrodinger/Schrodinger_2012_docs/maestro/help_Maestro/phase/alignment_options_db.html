Phase — Alignment Options Dialog Box

In this dialog box you can align ligands that are not used in the model for the hypothesis; and choose to display all aligned ligands, or only those ligands that match selected site points. The "non-model" ligands are those that are not in the active set, or those that are in the active set but do not match all sites in the hypothesis.

To align the ligands, select Align non-model ligands. Ligands that are in the model are aligned already.

To display only the non-model ligands that match certain features, select the features under Must match. If you set feature-matching tolerances in the Score Actives dialog box, these are applied when matching the non-model ligands. The set of features that must match is called a site mask.

To open the Alignment Options dialog box:

Click Alignment Options in the Score Hypotheses step of the Develop Common Pharmacophore Hypotheses panel.
