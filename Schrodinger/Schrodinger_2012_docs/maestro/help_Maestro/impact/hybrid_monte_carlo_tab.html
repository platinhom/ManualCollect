Summary

Use this tab to set options for Hybrid Monte Carlo calculations in Impact.

Hybrid MC Tab Features

This tab includes four options:

  • Number of HMC cycles text box
  • MD steps per HMC cycle text box
  • Time step (ps) text box
  • Target temperature (K) text box
Number of HMC cycles

This text box sets the number of HMC cycles for the simulation. The default value is 100; the acceptable range is any number greater than zero.

MD steps per HMC cycle

This text box sets the number of MD steps per HMC cycle for the simulation. The default value is 4; the acceptable range is any value greater than zero. Liaison calculations use 5 MD steps per HMC cycle.

Time step (ps)

This text box sets the MD time step (in picoseconds) for the simulation. The default value is 0.001 ps; the acceptable range is any value greater than zero. Because energy conservation is less important in HMC simulations, a time step of 0.002 ps or greater may be suitable.

Target temperature (K)

This text box sets the target temperature (in Kelvin) of the HMC simulation. The default target temperature, like the default initial temperature, is 298.15 K. Any target temperature greater than zero may be used.
