Summary

In this dialog box, you set the target temperature for each replica in the replica exchange MD simulation, and select the temperature of interest.

Opening the Temperature Settings Dialog Box

To open the Temperature Settings dialog box, click Temperature Settings in the Replica Exchange tab of the Impact Dynamics panel.

Temperature Settings Dialog Box Features

  • Temperature text boxes
  • Temperature of interest radio buttons
  • Auto Calculate button
  • Clear All button
Temperature text boxes

Enter the temperature for each replica in these text boxes. The temperatures must be entered in increasing order. The default value for the first temperature is 300 K.

Temperature of interest buttons

Select the temperature of interest. Only one temperature can be selected. This is the temperature of the replica whose trajectory will be incorporated into Maestro at the end of the job

Auto Calculate button

Calculate the temperatures as a geometric series. This button becomes available when the first two temperatures have been entered. These temperatures define the ratio that is used to generate all the other temperatures.

Clear All button

Clear all the temperature text boxes.
