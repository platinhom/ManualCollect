Summary

The Soak application surrounds a structure with a box of solvent molecules. The user specifies the box size and the requested solvent density. Then Impact adds the solvent molecules, removes any that are too close to the solute, and writes a Maestro-format file for the soaked system.

Opening the Soak Panel

To open the Soak panel, choose Soak from the Impact submenu of the Applications menu.

Using the Soak Panel

To model active sites or water shells around proteins with explicit solvent, use Soak followed by a short minimization and constant temperature MD equilibration at room temperature.

To set up a Soak calculation:

  1. Select the structure in the Project Table or include it in the Workspace.
  2. Open the Soak panel.
  3. Choose the source of structures.
  4. Choose the solvent and enter the solvent box size.
  5. Click Start
  6. Choose the job options, and click Start

By default, Soak places 216 water molecules in the smallest permitted solvent box (at least 18.65 Å in each dimension.) The resulting box is not equilibrated due to edge effects. A short minimization is usually sufficient to obtain a fully equilibrated solvated system. It is recommended that constant temperature molecular dynamics, described in the Impact Dynamics Panel and Impact Dynamics Tab topics, be used for this and any other explicit solvent systems.

The output file is jobname_out.mae for the soaked system. The soaked structure can be appended to the Project Table by choosing the appropriate incorporation option in the Start dialog box.

Soak Panel Features

The Soak panel has features common to Impact application panels, such as the Use structures from option menu and the Start, Write, Close, and Help buttons at the base of the panel. For information on these common features, see the Impact Panels Overview topic. The Soak panel also includes a single tab, labeled Soak, with the following features:

  • Solvent type option menu
  • Box [X,Y,Z] dimension text boxes

Solvent type

The solvent types available are SPC water (Simple Point Charge), TIP4P water, and TIP3P water.

Box [X,Y,Z] dimensions

These three text boxes enable you to specify the required size of the solvent box in angstroms. The minimum (and default) size for any dimension that Maestro will use is 18.65 Å. These values will also update the size of the simulation box when periodic boundary conditions are applied (see the Periodic Boundary Conditions topic).
