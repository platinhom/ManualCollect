Summary

In this tab, you select the level of theory for the calculation—SCF, DFT, GVB, LMP2—and select options related to the level of theory. The controls displayed in this tab depend on the level of theory selected.

For information about the controls at the top and bottom of the panel, see the Jaguar Panel topic.

Theory Tab Features

The controls displayed in the Theory tab depend on the choice made in the Level of theory option menu. Some of these controls are common to more than one level of theory. All of them are described below, with an indication of which levels of theory they apply to.

  • Level of theory option menu
  • Spin unrestricted option
  • Grid density option menu
  • Functional options
  • Excited state controls
  • Core localization method option menu
  • Valence localization method option menu
  • Resonance option menu
  • LMP2 Pairs options
  • GVB Pairs options

Level of Theory Option Menu

This option menu provides a selection of the available SCF and correlation treatments: Hartree-Fock (HF), density functional theory (DFT), local second-order Møller-Plesset perturbation theory (LMP2), generalized valence bond theory (GVB), and local second-order Møller-Plesset theory for a GVB wave function. The controls in the rest of the tab depend on the choice you make from this menu.

If you read an input file that contained gvb or lmp2 sections, the level of theory is set to Other. You can edit the gvb or lmp2 sections in the Edit Job dialog box. If you choose another level of theory from this menu, the old gvb and lmp2 sections are removed from the input file created by Maestro.

Spin Unrestricted Option

To run a spin-unrestricted (UHF or UDFT) calculation for an open-shell system, select this option (sets iuhf=1 in the gen section). Otherwise, a spin-restricted (ROHF or RODFT) calculation is performed. This option is only displayed if you choose HF (Hartree-Fock) or DFT (Density functional theory) from the Level of theory option menu.

Grid Density Option Menu

Choose the grid density for the DFT calculation, from Medium, Fine, or Maximum. If you read an input file that contained some other grid density, the grid density is set to Other, otherwise this option is unavailable. If you choose a grid density from this menu, the previous grid density specification is replaced. This option menu is only displayed if you choose DFT (Density functional theory) from the Level of theory option menu.

Functional Options

The density functional can be chosen from a list or read from the input file. If you read an input file that contained a density functional selection, the Specified in input file option is selected by default; otherwise the Selected option is selected, so that you can choose a functional from the list.

There are four different kinds of functionals. You can choose which kind to display in the list from the Show option menu, and then you can choose the functional that you want to use from the list. For more information on the functionals themselves, see the Jaguar User Manual. The functional that you choose is displayed in parentheses next to the Selected option.

This section is only displayed if you choose DFT (Density functional theory) from the Level of theory option menu. Making a choice from one of the option menus sets the dftname keyword in the gen section.

Excited State Controls

Select the Excited state option to perform an excited state calculation using either the configuration interaction singles (CIS) method, if the level of theory is HF, or the time-dependent density-functional method (TDDFT), if the level of theory is DFT. (Note that the option label includes (CIS) or (TDDFT) to distinguish the two levels of theory.) This option sets icis=1 in the gen section. Then specify the number of excited states in the Number of excited states text box (sets nroot). You can also specify the maximum number of iterations of the diagonalization procedure in the Maximum CIS iterations or Maximum TDDFT iterations text box (sets maxciit). These two text boxes are only available when you select the Excited state option.

These controls are only displayed if you choose HF (Hartree-Fock) or DFT (Density Functional Theory) from the Level of theory option menu.

Core Localization Method Option Menu

Choose the localization method for the core orbitals from this option menu.Sets loclmp2c in the gen section. This menu is only displayed if you choose LMP2 (Local MP2) or GVB-LMP2 from the Level of theory option menu. The options are:

None
Do not localize core orbitals (loclmp2c=0). This is the default.
Pipek-Mezey
Perform Pipek-Mezey localization, maximizing Mulliken atomic populations (loclmp2c=2).
Boys
Perform Boys localization (loclmp2c=1).
Pipek-Mezey (alt)
Perform Pipek-Mezey localization, maximizing Mulliken basis function populations (loclmp2c=3).

Valence Localization Method Option Menu

Choose the localization method for the valence orbitals from this option menu. Sets loclmp2v in the gen section. This menu is only displayed if you choose LMP2 (Local MP2) or GVB-LMP2 from the Level of theory option menu. The options are

Pipek-Mezey
Perform Pipek-Mezey localization, maximizing Mulliken atomic populations (loclmp2v=2). This is the default.
Boys
Perform Boys localization (loclmp2v=1).
Pipek-Mezey (alt)
Perform Pipek-Mezey localization, maximizing Mulliken basis function populations (loclmp2v=3).

Resonance Option Menu

Choose the handling of resonance structures for aromatic molecules and molecules with conjugated double bonds. Sets ireson in the gen section. This menu is only displayed if you choose LMP2 (Local MP2) or GVB-LMP2 from the Level of theory option menu. The options are:

None
Do not delocalize LMP2 pairs over other atoms (ireson=0).
Partial delocalization
Delocalize LMP2 pairs over neighboring atoms in aromatic rings (ireson=1).
Full delocalization.
Delocalize LMP2 pairs over all atoms in aromatic rings (ireson=2).

LMP2 Pairs Options

Select the types of LMP2 pairs to include in the calculation. Sets iheter in the gen section. This menu is only displayed if you choose LMP2 (Local MP2) from the Level of theory option menu. (For GVB-LMP2, the atom pairs specified for the GVB calculation are used.) The options are:

All atom pairs
Treat all atom pairs with LMP2 (iheter=0). This is the default.
Hetero atom pairs
Treat atom pairs in which the two atoms are different (except CH pairs) with LMP2 (iheter=1).

GVB Pairs Options

Select the types of GVB pairs to include in the calculation. Sets igvball in the gen section. This menu is only displayed if you choose GVB from the Level of theory option menu. The options are:

All atom pairs
Treat all atom pairs with LMP2 (igvball=1). This is the default.
Hetero atom pairs
Treat atom pairs in which the two atoms are different (except CH pairs) with LMP2 (igvball=2).
