Overview

The Jaguar pKa prediction module calculates the pKa (or pKb) of molecules that contain certain acidic or basic functional groups. The calculations involve geometry optimization of the ionic and neutral species, single-point energy and frequency calculations, and an empirical correction. The accuracy (RMSD) of the prediction is in the range 0.2-0.5 pKa units for bases, and 0.2-1.0 pKa units for acids, depending on the functional group. The functional groups recognized by Jaguar are:

Acids Bases
Alcohols Amines
Enols Anilines
Phenols Amidines
Carboxylic acids Heterocycles
Thiols Benzodiazepines
Sulfonamides Guanidines
Hydroxamic acids Pyrroles (C-2 protonation)
Imides Indoles (C-3 protonation)
Barbituric acids  
Tetrazoles  

If you want to calculate pKa values without the empirical correction, or for functional groups that don't have empirical corrections, you must add ipkaraw=1 to the gen section of the input file. The errors in the raw pKa calculations are much higher.

Running pKa Calculations

To run a pKa calculation, choose Tasks → Small Molecule Tasks → pKa Prediction → Quantum Mechanical. The Jaguar panel for pKa prediction has only the Molecule, SCF, and Solvent tabs, because all other input is predefined.

The structures that you select for the pKa calculation must be the neutral form of the acid or the base. Jaguar does not calculate pKa values for ions, even though one of the calculations must be performed on an ion.

In addition to selecting structures, you must designate an atom in the molecule for the acidic site. This atom must be the hydrogen atom in an acid, or the basic atom in a base. For a single molecule, you can enter the atom label in the Use this atom text box, or select Pick and pick the atom in the Workspace. For multiple molecules, you should select the Use pKa atoms from Project Table option. If pKa atom is not listed as a property in the project table, select Pick to add atoms to Project Table, then include the entries you want to use as input in the Workspace one-by-one and pick the pKa atom.

If you want to perform a conformational search on the protonated and the deprotonated species with MacroModel before the pKa calculation is performed, select Perform conformational searches on input structures. The lowest-energy conformer is used for each species. This option requires a MacroModel license.

If you want to calculate pKa values in DMSO (e.g. for carbon acids), you can choose DMSO from the Solvent option menu in the Solvent tab.

Usually it is not necessary to change any of the SCF convergence settings, but if you want to, you can do so in the SCF tab.

When you have selected the structures and pKa atoms, and made any settings, click Start.
