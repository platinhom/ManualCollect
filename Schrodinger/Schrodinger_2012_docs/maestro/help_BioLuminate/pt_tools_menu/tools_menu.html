Project Table — Tools Menu

This menu provides tools to use in the Project Table.

Find/Replace   Show or hide the Find toolbar to find and replace text in any table cell except the row number. This toolbar can also be displayed from the Project toolbar.
Sort   Sort project entries or entry groups. Opens the Sort Project Table panel to sort entries by property values. This panel can also be opened from the Project toolbar.
Plot Properties   Open the Manage Plots panel to plot property data for entries. This panel can also be opened from the Project toolbar.
2D Viewer   Open the 2D Viewer panel to display 2D structures for project entries in a grid. This panel can also be opened from the Property toolbar.
View Poses   Sets up group containing a receptor as the first entry and a set of ligands as the remaining entries to step through the ligands with the receptor fixed. Hydrogen bonds and contacts can be displayed if selected. The submenu offers the following choices:
  • Setup—Fix receptor, include first ligand, zoom to the ligand, and add Mark property for marking entries.
  • Export Rotatable Groups—Export the receptor and optionally poses for which rotatable groups in the receptor adopt a particular orientation.
  • Count H-Bonds—counts hydrogen bonds between ligands and the receptor and adds count as a property to the Project Table.
  • Display H-Bonds—selects display of hydrogen bonds between ligands and the receptor.
  • Define H-Bonds—opens the H-Bonds and Contacts panel at the H-bonds tab so that H-bond settings can be made.
  • Count Contacts—counts contacts between ligands and the receptor and adds counts as properties to the Project Table.
  • Display Contacts—selects display of bad and ugly contacts between ligands and the receptor.
  • Define Contacts—opens the H-Bonds and Contacts panel at the Contacts tab so that contact settings can be made.
  • Display Per-Residue Interactions—selects display of interactions between the ligand and selected receptor residues.
  • Define Per-Residue Interactions—opens the Per-Residue Interactions panel so that you can specify the interaction types and select the residues.
  • Help—Display the Viewing Poses help topic.

You can use LEFT ARROW and RIGHT ARROW to step through the ligand entries. You can also mark poses by pressing the M key. This action sets a Mark property in the Project Table. To clear the mark, press M again. You can use the Mark property to sort the poses, for example.

Record Movie   Record an MPEG movie of the selected entries in the Project Table. Opens the Record Movie dialog box, in which you can specify the file name or browse to the location, and set options for resolution and speed in an Options dialog box. The preference for fitting to screen when the Workspace changes is honored when the movie is recorded, so it is possible to have a blank frame in the movie.
