Creating a New Project

To create a new project, choose File → New Project or press Ctrl+N (⌘N). The New Project project selector opens. Specify the location and name of the new project, and click Create. This button is not active until you have provided a unique project name. Maestro adds a .prj extension to the project name if you did not supply one. When you click Create, the current project is closed, and a new project is created.

You can specify the location of the project in several ways:

  • Navigate to a directory using the Directories list
  • Type the location into the Project text box.
  • Create a new directory in the currently displayed directory by clicking Make Dir and supplying the new directory name in the Make Directory dialog box.

If you want to create a project with a large number of structures, you can do so from the command line with the project_create utility.
