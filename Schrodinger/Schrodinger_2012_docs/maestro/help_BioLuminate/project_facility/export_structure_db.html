Summary

This dialog box allows you to export structures from a trajectory.

Opening the Export Structure Dialog Box

To open the Export Structure dialog box:

  • Click Structure in the Trajectory panel.

Export Structure Dialog Box Features

  • Export to options
  • Structures options
Export to options

Select an option for the location of the structures, from Project Table or File. If you select File, enter the file name in the text box below the option, or click Browse to navigate to the location and name the file. Structures exported in PDB format are exported to a multi-model single PDB file.

Structures options

Select an option for the export of structures from the trajectory.

  • Selected frames (as single entries)—Export the frames that are selected in the Frame control section of the Trajectory panel. Each frame is exported as a single entry.

  • Current frame only (as a single entry)— Export the current frame as a single entry.

  • Current frame only (as multiple entries)— Export the current frame as multiple entries. How many entries and their composition depends on the simulation. For Desmond simulation, the possible entries consist of: the full system, the solute, the solvent, the membrane, ions added for neutralization, positive ions from added salt, negative ions from added salt.

Atoms to export section

Export only the atoms specified in this section from each frame. These standard atom selection controls allow you to select the atoms to export.
