Renaming a Project

There is no explicit facility inside Maestro for renaming a project. To rename a project from inside Maestro, you can save the project with a new name with Project → Save As, File → Save Project As, then delete the old project.

You can also rename a project using your file explorer, since a project is just a directory with a .prj extension. If you have created backups of the project, ensure that you also change the name of the backup directory, project.backup.

You can also use the mv command in a terminal window (or Schrodinger Command Prompt on Windows, prefixed with sh) to rename the project and any backup directory.
