Summary

The Plot Settings - One-Coordinate Scan panel is used to control the plot attributes of data sets that are displayed in the 1D Plot panel, and to delete data sets.

Opening the Plot Settings - One-Coordinate Scan Panel

To open the Plot Settings - One-Coordinate Scan panel:

  • Click Plot Settings in the Plot of One-Coordinate Scan panel.

Using the Plot Settings - One-Coordinate Scan Panel

First, select from the list the data set that you want to modify. To change options, select the desired options from the option menus. The selected data set is updated in the Plot of One-Coordinate Scan panel. To delete the currently selected data set from the Plot of One-Coordinate Scan panel, click Delete.

Plot Settings - One-Coordinate Scan Panel Features

  • Available data sets list
  • Symbol option menu
  • Symbol color option menu
  • Symbol size option menu
  • Curve option menu
  • Curve color option menu
  • Curve width option menu
  • Delete button
Available data sets list

The top portion of the panel contains a list of the datasets that are available. Only one data set in the list can be selected at any one time and the settings in the rest of the panel apply to that selection.

Symbol option menu

This option menu contains various symbols that can be used to mark data points on a 1D plot.

Symbol color option menu

This option menu allows you to select the color of the markers for the currently selected data set.

Symbol size option menu

This option menu can be used to change the size of the markers on the currently selected plot. To change the marker size, click on the option menu bar, and then click again on the desired size. The currently selected plot will be updated to reflect the change.

Curve option menu

This option menu allows you to choose whether the currently selected plot is displayed using a solid line, a dashed line, or no line (data point markers only).

Curve color option menu

This option menu allows you to change the line color of the currently selected plot.

Curve width option menu

This option menu allows you to vary the line thickness of the plot of the currently selected data set.

Delete button

Deletes the currently selected data set from the Plot of One-Coordinate Scan panel.
