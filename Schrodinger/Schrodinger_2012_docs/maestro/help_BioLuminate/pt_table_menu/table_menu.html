Project Table — Table Menu

The Table menu provides access to tools that operate on the contents of the table or configure the panel.

Import   Import structures or data from files. Submenu provides selection of structure, spreadsheet, vibrational data, or surfaces for import.
Export   Export structures or data to files or to Canvas. Submenu provides selection of structure or spreadsheet for export format, and an item for exporting to Canvas. The submenu items open file selectors or dialog boxes to perform the actions.
Preferences   >Open the Preferences panel at the Project Table - Properties tab, so you can set project-related preferences.
Customize   Customize the menus or toolbars. Submenu provides selection of Menus or Toolbars to open the Customize Menus panel or Customize Toolbars panel. Note: if you customize menus or toolbars, some of the information about them in the help topics or manuals will no longer apply.
Toolbars   Show or hide the toolbars. Submenu provides selection of the toolbars to show. See the topic Project Table Toolbars for information on the toolbars.
