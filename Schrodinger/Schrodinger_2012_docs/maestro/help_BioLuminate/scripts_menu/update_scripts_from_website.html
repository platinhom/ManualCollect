Summary

In this panel, you can download scripts from the Scripts Center on the Schrödinger web site.

Opening the Update Scripts From Website Panel

To open the Update Scripts From Website panel:

  • Choose Tasks → Scripts → Update.

    • Update Scripts From Website Panel Features

    • Location section
    • Filter section
    • Scripts table
    • Status display
    • Install button

    Location section

    In this section you can choose the location for the scripts, from one of two locations that are searched by Maestro. The two choices are:

    • User Area—Install scripts for your own use in your home directory. There are separate locations for Maestro scripts and for command-line scripts, which are listed below the Install scripts in options.

    • Common Area—Install scripts for the use of all users in the current installation ($SCHRODINGER). There are separate locations for Maestro scripts and for command-line scripts, which are listed below the Install scripts in options. You must have permission to write to these directories to install scripts in them.

    Filter section

    In this section you can filter the scripts that are shown in the scripts table.

    Show options

    Select an option to filter what is listed in the table. The options are:

    • All available scripts—Show all available scripts of the chosen type in the chosen category.
    • Only scripts updated or added within N weeks—Show scripts of the chosen type in the chosen category that have been updated or added within a specified number of weeks. Enter the number of weeks in the text box.
    • New versions of installed scripts—Show only new versions of scripts of the chosen type in the chosen category that are already installed in the selected location.
    • Scripts not currently installed—Show only scripts of the chosen type in the chosen category that are not currently installed in the selected location.
    Category option menu

    Choose a script category from this option menu to display only scripts from that category. To display all scripts, choose All.

    Type option menu

    Choose a script type from this option menu, or choose All to show all scripts. There are three types:

    • Command-line—Scripts that are to be used only from the command line. Can include shell scripts and Python scripts. These scripts will be installed in the command-line location given in the Location section.
    • Maestro Command—Scripts consisting of Maestro commands. These scripts will be installed in the Maestro location given in the Location section.
    • Maestro/Python—Python scripts for use from Maestro, including Python panels. These scripts will be installed in the Maestro location given in the Location section.

    Scripts table

    This table lists the available scripts that satisfy your filtering choices. For each script, the title, script name, category, and last update date are given. When you pause the cursor over the title, a description of the script is displayed in a tooltip.

    You can select multiple scripts using shift-click and control-click. The scripts you select are the ones that will be installed when you click Install.

    Status display

    This line, below the scripts table, shows the number of scripts displayed out of the total number available.

    Install button

    Click this button to install the scripts that are selected in the table. The scripts are downloaded from the Schrödinger web site. If you are not logged on, you will be prompted for your logon ID (email address) and password. Once you are logged on, the scripts will be downloaded and installed.
