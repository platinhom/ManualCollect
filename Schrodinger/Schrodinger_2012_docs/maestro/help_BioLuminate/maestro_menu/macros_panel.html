Summary

The Macros panel allows you to define macros that execute a series of commands in Maestro. These macros can then be called from the command input area or used in scripts.

Opening the Macro Panel

To open the Macros panel:

  • Choose Edit → Settings → Aliases and Macros → Macros.

Using the Macro Panel

The following example is a macro that undisplays side chains, hydrogens, carbonyl oxygen of the backbone, and waters. It colors the atoms by entry, which is useful for looking at multiple aligned proteins. It also installs the macro to the Scripts menu in the main window for easy future access.

  1. Choose Maestro g→ Macros.

  2. Click New to create a new macro and name it backbone.

  3. Enter the following commands in the text box:

    undisplayatom (sidechain) OR (atom.ptype " O ")
undisplayatom atom.ele H
undisplayatom water
colorscheme scheme=Entry all
fit
enhance3d

  4. Optional: Click Install to install the macro to the Scripts menu of the main window. For more information on working with the Scripts menu, see the topics Install Scripts panel and Manage Scripts panel.

  5. To run the macro, either choose backbone from the Scripts menu, or enter the following command in the command input area:

    macrorun backbone

Macro Panel Features

  • Macros List
  • Name text box
  • New button
  • Delete button
  • Write button
  • Install button
  • Definition text area
Macros List

This list displays the defined macros. Click on a macro to edit, delete or rename it.

Name text box

This text box allows you to rename the selected macro.

New button

Click New to create and name a new macro.

Delete button

Click Delete to delete the selected macro.

Write button

Click Write to export the selected macro as a .cmd file.

Install button

Click Install to install the macro on the Scripts menu of the main window. For more information on working with the Scripts menu, see Install Scripts panel and Manage Scripts panel.

Definition text area

This text area displays the definition of the currently-selected macro. To make changes to a macro, edit the definition.
