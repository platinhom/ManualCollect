Summary

The Display/Undisplay Atoms Panel is used to display or hide atoms as well as the associated labels and markers.

Opening the Display/Undisplay Atoms Panel

To open the Display/Undisplay Atoms panel:

  • Choose Edit → Display/Undisplay Atoms in the main window.

This panel is dockable, so it opens by default in the Workspace. You can change this behavior in the Preferences panel, under General – Panels.

Using the Display/Undisplay Atoms Panel

To display or undisplay all atoms:

  • Click All in any of the standard picking controls sections.

To display or undisplay atoms by picking:

  1. Select Pick in any of the three sections.

  2. Choose a structural unit from the corresponding option menu.

  3. Pick atoms in the Workspace.

    If you selected Pick in the Display only section, only the picked atoms are displayed, and all other atoms are undisplayed.

    If you selected Pick in the Undisplay section, the picked atoms are undisplayed.

To display or undisplay atoms of specific groups:

  1. Click the Atom Selection button.

  2. Choose the atom group from the list.

Displaying/Undisplaying Atoms Using Text Commands

The atoms can also also be displayed or undisplayed using text commands. The syntax is as follows:

displayatom ASL
undisplayatom ASL

Examples:

displayatom atom.mtype water, 

undisplayatom atom.ele H,  

displayatom mol.num1 and res.num5

Atom Labels When Displaying/Undisplaying Atoms

If an atom has an atom label, the label can be displayed or undisplayed together with the atom (the default), or it can be always displayed, by selecting Show label when atom hides in the Atom labels tab of the Preferences panel.
