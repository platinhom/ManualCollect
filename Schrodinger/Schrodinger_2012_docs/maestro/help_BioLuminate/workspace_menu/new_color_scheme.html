Summary

In this dialog box you can set up a new color scheme based on an atom-level property.

Opening the New Color Scheme Dialog Box

To open the New Color Scheme dialog box:

  • Click New in the Color Scheme tab of the Atom and Bond Coloring panel.

Using the New Color Scheme Dialog Box

New color schemes can be created from atom properties, i.e. properties that have a separate value for each atom. Before you set up a new scheme, you should include in the Workspace a structure that has the desired atom property. The range of values of the property in that structure should be representative of the range in any structure that you might want to color with the new scheme.

To set up the scheme:

  1. Choose a property from the Atom property list.

  2. Click Update to obtain the property range, and fill in the Min and Max text boxes with the range values.

  3. Edit the Min and Max text boxes to set the minimum and maximum values that are actually used for the color scheme.

  4. Enter the number of colors that you want to use. The range is partitioned equally.

  5. Choose a color ramp to define the colors. The colors are assigned from the available colors in the color ramp.

New Color Scheme Dialog Box Features

  • Name text box
  • Atom Properties list
  • Update button
  • Property range
  • Min and Max text boxes
  • Number of colors text box
  • Color ramp text box
Name text box

Name of the new color scheme. This name is shown on the list in the Color Schemes panel, the Atom Color tab of the Atom and Bond Coloring panel, on the Color Scheme button menu on the Representation toolbar, and on the Selection shortcut menu.

Atom Properties list

List of atom-level properties. You can select one property from this list as a basis for the color scheme.

Update button

Obtain the property range from the atoms in the Workspace, and update the Property range indicator and the Min and Max text boxes.

Property range

This text displays the range of the property value in the Workspace, after you click Update.

Min and Max text boxes

Minimum and maximum values of the property that are used to define the color scheme. These text boxes are populated initially from the Workspace. You can edit the values to define the range that is actually used.

Number of colors text box

Specify the number of colors to use in the scheme. The property range between the minimum and maximum values is partitioned equally, and a color assigned to property values in each partition.

Color ramp text box

Choose the color ramp from which the colors are assigned. Each ramp has a fixed number of colors from which the colors are assigned. The ramps are defined in the ramps.res resource file.
