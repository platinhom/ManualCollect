Choose Color Dialog Box

This dialog box consists of two tabs labeled By Palette and By Name. Each tab displays the currently selected color in a rectangular bar at the top of the tab. When a color selector is opened, the By Palette tab is displayed by default.

In the By Palette tab, all the available colors are arranged in a rectangular grid of color blocks. To select a color, click the desired color. The Atom color bar changes to the color you selected. The RGB values for the color blocks are displayed in a tooltip.

In the By Name tab, the colors are listed by name in a scrolling list. To select a color, click on the desired list item. The Atom color bar changes to the color you selected.

If you want to change a color, select the color (in either tab), and click Edit. A standard color chooser opens, in which you can change the color. When you click OK, the new color is saved immediately in the colors.res file in your Maestro user resources directory. To revert to the default colors, click Restore Defaults, which removes this file.
