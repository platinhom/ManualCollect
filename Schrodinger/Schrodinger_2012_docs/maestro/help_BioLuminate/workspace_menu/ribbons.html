Summary

The Ribbons panel provides options for visualizing proteins, DNA, and RNA structures, using representations such as ribbons, ladders, and alpha carbon traces.

Opening the Ribbons Panel

To open the Ribbons panel:

  • Choose Workspace → Ribbons in the main window.

This panel is dockable, so it opens by default in the Workspace. You can change this behavior in the Preferences panel, under General – Panels.

Using the Ribbons Panel

If ribbons are not displayed for parts or all of a protein, the problem might be that the secondary structures have not been assigned.

Ribbons Panel Features

Style option menu

This option menu provides a choice of seven display styles: Cartoon, Ribbon, Tube, Thin Tube, Curved Line, CA Trace, and CA Trace Tube

Color scheme option menu

This option menu provides a range of properties that can be used to select the color scheme, such as residue properties and secondary structure. Each of these has a predefined set of colors that is applied to the ribbon. The color schemes are explained in the Color Schemes panel, which you open from the Atom and Bond Coloring panel.

Two of the color schemes have additional settings, which are displayed when you choose these schemes:

Residue position limits options and text boxes

These options allow you to set residue number limits for applying the Residue position color scheme, and are displayed when you choose that scheme from the Color scheme option menu. Residues with residue numbers outside the range are colored with the color for the end of the range. Insertion codes are ignored.

  • Lower limit— Select this option to set the lower limit for coloring, and enter the lowest residue number in the text box.
  • Upper limit— Select this option to set the upper limit for coloring, and enter the highest residue number in the text box.

Constant color option menu

Choose the color to use for coloring the ribbons. This option menu is displayed if you choose Constant color from the Color scheme option menu.

Apply ribbons to atoms picking tools

Use the picking controls in this section to apply ribbons to an atom selection.

Update All Ribbons button

Change the representation of all existing ribbons to reflect the current choice of style and color. By default, the new style and color is applied only to ribbons created after the change.

Delete All Ribbons button

Delete all ribbons from the Workspace.

Preferences button

Set preferences for the display of ribbons: ribbon quality, width of ribbons, strands, tubes and ladders, thickness of ribbons and strands, blending of colors, and atoms that are hidden. Opens the Preferences panel to the Ribbons settings group, in which the settings can be made. You can also restore the default values in this panel.
