View Menu

The View menu allows you to manage views of the Workspace and to control various aspects of the view of the structures, including stereo and 3D effects, clipping, and tiling. Some of these items are available on the View toolbar.

  • Reset View— Resets the rotation, translation, and zoom of the Workspace to the default state, which has the screen axes along the coordinate axes of the Workspace structures. You can also do this with the Reset button on the View toolbar:

  • Fit to Workspace—Fit a group of atoms to the size of the Workspace. The submenu has items for fitting all atoms in the Workspace, the selected atoms, or the ligand to the size of the Workspace. See Fit to Workspace for more information.

  • Transform Mode—Select a transformation mode, to which the normal mouse actions (left button, by default) apply, rather than picking. The previous state is remembered when you exit the mode, either from the menu, with the Esc key or by selecting another action that involves picking. The submenu has four items:
    None (Picking)—Default mode in which the mouse is used for picking.
    Rotation—Use the mouse for rotation.
    Translation—Use the mouse for translation.
    Zoom—Use the mouse for zooming.

  • Rotate—Rotate the structures in the Workspace. This submenu has items for rotating around the x (horizontal) axis and the y (vertical) axis by 90 degrees (also on the View toolbar), and items for rotating about all three axes by an increment that is set in the Preferences panel.

  • Translate—Translate the structure parallel to one of the axes. The submen has items for translating in either direction parallel to any of the three axes by an increment that is set in the Preferences panel.

  • Zoom In—Zoom in by a small increment.

  • Zoom Out—Zoom out by a small increment.

  • Move Clipping Planes—Adjust the clipping planes, where the view of the structure is truncated. This submenu has the following items:

    • Back—Move clipping planes away from the viewer.
    • Forward—Move clipping planes toward the viewer.
    • Together—Move front plane back and back plane forward (decreases the viewing volume.
    • Apart—Move front plane forward and back plane back (increases the viewing volume).
    • To All Atoms—Move clipping planes to show all atoms.
    • To Selected Atoms—Move clipping planes to show the selected atoms.
    • To Ligand—Move clipping planes to show the Workspace ligands, as defined by the ligand detection settings.

    The last three items use the fitting margin defined in the Preferences panel.

    See Clipping Planes Window for more information.

  • Stereo—Turn stereo viewing on or off. See the Stereo tab in the 3D tab of the Appearance panel for more information.

  • Toolbars—Submenu to manage display of toolbars. The two items are:

    • Select Toolbars Displayed—Select the toolbars that are displayed. You can also do this with the Manager toolbar, if it is displayed. The toolbars are
      Find
      Project
      Edit
      Workspace
      View
      Style
      Saved Views
      Display Atoms
      Representation
      Labels
      Build
      Fragments
      PrimeX
      Favorites
      Manager

    • Select Buttons for each Toolbar—Opens the Customize Toolbars dialog box, in which you can choose the buttons that are displayed on each toolbar.

Advanced View Options

This submenu contains the less commonly used view options.

  • Align—Align the Workspace structure so that selected atoms lie on one of the Cartesian axes or planes, in the Align Atoms panel.

  • Tile—Arrange the Workspace entries in a rectangular grid of "tiles". See Tiling Entries for more information. You can also do this with the Tile button on the View toolbar:

  • Perspective—Turn perspective on or off. See the the Perspective tab in the 3D tab of the Appearance panel for more information.

  • Enhanced Depth View—Enhance the perception of depth (with ambient spherical occlusion). This option is most useful for large structures.

  • Display Dipole Moment— Display the dipole moment of the Workspace structure, if one is available. The dipole moment is displayed as an arrow with the base at the geometric centroid of the molecule. Dipole moments from Jaguar are displayed in mauve; dipole moments from MM charges are displayed in yellow-green. Not supported for scratch entries.

  • Show Alternate Positions— For protein structures with partial occupancies, display the atoms for all occupancies. The alternate positions are displayed as ghost atoms.

  • Fog Options—Enable or disable fogging, and enhance depth cues. This submenu has the items Enhance Depth Cues, Settings, and Automatic, On, or Off for applying fogging. Automatic fogging turns fogging on when the molecule is larger than 40 atoms. See the Fog tab of the Appearance panel for more information. You can also do this with the Fog button on the View toolbar:

  • Show Crystal Mates— Display atoms that are related to the Workspace structure by crystallographic symmetry. The display can be limited to residues within a given distance of the Workspace structure. The Crystal Mates submenu offers four distances, in addition to All (display all atoms within the unit cell) and None (undisplay crystal mates).

    The atoms in the crystal mates are included in the Workspace as scratch entries. If you want to make use of them for a calculation, you can create an entry from the Workspace, and it will include the crystal mate atoms.

    The chain names for the crystal mates can be made unique by setting a preference in the Preferences panel.

  • Workspace Appearance— Set options for the Workspace appearance: background color, lighting, and materals effects, in the Appearance panel.
