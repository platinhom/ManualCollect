Window Menu

The Window menu allows you to display windows, manage the open panels and save or load panel layouts.

  • Close All—Close all open windows.

  • Raise All—Raise all open panels.

  • Full Screen—Enter full screen mode, in which the Workspace occupies the entire screen.

  • Advanced Options—This submenu has some less used options for managing windows and opening scripting windows.

    • Reopen—Reopen all panels that have been opened in the Maestro session and closed.

    • Cascade—Place panels in a cascade from the top left towards the bottom right of the screen, in order of opening.

    • Save Layout—Save the current sizes and locations of the panels that have been opened in the current Maestro session.

    • Restore Layout—Restore the panels to the locations and sizes that were last saved.

    • Python Shell—Open a panel that can be used to run Python commands.

    • Command Script Editor—Edit and replay a command script.

  • Show—Submenu for showing various common panels.

    • Project Table—Open the Project Table panel.

    • Project Table Summary—View and add summary information for the project. Opens the Project Summary panel, in which you can see information on project entries, runs, and jobs, and view, add, or delete annotations.

    • Job Monitor—Monitor jobs that you have started, in the Monitor panel.

    • Surface Manager—Open the Manage Surfaces panel.

    • Toggle Table—Open the Toggle Table panel, which provides PyMOL-like interaction with the Workspace.

  • Panel names—Raise the named panel to the top.
