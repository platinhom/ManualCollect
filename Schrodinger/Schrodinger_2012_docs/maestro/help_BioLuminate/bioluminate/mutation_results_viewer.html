Summary

This panel displays the results of residue mutations that are made either in the Residue Scanning panel or the Humanize Antibody panel. You can examine changes in various properties, both in a table and in a graph, and you can view mutations in the Workspace.

Opening the Residue Scanning/Humanize Antibody Viewer Panel

To open the Residue Scanning Viewer panel:

  • Choose Tasks → Residue Scanning → View Results in the main window.

To open the Humanize Antibody Viewer panel:

  • Choose Tasks → Antibody Modeling → Humanization Results in the main window.

The panel opens automatically when a Residue Scanning or Humanize Antibody job incorporates, with the results loaded.

Residue Scanning/Humanize Antibody Viewer Panel Features

  • Import button
  • Mutations table
  • Display original structure in gray option
  • Graph property option menu
  • Plot area
  • Export button
  • Reset button
Import button

Import the structure results for the mutation job. Opens a file selector in which you can navigate to and select the results file, jobname-out.maegz. When a Residue Scanning job incorporates, the panel opens automatically with the results displayed.

Mutations table

This table lists the residues that were mutated, with the identity of the orginal residue and of the mutated residue for each mutation that was performed. It also shows the change in the following properties for each residue:

  • SASA (total, polar, and nonpolar)
  • pKa
  • Affinity
  • Hydropathy
  • Total rotatable bonds
  • Stability (in solution and in gas phase)

The table can be sorted by clicking on the column headings.

You can select a row in the table, and the Workspace view zooms in to the corresponding residue.

Display original structure in gray option

Display the original structure in gray in the Workspace as well as the mutated structure. This allows you to compare mutations with their originals.

Graph property option menu

Choose the property to graph in the plot area below. The properties are those displayed in the Mutations table.

Plot area

This area displays a plot of the property chosen from the Graph property option menu as a function of the row number in the table (residue position). The plot has a toolbar and a display area. You can select residues in the Workspace and the table by moving the dotted lines to restrict the range of rows and property values.

The toolbar provides tools for manipulation of the plot and for saving an image of the plot. This is a generic toolbar, and some of the actions may not be useful in the current context. The toolbar buttons are described below.

Reset
Reset the plot to the original pan and zoom settings.
Back
Display the previous view of the plot in the view history.
Forward
Display the next view of the plot in the view history.
Pan/zoom
Pan the plot with the left mouse button, zoom with the right button.
Zoom to rectangle
Drag out a rectangle on the plot to zoom in to that rectangle.
Configure subplots
Configure the margins and spacing of each plot in the panel.
Edit axis and curve parameters
Make settings for the title, range, labeling, and scale of the axes; the color, style, and width of lines; and the color, style, and size of markers.
Save image
Save an image of the plot to file. Opens a file selector in which you can browse to a location, select the image format, and name the image.
Export button

Export the contents of the Mutations table to a CSV file. Opens a file selector, in which you can navigate to a location and name the file.

Reset button

Delete all data from the panel.
