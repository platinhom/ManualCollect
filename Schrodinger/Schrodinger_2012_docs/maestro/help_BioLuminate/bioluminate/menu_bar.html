Main Menu Bar

The menus on the main menu bar provide access to panels, allow you to enter commands, and control the appearance and contents of the Workspace. The menu bar has the following menus by default:

  • File—change directories; perform tasks on projects such as opening and closing, importing and exporting structures, making backups; save files; and quit Maestro.
    [Display topic list]

  • Edit—undo actions, select atoms, build and modify structures, set preferences, customize menus, toolbars, mouse actions.
    [Display topic list]

  • Style—control the appearance of the contents of the Workspace.
    [Display topic list]

  • View—change the view of the Workspace (camera position and angle), set up 3D and stereo effects, show or hide toolbars.
    [Display topic list]

  • Tools —prepare proteins, show various protein analyses; align and superimpose structures; create and manage surfaces.
    [Display topic list]

  • Tasks—set up, run, and monitor jobs for Schrödinger's computational programs.
    [Display topic list]

  • Window—Open, close, and raise windows, arrange windows in a cascade, save and restore window layout.

  • Help—View various kinds of documentation, including online help, manuals, information about Maestro and other Schrödinger products; run diagnostics.
    [Display topic list]
