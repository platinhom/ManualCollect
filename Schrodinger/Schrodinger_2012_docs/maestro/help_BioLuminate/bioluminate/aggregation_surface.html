Summary

In this panel you can generate a surface that indicates regions with a propensity for aggregation.

Opening the Aggregation Surface Panel

To open the Aggregation Surface panel:

  • Choose Tasks → Aggregation Surface in the main window.

Using the Aggregation Surface Panel

Aggregation regions are defined by identifying clusters of exposed hydrophobic residues. A molecular surface is created for a large probe molecule, representing a protein. The surface is colored red in the regions near these residues, which are the regions where aggregation might occur. The side chains of the hydrophobic atoms are colored red and displayed in ball-and-stick. The alpha carbon is also colored red, and the other backbone atoms are colored gray. The backbone is displayed as lines. Residues that are in contact with the hydrophobic side chains are also displayed; all other residues are hidden. The residues involved are colored red in the Workspace sequence viewer.

Once you have located aggregation regions, you might want to mutate residues in this region to reduce the size of the region.

To create an aggregation surface:

  1. Select atoms to define the structure for the surface, if you want to limit the calculation to part of the protein.

  2. Choose an option for the part of the Workspace structure that you want to create the surface for.

  3. Enter a name in the Surface name text box, if you want to change the default name.

  4. Click Create Surface.

Because the surface color scheme depends on the atom coloring, the surface is removed if you change the atom color scheme. For a similar reason, the surfaces are removed when you close the panel. If you don't make any changes, the atom visibility and the color scheme are still in effect when you close the panel.

If you want to create a selection from the displayed atoms, you can do so by right-clicking in the Workspace (not on an atom) and choosing Visible → Select.

Once you have this selection, you can run a job to perform mutations on these residues, as follows:

  1. Choose Tasks → Residue Scanning → Perform Calculation to open the Residue Scanning panel.

  2. Select Analyze only selected Workspace residues.

  3. Click Analyze Workspace. You may be prompted to choose a chain, as only one chain at a time is mutated.

The residues that were found to contribute to the aggregation region are listed in the table in the Residues tab. You can then select any of them for mutation, define the mutations, and run the job.

Aggregation Surface Panel Features

  • Compute aggregation surface for options
  • Surface name text box
  • Options button
  • Create Surface button
  • Delete Surfaces button
Compute aggregation surface for options

Choose an option to define the structure for which the aggregation surface is computed.

  • Workspace—Compute the surface for the structure in the Workspace. You must have only a single entry in the Workspace.
  • Entire entry containing selected atoms—Compute the surface for the entry that contains the selected atoms. This is useful if you want to view how the aggregation surface on one protein matches residues in another protein.
  • Entire chain containing selected atoms—Compute the surface for the entire chains that contain the selected atoms. This is useful for computing a surface for one chain of a multi-chain protein, for example.
  • Selected atoms and attached hydrogen atoms only—Compute the surface for the selected atoms with their attached hydrogens only. This option allows you to calculate the surface for only part of a protein chain, for example.
Surface name text box

Enter the name you want to associate with the surface in this text box. This name is used in the Manage Surfaces panel to identify the surface.

Options button

Open the Aggregation Surface - Options dialog box, where you can set parameters that control the quality and appearance of the surface, and parameters for locating aggregation regions.

Create Surface button

Click this button to create the surface. A progress bar is displayed above the buttons while the surface is being created. Surface creation should take less than a minute.

When the surface is created, it is displayed in the Workspace and other surfaces are hidden. To display more than one surface, use the Manage Surfaces panel (Window → Show → Surface Manager).

Delete Surfaces button

Delete the aggregation surfaces and redisplay all atoms in the structure.
