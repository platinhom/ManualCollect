Edit Menu

The Edit menu provides access to tools for editing and adjusting the structures in the Workspace. The default menu items are listed below. Advanced editing options and settings are listed in the following sections.

  • Undo—Undo the last undoable action. The menu item changes to display the name of the last command executed. When you undo an action, the menu item changes to Redo. Undoing can also be performed with the Edit toolbar button:

  • Delete—Delete objects from the Workspace. This submenu has several sections. The first two sections have a single item, for picking objects to delete and for deleting the selected atoms. The next section lists other kinds of structural objects that can be deleted: hydrogens, waters, and zero-order bonds. The item in the next section opens the Atom Selection dialog box for defining which atoms to delete. The last section allows you to delete various kinds of markers associated with the structures in the Workspace, such as ribbons, and adjustments.

  • Display/Undisplay Atoms—Select the atoms to display or undisplay in the Workspace, in the Display/Undisplay Atoms panel.

  • Add Hydrogens—Submenu that allows you to add hydrogen atoms to complete the valence on atoms in the Workspace. Includes items to pick atoms for hydrogen addtion, select them in the Atom Selection dialog box, add hydrogens to all atoms, and open the Add Hydrogens - Advanced panel in which you can choose a force-field treatment and apply it to selected hydrogen atoms.

  • Adjust Distances and Angles—Submenu that allows you to perform adjustment by picking; adjust various defined quantities; open the Adjust panel for adjustment of distances, angles, dihedrals, and chirality; delete adjustments; or open the Rotamers dialog box to select rotamers of the selected residues.

  • 3D Builder—Build or modify structures in the Workspace, using the Build panel. This submenu has the following items:

    • Fragments—Add molecular fragments to a structure, or place them in the Workspace. Opens the Build Panel at the Fragments tab.

    • Atom Properties—Modify atom properties, such as element, atom type, partial charge. Opens the Build Panel at the Atom Properties tab.

    • Residue Properties—Modify residue properties, such as residue type and residue number. Opens the Build Panel at the Residue Properties tab.

  • Structure Sculpting

    —Mode in which you can drag atoms to a new location and the surrounding atoms are minimzed with a force field.

  • Pick Mode—Chose the structural unit to use when picking in the Workspace. This submenu allows you to select a structural unit, from Atoms, Residues, Chains, Molecules, or Entries. You can then pick these units in the Workspace to make an atom selection. You can do the same tasks with the Select button on the Edit toolbar.

  • Select Atoms—Select various classes of atoms, and save the selection as a numbered set. This submenu has three sections. The first section contains only one item, Pick to Select, which turns Workspace selection on and off, and is the same as cliking the Select toolbar button. The structural unit that is used for picking is the one chosen from the Pick Mode sumbenu. The second section contains items for selecting all atoms, inverting the selection, selecting the next or previous residue in the chain (if a single residue is selected), and for opening the Atom Selection dialog box. The third section has items for saving up to 10 atom selections, and restoring these selections.

Advanced Edit Options

This submenu contaning other options for structure editing.

  • Transform—Submenu that allows you to pick atoms to transform (rotate and translate), set display options for transformations, or open the Advanced Transformations panel, for more complex atom selections to transform and to set the center of rotation.

  • Define Centroids—Place an atom at the centroid of a set of atoms. Opens the Centroids panel.

  • Symmetrize Workspace—Impose point-group symmetry on the atoms in the Workspace. Opens the Symmetrize Workspace panel.

  • Create New Named Set—Create a named set of atoms that can be used when selecting atoms for a task. Opens the Sets panel.

  • Connect and Fuse Atoms—Combine molecules by connecting or fusing atoms. Opens the Connect and Fuse panel.

  • Assign Bond Orders—Assigns bond orders to structures that do not have the correct bond orders. See the Assign Bond Orders topic.

  • Assign Partial Charges—Assigns partial charges to the structures in the Workspace, based on the OPLS_2005 force field. See the Assign Partial Charges topic.

  • Clean Up Geometry (U)—Clean up the geometry of the Workspace structure by minimizing its energy. By default the OPLS_2005 force field is used. If this does not include the atoms you need to minimize, you can set a preference to use the UFF force field; however this force field can only be used on less than 300 atoms, and is slow for more than 200 atoms. See the Clean Up Geometry topic.

Settings

This submenu has items for settings within the GUI, including preferences, Workspace appearance, customization of menus, toolbars, mouse actions, GUI component visibility, and others.

  • Preferences— Set preferences for Maestro appearance and behavior, in the Preferences panel.

  • Toolbars—Submenu with items to select the toolbars that are displayed and to customize the toolbars in the Customize Toolbars panel.

  • Menus—Customize the menus, in the Customize Menus panel.

  • Mouse Actions—Customize mouse actions, in the Customize Mouse Actions panel.

  • Profiles—Create and edit profiles, in the Customize Profiles panel.

  • Workspace 3D Options—Set options for stereo display, perspective, and fogging (depth cueing), in the 3D tab of the Appearance panel.

  • Workspace Appearance— Set options for the Workspace appearance: background color, lighting, and materals effects, in the Appearance panel.

  • Show options—Show various main window components, including the status bar, clipping planes, command input area, sequence viewer, Find toolbar, Build toolbar, and Fragments toolbar.

  • View—Set various options that affect the Workspace view. This submenu duplicates items that are on the Advanced View Options of the View menu.

  • Aliases and Macros—Set up aliases and define macros. The submenu has the following items:

    • Command Aliases—Create and edit aliases for commonly used Maestro commands.

    • Function Key Macros—Create and edit macros that can be bound to the Fn function keys.

    • Macros—Define and edit Maestro command macros.
