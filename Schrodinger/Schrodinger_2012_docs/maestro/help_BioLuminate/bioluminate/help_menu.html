Help Menu

The Help menu provides access to a range of information on BioLuminate, Maestro, and the applications they support.

  • Check for Updates—Check the Schrödinger web site for software updates. Opens a web page that displays information on the status of your software.

  • Online Help — Displays the Getting Help topic in your browser. This topic has links to several other locations at which you can find information.

  • Manuals Index — Opens a PDF reader at the PDF documentation index. You can then click a link in the index to open a manual or other PDF document.

  • Search Manuals—Opens the full text search index for the PDF documentation (Manuals_Search.pdx) in Adobe Reader. You can then search across all the PDF documentation for words or phrases.

  • Keyboard Shortcuts — Opens a topic that describes use of keyboard keys in Maestro.

  • Knowledge Base — Provides links to the Knowledge Base on the Schrödinger web site, where you can do a keyword search for information.

  • Diagnostics — Opens a panel in which you can run licensing and installation diagnostic tests.

  • About — Submenu with information about products, applications, and utilities.

    • About Maestro — Displays version and other information for Maestro, such as copyright notices.

    • About Applications — Displays a topic that contains information on Schrödinger products.

    • About Utilities — Opens your PDF viewer with the quick reference sheet that describes the utilities in $SCHRODINGER/utilities.

  • Contact Technical Support — Displays a form in a browser window that you can fill in to create a mail message to send to technical support.

  • Advanced Help Options — This submenu contains the less frequently used help items.

    • Tips — Opens a topic that supplies some tips for using Maestro.

    • Python Module Overview—Opens the top-level document that provides an overview of the Python modules supplied for use with Schrödinger software.

    • Python API Reference—Opens the top-level document that provides access to detailed descriptions of the Python modules supplied for use with Schrödinger software.

    • New Features — Opens a help topic that provides links to the New Features topics for each product.

    • Known Issues — Provides links to the Known Issues pages on the Schrödinger web site. You must log on to view the pages, if you are not already logged on.

    • Demo — Starts a Python-based demonstration of some of the basic features of Maestro.
