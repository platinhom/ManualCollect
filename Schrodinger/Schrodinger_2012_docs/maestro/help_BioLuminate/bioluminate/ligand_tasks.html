Small Molecule Tasks Subenu

The Small Molecule Tasks menu provides access to applications, workflows, and tools that apply principally to ligand-sized molecules or are used for interaction of ligands with proteins. Most of the Maestro applications are available from this submenu.

  • ADME—Calculate ADME properties with QikProp.

  • Binding Energy Estimation—Estimate the binding energy of a ligand to a protein. The submenu offers the following choices:

    • MM-GBSA—Calculate ligand binding energies using Prime MM-GBSA.
    • Linear Interaction Model—Generate a model for predicting the binding free energy of new ligands using a model based on known binding energies and simulations, using Liaison.

  • Copy/Paste/Cut—Submenu for copying, cutting, and pasting ligand structures. Structures can be copied from other applications and pasted into the Workspace, and conversion to 3D can be performed. Cutting structures can only be done in the Workspace.

  • Core Hopping—Substitute the core of a ligand for other cores, retaining the side chains. The submenu offers the following choices:

    • Receptor-Based—Substitute cores in the presence of a receptor, using Glide for docking of the new ligands.
    • Ligand-Based—Substitute cores using alignment to find similar cores, based only on the ligand.
    • Isosteric Matching—Substitute cores using shape overlap to find similar cores, based only on the ligand.

  • Docking—Dock potential ligands to a receptor . The submenu offers the following tasks:

    • Grid Generation
    • Glide Docking
    • Interactive (enumeration and docking, combiglide)
    • Combinatorial Screening
    • Covalent Docking
    • Visualize XP Interactions
    • Virtual Screening Workflow
    • Induced-Fit Docking
    • QM-Polarized Ligand Docking

  • Flexible Ligand Alignment—Align ligands to the first in the set by performing a quick conformational search and aligning the best conformer to the reference (first) ligand. The aligned structure replaces the original. Opens the Flexible Ligand Alignment panel.

  • Ligand and Structure-Based Descriptors—Generate properties for ligand molecules that can be used as descriptors in a QSAR or QSPR model. Opens the Ligand and Stucture-Based Descriptors panel.

  • Ligand Filtering—Filter ligands based on their properties, in the Ligand Filtering panel.

  • Merge Duplicate Ligands—Merge properties of duplicate ligands and write the merged records to a CSV file with a SMILES string for the structure.

  • Ligand Preparation—Prepare ligand-sized molecules for use in modeling, with LigPrep. Includes 2D→3D conversion, adjustment or expansion of ionization states, tautomers, ring conformations, stereochemistry, and minimization.

  • Ligand Properties—Generate properties for ligand molecules that can be used as descriptors in a QSAR or QSPR model. (See Ligand and Stucture-Based Descriptors.)

  • Library Design—Prepare a library of molecules from a core structure by addition of fragments at designated attachment points (see Combinatorial Library Enumeration).

  • Model Building and Statistics—Build and apply a QSAR or QSPR model based on molecular properties, with Strike.

  • P450 Site of Metabolism—Predict the likely sites of metabolism of ligands in various P450 sites.

  • Pharmacophore Modeling—Develop pharmacophore models of ligands (hypotheses) and use them to screen a database for matches; create and manage a database for screening. Uses Phase.

  • pKa Prediction—Predict the pKa of ligand-sized molecules. Two methods are available: Empirical, which uses Hammett and Taft methodology (Epik), and Quantum Mechanical, which uses ab initio methods and continuum solvation, with an empirical correction term (Jaguar).

  • Quantum Mechanics—Run electronic structure calculations with Jaguar and the semiempirical NDDO module. The submenu offers the full range of tasks.

  • R-Group Analysis—Analyze the side chains of a set of ligands and display properties related to these side chains, using the R-Group Analysis panel.

  • Shape Screening—Screen a set of molecules to find those that have the same shape as a query molecule (or shape), using Phase Shape Screening.
