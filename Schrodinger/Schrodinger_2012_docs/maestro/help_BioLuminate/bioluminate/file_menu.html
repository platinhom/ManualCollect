File Menu

This menu provides commands that perform actions on files and projects. The main menu items are described below, followed by the items on the Advanced File Options submenu.

  • Change Directory—Change Maestro's working directory, using a directory chooser. You can also change the directory in the Commands text box with the changedirectory Maestro command.

  • New Project—Create a new project. The current project is closed.

  • Open Project—Open a project. Displays a project selector in which you can navigate to the desired project. The current project is closed.

  • Open Recent Project—Open a recently used project. This submenu lists the most recently used projects, from which you can choose. The current project is closed.

  • Close Project—Close the current project. A scratch project is created to replace it.

  • Save Project As—Save the project under a new name, and continue working in the new project. The old project remains as it was when you performed the save.

  • Delete Project—Delete the current project. Any uncompleted jobs started from this project will finish, but cannot be incorporated.

  • Import Structures—Import structures into the project. Opens the Import panel.

  • Export Structures—Export project entries to files. Opens the Export panel.

  • Get PDB—Import a structure from the PDB, either from a local installation or from the web. Opens the Get PDB File dialog box.

  • Save Image—Save an image of the Workspace to a file in TIFF, JPEG, or PNG format, in the Save Image panel.

  • PyMOL—Open the PyMOL for Maestro panel, which allows you to start PyMOL, load structures from Maestro to PyMOL, and make settings for the PyMOL scene and image.

  • Show Project Table—Open the Project Table panel.

  • Show Toggle Table—Open the Toggle Table panel, which provides PyMOL-like interaction with the Workspace.

  • Quit—Quit the Maestro Elements application.

Advanced File Options

This submenu collects many options that are less commonly used. Some of these have toolbar buttons or keyboard shortcuts, so that the menu item is not usually needed.

  • Import from Project—Import entries from another project. Opens a project selector, then a dialog box to choose the entries.

  • Export Project—Export part or all of the project to a new project, and continue working in the current project. The project selector that opens offers options for inclusion of user-created files, for saving as a compressed file (.prjzip, the default), for copying only the selected or included entries, for inclusion of workflow data from CombiGlide, Phase, or Prime, and for saving in Suite 2012 project format. Plots are only copied if you copy all entries to the new project.

  • Merge 2nd Project into Currently Opened—Merge another project into the current project. Opens a project selector, in which you can navigate to and select the project to merge.

  • Update Project—Submenu for updating the structures in the project when this is not done automatically.

    • Save Workspace Changes—Save changes made in the Workspace to the project. This is only necessary when you don't have the preference for automatic saving of changes set (in the Preferences panel, under Project - Entry changes.)

    • Update Coordinates—Change the coordinates of the entries in the Workspace to the coordinates of the current view.

  • Create Project Entry from Workspace—Create a new project entry from the Workspace structure. A dialog box is displayed, in which you can enter a title for the entry, and choose whether to create the entry from all atoms, the selected atoms, or the displayed atoms. You can also do this with Ctrl+Shift+N (⇧⌘N)) or the Create Entry button on the Workspace toolbar.

  • Publish Project—Create a web page (HTML file) with a relative link to the project, which is stored as a zip file (.prjzip). You can provide a summary of the project, which is included in the web page. The page and project can then be placed on a web site.

  • Save/Restore/Export Full Workspace (Scenes)—Create and manage "scenes", which are snapshots of part or all of the project, including the Workspace content and display settings. This submenu offers the choices New, for creating a new scene, View, for viewing scenes, and Manage, for managing scenes; and commands for navigation or exiting when viewing scenes. The navigation and exit commands are also available from the Scenes toolbar, which is displayed below the Workspace when you view scenes. See the Scenes topic for details.

  • Save/Restore Project Snapshot (States)—Save a "snapshot" of the current state of the project. This is useful if you are about to make some extensive changes that you might want to revert. The snapshot is stored inside the project.

    • Save Snapshot—Make a copy of the project in its current state; the copy is stored inside the project.

    • Revert to Saved Snapshot—Revert the state of the project to the last saved state, discarding changes made since it was last saved.

  • Save/Restore Workspace Orientation (Views)—Submenu for saving and restoring the orientation of the Workspace (the "camera angle"). These features are also available from the Saved Views toolbar.

    • Save View—Save the current view of the Workspace as a named view. Opens a dialog box, in which you can name the view.
    • Restore View—Restore a saved view of the Workspace. This submenu has Previous and Next items for restoring the previous and the next view in the play sequence of the saved views, and items for up to five views. These are the top views in the Views table of the Manage Views panel, and are listed by the row number, which is also the mnemonic for the view.
    • Manage Views—Open the Manage Views panel, in which you can select, save, restore, delete, import, export, play through, and record movies of views of the Workspace.

  • Backup/Restore Project—Back up the project into a location that is outside the project. See Backing Up and Restoring Projects for more information.

    • Back Up—Make a backup copy of the project in a zip file (.prjzip); the copy is stored in the directory project.backup, where project.prj is the name of the project.

    • Restore from Backup—Restore the state of a project from a backup copy.

  • Project Table Annotate—This submenu provides commands for adding annotations to project entries or the Workspace. These features are also available as keyboard shortcuts.

    • Star Target Entry—Increment the number of stars assigned to the current entry in the Start column of the Project Table. When the maximum number is reached, the next execution of this commands sets the number of stars to zero. Keyboard shortcut: *.

    • Mark Target Entry—Mark or clear the mark for the current entry, creating the Mark property as necessary. Keyboard shortcut: X.

    • Single-Entry Feedback—Display or undisplay entry information in the Workspace, subject to the related preference. Keyboard shortcut: S.
