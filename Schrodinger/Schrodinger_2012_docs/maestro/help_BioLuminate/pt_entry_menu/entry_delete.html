Deleting Project Entries

To delete the selected entries, removing all the associated data, choose Entry → Delete in the Project Table panel. There is no panel or dialog associated with the Delete command.

This command can be undone using the Undo button on the Edit toolbar or by choosing Undo from the Edit menu in the main window.
