Project Table — Entry Menu

The Entry menu contains commands that act upon all selected project entries. There are three groups of items, for Workspace inclusion, for entry modification, and for entry grouping and other entry operations.

Include Only   Include only the selected entries in the Workspace, and exclude all others. Does not affect fixed entries. An X appears in the diamond of the In column for these entries.
Include   Include the selected entries in the Workspace along with the entries that are already included. An X appears in the diamond of the In column for these entries.
Exclude   Exclude (remove) the selected entries from the Workspace. The X that appears in the diamond of the In column for these entries is removed. Fixed entries are also removed with this command.
Fix   Include the selected entries in the Workspace, and keep them in the Workspace until explicitly excluded or unfixed. Actions that exclude other entries from the Workspace, such as Include Only, have no effect on the fixed entries. A padlock icon appears in the In column for these entries.
Unfix   Change the status of the selected entries so that any fixed entries remain in the Workspace but are no longer fixed. The padlock icon is removed from the In column for these entries.
Lock   Lock the selected entries so that they cannot be modified or deleted until they are unlocked. A padlock icon appears in the Row column for these entries. You can change locked entries in the Workspace, but the changes will not be saved to the project.
Unlock   Unlock the selected entries so that they can be modified or deleted. The padlock icon is removed from the Row column for these entries.
Stars   Mark the selected entries with the number of stars chosen from this submenu. The star rating is displayed in the Stars column. You can also click in the Stars column to set the number of stars for a single entry, or use the 0, 1, 2, and 3 keys to set the number of stars.
Mark Target Entry   Mark the target entry. A check box is added to the Mark column for that entry, which is created if it does not exist. The target entry is the entry that is included in the Workspace that meets one of the following conditions:
  • It is the only entry included in the Workspace
  • It is the only entry included in the Workspace that is not fixed
  • It is the only entry included in the Workspace that is selected
Delete   Delete the selected entries
Duplicate   Duplicate the selected entries, and select the duplicates. There are three choices on this submenu:
  • In Place—Each duplicated entry is placed immediately below its source entry, and is included in the source's entry group if one exists.
  • Ungrouped—Each duplicated entry is placed at the end of the ungrouped entries, in sequential order.
  • Into New Group—Each duplicated entry is placed into a new entry group. You can name the group in a dialog box that is displayed. If you click Cancel in this dialog box, the entries are duplicated in place.
Split   Split the entry into several entries by structural unit. The new entries inherit the title of the source entry. There are three actions and an option on this submenu:
  • By molecule—Create one entry for each non-water molecule, and one for all water molecules. The entries are named by appending _moln to the entry name of the source entry, where n is an integer starting from 1.
  • By chain—Create one entry for each chain and one for all water molecules. The entries are named by appending the chain name to the entry name of the source entry. A Chain Name property is added that stores the chain name for each entry.
  • Into Ligands, Water, Other—Create one entry for each ligand-sized molecule, one for all water molecules, and one for all other atoms, such as receptor and cofactors. The ligands are named by appending _moln to the entry name of the source entry, where n is an integer starting from 1. The water entry has the suffix _waters, and the entry for all other molecules has the suffix _others.
  • Copy Properties<Copy the entry properties from the source entry to the new entries.
Merge   Merge the selected entries into a single entry. The merged entry is placed after the last selected entry, inside the last entry's entry group if one exists.
Move to Row   Move the selected entries to a new location in the table. Opens a dialog box in which you can specify a row number. The entries are placed above the row with this number.
Group   Move entries into and out of groups. Submenu provides the following items:
  • New<Create a new entry group for the selected entries. Opens a dialog box in which you can provide a name for the entry group. The group is added to the end of the Project Table, and the entries are moved to the new group and removed from any existing group.
  • Move to—Move the selected entries to an entry group. Opens a dialog box in which you can select the entry group or create a new entry group.
  • By Entry Property—Create groups based on ranges of property values. Opens the Group by Entry Property panel, in which you can select the property, define the property ranges, and create the groups.
  • Ungroup—Move the selected entries out of the group, and place them above the groups at the end of the ungrouped entries.
Color   Color the entries (table rows). Submenu provides the following:
  • Constant Color— Color the background of the entire row for the selected entries with a chosen color. Opens a color selector, in which you can choose the color to apply to the rows. The color is stored in the project, associated with the entries that are colored.
  • By Entry Property—Color table cells based on ranges of property values. Opens the Color by Property dialog box, in which you can select the property, define the property ranges and select and apply the colors.
  • Reset Colors—Reset the background color for the selected entries to the default color.
