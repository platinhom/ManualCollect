Project Selectors

Project selectors in Maestro are a special class of file selectors. The navigation and file selection mechanisms are the same as for file selectors—see the File Selectors topic for more information. The selection of the type of file is limited to Maestro projects, which can have the extension .prj, .prjzip, or .prj.zip.

Projects are indicated in the directory list with a project icon. If the project is in use, either by you or by another user, a lock icon appears over the project icon.

In some project selectors that save a project, there are options for the content and the format:

  • Include user-created files—Include files created by the user when the project is saved.

  • Save as a compressed file (.prjzip)—When the project is saved, compress the entire project into a zip archive, and save only the archive.
