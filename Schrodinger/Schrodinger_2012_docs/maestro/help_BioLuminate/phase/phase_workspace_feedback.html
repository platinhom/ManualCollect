Summary

In this dialog box, you can set options for feedback information that is displayed in the Workspace for ligands used to develop common pharmacophore hypotheses.

Opening the Phase Workspace Feedback Dialog Box

To open the Phase Workspace Feedback dialog box:

  • Choose Display → Workspace Feedback in the Develop Common Pharmacophore Hypotheses panel.

Phase Workspace Feedback Dialog Box Features

  • Show single-entry feedback in Workspace option
  • Include property names option
  • Phase property fields list
  • Save Preferences button
Show single-entry feedback in Workspace option

Display information about a single entry, termed the "target" entry, in the top left corner of the Workspace. The target entry is the entry defined by the following conditions:

  • Only one entry is included in the Workspace
  • More than one entry is included in the Workspace, but only one is not fixed
  • More than one entry is included in the Workspace, but only one is selected

If none of these conditions is met, there is no target entry, and no feedback is displayed. The symbol in the In column of the Project Table is red for the target entry.

Include property names option

Select this option to include the names of the properties in the single-entry feedback along with the property value, in the form name:value.

Phase property fields list

Choose the properties to be show in the Workspace from the properties that are shown in the Ligands table or Alignments table (depending on which step is displayed). When you include a ligand in the Workspace from this panel, the Phase property feedback is displayed if Show single-entry feedback in Workspace is selected.

Save Preferences button

Click this button to save the settings in this dialog box as preferences that apply to all projects. The default is that the settings apply only to the current project.
