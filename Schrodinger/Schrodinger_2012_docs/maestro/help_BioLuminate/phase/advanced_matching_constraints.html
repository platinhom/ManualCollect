Summary

In this dialog box you can specify distance, angle, and dihedral constraints for a hypothesis, to be used when finding matches.

Opening the Advanced Matching - Constraints Dialog Box

To open the Advanced Matching - Constraints dialog box:

  • Click Constraints in the Matching Conditions dialog box.

Using the Advanced Matching - Constraints Dialog Box

To define a constraint, select sites in the Available Sites list and add them to the Constraint list, one at a time, with the Add button. The sites must be added in the correct order for defining the desired angle or dihedral. If you make a mistake, you can remove a site from the Constraint list by selecting it and clicking Remove. When you have added the correct number of sites (2 for a distance, 3 for an angle, 4 for a dihedral), click the Add button to the right of the Constraint list to add it to the constraints table. The value of the distance, angle, or dihedral is listed in the table. A default tolerance is assigned, which you can change by editing the table cell. You can display the constraint in the Workspace by checking the box in the Mark column.

To change the sites used in a constraint, select the constraint in the table and click Edit. The constraint is removed from the table and placed in the Constraint list, where you can add or remove sites.

To delete a constraint, select it in the constraints table and click Delete.

Advanced Matching - Constraints Dialog Box Features

  • Available sites list
  • Add and Remove buttons
  • Constraint list
  • Add button
  • Edit button
  • Delete button
  • Constraints table
Available sites list

Lists the sites that are available in the hypothesis. You can select a single site from the list to add to the Constraint list.

Add and Remove buttons

Add the site that is selected in the Available sites list to the Constraint list, or remove the site that is selected in the Constraint list.

Constraint list

List of sites for a constraint. An angle or a dihedral is defined by the order of the sites in this list. You can select a single site to remove from the list.

Add button

Add the constraint that is defined in the Constraint list to the constraints table.

Edit button

Remove a constraint from the constraints table and place it in the Constraints list for editing (changing of the sites in the constraint).

Delete button

Delete the constraints that are selected in the constraints table

Constraints table

Table of constraints that have been defined. You can select a single row to change the sites (by clicking Edit) or select multiple rows to delete.

Sites List of sites that define the constraint. Two sites define a distance constraint, three sites define an angle constraint, and four sites define a dihedral constraint.
Mark Column of checkboxes. Checking a box displays markers for the constraint in the Workspace.
Value Value of the distance, angle, or dihedral that is being constrained, in angstroms or degrees.
Tolerance Tolerance to be applied when matching the constraint, in angstroms or degrees. You can edit the tolerance by double-clicking in the cell and changing the value.
