Summary

In this panel, you can set up and run a semiempirical calculation on one or more structures using the semiempirical NDDO module.

Opening the Semiempirical NDDO Module Panel

To open the Semiempirical NDDO Module panel:

  • Choose Tasks → Small Molecule Tasks → Quantum Mechanics → Semiempirical NDDO in the main window.

Semiempirical NDDO Module Panel Features

  • Use structures from option menu
  • Input file text box and Browse button
  • Method option menu
  • Molecular and atomic properties section
  • Plot properties section
  • Optimize geometry option
  • Archive output files option
  • Keywords text box
Use structures from option menu

Choose the source of input structures from this option menu.

Input file text box and Browse button

If the input structure source is a file, enter the file name in this text box, or click the Browse button and navigate to the input file. The structure file can be in Maestro, SD, or MOL2 format.

Method option menu

Choose the method to use from this option menu. The available methods are RM1, AM1, PM3, MNDO and MNDO/d.

Molecular and atomic properties section

Select options for the calculation of atomic and molecular properties. With the exception of bond orders, the properties are added to the Maestro output file as atom or entry properties. The options are:

  • Atomic ESP charges—Calculate atomic charges that are fit to best reproduce the electrostatic potential (ESP keyword).
  • Mulliken charges—Calculate atomic charges from Mulliken population analysis (MULLIK keyword).
  • Bond orders—Calculate bond orders, including sigma-pi components (BONDS and PI keywords).
  • SAR properties—Calculate HOMO and LUMO energies, molecular hardness and electronegativity, electrophilic and nucleophilic frontier electron densities, electrophilic, nucleophilic, and radical superdelocalizabilities, and atom self-polarizabilities (SUPER keyword).
Plot properties section

In this section you can select the properties to tabulate on a grid for display in Maestro as surfaces. For molecular orbitals, you can choose how many orbitals on either side of the HOMO-LUMO gap are plotted. When you plot the electrostatic potential or the average local electron affinity or ionization potential, an analysis of the property on the isodensity surface is included in the output and recorded in the structure output file as Maestro properties.

Optimize geometry option

Select this option to optimize the geometries of the input molecules; deselect it to run single-point calculations. The default is to optimize geometries.

Archive output files option

Archive all the .out files to a zip file, which is named jobname.zip and is copied to the launch directory. If this option is deselected, the .out files are copied back individually to the launch directory, one for each structure. The default is to archive the .out files.

Keywords text box

Enter any additional keywords in this text box. See the Semiempirical NDDO Guide for more information on supported keywords.
