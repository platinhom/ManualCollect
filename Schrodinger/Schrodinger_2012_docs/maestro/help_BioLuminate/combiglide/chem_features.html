Summary

This panel displays plots of various analyses of the reagent side chains in terms of chemical features and compares the distribution of these features in the full library and the selected library.

Opening the Chemical Features Panel

To open the Chemical Features panel

  • Click Chem Features in the Analyze Library step of the Combinatorial Screening panel.

Using the Chemical Features Panel

The chemical features defined in CombiGlide are categories of functional groups that have certain properties. These features are essentially equivalent to pharmacophore features. There are six basic features, which are identified by a single letter:

  • A—Hydrogen-bond Acceptor
  • D—Hydrogen-bond Donor
  • H—Hydrophobe
  • N—Negative center
  • P—Positive center
  • R—Aromatic group

Each side chain can be characterized by the presence of one or more of these features, and assigned a "type", denoted by a label composed of the feature letters. For example, side chains of the type AHN would contain at least one acceptor, one hydrophobe, and one negative center. The side chains can then be analyzed in terms of the presence of multiple features.

The displays in this panel show what types of side-chains tend to concentrate at each position in the ligands that bind well to the receptor of interest, and are intended to assist in the selection of reagents for a second-generation library.

Chemical Features Panel Features

The display in the center of the panel and the controls available depend on the option selected at the top of the panel, and are described with the option.

  • File menu
  • Side Chain Features option
  • Overall Type Selectivity option
  • Type Preferences option
  • Position buttons

File Menu

The File menu allows you to export the chemical feature information in either graphical or text form.

Save as Postscript and Save as Gif save the plotting area of the panel (i.e. excluding the borders, options and buttons) as an image in the relevant format. Only the current display is saved: to save images for the other positions they must be displayed first.

Export to Spreadsheet writes a CSV file that contains the chemical feature data and the chemical feature type data. Export Text File generates a text file that contains the same data and information on the library selection.

Side Chain Features Option

When you select this option, the display shows a bar graph of the fraction of side chains that have a particular feature against the feature, for a given position. The position name is displayed above the graph. The Previous Position and Next Position buttons allow you to select the position that is used. Data for the input reagent list are shown in red; data for the reagents in the selected library are shown in blue. The "error bars" show a measure of the significance of the deviation of the selected library values from the input reagent list values. If the top of the blue bar is outside this range, the differences are statistically significant, and it may be inferred that there is a preference or an aversion for this feature at this position.

Overall Type Selectivity

When you select this option, a bar chart is displayed that shows a measure of how selective the position is. More selective positions are ones where the distribution of types in the selected library significantly differs from the distribution in the input side-chain collection. A value of 1 indicates an unselective position; the larger the value, the greater the selectivity. Selective positions are positions in which it matters more what kind of side chain is added at that position; unselective positions are positions in which the nature of the side chain is relatively unimportant. Thus, when designing a next-generation library, you will probably want to use more reagents at the more selective positions.

The usefulness of the selectivity analysis does depend on how diverse the input collection is. A collection that is uniform in the features it contains (only one type) cannot display selectivity.

You can switch to a view of the details on which the selectivity analysis was based, by clicking Show Type Details. The button changes to Show Type Selectivity so you can switch back to the selectivity view. The details view shows the number of types represented at each position in the input collection and the selected library, and an effective number of types, which is an entropy-based measure that is like a number of types corrected for the fact that not all types occur equally frequently.

Type Preferences

When you select this option, a chart is displayed for a given position that shows which types are represented in the input collection and in the selected library, and what fraction of each type is present. There is one "table" for each of the collections. Each row represents a chemical feature; each column represents a type. The width of the column corresponds to the fraction of that type in the collection. A given cell in a column is colored if the feature is represented in the type, and is left white if the feature is not represented. The color legend for the features is given to the right of the chart.

The types are in the same order in each table, and the boundaries between the types in each table are connected by a line. The options at the bottom of the panel allow you to order the types alphabetically, by the fraction of the type in the input library, or by the fraction of the type in the selected library.

Position Buttons

The Previous Position and Next Position buttons allow you to select the position that is used.
