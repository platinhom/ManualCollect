Summary

Make settings to control the docking of the combinatorial libraries.

Working in the Configure Docking Step

In this step, you select a receptor grid that has already been computed, and set options for docking the variations on the core structure for each set of attachments. The options are a reduced set of the full Glide options. The options in the Settings tab are applied to all docking jobs.

You can also apply Glide constraints to the docking of the structures in the final phase of docking, in which all attachments are in place. Glide constraints are only applied to the fully substituted core; they are not applied to the docking of the core structures, or of the core with a single attachment. Constraints can be used to filter out reagents when the part of the ligand that satisfies the constraint is not in the core.

Configure Docking Step Features

  • Settings tab
  • Constraints tab

Settings Tab

The Settings tab is a combination of parts of the Glide Settings tab and Ligand tab, with the following features:

  • Receptor grid section
  • Docking section
  • Scaling of van der Waals radii section

Receptor grid section

In this section you select the receptor grid file. Either enter the base name of the receptor grid file in the Receptor grid base name text box, or click Browse and navigate to the receptor grid file.

If you have not generated a grid, you must do so before proceeding, in the Glide Receptor Grid Generation panel, which you can open from the CombiGlide submenu of the Applications menu in the main window.

Docking section

In this section you select options for docking the combinatorial library.

Allow inversion of tetrahedral nitrogens—Select this option to allow inversions at tetrahedral nitrogen atoms. Selected by default.

Allow flips of 5- and 6-member rings—Select this option to allow the conformation of nonaromatic 5- and 6-membered rings to be varied during docking. Selected by default. If you select this option, you can also select Include input ring conformation to ensure that the input ring is included; otherwise it is only included if it is sufficiently low in energy.

Amide bonds—Choose an item from this option menu for the treatment of amide bonds. The choices are:

  • Penalize non-planar conformation—penalize amide bonds that are not cis or trans (default)
  • Vary conformation—allow nonplanar amide bonds
  • Retain original conformation—freeze amide bonds in their input conformation throughout docking
  • Allow trans conformation only—enforce trans conformation to within a small angle range.

Do not dock or score ligands with more than n atoms—Enter a number to restrict the size of the molecules that are docked. Default is 300.

Do not dock or score ligands with more than n rotatable bonds—Enter a number to restrict the number of rotatable bonds to sample during docking. Default is 50.

Scaling of van der Waals radii section

Enter a scaling factor and a partial charge in the text boxes to scale the van der Waals radii of ligand atoms with small partial charges. The scaling simulates flexibility of nonpolar groups.

Constraints Tab

This tab contains controls for applying Glide constraints to the docking of the combinatorial library. These constraints do not apply to the docking of the core molecule or to single-position docking. The contents of this tab are identical to those of the Glide ligand docking Constraints tab.
