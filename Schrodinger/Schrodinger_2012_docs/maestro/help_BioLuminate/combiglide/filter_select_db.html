Summary

In this dialog box you can choose a selection strategy, set up property filters, and manually select reagents for inclusion or exclusion from the library. When you click OK or Apply. a job is run to apply the filters and the strategy to obtain reagent lists for each position.

Opening the Filter and Select Dialog Box

To open the Filter and Select dialog box

  • Click Filter and Select in the Analyze Library step of the Combinatorial Screening panel.

Filter and Select Dialog Box Features

The Filter and Select dialog box is a nonmodal dialog box, which allows you to interact with the Combinatorial Screening panel while you are working in the dialog box. The dialog box consists of three tabs, which are described in separate topics.

  • Selection Strategy Tab
  • Filter Tab
  • Include/Exclude Tab

Below the tabs is a set of buttons that perform the standard actions.
