Ligand Docking — Torsional Constraints Tab

  • Summary
  • Torsional Constraints Tab Features
  • Related Topics

Summary

In this tab you can set up torsional constraints for all ligands, in terms of SMARTS patterns and sets of torsions that are constrained relative to those SMARTS patterns.

This tab is only available when either Dock flexibly or Refine is selected in the Settings tab.

Torsional Constraints Tab Features

  • Apply torsional constraints option
  • Patterns table
  • Pattern action buttons
  • Constrain options
  • Torsions table
  • Pick option and menus
  • Torsion action buttons
Apply torsional constraints option

Select this option to apply torsional constraints when docking ligands. When you do so, the rest of the tab becomes available.

Patterns table

This table lists the SMARTS patterns used to define the torsional constraints.

Row Table row index.
Status Text message that indicates whether the pattern is suitable for defining torsional constraints, and if not, the reason it is not suitable.
SMARTS pattern SMARTS pattern that is to be used for a constraint.
Atoms Number of atoms in the corresponding SMARTS pattern
Torsions Number of torsions that are defined for the corresponding SMARTS pattern.
Pattern action buttons

These buttons perform actions on the rows of the Patterns table.

Create from Selection button

Create a new SMARTS pattern in the table from the current Workspace selection.

Create Blank Row button

Create a new blank row in the table.

Replace from Selection button

Replace the SMARTS pattern in the selected row with a SMARTS pattern derived from the current Workspace selection. Only available if a single row is selected in the table

Delete button

Delete the rows that are selected in the table.

Delete All button

Delete all rows in the table.

Constrain options

These options allow you to choose which torsions to constrain.

  • All torsions for pattern—Constrain all torsions for the pattern that is selected in the table, subject to the conditions for a torsion to be valid for use as a constraint.
  • Selected torsions for pattern—Constrain the torsions that are listed in the torsions table, below. When you select this option, the torsions table and related tools become available.

Torsions table

This table lists the torsions that have been selected for use as constraints with the pattern that is selected in the Patterns table.

Row Table row index.
Status Text message that indicates whether the pattern is suitable for defining torsional constraints, and if not, the reason it is not suitable.
Atoms Number of atoms in the corresponding SMARTS pattern
Set Angle Check boxes for setting the corresponding angles to the value specified in the Angle column for each ligand.
Angle Angle to which the torsion should be constrained in all ligands. By default, the torsion is constrained for each ligand to the value it had when it was read by Glide.
Pick option and menus

Select this option to pick torsions in the Workspace. The first option menu allows you to choose the type of object to pick (Atoms or Bonds), and the second option menu allows you to choose whether to create a new row in the table (Create), or to replace the row that is currently selected in the table (Replace).

Torsion action buttons

These buttons perform actions on the rows of the torsions table.

Create Blank Row button

Create a new blank row in the table.

Delete button

Delete the rows that are selected in the table.

Delete All button

Delete all rows in the table.

Delete Invalid button

Delete the rows in the table for which the torsion is not valid for use with the selected SMARTS pattern.
