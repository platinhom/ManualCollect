    Summary

    In this tab, you define a template for the replacement of the core. This involves selecting the template molecule and defining its core.

    Define Template Tab Features

    • Input Structure button
    • Start Defining Template button
    • Export Template button
    Input Structure button

    Import a structure into Maestro and display it in the Workspace. Opens a file selector, in which you can choose a Maestro file or a combinatorial definition file.

    Start Defining Template button

    When you have a structure in the Workspace, click the Start Defining Template button to start defining the template for core hopping. The button text changes to Stop Defining Template.

    Click the bonds to the side chains that you want to keep as part of the new structures. The bonds are marked with an arrow pointing towards the side chain. Multiple clicks on the first bond change the direction of the arrow, then clear the bond selection, then select it again. For the subsequent bonds, multiple clicks select and deselect the bond.

    When you have finished selecting bonds, click Stop Defining Template to exit template definition mode.

    Export Template button

    Export the template to a Maestro file. The file includes the core definition information.
