Summary

This panel allows you to preview the results of attachment-based core hopping as they are being generated, and examine scores for the best output structures.

Opening the Results Preview Dialog Box

To open the Results Preview dialog box:

  • Select Use immediate results visualizer in the Find New Cores tab of the Ligand-Based Core Hopping panel. The panel opens after you click Start and the job has started.

Using the Results Preview Dialog Box

This panel allows you to view the results of the core-hopping calculation while it is in progress. The top-scoring 20 structures are displayed in a table for two or three different scores: side-chain RMSD, core overlap, and synthesizability. The third of these is only available if you use an .sqlite file as input.

The scores are evaluated as output structures are generated, and the results in the tables are updated. If you want to prevent the table for a particular score from being updated, select Freeze updates. The table is not updated until you deselect this option, and you can examine the structures in the table. The structures are ordered from the best to the worst.

To view one of the core structures in the Workspace, click on the cell for the structure. To display the scores for a structure, pause the pointer over the structure.

Results Preview Dialog Box Features

  • Template structure
  • Inputs processed and Results returned text boxes
  • Progress bar
  • Substitutability Analysis button
  • Side chain RMSD score section
  • Core overlap score section
  • Synthesizability score section
  • Freeze updates option
  • Statistics text boxes
  • Structure display tables
Template structure

The template structure with the attachment points marked is shown at the top of the panel, with various properties listed to the left.

Inputs processed and Results returned text boxes

These text box reports the number of input core structures processed, the total, and the number of core structures found that match. They are updated as the structures are processed.

Progress bar

This bar reports the progress of the job. It is only present if you selected an .sqlite input file.

Substitutability Analysis button

This text box reports the number of core structures that have been generated, and is updated as they are processed.

Side chain RMSD score section

This section reports the RMSD of the side chains of the generated core structures with respect to the template, when superimposed on the template side chains. The lowest scores represent the best match. The 20 structures with the lowest RMSD values are shown in the table, ordered from the lowest to the highest.

Core overlap score section

This section reports a score that is related to the overlap of the new core and the template core. The score is evaluated by averaging the fraction of atoms in the new core that have a template core atom within 0.5 Å and the fraction of atoms in the template core that have a new core atom within 0.5 Å, when the side chains are superimposed. Larger scores represent better matches. The 20 structures with the highest overlap scores are shown in the table, ordered from the highest to the lowest.

Synthesizability score section

This section reports a synthesizability score. The score is based on whether a particular substitution pattern has been observed in drug-like molecules, and how many of the attachment points are substituted in the new core molecule. Larger scores represent better results. The 20 structures with the highest synthesizability scores are shown in the table, ordered from the highest to the lowest.

This section is only present if you selected an .sqlite input file.

Freeze updates option

Prevent the table for a particular score from being updated. This allows you to examine the results at a particular point without the table being updated while you are doing so. Updates resume when you deselect this option.

Statistics text boxes

These text boxes report the best score, the worst score, the average score, and the standard deviation of the scores. Each score type has a set of these text boxes. They are updated as results are processed.

Structure display tables

These tables contain structures for the 20 best-scoring new core structures. You can click on a table cell to view the structure in the Workspace. The tooltip for the cells contains the scores, and a number of other properties of the molecule.
