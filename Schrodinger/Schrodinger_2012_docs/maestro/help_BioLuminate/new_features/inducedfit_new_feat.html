New Features in Induced Fit Docking 2012

  • IFD jobs can be run entirely on a remote host, without a shared file system. This means that only the results files are returned from the job, and the job will not be restartable.

  • Two new stages have been added to the input file: PRIME_ECALC, which performs an energy calculation, and PRIME_SIDECHAIN, which performs a side-chain prediction. These stages make the entire set of Prime refinement calculations available to IFD.
