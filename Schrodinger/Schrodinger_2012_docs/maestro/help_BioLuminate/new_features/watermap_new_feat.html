WaterMap 1.4 New Features

  • You can select multiple WaterMap files when importing WaterMap results into the Analyze Results panel.

  • Shape by entry is selected by default.

  • WaterMap archives are now stored in zip format (.zip). The previous .tgz format can still be read, but will no longer be written.
