New Features in SiteMap 2.6

  • An option for detecting sites suitable for protein-protein interactions, Detect shallow binding sites, was added to the SiteMap panel. This option sets the command options -enclosure 0.4 and -maxvdw 0.55.

  • Input and output Maestro files can be in compressed format.

  • A command input file can be written that encodes the command options as keyword-value pairs, with the – sign stripped from the option to form the keyword.

  • The -j jobname is now optional. When it is not used, the job name is taken from the stem of the command input file or of the protein structure file.
