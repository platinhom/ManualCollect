New Features in Core Hopping 1.2

The list below summarizes the additions and changes to Core Hopping.

  • The graphical interface for core hopping has been redesigned. The three types of core hopping procedure—receptor-based, ligand-based, and shape-based— now have the same kind of interface, based on the previous Ligand-Based Core Hopping panel. The shape-based core hopping has been split off into a separate panel, Isosteric Matching. The four former receptor-based core hopping panels are no longer available, as they are replaced by the new Glide-Based Core Hopping panel.

  • The substitutability score can be calculated for all three methods hopping when an SQLite database is used for input. In addition, you can run an analysis on the structure to find other molecules that have the same core, and show the substitution patterns, in the Substitutability Score Analysis panel.

  • An SQLite database can be used for all three methods, and filters on various structural properties can be applied before the protocores are aligned.

  • You can now visualize previous results from the Project Table, in addition to the run just done and results from file.

  • You can write out a shell script that contains the commands for running the job by clicking Write.
