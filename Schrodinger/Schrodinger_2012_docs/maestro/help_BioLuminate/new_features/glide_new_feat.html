New Features in Glide 5.8

New Features

  • Excluded volumes can be defined during grid generation, and applied in docking. Each volume is spherical, and has a penalty associated with it that ramps up from zero at the edge of the sphere to its full value at some distance inside the sphere. The feature is supported in a new Excluded Volumes tab in the Receptor Grid Generation panel, and in the Settings tab of the Ligand Docking panel.

  • Cysteine thiols can be selected as rotatable groups when generating grids.

  • You can restrict the torsional sampling of a wider range of functional groups than amides, in the Settings tab of the Ligand Docking panel. The existing amide bond feature is retained, but is now a choice under Bias torsion sampling for. The functional groups are specified in a resource file, torcontrol.txt, in the impact-vversion/data directory, which you can customize and place in the impact subdirectory of your user resources directory ($HOME/.schrodinger on Linux and Mac, %APPDATA$\Schrodinger on Windows).

  • The structures that are skipped or rejected during docking can be written to a file, jobname_skipped.maegz by adding the KEEPSKIPPED keyword to the input file.

  • Docking output can be written in SD format rather than in Maestro format, except when XP descriptors are written or rotatable groups are present in the grid.

  • New keywords have been added for specifying constraints in the Glide input file, which are consistent with those used in VSW. The old keywords are still available but are no longer written by Maestro.

  • A boolean property for each constraint is added to the pose file to indicate whether the constraint was satisfied during docking.

  • The grid center can be specified with two new keywords, GRID_CENTER_ASL, corresponding to Centroid of selected residues in Maestro, and LIGAND_MOLECULE, corresponding to Centroid of Workspace ligand in Maestro.

  • The maximum number of poses that can be compared for redundancy using symmetry has been increased to 100, and the NMAXRMSSYM keyword has been added to set this value.

  • Glide has been ported to MacOSX and is now available on Mac computers.
