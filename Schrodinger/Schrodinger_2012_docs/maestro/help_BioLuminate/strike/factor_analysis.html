Summary

Performs a principal component analysis of a set of descriptors for selected entries and displays information about the analysis.

Opening the Factor Analysis Panel

To open the Factor Analysis panel:

  • Choose Tasks → Small Molecule Tasks → Model Building and Statistics → Factor Analysis.

Using the Factor Analysis Panel

To do an analysis, you must select a set of entries in the Project Table. When you open the panel, the properties that are available in the Project Table are displayed in the descriptor list. You can then select a set of descriptors, name the job, and click Generate Factors to perform the analysis. If you want to perform multiple analyses, you can select a different set of descriptors or entries, name the job, then click Generate Factors. The results from each job appear on the Visualize data from option menu. You can then select an analysis and plot the scores or the loadings for various combinations of the principal components.

When plotting the data, it is often useful to view a scores plot and a loadings plot side-by-side, to identify relationships between molecules and variables. Clustering of molecules in the scores plot indicates that the molecules have similar variable (property) values. Clustering in the loadings plot indicates that the variables have similar weights in the principal components.

Factor Analysis Panel Features

  • Descriptor list
  • Job name text box
  • Generate Factors button
  • Visualize data from option menu
  • Plot Scores button and axis menus
  • Plot Loadings button and axis menus
  • Reset button
Descriptor list

Lists the available properties from the project table for the selected entries. You can select multiple list items using shift-click and control-click in the usual way.

Job name text box

Specifies the name for the factor analysis job. The name is used in the Visualize data from option menu to identify the analysis, so you should choose unique and meaningful names.

Generate Factors button

Runs the analysis job. When the job finishes, the visualization tools are displayed in the panel if they are not already displayed.

Visualize data from option menu

Lists the available analyses, identified by the job name. Select a data set from this list for plotting. This menu is unavailable until the job finishes, which should be in less than a minute.

Plot Scores button and axis menus

Plot the value of the principal component for each selected project entry. The axis menus allow you to choose which principal components are used to determine the value.

Plot Loadings button and axis menus

Plot the loadings (or weights) for each selected descriptor. The axis menus allow you to choose which principal components are used to determine the loadings. A high loading means that the descriptor has a large relative contribution to the principal component.

Reset button

Clear all analysis jobs from the panel.
