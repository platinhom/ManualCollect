Project Table — Property Menu

The Property menu contains commands that act upon the entry properties in the Project Table.

Reorder Property Columns   Reorder the property columns. Opens the Reorder Properties dialog box so you can set the column order.
Tree   Display a tree view of the properties in the Project Table. Opens the Property Tree panel, which is docked to the right of the properties in the table by default. You can display the properties alphabetically or by family, and you can filter the properties that are shown in the tree. The alphabetical view has the letters of the alphabet at the top level, with the properties at the next level. The family view lists the property families at the top level, with the primary and secondary property sets at the next level, and the properties at the lowest level. Click on the turners to show the next level. The check boxes can be used to show or hide individual properties or groups of properties. You can drag properties between Primary and Secondary sets within a family to redefine these sets.
Show   Show properties in the Project Table. Submenu provides the following selection:
  • All—show all properties
  • Primary Only—show only the primary properties in each property family
  • For Selected Entries—show only those properties in the Project Table for which at least one of the selected entries has a value for the property.
  • Select—open the Show Properties dialog box, in which you can choose the properties to show.
  • Show 2D Structure—Display 2D structures as a third column in the Project Table. The height of the structures can be adjusted with the 2D structure height slider, which is displayed below the table when you show this column in the Project table. Structures with more than a specified number of atoms (default 200) are not displayed. For protein-ligand complexes, the ligand is displayed, with an icon in the top right to indicate that a protein is present. The ligand is detected with the settings in the Ligand Detection Settings dialog box. You can control the appearance of the structures and the size limit in the 2D Structure - Display tab of the Preferences panel.
Show Only Family   Show only the properties or the primary properties from the selected property family. Choose the family and property set from the submenu, which is dynamically constructed from the properties present in the Project Table.
Hide Family   Hide properties from the selected family. Choose the family from the submenu, which is dynamically constructed from the properties present in the Project Table.
Custom Families   Create, edit, and delete custom property families. Opens the Manage Custom Property Families panel.
Show 2D Structure   Display 2D structures as a third column in the Project Table. The height of the structures can be adjusted with the 2D structure height slider, which is displayed below the table when you show this column in the Project table. Structures with more than a specified number of atoms (default 200) are not displayed. For protein-ligand complexes, the ligand is displayed, with an icon in the top right to indicate that a protein is present. The ligand is detected with the settings in the Ligand Detection Settings dialog box. You can control the appearance of the structures and the size limit in the 2D Structure - Display tab of the Preferences panel.
Single-Entry Feedback   Display or undisplay entry information in the Workspace for a single "target" entry, subject to the related preference.
Add Property Column   Add a new property column to the project table. Opens the Add Property dialog box, in which you can specify the name, type, and precision of the property and its initial value.
Copy Property Column   Copy values from one property to another. Opens the Copy Property panel, in which you can specify the target and destination property. Properties can be copied for the selected entries or for all entries.
Calculate Property Column   Calculate predefined structure-based properties or properties derived from other properties. Submenu provides selection of Structure-Based, which opens the Calculate Property panel, or Property-Based, which opens the Project Table Calculator panel.
Clear Property Values   Clear the values for chosen properties. Opens the Clear Values dialog box, in which you can select properties and clear their values, either for the selected entries or for all entries.
Delete Property Columns   Delete chosen properties. Opens the Delete Property dialog box, in which you can select the properties to delete.
Merge Property Columns   Merge the values of properties into a destination property. Opens the Merge Properties dialog box, in which you can select the destination property, the source properties, and a separator. Only string-valued properties are listed for the destination. The source properties are concatenated into a string, in the selected order, separated by the specified separator. The destination property is replaced by the concatenated string of source properties. If you want to include the destination property in the concatenated string, select it as one of the source properties.
