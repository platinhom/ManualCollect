Summary

In this dialog box you define regions to be excluded from ion placement and candidate regions for ion placement.

Opening the Advanced Ion Placement Dialog Box

To open the Advanced Ion Placement dialog box:

  • Click Advanced Ion Placement in the Ions tab of the System Builder panel.

Advanced Ion Placement Dialog Box Features

  • Candidates list
  • Candidates button
  • New button
  • Reset button
Candidates list

Lists the candidates for placement of ions. Ions are placed in the candidate regions that are selected in the table when you click OK.

Candidates button

Automatically find candidate regions to place the ions and fill the table with these candidate regions. When positive [negative] ions are selected, negatively [positively] charged protein residues are shown in the tables. These candidate regions do not include the excluded region defined in the Excluded region section. Control-click rows in the table to select the ion locations. Selected rows [regions] are displayed in CPK, colored red, in the Workspace.

New button

Open the Atom Selection dialog box to define a region to locate ions. The new region is added to the table.

Reset button

Remove all rows from the table.
