Summary

This panel is used to manage the plots that are in the project - create, show, hide, delete, and rename.

Opening the Manage Plots Panel

To open the Manage Plots panel, do one of the following:

  • Choose Tools → Plot Properties in the Project Table panel.

  • Click the Plot button on the Project toolbar of the Project Table panel.

Manage Plots Panel Features

  • Plot list
  • New Scatter Plot button
  • Delete button
  • Show button
  • Hide button
  • Rename button
Plot list

This area lists the plots that are stored in the project. You can select multiple plots in this list to delete, show, or hide, and you can select a single plot to rename. Double-clicking a plot in the list shows that plot.

New Scatter Plot button

Create a new scatter plot. Opens the Scatter Plot panel with the default plot name Scatter-N.

Delete button

Delete the selected plots.

Show button

Show the Scatter Plot panels for the selected plots.

Hide button

Hide the Scatter Plot panels for the selected plots.

Rename button

Rename the selected plot. Only available when a single plot has been selected. Opens the New Plot Name dialog box, in which you can enter a name for the plot. The name may contain spaces.
