Revert to Opening State

The Revert to Opening State command returns the currently open project to the state it was in when the project was last opened, discarding all changes made to the project since it was last opened.

To revert the project to the opening state:

  • Choose File → Advanced File Options → Revert Project to Opening State in the main window.

Note that there is no panel associated with this menu item.

The opening state of the project is not available if you selected Fast close under Options for saving project on close in the Project - Opening projects section of the Preferences panel.
