Save State

You can store a copy of the project in its current state within the project. To save the state of the project, choose File → Advanced File Options → Save/Restore Project Snapshot (States) → Save State in the main window. This may take some time if the project is large, and Maestro will be unavailable until the state is saved.

This facility provides a single-level version history for a project. Once the state is saved, you can discard the current version of the project and revert to the snapshot version by choosing File → Advanced File Options → Save/Restore Project Snapshot (States) → Revert to Saved State in the main window.
