Summary

This panel displays information about the current project, including annotation, jobs, and runs, and allows you to edit the annotation and open the wizard-based panels.

Opening the Project Summary Panel

To open the Project Summary panel:

  • Choose Window → Advanced Window Options → Show Project Table Summary in the main window.

Project Summary Panel Features

  • Annotation section
  • Project section
  • Jobs section

Annotation Section

This section displays the project annotation. The project creation date and a record of jobs run from the project are automatically added to the project annotation

You can edit the text in the annotation section, including the automatically generated text. The standard mouse actions (highlight to copy, middle-click to paste) can be used to copy and paste text. The main key strokes for editing text are:

Ctrl+W (⌘W) to cut the selection
Alt+W (⌥W) to copy the selection
Ctrl+Y (⌘Y) to paste the selection

These and other conventions are the same as those used by Emacs. For more information, visit the web site http://www.tcl.tk/man/tcl8.4/TkCmd/text.htm#M152.

Project Section

This section displays statistics on the project, under two headings, Entries and Runs. Under Entries, there is a button to open the Project Table panel. Under each product in the Runs subsection, there is a button that opens the panel for that product. The data shown is the number of jobs run in each category represented.

Jobs Section

This section displays the jobs in the job database for the current project. Clicking Refresh updates the display for all jobs, and is equivalent to clicking Monitor in the Monitor panel. Clicking Delete deletes the selected jobs from the database. You can select jobs with the usual click, shift-click and control-click key strokes.
