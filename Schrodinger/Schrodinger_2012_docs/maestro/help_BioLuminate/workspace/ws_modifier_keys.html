Workspace Modifier Keys

Modifier keys (Shift and Ctrl or ⌘) can be used with the mouse or trackpad for access to different Workspace functions.

By default modifications are applied when using the mouse to translate or rotate structures in the Workspace. The Shift (⇧) key and Ctrl (⌘) key are separate modifiers that work with both translation (mouse button 3) and rotation (mouse button 2).

The Shift key acts to limit rotation or translation to either the x or y directions. Maestro observes the primary direction of your mouse movements to determine which of these directions to choose.

For translation, if the primary direction is in the y direction (up/down), Maestro restricts the translation of the Workspace structures to the y axis. If the determined primary direction is in the x direction (left/right), Maestro restricts the translation of the Workspace structures to the x axis.

For rotation, if the primary direction is in the y direction (up/down), Maestro restricts the rotation to rotation around the x axis. If that direction is in the x direction (left/right), Maestro restricts the rotation to rotation around the y axis.

The Ctrl key acts to shift the rotation or translation to the z direction ("in/out", so to speak). If you are doing translation and hold the Ctrl key down, moving the mouse in the x direction (left/right) moves the structures along the z axis. (This may not be apparent unless you have the clipping planes window displayed.) If you are doing rotation and hold the Ctrl key down, moving the mouse in the x direction (left/right) rotates the structures around the z axis.

You can change the mapping of the modifier keys in the Customize Mouse Actions panel, which you open with Edit → Settings → Mouse Actions.
