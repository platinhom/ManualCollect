Fixing Entries in the Workspace

To fix the selected entries in the Workspace, choose Entry → Fix.

You can make changes to entries that are fixed in the Workspace. Fixing an entry only means that it remains in the Workspace until you explicitly exclude it, with Entry → Unfix.

Fixed entries are marked with a padlock icon in the In column of the Project Table panel.
