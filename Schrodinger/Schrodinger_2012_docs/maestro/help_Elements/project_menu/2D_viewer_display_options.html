Summary

In this dialog box you can set options for the generation and display of the images of 2D structures that are displayed in the 2D Viewer panel.

Opening the 2D Viewer - Display Options Dialog Box

To open the 2D Viewer - Display Options dialog box

  • Click Display Options in the 2D Viewer panel.

Maximum number of atoms text box

Display the 2D structure only for molecules that have no more than the specified number of atoms.

Maximum scale factor text box

Scale down images by at least this amount, relative to the size of the cell or display area. This cutoff can be used to prevent small molecules from appearing on a much larger scale than large molecules. Set this factor to 1 to fit the image into the cell.

Atom label font size text box

Point size of the font used for the atom labels in the generation of the image.

Label margin text box

Margin around atom labels, in pixels.

Bond line width text box

Width of bond lines in pixels for image generation.

Hash spacing text box

Width of wedge bonds in pixels for image generation.

Bond spacing text box

Space between bond lines for multiple bonds in pixels for image generation.

Use color in 2D images option

Color the atoms by element in the 2D images.

Show implicit hydrogens in 2D images option

Show all hydrogens, rather than leaving them implicit. Nonpolar hydrogens are always displayed.

Label all carbons in 2D images option

Display a C for each carbon atom, rather than leaving them implicit.

Set to ACS Style button

Set the options to the values recommended for ACS publications.

Scale images on export option

Scale images to a uniform image size of 400x200 pixels when they are exported. Images are always scaled when exported to a PDB file. Deselect this option for images for which the bond have the same length.

Include hydrogens when exporting to 2D SD format option

Include hydrogens in the 2D structure when exporting structures to a 2D SD file. The default is not to include hydrogens.

Set to Defaults button

Set the options to the default values.
