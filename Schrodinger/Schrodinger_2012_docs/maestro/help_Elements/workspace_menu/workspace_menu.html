Workspace Menu

The Workspace menu controls what is displayed in the main window. Several of the items on the menu control the appearance of the atoms in the Workspace, the display of extra properties of the Workspace structure such as surfaces and dipole moments, and the selection of objects to display in the Workspace.

  • Clear—Clear the Workspace. This operation removes scratch entries as well as project entries from the Workspace. It does not remove any data from project entries.

  • Create Project Entry—Create a new project entry from the Workspace structure. A dialog box is displayed, in which you can enter a title for the entry, and choose whether to create the entry from all atoms, the selected atoms, or the displayed atoms.

  • Save Image—Save an image of the Workspace to a file in TIFF, JPEG, or PNG format, in the Save Image panel.

  • Fit to Workspace—Fit a group of atoms to the size of the Workspace. The submenu has items for fitting all atoms in the Workspace, the selected atoms, or the ligand to the size of the Workspace. See Fit to Workspace for more information.

  • Full Screen—Enter full screen mode, in which the Workspace occupies the entire screen.

  • Scenes—Create and manage "scenes", which are snapshots of part or all of the project, including the Workspace content and display settings. This submenu offers the choices New, for creating a new scene, View, for viewing scenes, and Manage, for managing scenes; and commands for navigation or exiting when viewing scenes. The navigation and exit commands are also available from the Scenes toolbar, which is displayed below the Workspace when you view scenes. See the Scenes topic for details.

  • Style—Apply a defined visual style to the Workspace contents. This submenu contains commands for applying and reapplying the style, making style settings, creating binding site surfaces, and fixing the receptor in the Workspace. The commands are also available from the Style toolbar.

  • Atom Labels—Set up and display atom labels, in the Atom Labels panel.

  • Atom and Bond Coloring—Change the coloring of atoms and bonds, in the Atom and Bond Coloring panel.

  • Molecular Representation—Change the representation of atoms and bonds. Opens the Molecular Representation panel.

  • Ribbons—Change ribbon representations. Opens the Ribbons panel.

  • Display/Undisplay Atoms—Select the atoms to display or undisplay in the Workspace, in the Display/Undisplay Atoms panel.

  • Clipping Planes—Show or hide the clipping planes window.

  • Stereo—Turn stereo viewing on or off. See the Stereo tab in the 3D tab of the Appearance panel for more information.

  • Enhanced Depth View—Enhance the perception of depth (with ambient spherical occlusion). This option is most useful for large structures.

  • 3D Options—Set options for stereo display, perspective, and fogging (depth cueing), in the 3D tab of the Appearance panel.

  • Appearance— Set options for the Workspace appearance: background color, lighting, and materals effects, in the Appearance panel.

  • Tile Workspace—Arrange the Workspace entries in a rectangular grid of "tiles". See Tiling Entries for more information.
