Renaming Project Entries

Warning: Do not use parentheses, # symbols, or quotes in an entry name.

To rename a project entry:

  1. If the entry is not selected, select it.

  2. Click in the Entry Name table cell and edit the name.

  3. Press ENTER or click somewhere other than the table cell.

To assign the same base name to several project entries:

  1. Select the desired entries.

  2. Click in the Entry Name table cell for one of the entries and edit the name.

  3. Press Ctrl+ENTER (⌘ENTER).

    The entries are assigned the given name, and then Maestro appends an index number to make the names unique.
