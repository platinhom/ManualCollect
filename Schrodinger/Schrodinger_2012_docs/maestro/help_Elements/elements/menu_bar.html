Main Menu Bar

The menus on the main menu bar provide access to panels, allow you to enter commands, and control the appearance and contents of the Workspace. The menu bar has the following menus by default:

  • File—perform tasks on projects such as opening and closing, importing and exporting structures, showing project data; quit Maestro.
    [Display topic list]

  • Edit—undo actions, copy and paste structures or images, build and modify structures, show toolbars, customize Maestro, and find atoms in the Workspace.
    [Display topic list]

  • Display—control the display of the contents of the Workspace, including 3D and stereo effects; save Workspace images.
    [Display topic list]

  • Tasks —Set up and run programs and perform other tasks on structures.
    [Display topic list]

  • Help —View various kinds of documentation, including online help, manuals, information about Maestro and other Schrödinger products; run diagnostics.
    [Display topic list]
