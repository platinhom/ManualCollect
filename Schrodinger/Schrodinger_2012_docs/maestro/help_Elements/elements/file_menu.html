File Menu

This menu provides commands that perform actions on projects. At the end of the menu is a list of recent projects that can be opened by choosing the menu item for the project. The other menu items are as follows:

  • Show Table—Open the Project Table panel.

  • 2D Viewer—Open the 2D Viewer panel, to display 2D structures in a rectangular grid.

  • New Project—Create a new project. The current project is closed.

  • Open Project—Open a project. Displays a project selector in which you can navigate to the desired project. The current project is closed.

  • Close Project—Close the current project. A scratch project is created to replace it.

  • Save Project As—Save the project under a new name, and continue working in the new project. The old project remains as it was when you performed the save.

  • Delete Project—Delete the current project. Any uncompleted jobs started from this project will finish, but cannot be incorporated.

  • Export Project—Export part or all of the project to a new project, and continue working in the current project. The project selector that opens offers options for inclusion of user-created files, for saving as a compressed file (.prjzip, the default), for copying only the selected or included entries, for inclusion of workflow data from CombiGlide, Phase, or Prime, and for saving in Suite 2012 project format. Plots are only copied if you copy all entries to the new project.

  • Import Structures—Import structures into the project. Opens the Import panel.

  • Export Structures—Export project entries to files. Opens the Export panel.

  • Get PDB—Import a structure from the PDB, either from a local installation or from the web. Opens the Get PDB File dialog box.

  • Merge—Merge another project into the current project. Opens a project selector, in which you can navigate to and select the project to merge.

  • Quit—Quit the Maestro Elements application.
