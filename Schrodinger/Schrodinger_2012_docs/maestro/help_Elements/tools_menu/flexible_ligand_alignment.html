Flexible Ligand Alignment Panel

This panel allows you to perform a flexible alignment of the entries that are selected in the Project Table. The first selected entry is used as a template, and is kept as is. For the remaining ligands, a ligand torsional search is performed with ConfGen (see the ConfGen User Manual for details). The conformers generated by ConfGen are then aligned in turn to the reference ligand, and the conformer that has the best overlap with the reference ligand is chosen. This conformer replaces the existing entry, so you should ensure that you copy the original structures if you want to keep them. It is also advisable to use well-minimized structures as input to flexible ligand alignment. The structures must include hydrogens—no implicit hydrogens are permitted. The alignment can take 1–2 seconds per entry.

To open the Flexible Ligand Alignment panel:

This panel is dockable, so it opens by default in the Workspace. You can change this behavior in the Preferences panel, under General – Panels.

To align the ligands click Align Selected Entries.
