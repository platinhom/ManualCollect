Summary

In this panel you can create a hypothesis from a ligand and screen compounds against this hypothesis.

Opening the Simplified Pharmacophore Modeling and Screening Panel

To open the Simplified Pharmacophore Modeling and Screening panel:

  • Choose Tasks → Pharmacophore Search.

Using the Simplified Pharmacophore Modeling and Screening Panel

This panel provides a simple interface for the creation of a pharmacophore hypothesis and its use to screen a set of compounds. The range of features is necessarily limited, but provides the main options.

The hypothesis can be created from a ligand. When you click either of the buttons to specify the ligand source, the pharmacophore features are detected and listed in the Features table. If you edit the ligand, the panel is reset, because the feature list is no longer valid. You can use a ligand that is part of a ligand-receptor complex, and the ligand is extracted for the hypothesis reference structure.

Simplified Pharmacophore Modeling and Screening Panel Features

The panel has two tabs, which are described in separate topics, linked below. The panel features are described here.

  • Hypothesis ID text box
  • Edit tab
  • Find Matches tab
  • Start button
  • Reset button
Hypothesis ID text box

This text box reports the hypothesis ID, which is read in when you import a hypothesis, and created automatically when you use a ligand from the Workspace or a file.

Start button

Start the job to find matches to the hypothesis.

Reset button

Clear all data from the panel and reset all options to their defaults.
