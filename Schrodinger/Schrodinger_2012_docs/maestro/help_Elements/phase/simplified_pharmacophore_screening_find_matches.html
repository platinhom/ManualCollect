Summary

In this tab, you can select the source of structures to screen, and set up and run the screening job.

Find Matches Tab Features

  • Screen structures in option menu
  • Remote option
  • File path text box and Browse button
  • Source of conformations options
  • Must match on at least N site points text box
  • Return at most N hits per molecule text box
  • Return at most N total hits text box
Screen structures in option menu

Choose the source of structures to screen against the hypothesis. You can choose from a Phase 3D database, a file or selected entries in the Project Table. If you choose a database or a file, you must supply the path in the File path text box, or click Browse to navigate to the file or database.

Remote option

If the database is not accessible to the host on which you are setting up the job, select this option. This option is only available when you select 3D database from the Screen structures in option menu.

File path text box and Browse button

If you chose to screen structures in a 3D database or in a file, enter the path to the database or file in this text box, or click Browse to navigate to the database or the file. For a file, the file type must be either Maestro or SD.

Source of conformations options

Select the source of the conformations to use:

  • Use existing conformers—Use the existing conformers in the database or the file. You must generate conformers before you run the search.
  • Generate during search—Generate the conformers as they are needed during the search.
Must match on at least N site points text box

Specify the number of site points in the hypothesis that must be matched in any structure for it to be considered a hit. If you specify less than the maximum number, the Must match settings in the Edit tab determine whether matching a particular site is required or optional.

Return at most N hits per molecule text box

Specify the number of hits to return per molecule. The best scoring N hits for each molecular are returned. Some molecules can return more than one hit because different alignments or different conformers might match the hypothesis.

Return at most N total hits text box

Limit the maximum number of hits returned to the specified number. The hits that are returned are the top-scoring hits, ordered from highest to lowest score of the best hit for each molecule.
