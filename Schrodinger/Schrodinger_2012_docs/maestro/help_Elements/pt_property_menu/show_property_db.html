Summary

In this dialog box you can select the properties to be shown in the Project Table.

Opening the Show Properties Dialog Box

To open the Show Properties dialog box, choose Property → Show → Select in the Project Table panel.

Using the Show Properties Dialog Box

To show the desired properties, select them in the Available properties list and click the right arrow button or press the RIGHT ARROW key to move them to the Properties to show list.

To hide properties that are currently shown, select them in the Properties to show list and click the left arrow button or press the LEFT ARROW key to move them to the Available properties list.

To limit the list of available properties to a particular property family, choose the family from the Subset option menu.

To filter the list of available properties, enter the text you want to match in the Filter text box. The text can include wildcard characters (? for any single character, * for any number of characters) and character sets, enclosed in square brackets.

Show Properties Dialog Box Features

  • Available properties list
  • Subset option menu
  • Filter text box
  • Sort option menu
  • Arrow buttons
  • Properties to show list
Available properties list

List of properties that are available for showing. The list can be restricted by using the Subset option menu or by filtering, or both. You can select items with click, shift-click and control-click. You can select all items with Ctrl+A (⌘A).

Subset option menu

Restricts the range of properties shown in the Available properties list. This option menu lists the defined property families and two other items, All properties, and Properties shown in Project Table.

Filter text box

Filter the contents of the Available properties list so that only the items that match the expression in the text box are displayed. You can use the wildcard characters * for zero or more characters and ? for a single character, and square brackets to enclose a range of characters. The filtering is done as soon as you type in the text box. To remove the filter, clear the text box.

Sort option menu

Sort the property list according to the option selected from the menu: Alphabetically or By project table order.

Arrow buttons

These buttons are used to transfer properties between the Properties in family list and the Available properties list. The upper button (pointing right) moves the properties that are selected in the Available properties list to the Properties in family list; the lower button (pointing left) moves the properties that are selected in the Properties in family list to the Available properties list. You can also use the RIGHT ARROW and LEFT ARROW keys.

Properties to show list

List of properties that will be shown in the Project Table when you click Show. You can select items with click, shift-click, and control-click. You can select all items with Ctrl+A (⌘A).
