Summary

The Calculate Property panel allows you to generate one or more new properties from a fixed list, based on the attributes of the structures.

Opening the Calculate Properties Panel

To open the Calculate Properties panel

  • Choose Property → Calculate in the Project Table panel.

Using the Calculate Properties Panel

To calculate a property, select the property from the list, set any options for the property calculation, select an option for the entries to calculate the property for, and click Calculate.

To recalculate a property, select Recalculate existing values before you click Calculate.

Property

The properties you can generate are listed in this section. To calculate a property, select it from the list. The available properties are:

  • Chiral centers—calculates the number of chiral centers for ligand-sized molecules and adds the chirality properties used by LigPrep (single molecule entries only)
  • Molecular weight (single molecule entries only)
  • Molecular formula (single molecule entries only)
  • Molecular charge (single molecule entries only)
  • Number of atoms
  • Number of heavy atoms
  • Number of residues
  • Number of molecules
  • pKa atom
  • Secondary structure content—generates three properties: percentage of helices, percentage of beta strands, and percentage of random coils
  • Sequential index
  • Spin multiplicity
  • Substructure—calculates the number of occurrences in each entry of the substructure that you provide
  • Surface area—calculates solvent-accessible surface area (SASA) and partitions it into hydrophobic and buried portions. Generates up to four properties.
Options for property

When you select a property, any options that can be set for the property are displayed in the section below the property list. The following properties have options that can be set:

  • Molecular formula—enter a limit on the number of atoms in the text box, The default is 100 atoms.

  • Substructure—name the property in the Property name text box, and enter substructure notation for the substructure whose occurrences you want to count in the text area. This notation is described in detail in Appendix C of the Maestro User Manual

  • pKa atom—pick an atom for which Jaguar pKa calculations will be performed. This atom should be an electronegative atom to which a proton can be added or removed, or an acidic hydrogen.

  • Surface area—Set up the surface type and its parameters, and choose the types of surface area to calculate. The following controls are displayed:

    Surface type options

    Choose the surface type, from Extended radius or Molecular.

    Probe radius text box

    Enter the probe radius for the surface. The default value of 1.4 Å is the value for water.

    Grid spacing text box

    Enter the grid spacing. Only available for the molecular surface.

    Calculate hydrophobic surface area option

    Calculate the hydrophobic surface area of the molecule. Selecting this option produces the Hydrophobic SA property, and the Buried Hydrophobic SA property if Calculate buried surface area is also selected. Hydrophobic atoms are defined in the same way as in Glide for hydrophobic constraints.

    Calculate buried surface area option

    Calculate the buried surface area of a receptor in the presence of a ligand. Selecting this option requires you to choose a receptor, and produces the Buried SA property, and the Buried Hydrophobic SA property if Calculate hydrophobic surface area is also selected. For meaningful results, the entries you calculate the property for should be docked to the receptor. The buried surface area is the area that is not solvent-accessible when the ligand is present.

    Receptor controls

    When calculating buried surface areas, these controls enable you to choose a receptor. You can either pick a receptor in the Workspace, if Pick to define receptor is selected, or click Choose to choose an entry for the receptor.

Calculate for entries

There are two options for the entries for which properties are calculated: All and Selected.

Recalculate existing values

If this option is selected, calculating a property that already exists replaces the property values with new values. If it is not selected and the property exists, the property will be calculated only for the entries that do not already have values.

Calculate

Click this button to calculate the property.
