Prime Database Update Utilities

Prime uses two databases, the PDB database and the BLAST database. As these databases are continually being updated, any installed version might not have the latest contributions. Utilities have been provided for updating these databases from the web sites. You can run these utilities on Windows from a Schrödinger shell—see Installing Prime Third-Party Software and Databases on Windows for details. You must have permission to write to the destination in order to run these utilities.
