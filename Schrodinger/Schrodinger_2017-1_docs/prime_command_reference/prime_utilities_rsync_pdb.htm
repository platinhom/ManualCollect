Updating the PDB Datbase: rsync_pdb

This utility creates or updates a local mirror of the PDB. The syntax is:

$SCHRODINGER/utilities/rsync_pdb [options]

For a description of the options, run the command with the -h option.

The PDB is installed in the first location found in the following list:

  • $SCHRODINGER_PDB
  • $SCHRODINGER_THIRDPARTY/database/pdb
  • $SCHRODINGER/thirdparty/database/pdb

These environment variables are described in Setting Environment Variables for the Prime Third-Party Software and Databases.

This utility uses an rsync server on the Schrödinger web site that enables the files to be downloaded. The files are written in the new remediated format. The Schrödinger site is updated nightly from the PDB. On Windows, the utility wget is used to update the local copy.
