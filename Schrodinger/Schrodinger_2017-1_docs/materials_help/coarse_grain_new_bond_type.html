Define New Coarse-Grained Particle Bond Type Dialog Box

Set up new artificial bond types between coarse-grained particles that are separated by more than one bond.

To open this dialog box, click the Define New Bond Type button in the Bond tab of the Coarse-Grained Force Field Assignment panel.

Using the Define New Coarse-Grained Particle Bond Type Dialog Box

The purpose of this dialog box is to define new artificial bonds between particles that are not bonded, so that potentials other than nonbonded potentials can be applied, and the bonds can be taken into account when defining potentials for angles and dihedrals.

When a new bond type is defined, it is added to the lists of bonds that can be used to define angles and dihedrals, and it is removed from the list in the Nonbond tab.

Define New Coarse-Grained Particle Bond Type Dialog Box Features

  • Particle 1 option menu
  • Particle 2 option menu
  • Number of bonds by which particles are separated text box
  • Reset button
Particle 1 option menu
Particle 2 option menu

These two option menus allow you to choose the particle types between which the new bond is defined.

Number of bonds by which particles are separated text box

Specify the number of bonds by which the two particles must be separated for the new bond type to be identified. The bonds are counted along the bond path between the particles. New bond types created with this dialog box are not included in the bond path.

Reset button

Reset the panel settings to their default values.
