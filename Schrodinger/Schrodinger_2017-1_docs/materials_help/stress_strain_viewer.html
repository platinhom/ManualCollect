Stress Strain Viewer Panel

Display a plot of the stress against the strain from a stress-strain calculation.

To open this panel, choose Tasks → Stress Strain → Analyze Results in the Task Tool.

Stress Strain Viewer Panel Features

  • Load Data from Workspace button
  • Plot toolbar
  • Plot area
Load Data from Workspace button

Load the stress-strain data from the Workspace structure, which must contain the output structure from a stress-strain calculation.

Plot toolbar

The toolbar has tools for manipulating the plot and for saving images. The buttons are common to all plot toolbars, and are described in the Plot Toolbar topic.

Plot area

This area displays the plot of the stress against the von Mises strain.
