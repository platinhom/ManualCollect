Using Features

Locate Ring-Spears Panel

Locate parts of the structure in the Workspace that go through the middle of ("spear") a ring.

To open this panel, choose Tasks → Materials → Locate Ring-Spears in the Task Tool.

Using the Locate Ring-Spears Panel

This panel enables you to find parts of a structure that are entangled, where a bond goes through the middle of a ring ("spears the ring"). Such structures can be generated during automated building procedures, such as when building polymers or enumerating structures on a scaffold. Once the ring-spears are located, you can step through each instance to zoom in on it in the Workspace, and take whatever action you need to fix the structure. You might want to manually move the atoms so that the bond no longer traverses the ring, or you might want to try rebuilding the structure with different choices of options so that it is not entangled.

The ring-spear instances are colored with a color of your choice (red by default), and the representation for the atoms is set to ball-and-stick. Note that the color scheme and the representation remain when you close the panel. You can use the toolbars to reapply a color scheme or representation.

We recommend that you use this panel after building a polymer, especially if you have built the polymer with specific dihedrals, or chosen not to avoid clashes.

Locate Ring-Spears Panel Features

  • Color ring-spear atoms color button
  • Color non-ring-spear atoms option and color button
  • First ring-spear only option
  • Atoms to search options
  • Cycle through ring-spears buttons
  • Reset button
  • Locate button
Color ring-spear atoms color button

Choose the color for the ring atoms and the atoms of the bond that traverse the ring. The color used is shown on the color button. Click the button to open a color selector to choose a new color.

Color non-ring-spear atoms option and color button

Select this option if you want to color all atoms other than those involving the ring-spearing with the color shown on the color button. Click the button to open a color selector to choose a new color.

First ring-spear only option

Select this option to color just the first instance of ring-spearing found, and zoom the view in to this instance.

Atoms to search options

Select an option for the atoms to search for ring-spears, from All or Workspace selection.

Cycle through ring-spears buttons

Click the arrow buttons to cycle through the instances of ring-spears in the Workspace. The view zooms in to each instance so you can examine it more closely.

Reset button

Reset the settings in the panel to their default values.

Locate button

Click this button to locate the ring-spear instances. The atoms involved in the ring-spearing are colored and drawn in ball-and-stick.
