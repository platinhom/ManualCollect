Build Single Complex Panel

Build a single organometallic complex with a single metal center, in octahedral, tetrahedral, or square planar geometry.

To open this panel, choose Tasks → Materials → Build → Single Complex in the Task Tool.

Using the Build Single Complex Panel

There are three main steps to building an organometallic complex:

  1. Choose the metal and the coordination geometry.

    You can do this in the Complex information section. Type in the atomic symbol for the metal, and choose the geometry from the Geometry option menu.

  2. Choose the ligands.

    In this step, you decide how many different ligands you want to use, how many of each ligand to use, and select or sketch the ligands. Each ligand is represented by a row in the Ligands section. Each row specifies the number of copies of the ligand, and has a button to assign a ligand structure to this row. For example, for an ML3X3 complex, you would have two rows, one for ligand L and one for ligand X, and for each you would set the number of copies to 3. Rows can be added by clicking Add Ligand Row.

    The actual structures can be defined in the Ligand sketcher section, either by choosing a template for the ligand, which you can modify, or drawing the ligand structure in the sketcher. To draw a ligand, choose Custom from the Template option menu. When you have drawn or chosen a ligand, click Use Current Structure on the row in the Ligands section that you want to assign the structure to. The ligand formula and whether the ligand is monodentate or bidentate is shown in the ligand row. In addition, the number of occupied coordination sites listed above the ligand rows is updated.

  3. Choose the arrangement of ligands in the complex.

    For octahedral complexes with bidentate ligands, you can automatically arrange them in a facial or meridional conformation, by choosing the isomer in the Complex information section. Both isomers are chiral, but only one of the stereoisomers is built. The facial stereoisomer is in the Λ form, the meridional stereoisomer is in the Δ form. The geometry is applied when you save the structure. To convert to the other stereoisomer, display the Build toolbar, and double-click the R/S button. (If the ligand is symmetric, there are only the two stereoisomers.)

    For tetrahedral complexes with two pairs of monodentate ligands, you can choose either the cis or the trans isomer in the Complex information section.

    For any other type of complex, you must pick the sites on the metal atom for each ligating atom on each ligand, by clicking Specify Ligand Coordination Sites and using the diagram of the complex in the dialog box that opens to pick the sites for each ligand.

When you have finished defining the complex, click one of the action buttons to save it, either to the Maestro project with Create Project Entry or to a file, with Write File.

If you want to create new ligand templates, choose Custom from the Template option menu, then draw the new ligand in the sketcher. You must indicate which atoms are the ligating atoms, by adding a bond to them and labeling the terminal atom as R1 for the first ligating atom, and R2 for the second. Right-click on the atom and choose Set R Group → R1 or R2 to perform the labeling. Click Save New Template to save the ligand structure you drew, and name the structure in the dialog box that is displayed. The custom templates are stored in your Schrödinger user resources directory.

When you have built the complex, you should consider cleaning it up with the Ligand Cleanup panel, and then running a geometry optimization with Jaguar before proceeding with any calculations on the complex.

Build Single Complex Panel Features

  • Complex information section
  • Ligands section
  • Ligand sketcher section
  • Sketcher
  • Clean geometry option
  • Action buttons

Complex information section

In this section, you provide information on the metal and the coordination geometry, and name the complex.

Central atom text box

Enter the atomic symbol for the metal atom in this text box. You can only enter valid atomic symbols: all other text is disallowed.

Geometry option menu

Choose the geometry of the complex. Three choices are available: Octahedral, Square planar, and Tetrahedral.

Isomer option menu

For octahedral complexes, this option menu allows you to specify whether the complex is facial or meridional. This choice is only available if you use three copies of a bidentate ligand, or three copies of two monodentate ligands. For asymmetric bidentate ligands, the facial isomer is the Λ stereoisomer, the meridional isomer is the Δ stereoisomer. (If the ligands are symmetric, there are only the two stereoisomers, which are built by choosing the facial or meridional isomer.)

For square planar complexes consisting of two pairs of monodentate ligands, you can choose to build either the cis or the trans isomer from this option menu.

Title text box

Specify the title for the complex. This title is used as the entry title when the complex is exported to a file or added to the Project Table.

Ligands section

In this section, you specify how many different ligands you want to use, how many of each ligand to use, and how to arrange the ligands. The number of occupied coordination sites is shown at the top of the section.

Add Ligand Row button

Add a row of controls for another ligand. By default, one ligand row is displayed.

Occupied coordination sites display

This text area shows the number of coordination sites that are occupied by the ligands specified in the Ligand rows. The value is updated when you click Use Current Structure to add a ligand to a row.

Ligand row

Each ligand row has a set of controls for specifying a ligand.

Copies box

Set the number of copies by typing the number into the text box, or use the arrow buttons to set the number. Changing the number of copies changes the information on occupation of the coordination sites if a ligand has been assigned to the row.

Use Current Structure button

Click this button to use the structure that is currently displayed in the sketcher for this ligand. The structure must have one or two R groups, labeled R1 for the first and R2 for the second, if the ligand is bidentate. When you click this button, the ligand information and the information on occupation of the coordination sites is updated.

Ligand information

The formula of the ligand and whether it is monodentate or bidentate is displayed to the right of the Use Current Structure button. This information is displayed when you click Use Current Structure. Pausing the pointer over the information shows the 2D structure of the ligand in a tooltip.

Delete button

Click this button (red X) to delete the ligand row.

Specify Ligand Coordination Sites button

Specify the coordination sites for the ligands explicitly, by selecting each site on a diagram of the complex. Opens a dialog box, in which all copies of the ligand are listed with their attachment points. Clicking on a site in the diagram associates the site with the selected attachment point. By default the first attachment point is selected, and the selection moves to the next point (top down, left-right order) when the association is made.

Ligand sketcher section

The ligand sketcher allows you to modify a template structure for a ligand, or add your own structures.

Template option menu

This option menu offers a set of template ligand molecules that you can use directly or edit to create the ligand that you want. When you choose one of the templates, the structure is displayed in the sketcher. You can then edit it with the sketcher tools.

If you choose Custom, you can draw your own ligand in the sketcher. When you have created your ligand, you must draw in a bond from the ligating atom or atoms to a potential metal atom, and label the terminal atom on the bond with R1, or R2 for the second ligating atom of a bidentate ligand. To label the atom, you can either pause the pointer over the atom and type in the label, or right-click on the atom and choose Set R Group → R1 (or R2). The labeled terminal atom is replaced by the metal when the complex is constructed. If the ligand forms a dative bond, you must still add a bond and label it, even if it violates the valence of the ligating atom.

The menu is reordered each time you choose a ligand, so that the last chosen is at the top.

Clear Sketcher button

Remove all structures from the sketcher.

Save New Template button

Save the structure that is currently displayed in the sketcher as a new template. A dialog box opens, in which you can name the template. The template is added to the Template option menu, and is saved for future Maestro sessions in your Schrödinger user resources directory.

Manage Templates button

Manage user-defined templates. Opens the Manage Templates panel, in which you can select saved templates from a menu and delete them, or set the directory in which to store the templates. Only user-defined templates can be deleted; the standard templates cannot be deleted.

Sketcher

Sketch the ligand structure using these tools. The tools are described in the 2D Sketcher Dialog Box topic.

Clean geometry option

To clean up the geometry of the complex with a force-field minimization, select Clean geometry after building. The method is the same as that used in the Clean Up Complexes panel.

Action buttons

Use these buttons to save the structure in the Maestro project or in a Maestro file. The structure is converted to 3D before saving.

Create Project Entry button

Create an entry in the current project for the complex.

Write File button

Write the structure of the complex to a Maestro file. A file selector opens, in which you can navigate to a location and name the file.

Reset button

Reset the panel to its initial state. All changes in the panel are discarded.
