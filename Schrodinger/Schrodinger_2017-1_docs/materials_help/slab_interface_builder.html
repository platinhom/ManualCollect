Build Slabs and Interfaces Panel

Build a slab of a specified thickness by slicing a crystal structure along a specified plane, and optionally add a second slab above or below the first, taken from another crystal structure. Multiple slabs can be generated that are sliced along different planes.

To open this panel, choose Tasks → Materials → Slabs and Interfaces in the Task Tool.

Using the Build Slabs and Interfaces Panel

This panel enables you to generate new periodic systems that are composed of slabs of other periodic systems. Each new system can be composed of a slab of one system, sliced along a particular plane and with a specified thickness, with an optional vacuum layer above it (for simulating surfaces in vacuo). It may also contain a slab of another system, with optional vacuum layers between the two different slabs. The second slab (the "adsorption" layer) is stretched or contracted if necessary so that its lattice constants match those of the first slab. The matching is done on multiples of the lattice constants of each slab, so that the amount of stretching or contraction is small.

If the source structure is a molecular crystal, molecules that span the two planes that define the slab are included in the slab. The atoms in a molecule that spans a plane are considered to be in the vacuum layer. For other structures, the partitioning is done on an atomic basis: atoms that are between (or on) the two planes that define the slab are included in the slab. Any terminating groups added to satisfy valences at the surface are considered to be in the vacuum layer.

To define slabs composed of a single layer:

  1. Display the first source structure in the Workspace.

  2. In the Base layer section, click Load.

  3. Select Define slab.

  4. Specify the Miller indices that define the surfaces of each slab you want to build.

    The h, k, and l text boxes allow you to specify ranges of indices, and all combinations within these ranges are used to define slabs. You must define at least one slab with these text boxes. You can also use the hkl triples text boxes to add a list of triples to use for defining slabs.

    It might be useful to display the planes in the Workspace, on the structure. To do this, you can use the Planes button on the Periodicity toolbar.

    You might have to extend the crystal to obtain a reasonable view of the planes, which you can also do from the Periodicity toolbar.

  5. Specify the distance perpendicular to the plane at which the bottom of the slab starts.

    The slice defining the bottom of the slab is made parallel to the defined plane at the specified distance.

  6. Specify the thickness of the slab.

  7. (Optional) If you want space between this instance of the slab in the periodic structure and the next instance, specify the amount of space in the Vacuum buffer box.

    A large distance allows you to simulate a surface in a vacuum (essentially a 2D-periodic structure).

  8. (Optional) If the structure is not a molecular solid, you can choose a fragment to terminate any dangling bonds on the surface, and specify the length of the bond to the fragment from the surface atoms.

  9. Name the job and click Run to run it, or click the Settings (gear) button to make job settings and run the job.

To define slabs composed of two layers:

  1. Complete the steps above to define the base layer, but do not run the job.

  2. Select Adsorption layer.

  3. Display the structure for the adsorption layer in the Workspace.

  4. Click Load in the Adsorption layer section.

  5. Define the slab, using steps 3 – 8 above.

  6. In the interface section, select an option for the bottom layer of the new system.

    You can choose either layer, depending on which faces of the slabs you want to be next to each other.

  7. Set the inter-layer separation.

    The separation is measured from the top of the vaccum buffer, if you defined one. You can make this separation negative to move the top layer down into the vacuum buffer (or you can decrease or remove the vacuum buffer).

  8. (Optional) Translate the top layer along the new lattice vectors by a specified amount in angstroms.

    This may be useful to get better alignment of atoms at the interface.

  9. (Optional) Change the tolerances for stretching or contracting the adsorption layer to match the base layer lattice constants, and the maximum number of cell extents permitted to do the matching.

  10. Name the job and click Run to run it, or click the Settings (gear) button to make job settings and run the job.

Base layer section

Define the slab for the base layer in this section. For a single slab, you need only define the base layer. For a interfacial (two-layer) system, the base layer is the one onto which the adsorption layer is added. The base layer maintains its geometry, whereas the adsorption layer can have a strained geometry.

Adsorption layer option and section

Define the slab for the adsorption layer in this section. The adsorption layer is added to the base layer, and can have a strained geometry, specified in the Interface section.

Load cell from Workspace button

Click the Load button to load a crystal structure (asymmetric unit) from the Workspace as the source for a particular layer. There is an instance of this button in the Base layer section and the Adsorption layer section.

Define slab option and section

Select this option to define the slab for a particular layer, and enter the information needed to define the slab. There is an instance of this option and section in the Base layer section and the Adsorption layer section.

Miller indices settings

Specify the Miller indices that define the planes used to slice the crystal to generate the slabs.

It might be useful to display the planes in the Workspace, on the structure. To do this, you can use the Planes button on the Periodicity toolbar.

You might have to extend the crystal to obtain a reasonable view of the planes, which you can also do from the Periodicity toolbar.

h, k, l boxes

Set the range of Miller indices by entering the minimum and maximum values for each index in the boxes. All possible triples of Miller indices are generated from the ranges specified, to define the planes used to create the slabs. You must specify at least one plane using these boxes.

To define a single plane, set the values in both boxes to the same value for each index. For example, to specify the 100 plane, set the two values for h to 1, the two values for k to 0, and the two values for l to 0.

hkl triples option and text box

Define the planes used to generate the slabs in terms of triples of Miller indices, and enter the list of triples in the text box. Each triple defines one slab. The indices within a triple must be space-separated, and there must be a comma between each triple.

Geometry settings

These settings define the details of the two slab surfaces and the space above the surface (if any) that defines the new structure.

Units options

Select the units that define the slab bottom, slab thickness, and vacuum buffer, from Inter-planar spacing or Å. The inter-planar spacing is the distance between the closest two instances of the plane that defines the slab.

Slab bottom box

Specify the distance above the chosen plane at which the bottom surface of the slab starts, using the selected units to define the distance. The distance is taken normal to the plane.

Slab thickness box

Specify the distance above the slab bottom at which the top surface of the slab is placed, using the selected units to define the distance. The distance is taken normal to the slab bottom plane.

Vacuum buffer box

Specify the amount of empty space (measured normal to the surface) to leave above the top surface of the slab in the new periodic system.

Termination settings

If the input structure is not a molecular crystal, specify how to terminate any dangling bonds or unsatisfied valences when the new surfaces are exposed.

Fragment option menu

Choose the fragment used to terminate any dangling bonds on the surfaces of the slab.

Bond length box

Specify the length of the bond to the terminating fragment. This value will depend on the nature of the atoms involved in the bond. Only a single value can be specified.

Interface section

Specify how the two slabs are placed to create a system with an interface.

Bottom layer options

Specify which layer should be the bottom layer of the interface model. The layers are stacked with their surfaces parallel.

Inter-layer separation box

Specify the separation between the two layers, in angstroms. If a vacuum buffer is defined for the bottom layer, this value can be negative to reduce the gap between the layers.

Translate top layer along lattice vectors boxes

Translate the top layer along the lattice vectors a and b by the amounts specified in the boxes. The translation can be used to get a better match between the two structures.

Stretch or contract adsorption layer by up to boxes

To create a system that is periodic, the periodicity in the two layers must match or be made to match. This is done by stretching or contracting the adsorption layer by a small amount, whose limit is specified in these text boxes. For the a and b values, a multiple of the a and b values in each slab is found for which the differences are less than the specified tolerance, i.e.

|ma1 − na2| < δa
|jb1 − kb2| < δb

For the angle, |γ1 − γ2| < δγ.

Maximum cell extents box

Place a limit on the number of multiples of a and b that can be used to match the two structures: in terms of the equations above, m, n, j, and k must not be greater than the value specified, otherwise the bilayered system is discarded.

Job toolbar

Manage job submission. See Job Toolbar for a description of this toolbar.

Status bar

The status bar displays information about the current job settings for the panel. This includes the job name, task name and task settings (if any), number of subjobs (if any) and the host name and job incorporation setting. It also displays a message for a short time when you start a new job.

The status bar also contains the Help button, which opens the help topic for the panel.
