Reading and Writing LigPrep Settings

You can save the settings in the panel, including the choice of input and output, by writing them to a LigPrep input file. This file can be used as an alternative to command-line options when running a LigPrep job from the command line. It can also be read in to the LigPrep panel, to set all the options a particular way. This feature is useful if you run LigPrep with nonstandard settings (although these are remembered by the panel), particularly if you use different settings in different circumstances. All the settings for the particular circumstance can be made by reading the appropriate input file. This file is a simple keyword-value file. For more information on the file, see The ligprep Command.

  • To write a LigPrep input file, click the Settings button arrow and choose Write. The file is written to the current working directory with the name jobname.inp.

  • To read a LigPrep input file, click the Settings button arrow and choose Read, then navigate to the input file in the file selector that opens.
