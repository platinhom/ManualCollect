Summary of Procedures for KNIME Workflows in Maestro

To set up and install a new workflow:

1. Choose Scripts → New KNIME Workflow.
2. Import or build a new workflow in the KNIME workspace.
3. Edit the Reader, Writer, and Quick Form node names to configure them for input, output and parameter passing.
4. Rename your workflow without spaces.
5. Save it and close KNIME.
6. Test it in the startup panel.
7. Install it in Maestro (see below).

To run a workflow:

1. Choose Scripts → workflow-name.
2. Make settings in the panel for input and output.
3. Click Run.

To install a workflow:

1. Unzip the zip file containing the workflow, if necessary.
2. In Maestro, choose Scripts → Install.
3. In the Install Scripts panel, click Browse.
4. Navigate to and select the directory where the workflows are installed.
5. Click Choose.
6. In the Install Scripts panel, check the box next to the script name, and click OK.
