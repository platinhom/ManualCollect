Installing and Updating Pregenerated KNIME Workflows

If you have wrapped workflows that are not installed in Maestro, you can install them by choosing Scripts → Install. In the Install Workflows panel, you can browse to the desired location, and select workflows to install. The items that are listed are Python scripts. These can include ordinary Python scripts as well as KNIME workflow wrappers, so you must take care to choose the KNIME workflow wrappers for installation.

The installed workflows are kept in the workflows folder of your Maestro user resources directory. See Installing KNIME Workflows for information on installing workflows for all users.
