Running KNIME from the Schrödinger Installation

When you install KNIME and the Schrödinger KNIME extensions from the Schrödinger distribution, they are installed into $SCHRODINGER/knime-vversion.

Linux: Use the following command to run KNIME:

$SCHRODINGER/knime [options]

For a description of the options, run this command with the -h option.

Windows: Double-click the KNIME icon on your desktop. If you want to use any of the options, you can open a Schrödinger Command Prompt window from the Start menu, and use this command:

knime [options]

If you want to use specific options every time you start KNIME, right-click on the KNIME icon on your desktop and choose Properties, then add the options to the end of the text in the Target text box, making sure that there is a space before each option.

Mac: Click the KNIME icon on the dock. If there is no KNIME icon on the dock, you can put it there by dragging it from the SchrodingerSuite2017-1 folder in your Applications folder. If you want to run KNIME from the command line, open a terminal window and use the same command as for Linux.

Some of the knime command options are stored as preferences, and can be changed by opening the Preferences panel from the File menu, and choosing KNIME → Chemistry → Schrödinger. Options are stored in a custom knime.ini file in your Schrödinger user resources directory.
