Troubleshooting Semiempirical NDDO Calculations

If you use a Maestro structure file as input, and you experience a problem running the module and want to see the input file that is created from the .mae file, include the -SAVE option on the command line. This option creates a .zip file of the job scratch directory, including the input file, which has the name jobname.mop. You might be able to edit this file to work around the problem, and then submit the job using the .mop file as input.
