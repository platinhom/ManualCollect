The MOPAC 2012 and Semiempirical NDDO Module Panels

Semiempirical NDDO jobs can be set up and run from Maestro by using the Semiempirical NDDO Module panel or the MOPAC2012 panel. To open these panels, choose Tasks → Quantum Mechanics → Semiempirical NDDO or Tasks → Quantum Mechanics → MOPAC2012 in the Task Tool.

Figure 1:The Semiempirical NDDO Module panel.

Figure 2:The MOPAC2012 panel.

The input structures can come from a file, from the Project Table, or from the Workspace. You can choose the structure source from the Use structures from option menu. If the input structure source is a file, enter the file name in the Input file text box, or click the Browse button and navigate to the input file. The structure file can be in Maestro, SD, or MOL2 format.

You can choose the method to use from the Method option menu. The available methods in the Semiempirical NDDO panel are RM1, AM1, PM3, MNDO and MNDO/d. In the MOPAC2012 panel, these are supplemented with PM6, PM6-D3, PM6-DH+, PM6-DH2, PM6-DH2X, PM7, and PM7-TS [3,4]. The variants of PM6 are corrected for one or more of dispersion, hydrogen bonding, and halogen bonding. PM7-TS is intended only for barrier heights of transition states. If you want to optimize the geometry, select Optimize geometry; to run a single point calculation, deselect this option.

You can set the charge and spin multiplicity for the molecule in the Charge and Multiplicity boxes, and choose the type of wave function for open-shell molecules. The type is automatically set to ROHF if you set the multiplicity higher than 1. The maximum multiplicity is 6.

If you want to calculate properties, select the appropriate options in the Molecular and atomic properties section. These options correspond to keywords given in Molecular Properties Keywords for NDDO Calculations: the SAR properties option corresponds to the SUPER keyword. Likewise, if you want to generate grids for the display of surfaces in Maestro, select the appropriate options in the Plot properties section. These options correspond to keywords given in Plot Keywords for NDDO Calculations for the Semiempirical NDDO module.

To specify other options, you can enter keywords in the Keywords text box. The keywords you can use with the Semiempirical NDDO module are described in Keywords Supported in the Semiempirical NDDO Module.

If you are running a calculation on a large structure (1000 or more atoms) with the MOPAC2012 program, you can add the MOZYME keyword to the Keywords text box to speed up the calculations, especially if you are running on biological systems, for which this keyword was designed.

In addition to a Maestro file, the output from the job includes output files (.out) for each structure. You can collect these files into a zip file by selecting Archive output files (the default).

To start the job, enter a name in the Job name text box and click Run, or click the Settings button, make settings in the Job Settings dialog box, then click Run. You can choose not to incorporate the results into the Maestro project, or to append new entries to the project. The output properties include the semiempirical energy as an entry property, and the atomic partial charges as atom-level properties.
