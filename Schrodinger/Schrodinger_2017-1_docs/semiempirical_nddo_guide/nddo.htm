Semiempirical NDDO Guide

Schrödinger provides access to programs for basic semiempirical NDDO capabilities. Two versions are available: the Semiempirical NDDO module is based on the public domain version of MOPAC 7.1 available at www.openmopac.net; and MOPAC2012 is the commercially available version of MOPAC, which requires a license. With these tools you can calculate heats of formation, optimized geometries, force constants, and a selection of properties as indicated below. The semiempirical NDDO programs can be run from Maestro or from the command line.

In the version that is based on the public domain version of MOPAC 7.1, we have changed some of the behavior. The use of some keywords has been explicitly blocked because the feature does not actually work correctly. The supported keywords are listed in this manual. Input files for commercial releases of MOPAC may not work with this module.

If you publish work that uses the semiempirical NDDO modules, please use the citation shown on our citation page.
