Starting the Substructure Conformational Search Job

1. Click the Settings button to open the Job Settings dialog box.

2. Choose Append new entries as a new group from the Incorporate option menu.
3. Enter SubsAuto in the Name text box.
4. Click Run to launch the job.

The sample files included in the distribution have conformational search variables defined only for the ligand.
