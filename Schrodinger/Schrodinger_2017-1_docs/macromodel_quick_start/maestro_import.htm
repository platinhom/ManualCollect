Importing the 1err Structure

To display an existing structure in the Workspace, you must import the structure into the current project. Follow the instructions below to import the 1err protein-ligand complex into the project.

1. Choose File → Import Structures.

The Import panel opens. It should display the common structure files in your working directory (Maestro, PDB, SD, MOL2). By default the import options are not shown, but you can display them by clicking Options.

You can also open the Import panel by pressing Ctrl+I (⌘I) in the main window or by choosing Table → Import → Structures in the Project Table panel.

2. Select 1err.pdb from the list of files.
3. Click Open.

The Import panel closes.

When Maestro imports a PDB file, problematic parts of imported structures, such as nonstandard functional groups, are colored orange, red, green, or blue. For this exercise, the protein structure is already correct. However, when you begin to work on other proteins, you may want to investigate and manually adjust marked portions.
