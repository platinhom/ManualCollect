Energy Calculation Exercises

Many types of energetic calculations are available using MacroModel. This section introduces the MacroModel energetic panels and basic energetic parameters. These exercises calculate the molecular mechanics energy of a structure in gas phase, then in solution phase.

Before starting the calculations, import the substituted thymine structure from Ecalc.mae, which you copied to your working directory in Copying the MacroModel Tutorial Files. If you have not copied these files, do so now. See Importing the 1err Structure for instructions on importing structures.
