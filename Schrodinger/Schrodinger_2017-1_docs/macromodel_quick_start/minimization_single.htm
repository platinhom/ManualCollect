Energy Minimization of a Single Structure

1. Choose Tasks → General Modeling → Minimization → Force-Field or Tasks → General Modeling → Minimization → Force-Field in the Task Tool.

2. Choose Workspace (included entries) from the Use structures from option menu.
3. Click the Settings button to open the Job Settings dialog box.

4. Under Incorporate, select Append new entries individually.
5. In the Name text box, type Mini.
6. In the Host list table, change the Use column value to 1 for the highlighted host.
7. Click Run to launch the job.

After job completion, the final minimized structure is incorporated into the project as a new entry. If you wish to change the default minimization setting, click the Mini tab.
