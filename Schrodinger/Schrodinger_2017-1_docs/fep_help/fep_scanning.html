FEP+ Scanning Panel

Set up new ligands by scanning through the hydrogens on a ligand structure, replacing them with selected small groups, and scanning ring carbons for heteroatom substitution. FEP+ jobs are run for each ligand substitution to order them by binding free energy.

To open this panel, choose Tasks → Free Energy Perturbation → FEP Scanning in the Task Tool.

Using the FEP+ Scanning Panel

To set up an FEP+ Scanning job:

  1. Import a ligand-receptor complex into the panel, from the Workspace, a file, or the Project Table.

    The complex must consist of a receptor and a single ligand. They can be in separate entries.

  2. (Optional) To restrict the places where substitutions are made in the ligand, pick the atoms in the Workspace where you want substitutions to occur, and select Use selected atoms only.

  3. Select the types of substitution you want to make, by checking the box for the group or atom.

  4. (Optional) Click View to view the structures and reports for the generated ligands.

    Here, you can also remove generated ligands from the list to be included in the calculations, either individually or by excluded all ligands for which problems were detected.

  5. Click Run to run the job with the default host and settings, or click the Job Settings button to choose the host and other job-related settings then run the job.

The results of an FEP+ Scanning job are in the same form as from an FEP Mapper job. You can use the FEP Mapper Analysis Panel to analyze the results of the job.

FEP+ Scanning Panel Features

  • Import receptor-ligand complex from controls
  • Receptor section
  • Ligand section
  • Use selected atoms only option
  • Select scanning type options
  • Total number of new ligands report
  • View button
  • Job toolbar
  • Status bar
Import receptor-ligand complex from controls

Specify the source of the receptor-ligand complex.

  • Project Table (selected entries)—Use the selected entries in the Project Table. The first entry that contains a receptor is chosen for the receptor. If there is more than one ligand selected, a list of ligands is shown in the Choose ligand section so you can choose the ligand to use.
  • Workspace—Use the contents of the Workspace as the source of the receptor-ligand complex. The Workspace must contain only one protein and one ligand.
  • File—Use the structures from the specified file. When this option is selected, the text box and Browse button are activated. Click Browse and navigate to the file you want to use. The file name is displayed in the text box when you click Open in the file selector. You can also enter the file name in the text box. The first protein structure is used as the receptor, and a list of ligands is presented so you can choose the ligand, if the file contains more than one ligand.
Receptor section

The title of the receptor is reported in this area. Next to the title, an icon is displayed that indicates the "health" of the receptor. If it does not pass all the tests of receptor health, the icon is on a button, which you can click to open a protein reliability report that gives details of the problems found. This is a simplified version of the Protein Reliability Report Panel.

Ligand section

This section displays the title of the ligand, and an option for the ligand atoms to use in the scan. If more than one ligand is included in the source of the complex, a list of ligands is given, and the section is entitled Choose ligand. You can choose one ligand from the list for the scan.

Use selected atoms only option

Select this option to restrict the atoms used in the scan to the ligand atoms that are selected in the Workspace. By default, all eligible atoms in the ligand are used in the scan.

Select scanning type options

Select the type of substitutions that you want to make on the ligand. The R Group options allow you to select groups that will replace hydrogen atoms on the ligand. The Heterocycles option allows you to substitute N for C in rings in the ligand.

Total number of new ligands report

The number of new ligands generated by the substitution of groups or atoms is reported here, and is updated as you change your selection.

View button

Opens the Preview Ligands for FEP+ Scanning dialog box, where you can view a list of the generated ligands and select the ligands to use in the scanning job. The ligands are identified by the substitution A->B, where A is the ligand atom and B is the new atom or group. The 2D structure is also given, and a report on issues with the ligand.

Job toolbar

Manage job submission. See Job Toolbar for a description of this toolbar.

Status bar

The status bar displays information about the current job settings for the panel. This includes the job name, task name and task settings (if any), number of subjobs (if any) and the host name and job incorporation setting. It also displays a message for a short time when you start a new job.

The status bar also contains the Help button, which opens the help topic for the panel.
