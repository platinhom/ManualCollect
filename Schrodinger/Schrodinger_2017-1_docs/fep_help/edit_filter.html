FEP+ — Edit Filter Dialog Box

Edit a property filter for the ligands in the FEP+ panel.

To open this dialog box, choose Create Filter from the Apply filter option menu in the FEP+ review panel, or click the Settings button (gear icon) next to this option menu and choose Edit.

Using the Edit Filter Dialog Box

A filter can consist of a set of criteria for values of one or more properties.

To set up a filter:

  1. Enter a name in the Filter name text box.

  2. Click Add Property to open a property selector.

  3. Choose the properties you want to filter on, and click Select.

  4. Set the minimum and maximum value of the properties using the text boxes, the arrow buttons, or the slider.

  5. Repeat from step 2 to add more properties to the filter.

  6. Click Save to save the filter and apply it.

A ligand must meet all of the criteria set in the filter (i.e. have a value anywhere from the minimum value to the maximum value for each property) to be included in the set shown in the main panel.

  • To delete a filter criterion, click its settings button and choose Delete.

  • To reset the range for a filter criterion, click its settings button and choose Reset.

Edit Filter Dialog Box Features

  • Filter name text box
  • Add Property button
  • Filter criterion controls
  • Filter information
  • Reset All Criteria button
Filter name text box

Specify the name of the filter in this text box.

Add Property button

Add a property to filter from. Opens a property selector, in which you can choose one or more properties.

Filter criterion controls

The controls for a property filter criterion consist of the property name; with two text boxes and a slider below, to set the minimum and maximum (inclusive) values of the property; and a settings button for removing the criterion or resetting it to its default values (the full range).

Filter information

The number of ligands that match the filter in its current state is reported here. It is updated for each change in the filter criteria.

Reset All Criteria button

Reset all filter criteria to their default ranges (the full range of values in the ligand set).
