Simulation Event Analysis Panel

In this panel you can analyze the output from a Desmond simulation and display the results of the analysis in various forms. The analysis consists of selection of properties that are extracted from the trajectory; these properties can then be exported and plotted, and statistics calculated for the properties.

To open this panel, choose Tasks → Classical Simulation → Molecular Dynamics Analysis → Simulation Event Analysis in the Task Tool.

Simulation Event Analysis Panel Features

  • Trajectory selection
  • Properties section
  • Data analysis section
  • Display section

Trajectory selection

At the top of the panel are tools for selecting the trajectory.

To import the trajectory from a file, click Browse. The file that you select is an output CMS file from a simulation, which has the extension -out.cms (or the compressed equivalents). If for some reason, the CMS file associated with the trajectory does not have this extension, you can choose Any CMS file from the Files of type option menu to look for othe CMS files.

You should ensure that the option For Desmond files, import only first structure is selected in the Preferences panel, under Projects – Importing Structures, otherwise you will get an "Invalid structure" error.

To use a trajectory that is already imported into the project, select the entry for the trajectory in the Project Table (the full system entry), and click Import from Project Table.

Energy tab

In this tab, you can specify the energy properties to analyze for a molecule that you pick. Several properties are listed for each energy type:

  • Self—energy of the picked molecule
  • Solute—energy of interaction of the picked molecule with the solute
  • Membrane—energy of interaction of the picked molecule with the membrane
  • Solvent—energy of interaction of the picked molecule with the solvent
  • Other—energy of interaction of the picked molecule with molecules other than itself, the solute, the solvent, and the membrane. Includes added ions.
Energy properties to monitor options

The energy properties available for analysis are:

  • Coulomb—Coulomb energy
  • van der Waals—van der Waals energy
  • Bond—bond energy
  • Angle—bond angle energy
  • Torsion—torsional energy

The total energy is always included in the energy properties.

Pick molecule for energy properties option

Select this option and pick the molecule in the Workspace for which you want energies to be analyzed. When you pick a molecule, properties for the selected energy types are listed in the Properties to analyze list, as well as the total energy properties.

H-Bonds tab

In this tab you can select the atoms or molecules for which you want to monitor hydrogen bonds. There are two choices: pick molecules for which intramolecular H-bonds are monitored, or pick two atom sets to monitor H-bonds between the sets.

Pick molecule for H-bond monitoring option

Select this option and pick the molecule in the Workspace for which you want intramolecular hydrogen bonds to be monitored. When you pick a molecule, Intra, Nonsolvent, Solute, and Membrane properties for the molecule are listed in the Properties to analyze list.

H-bonds between atom subsets section

In this section you can specify two subsets of molecules, between which H-bonds are monitored. You must include all the atoms that are used to define H-bonds: the H and the heavy atom it is attached to in one set, and the acceptor and the atom it is attached to in the other. Click the Select button for each subset to specify the atoms for that subset. This button opens the Atom Selection dialog box, in which you can make the selection. The ASL expression appears in the text box when you click OK in this dialog box. When you have specified both subsets, click Add to add the property to the Properties to analyze list.

RMSD tab

In this tab you can add the calculation of the RMSD of atoms relative to those in a particular frame or to a reference structure to the list of properties to evaluate.

RMSD reference section

In this section you specify the reference structure for the RMSD.

RMSD to frame option and box

Select this option to calculate the RMSD relative to a particular frame, and specify the reference frame in the box.

RMSD to reference structure option and Browse button

Select this option to calculate the RMSD relative to a reference structure. Click Browse to navigate to a Maestro file to use for the reference structure. The file name is displayed in the text box below.

Atoms for RMSD and fitting section

In this section you specify the atoms in the reference for which the RMSD is calculated, and the atoms that are used for superimposing the structures on the reference for each frame.

Predefined sets option and menu

Select this option to calculate the RMSD for predefined atom sets, and choose the set from the option menu.

Atom selection option and Select button

Select this option to calculate the RMSD for an atom set that you choose. Click Select and use the Atom Selection dialog box to define the atom set. The ASL expression is displayed in the text box to the right of the Select button.

Fit to atoms option and Select button

Select this option to define the atoms that are used for superimposing the structures in each frame to the reference. If no atoms are selected, all atoms in the reference are used for the superposition. Click Select to choose the atoms that are used to superimpose the frames in the Atom Selection dialog box. The ASL expression is displayed in the text box to the right of the Select button.

Add button

Click this button to add the property to the properties list.

Note that the RMSF is averaged over protein chains if the chain is not specified, so you should ensure that you specify the desired chain.

Measurements tab

In this tab you can set up measurements of distances, angles, dihedrals, plane angles, and radius of gyration.

Pick atoms for measurements option and Add button.

Select this option to pick atoms for a measurement. Picking two atoms defines a distance, three defines an angle, four defines a dihedral, and six defines an angle between two planes (defined by the first three and the second three atoms). When you have picked the atoms, click Add to add the measurement to the Properties to analyze list.

Radius of gyration section

Add the radius of gyration of a set of atoms around its center of mass to the properties to be calculated. To specify the atoms, select Use ASL, then click Select to select the atoms in the Atom Selection dialog box. Click Add to add the radius of gyration to the Properties to analyze list.

Properties to analyze list

Lists the properties that have been chosen for analysis. You can select multiple items on this list using shift-click and control-click. The selected items can be acted on with the action buttons and with the plotting and statistics buttons. The properties listed here are also known as "simulation analysis keywords"; they are used as input to the simulation analysis script. You can double-click on a property to view it or to edit it.

Action buttons

These buttons allow you to edit the Properties to analyze list, analyze the properties on the list, read and write the list, and read and write the results of an analysis.

  • Remove—Remove the selected properties from the list.

  • ResetClear the list.

  • Analyze—Run an analysis on the properties (keywords) that are selected in the list. Opens a dialog box in which you can specify a job name, then starts the job when you click OK.

  • Analyze All—Run an analysis on all properties (keywords) in the list. Opens a dialog box in which you can specify a job name, then starts the job when you click OK.

  • Read Keywords—Read a list of properties (keywords) into the list. The keywords are appended to the list, and duplicates are removed. Opens a file selector, in which you can navigate to and select the file, which has a .sat extension.

  • Write Keywords—Write the selected properties to a file. Opens a file selector, in which you can navigate to a location and name the file, which has a .sat extension.

  • Export Results—Write a Desmond analysis results file, after an analysis has been run. Opens a file selector, in which you can navigate to a location and name the file.

Data analysis section

In this section, you can create plots for the properties that have been analyzed. A plot is created for each property that is selected in the Properties to analyze list.

  • Analysis type option menu
  • Analyze button
Analysis type option menu

Choose the type of data analysis you want to perform:

Time series

Plot the selected properties as a function of time, when you click Plot. Each property is plotted in a new window, unless you select Multiple time series plots under Plot options

Histogram

Display a histogram of the property values, with the number of bins specified. The range of property values is divided evenly into the specified number of bins.

Multi-variable plot

Display a scatter plot of up to four variables, where the first two are represented on the axes, and the third and fourth are represented by symbol color and symbol size.

Statistics

Display the mean, median, standard deviation, and correlation matrix for the selected properties. Opens a panel in which the information is displayed in a table.

Heat Map

Display a correlation coefficient heat map for the selected properties.

Polar plot

Display polar plots of each of the selected properties as a function of time. Useful for plots of angles and dihedrals.

RMSF plot

Plot the RMSF as a function of residue number.

Analyze button

Prepare the plots or statistics and display them.

Display section

This section allows you to control some aspects of the interaction between the Workspace and plots, the plots, and the Workspace contents. The sections that are displayed in this section depend on the choices made from the option menu in the Data analysis section.

  • Time series options
  • Histogram options
  • Solvent display options
Time series options

  • Auto-zoom—Zoom in to the atoms represented by a property in the Workspace when a point is clicked on the time series plot for that property.

  • Auto-select—Select the atoms represented by a property in the Workspace when a point is clicked on the time series plot for that property.

  • Multiple windows—Plot time series for each selected variables in its own window.

  • Multiple plots per windows—Plot time series for all selected variables in the same window.

  • Multiple lines per plot—Plot time series for all selected variables in the same window.

Histogram options

In this section you can specify the number of bins to use in the histogram.

Solvent display options

These options can be used to control which solvent molecules are displayed in the Workspace. This is useful when you want to analyze solute-solvent hydrogen bonds, or to turn off water display.

  • Hide solvent—Do not show any solvent molecules.
  • Show solvent within—Show solvent molecules that are within the specified distance of any solute atom.
  • Show all solvent—Show all solvent molecules.
