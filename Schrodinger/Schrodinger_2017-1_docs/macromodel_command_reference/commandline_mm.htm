Running MacroModel from the Command Line

Running MacroModel from the command line gives you the ability to customize batch processing. In addition, some types of MacroModel calculations are not supported by the Maestro GUI. This chapter gives instructions on how to launch MacroModel jobs from the command line. On Windows, you can open a Schrodinger Command Prompt window from the Start menu, and use this window to run command-line jobs. See Running Schrödinger Software for more information.
