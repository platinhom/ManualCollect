MacroModel Python Scripts

The MacroModel distribution contains several Python scripts that can be used for running various jobs. These scripts are stored in $SCHRODINGER/macromodel-vversion/python. To run a script, use the following syntax:

$SCHRODINGER/run -FROM macromodel script-name [options]

For more information on using Python for scripting, see the Python API web page. See Scripting in Maestro for information on using scripts with Maestro.
