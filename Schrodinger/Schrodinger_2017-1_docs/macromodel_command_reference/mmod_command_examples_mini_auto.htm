MacroModel Example: Multiple Conformer Minimization With Automatic Redundant Conformer Elimination

Below are two examples of command files for energy minimization of multiple conformers with redundant conformer elimination. The first uses the AUTO opcode; the second uses the COMP opcode with arg1=0. Explanations of the opcodes used in the file that are essential to this example are given.

mult_auto-out.mae

 EXNB       0      0      0      0     0.0000     0.0000     0.0000     0.0000
 BDCO       0      0      0      0    89.4427 99999.0000     0.0000     0.0000
 CRMS       0      0      0      0     0.0000     0.2400     0.0000     0.0000
 DEMX       0      0      0      0    51.0000     0.0000     0.0000     0.0000
 MULT       0      0      0      0     0.0000     0.0000     0.0000     0.0000

MULT: Mandatory for this computation, due to the presence of the AUTO opcode.

AUTO: Automatic parameter assignment. Arg1 turns on AUTO only for the first conformer. Arg5 turns off torsional parameters, which are not needed for this minimization.
