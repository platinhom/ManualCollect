MacroModel Capabilities

MacroModel is the batch-mode molecular modeling computational facility for use with Maestro. It is designed to minimize the energy of one structure or a series of structures, to eliminate duplicate conformations, to do conformational searches, and to conduct molecular dynamics simulations, including free-energy perturbation methods and mixed-mode Monte Carlo/stochastic dynamics procedures. MacroModel jobs may be started from the Maestro GUI, or may be run from the command-line. Some MacroModel features are not available from the Maestro interface. The use of Maestro for running MacroModel jobs is described in the MacroModel User Manual. This manual attempts a complete description of MacroModel’s command-line facilities.

Note: MacroModel was originally the name for the graphical interface; the executable was called BatchMin (bmin). The original interface was replaced by Maestro, and MacroModel now refers to the software package that contains the bmin executable.
