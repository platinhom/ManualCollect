Running Field-Based QSAR from Maestro

Field-based QSAR models can be built and applied from Maestro in the Field-Based QSAR panel. See Field-Based QSAR Tutorial for a tutorial introduction.

To open the Field-Based QSAR panel, choose Tasks → Discovery Informatics and QSAR → 3D Field-Based in the task Tool.
