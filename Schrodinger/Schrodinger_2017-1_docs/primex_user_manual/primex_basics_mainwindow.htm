PrimeX Basics: The Maestro Main Window

At the top of the Maestro main window are the title bar and the menu bar, followed by the Selection toolbar. In the center is the Workspace, where structures and surfaces are displayed. At the bottom is the Footer bar, which contains. Several optional components can be displayed at the bottom of the window: the footer bar, which contains the Status bar and Workspace Configuration toolbar, the sequence viewer, the command input area,and the clipping planes window. Some of these are displayed by default. For PrimeX, it is useful to display the sequence viewer and the status bar, which you can do from the Window menu.

There are several shortcut menus in the main window, which you display by right-clicking and holding over an atom, an empty part of the Workspace, or the sequence viewer.

Maestro is capable of displaying structures in stereo, and can use perspective and fogging to enhance the stereo effect. For information on these capabilities, see Appearance Panel.
