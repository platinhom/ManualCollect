PrimeX Map Colors

To change the default map colors, click Set Default Map Colors. You can then choose colors for each of the map types in the Set Default Map Colors dialog box. Default map colors can also be set in the Preferences panel, in the Surfaces – Creation section. Colors for individual maps can be changed from the Manage Surfaces panel.
