Finding Binding Sites with SiteMap

SiteMap is a tool for finding potential binding sites for small-molecule ligands on a protein. It maps and scores regions on the protein surface that are likely to accommodate a ligand.

SiteMap can be run either from Maestro or from the command line. A SiteMap calculation typically takes a few minutes for proteins having up to 5000 atoms.

If you publish work that uses SiteMap, please use the citation shown on our citation page.
