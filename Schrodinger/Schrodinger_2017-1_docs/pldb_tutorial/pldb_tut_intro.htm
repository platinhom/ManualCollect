Searching the Protein-Ligand Database (PLDB)

The Protein-Ligand Database (PLDB) is designed to provide insight from protein-ligand complexes, validate unusual interactions from crystal structures, docking results, and design ideation. Access to the PLDB is done through a connection made to a server located centrally within your organization or hosted by Schrödinger.

This tutorial will explain how to get started with the PLDB, through a set of query examples that are linked below.
