Logging In to the PLDB

These exercises use the PLDB Search panel from Maestro. To do a search you must log in to the PLDB.

  1. If you haven't already done so, start Maestro.

  2. Open the PLDB Search panel from the Task Tool.

    You can do this by searching for PLDB or using the Browse tool to go to the Structure Analysis section.

    When you select PLDB search, a login dialog opens.

  3. Log in to the PLDB. Your administrator will provide you with the Server URL, User Name, and Password.

Once you have logged in, the history panel is displayed.

The Exit menu on the top-right corner of the screen allows you to close the PLDB, log in as another user or into another server, or log out of PLDB completely.

When you first log in, you will have no saved searches in the "My Searches" tab.

However, once you have run queries, the most recent four queries will be displayed, and you can page through older queries for review.
