Finding Serine Proteases by Sequence Similarity

In this exercise, we want to search for structures whose sequences have at least 80% sequence similarity with the serine protease, 1S0Q.

  1. Open the search form, and run a search for PDB Id 1S0Q. The results of this query should be available in under five seconds.

  2. Click the View Results button from the history panel to see the result.

  3. From the results panel, click the Import All Results to Project Table button to import your result into the Maestro workspace.

  4. With 1S0Q still selected in the Maestro workspace, click the New Search button from the history panel to start a new search.

  5. Scroll down to the Sequence Features section of the form. This is where you will be able to launch BLAST sequence similarity searches.

  6. From the Import sequence menu, choose From workspace.

    The Sequence box should be populated with the sequence for 1S0Q.

  7. In the Similarity Filter threshold box, enter 80. This will ensure that only sequences with at least 80% sequence similarity to the query are returned.

  8. Enter the title "1S0Q sequence similarity" for your search and click the Start Search button to submit it. This query finishes in less than five minutes. Hand-investigated structures from the result set of this query can then be used for homology modeling.
