rmsdcalc

This utility computes the RMSD between each structure in a given Maestro file and a corresponding reference structure from a second Maestro file. The correspondence between structures is done by title, so reference titles must be unique, and each structure for which the RMSD is sought must have a title that matches one of the reference structure titles. Note that RMSD cannot be computed between two structures unless they have the same connectivity. The syntax and options descriptions can be listed by running the command with the -h option. The file specifications are given in Table 1.

Table 1. File specifications for the rmsdcalc command

File Specification

Description

-screen screenFile

Maestro file (.mae, .maegz, or .mae.gz) containing the structures for which RMSDs are sought.

-ref refFile

Maestro file (.mae, .maegz, or .mae.gz) containing the reference structures.

-out {csvFile|maeFile}

Output file. If the extension is .csv, a comma-separated file is created with the title and RMSD for each structure in screenFile. If the extension is .mae, .maegz, or .mae.gz, a Maestro file is created with each structure from screenFile and the property r_rmsdcalc_RMSD.
