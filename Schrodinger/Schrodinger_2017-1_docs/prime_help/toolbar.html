Prime Structure Prediction Toolbar

The Prime Structure Prediction toolbar provides convenient access to commonly used structure prediction and display tasks. The names of the tasks are given in the button tooltips. The tasks can also be found on the Edit and Display menus. The buttons are described below.

Color Scheme (Display menu)
Select a color scheme for coloring sequences in the sequence viewer. If there is a structure in the Workspace associated with the sequence, it is also colored.
View Template Structure (Display menu)
Display the template structures in the Workspace. In the Find Homologs and Edit Alignment step, it is the selected structures that are displayed. In the Build Structure step, all templates carried into this step are displayed.
View SSA (Display menu)
Show or hide the secondary structure assignment (SSA) of the template in the Sequence Viewer.
Wrap Sequences (Display menu)
Switch between wrapping the sequences in the Sequence Viewer into multiple lines if they are longer than the display area, or displaying them as a single scrolling line.
Select (Edit menu)
Select a region of the sequence. The corresponding residues highlighted in the Maestro Workspace.
Crop (Edit menu)
Truncate the query sequence at either end. You can move the truncation point by clicking on the desired residue.
Edit SSP (Edit menu)
Edit secondary structure predictions. Highlight a region of the SSP, and then type either H for alpha-helix, E for beta-strand, or - for loop.
Slide Freely (Edit menu)
Edit the alignment between query and template. Selecting a residue and sliding to the right creates N-terminal gaps; selecting a residue and sliding to the left creates C-terminal gaps.
Slide As Block (Edit menu)
Edit the alignment between query and template by moving residues as blocks. Selecting a residue and sliding to the right creates N-terminal gaps. Selecting a residue and sliding to the left will slide all residues as a block to the left.
Add Anchors (Edit menu)
Set alignment constraints by setting anchors at residue positions. To remove anchors, click again on the same residue.
Lock Gaps (Edit menu)
Lock gaps, to prevent gaps from being collapsed during manual editing. Locked gaps are indicated with a -.
Unlock Gaps (Edit menu)
Unlock gaps, to allow previously locked gaps to collapse during manual editing. Unlocked gaps are indicated with a ~.
