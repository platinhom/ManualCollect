Structure Prediction - Job Options Dialog Box

Select host and login information for each type of job in the Prime Structure Prediction workflow.

To open this dialog box, click Job Options in any step of the Structure Prediction panel.

Using the Structure Prediction - Job Options Dialog Box

The Structure Prediction workflow involves running a number of jobs. The jobs are started by clicking buttons in the various steps. As some of the jobs in the Structure Prediction workflow can take some time, you might want to choose which host the job will run on, which you can do in this dialog box.

To make settings for a single job type:

  1. Select the row in the Jobs table for which you want to change the job settings.

  2. Choose the host from the Host option menu.

  3. If you have a different user name on the host you selected, enter the user name for that host in the Login text box.

  4. Click Selected.

To make settings for all job types:

  1. Choose the host from the Host option menu.

  2. If you have a different user name on the host you selected, enter the user name for that host in the Login text box.

  3. Click All.

Structure Prediction - Job Options Dialog Box Features

  • Jobs table
  • Job Options section
    • Host option menu
    • Login text box
    • Set for buttons
Jobs table

This table lists the job types for each step in the Structure Prediction workflow, giving the step name, the job type, the host, and the login (user name) for the job. The host and the login can be changed by using the controls below the table.

Job Options section

Use the controls in this section to set options for an individual job type or for all job types.

Host option menu

Select the host to use for the job type from this option menu. The menu is populated from the schrodinger.hosts file.

Login text box

Specify the user name to use for the host, if it is different from the one listed for the job type. This may be useful if you are launching jobs from Windows or Mac to Linux, for example, and your user name on Linux is different from that on Windows or Mac.

Set for buttons

Click All to set the host and login for all job types; click Selected to set the host and login for the selected job type. The Jobs table is updated with the new settings.
