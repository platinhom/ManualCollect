Centering the Display on Residue 167

Next, the display will be centered on residue 167.

1. Type Ctrl+F (⌘F) in the main window.

The Find toolbar is displayed.

2. From the Find option menu, select Residue number.
3. Enter A:167 in the Residue number text box.
4. Click N.

The display is centered on residue 167 and zoomed in.

5. In the Manage Surfaces panel, check the box next to the Display at most text box and change the value to 30, then press ENTER.
