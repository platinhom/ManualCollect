Refining the Loop from Residues 167 to 175

The next task is to refine the loop that was examined in Examining the Density Fit for p38 Kinase, using the results of the minimization just performed.

1. Click Refine loops in the Real-Space Refinement section of the PrimeX panel.

The Real-Space Refinement panel opens.

2. Ensure that the loop to be refined is specified by Chain: A, Start: 167 and End: 175.

These are the residues identified above as good “anchors”. The remaining residues in this loop, 175–183, have much better real-space R-factors than residues 168–174. Reducing the size of the loop also makes the calculation faster.

3. Under Build mode, ensure that Automatic - resolution > 2.5Å is selected.
4. Start the job with an appropriate entry title.

This job takes about 10 minutes. More than one structure is returned from loop fitting and are selected in the Project Table. There is some variation between the loop predictions, but they cover a different region of space from the original. You can use the display techniques in Step 1 through Step 11 to view the fit of the loop predictions to the map. To select multiple entries in the Project Table use control-click.
