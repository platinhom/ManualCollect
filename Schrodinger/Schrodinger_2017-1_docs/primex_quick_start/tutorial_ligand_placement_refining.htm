Refining the p38 Kinase Ligand

1. Include the first ligand pose in the Workspace.

If not all the atoms are displayed, choose All from the Display Only menu button on the Display Atoms toolbar.

2. Run a reciprocal-space coordinate minimization with an appropriate entry title, following the instructions in Reciprocal-Space Minimization for p38 Kinase.
3. Run a B-factor refinement with an appropriate entry title, following the instructions in B-Factor Refinement for p38 Kinase.
4. Repeat the coordinate minimization with an appropriate entry title.
