Use of Similarity in QikProp

QikProp uses similarity in property space to improve predictions of key properties, and to return similar molecules from a database. The similarity calculation uses a normalized Euclidean functional form with a proprietary set of per-property weights. Similarities range from a value of 0.0 (no similarity) to 1.0 (exactly identical). The two tasks are described in the topics listed below.
