Manage KNIME Workflows Panel

The Manage KNIME Workflows panel allows you to manage KNIME workflows or named macros that appear on the KNIME submenu of the Workflows menu in the main window.

Using the Manage KNIME Workflows Panel

To change the name of a workflow, edit the Name field directly in the table.

To change the order in which the items appear in the Workflows menu, select a workflow and click the up or down arrow buttons to move it in the list.

To delete or duplicate a workflow, select the workflow and click Delete or Duplicate.

To create submenus for the workflows, edit the workflow names, and use a colon to separate the submenu name from the menu item.

To see the changes you have made to workflow names and locations, click Apply, then choose Workflows → KNIME. When you are satisfied with the changes, click Close to close the Manage KNIME Workflows panel.

Note: Workflows that are placed in the mmshare-vversion/python/common directory of your installation and are referenced in a scripts.mnu file in this directory do not need to be installed. They automatically appear on the Workflows menu, in a separate section.

For information on creating KNIME workflows to use with Maestro, see Running KNIME Workflows from Maestro.

Manage KNIME Workflows Panel Features

  • Contents of the KNIME Workflows menu table
  • Delete button
  • Duplicate button
  • Description button
  • Arrow buttons
  • Apply button
Contents of the KNIME Workflows menu table

This table lists all the workflows currently installed and visible in the KNIME submenu of the Workflows menu. The Name field lists the name of the workflow as it appears in the Workflows menu, or the submenu and the name, separated by a colon. The Command field lists the command that is run when you choose the workflow from the KNIME submenu. The workflow is run via a Python script, for which the appropriate Maestro command is listed.

You can select a workflow and delete it or duplicate it. You can reorganize the workflows by selecting them and using the arrow buttons to move them up or down in the table. The order they appear in the table is the order they appear on the KNIME submenu.

Delete button

Select a workflow in the table and click this button to remove it from the Workflows menu.

Duplicate button

Select a workflow in the table and click this button to duplicate it. The duplicate workflow appears at the bottom of the table. You can then rename it, if you wish.

Description button

Select a workflow in the table and click this button to display its description in a dialog box. The button is not available if the workflow does not have a description or if no workflow is selected.

Arrow buttons

Select a workflow in the table and click these buttons to move it up or down in the table, and thus in the KNIME submenu.

Apply button

Click this button to apply your changes to the Workflows menu. You can then choose the Workflows → KNIME menu and view your changes.
