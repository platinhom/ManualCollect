Conformer Cluster Panel

Cluster a set of conformers using hierarchical agglomerative clustering based on the atomic or torsional RMSD of the conformers, and apply the clustering to produce a file for each cluster or a single file containing representatives of each cluster.

Using the Conformer Cluster Panel

If you already have an RMSD or other similarity matrix between your conformers, you can import this matrix to perform the clustering. The matrix must be in a CSV file that has both row and column labels that identify the conformers.

Use structures from option menu

Choose the source of the structures for clustering.

  • Project Table (n selected entries)—Use the entries that are currently selected in the Project Table. The number of entries selected is shown on the menu item. An icon is displayed to the right which you can click to open the Project Table and select entries.
  • File—Use the structures from the specified file. When this option is selected, the File name text box and Browse button are displayed. Click Browse and navigate to the file you want to use. The file name is displayed in the text box when you click Open in the file selector. You can also enter the file name in the text box. The allowed file types are: Maestro, PDB, SD.
Open Project Table button

Open the Project Table panel, so you can select or include the entries for the source of the structures.

File name text box and Browse button

If you chose File from the Use structures from option menu, enter the file name in this text box, or click Browse and navigate to the file. The name of the file you selected is displayed in the text box.

Generate RMSD matrix section

Set options for calculating the RMSD matrix to be used in the clustering, and run the calculation.

Cluster by options

Choose the quantity used to evaluate the RMSD matrix.

  • Atomic RMSD—Use the RMSD between corresponding atoms in the conformers. The Define comparison atoms controls are displayed when you choose this option.

  • Torsional RMSD—Use the RMSD between corresponding torsions in the conformers. Controls for defining the torsions (dihedrals) are displayed when you choose this option.

List of atoms or torsions for comparison

Displays the list of atoms or the list of torsions (given as a quadruple of atom numbers) that is to be used for comparing conformers. The list is populated by using the tools below the list. You can select rows in the list to delete using the Delete button (to the right), and you can delete all rows by clicking Delete All.

Define comparison atoms controls

Choose the atoms that you want to use to calculate the atomic RMSD.

Define torsions controls

Choose the torsions (dihedrals) that you want to use to calculate the torsional RMSD. There are two options for choosing the torsions:

  • Pick—Select this option to pick torsions or residues in the Workspace or use a predefined set of torsions. The menu provides the following options:

    • Atoms—Pick four atoms in the Workspace to define a torsion. When you have picked four atoms, the torsion is added to the list and you can continue picking to define more torsions.
    • Bonds—Pick three bonds in the Workspace to define a torsion. When you have picked three bonds, the torsion is added to the list and you can continue picking to define more torsions.
    • Residues—Pick residues in the Workspace to add all the torsions (backbone and side chain) in the residues to the list.
    • All (include terminal dihedrals)—Include all dihedrals in the list, including those to terminal atoms (such as methyl hydrogens)
    • All (exclude terminal dihedrals)—Include all dihedrals in the list except those to terminal atoms (such as methyl hydrogens)
    • All torsions along rotatable bonds—Include dihedrals in the list for rotatable bonds only. This option excludes ring dihedrals, for example.

  • Select—Select this option to choose protein dihedrals or use the Atom Selection Dialog Box to select atoms that define torsions. Select the options for the named protein dihedrals or click Select to open the Atom Selection dialog box.

Retain mirror-image conformers option

Treat mirror-image conformers (enantiomers) as separate structures in the clustering. If this option is not selected, enantiomers are treated as the same conformer (stereochemistry is ignored).

Perform structure equivalence test for large molecules option

Check that the structures are actually conformers. This is not done by default for structures with more than 300 atoms, to save time. If you want to perform the check on structures with more than 300 atoms, select this option.

Per cluster, output structure with options and menu

Calculate RMSD Matrix button

Run a job to calculate the RMSD matrix. A dialog box opens so you can name the job and submit it to a chosen host. The job name is used to name the RMSD matrix, which is written as a CSV file with row and column labels.

RMSD in place option

If using atomic RMSD to calculate the matrix, calculate it without superimposing the structures. If this option is not selected, a superposition is performed on the structures first. You might want to use this option if the conformers are already in the desired relative positions and do not need to be superimposed (e.g. for aligned proteins, docked poses).

Clustering section

Cluster the conformers using the calculated or supplied RMSD matrix. The controls in this section are unavailable until an RMSD matrix has been calculated or supplied.

Calculate tab

Set options for the clustering and run the job.

Linkage method option menu

Choose a linkage method for clustering from the following:

  • Complete—The distance between clusters is the largest distance between any pair of objects (one object from each cluster). This option produces compact, spherical clusters.

  • Average—The distance between clusters is the average distance between all pairs of objects in the two clusters.

  • Single—The distance between clusters is the smallest distance between any pair of objects (one object from each cluster). This option produces diffuse, elongated clusters.

Calculate Clustering button

Click this button to perform the clustering. When it is done, the results are used to populate the Results and Apply tabs.

View the results of the clustering in various plots.
