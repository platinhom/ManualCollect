Using Features Links

Sets Panel

The Sets panel is used to define sets of atoms that can be used in subsequent Maestro operations. It also allows immediate use of the sets to select atoms in the Workspace.

To open this panel, choose Select → Custom Atom Sets. You can also open this panel from the Quick Select common objects button, by choosing Custom Sets → Manage Sets.

Using the Sets Panel

To define a set:

  1. Click New.

  2. Enter a name or use the default name Setn (where n is the total number of sets that have been created using the default name).
    Once the set is created, it appears as the currently selected item in the list at the top of the panel.

  3. Use the standard picking controls to define which atoms belong to the set.

To select the atoms in the Workspace that match a set:

  1. Select the set in the sets list.

  2. Click Replace in the Workspace selection section.

To add atoms to the Workspace selection that match a set:

  1. Select the set in the sets list.

  2. Click Add in the Workspace selection section.

User-defined sets are automatically saved with the current project. To use the currently defined sets with another project, click Write to save them to a separate sets file. Then, when you open another project, you can click Read to read the sets file into that project.

In most cases in Maestro, a set can be specified wherever atom selection is required. It is also possible to use sets in ASL expressions with the set syntax. For example, if you define a set "alpha" that contains all the alpha carbons of a protein, you can use the following command:

coloratom color=green set alpha

to color all the alpha carbons green.

Sets list

Once a set is created, it appears as the currently selected item in the sets list at the top of the panel. The set definitions for the sets shown in the list can be written to and read from a .set file.

Name text box

Displays the name of the currently selected set. To rename the set, enter a new name in this text box.

Atoms for set section

After a set has been created, it can be defined by using the standard picking controls to select atoms for the set. If you select Markers, the atoms that are selected by the set are marked in the Workspace.

Apply marker offset

Displays set markers beside the atoms in the set, rather than on top of the atoms.

New button

Click to define a new set. Then enter a name in the dialog box, or use the default name Setn (where n is the total number of sets that have been created using the default name).

Delete button

Click to delete the currently selected set.

Read button

Click to read in previously written set files. This button opens a file selector so you can browse for the file you want to read.

Write button

Click to save the set definitions to a file for subsequent use. Opens a file selector (see the related topic list below) that allows you to either enter a name for the file or to select an existing file.

Note: All currently defined sets are written to one file. By convention, set files are expected to end in the .set suffix.

Workspace selection section

Click the buttons in this section to apply the current set to the Workspace selection. There are three actions that you can take:

  • Replace—Replace the Workspace selection with the atoms that match the current set.
  • Add—Add the atoms that match the current set to the Workspace selection.
  • Subtract—Remove the atoms that match the current set from the Workspace selection.
