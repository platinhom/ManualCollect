Features

Structure File Converter Panel

Read structures from a structure file in a chosen format, and write them out to another file in a different format.

Structure File Converter Panel Features

  • Input file text box and Browse button
  • File type option menu
  • Structures to convert options
  • Output file text box and Browse button
  • File type option menu
  • Append to existing file option
  • Convert button
  • Reset button
Input file text box and Browse button

Enter the name of the file that you want to read structures from in this text box, or click Browse and navigate to the file. If you choose a specific format from the File type option menu, the file extension can be omitted if you type in the file name.

File type option menu

Choose the file format from this option menu. The default is Determine from extension, which requires you to provide the file extension. If you browse for a file, the menu is set to the file format you choose; however it must be one of the allowed formats, Maestro, PDB, SD, SMILES, MOL2, or (for input only) MacroModel.

Structures to convert options

Specify the range of structures to convert. If you select All, all structures are converted from the input file. If you deselect All, you can speciy the range of structures to convert in the Range text boxes.

These options are not available until you have specified an output file.

Output file text box and Browse button

File type option menu

Enter the name of the file that you want to write structures to in this text box, or click Browse and navigate to the file. If you choose a specific format from the File type option menu, the file extension can be omitted if you type in the file name.

Append to existing file option

Select this option to append the structures to an existing file. Otherwise any existing file is overwritten. This option is only available if the output file already exists. You must make sure it has write permiision.

Convert button

Perform the file format conversion. The conversion is run immediately, not under job control.

Reset button

Reset all the panel settings to their defaults.
