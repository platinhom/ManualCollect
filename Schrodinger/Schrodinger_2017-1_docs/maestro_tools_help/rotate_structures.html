Rotate Structures Panel

This panel enables you to rotate a set of protein structures (and associated objects) using the same rotation for each structure. The rotation matrix is determined by aligning the first structure to a reference structure using the protein structure alignment tool.

Rotate Structures Panel Features

  • Use reference from option menu
  • File name text box and Browse button
  • Source of structures to rotate options
  • Watermap files option
  • Action buttons
Use reference from option menu

Select the source of the reference structure. The choices are Workspace and File. If File is chosen, then you can specify the file name in the text box below.

File name text box and Browse button

If File is chosen from the Use reference from option menu, enter the file name for the reference protein structure in the text box, or click Browse and navigate to the file.

Source of structures to rotate options

Select the source of structures to rotate. There are two options:

  • Selected entries—Rotate the entries that are selected in the Project Table.
  • File—Rotate the structures that are in the specified file. Enter the file name in the text box, or click Browse and navigate to the file.
Watermap files option

If the structures are from a WaterMap calculation, rotate the objects in the associated files as well as the structures.

Action buttons

Click Start to start the job that performs the rotations.
Click Reset to clear the settings in the panel, so that you can set up another job.
