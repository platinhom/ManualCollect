Rapid Torsion Scan Panel

This panel enables you to scan the orientations around a ligand bond or view rotamer states of a selected protein residue and see a breakdown of the energy for each orientation or state. No relaxation of the structure is done during the scan.

To open this panel, choose Edit → Rapid Torsion Scan

Rapid Torsion Scan Panel Features

  • Pick ligand bond to rotate option
  • Number of rotamers and Angle increment slider and text boxes
  • Recalculate button
  • Pick standard protein residue to sample option
  • Scanned object text
  • Plot button
  • Rotamer states table
  • Next and Previous buttons
  • Select Best State button
  • Restore to Original button
  • Reset button
Pick ligand bond to rotate option

Select this option and pick an atom in the Workspace to define the bond to rotate around. The atom should be on the end of the bond that is rotated. The bond is scanned at the interval specified by the Angle increment slider, which is 30° by default.

Number of rotamers and Angle increment slider and text boxes

This slider allows you to change the angle increment for the scan. The allowed increments are integer divisors of 360° from 1° to 30°. The increment and the number of rotamer states generated are displayed in the text boxes.

Recalculate button

Recalculate the energies along the torsional coordinate. If you change the number of rotamers, click this button to recalculate the energies at the new points.

Pick standard protein residue to sample option

Select this option and pick a protein residue for which you want to see energetic information on the rotamer states.

Scanned object text

Displays the identity of the protein residue or ligand bond. The ligand bond information gives the atom numbers and the direction to the rotated group.

Plot button

Open the Plot of One-Coordinate Scan panel to display an interactive plot of the energy against the torsional coordinate.

Rotamer states table

Lists the rotamer states for the protein or the ligand bond with their energy contributions, in kcal/mol. The columns can be sorted by clicking in the column header. The arrow displayed in the header shows the direction of the sort. The first row contains the original state.

#   State number
vdW+internal   Sum of van der Waals and internal energies.
electrostatic   Electrostatic energy.
WaterMap   WaterMap contribution to energy.
Total   Total energy, which is the sum of the contributions in the other columns.
Probability/Angle   Probability of the protein rotamer state, based on frequency of occurrence in known proteins, or angle around the ligand bond. The original rotamer state does not have a probability value, because the probability is derived from a data set of rotamer states and the original state might not be in this set. The heading changes depending on whether a protein or a ligand is scanned.
Next and Previous buttons

Select the next or previous rotamer state in the table.

Select Best State button

This button selects the rotamer state in the table that has the lowest total energy.

Restore to Original button

Restore the structure to the original rotamer.

Reset button

Reset the panel to its default (clear all information).
