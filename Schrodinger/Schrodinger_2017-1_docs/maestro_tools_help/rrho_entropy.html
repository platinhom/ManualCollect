RRHO Entropy Panel

Calculate the change in translational, rotational, vibrational, and total entropy of each ligand upon binding to a receptor.

Using the RRHO Entropy Panel

The entropy calculations are done with MacroModel in the rigid rotor harmonic oscillator (RRHO) approximation. The output is a CSV file or a Maestro file with the entropy values as properties. Conformational entropy is not calculated, as this would require sampling of the energy landscape of the ligand in solution and in the complex to obtain the partition function.

For each ligand, two MacroModel RRHO jobs are run. The first is done on the ligand-receptor complex, with the ligand as the substructure (SUBS) and receptor residues within a specified radius of the ligand as frozen atoms (FXAT). These residues are used to calculate the potential field around the ligand but are not themselves included in the entropy calculation. The second calculation is done on the ligand alone. When the jobs are complete, the differences between the ligand-receptor and ligand entropy values are tabulated and stored as properties in the output structure file.

The input structure set must have the receptor as the first entry and the ligands as the remaining entries ("pose viewer" format for a file). The job can be distributed over multiple processors. The properties that are calculated are labeled TdS trans, TdS rot, TdS vib, and TdS total in Maestro, and are given in_kcal/mol at 300K.

Use structures from option menu

Choose the source of the structures for the current task. The structures must have the receptor as the first entry and the ligands as the remaining entries.

  • Project Table (n selected entries)—Use the entries that are currently selected in the Project Table. The number of entries selected is shown on the menu item. An icon is displayed to the right which you can click to open the Project Table and select entries.
  • File—Use the structures from the specified file. When this option is selected, the File name text box and Browse button are displayed. Click Browse and navigate to the file you want to use. The file name is displayed in the text box when you click Open in the file selector. You can also enter the file name in the text box.
Open Project Table button

Open the Project Table panel, so you can select or include the entries for the source of the structures.

File name text box and Browse button

If you chose from the Use structures from option menu, enter the file name in this text box, or click Browse and navigate to the file. The name of the file you selected is displayed in the text box.

Force field option menu

Choose the force field to use in the calculations of the entropy.

Radius text box

Set the radius for the shell of frozen receptor atoms. Residues with any atom that is within this distance of any ligand atom are included in the shell. These atoms are included in the MacroModel calculations to create a potential field around the ligand, but are not part of the entropy calculation itself; the rest of the protein is ignored.

Write CSV file option

Write the entropy values to a comma-separated value (CSV) file. The file is named using the job name as the file stem.

Job toolbar

Manage job submission. See Job Toolbar for a description of this toolbar.

Status bar

The status bar displays information about the current job settings for the panel. This includes the job name, task name and task settings (if any), number of subjobs (if any) and the host name and job incorporation setting. It also displays a message for a short time when you start a new job.

The status bar also contains the Help button, which opens the help topic for the panel.
