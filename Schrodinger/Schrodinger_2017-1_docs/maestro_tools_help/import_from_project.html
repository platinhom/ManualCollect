Using Features

Import Structures from Project Panel

Import structures from one project into another project. The entries to be imported can be selected in a table, which can show the entry properties.

Using the Import Structures from Project Panel

To import structures from one project into another project:

  1. Choose the source of the project to import from, from the Import from option menu.

    You can choose the current Maestro project, or an external project (choose File). If you choose File, click Open to locate the project and open it.

  2. If you want to display the project properties, select Show properties.

    This can be useful for selecting entries to import, as you can sort the table by property.

  3. Choose the source of the project to import into, from the Import to option menu.

    You can choose the current Maestro project or an external project (choose File). If you choose File, click Open to locate an existing project and open it, or click New to create and name a new project.

  4. Select entries from the source project in the table on the left.

    You can use the Selection buttons in this task. Selection of multiple entries can be done with shift-click and control-click.

  5. Click the right arrow button to add the selected entries to the list of entries to import.

  6. Repeat the entry selection and transfer as many times as needed, to build up a list of entries to import.

  7. (Optional) If you need to remove entries from the list, select them and click the left arrow.

    You can use the Sort by options to sort the list, and use the selection buttons to select entries in the list, as well as the usual selection actions (shift-click, control-click).

  8. Click Import.

    The entries on the list are imported into the destination project.

From section

In this section you choose a project and select the entries that you want to import into the other project.

Import from option menu

Select the source of the project to import from. You can choose Current Maestro project or File. If you choose File, the Open button is activated.

Open button

Open the source project. Opens a project selector, in which you can navigate to and select the project.

Show properties option

Show the project properties in the entry table. This allows you to sort the entries by property to help in selecting the entries to import.

Project text

Displays the name of the source project.

Selection buttons

Use these buttons to select all entries in the entry table (All), clear the selection (None), or invert the selection (Invert). The selected entries can be added to the list of entries to be imported by clicking the right arrow button.

Entry table

This table lists all the entries in the source project. The first three columns show the entry title, the entry ID, and the row number in the project (Project Order). If you selected Show properties, the project properties are also shown. You can click on the header of any column to sort the table by the property shown in that column. A second click reverses the order of the sort. You can select multiple rows in the usual way with shift-click and control-click.

Structure display area

This area displays the 2D structure of an entry in the table when the pointer is resting on the entry row.

Transfer buttons

Use these buttons to move entries to the list of entries to be imported (in the To section), or move them out of the list. To move entries to the list, first select entries in the entry table of the From section, then click the right arrow. To remove entries from this list, select them and click the left arrow.

To section

In this section you choose the destination project and construct a list of entries to import.

Import to option menu

Select the source of the project to import into. You can choose Current Maestro project or File. If you choose File, the Open button is activated.

Open button

Open the destination project. Opens a project selector, in which you can navigate to and select the project.

New button

Create a new project to import the structures into. Opens a project selector, in which you can navigate to a location and name the project.

Project text

Displays the name of the destination project.

Selection buttons

Use these buttons to select all entries in the list of entries to be imported (All), clear the selection (None), or invert the selection (Invert). The selected entries can be removed from the list of entries to be imported by clicking the left arrow button.

Sort by options

Sort the list of entries by one of four criteria:

  • Project—Sort by the order in the source project.
  • Title—Sort by title.
  • ID—Sort by entry ID in the source project.
  • Project—Sort by the order in which the entries were added to the list.
Entry list

This list shows the entries that are staged for import into the destination project. You can select entries in this list and click the left arrow button to remove them from the list.

Structure display area

This area displays the 2D structure of an entry in the list when the pointer is resting on the entry.

Import button

Click this button to perform the import.
