R-Group Analysis

If you have a set of structures that are built on a common or similar scaffold, by attaching different groups at one or more points on the scaffold, you might want to see the properties of the structures as a function of the groups at the various attachment points. The R-Group Analysis facility provides tools for identifying the scaffolds, the attachment points, and the R groups at each point, and displaying the R groups and property information in various forms. These tools are described in the topics listed below.
