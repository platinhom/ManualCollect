Custom Field Style Dialog Box

Select the fields that you want to use in the field-based QSAR model from a list of available fields.

To open this dialog box, choose Custom from the Field style option menu and click Edit in the Build Field-Based Model dialog box.

Custom Field Style Dialog Box Features

  • Built-in fields list
  • Import Feature Definitions button
Built-in fields list

Select the fields that you want to use in the field-based QSAR model from this list. Use shift-click and control-click to select multiple items.

Import Feature Definitions button

Import a set of feature definitions that are used for hydrophobic, acceptor, and donor fields. Opens a file selector, in which you can navigate to and select a Phase feature definition file (.def).
