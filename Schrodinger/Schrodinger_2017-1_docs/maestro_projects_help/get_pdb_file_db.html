Get PDB File Dialog Box

This dialog box allows you to obtain a protein from either the PDB web site or a local copy of the PDB database and import it into Maestro.

If you plan to use the structure for modeling, you should prepare it with the Protein Preparation Wizard first. The Protein Preparation Wizard panel also has a facility for obtaining a protein structure from a local database or the web site.

Get PDB File Dialog Box Features

  • PDB ID text box
  • Chain name text box
  • Auto option
  • Retrieve from local installation option
  • Download from Web option
PDB ID text box

Enter the 4-character PDB code into this text box for the desired protein. You can use upper case or lower case for the code.

Chain name text box

If you only want to import a single chain, enter the single-letter chain name in this text box. To extract the chain with a blank name, use " "

Auto option

Retrieve the protein structure from a local copy of the PDB database, and if that fails, download the protein structure from the RCSB web site.

Retrieve from local installation option

Retrieve the protein structure from a local copy of the PDB database. The possible locations for this copy are specified by environment variables. The structure is retrieved from the first location in the following list that has a valid PDB database:

  • $SCHRODINGER_PDB
  • $SCHRODINGER_THIRDPARTY/database/pdb
  • $SCHRODINGER/thirdparty/database/pdb
Download from Web option

Download the protein structure from the RCSB web site.
