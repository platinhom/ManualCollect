2D Viewer - Select Properties Dialog Box

This dialog box allows you to choose the properties that are displayed in the table cells of the 2D Viewer display area.

2D Viewer - Select Properties Dialog Box Features

  • Available properties list
  • Selected properties list
  • Select All button
  • Deselect All button
  • Show family option menu
  • Sort alphabetically option
  • Filter text box
  • Display property names option
  • Atom-level annotation property option menu
Available properties list

List of properties that are available for selection as displayed properties. To add one or more properties, select them in the list and click the arrow button or press the RIGHT ARROW key to move them to the Selected properties list.

Selected properties list

Lists the properties that are selected for display.
To remove properties from this list, select them and click the arrow button to the left or press the LEFT ARROW key to move them back to the Properties list.
To change the position of a property in the list, select it and click one of the arrow buttons to the right to move it up or down. These buttons are only available when you have a single item selected in the list.

Select All button

Select all the properties in the Properties list, and move them to the Selected properties list.

Deselect All button

Select all the properties in the Selected properties list, and move them to the Properties list.

Show family option menu

Choose a family to show only those properties that belong to this family in the Properties list. The option menu lists all families that are available in the Project Table. The family is listed in parentheses after each property in the list.

Sort alphabetically option

Display the properties in the Properties list in alphabetical order. The default order is the Project Table order.

Filter text box

Show only those properties that match the filter.

Display property names option

Display the property names as well as the property values. The name is followed by a colon and a space, then the property.

Atom-level annotation property option menu

Display the selected property for each atom instead of the atomic symbol. The property is displayed at the position of the atom and is displayed for all atoms, including carbon, and hydrogen if the Show all hydrogens preference is selected. You must click Update in the 2D Viewer panel after choosing a property to see the property in the display.
