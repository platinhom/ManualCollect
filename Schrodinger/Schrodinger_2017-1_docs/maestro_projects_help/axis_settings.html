Axis Settings Dialog Box

In this dialog box you can make settings that control the appearance of the axis labels in the Scatter Plot panel.

Axis Settings Dialog Box Features

This dialog box has three tabs, X, Y, and Color Scale. These tabs share a common set of controls, which are described below with any exceptions.

  • Axis label text box
  • Label font size option menu
  • Axis limits section
  • Axis ticks section
  • Tick label decimal places section
  • Tick label font size option menu
Axis label text box

Enter the label to be used for the axis in this text box. The default is the property name.

Label font size option menu

Choose the point size for the label font from this option menu. The default is 10.

Axis limits section

In this section you can select an option for determining the axis limits. This section is not included in the Color Scale tab. The two options are:

  • Automatic—Determine the axis limits from the range of values for the axis. Some rounding is done to decrease the number of significant digits in the end points.
  • Manual—Set the axis limits in the Minimum and Maximum text box. The defaults are the automatically determined limits.
Axis ticks section

In this section you can select an option for determining the number of tick marks on the axis. The options are:

  • Automatic—Determine the number of tick marks from the axis range to give a reasonable appearance.
  • Number of ticks—Specify the number of tick marks to use, including the ends of the range. The axis range is divided by this number minus one to determine the increment.
  • Tick increment—Specify the increment to use for the tick marks. The number of ticks is determined by the range and the increment.
Tick label decimal places section

In this section you can select an option for determining the number of decimal places used on the numerical labels for the tick marks. The two options are:

  • Automatic—Determine the number of decimal places from the tick increment.
  • Manual—Set the number of decimal places in the text box.
Tick label font size option menu

Choose the point size for the tick label font from this option menu. The default is 8.
