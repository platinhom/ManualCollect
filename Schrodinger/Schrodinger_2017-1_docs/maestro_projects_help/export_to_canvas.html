Features

Export to Canvas Dialog Box

In this dialog box you can export the selected entries to a Canvas project, and start Canvas with this project loaded. The dialog box allows you to set options for the Canvas project and disposition of structures that are to be exported. When you click OK, a file selector opens, in which you can navigate to and select the project. On clicking Save in the file selector, Canvas is started.

To open this panel, choose Table → Export → To Canvas

Export to Canvas Dialog Box Features

  • New project option
  • Existing project option
  • Append structures option
  • Replace existing structures option
New project option

Select this option if you want to create a new Canvas project for the selected entries.

Existing project option

Select this option if you want to export the selected entries to an existing Canvas project. When you do, the options for the disposition of the structures become available.

Append structures option

Append the structures to the existing Canvas project.

Replace existing structures option

Replace the structures in the Canvas project with the selected entries.
