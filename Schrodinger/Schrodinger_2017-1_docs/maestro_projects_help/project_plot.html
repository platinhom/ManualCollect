Using Features Links

Project Plot Panel

Create and manage scatter plots and histograms of entry properties.

To open this panel, choose Tasks → Workspace and Project Table Operations → Advanced Project Table Plotting.

Using the Project Plot Panel

Plots are created for the entries that are selected in the Project Table, so you should select the desired entries before creating new plots.

When a plot is created, you can save the settings for the plot as a named template, and use them for other plots. The templates are stored in your Maestro user resources directory.

Project Plot Panel Features

  • Plot list
  • Scatter Plot button
  • Histogram button
  • 2D Histogram button
  • From Template button
  • Delete button
  • Show button
  • Hide button
  • Rename button
  • Tile button
Plot list

Lists the plots created with this panel. You can select multiple plots in the list to delete, show, hide, or tile.

Scatter Plot button

Create a new scatter plot, from up to 4 entry properties. Opens the Scatter-N Panel.

Histogram button

Create a new histogram from a single entry property. Opens the Histogram-N Panel.

2D Histogram button

Create a new 2D histogram from a two entry properties. Opens the 2DHistogram-N Panel.

From Template button

Create a new plot from a template. Opens the Select a Template dialog box, which lists the available templates. Choose a template from the list and click OK. The relevant plot panel then opens with the settings from the template.

Delete button

Delete the selected plots. The data for these plots is cleared.

Show button

Show the selected plots.

Hide button

Hide the selected plots. You can also hide a plot by closing the plot window. The plot data is retained even though the window is not visible.

Rename button

Rename the selected plot. This button is only available when a single plot is selected.

Tile button

Tile the selected plot panels in a grid that covers the screen.
