Vibration Panel

This panel provides controls for the animation of vibrational frequencies in the Workspace.

Using the Vibration Panel

This panel controls the display of the vibrational motion of the atoms in the Workspace in a particular normal mode. You can control the speed and amplitude of the displayed motion with the Amplitude and Speed sliders. You can switch to a new mode and you can rotate the molecule while the animation is occuring. You can stop animation by pressing ESC.

The vibrational data come from a Jaguar frequency calculation and are read from a .vib file, which contains the normal modes and frequencies. The .vib file is automatically incorporated into the project at the end of a frequency calculation, and a V button is added to the Title column in the Project Table for the incorporated entry.

If you import a structure for which a frequency calculation was performed, you can also import the data, by choosing Table → Import → Vibrational Data. If the structure that you imported does not have an associated .vib file, you can import the frequency data from the Jaguar output file.

The vibrational animation performed with the controls in this panel displays a harmonic oscillation of the atoms in the Workspace for a single vibrational mode. There is no information about anharmonicity or curvature of the coordinates in this display, so you should beware of drawing any structural inferences from large amplitude motions.

Vibration Panel Features

  • Entry text box
  • Choose button
  • Vibration table
  • Animate option
  • Amplitude text box
  • Speed slider
Entry text box

This noneditable text box displays the entry ID and title of the entry chosen for vibrational animation.

Choose button

Click this button to open an entry selector in which you can choose an entry for vibrational animation. This button allows you to change the entry after opening the panel.

Vibration table

This table lists the frequencies in cm-1 of the molecule. You can select a single frequency in the table to animate, and you can select a new frequency while the animation is in progress.

Animate option

This option starts and stops the animation.

Amplitude text box

This text box specifies the scaling factor for the amplitude of the vibrational motion that is displayed in the animation.

Speed slider

This slider allows you to adjust the speed of the animation.
