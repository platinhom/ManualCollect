Renaming Project Entries and Groups

Warning: Do not use double quotes in an entry name.

To rename a project entry:

  1. If the entry is not selected, select it.

  2. Click in the Title table cell and edit the name.

  3. Press ENTER or click somewhere other than the table cell.

To rename an entry group:

  1. Double-click in the Title table cell, or right-click in the Title table cell and choose Edit Title.

  2. Edit the text.

  3. Press ENTER or click somewhere other than the table cell.

To assign the same title to several project entries:

  1. Select the desired entries.

  2. Click in the Title table cell for one of the entries and edit the name.
