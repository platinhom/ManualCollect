Using Features

Manage Custom Property Families Dialog Box

In this dialog box you can create, edit and delete custom property families. Property families are also created by applications; these families cannot be changed.

To open this panel, choose Property → Custom Families

Using the Manage Custom Property Families Dialog Box

Some procedures are given below for creating and editing custom property families. Changes that you make to a property family are applied immediately: there is no confirmation for changes to the properties included, or for deleting a family. You should therefore take care to make a copy of any family that you do not want to change or delete accidentally.

The primary property families for each application are custom families, which you can change if you want. To restore the default definitions of these families, click Reset Primary Families. You can change the primary property families from the Tree view of the properties by dragging properties between the primary and secondary sets. To open the Tree view, choose Properties → Tree or click the Tree toolbar button.

Property family definitions are stored in the file custom_property_groups in your user resources directory. Thus, they are not tied to a particular project, but are part of your Maestro customization.

To create a new property family from scratch:

  1. Click New.

    The New Family dialog box opens.

  2. Enter a name for the property family and click OK.

    The dialog box closes and the family name is displayed in the Custom property families list.

  3. Select the properties that you want to add to the family in the Available properties list.

    To select the properties, you can use shift-click and control-click as usual; you can choose a subset of the properties from the Subset option menu; and you can filter them by entering an expression in the Filter text box.

  4. Click the arrow button that points to the right (the upper one) or press the RIGHT ARROW key.

    The properties you selected are transferred to the Properties in family list.

  5. Repeat the selection and transfer steps until you have all the properties you want in the family.

To create a new property family from an existing family:

  1. Select the existing family in the Custom property families list.

  2. Click Copy.

    The Copy Family dialog box opens.

  3. Enter a name for the property family and click OK.

    The dialog box closes and the family name is displayed in the Custom property families list.

  4. Select the properties that you want to add to the family in the Available properties list, and click the right arrow button or press the RIGHT ARROW key.

  5. Select the properties that you want to remove from the family in the Properties in family list and click the left arrow button.

To change an existing property family:

  1. Select the existing family in the Custom property families list.

  2. Select the properties that you want to add to the family in the Available properties list, and click the right arrow button.

  3. Select the properties that you want to remove from the family in the Properties in family list and click the left arrow button or press the LEFT ARROW key.

Note that the changes to the family are immediate: there is no confirmation.

To delete a property family:

  1. Choose the family in the Custom property families list.

  2. Click Delete.

    The family is deleted immediately.

Manage Custom Property Families Dialog Box Features

  • Custom property families list
  • Action buttons
  • Available properties list
  • Subset option menu
  • Filter text box
  • Sort options
  • Properties in family list
  • Arrow buttons
Custom property families list

This list shows the property families that you have defined. You can select a single family from the list to operate on.

Action buttons

These buttons allow you to perform various operations on custom property families.

  • New—Create a new custom family. Opens the New Family dialog box, in which you can specify the name of the family. You must provide a name that is not already in use.

  • Copy—Make a copy of an existing custom family. Opens the Copy Family dialog box, in which you can specify the name of the family you want to copy to. If you want to overwrite an existing property family, you can provide a name that is already in use. The family is overwritten when you confirm the action. Otherwise, the Copy Family dialog box is redisplayed so that you can choose another name for the copy. The default name has _copy appended to the source family name.

  • Rename—Rename an existing custom family. Opens the Rename Family dialog box, in which you can specify the new name of the family. If you want to overwrite an existing property family, you can provide a name that is already in use. The family is overwritten when you confirm the action. Otherwise, the Rename Family dialog box is redisplayed so that you can choose another name for the family.

  • Delete—Delete the selected property family. The action is immediate: no confirmation dialog box is displayed to confirm the action.

  • Reset All—Reset all property families to their defaults. This action deletes any property familiies that you have defined.

  • Reset Primary Families—Reset the primary property families to their defaults. This action retains any property families that you have defined, but discards any changes you have made to the primary families.

Available properties list

List of properties that are available for inclusion in the family. The list can be restricted by using the Subset option menu or by filtering, or both. You can select items with click, shift-click and control-click. You can select all items with Ctrl+A (⌘A).

Subset option menu

Restricts the range of properties shown in the Available properties list. This option menu has two items in addition to the existing families: All properties, and Properties shown in Project Table.

Filter text box

Filter the contents of the Available properties list so that only the items that match the expression in the text box are displayed. You can use the wildcard characters * for zero or more characters and ? for a single character, and square brackets to enclose a range of characters. The filtering is done as soon as you type something in the text box. To remove the filter, clear the text box. If a subset is selected, the filter is applied to the subset.

Sort options

Sort the Available properties list, either alphabetically, or by the existing order in the Project Table. The sort is performed when you select the desired option.

Properties in family list

List of properties that are in the specified property family. You can select items with click, shift-click, and control-click. You can select all items with Ctrl+A (⌘A). Adding properties to this list or removing propertis from this list updates the family definition immediately.

Arrow buttons

These buttons are used to transfer properties between the Properties in family list and the Available properties list. The upper button (pointing right) moves the properties that are selected in the Available properties list to the Properties in family list; the lower button (pointing left) moves the properties that are selected in the Properties in family list to the Available properties list. You can also use the RIGHT ARROW and LEFT ARROW keys.
