Importing Structures

When running QikProp from Maestro, you can use as input for a job either the structures in the Workspace or entries that are selected in the Project Table panel. In this exercise, you will import structures into Maestro to use as QikProp input.

1. Choose File → Import Structures.

The Import panel opens.

2. If the import options are not displayed, click Options.
3. From the Create groups for option menu, choose None.
4. Select ligs.mae, and click Open.

The structures are imported as individual entries, and the first structure is displayed in the Workspace.

5. Click the Show Project Table button on the toolbar, or choose Window → Project Table.

The Project Table panel opens. The contents of the ligs.mae file are listed, and all 200 entries are selected.
