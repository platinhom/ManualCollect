Custom Chemistry Definitions for Covalent Docking

If you want to use a reaction other than the predefined reactions, you can define your own custom chemistry, by using keywords that you add to a file, one keyword per line. You can also use the corresponding command options to define a custom chemistry. However, storing the keywords in a file is more useful, as it allows reuse of the definitions. The procedure for creating custom chemistry definitions is described in the topics listed below.
