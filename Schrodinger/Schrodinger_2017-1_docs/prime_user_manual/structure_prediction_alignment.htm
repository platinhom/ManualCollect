PSP—Edit Alignment Step

In order to build a model structure of the query, the Build Structure step requires a satisfactory alignment between templates and query. The alignment of templates to query generated in the Find Homologs step is based only on sequence. This alignment usually needs some improvement. The Edit Alignment step allows you to improve alignments using various tools before building a model in the Build Structure step.

There are several ways to produce an alignment for use in the Build Structure step:

  • Accept the BLAST/PSI-BLAST alignment generated in the Find Homologs step and currently displayed in the sequence viewer.

  • Import an alignment.

  • Generate or import secondary structure predictions, then use the alignment program (STA) to create a new alignment.

  • Edit any of these alignments manually.

If you want to identify globally conserved residues to assist in the alignment process, but did not do so in the Find Homologs step, click Identify Globally Conserved Residues to run HMMER/Pfam. For more information on this action, see Searching for Homologs in PSP.

See sta—Single Template Alignment for technical details on the alignment program, including instructions for running it from the command line.

When you are satisfied with the alignment, click the Next button to proceed to the Build Structure step.
