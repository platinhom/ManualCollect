Saving Prime Runs With Different Templates

If you decide to try different templates, you can avoid overwriting your existing workflow by starting a new one before you navigate back to the Find Homologs step to choose a different homolog. Choose Save As from the File menu. This creates a copy of the current run, allowing you to select a new template and build on it. You can then compare results from each run.

If you are building a heteromultimer, you must model each unique chain of the multimer, and save a run for each chain. The runs are combined to build the final model.
