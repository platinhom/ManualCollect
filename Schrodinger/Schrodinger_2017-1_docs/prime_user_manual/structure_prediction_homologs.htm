PSP—Find Homologs Step

This step is used to search for homologs of the query sequence. These homologs can then be used as templates to build a model structure.

When you are satisfied with the template (or structurally aligned multiple templates) you have selected, you can proceed to the Edit Alignment step by clicking Next.

If there is an error, a message appears when you click Next to go to the next step. See Find Homologs Error Messages for a list of error messages for Find Homologs.
