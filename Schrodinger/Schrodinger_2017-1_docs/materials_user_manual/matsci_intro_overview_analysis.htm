Structure Analysis and Annotation with the Materials Science Suite

Several tools that can be used for analysis and annotation of common elements in a structure or set of structures are provided in the Materials Science interface:

  • R-group analysis—analyze the properties of a set of structures as a function of the groups at the various attachment points on a common scaffold (R-Group Analysis manual).

  • Scaffold decomposition—find common scaffolds in a set of molecules (Scaffold Decomposition Panel).

  • Ligand filtering—Filter a set of structures based on properties of the structures, such as functional group counts or physical properties (Ligand Filtering Panel).

  • Annotating monomers—Mark the heads and tails of a monomer unit for identification in other applications (Mark Monomer Head and Tail Panel).

  • Locating ring-spears—Find locations where a ring has a chain or another ring going through it (Locate Ring-Spears Panel).

Analysis tools are also provided for the output of various calculations that produce a graphical representation of the analysis—see Visualization Tools in the Materials Science Suite.
