The Maestro - Materials Science Interface

The Maestro - Materials Science interface is a customized form of the Maestro interface that is specially designed for materials science use. It inherits most of the capabilities of the Maestro interface, and it has features of its own.

If you prefer to use the standard Maestro interface, you can do so. Most of the capabilities of the Materials Science Suite are available from the Task Tool; however some features are only available in the Maestro - Materials Science interface.

You can open the Maestro - Materials Science interface as follows:

  • Windows: Double-click the Maestro - Materials Science icon on the desktop
  • Mac: Go to Applications → SchrodingerSuite2017-1 and double-click the Maestro - Materials Science icon
  • Linux: Use the command $SCHRODINGER/maestro -profile MaterialsScience

See Running Schrödinger Software for more information.

Figure 1:The Materials Science main window.

The main window opens with the following features displayed by default:

  • Menu bar. This is at the top of the window on Linux and Windows, and is the menu bar on a Mac.

  • Toolbar. This is below the menu bar on Linux and Windows, and is at the top of the window on a Mac. It has tools for selecting atoms, controlling the appearance of the structures in the Workspace, and building and editing structures. At the right is a button for information on what jobs are running, and a button for opening panels to perform tasks (the Task Tool).

  • Favorites Toolbar. This is below the main toolbar, and provides quick access to panels. You can add items by clicking the pale star next to any action in the Task Tool, and remove items by right-clicking and choosing Delete.

  • Workspace Navigator. This dockable panel is displayed on the left side of the main window. It displays a list of all the structures (“entries”) in the current Maestro project, and a list of all the structures in the Workspace with their substructures. See Workspace Navigator Panel for more information. You can undock it from the main window and redock it with the docking button.

    Many other panels are also dockable. You can change the docking behavior in the Preferences panel, which you open with Edit → Preferences, or Ctrl+, (⌘,) .

  • Workspace. This is the large black area that occupies the main part of the main window. It is where 3D structures are displayed, along with any associated objects such as surfaces and text labels.

  • Status bar. This bar is below the Workspace. When the pointer is not over an atom in the Workspace, the status bar gives information on the contents of the Workspace. When the pointer is over an atom, the status bar gives information on the identity of the atom. For more information, see Status Bar. The Status bar has added features for periodic structures: cell information and a Periodicity button, which are described below.

  • Workspace Configuration toolbar. On the right of the Status bar is a set of buttons for turning on and off various Workspace display features, and a button to open the Workspace Configuration toolbox. For more information, see Workspace Configuration Toolbox.

Other Workspace components can be displayed from the Window menu.

Some of the tools for biological systems are not present in the Materials Science profile:

  • Ribbons button on the Workspace Configuration toolbar
  • Crystal Mates and Alternate Positions buttons, under Workspace Toggles in the Workspace Configuration toolbox.
  • 2D Overlay and Sequence Viewer buttons, under Workspace Gadgets in the Workspace Configuration toolbox.
  • Ligand-receptor and Intra-ligand options for hydrogen bonds in the Interactions toolbox.

The Materials Science profile has additional features that are described in Workspace Tools for Periodic Structures.
