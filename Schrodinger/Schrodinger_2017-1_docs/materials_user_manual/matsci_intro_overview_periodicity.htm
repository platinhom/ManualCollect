Workspace Tools for Periodic Structures

In the Materials Science profile, Maestro has additional tools for working with periodic structures.

The Status bar is customized to provide information on periodic structures and access to tools for display of periodic structures.

  • Cell information. When a periodic structure is detected, cell information is displayed at the right end of the Status bar.

    • If a cell has not been built, the cell information shows the space group hash from the PDB CRYST1 Space Group property.
    • If a cell has been built, the cell information shows the dimensions of the cell.

  • Periodicity toolbox. When a periodic structure is detected, the Periodicity button is enabled.

    This button opens the Periodicity toolbox.

    The toolbox provides tools for display of periodic structures, allowing you to show or hide information on the bonding, show or hide symmetry-related atoms, and create a supercell structure.

    The four buttons are described below.

    Translate to First Unit Cell—translate all atoms in the system so that they are inside a single unit cell, which is the one represented by the box in the Workspace (if shown).
    Recalculate Connectivity—Recalculate the connectivity between the atoms after Workspace changes.
    Recalculate Bond Orders—Recalculate the bond orders between the atoms after Workspace changes. Bond orders are assigned if the asymmetric unit has less than 50 atoms, and is not for an atomic solid.
    Extents—Display replicas of the unit cell along the lattice vectors. You can set the number of replicas in the positive and negative directions for each lattice vector, in the text boxes. The replicas are not displayed until you click BUILD CELL.

    The first three buttons have a menu from which you can choose when the operation is applied.

    • Auto—apply the operation if needed.
    • Always—always apply the operation, even if not needed.

    The three buttons along the bottom of the toolbox do the following:

    • BUILD CELL—Create and display a supercell in the Workspace based on the Extents settings. The new atoms are virtual atoms: displayed but not added to the project entries.
    • REVERT TO ASU—Revert the display to show only the asymmetric unit. Extents are discarded as well as symmetry-related atoms in the first unit cell.
    • MAKE P1—Create a new project entry from the supercell built on the extents, with P1 crystal symmetry.

    If there are multiple entries displayed in the Workspace, the actions are applied to each entry separately.

  • Workspace Configuration toolbox. There are two extra buttons in the Workspace Toggles section, for displaying objects related to periodic structures.

    Bonds. Show the connectivity of the structure, displayed as single bonds. Two atoms are considered connected if the distance between them is smaller than the sum of their covalent radii plus an offset. No account is taken of bond order in the display. Clicking the button displays or hides the connectivity. There are some buttons for controlling the bonds:

    Update Bonds—update the connectivity by reapplying the distance criterion.
    Dynamic Bonds—update the connectivity after each change in the Workspace.
    Bond Options—opens a dialog box for changing the distance offset.

    Planes. Display up to 4 crystal planes, specified by their Miller indices. Opens the Crystal Planes panel, which allows you to set the Miller indices, color, and opacity for each plane. If more than one unit cell is displayed in the Workspace, you can display replicas of each plane by setting a range of displacements from the primary unit cell (Min and Max values), or move the planes parallel to the normal (Translation).
