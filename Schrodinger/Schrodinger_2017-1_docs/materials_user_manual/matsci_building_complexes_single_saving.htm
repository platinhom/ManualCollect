Saving the Organometallic Complex

When you have finished defining the complex, click one of the action buttons to save it, either to the Maestro project with Create Project Entry or to a file, with Write File.

By default, the complexes are cleaned up after building (using the same method as the Ligand Cleanup panel). You can deselect this option if you don’t want to clean them up, and subsequently use the Ligand Cleanup panel if you later decide you do want to clean them up. In any case, it is recommended that you run a geometry optimization with Jaguar before proceeding with any calculations on the complexes.
