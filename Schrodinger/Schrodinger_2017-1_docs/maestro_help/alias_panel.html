Using the Command Aliases Panel

Maestro is supplied with a number of standard aliases, a list of which is displayed in the Command Alias text box at the top of the Command Aliases panel. You can select an alias for editing by double-clicking it in the list. The name of the selected alias is displayed in the Alias Name text box, amd the corresponding command is displayed in the Alias Definition text box.

Once an alias has been defined, it can be used in place of the original command. To illustrate, the steps in assigning an alias for the coloratom command are given below.

  1. Open the Command Aliases panel.
  2. Click New to define an alias that is not on the standard alias list.
  3. Enter the name of the alias in the Alias Name text box. In this case we will use col. Click OK or press ENTER.
  4. Enter coloratom in the Alias Definition text box and press ENTER.
  5. Save your alias for use in subsequent Maestro sessions (if desired), by clicking Save. The defined alias is stored in the alias.cmd file located in the resource directory, $HOME/.schrodinger/maestro90.
  6. Test your new alias (if desired). For our example, enter the command col color=green all in the Commands text box. All the atoms in the Workspace are colored green.
