Epik pKa Prediction

Epik’s pKa prediction is based upon the Hammett and Taft equations. The implementation used largely follows that described in pKa prediction for Organic Acids and Bases (Perrin, D.D.; Dempsey, B.; Serjeant, E.P.; Chapman and Hall, London (1981)).
