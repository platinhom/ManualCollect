Define Actives and Inactives Dialog Box

Assign ligands to the active set or the inactive set for pharmacophore model development from multiple ligands, and optionally set the activities of the ligands.

To open this dialog box, click the Define button in the Actives / Inactives Split section of the Develop Pharmacophore Model panel.

Using the Define Actives and Inactives Dialog Box

The primary task in this dialog box is to assign ligands to the active set for pharmacophore model development from multiple ligands. You can also assign ligands to the inactive set, which will be used in scoring and can be used to create excluded volumes. A secondary task is to assign activities to the ligands.

There are two main ways to make the assignment:

  • Manual—assign the ligands to a set by changing the values in the Pharm Set column of the ligand table.To change the value for a single ligand, click the cell and choose from the menu. To change the values for multiple ligands, select them in the table, then right-click and choose from the shortcut menu.
  • By activity—assign the ligands to a set using thresholds on the ligand activities. This method requires activities, which you can set for single ligands by editing the table cells, or for all ligands by choosing a property from the Available properties list. To assign the ligands, choose the activity property type from the Define units option menu, set the thresholds and the relationship (above or below), then click Apply.

Selected Project Table entries table

This table lists the entries selected for development of a pharmacophore model. You can use this table to designate which entries are actives and which are inactives, and set the activity for the entries. You can select multiple rows in the table and apply an action from the shortcut (right-click) menu: set the Pharm Set property, modify the selection, or remove the ligands from the table.

Name Title of the project entry for the ligand. Noneditable.
Conformers Number of conformers for the ligand. Noneditable.
Activity Activity of the ligand. You can set the activity of a single ligand by editing the table cell. You can set the activity of all ligands by choosing a property from the Available properties list.
Pharm Set Indicates whether the ligand is in the active set, the inactive set, or neither. You can change the value for a single ligand by clicking in the table cell and choosing from the menu. You can change the value for the selected ligands by right-clicking and choosing from the shortcut menu. By default all ligands are assigned to the active set.
Required Matches A checked box indicates that the ligand is required to match all hypotheses in the pharmacophore model. The checkboxes can ony be checked for the active ligands. The requirement is only relevant when the number of actives required to match a hypothesis is less than the total.
Available properties list

Lists properties that are available for selection as the activity property of the ligands. Only real and integer properties are shown. Select a single property to populate the Activity column of the table with the values for the ligands.

Show family option menu

Prune the list of available properties to show only the properties for the chosen family.

Define units option menu

Set the type of the activity property by choosing from this option menu. This choice defines the units of the activity.

Active if activity is option menu and text box

Set the threshold for assigning ligands as actives in the table, and choose the appropriate relation for the activity property type from the option menu. All ligands whose activity meets the condition set are assigned to the active set. Choosing the relation changes the relation for the inactives as well.

Inactive if activity is option menu and text box

Set the threshold for assigning ligands as inactives in the table, and choose the appropriate relation for the activity property type from the option menu. All ligands whose activity meets the condition set are assigned to the inactive set. Choosing the relation changes the relation for the actives as well.

Apply button

Apply the criteria for assigning ligands to the active and inactive sets. Sets the value in the Pharm Set column according to the ligand activity thresholds.

Reset button

Clear all changes made in the panel and reset settings to the defaults.
