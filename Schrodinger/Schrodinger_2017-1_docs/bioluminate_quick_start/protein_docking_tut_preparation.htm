Preparing for the Protein-Protein Docking Exercise

As the jobs take 4 hours to run, you should first create a project for the jobs, so that you can close the BioLuminate interface and return to the project later to view and analyze the results, if you want.

1. If there is a project open, choose File → Close Project.

The previous project is closed and the Workspace is cleared. If you did not name the previous project, you can choose to save it at this time.

2. Choose File → Save Project As.
3. Navigate to a location, and enter a name in the File name text box, such as 1avx_docking_exercise.
