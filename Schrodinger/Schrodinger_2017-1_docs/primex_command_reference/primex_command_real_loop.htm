PrimeX Keywords for Loop Prediction

The following keywords are needed for loop predictions, in addition to the general real-space refinement requirements.

  • PRIMEX_TYPE LOOP_BLD
  • Loop prediction keywords, given in Table 1.

 

Table 1. Loop prediction keywords

Keyword

Description

USE_BIAS

Bias sampling of loops to existing loop structure. Conformational space is sampled more heavily around the existing structure. Can only be used if the loop is complete (no missing residues) Allowed values: yes, no.

LOOP_FILE

Name of file containing definition of loop. Must contain lines with the format chain0:resnum0 chain1:resnum1, specifying the beginning and end of the loop.

LOOP_0_RES_0

LOOP_0_RES_1

Beginning (_0) or end (_1) of loop. The value for these keywords is a residue specification in the format chainID:resnum, for example A:118.
