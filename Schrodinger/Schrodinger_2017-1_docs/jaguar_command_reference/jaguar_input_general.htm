General Description of the Jaguar Input File

The input file often begins with an optional line indicating the version number of Jaguar. The other parts of the input file are either single lines composed of options in capital letters followed by arguments on the same line; sections describing the molecule and the calculation, whose formats will be described later in this chapter; or comments.
