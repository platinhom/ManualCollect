The Basis Set File

The basis sets available for use in Jaguar appear in the file default.basis, in the standard data directories. Portions of this file are shown in this section; you might want to refer to them as you read the description of the file. Basis sets at the top of the file do not contain effective core potentials, and will be described first. The basis sets with effective core potentials will be described later.
