Summary of Ark Syntax

The PBF radii file is formatted according to the Ark syntax, which is described in brief in Ark Format Syntax Summary. The format is based on a keyword=value notation. Keywords can be composed of alphanumeric characters and underscores, and must not begin with a numerical character. Values can be arbitrary strings (including integer, fixed-point, and floating-point numbers), tables, and lists. A table is an ordered set of keyword=value pairs enclosed in braces, in the following form:

{key1 = value1 key2 = value2 ... keyN = valueN}

Lists are sequences of values of the form

[value1 value2 value3 ... valueN]

They can be used to assign values for an array, for example. Note that pairs of keywords and values in tables and values in lists are separated by spaces or new line characters, not commas. As values can be tables or lists, nested tables and lists are allowed. Empty strings are ignored. The # sign is used to start comments, which are also ignored.
