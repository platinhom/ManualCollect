Filtering by Lipinski’s Rule for VSW

To prefilter the ligands using Lipinski’s Rule of 5, select Prefilter by Lipinski’s Rule. Ligands that do not satisfy this rule are not docked. This option requires QikProp properties. If the input structure files do not have QikProp properties, select Run QikProp.
