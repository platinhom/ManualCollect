Filter and Select Dialog Box — Include/Exclude Tab

In this tab you can choose reagents for explicit inclusion or exclusion from the combinatorial library.

  1. Select an attachment position from the Position option menu.
  2. Select reagents from the Reagents list.
  3. Click Include to add the selected reagents to the Always Included list, or click Exclude to add the selected reagents to the Always Excluded list.

If you add a reagent to one list that is already in another list, it is removed from the other list. To remove reagents from a list, select the reagents in the list and click the Remove button immediately below the list.

Include/Exclude Tab Features

  • Position option menu
  • Reagents list
  • Always Included list
  • Always Excluded list
  • Include, Exclude, and Remove buttons
  • Filter text box
Position option menu

This option menu contains the names of the attachment positions. When you select a position, all reagents in the reagent file for that position are displayed in the All reagents list, and you can select them for inclusion or exclusion.

Reagents list

Contains the list of reagents for the selected position. The list can be filtered.

Always Included list

List of reagents that will always be included in the definition of the combinatorial library.

Always Excluded list

List of reagents that will always be excluded from (never included in) the definition of the combinatorial library.

Include, Exclude, and Remove, buttons

Include includes the selected reagents in the Always Included list.
Exclude includes the selected reagents in the Always Excluded list.
Remove removes the selected reagents from the relevant list.

If you add a reagent to one list that is already in another list, it is removed from the other list.

Filter text box

Specify a filter to apply to the Reagents list. You can use "*" to represent an arbitrary string and "?" to represent an arbitrary single character. You can also specify alternatives by listing them in braces { }, separated by commas. For example, 3-{F,Cl,Br}* would match any reagent that began with 3-F, 3-Cl, or 3-Br. All reagents whose titles contain the specified string are listed in the Reagents list when you press ENTER.
