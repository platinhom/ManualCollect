The mopac Section

This section is used to specify keywords for semiempirical NDDO calculations on the QM region. The keywords are the same as in the semiempirical NDDO input file—see Keywords Supported in the Semiempirical NDDO Module. In the mopac section, they must be assigned values of 1 (true) or 0 (false). Thus, for each keyword in a semiempirical NDDO input file, the mopac section would include these keywords and set their values to 1.
