Starting Maestro

If Maestro is not running, you can start it as follows:

Linux:

1. Open a terminal window.
2. Set the SCHRODINGER environment variable to the Schrödinger software installation directory:

csh/tcsh:

setenv SCHRODINGER installation-path

sh/bash/ksh:

export SCHRODINGER=installation-path

3. Enter the following command: 
$SCHRODINGER/maestro &

Windows: Double-click the Maestro icon on the desktop.

You can also use Start → All Programs → Schrodinger-2017-1 → Maestro.

Mac: Click the Maestro icon on the dock.

If it is not on the dock, drag it there from the SchrodingerSuites2017-1 folder in your Applications folder, or start Maestro from that folder.
