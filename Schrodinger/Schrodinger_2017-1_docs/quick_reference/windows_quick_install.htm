Quick Guide to Installation on Windows

This topic describes how to install Schrödinger’s Release 2017-1 software for use on a single Windows machine. If you need more details or encounter problems, see Installing the Software on Windows. If you want to run jobs on remote hosts, or perform any other nondefault installation tasks, see the Installation Guide.

It is assumed that you have downloaded the software into your Downloads folder.

Installing the software

1. Open your Downloads folder: Start → Documents, then click Downloads in the side bar.
2. Right-click on the downloaded zip file (the name starts with Schrodinger_Suite) and choose Extract All.

A panel opens so you can choose the destination where the extracted files will go.

a. Leave the destination at the default.
b. Ensure that Show extracted files when complete is selected.
c. Click Extract.
4. Double-click the folder that starts with Schrodinger_Suite (there should be only one folder).
5. Double-click setup, and click Run in the Open FIle - Security Warning dialog box.
6. Click Next in the installer.
7. In the Destination folder step, click Next to accept the default location and user.
8. In the Setup Type step, leave the selection at the default, and click Next.
9. In the Product Selection folder, check Select All and click Next.

If you want to install only some products, check the boxes for those products instead of checking Select All.

10. Click Start to start the installation.
11. If your virus checker objects to any executables, allow the executable to be used.
12. When the installation finishes, click Next.
13. In the Profile Selection step, click Next.
14. In the File Types step, click Next.
15. Deselect View Release Notes.
16. Click Configure to proceed to licensing your installation, or click OK to finish and do the licensing later.

If you installed the free version of Maestro, click OK, as the license is already installed.

Setting up licensing

If you obtained the free version of Maestro as an academic user, you can skip to the next section, as the license is already installed and activated.

If this is a new installation, or you are purchasing a renewal, you should receive an email message with a license key. Follow the instructions below. Skip the first step if the Configure Schrodinger Software panel is already open.

1. Choose Start → All Programs → Schrodinger-2017-1 → Configure.

The Configure Schrodinger Software panel opens and checks for valid existing licenses. In the Add Licenses section, I have a key should be selected.

2. Copy and paste the key from the email message into the text box.
3. Click Install Licenses, and click OK in the dialog box that opens.

If you are installing a new release and already have valid licenses, follow the instructions below. Skip the first step if the Configure Schrodinger Software panel is already open.

1. Choose Start → All Programs → Schrodinger-2017-1 → Configure.

The Configure Schrodinger Software panel opens and checks for valid existing licenses. In the Add Licenses section, I have a license file should be selected, and the location of the license file should be shown in the text box.

2. Click Install Licenses, and click OK in the dialog box that opens.

Checking that the software and documentation is working

1. Double-click the Maestro icon on your desktop to start Maestro, to check that Maestro is properly installed and licensed.
2. Choose Help → Help in Maestro to ensure that the documentation is properly installed.
