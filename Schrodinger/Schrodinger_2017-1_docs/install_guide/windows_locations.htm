Schrödinger File and Resource Locations on Windows

This topic contains information on where various resources or files are kept on Windows. These are relative to the standard application data location (%APPDATA%), local application data location (%LOCALAPPDATA%), and user location (%USERPROFILE)%), whose definitions are listed below.

%USERPROFILE% is set to C:\Users\username.
%APPDATA% is set to %USERPROFILE%\AppData\Roaming.
%LOCALAPPDATA% is set to %USERPROFILE%\AppData\Local.
