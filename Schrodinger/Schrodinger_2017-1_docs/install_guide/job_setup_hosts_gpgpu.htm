The gpgpu Setting in the Hosts File

To make use of GPU cards for Desmond calculations, the hosts file must be configured for each host. For example, on a local workstation, you could use an entry like this:

# Local workstation
name: myhostname
host: localhost
processors: 4
tmpdir /usr/tmp
gpgpu: 0, GTX 680

This entry indicates that the local workstation has a single GeForce GTX 680 card available at device index 0. For a remote host you could use an entry like this:

# Remote host
name: otherhost1
host: remote1.mycorp.com
processors: 8
schrodinger: /path/to/schrodinger/
tmpdir: /usr/tmp
gpgpu: 0, Tesla K20
gpgpu: 1, Tesla K20

Here, the remote workstation has two Tesla K20 cards, residing at device indices 0 and 1.
