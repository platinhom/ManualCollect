Configuring the Client for Offline Job Management

Once the job server host is set up, you can set up the client to make use of the job server.

  1. Copy the hosts file that you modified on the job server host to the client Schrödinger software installation.

  2. Set the following Maestro preferences, which are listed under Jobs - Starting in the Preferences panel.

    • Always run job driver script on remote host
    • Allow jobs on remote hosts to run detached from local machine

    Ensuring that the driver runs on the remote host is essential.

    These preferences could also be set by adding the following to prefer.cmd in the Maestro user resources directory on your client machine:

    prefer useremotedriver=true
prefer detachedjobs=true

For information on monitoring and incorporation of offline jobs, see the Monitor Panel topic.
