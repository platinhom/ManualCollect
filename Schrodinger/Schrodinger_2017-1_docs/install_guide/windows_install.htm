Installing the Software on Windows

We recommend that you install Schrödinger software as an administrator, and install for all users (AllUsers mode). However, if you do not have administrator privileges, you can install Schrödinger software in single-user (OnlyForMe) mode. To do so, you must have permission to write in the User folder of the registry and permission to create a directory that does not have spaces in the path (see below).

If Schrödinger software is installed by an administrator in OnlyForMe mode, the mode cannot be changed to AllUsers. Instead, the software must be uninstalled and reinstalled in AllUsers mode.

After you have installed the software, you must then obtain a license. If you are an academic user installing only Maestro, the license is included and installed automatically.
