Customizing the Hosts File

You can copy and edit the schrodinger.hosts file from the installation directory to customize its settings. The edited hosts file should be placed in your Schrödinger user resources directory. You usually do not need to do this unless you have different user names on different hosts. If you have installed Schrödinger products on multiple hosts, you may need to edit the schrodinger.hosts file on each host to add entries for the other hosts.
