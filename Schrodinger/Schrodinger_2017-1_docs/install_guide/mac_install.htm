Installing the Software on a Mac

The topics listed below provide information on installing and uninstalling Schrödinger software and the third-party databases needed for protein modeling.

After the software is installed, you must also install the license. You should receive instructions by email for obtaining and installing the license. For licensing information, go to https://www.schrodinger.com/licensing.
