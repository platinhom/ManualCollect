Obtaining Homologs from a BLAST Search for Residue Scanning

If you don’t have homologs for your protein, you can run a BLAST search to find homologs.

1. Click Run a Blast Search.

The BLAST Search Settings dialog box opens, so you can change settings if you want.

2. Click Start Job in this dialog box, after changing any settings.

The job starts, and its log file is displayed in the Job Progress dialog box.

At the end of the job, the BLAST Search Results dialog box opens, so that you can choose how many of the homologs you want to use. You can do one of the following to select homologs:

  • Select the homologs in the table (with shift-click and control-click).

  • Enter the number of the top homologs in the text box at the bottom of the panel and click Select Top.

When you have selected the homologs, click Incorporate Selected Rows to add the structures to the sequence viewer.
