Peptide Tools

The peptide tools allow you to predict various properties of small peptides from their sequences—see Predicting Peptide Properties.

  • Peptide Helicity (Tasks → Peptide Alpha Helicity): Predict the stability of alpha helices for small peptide sequences, using molecular dynamics.

  • Peptide Docking (Tasks → Peptide Docking): Dock peptides to a receptor, starting from the sequence. The receptor is largely rigid, the conformational space of the peptide is explored.

  • Peptide QSAR (Tasks → Peptide QSAR): Predict properties of small peptides using a QSPR (sequence-property) model based on peptide descriptors.
