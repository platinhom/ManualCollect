Running Residue Scanning from the Command Line

You can run residue scanning and affinity maturation jobs from the command line with the following command:

$SCHRODINGER/run residue_scanning_backend.py [options] structure-file

If you are running on Windows from a Schrodinger Command Prompt window, you can omit $SCHRODINGER/. To view the command options, run the command with the -h option and no structure file.

The structure file must be in Maestro format and contain either a single protein structure, with or without a ligand, or a receptor structure followed by a set of ligand structures (a “pose viewer” file).

The output Maestro file (jobname-out.maegz) contains first the starting structure, then the refined mutant structures. The property s_bioluminate_Mutations identifies the mutation made in each structure relative to the starting structure (for which this property is None), in the form chain:resnum(oldres>newres), e.g. H:103(TYR>ALA).
