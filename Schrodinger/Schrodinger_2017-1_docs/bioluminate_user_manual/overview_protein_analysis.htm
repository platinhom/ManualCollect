Protein Analysis Tools

The protein analysis tools provide information on a protein and its properties. No change is made to the protein structure.

  • Protein Structure Quality Viewer (Tools → Protein Structure Quality): Show reports on deviations of protein parameters from standard values, in graphical and tabular form. See Analyzing Protein Quality.

  • Residue Analysis (Tasks → Residue Analysis): Calculate energetic and other properties of residues. See Analyzing Residue Properties.

  • Consensus Visualization (Tools → Protein Consensus Viewer): Locate consensus waters, counter ions and ligands in a set of homologs to a reference protein. See Identifying Consensus Molecules.

  • Reactive Protein Residues (Tools → Reactive Residue Identification): Identify residues that are prone to specified reactions, by matching sequence patterns and some structural information. See Identifying Reactive Residues.

  • Aggregation Surface (Tasks → Aggregation Surface): Predict regions on a protein surface that have a propensity for aggregation. See Predicting Aggregation Regions.

  • Protein Interaction Analysis (Tasks → Protein Interaction Analysis): Analyze the interactions at the interface of two proteins. See Analyzing Protein Interface Interactions.

  • Low Mode Vibrational Sampling (Tasks → Low Normal Mode Analysis): Locate and visualize large-scale vibrational motions in a protein. See Locating Large-Scale Motions.

  • SiteMap (Tools → Binding Site Identification): Locate druggable sites on a protein. See the SiteMap User Manual.
