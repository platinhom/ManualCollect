Protein-Protein Docking Distance Restraints

If you know which parts of the protein should be in contact, you can add distance restraints between residues in the two proteins. These restraints define a minimum and a maximum distance between a residue in one protein (the receptor) and a residue in the other protein (the ligand). Individual restraints can be combined to satisfy some or all of a set of restraints. Restraint groups can be defined, and each group must be satisfied (there is a logical AND between groups).

Restraints can be added in the Distance Restraints tab. There are four Group tabs, allowing you to define four restraint groups. Within each group, each restraint between a pair of residues is represented by a row in the table.

To add a new restraint row to the table, click New Restraint. The Define New Restraint dialog box opens, so that you can define the ligand residue and the receptor residue for the constraint and set the maximum and minimum distance. The residues can be chosen by picking in the Workspace, by using the Workspace selection, by using the Atom Selection dialog box (see Atom Selection Dialog Box), or by typing in the chain, residue name and residue number, e.g. A:GLY44. When you click OK, a new row is added to the table with the information given in the dialog box. You can then edit the distances in the table.

To delete a restraint, select it in the table and click Delete column.

When you have defined the restraints in a group, you can then decide whether you want all of them to be satisfied (select All for Poses must satisfy), or a minimum number (select At least for Poses must satisfyand enter the minimum number).

If you want to define restraints between two surface patches, for example, you can do so by defining groups. In this scenario, each residue in one patch must satisfy a constraint with one of the residues in the other patch. To set up the restraints, you define a group for each residue in one patch. This should be the smaller patch, as the maximum is four groups, and hence the patch must have no more than four residues. In each group you define restraints between one residue in the first patch and all residues in the other patch. You then select At least for Poses must satisfyand enter 1 for the minimum number, so that the residue from the first patch is restrained to only one of the residues in the second. As you added all the residues from the other patch, the effect is that it doesn't matter which residue in the second patch is restrained to the residue from the first patch.
