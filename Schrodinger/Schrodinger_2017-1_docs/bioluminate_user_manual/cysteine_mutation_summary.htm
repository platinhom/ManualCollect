Cysteine Mutation Workflow Summary

1. Display the protein you want to analyze in the Workspace. You should ensure that it is properly prepared (with the Protein Preparation Wizard).

See Selecting and Analyzing the Protein for Cysteine Mutation.

2. If you want to analyze only part of the protein, select the residues that you want to analyze, and select Analyze only selected Workspace residues.

See Selecting and Analyzing the Protein for Cysteine Mutation.

3. Click Analyze Workspace or Analyze MD Trajectory.

See Selecting and Analyzing the Protein for Cysteine Mutation.

When the analysis finishes, the Residue pairs for mutation table is populated with the residue pairs that were found.

4. Filter the list to show only the mutations you are interested in:
a. Use the display options to display the mutation types that you are interested in.
b. Use the menus below the table to select the residues that you want to mutate to or from. The table is updated to show only those residues.
c. Use the cutoff to filter out pairs for which the distance is too large to form a disulfide bond.

See Choosing Residue Pairs to Mutate to or from Cysteine.

4. Select the pairs that you want to mutate in the table. Each pair is mutated independently to produce a structure and an energy evaluation.

See Choosing Residue Pairs to Mutate to or from Cysteine.

5. Decide whether you want to optimize residues near to the mutated residues by selecting from the Gas phase optimization shell options. The mutated residues are always optimized.

See Relaxing the Structure Around the Cysteine Mutation Site.

6. Enter a name in the Job name text box and click Run to run the mutation job. To decide how to handle the output, click the Settings button instead.

See Running the Cysteine Mutation Job.

7. When the job finishes, go to the Results tab. If the results are not listed in the Mutations table, click Load Results from Previous Run, navigate to and open the output Maestro file from the job.

See Examining the Cysteine Mutation Results.
