Using the Symmetrize Workspace Dialog Box

To symmetrize the structure that is included in the Workspace, enter a tolerance and click Find Point Group. If the point group displayed is the correct group, click Symmetrize. If not, increase the tolerance and click Find Point Group again, until the correct group is identified; then click Symmetrize.

Symmetrize Workspace Dialog Box Features

  • Tolerance: Allows you to adjust the tolerance for moving the atoms to find symmetry.

  • Find Point Group: Finds the point group within the given tolerance range.

  • Symmetrize: Adjusts the structure in the Workspace to conform to the given point group.
