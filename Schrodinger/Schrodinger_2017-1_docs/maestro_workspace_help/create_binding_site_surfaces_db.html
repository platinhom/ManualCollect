Create Binding Site Surfaces Dialog Box

In this dialog box you can set options for the generation of binding site surfaces and create the surfaces. These surfaces are molecular surfaces—see the Advanced Surfaces Panel topic for more information.

To open this panel, choose Workspace → Surface → Binding Site

Create Binding Site Surfaces Dialog Box Features

  • Create receptor surfaces option
  • Create ligand surfaces option
  • Truncate recptor surface at N ang from ligand text box
  • Surface transparency text box
  • Surface grid spacing text box
  • Style options
  • Color scheme option menu
  • Reset button
Create receptor surfaces option

Select this option to create a surface for the receptors that are displayed in the Workspace.

Create ligand surfaces option

Select this option to create a surface for the ligands that are displayed in the Workspace.

Truncate receptor surface at N Å from ligand text box

Specify the distance from the ligand at which the receptor surface is truncated.

Surface transparency text box

Specify the surface transparency as a percentage.

Surface grid spacing text box

Specify the grid spacing for surface generation. The image quality increases as the spacing gets smaller.

Style options

Select an option for the kind of surface to draw, from Solid, Mesh, or Dot.

Color scheme option menu

Choose a color scheme for coloring the surface. Apart from a constant color (gray), the option menu lists various properties by which the surface can be colored.

Reset button

Reset the values in the panel to their defaults.
