Edit Custom Label Panel

This panel is used to compose labels for assignment to atoms. Options for customizing the label appearance are available in the Preferences Panel.

To open this panel, choose Edit Custom Label from the Apply Label menu in the Style toolbox.

Using the Edit Custom Label Panel

To create a label you need to choose the fields (properties) that will be displayed in the label. The default fields are listed in the Select and order label fields list. To add a field to the list, click Add, and select the fields you want to use from the lists in the Add Label Fields Dialog Box. The fields are displayed in the order that they appear in the list. You can drag the fields to the desired position in the list. If you want to remove fields, select them in the list and click Remove. You can hide fields in the label by clearing the check box for the field. In this way you can keep a set of commonly used fields in the Select and order label fields list but choose which of these are actually displayed in the label by using the check boxes.

The formatting of the atom labels is set up in the Preferences panel, in the Atom/Bond labels group. The formatting includes the field separator, names of the fields, font, color, and default position. Applying fog to labels is controlled from the Fog tab of the Appearance panel.

Edit Custom Label Panel Features

  • Select and order label fields list
  • Add button
  • Remove button
  • Clear All Labels button
Select and order label fields list

Lists the fields to include in the atom labels, in the order shown. You can select multiple fields in the list, and drag them to reposition them in the list, or click Remove to remove them from the list. Each field has a check box that indicates whether it is included in the label or not. When a new field is added, it is added to the end and automatically included in the label.

Add button

Add fields to the Select and order label fields list. Opens the Add Label Fields Dialog Box, in which you can select atom properties and entry properties for the label. When you click OK in the dialog box, the fields are listed in the Select and order label fields list.

Remove button

Remove the selected fields from the Select and order label fields list.

Remove All button

Remove all fields from the Select and order label fields list.
