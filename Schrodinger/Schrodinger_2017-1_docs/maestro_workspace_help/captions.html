Captions

Captions can be used to display any kind of text in the Workspace, in a text box. They are stored as part of the project.

To create a caption, choose Workspace â†’ Add Caption. The New Caption dialog box opens. Enter the caption text in the Workspace caption text area, choose the background style (transparent or automatic contrast to the text), font attributes and color, and click OK.

The caption size is determined by the size of the text and the font. It does not change when you zoom in and out. The shape of the caption is determined by where you place line breaks in the text that you enter.

You can select one or more captions in the Workspace with shift-click and control-click. Selected captions are displayed with a gray border. You can reposition the selected captions as a group by dragging them. To delete them, press Delete, right-click and choose Delete Selected Captions.

If you want to change the text of a caption, double-click on the caption or right-click on the caption and choose Edit Caption. You can change the text, font, background, and color in the Edit Caption dialog box.

To show or hide the captions, use the Annotations button on the Workspace Configuration toolbar.

Other options for placing text in the Workspace include atom labels and showing entry properties.
