Force Field Used for IFD

You can choose the force field used from the Force field option menu. The default is OPLS3, but you can also choose OPLS_2005. The OPLS3 force field has a wider range of parameters and significant improvements over OPLS_2005, and is recommended. However, you may want to use OPLS_2005 to compare with previous results.

If you choose OPLS3 and you have a license for force field customization, customization controls are displayed to the right.

  • To customize the unparametrized torsions in the OPLS3 force field for a set of molecules, click Customize. The parametrization is done by fitting the torsions to a set of quantum-mechanical calculations, which can take several hours to complete. Clicking the Customize button opens the Force Field Builder panel.

  • To use your customized version of the OPLS3 force field, rather than the standard version in the distribution, select Use customized version. This option is set by default to the value of the Use custom parameters by default option in the Preferences panel, under Jobs - Force field. This section in the Preferences panel also allows you to specify the path to the custom force field.
