IFD H-Bond and Metal Constraints

In the H-Bond/Metal section you can set up constraints for hydrogen bonding and for association with a metal atom.

H-bond and metal constraint atoms

Select Pick to pick atoms in the receptor structure to be used for Glide H-bond or metal constraints. The atoms should be hydrogen-bond acceptors (e.g. O, N, S, with lone pairs available), hydrogen-bond donors (e.g. H in OH, NH, SH groups), or metal atoms. The atoms that are picked are listed in the text box. Symmetry-related atoms (such as the other O atom in a carboxylate group) are automatically included as constraints, so you only need to pick one. The constraints are applied using the default feature sets for hydrogen-bond donors and acceptors, and metal atoms. For more information on H-bond and metal constraints in Glide, see Setting Up H-Bond and Metal Constraints in the Receptor and Applying Constraints for Glide Docking.

Docking stages in which to apply H-bond/metal constraints

This option menu allows you to choose the Glide docking stages in which to apply the constraints, from Both, Initial, or Redocking.
