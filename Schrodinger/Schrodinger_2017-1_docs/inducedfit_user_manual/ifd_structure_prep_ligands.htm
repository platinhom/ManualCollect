Preparation of the Ligands To Be Docked with IFD

Each ligand that will be docked to the receptor must also meet certain requirements. Like the receptor, it must have correct bond orders, formal charges, and a complete set of hydrogens for a valid ionization state. You can run the ligands through the Schrödinger application LigPrep to produce one or more desired ligand conformations and ionization states. See the LigPrep User Manual for information on this application.

In addition to the structure preparation, the names of the atoms in each ligand to be docked must satisfy two conditions:

  • All atoms in the ligand must have the same PDB Residue Name, Residue Number, and Chain Name. This condition is satisfied automatically during job execution, so you do not need to do anything. The program sets the PDB residue name to “UNK”, the chain name to Z, the residue number to 999, and the insertion code to blank. These values ensure that there is no conflict with the receptor.

  • All atoms must have PDB Atom Names that are unique within the ligand residue, as required for parameter generation. Prime attempts to ensure that atoms have unique PDB atom names, but if it does not succeed in this task, the job will fail. It is therefore highly recommended to correct the ligand if it does not satisfy this condition. If it does not, you must first ensure that the ligand is a single residue.

To check whether a ligands satisfies these conditions and correct it if it does not, use the procedures below. Before doing so, ensure that only the ligand is displayed in the Workspace.

To check that all atoms in the ligand residue have the same residue information:

  1. Select the ligand in the Workspace. (Use the Quick Select ligand button.)

  2. In the Style toolbox, click the arrow next to the Apply Labels button, and choose Edit Custom Label.

    The Edit Custom Label dialog box opens.

  3. In the Select and order label fields list, clear all check boxes.

  4. Click Add.

    The Add Label Fields dialog box opens.

  5. Select Residue name, Residue number, and Chain name, and click OK.

  6. Click OK in the Edit Custom Label dialog box.

The labels are displayed, which you can examine to ensure that they are the same. To remove the labels, click the Delete Labels button in the Style toolbox.

To correct the residue information:

  1. Select the atoms that need to be changed in the Workspace.

  2. In the 3D Builder panel, click Other Edits and choose Change Other Properties.

    The Change Other Properties panel opens.

  3. Choose the property from the Property option menu.

  4. Enter a value in the appropriate text box.

  5. Click Apply.

To check that all atoms have unique PDB atom names within the residue:

  1. Select the ligand in the Workspace. (Use the Quick Select ligand button.)

  2. In the Style toolbox, click the arrow next to the Apply Labels button, and choose Edit Custom Label.

    The Edit Custom Label dialog box opens.

  3. In the Select and order label fields list, clear all check boxes.

  4. Click Add.

    The Add Label Fields dialog box opens.

  5. Select PDB atom name, and click OK.

  6. Click OK in the Edit Custom Label dialog box.

To correct ligands that do not have unique PDB atom names:

  1. Select the ligand in the Workspace.

  2. In the 3D Builder panel, click Other Edits and choose Change Other Properties.

    The Change Other Properties panel opens.

  3. Choose PDB Atom Name from the Property option menu.

  4. Select Set unique PDB atom names within residues.

  5. Click Apply.
