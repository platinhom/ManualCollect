Checking and Interpreting Results of MacroModel Coordinate Scans

Coordinate scan calculations involve minimizations, and you should check that minimizations for all coordinate values converge.

You can display the results of a scan in one or two coordinates in the Plot of One-Coordinate Scan panel or the Plot of Two-Coordinate Scan panel, which you open by choosing Tasks → Structure Analysis → Coordinate Scan → Plot One Coordinate or Tasks → Structure Analysis → Coordinate Scan → Plot Two Coordinates.
