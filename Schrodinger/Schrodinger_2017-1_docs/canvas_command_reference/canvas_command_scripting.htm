Scripting with Canvas

In addition to the command-line utilities, Canvas has a Python API that can be used to write Python scripts. The API is described in the document Canvas Python API. For the general Schrödinger Python API, visit the Python API web page. Canvas utilities are also available for use with KNIME, along with many other Schrödinger products and utilities.
