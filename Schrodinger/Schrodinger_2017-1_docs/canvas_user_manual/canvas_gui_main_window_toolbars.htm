The Canvas Toolbars

The Canvas interface has four toolbars: File, Edit, Select Rows, and Structure Size. You can show or hide the toolbars from the shortcut menu, and you can show or hide toolbar buttons in the Preferences panel (File → Preferences → Toolbar Buttons).

The File toolbar has buttons for many of the common actions. A brief description of the File toolbar is given below, with cross-references to fuller descriptions of the actions.

New Project

Create a new Canvas project. The current project is closed. See Opening and Closing Canvas Projects.

Open Project

Open an existing Canvas project. Opening and Closing Canvas Projects.

Import

Import structures into the project. See Importing Structures and Data into Canvas Projects.

Export

Export structures from the project to an external file. See Exporting Structures and Data from Canvas Projects.

Find

Find text in the spreadsheet. Opens the Find panel. See Finding Text in the Canvas Spreadsheet.

Collapse to Selected

Collapse the view to show only the selected structures. Same as View → Collapse to Selected.

Hide Selected

Hide the selected structures. Same as View → Hide Selected. See Displaying Different Views of the Canvas Spreadsheet.

Apply to Master

Apply the current custom view to the master view. Only available in custom views. See Displaying Different Views of the Canvas Spreadsheet.

Save View

Save the current view. In the master view, the view is saved as a new custom view, and the Save Custom View dialog box opens. See Displaying Different Views of the Canvas Spreadsheet.

Restore Default View / Revert All Changes

Restore the default view in the master view, or revert all changes to the view in a custom view. In either case, the view is restored to its initial (default) state. See Finding Text in the Canvas Spreadsheet.

Edit Spreadsheet

Edit cells in the spreadsheet. When you click this button, the Edit toolbar is displayed.

Add Row

Add a new empty row to the end of the spreadsheet.

Add Column

Add a new empty column to the right side of the spreadsheet.

Substructure Query

Run a substructure query. See Filtering the Canvas Spreadsheet by Substructure.

Scatter Plot

Create a scatter plot. See Scatter Plots of Canvas Data.

Histogram

Create a histogram. See Histograms of Canvas Data.

Pie Chart

Create a pie chart. See Pie Charts of Canvas Data.

The Edit toolbar is only active when you are editing the spreadsheet. It is displayed in place of the File toolbar when you start editing, and has the following two buttons.

Save Spreadsheet Changes

Save the spreadsheet with the changes included. When you click this button, the spreadsheet is saved, and the File toolbar is redisplayed.

Exit Without Saving Spreadsheet Changes

Discard all the changes to the spreadsheet. When you click this button, the spreadsheet is reverted to its state before you started editing, and the File toolbar is redisplayed.

The Select Rows toolbar is described in Selecting and Copying Rows, Columns, and Cells in the Canvas Spreadsheet, and the Structure Size toolbar is described in Configuring and Navigating the Canvas Spreadsheet.
