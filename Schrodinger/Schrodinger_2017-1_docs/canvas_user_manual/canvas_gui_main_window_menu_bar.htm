The Canvas Menu Bar

The menu bar has the following menus:

  • File—Open and close projects, import and export structures and data, set preferences, and quit Canvas. This menu is described in detail in Canvas Projects.

  • Edit—Copy cells, clear cell content, add or delete rows or columns, select cells, rows, or columns, find and select text. This menu is described in detail in The Canvas Spreadsheet.

  • View—Create, delete, display, and perform operations on custom views. This menu is described in detail in Displaying Different Views of the Canvas Spreadsheet.

  • Partition—Create, store, modify, apply, and delete partitions of the spreadsheet rows. This menu is described in detail in Partitioning Rows Into Classes in the Canvas Spreadsheet.

  • Structure—Configure the structure display and copy and paste structures. This menu is described in detail in Structure Menu, and its use in several topics in this manual.

  • Data—Perform operations on the table data, such as sorting, filtering creating a heat map, calculating statistics and new properties, and editing data. This menu is described in detail in Organizing the Canvas Spreadsheet Data and Filtering and Querying the Data in the Canvas Spreadsheet.

  • Query—Query the database and return rows that match the query. This menu is described in detail in Filtering and Querying the Data in the Canvas Spreadsheet.

  • Chart—Create and display scatter plots, histograms, and pie charts. This menu is described in detail in Making Charts of the Canvas Spreadsheet Data.

  • Applications—Run Canvas applications. This menu is described in detail in Running Applications from Canvas.

  • Tasks—Run Canvas applications. This menu is organized by the kind of task, rather than the list of applications. It opens the same panels as the Applications menu.

  • Project—Show or hide the Project View and the Message View.

  • Python —Open a Python interpreter to run Python commands or scripts.

  • Scripts—Install, manage, and run Python scripts. This menu is described in detail in Running Python Scripts from Canvas.

  • Help—Display online help, the user manual, Canvas legal notices, and the Knowledge Base; open the Diagnostics panel.

If you start Canvas without opening a project, only the File and Help menus are available.
