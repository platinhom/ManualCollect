Changing the Classes in a Partition

When a partition has been created, you can add classes, move rows to new classes or declassify them, edit the partition, or delete the partition.

To create a new class, first select the rows in the spreadsheet. The rows can be unclassified rows, or rows that are already in one or more classes. Next, choose Partition → Add/Move to → New Class, or press ALT+L (UL). The New Class dialog box opens, showing the class table. You can then edit the class value and change the color, as described above.

To move rows to an existing class, select the rows and choose Partition → Add/Move to → Class n, or type Alt+n (Un), where n is the class index. Classes are indexed sequentially from 1, and the class value is displayed on the Add/Move to submenu.

To remove rows from their classes, so that they are not in any class, select the rows and choose Partition → Unclassify, or press ALT+0 (U0). You do not need to select entire rows to add or remove them from a class: any row with a selected cell is included in operations on a class.

To make changes to the partitions, choose Partition → Edit. In the Edit Partition dialog box, you can change the name and description of the partition, change the class values and colors, and delete classes. To delete a class, right-click on it in the Class table and choose Delete.
