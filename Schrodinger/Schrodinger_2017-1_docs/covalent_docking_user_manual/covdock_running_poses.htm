Setting the Number of Output Poses from Covalent Docking

By default, covalent docking only returns a single pose per ligand, which is the lowest-energy pose. If you want multiple poses to be returned, you can enter the number of poses you want in the Output poses per ligand reaction site text box. This could be important if there are multiple matches of the ligand SMARTS pattern or the receptor site, as the ranking of poses by Prime energy can only be done on the basis of a particular binding site.

For virtual screening, you can limit the number of ligands whose poses are returned, by setting a value in the Max number of top-scoring ligands to report text box. The poses are sorted by score and the cutoff is applied to the best scoring ligands. This limit can be applied to pose prediction as well, but is probably more useful for virtual screening, to limit the output.
