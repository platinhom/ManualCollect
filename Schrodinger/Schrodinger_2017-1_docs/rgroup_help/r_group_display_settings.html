R-Group Analysis — Display Settings Dialog Box

In this dialog box you can make settings for the display of the R-Group heat map.

Display Settings Dialog Box Features

  • Notation options
  • Digits after decimal box
  • Maximum value and Minimum value text boxes
  • Min and Max sliders
  • Reverse Map button
Notation options

Select a notation option for the display of the values in the heat map. The two options are Floating point and Scientific notation.

Digits after decimal box

Specify the number of digits to display after the decimal point in the values shown in the heat map. You can use the arrow buttons or enter the value in the text box.

Maximum value and Minimum value text boxes

Enter the maximum value and the minimum value that is represented by the colors in the heat map. Values that lie outside the range represented by these maximum and minimum values are assigned the color of the maximum or the minimum. You can enter arbitrary values in these text boxes, so that the range of colors can be set to an absolute range rather than relative to the data range.

Min and Max sliders

Change the maximum and minimum values that are represented by the colors in the heat map. The slider range is restricted to the data range. The heat map is updated while you move the sliders, so you can see the cell colors changing as the range represented is increased or decreased.

Reverse Map button

Reverse the direction of the color ramp, so that it changes from red to blue instead of from blue to red.
