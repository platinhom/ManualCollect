Predicting the pKa Value for Protonated Methylamine

In this exercise, you will calculate the pKa of methylamine (or rather, of its conjugate acid, the methylammonium cation).

1. Import the file methylamine.maegz from your tutorial directory (File → Import Structures).
2. Choose Tasks → Quantum Mechanics → pKa.

The Jaguar panel opens with the Input tab displayed, set up for pKa calculations. In the table section, you can specify the pKa atom.

3. Double-click the pKa Atom cell in the table for the methylamine row.

A text box is displayed in the cell, for the atom name.

4. Click on the nitrogen atom in the Workspace.

The nitrogen atom is marked and its label is displayed in the text box.

5. Press Enter.
6. Set the job name to methylamine.
7. Click Run.

The calculation of pKa values involves several steps and can be time-consuming for large molecules. This job takes a few minutes to finish. The output pKa is the only data written to the file methylamine.out. When the job finishes, a new entry is added that contains the pKa value as a property, and the pKa value is displayed next to the nitrogen atom in the Workspace.
