Setting Preferences for the Jaguar Exercises

By default, only the primary Jaguar properties are shown in the Project Table. In these exercises you will be using other properties, so the preference to show these properties by default must be set.

  1. Choose Edit → Preferences.

  2. Under Project Table in the tree on the left, click Behavior.

  3. Under When new entries are added, select Show all properties.

  4. Close the Preferences panel.
