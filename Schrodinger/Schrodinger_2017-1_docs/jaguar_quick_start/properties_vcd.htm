Calculating and Displaying a VCD Spectrum for Alanine

In this set of exercises, you will run calculations to generate the data for a vibrational circular dichroism (VCD) spectrum. Experimental spectra are usually an average over the low-lying conformations of the molecule. To compare with experiment, it is therefore is necessary to perform a conformational search and generate the spectrum for each conformer. This is done automatically in the VCD workflow.

The exercises in this section can take a substantial amount of computing time. To perform this exercise, you need a MacroModel license as well as a Jaguar license.
