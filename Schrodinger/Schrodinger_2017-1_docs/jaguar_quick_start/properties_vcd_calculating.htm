Calculating VCD Spectral Data for Alanine

In this exercise, you will set up and run the job to calculate VCD spectral data for alanine.

  1. Import the file alanine.maegz from your tutorial directory (File → Import Structures).

    The alanine molecule is imported and displayed in the Workspace.

  2. Open the VCD Workflow panel.

    How do I do this?

    1. Click the Tasks button.

      A menu is displayed. To the left there should be a pane with a TASKS heading and an APPLICATIONS heading. If it is not displayed, click Browse on the menu.

    2. Under TASKS, choose Quantum Mechanics.

      The contents of the pane are replaced with the tasks available for Quantum Mechanics.

    3. Click VCD Spectra.

  3. Ensure that Use conformational search is selected.

    This is the default. The conformational search settings can also be left at the defaults.

  4. From the Solvent option menu, choose Chloroform.

    VCD spectra are usually measured in a solvent, such as CCl4, CHCl3 or CS2, which can affect the conformations that are present.

  5. Click the Settings button.

    The Jaguar - Job Settings dialog box opens.

    If you have access to multiple processors, you can distribute the job. The optimization of the conformers takes 1–2 hours per conformer, of which there are six. To do this:

    1. Choose a multiprocessor host from the Host option menu. The number of processors (cores) is given in parentheses next to the host name.

    2. Enter the number of processors to use in the CPUs text box. This number should be 6 or less.

  6. Name the job vcd, and click Run.
