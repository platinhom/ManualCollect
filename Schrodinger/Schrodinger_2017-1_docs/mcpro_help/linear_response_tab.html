MCPRO+ — Linear Response Tab

In this tab you set up the parameters of the simulations that will be done for the linear response calculation. For more information on these calculations, see MCPRO+ Linear Response Calculations.

Linear Response Tab Features

  • Monte Carlo settings section
  • Bound Monte Carlo sampling section
  • Free Monte Carlo sampling section

Monte Carlo settings section

In this section you set the pressure and temperature for the ensemble, and specify the block size.

Temperature text box

Specify the temperature for the simulation in Celsius.

Pressure text box

Specify the pressure for the simulation in atmospheres.

Num configurations per block text box

Specify the number of configurations per block. . Each block is run as a separate MCPRO+ calculation with the given number of configurations in the block, but the blocks are run sequentially to maintain the Markov chain.

Bound Monte Carlo Sampling Section

In this section you specify the parameters for the sampling of the system in which the ligand is bound to the receptor.

Solvent-only equilibration for N x 1000 configurations option and text box

Select this option to perform equilibration of the solvent at the given temperature and pressure, and enter the number of configurations (in thousands) for the equilibration in the text box.

System equilibration for N x 1000 configurations option and text box

Select this option to equilibrate the entire system, and specify the number of configurations (in thousands) for the equilibration in the text box. This step is performed after any solvent equilibration. All degrees of freedom are sampled.

System averaging for N x 1000 configurations option and text box

Select this option to perform sampling of the system to obtain the final results from the usual averaging techniques, and specify the desired number of configurations (in thousands) in the text box.

Free Monte Carlo Sampling Section

In this section you specify the parameters for the sampling of the free ligand in solvent.

Run annealing protocol N times text box

Specify the number of times to run the annealing protocol, which includes the steps given in the Annealing Protocol section. This protocol is repeated in order to obtain good Coulomb energies for the ligand-solvent system.

Solvent-only equilibration for N x 1000 configurations option and text box

Select this option to perform equilibration of the solvent at the given temperature and pressure, and enter the number of configurations (in thousands) for the equilibration in the text box.

Solvent and ligand torsion high-temperature equilibration for N x 1000 configurations option and text box

Select this option to perform equilibration of the solvent and the ligand torsional angles at high temperature, and enter the number of configurations (in thousands) for the equilibration in the text box. This step ensures that the ligand can sample higher-energy configurations and therefore a wider range of torsional angles. The higher temperature is quenched by the next equilibration.

System equilibration for N x 1000 configurations option and text box

Select this option to equilibrate the solvated ligand system, and specify the number of configurations (in thousands) for the equilibration in the text box. This step is performed after any solvent equilibration. All degrees of freedom are sampled.

System averaging for N x 1000 configurations option and text box

Select this option to perform sampling of the system to obtain the final results from the usual averaging techniques, and specify the desired number of configurations (in thousands) in the text box.
