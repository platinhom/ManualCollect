MCPRO+ — Solvation Tab

In this tab you can select the solvent model and set options for the solvent.

Solvation Tab Features

The Solvation tab has a single section, Solvent Options, with up to five choices for a solvation model. The choices available depend on the type of calculation and the model system. Some choices are further restricted by choices made in other tabs. These restrictions are noted along with the descriptions.

Constant dielectric of option and controls

Use this option to specify a general solvent using a continuum model. You can specify the dielectric constant in the text box. If you are minimizing a structure with the steepest descent or conjugate gradient method, you can choose to use a distance-dependent dielectric. The method is chosen in the Minimization tab.

GB/SA aqueous continuum solvent option

Run the simulation using the GB/SA model for aqueous continuum solvation. For minimization calculations, this solvation method is available with all except the conjugate gradient method.

Employ a periodic solvent box option and controls

Use explicit water molecules for the solvent model, and run the simulation with periodic boundary conditions. When you choose this option, the controls for it become available:

  • Option menu—Choose the solvent model (TIP3P or TIP4P) and the box dimensions. Each item corresponds to a combination of solvent and box size, which determines the number of water molecules in the box, before the solute (protein, ligand, or complex) is added to the box.

  • Number of solvent molecules to remove options—Choose whether to automatically remove solvent molecules according to the solute volume, or to specify the number of solvent molecules to remove. Solvent molecules must be removed to accommodate the solute, and are chosen for removal in order of their interaction energy with the solute, from highest to lowest, i.e. high-energy molecules are removed first.

    Warning: If you choose to specify the number of solvent molecules to remove, you should make an accurate estimate, otherwise the job could fail or the results of the calculation might be erroneous.

Employ a solvent cluster option and controls

Use explicit water molecules for the solvent model. When you select this option, you can choose the solvent model and the size of the solvation shell with the following controls:

  • Solvent option menu—Choose the model for the water molecules, from TIP3P and TIP4P.

  • Cap radius text box—Enter the radius of the solvent cap. This radius is measured from the centroid of the ligand (either the reference ligand, if one was selected, or the first ligand specified for the simulation). The solvent molecules are added from pre-equilibrated caps of the given radius, and solvent molecules with in a given distance of solute atoms are removed. When you choose a solvent cap, the sphere is represented in the Workspace in semi-transparent blue.

Gas phase option

Run the simulation in the gas phase.
