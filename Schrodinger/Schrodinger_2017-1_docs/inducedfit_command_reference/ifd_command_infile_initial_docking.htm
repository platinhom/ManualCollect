The INITIAL_DOCKING Stage of the IFD Input

Perform the initial docking for the extended sampling stage. This involves 7 different docking protocols using combinations of the vdw scaling and alanine mutation from the above two stages, plus specific Glide settings to generate 20,000+ initial poses which are then filtered and clustered down to 80. The keywords in Table 2 can be used. You can add Glide keywords to this stage, prefixed with GRIDGEN_ or DOCKING_. As this stage performs both grid generation and docking, the prefixes are necessary to direct the keyword to the appropriate Glide run. The Glide keywords are described in The glide Command.
