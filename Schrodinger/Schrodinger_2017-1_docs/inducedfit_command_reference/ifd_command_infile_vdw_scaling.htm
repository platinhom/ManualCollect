The VDW_SCALING Stage of the IFD Input

This stage estimates the flexibility of side chains within the binding site for the extended protocol, and sets the atom-level property r_glide_atom_vdwscale for each atom on those side chains to a value related to its potential range of motion. Atoms that are more mobile are scaled to lower values. This property can then be used by Glide to perform atom-specific van der Waals scaling during grid generation. The BINDING_SITE keyword from the common Glide keywords (Table 2) can be used in this stage to define the binding site.
