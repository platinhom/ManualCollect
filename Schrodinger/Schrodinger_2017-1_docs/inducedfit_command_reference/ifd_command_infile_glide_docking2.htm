The GLIDE_DOCKING2 Stage of the IFD Input

This stage performs both grid generation and ligand docking. It can be used with either protocol, and supersedes the GLIDE_DOCKING stage. The keywords in Table 2 apply to this stage. You can add Glide keywords to this stage, prefixed with GRIDGEN_ or DOCKING_. As this stage performs both grid generation and docking, the prefixes are necessary to direct the keyword to the appropriate Glide run. So for example, to set the option to sample rings during docking, you would use the keyword DOCKING_SAMPLE_RINGS. The Glide keywords are described in The glide Command.

If multiple receptors are specified in the input files, the settings are applied to each receptor. This means that the binding site is defined by the same residue specifications for each receptor. For example, if a ligand is used, it must have the same residue name, residue number, and chain ID in each complex.
