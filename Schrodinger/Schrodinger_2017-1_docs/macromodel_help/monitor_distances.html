Monitored Distances Panel

This panel is used to set up distances to be monitored during a dynamics simulation. Results are saved in the .mmo file.

To open this panel, click Distances in the Monitor tab of the Dynamics panel or the MC/SD panel.

Using the Monitored Distances Panel

To define a distance to be monitored, choose Atom or Bond from the Pick menu, and click on two atoms or one bond in the Workspace. When the distance is defined, the two atoms appear in the list at the top of the panel. The defined distances are marked with a purple dotted line and eye icon. The selected distance is distinguished by a thicker line.

To redefine an item, select it in the list and then pick the atoms or bond in the Workspace.

To delete a distance from the list, select it in the list and click Delete. To delete all monitored distances, click Delete All.

Monitored Distances Panel Features

  • Monitored distances list
  • Define monitored distances section
  • Delete button
  • Delete All button
Monitored distances list

Lists the distances to be monitored. After the second atom is picked, a new entry appears in the list.

Define monitored distances section

The Define monitored distances section contains the standard picking controls, which are used to select atoms that define a distance. It also contains a Markers option, which you can use to show markers (a purple dotted line and eye icon) for the monitored distances in the Workspace.

Delete button

To delete a monitored distance, select it in the Monitored Distances List, then click Delete.

Delete All button

Click this button to delete all monitored distances.
