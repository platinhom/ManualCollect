MC/SD Panel

The MC/SD tab is used to set up mixed-mode Monte Carlo/Stochastic Dynamics simulations.

To open this panel, choose Applications → MacroModel → MC/SD in the Task Tool.

Using the MC/SD Panel

MacroModel panels allow you to choose structures from either the Workspace or the Project Table (see Energy Overview for details). The MC/SD panel contains seven tabs. For descriptions of the controls in the first six tabs, see Potential, Constraints, Substructure, Mini, Monitor, and Dynamics. The seventh tab, MS/SD, contains controls for setting up mixed-mode Monte Carlo/Stochastic Dynamics simulations. These are described below. Once you have finished configuring the settings on the panel, you can click Run to run the job with the current job settings, or you can choose Job Settings from the Settings button menu to open the Job Settings dialog box to make job settings, or choose Write to write the files without starting the job (see Energy Overview for details).

The MC/SD procedure (Guarnieri, F.; Still, W. C. J. Comput. Chem. 1994, 15, 1302-1310) differs from normal dynamics simulations in that it uses a mixture of Metropolis Monte Carlo and dynamics steps in order to increase the rate at which simulations explore conformational space. For MC/SD simulations, it is necessary to specify torsions to be rotated and, if there is more than one molecule in the system, molecules to be translated and rotated.

The simplest way to define torsions and molecules for translations/rotations is to click Perform Automatic Setup. The program then searches the molecules and sets up all the necessary torsion rotations and molecule translations/rotations. These variables can be viewed or edited by clicking Torsion Rotations and Molecule Trans/Rot to open the corresponding panels (see related topics below for descriptions of these panels). All torsion rotations and molecule translations/rotations can be removed by clicking Reset All Variables.

Perform Automatic Setup

Click to automatically define torsions and translations/rotations. To view and edit the automatically generated settings, click Torsion Rotations and Molecule Trans/Rot.

Ratio of MC to SD steps

Specify how many Monte Carlo trials are performed for each dynamics time step. The default is 1, but this number can be increased as required to give more dynamics steps per Monte Carlo step.

Minimum number of torsions to vary in a MC step

Specify the lower limit for the number of torsions that can be varied in each Monte Carlo step of an MC/SD simulation. The number of torsions varied is randomly selected from the range defined by the minimum and maximum values.

Maximum number of torsions to vary in a MC step

Specify the upper limit for the number of torsions that can be varied in each Monte Carlo step of an MC/SD simulation. The number of torsions varied is randomly selected from the range defined by the minimum and maximum values.

Torsion Rotations

Click this button to open the Torsion Rotations panel for defining specific dihedral angles to sample during a simulation.

Molecule Trans/Rot

Click this button to open the Molecule Trans/Rot panel for viewing automatically generated molecule translation and rotation selections.
