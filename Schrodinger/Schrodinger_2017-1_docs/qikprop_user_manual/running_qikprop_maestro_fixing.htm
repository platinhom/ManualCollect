Fixing Noncompliant Structures for QikProp

If your input structures are missing hydrogen atoms or other structural information needed by QikProp, the Hydrogen/Lone-Pair Fixup dialog box (Figure 1) is displayed when you click Run in the Job Settings dialog box. This dialog box presents a list of valid treatments to fix up the structures, and buttons to select the action to take.
