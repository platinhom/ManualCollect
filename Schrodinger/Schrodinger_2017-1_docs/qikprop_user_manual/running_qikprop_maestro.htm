Running QikProp From Maestro

To run QikProp from Maestro, choose Tasks → ADME and Molecular Properties → Ligand-Based ADME/Tox Prediction or Applications → Other Applications → QikProp in the Task Tool. The QikProp panel opens.

Figure 1:The QikProp panel.

To launch a job from the QikProp panel, specify an input source and select options. After you make your selections, click the Settings button. The Job Settings dialog box is displayed, in which you can specify an incorporation mechanism, a job name, and a host. The job is started when you click Run, and you can monitor its progress in the Monitor panel.
