Making Ligand Settings for Glide Docking

For a tutorial example, see Specifying Ligands To Dock.

In the Ligands tab you specify the source of ligands to be docked or scored and set size limits for skipping ligands. You can also choose the partial charges to use and change the settings for van der Waals radii scaling of nonpolar ligand atoms. The ligand settings are described in the following topics:
