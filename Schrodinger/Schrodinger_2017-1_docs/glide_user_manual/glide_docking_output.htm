Selecting Output Options for Glide Docking

For a tutorial example, see Specifying Output Options for Ligand Docking.

The options in the Output tab control the final output of ligand poses that pass successfully through Glide’s various scoring stages. The features of this tab are options in a section called Structure output and an Advanced Settings button. Clicking the button opens the Output - Advanced Settings dialog box.
