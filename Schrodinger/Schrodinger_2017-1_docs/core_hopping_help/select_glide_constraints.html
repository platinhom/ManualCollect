Select Glide Constraints Panel

In this panel you can select the Glide receptor constraints you want to use when aligning or docking the cores, and decide which onet to apply. You can only use H-bond constraints.

Select Glide Constraints Panel Features

  • Select available constraints table
  • Align stage: Must match options and box
  • Dock stage: Must match options and box
Select available constraints table

This table lists the constraints that are available with the Glide grid, and allows you to select them for application in either the alignment stage, the docking stage, or both. The columns are described below.

Use Align Check box for selecting the constraint for use in the alignment stage.
Use Dock Check box for selecting the constraint for use in the alignment stage.
Name Constraint name, as defined when the grid was set up. Noneditable.
Constraint Type Type of constraint, which can be H-bond donor or H-bond acceptor. No other constraint types can be used. Noneditable.
Align stage: Must match options and box

Select All if you want protocore alignment to match all constraints, or select At least and enter the minimum number of constraints that must be met, in the box. There is no facility for choosing which of the minimum number of constraints is actually met.

Dock stage: Must match options and box

Select All if you want the docking of the aligned protocores to match all constraints, or select At least and enter the minimum number of constraints that must be met, in the box. There is no facility for choosing which of the minimum number of constraints is actually met.
