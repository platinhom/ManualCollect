Creating a Numeric QSAR Model for Binding Affinity

This exercise demonstrates how to use the AutoQSAR panel to set up, create, interpret, and apply a QSAR model for a congeneric series of ligands that bind to FactorXa. The QSAR models include all the information necessary to make predictions for compounds using a consensus of QSAR models or any single one of the top-ranked QSAR models.
