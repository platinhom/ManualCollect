Running the tautomerizer

To run the tautomerizer from the command line, use the syntax below. The tautomerizer is part of the LigPrep collection of programs and requires a LigPrep license.

$SCHRODINGER/utilities/tautomerizer [options] input_file.mae output_file.mae

For a description of the options, run the command with the -h option.

The tautomerizer can also be run as part of the LigPrep process. When LigPrep is run from the LigPrep panel in Maestro, the default options are used. When it is run from the command line, the defaults can be overridden by options to the ligprep command. See The ligprep Command and LigPrep Use of the tautomerizer Utility for more information.
