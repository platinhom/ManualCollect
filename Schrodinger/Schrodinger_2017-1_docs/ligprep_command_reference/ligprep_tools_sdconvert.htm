LigPrep Use of sdconvert

This utility converts between MDL SD, Maestro, and MacroModel format files. See Conversion To and From SD Format: sdconvert for details of this utility. In LigPrep, the tool is used with the option to save structures that could not be converted (the -bad option of sdconvert).
