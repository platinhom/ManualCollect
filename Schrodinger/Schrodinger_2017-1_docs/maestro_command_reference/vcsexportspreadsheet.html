vcsexportspreadsheet

Export actives found by libselector to a file for use in a spreadsheet. This can be comma-separated value (.csv) format or tab-delimited format.

Syntax: vcsexportspreadsheet delimiter= <text> [<filename> ]

Options:

delimiter

This option sets the delimiter to use to separate columns.

Valid values: text strings
Default value: ,

Operands:

<filename>

The name of the file to which actives will be written. If no name is specified, then no export will be done.
