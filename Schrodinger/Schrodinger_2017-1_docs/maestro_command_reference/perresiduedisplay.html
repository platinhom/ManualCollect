perresiduedisplay

Specify whether or not display per-residue interactions.

Syntax: perresiduedisplay display= yes | no

Options:

display

If this option is true, display per-residue interactions.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: false
