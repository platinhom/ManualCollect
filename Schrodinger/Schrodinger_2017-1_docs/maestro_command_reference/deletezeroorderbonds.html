deletezeroorderbonds

Delete zero order bonds connected to atoms in the specified set of atoms. The zero order bond will be deleted if it connects any atom within the set.

Syntax: deletezeroorderbonds [<ASL> ]

Operands:

<ASL>

The set of atoms which will have zero order bonds deleted.
