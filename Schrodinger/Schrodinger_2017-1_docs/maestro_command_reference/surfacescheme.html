surfacescheme

Sets the color scheme for the given surface.

Syntax: surfacescheme color= <text> colorblue= <n> colorgreen= <n> colorramp= <text> colorred= <n> datarange= entire_volume dotradius= <x> entry= <text> espcolorramp= <text> espmax= <x> espmin= <x> iterations= <n> linewidth= <x> mapmax= <x> mapmin= <x> negativebluecolor= <n> negativecolor= <text> negativegreencolor= <n> negativeredcolor= <n> scheme= <text> schemevolume= <text> showlegend= yes | no smooth= yes | no [<surface> ]

Options:

color

Sets the color for a constant color.

Valid values: text strings
Default value: gray

colorblue

Sets the blue color component for a constant color.

Valid values: integers
Default value: 160

colorgreen

Sets the green color component for a constant color.

Valid values: integers
Default value: 160

colorramp

Sets the color ramp for the surface when color scheme is Map Values From Volume.

Valid values: text strings
Default value: redwhiteblue

colorred

Sets the red color component for a constant color.

Valid values: integers
Default value: 160

datarange

Sets the data range to which we apply color ramp.

Valid values: on_surface
entire_volume
Default value: on_surface

dotradius

Sets radius for surface dots, unit is Angstroms.

Valid values: reals
Default value: 0.1
Minimum: 0.0001

entry

The entry that the surface belongs to.

Valid values: text strings
Default value:  

espcolorramp

Sets the color ramp for the surface when color scheme is electrostatic potential (ESP).

Valid values: text strings
Default value: redwhiteblue

espmax

Sets maximum ESP value for ESP color scheme.

Valid values: reals
Default value: 0.3
Minimum: 0.001

espmin

Sets minimum ESP value for ESP color scheme.

Valid values: reals
Default value: -0.3
Minimum: -0.001

iterations

Specify the number of iterations for Laplacian smoothing.

Valid values: integers
Default value: 35
Minimum: 0
Minimum: 100

linewidth

Sets line width for surface mesh, unit is pixel.

Valid values: reals
Default value: 0.025
Minimum: 0.0001

mapmax

Sets maximum mapping value.

Valid values: reals
Default value: 0

mapmin

Sets minimum mapping value.

Valid values: reals
Default value: 0

negativebluecolor

Sets the blue color component for the negative surface.

Valid values: integers
Default value: 255

negativecolor

Sets the color for the negative surface.

Valid values: text strings
Default value: white

negativegreencolor

Sets the green color component for the negative surface.

Valid values: integers
Default value: 255

negativeredcolor

Sets the red color component for the negative surface.

Valid values: integers
Default value: 255

scheme

Sets the color scheme.

Valid values: text strings
Default value:  

schemevolume

Sets the color scheme volume name.

Valid values: text strings
Default value:

showlegend

Sets whether or not show legends in workspace.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: true

smooth

Turn on/off surface smoothing processing.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: true

Operands:

<surface>

The name of the surface to set the color scheme for.
