bondorder

Increment, decrement or set the bond order for the specified bond.

Syntax: bondorder [increment|decrement|<bond_order> <atom1> <atom2> ]

Operands:

increment|decrement|<bond_order> <atom1> <atom2>

The first operand is either the word "increment", the word "decrement", or an integer value representing the bond order to be applied to the specifed bond. Note that "0" is a valid bond order. The second and third operands are the atoms which define the bond which is to have its bond order changed.
