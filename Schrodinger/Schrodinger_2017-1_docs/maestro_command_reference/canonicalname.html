canonicalname

Sets atom names to one generated from atom properties (such as element or PDB names) for all the atoms matched by the ASL operand.

Syntax: canonicalname [<separator> <option_flag> <ASL> ]

Operands:

<separator> <option_flag> <ASL>

The first operand is the separator symbol to be used in generating the canonical names of all atoms. The second operand is the option_flag, which indicates which two atom properties will be used to construct the canonical names. Possible values are "3" - from element and atom number and "12" PDB chain, residue and atom name. The final operand is the ASL expression to indicate which atoms are to have a canonical name set.
