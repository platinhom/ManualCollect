workspacecopy

Currently this is only used by Edit->Copy to determine if it generates in the current working directory a number of files corresponding the data it puts on the clipboard. It does not actually cause the copy to execute. This has to be done graphically.

Syntax: workspacecopy writefiles= yes | no

Options:

writefiles

If true, write out files corresponding to the clipboard data. If false, do not write out the files.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: false
