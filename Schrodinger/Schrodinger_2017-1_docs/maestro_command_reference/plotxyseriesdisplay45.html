plotxyseriesdisplay45

Shows or hides a 45-degree reference line for the given series.

Syntax: plotxyseriesdisplay45 [<plotname> <series> yes|no [slope] ]

Operands:

<plotname> <series> yes|no [slope]

The name of the plot to operate on, the series to operate on, and a yes or no to display or hide the 45-degree reference line. The slope is optional but should be 1 or -1.
