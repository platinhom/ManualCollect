phaseaddligands

Adds the given ligands to the table.

Syntax: phaseaddligands activity= <text> convert_activity= yes | no convert_scale= <x> [<ESL> ]

Options:

activity

This determines which property (if any) to use as the activity property for the ligands.

Valid values: text strings
Default value:  

convert_activity

Set to true if the activity values should be converted from concentration to -log[concentration]

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: false

convert_scale

A scale factor for conversion.

Valid values: reals
Default value: 1

Operands:

<ESL>

The entries to add as ligands.
