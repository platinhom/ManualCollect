qsiteresidue

Defines a residue as part of the QM region for an Impact QSite simulation

Syntax: qsiteresidue backboneresidue= yes | no backbonestring= <text> basis= <text> cuttype= <n> [<chain>:<molnum>:<resnum>:<inscode> ]

Options:

backboneresidue

The flag indicates if this residue is a lower or upper bound residue of Free ligand/ion residues.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: false

backbonestring

The string for all residues of a Free ligand/ion section.

Valid values: text strings
Default value:  

basis

The basis set for the residue to be included in the QM region of an Impact QSite simulation.

Valid values: text strings
Default value: lacvp*

cuttype

The cut type for the residue to be included in the QM region of an Impact QSite simulation.

Valid values: integers
Default value: 1
Minimum: 0
Minimum: 5

Operands:

<chain>:<molnum>:<resnum>:<inscode>

The number of a residue to be included in the QM region in a Impact QSite simulation.
