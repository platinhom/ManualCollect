Customizing Maestro with Resource Files

Maestro uses resource and configuration files to make settings for various parts of its operation, for example, for the display of colors and fonts. The default versions of the resource files are stored in subdirectories (folders) of the data subdirectory in the maestro-vversion directory of your installation. General resources are in the res subdirectory, which will be referred to as the Maestro resources directory. The default versions of project configuration files, rotamer resource files, and other resources are stored in the data subdirectory in the mmshare-vversion directory of the installation, which will be referred to as the installation resources directory.

You can customize resources for a particular Maestro profile by copying the resource or configuration files to the relevant profile subdirectory (folder) in your profiles directoryand then editing them. Instructions for editing these files are given in the topics listed below.

When Maestro is started, resources and configurations are initialized from the Maestro resources directory, then resources that are set in the profile directory are read, and replace those read from the installation. The resource files in the profile directory for a new release are copied from the profile directory for the previous release.

You can also change the standard resources and configurations by editing these files in the Maestro resources directory. If you do, the resources are changed for all users.
