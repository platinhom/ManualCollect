calculateangle

Measure bond angle between a triplet of atoms of all selected entries in the project. If any entry does not have associated atoms, then that entry is ignored.

Syntax: calculateangle [<atom1> <atom2> <atom3> <prop_name> ]

Operands:

<atom1> <atom2> <atom3> <prop_name>

The three atoms between which the angle is to be measured. Note that the specifying a-b-c is the same as specifying c-b-a.
