primexincreaseisovalue

Increase isovlaue for currently selected map surfaces.

Syntax: primexincreaseisovalue entry= <text> numsigma= <x> volume= <text> [<surface> ]

Options:

entry

The entry name of the entry that the surface belongs to.

Valid values: text strings
Default value:  

numsigma

The amount of increasing isovalue, e.g. 0.1*sigma.

Valid values: reals
Default value: 0.1
Minimum: 0.01

volume

The name of the volume that the surface belongs to.

Valid values: text strings
Default value:  

Operands:

<surface>

The name of the surface to increase the isovalue for.
