dihedral

Specifies a quartet of atoms to have their dihedral angle measured and displayed.

Syntax: dihedral xoffset= <x> yoffset= <x> [<atom1> <atom2> <atom3> <atom4> ]

Options:

xoffset

Specifies the X offset in Angstroms. Any dihedral created after this value is set will use this new value. Can be applied to an existing measurement by respecifying the measurement. Does not affect any already created dihedrals.

Valid values: reals
Default value: 0

yoffset

Specifies the Y offset in Angstroms. Any dihedral created after this value is set will use this new value. Can be applied to an existing measurement by respecifying the measurement. Does not affect any already created dihedrals.

Valid values: reals
Default value: 0

Operands:

<atom1> <atom2> <atom3> <atom4>

The four atoms between which the dihedral angle is to be measured. Note that the specifying a-b-c-d is the same as specifying d-c-b-a.
