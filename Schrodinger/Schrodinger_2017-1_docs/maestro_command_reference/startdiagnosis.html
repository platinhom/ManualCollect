startdiagnosis

This commands collects diagnostic information such as total system memory, available system memory, running maestro process peak virtual memory usage, current virtual memory usage, peak resident memory usage (only Linux), current resident memory usage (Only Linux), total swap disk space, free swap disk space, page fault etc. It also starts timer to collect user, sys, and real time information when explicitly called to stop it. All information are written into the file residing LINUX : $HOME/.schrodinger/maestro_diagnosis_<unique id>.txt Winows : AppData/Schrodinger//maestro_diagnosis_<unique id>.txt Unique file name is generated using second, minute, hour, day, month, and year. This file is created for each session.

Syntax: startdiagnosis
