entrygroupcollapse

Collapses the given group.

Syntax: entrygroupcollapse isexp= yes | no [<group_name> ]

Options:

isexp

Specifies whether the <group_name> is actual group name or it is expression such as 'selected'.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: false

Operands:

<group_name>

Group name. Collapses the given group and hides all the entries in that group.
