hidetoolbar

Hide the toolbar of given id displayed under given panel.

Syntax: hidetoolbar [<panel_name:toolbar_id> ]

Operands:

<panel_name:toolbar_id>

The first part is the name of the panel under which toolbar would be marked as hidden. The name must match to all characters and it is a case sensitive. The second part is the name of a toolbar id which needs to be marked as hidden. Id must match to all characters and it is a case sensitive. Any toolbar can not be displayed under any panel. There is a fix set of toolbars which can be displayed under given panel.
