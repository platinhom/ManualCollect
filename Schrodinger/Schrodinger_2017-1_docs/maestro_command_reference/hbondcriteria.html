hbondcriteria

Specify the criteria for calculating H-bonds in the workspace. The acceptor atom (A), H-bonded to the donor Hydrogen (H), is Oxygen, Nitrogen, Sulfur, or Fluorine.

Syntax: hbondcriteria acceptorangle= <x> acceptormaximumangleasacceptor= <x> acceptorminimumangleasacceptor= <x> acceptorminimumangleasdonor= <x> aromatichbondacceptorminangle= <x> aromatichbonddistance_n= <x> aromatichbonddistance_o= <x> aromatichbonddonormaxangle_n= <x> aromatichbonddonorminangle_n= <x> aromatichbonddonorminangle_o= <x> display= yes | no displayaromatichbond= yes | no displayhalogen= yes | no displayhbond= yes | no displaysaltbridge= yes | no distance= <x> donorangle= <x> donorminimumangleasacceptor= <x> donorminimumangleasdonor= <x> halogendistance= <x> saltbridgedistance= <x> set1= <text> set2= <text>
