symmetrizeworkspace

Symmetrizes the workspace finding the point groups.

Syntax: symmetrizeworkspace tolerance= <x> [analyze|update ]

Options:

tolerance

Tolerance for finding the pointing groups and symmetrizing the workspace

Valid values: reals
Default value: 0.04
Minimum: 0.04
Minimum: 1.00

Operands:

analyze|update

If analyze, finds the point groups for the current tolerance. If update, symmetrize the Workspace with the current tolerance.
