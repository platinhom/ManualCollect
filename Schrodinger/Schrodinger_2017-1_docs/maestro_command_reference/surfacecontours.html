surfacecontours

Control surface contour settings Edges are contoured when the distance along z to the nearest surface point behind it is at least the distance specified in the contour 'depth' value. The contour color is controlled by the red, green, and blue option values. The contour line's thickness is controlled via the 'thickness' option. The color rendered for the contour is additionally affected by 'intensity' which affects the final color by blending the contour color and the surface color according to 'intensity'.

Syntax: surfacecontours blue= <x> contours= yes | no depth= <x> green= <x> intensity= <x> red= <x> thickness= <x>

Options:

blue

Sets blue component of contour color

Valid values: reals
Default value: 0
Minimum: 0.0
Minimum: 1.0

contours

Display or hide contours on opaque surfaces. For transparent surfaces these are never shown regardless of what value this setting has.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: false

depth

Display contours where the distance to the closest surface point is at least this distance in z.

Valid values: reals
Default value: 1.2
Minimum: 0.0
Minimum: 5.0

green

Sets green component of contour color

Valid values: reals
Default value: 0
Minimum: 0.0
Minimum: 1.0

intensity

Intensity of the contour. A value of 1.0 is the most intense showing only the contour color. A value of zero is the least intense showing none of the contour color and only the surface color.

Valid values: reals
Default value: 0.2
Minimum: 0.0
Minimum: 1.0

red

Sets red component of contour color

Valid values: reals
Default value: 0
Minimum: 0.0
Minimum: 1.0

thickness

Sets thickness of contour

Valid values: reals
Default value: 0.1
Minimum: 0.0
Minimum: 1.0
