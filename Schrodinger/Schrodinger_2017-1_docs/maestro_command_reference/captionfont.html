captionfont

Font settings for the caption

Syntax: captionfont name= <text> size= <x> style= bold | italic | italicbold

Options:

name

Font name for the caption (helvetica, etc.)

Valid values: text strings
Default value: helvetica

size

Font size for the caption

Valid values: reals
Default value: 14
Minimum: 3
Minimum: 96

style

Font style for the caption (regular, bold, etc.)

Valid values: regular
bold
italic
italicbold
Default value: regular
