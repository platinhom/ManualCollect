dipolemoment

Set dipole moment options.

Syntax: dipolemoment ambient= <x> arrowwidth= <x> chargeblue= <x> chargegreen= <x> chargered= <x> diffuse= <x> diskblue= <x> diskgreen= <x> diskoffset= <x> diskred= <x> diskwidth= <x> display= yes | no emission= <x> frompositive= yes | no jagusrblue= <x> jagusrgreen= <x> jagusrred= <x> negativeblue= <x> negativegreen= <x> negativered= <x> positiveblue= <x> positivegreen= <x> positivered= <x> ringblue= <x> ringgreen= <x> ringred= <x> ringustep= <n> ringvstep= <n> ringwidth= <x> scale= <x> shaftwidth= <x> shininess= <x> showdisk= yes | no showpositive= yes | no specular= <x> transparency= <x> ustep= <n> vstep= <n>

Options:

ambient

Set material property - ambient, to its red, green, and blue components, for front face.

Valid values: reals
Default value: 0.5
Minimum: 0.0
Minimum: 1.0

arrowwidth

The width of dipole moment arrow.

Valid values: reals
Default value: 0.2
Minimum: 0.0001

chargeblue

The blue color component for dipole moments defined by partial atomic charges.

Valid values: reals
Default value: 0.5
Minimum: 0.0
Minimum: 1.0

chargegreen

The green color component for dipole moments defined by partial atomic charges.

Valid values: reals
Default value: 1
Minimum: 0.0
Minimum: 1.0

chargered

The red color component for dipole moments defined by partial atomic charges.

Valid values: reals
Default value: 0.8
Minimum: 0.0
Minimum: 1.0

diffuse

Set material property - diffuse, to its red, green, and blue components, for front face.

Valid values: reals
Default value: 0.4
Minimum: 0.0
Minimum: 1.0

diskblue

The blue color component of disk.

Valid values: reals
Default value: 0
Minimum: 0.0
Minimum: 1.0

diskgreen

The green color component of disk.

Valid values: reals
Default value: 1
Minimum: 0.0
Minimum: 1.0

diskoffset

The offset (percentage) of distance between the dipole moment disk and the positive end.

Valid values: reals
Default value: 0.05
Minimum: 0.0
Minimum: 1.0

diskred

The red color component of disk.

Valid values: reals
Default value: 1
Minimum: 0.0
Minimum: 1.0

diskwidth

The width of dipole moment disk.

Valid values: reals
Default value: 0.3
Minimum: 0.0001

display

Whether or not display dipole moment in workspace.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: false

emission

Set material property - emission, to its red, green, and blue components, for front face.

Valid values: reals
Default value: 0.15
Minimum: 0.0
Minimum: 1.0

frompositive

Set to true if the arrow is drawn from positive to negative.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: true

jagusrblue

The blue color component for dipole moments defined by Jaguar properties.

Valid values: reals
Default value: 1
Minimum: 0.0
Minimum: 1.0

jagusrgreen

The green color component for dipole moments defined by Jaguar properties.

Valid values: reals
Default value: 0.7
Minimum: 0.0
Minimum: 1.0

jagusrred

The red color component for dipole moments defined by Jaguar properties.

Valid values: reals
Default value: 0.8
Minimum: 0.0
Minimum: 1.0

negativeblue

The blue color component of dipole moment markers for negative region.

Valid values: reals
Default value: 0
Minimum: 0.0
Minimum: 1.0

negativegreen

The green color component of dipole moment markers for negative region.

Valid values: reals
Default value: 0
Minimum: 0.0
Minimum: 1.0

negativered

The red color component of dipole moment markers for negative region.

Valid values: reals
Default value: 1
Minimum: 0.0
Minimum: 1.0

positiveblue

The blue color component of dipole moment markers for positive region.

Valid values: reals
Default value: 1
Minimum: 0.0
Minimum: 1.0

positivegreen

The green color component of dipole moment markers for positive region.

Valid values: reals
Default value: 0
Minimum: 0.0
Minimum: 1.0

positivered

The red color component of dipole moment markers for positive region.

Valid values: reals
Default value: 0
Minimum: 0.0
Minimum: 1.0

ringblue

The blue color component of the ring.

Valid values: reals
Default value: 1
Minimum: 0.0
Minimum: 1.0

ringgreen

The green color component of the ring.

Valid values: reals
Default value: 1
Minimum: 0.0
Minimum: 1.0

ringred

The red color component of the ring.

Valid values: reals
Default value: 0
Minimum: 0.0
Minimum: 1.0

ringustep

The U step domain tolerance for the ring.

Valid values: integers
Default value: 5
Minimum: 2

ringvstep

The V step domain tolerance for the ring.

Valid values: integers
Default value: 3
Minimum: 2

ringwidth

The width of the ring around the disk.

Valid values: reals
Default value: 0.05
Minimum: 0.0001

scale

The scale to convert unit from Debye to Angstrom for the length of dipole moment arrow.

Valid values: reals
Default value: 1
Minimum: 0.0001

shaftwidth

The width of dipole moment shaft.

Valid values: reals
Default value: 0.1
Minimum: 0.0001

shininess

Set material property - shininess, for front face.

Valid values: reals
Default value: 80
Minimum: 0.0
Minimum: 128.0

showdisk

Set to true if the disk for positive sign is shown.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: true

showpositive

Set to true if positive and negative colors are used.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: true

specular

Set material property - specular, to its red, green, and blue components, for front face.

Valid values: reals
Default value: 0.7
Minimum: 0.0
Minimum: 1.0

transparency

The transparency of dipole moment markers.

Valid values: reals
Default value: 0.75
Minimum: 0.0
Minimum: 1.0

ustep

The U step domain tolerance for the cylinder.

Valid values: integers
Default value: 3
Minimum: 2

vstep

The V step domain tolerance for the cylinder.

Valid values: integers
Default value: 5
Minimum: 2
