visexport

Export a volume to the given file.

Syntax: visexport entry= <text> format= cnsformatted | cnsccp4 surface= <text> surfacetype= <text> surfacetypecomment= <text> volume= <text> [<file name> ]

Options:

entry

The entry name of the entry that the surface belongs to.

Valid values: text strings
Default value:  

format

The format of the file that will be exported.

Valid values: cnsunformatted
cnsformatted
cnsccp4
Default value: cnsformatted

surface

The name of the surface.

Valid values: text strings
Default value:  

surfacetype

The surface type of the surface that will be exported.

Valid values: text strings
Default value:  

surfacetypecomment

The surface type comment of the surface that will be exported.

Valid values: text strings
Default value:  

volume

The name of the volume that the surface belongs to.

Valid values: text strings
Default value:  

Operands:

<file name>

The name of the visio file to export.
