entrywscreate

Create project entry from atoms in the workspace. If source entry is only one entry, then it also copies properties in the newly created entry.

Syntax: entrywscreate removefromsource= yes | no [<new_name> <ASL> ]

Options:

removefromsource

This determines whether or not atoms matching given ASL should be removed from the one or more associated source entries.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: false

Operands:

<new_name> <ASL>

The name to be given to the created project entry, followed by a valid ASL expression to specify which atoms constitute the new project entry.
