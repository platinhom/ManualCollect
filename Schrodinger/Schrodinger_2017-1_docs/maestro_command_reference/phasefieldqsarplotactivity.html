phasefieldqsarplotactivity

Creates a scatter plot of predicted versus actual activities for the Field-Based QSAR Model.

Syntax: phasefieldqsarplotactivity draw_line= yes | no pls_factor= <n> [<all|selected|training|test> ]

Options:

draw_line

If set, then the plot will include a line at 45 degrees.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: true

pls_factor

This is which set of PLS factors to plot.

Valid values: integers
Default value: 1
Minimum: 1

Operands:

<all|selected|training|test>

Which ligands to plot: all, selected, the training set, or the test set.
