bondproplabel

Turns the display of bond properties on or off. These properties are read from the structure file at run-time.

Syntax: bondproplabel display= yes | no [<property display name> ]

Options:

display

A boolean option which determines if the bond property is to be displayed or not in the current label definition.

Valid values: boolean (true|false; yes|no; y|n; on|off)
Default value: true

Operands:

<property display name>

The name of the bond property. It's not an error if this property does not exist. Note this can be the 'Display Name' as it appears in the bond labels panel or the 'Data Name' as it appears in the Maestro format file.
