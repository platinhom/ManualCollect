plotxysaveimage

Capture the current XY plotting window and save it to an image file.

Syntax: plotxysaveimage format= jpeg [<file_name> ]

Options:

format

Specifies the format of the saved image.

Valid values: tiff
jpeg
Default value: tiff

Operands:

<file_name>

The file where the image will be saved.
