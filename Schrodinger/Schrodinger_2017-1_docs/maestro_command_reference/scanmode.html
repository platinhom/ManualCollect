scanmode

Set the Coordinate Scan mode.

Syntax: scanmode modetype= angle | dihedral

Options:

modetype

Determines the mode for coordiante scan softlimit=<n> <input_file_name>

Valid values: distance
angle
dihedral
Default value: distance
