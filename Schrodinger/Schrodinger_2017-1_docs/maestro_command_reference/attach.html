attach

Attach the currently selected fragment to the specified atom. The bond to this atom (and the atom itself) will be replaced by a bond from the incoming fragment.

Syntax: attach [<atom_num> ]

Operands:

<atom_num>

The atom number defines the bond to which the fragment be attached. The atom specifed must be a terminally attached atom.
