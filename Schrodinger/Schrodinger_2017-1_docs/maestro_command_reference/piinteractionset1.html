piinteractionset1

Specify the first set of atoms used in finding Pi interactions in the 3D workspace.

Syntax: piinteractionset1 [<ASL> ]

Operands:

<ASL>

A string in the atom specification language. Typical usage is to define piinteractionset1 and piinteractionset1. This set, piinteractionset1, defines the "from" atoms. The piinteractionset2 atoms define the "to" atoms. Pi interactions are calculated between these two sets. That is, the Pi interactions are inter-set Pi interactions. No intra-set Pi interactions are calculated. If piinteractionset2's ASL string is empty, then Pi interactions are calculated for all atoms in piinteractionset1.
