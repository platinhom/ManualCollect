proteinsculpting

Define an atom for a protein sculpting operation and/or translate it by X, Y, Z angstroms

Syntax: proteinsculpting [<atom> [ <X> <Y> <Z> ] ]

Operands:

<atom> [ <X> <Y> <Z> ]

An atom number to be transformed and the X, Y and Z extents for transformation.
