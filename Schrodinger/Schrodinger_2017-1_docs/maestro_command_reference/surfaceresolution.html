surfaceresolution

Set surface displaying resolution options.

Syntax: surfaceresolution lowresstyle= mesh | dot resolution= low

Options:

lowresstyle

Specifies drawing style for sloid surfaces when rotating. Valid values are `solid', `mesh', and `dot'.

Valid values: solid
mesh
dot
Default value: solid

resolution

Set the resolution for drawing solid surfaces to low or high .

Valid values: high
low
Default value: high
