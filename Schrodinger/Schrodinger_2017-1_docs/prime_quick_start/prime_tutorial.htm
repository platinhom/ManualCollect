Homology Modeling With Prime

This set of exercises provides a tutorial introduction to using Prime for protein structure prediction by homology modeling.

  • Homology Modeling of a Single Chain provides an introduction to homology modeling for a single chain that covers most of the tasks that are performed in homology modeling, including running a BLAST search, selecting a template, editing the alignment, building the model, and refining the model.

  • Homology Modeling of a Homomultimer provides an exercise on building a homology model for a homomultimer with four identical chains. This exercise includes the preparation of a tetrameric template in the Protein Preparation Wizard panel from a template that includes only the asymmetric unit.

  • Homology Modeling of a Heteromultimer provides an exercise on building a homology model for a heteromultimer with two different chains.

It is assumed that you have already installed Maestro and Prime, and have access to the third-party programs and databases (PDB, BLAST, HMMER/Pfam). If you are running Prime on Linux, it is assumed that you have downloaded and installed the optional (but highly recommended) third-party secondary structure prediction program PSIPRED. To find out how to obtain third-party programs, go to the Third Party Programs page of our website.

The tutorial is designed to run from a local copy of the PDB and BLAST databases. If you run the tutorial with web download, you might notice some differences.
