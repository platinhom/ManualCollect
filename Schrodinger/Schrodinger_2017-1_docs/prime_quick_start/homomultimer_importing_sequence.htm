Importing the hERG Sequence

Now that the template is prepared, you can start the homology modeling.

  1. If the Structure Prediction panel is not already open, open it now by choosing Tasks → Biologics → Advanced Homology Modeling in the Task Tool.

    The Structure Prediction panel opens at the first step, Input Sequence. By default the Guide is not displayed. If you want to display it, choose Step → Guide.

  2. Choose File → New, and name the run hERG.

  3. For Get sequence from, click File.

  4. Select hERG.fasta, then click Open.

    The sequence is displayed in the Prime sequence viewer.

  5. Click Next to proceed to the next step, Find Homologs.
