Homology Modeling of a Homomultimer

This exercise builds a homomultimer for hERG, which is a tetramer. The template that will be used is 1orq. This is not the highest scoring template that can be found from a BLAST search, but it is chosen to illustrate the use of a template in a realistic situation, where the similarity may not be high.

PDB structures often only include the asymmetric unit, so it may be necessary to build the multimeric template structure by applying the crystallographic symmetry operations. In the first part of the exercise, you will produce the desired tetrameric template from the PDB structure 1orq. Following that, you will create the homology model for hERG.
