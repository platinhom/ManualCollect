WScore Visualizer Panel

Visualize WScore scoring terms in the Workspace, showing the parts of the structure that contribute to the score.

To open this panel, click the WS button for the desired ligand in the Entry List.

WScore Visualizer Panel Features

  • Receptor and ligand titles
  • Table
Receptor and ligand titles

These text fields report the title of the receptor and the title of the ligand that were used for the visualization.

Table

The table lists the WScore terms with values and descriptions. Not every term has a visualization, but all terms are listed.

View Column of buttons for visualizing terms. The button is present if the term has a visualization. Only one term can be visualized at a time.
WScore Term Name of the WScore term
Score Value of the WScore term for the particular instance of the term.
Trigger Indicates whether the model generation stopped due to triggering an issue for this term. This column is only displayed for the ligands that contributed to the model. Typically, inappropriate penalties trigger for actives, whereas inappropriate rewards trigger for decoys.
Description Description of the WScore term.
