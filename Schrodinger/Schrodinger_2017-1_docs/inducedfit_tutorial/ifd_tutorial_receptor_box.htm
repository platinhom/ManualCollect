Defining the Receptor and Enclosing Box for IFD

The receptor must be distinguished from the complexed ligand in order for the Glide grid generation portion of the protocol to run correctly, and the parameters of the grid box defined.

1. Ensure that the Box center option selected is Centroid of Workspace ligand.
2. Select Pick to the right of the Centroid of Workspace ligand text box.
3. Click on a ligand atom in the Workspace.

The ligand is colored orange. When you pick the ligand, it is listed in the text box, and the grid center, the ligand center box (green) and the grid box (purple) are displayed in the Workspace. The ligand is marked with green markers.

4. Ensure that the Box size option selected is Auto (the default).

The position and size of the grid box (or enclosing box) are defined automatically, based on the selected ligand.

You can also specify hydrogen-bond and metal constraints, and choose the docking stage to which they apply. In this tutorial, constraints are not used.
