To convert the relative binding free energies to absolute binding free energies, we need to know the absolute binding free energy for at least one of the molecules. In retrospective studies, in principle, we can choose any molecule as the reference compound, and force the predicted ΔG value of the reference compound to be equal to the experimental value ΔGexp. However, if the experimental measurement for the reference compound has a large error, the error will propagate to the estimated binding free energies for all the other molecules using this method. So to minimize the error introduced in choosing an arbitrary compound as the reference compound, we instead choose all the compounds with experimental binding free energies available as the reference.
