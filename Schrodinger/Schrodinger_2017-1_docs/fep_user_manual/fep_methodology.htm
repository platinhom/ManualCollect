FEP+ Methodology

FEP+ is a tool for accurate calculation of the relative binding free energies among congeneric ligands. The key components of FEP+, with their primary citations, are:

  • State of the art protein and small molecule force fields, OPLS3 [1]
  • Enhanced sampling method, FEP/REST [2,3,4]
  • Automated setup and analysis tools [5]
  • Robust error estimates through cycle closure [6]
  • Large scale validation [5]
  • GPU hardware acceleration

A brief introduction to FEP theory, the FEP/REST enhanced sampling method, and the cycle closure error estimate are given in the topics listed below.
