Display Tasks for Correcting Protein Structures

When making corrections, it is useful to display information that is relevant to the task: for example, to display formal charges when correcting the charge or the atom type. This section presents some procedures for displaying information. Display tasks are performed using the Style toolbox.

To display formal charges for all atoms:

  1. Click the All button on the Select toolbar.

    You can also clear the selection, as the action is applied to all atoms if there is no selection.

  2. Choose Formal Charge from the Apply Labelsmenu in the Style toolbox.

To display element labels and formal charges for all atoms:

  1. Click the All button on the Select toolbar.

  2. Choose Edit Custom Labelfrom the Apply Labelsmenu in the Style toolbox.

    The Edit Custom Label dialog box opens.

  3. Click Add.

    The Add Label Fields dialog box opens.

  4. Select Element and Formal charge from the Atom properties list.

  5. Click OK in the Add Label Fields dialog box and then in theEdit Custom Label dialog box.

    All atoms in the Workspace are labeled.

To clear all labels:

  1. Click the All button on the Select toolbar.

  2. Click the Delete labelsbutton in the Style toolbox.

To undisplay the protein:

  1. Click the Select proteins (only) button under Quick Select on the toolbar.

  2. Click the Undisplay atomsbutton in the Style toolbox.

This is useful for locating metal ions, cofactors, and waters.

To display only the cofactor:

  1. Click the Define button on the Select toolbar.

    The Atom Selection dialog box opens.

  2. In the Residue tab, choose Residue Type.

  3. Click the residue type of the cofactor, which will be near the end of the list.

    The cofactor is highlighted. If there is more than one of this type, they are all highlighted.

  4. Click Add, then click OK.

  5. Click the Display selected atoms only button in the Style toolbox.

The cofactor is displayed. Because the cofactor was chosen by residue type and not molecule number, this method works even if the cofactor is covalently bonded to another residue. If there is more than one cofactor, they are all selected.
