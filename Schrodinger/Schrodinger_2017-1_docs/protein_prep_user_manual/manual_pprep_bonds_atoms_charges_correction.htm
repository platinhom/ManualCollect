Atom and Bond Correction Tasks

The basic tasks for correcting bond orders, atom types, and formal charges are given below. It is a good idea to display formal charges for both atom type changes and formal charge changes, since atom typing can change the formal charge, and Maestro corrects the formal charges of surrounding atoms to compensate, if necessary.

To delete bonds use one of the following methods:

  1. Select the bonds to be deleted.

  2. Click the Delete bonds button menu in the 3D Builder panel.

or

  • Right-click on the bond and choose Delete from the shortcut menu.

To set or change bond orders:

  1. Select the bonds whose order you want to increase (decrease).

  2. Click the Increment bond order (Decrement bond order) button.

or

  • Right-click on a bond and choose Increase Bond Order or Decrease Bond Order.

To set or correct the formal charge on any atom:

  1. Select the atom or atoms whose formal charge you want to increase (decrease).

  2. Click the Increment formal charge (Decrement formal charge) button as many times as necessary.

To correct atom types:

  1. Pick the atom to be changed.

  2. Choose Change Other Properties from the Other Edits menu in the 3D Builder panel.

  3. Choose Atom Type (MacroModel) from the Property option menu.

  4. Select the correct atom type from the list.

    Atom types include both element name and formal charge. Atom type numbers are in parentheses.

  5. Click Apply, then click Close.

If you want to change multiple atoms you can leave the panel open, pick other atoms, choose the relevant type, and click Apply.
