Displaying Hydrogens for Protein Preparation

The many hydrogen atoms in an all-atom PDB structure can clutter the display, making it difficult to see what is important. The Protein Preparation Wizard provides three options for display of hydrogen atoms:

  • None—Do not display hydrogens
  • Polar only—Display only the polar hydrogens
  • All ligand, polar receptor—Display all ligand hydrogens, but only the polar hydrogens on the receptor
  • All—Display all hydrogens

If hydrogens are added later, they are displayed according to the options selected here. It is useful to display only the polar hydrogens when checking hydrogen bonds, for example.
