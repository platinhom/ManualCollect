Save Custom View Dialog Box

Save a custom view as a named view.

Save Custom View Dialog Box Features

  • Use selected rows and Use all rows options
  • Name text box
  • Description text box
  • Number of rows text box
  • Number of columns text box
  • Open view option
Use selected rows and Use all rows options

Select Use selected rows to include only the selected rows in the new view. The selection must consist of entire rows for this option to be available. Select Use all rows to include all displayed rows in the new view.

Name text box

Enter a name for the view in this text box.

Description text box

Enter a description of the view in this text area. You can change the description later in the Details dialog box.

Number of rows text box

Number of rows in the new view. Not editable.

Number of columns text box

Number of columns in the new view. Not editable. If rows are selected, the value displayed is 0, meaning all columns.

Open view option

Select this option to open the new view when you click OK to create the view. The parent view remains open.
