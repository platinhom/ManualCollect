Choose Random Subset Dialog Box Features

  • Choose rows option menu and box
  • using seed text box
  • Generate Seed button
  • Store results in options
  • Permute rows option
Choose rows option menu and box

Select the means of choosing the number of rows, from by percent or by count, and specify the percentage or the count in the corresponding box.

using seed text box

Specify the seed for random number generation in this text box.

Generate Seed button

Generate a random seed. The seed is displayed in the text box to the left.

Store results in options

Choose an option for storing the results. If you choose Current view, the view is altered to show only the randomly selected rows. If you choose New view, the Save Custom View dialog box opens, in which you can name and describe the new view, and choose to open it.

Permute rows option

Randomize the row order in the random subset. The default is to leave the rows in their original order.
