Export Options Dialog Box

In this dialog box, you can select the properties to export, including fingerprints, and set options for how structures and fingerprints are exported. Note that properties are exported in the order that they appear in the view you export them from.

Export random subset of N rows option and box

Select this option to export a random subset of the rows in the spreadsheet, and specify the number of rows in the box.

Seed text box

Specify a seed for the random subset selection.

Generate Seed button

Generate a seed for the random subset selection. The seed is displayed in the Seed text box.

Export fingerprint option and menu

Select this option to export a fingerprint, either to a CSV file (SMILES+Properties) as a set of properties with values of 0 and 1 for each bit in the fingerprint, or to a binary fingerprint file (.fp). Choose the fingerprint from the option menu.

Property selection tools

Select the properties to export to file with the property selection tools.

Export unique SMILES option

When exporting to a SMILES format, export unique (canonical) SMILES. If this option is not selected, export a SMILES string that depends on the atom order in the structure (usually the first atom in the SMILES string is the first atom in the structure).

Use MDL version 3.0 format for SD files option

Use the V3000 format when writing SD files. This is a free-format version that allows the writing of structures with more than 999 atoms. The default is to use the V2000 format for SD files, which is fixed-format.

Export largest fragment only option

If a structure in the spreadsheet contains disconnected fragments (e.g. water molecules), export the largest fragment and discard the rest.

Force 2D coordinates options

If the structures have 3D coordinates, convert them to 2D coordinates and export the structure in 2D. When you select this option, the options for exporting hydrogens become available:

  • Suppress hydrogens—Do not export hydrogens
  • Export polar hydrogens only—Export only the polar hydrogens (those attached to electronegative atoms).
  • Export all hydrogens—Export all hydrogen atoms, regardless of polarity.
Export Canvas UIDs option

Select this option to export the Canvas unique identifiers to the file as a property. Exporting the UID allows you to re-import the properties to the same project and use the UID to match the structures in the file to the records in the project.

Autoscale properties option

Shift and scale each numeric property exported so that its mean is 0 and its standard deviation is 1. To autoscale only a subset of properties, select these properties in the Selected properties list.

Fingerprint Options button

Display the options for exporting fingerprints in the lower part of the panel. The options described below are the options that are displayed.

Export bits options

Choose an option for the bits to export. If you choose Export the N most significant bits, you can set the number of bits to export in the box. The significance is determined relative to the fingerprints for the selection of rows being exported. The default is 200 bits for CSV files, and all bits for binary (.fp) files.

Prefix for bit columns text box

Set the prefix for the bit column names. This option is only available for export to CSV files.

Save bit set option and text box

Select this option to save data that identifies the bit set in the project, so that it can be identified later and associated with the source fingerprint. Specify the name of the bit set in the text box.

Export to file option, text box and Browse button

Export the bit set to a bit set file (.bs). Specify the file name in the text box or click Browse and navigate to the file.

Use saved bit set option and menu

Select this option to use an existing bit set to identify the bits, and select the name of the bit set from the option menu. The items on the option menu are the saved bit sets for the selected fingerprint. The Prefix for bit columns text box is filled in with the saved prefix.

Import bit set from external file text box and Browse button

Import a previously exported bit set from an external bit set file (.bs). Enter the file name (including the path) in the text box, or click Browse and navigate to the file.
