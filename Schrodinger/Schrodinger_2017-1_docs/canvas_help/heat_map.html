Heat Map Dialog Box

Color the cells in the current view according to values of their properties.

Heat Map Dialog Box Features

  • Show Heat Map option
  • Color Space options
  • Properties list and search box
  • Add and Remove buttons
  • Heat map table
Show Heat Map option

Select this option to display the background coloring of the cells. Deselect this option to turn off the background coloring.

Color Space options

Select the scheme for determining the mapping of colors to values. The options are:

  • RGB—Color is interpolated linearly between the RGB colors for the maximum and minimum values
  • Rainbow (HSV)—Color follows the spectrum (rainbow) between the maximum and minimum values
  • Transparency—Transparency increases from the maximum value and the minimum value towards the mean value (which is white).
  • Color Wheel (discrete)—A selection of colors from the spectrum is applied to individual integer values. For real-valued properties, the value is rounded down to the nearest integer and the color assigned based on the integer value.
Properties list and search box

Lists the properties that are available for coloring in the current view. You can select multiple properties to be added to the list of properties to which the heat map is applied.

Add and Remove buttons

Use these buttons to move properties between the Properties list and the heat map table. Select one or more properties and click the appropriate button to move those properties.

Heat map table

This table displays the properties for which a heat map is defined. The table columns are described below.

Property Name of the property
Display Checkbox for displaying the heat map for this particular property.
Min Minimum value of the property, and color used for the minimum value. Right-click in the table cell to edit the color.
Max Maximum value of the property, and color used for the maximum value. Right-click in the table cell to edit the color.
