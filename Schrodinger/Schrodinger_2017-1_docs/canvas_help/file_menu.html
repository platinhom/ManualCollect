File Menu

The File menu allows you to create, open and close projects, import and export data, and quit Canvas. Canvas projects are saved whenever a change is made, so there is no Save menu item. You must have a Canvas project open to import structures and data.

New Project

Create a new Canvas project. Opens a file selector in which you can navigate to a location and name the project. If a project is already open, this project is closed first.
Keyboard shortcut: CTRL+N (⌘N)

Open Project

Open an existing project. Opens a file selector in which you can navigate to a location and select the project, which can be a regular project (.cnv) or a zipped project (.cnvzip or .cnv.zip). Zipped projects are unzipped in place (that is, in the same location as the zip file). If the project directory is read-only, the project is opened in read-only mode, and no changes can be made to the project data.
Keyboard shortcut: CTRL+O (⌘O)

Recent Projects

Open one of the most recently used projects. The submenu displays a list of up to 10 projects.

Archive Project

Archive the current project in zip format. Opens a file selector, in which you can name the archive file. The default name is the name of the project, and the extension used is .cnvzip. Archiving a project is a convenient way of moving it to a new location or sending it to a collaborator.

Close Project

Closes the current project. The data in a project is automatically saved whenever you make a change.
Keyboard shortcut: CTRL+W (⌘W)

Import

Import all structures and properties from a Maestro file, an SD file, a CSV file with SMILES strings and properties, or a SMILES file with SMILES strings and optional titles. Opens a file selector in which you can navigate to a location, choose the format, and select the file, then opens a dialog box in which you can set options for importing and matching the properties.

You can select multiple files for import in all formats except CSV. If you do, all structures in all files are appended to the project, and the dialog box is not displayed.

The progress of the import is shown in the status bar, along with a button that you can use to stop (abort) the import operation.

Import Project

Import structures and data from another Canvas project. All structures and all data are imported. Opens a file selector in which you can navigate to a location and select the project.

The progress of the import is shown in the status bar, along with a button that you can use to stop (abort) the import operation.

Export

Export structures and properties to a Maestro file, an SD file, a CSV file with SMILES strings and properties, or a SMILES file with SMILES strings and optional titles. Opens a file selector in which you can navigate to a location, choose the format, and name the file, then opens a dialog box in which you can set options for the properties or fingerprint to export.

The progress of the export is shown in the status bar, along with a button that you can use to stop (abort) the export operation.

Export to Maestro

Export all the structures and properties from the Canvas project to a temporary Maestro file, then start Maestro and import the structures.

The progress of the export is shown in the status bar, along with a button that you can use to stop (abort) the export operation.

Preferences

Set preferences to customize Canvas layout and behavior. Opens the Preferences dialog box.

Quit

Close the current Canvas session.
Keyboard shortcut: CTRL+Q (⌘Q)
