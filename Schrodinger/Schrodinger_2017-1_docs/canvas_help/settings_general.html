General Settings Dialog Box

This dialog box allows you to set the title of the plot and the foreground and background colors.

General Settings Dialog Box Features

  • Chart title text box
  • Font size box
  • Color button
  • Foreground button
  • Background button
Chart title text box

Enter the title of the chart in this text box. The title is displayed at the top of the chart area; it is different from the name of the chart, which is used to identify the chart in Canvas.

Font size box

Specify the size of the font to use for the chart title.

Color button

Specify the color of the title. Opens a standard color chooser so that you can select the color. The color is displayed to the right of the button. Default: black.

Foreground button

Specify the color of the foreground, i.e. the chart itself. Opens a standard color chooser so that you can select the color. The color is displayed to the right of the button. Default: black.

Background button

Specify the color of the background of the chart. Opens a standard color chooser so that you can select the color. The color is displayed to the right of the button. Default: white.
