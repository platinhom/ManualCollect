Self-Organizing Map Viewer Panel

This panel displays a self-organizing map and provides choices for coloring the map.

File menu

This menu contains three items.

  • Close—Close the panel.

  • Export—Export cell data to a file. Opens a dialog box in which you can choose to export to a file or to the spreadsheet. Two properties are exported with the structures: Cell, which is the cell index (running by rows from the top left) and Total, which is the number of structures in the cell.

    If you choose To file, click Browse to navigate to the location for the file, select the file format from Maestro, SD, or CSV, and name the file. You can export the structures in each cell to a separate file, labeled with the cell index, by selecting Place results into separate files. CSV files include the structure name, a SMILES string for the structure, and the properties.

    If you choose To spreadsheet, the column names for the properties are derived from the property name by adding a prefix, which you can specify in the Column prefix text box. The resulting column names are prefix:Cell, and prefix:Total. The default prefix is SOM:jobname.

  • Save Image—Save an image of the map in PNG, JPEG, or TIFF format. Opens a dialog box in which you can name the file and set the dimensions of the map, in pixels.

Properties menu

This menu contains two items.

  • Numeric—Opens a dialog box in which you can choose properties that are not necessarily in the map for coloring by value. The properties you choose are displayed on the Average property value option menu.

  • Categorical—Opens a dialog box in which you can choose properties that are not necessarily in the map for coloring by category. The properties you choose are displayed on the Category option menu.

Help menu

This menu contains one item, SOM Viewer Help, for displaying this help topic.

Display area

Displays the map. The cells are colored according to the coloring options. The shade of gray in the cell borders indicates the distance between adjacent cells: the darker the border, the larger the distance.

For properties other than the distance to the selected cell, you can right-click on a cell to open a custom view showing the structures that are in the cell. To add other cells to this view, control-right-click the cells. Cells for the current view are outlined in yellow.

Color by options

Select an option to color the map.

  • Distance to selected cell—Color the map by the distance to the selected cell, which is outlined in red. Click on a cell in the map to select it and color the map by the distance to this cell.
  • Cell population—Color the map by the number of compounds in each cell.
  • Cell property value—Color the cell according to the property chosen from the option menu.
  • Average property value—Color the cell according to the average value of the property chosen from the option menu. This menu is populated by choosing Numeric from the Properties menu, and selecting the properties you want to use from the dialog box that opens.
  • Category—Color the cell according to the number of structures in the cell with the selected value of the selected categorical property. The menu of categorical properties is populated by choosing Categorical from the Properties menu, and selecting the properties you want to use from the dialog box that opens.
Display values in cells option

Select this option to display the property value in the cell that was used to determine the cell color.

Min and Max buttons

Choose the colors for the minimum and maximum values of any quantity used to color the map. Each button opens a color chooser. The color is displayed below the button. Intermediate values are interpolated on a spectrum.

Bar and boxes

The boxes display the minimum and maximum values of the property used to color the map. You can adjust these values so that the cells with values outside the range specified in the boxes are colored by the colors for the minimum and maximum. The bar above these boxes indicates the range of possible values for which a color ramp is applied. You can drag the sliders on the bar to adjust the minimum and maximum property values used.

Distance between cells legend

The distance between adjacent cells is indicated by grayscale coloring of the border between the cells. This legend provides a scale of the distances.
