Analyze Atomic Contributions Dialog Box

This dialog box presents an analysis of the atomic contributions of a structure to its predicted property value from the kernel-based PLS regression.

Using the Analyze Atomic Contributions Dialog Box

You can use this dialog box to examine the contributions from each atom to the predicted property. Pausing the pointer over the atom shows its contribution. The observed and predicted properties are listed at the top. The color coding indicates the size (intensity) and sign (red or blue) of the contribution.

As well as examining contributions, you can edit the structure to create a new structure, by clicking Edit Structure. When you click OK, the new structure is displayed, its fingerprint is calculated, and the contributions are reanalyzed and displayed. The observed value is still that of the original, but the predicted value is the value for the new molecule. You can then save the new molecule to the project by clicking Save to Project. It is added as a new row, with the predicted property value.

Analyze Atomic Contributions Dialog Box Features

  • Information text boxes
  • Structure display area
  • Edit Structure button
  • Reset button
  • Save to Project button
Information text boxes

The Observed and Predicted text boxes show the observed and predicted property value for the structure. The Model set box shows whether the structure was in the training set or the test set.

Structure display area

The 2D structure of the molecule is displayed in this area. Each atom is colored according to its contribution to the property, as shown in the Model Visualization panel. Pausing the pointer over an atom displays the contribution to the property due to that atom and the atom number. Double-clicking an atom opens the Fragment Contribution dialog box, which shows the fragments that include the atom and the contributions from each fragment to the property and the contribution from each fragment to the atom's contribution to the property. The fragment contribution is partitioned equally between the atoms in the fragment.

Edit Structure button

Edit the 2D structure in the Edit Structure dialog box. This allows you to make changes on the basis of the atomic contribution information. Fingerprints are generated for the edited structure, and the new predicted property value is displayed and analyzed into atomic contributions.

Reset button

Reset the structure to the original, after editing. This button is essentially an Undo button.

Save to Project button

Save the structure to the project. You can save the original structure or an edited version of the structure. Opens the Export to Project dialog box, in which you can name the structure and set the column prefix for the predicted property. Only the predicted property is added to the spreadsheet,
