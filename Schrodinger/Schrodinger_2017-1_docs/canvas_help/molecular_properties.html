Molecular Properties Dialog Box

In this dialog box you can set up and run a job to calculate various molecular properties (descriptors).

Molecular Properties Dialog Box Features

  • Data set option menu
  • Save results as text box
  • Descriptor class tree
  • Calculate class Descriptors option
  • classDescriptors Job Options section
  • Physicochemical Descriptors options
  • Topological Descriptors options
  • LigFilter Descriptors options
  • QikProp Descriptors options
  • Compute button
Data set option menuSelect the source of the data. The data source can involve both row and column selection. The choices are: Selected rows—Use the rows that are selected in the master view. You must select the rows before opening the dialog box. Visible rows—Use the rows that are displayed (visible) in the master view. You must ensure that the required rows are visible before opening the dialog box. Saved view—Use the rows and columns in a chosen saved view. When you select this item, an option menu that lists the saved views in the project is displayed to the right. You can choose the desired view from this option menu. A second option menu, labeled Domain, allows you to choose whether to restrict the rows or the columns, or both, to those of the saved view. Otherwise the visible rows or columns of the Master View are used. Original data set for jobname—Use the rows and columns that were used for the specified job. This item is only available when you are cloning a job or viewing the results of a job.
Save results as text box

Provide a name for the property calculation. This name is listed under Molecular Properties in the Project View.

Descriptor class tree

When you choose a descriptor class in the tree, the options for this class are displauyed in the panel to the right.

Calculate class Descriptors option

Select this option to calculate the descriptors for the particular descriptor class. This option is present for each descriptor class.

class Descriptors Job Options section

Set options for running of the job for the descriptor class. These options are set independently for each class, so that you can run each calculation on a different host with different job options.

  • Host—Select the host on which to run the job.
  • CPUs—Set the number of processors (cores) to use for the job.
  • Incorporate—Choose whether or not to incorporate the results into the project automatically on completion.
Physicochemical Descriptors options
Available properties list

Select the properties that you want to calculate from this list.

If you select Rotatable bonds, you can also choose Use LigParse definitions to use the definitions from LigParse for rotatable bonds. The Canvas definitions differ from the LigParse definitions by the exclusion of amide bonds and bonds that connect to a terminal heavy atom (one that otherwise only has H atoms attached, such as methyl, amino, or hydroxyl).

If you select Hydrogen bond donors, you can also select Count multiple to count donors that can donate multiple hydrogen bonds (e.g. an NH2 group) as multiple donors.

Advanced button

Open the Molecular Properties - Advanced Options dialog box, in which you can customize some of the property calculations.

Consider only the largest fragment from each structure option

Perform calculations of properties on the largest fragment in each structure. This may be necessary if the structure consists of several molecules, such as a ligand, waters, and ions.

Topological Descriptors options

These options consist of a standard set of property selection tools.

LigFilter Descriptors options
Select All button

Use this button to select all LigFilter descriptors.

General attributes list

Select attributes from this list for calculation.

Functional group counts tree

Select specific functional groups to count in each structure. The functional groups are grouped and presented in a tree structure, so that you can display or hide each group.

QikProp Descriptors options

There is only one functional option: Compute PM3 properties, which calculates the ionization potential, electron affinity, and dipole moment from a PM3 semiempirical quantum mechanical calculation. This calculation takes the majority of the time for each structure, so turning it off ("fast mode") increases the throughput by a factor of about 30.

Compute button

Start the jobs to calculate the properties. As the job options are set for each descriptor class, the jobs for each descriptor are submitted to the selected host when you click this button.
