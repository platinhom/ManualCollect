Histogram Panel

This panel displays a histogram plot for a selected property.

Histogram Panel Features

  • Menu bar
  • Chart area
  • Property option menu
  • Property type options
Menu bar

The menu bar has two menus: File and Settings. The File menu has two items, Save Image, for saving the image in png, jpeg, or tiff format, and Close, for closing the chart. The Settings menu has the following items:

  • General—Make general plot settings. Opens the General dialog box.
  • Y Axis—Change settings for the y axis. Opens the Y Axis dialog box.
  • Bars—Change settings for the number and appearance of the bars. Opens the Bars dialog box.
Chart area

Displays the chart. The characteristics of the chart are changed using the Settings menu.

You can select bars in the chart, and the corresponding rows in the spreadsheet are selected. To add to the bar selection, use shift-click; to toggle the bar selection, use control-click. To clear the selection, click on an empty part of the chart. The row selection is displayed as a patterned area on the bars. If only some of the rows in a bar are selected, the patterned area represents the proportion of selected rows.

Property option menu

Choose the property for which the histogram is displayed. The properties listed are the properties that are selected in the current view.

Property type options

Select Property is numeric for properties that have numeric values (real or integer); select Property is categorical for properties whose values are integers or strings.
