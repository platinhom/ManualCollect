Help Menu

The Help menu provides access to information about Canvas.

  • About Canvas—Displays a panel that shows the Canvas version and legal notices relating to Canvas.

  • Online Help—Opens the Canvas Online Help in your browser. See Getting Help for more information about the help system, other sources of information, and contacting Schrödinger.

  • Diagnostics—Opens the Diagnostics panel, in which you can run tests on the installation, remote hosts, and licensing.

  • User Manual—Opens the Canvas User Manual in your PDF reader.

  • Python Module Overview—Opens the top-level document that provides an overview of the Python modules supplied for use with Schrödinger software.

  • Python API Reference—Opens the top-level document that provides access to detailed descriptions of the Python modules supplied for use with Schrödinger software.

  • Known Issues—Opens the Canvas Known Issues page on the Schrödinger web site in your browser. If you are not logged on, the logon page is displayed first, and the Known Issues page is automatically displayed once you log on.

  • Knowledge Base—Provides links to the Knowledge Base on the Schrödinger web site, where you can do a keyword search for information.
