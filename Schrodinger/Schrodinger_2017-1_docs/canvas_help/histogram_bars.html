Bars Dialog Box

In this dialog box you can change the number and appearance of the bars, and the display of the horizontal axis.

Bars Dialog Box Features

  • Bar specification section
  • Axis line section
  • Tick marks section
  • Tick mark labels or Category labels section
  • Axis title section
  • Color buttons
Bar specification section

For numerical properties, set the bin width for each bar (the data range covered by a bar) and the minimum and maximum data values for the histogram. For both numerical and categorical properties, set the width of the bars as a percentage of the space occupied by each bar, and the color of the bars.

Axis line section

Set the color and the weight, in points, of the axis line.

Tick marks section

Set the color and weight of the marks, and for numerical properties choose whether to center the tick marks on the bars (rather than between the bars).

Tick mark labels or Category labels section

Set the font size in points, the color, and for numerical properties, set the number of decimal places displayed for the labels.

Axis title section

Set the axis title, font size in points, and color of the text.

Color buttons

The Color buttons open a color chooser, in which you can select the desired color. The selected color is displayed to the right of the button.
