Diversity-Based Selection Dialog Box

Select structures for maximum dissimilarity based on a similarity or distance matrix, or on a set of fingerprints. This method is useful for selecting a set of compounds that is different from an existing set, for example.

Using the Diversity-Based Selection Dialog Box

One typical scenario for using diversity-based selection is to determine which compounds to buy from one library to supplement another, and export the structures to a file. This can be done as follows:

  1. Import both the existing library and the pool of compounds to choose from into a single project. When you import the libraries, ensure that you create a view from each library.

  2. Create a fingerprint for all compounds, with Applications → Binary Fingerprints.

  3. Open the Diversity-Based Selection dialog box.

  4. Choose the fingerprint you generated from the Fingerprint column option menu.

  5. For the Diversity selection method, choose Sphere.

  6. Enter the number of compounds you want in the Diverse subset size text box.

  7. For the Initialization method, choose Existing structures.

  8. For Use structures from, choose the custom view that contains the existing library.

  9. Click Choose Compounds.

    A diverse collection is selected that avoids the structures in the existing library, i.e., all diverse compounds will lie outside the spheres of the existing library structures. The collection is displayed in a new custom view.

  10. Choose View → Apply to Master in the new custom view.

  11. Export the structures from the Master View.

An alternative to using this approach is to use the Hole-Filling and Library Optimization dialog box, which also uses fingerprint similarity, but with a different optimization strategy that can include property values.

Data set option menuSelect the source of the data. The data source can involve both row and column selection. The choices are: Selected rows—Use the rows that are selected in the master view. You must select the rows before opening the dialog box. Visible rows—Use the rows that are displayed (visible) in the master view. You must ensure that the required rows are visible before opening the dialog box. Saved view—Use the rows and columns in a chosen saved view. When you select this item, an option menu that lists the saved views in the project is displayed to the right. You can choose the desired view from this option menu. A second option menu, labeled Domain, allows you to choose whether to restrict the rows or the columns, or both, to those of the saved view. Otherwise the visible rows or columns of the Master View are used. Original data set for jobname—Use the rows and columns that were used for the specified job. This item is only available when you are cloning a job or viewing the results of a job.
Save results as text box

Assign a name to the results. This name is listed under Diversity-Based Selection in the Project View window.

Fingerprint column option menu

Choose the fingerprint column to use for the matrix. You must generate fingerprints beforehand and incorporate them into the project. The available fingerprints are those that are listed under Binary Fingerprints in the Project View window and marked as incorporated.

Metric option menu

Choose the metric used to calculate the similarity or distance matrix.

Formula Key button

Open a panel that displays the definitions of the terms in the Description text box.

Description text box

Noneditable text box that displays the definition of the metric.

Diversity selection method option menu

Choose the method for selecting structures based on their diversity. The available options are:

  • Sphere—Sphere exclusion. Exclude compounds that are within a specified distance of the current subset.
  • DISE—Directed sphere exclusion. Exclude compounds that are within a specified distance of the current subset.
  • MAXSUM—Maximum sum of distances. Each round adds the compound with the largest total distance from the current subset.
  • MAXMIN—Largest minimum distance. Each round adds the compound with the largest minimum distance from the current subset.

A description of the method is displayed to the right of the option menu.

Diverse subset size option and text box

Specify the size of the diverse subset that is selected.

Exclusion sphere size text box

Specify the radius of the exclusion sphere for sphere exclusion methods (Sphere and DISE). Compounds whose distance is smaller than this value are excluded.

Initialization method option menu

Choose the method for selecting the first structure in the subset. Not available for the DISE method (which has its own initialization method). The available methods are:

  • Random—Choose a structure at random.
  • Representative—Choose the structure with the greatest similarity to the other structures in the dataset. Not available for the Sphere method.
  • Dissimilar—Choose the structure with the least similarity to the other structures in the dataset. Not available for the Sphere method.
  • Existing structures—Choose structures that are not within the exclusion sphere of the specified existing structures. This option activates the Use structures from option menu.
Seed text box

Specify the seed used to generate random numbers for the Random initialization method.

Generate Seed button

Click this button to generate a seed. The seed is displayed in the Seed text box.

Use structures from option menu

Select the view that contains the existing structures for initialization with the Sphere method. The diverse set is chosen so that it does not include any structure within the exclusion sphere of these structures.

Choose Compounds button

Run the job to choose the diverse subset of structures.
