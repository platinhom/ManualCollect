Similarity/Distance Screen Dialog Box

Screen structures for similarity to one or more reference structures based on a pregenerated fingerprint. The similarities or distances are added as new properties to the spreadsheet.

Similarity/Distance Screen Dialog Box Features

  • Data set option menu
  • Fingerprint column list
  • Base name for new similarity/distance columns text box
  • Naming policy option
  • Reference structures section
  • Metric option menu
  • Formula Key button
  • Description text box
  • Tversky α and β text boxes
  • Screen button
Data set option menu
Data set option menuSelect the source of the data. The data source can involve both row and column selection. The choices are: Selected rows—Use the rows that are selected in the master view. You must select the rows before opening the dialog box. Visible rows—Use the rows that are displayed (visible) in the master view. You must ensure that the required rows are visible before opening the dialog box. Saved view—Use the rows and columns in a chosen saved view. When you select this item, an option menu that lists the saved views in the project is displayed to the right. You can choose the desired view from this option menu. A second option menu, labeled Domain, allows you to choose whether to restrict the rows or the columns, or both, to those of the saved view. Otherwise the visible rows or columns of the Master View are used. Original data set for jobname—Use the rows and columns that were used for the specified job. This item is only available when you are cloning a job or viewing the results of a job.
Fingerprint column list

Choose the fingerprint column to use for the matrix. You must generate fingerprints prior to calculating the matrix, and incorporate them into the project. The available fingerprints are those that are listed under Binary Fingerprints in the Project View window and marked as incorporated.

Base name for new similarity/distance columns text box

Specify the base name that is used to construct the names of the columns to be added to the spreadsheet that contain the similarity to each reference structure. A default name is supplied.

Naming policy option

Select an option for constructing the column names of the similarities to the reference structures.

  • Use generated sequence—Generate a numerical sequence of names, as basename-number.
  • Use structure name—Append the structure name to the basename, as basename-structurename.
Reference structures section

Specify the reference structures to which the similarity is calculated. The options for specifying the reference structures are:

  • Select by hand—Select the structures manually. The structures in the screen are listed in the table, and you can select the desired rows of the table for the reference structures. The column name is displayed when a row is selected.
  • Use saved view—Use the structures in the view chosen from the option menu as the reference structures. The structures and column names are displayed in the table.
  • Use selected rows—Use the rows selected in the Master View as the reference structures. The structures and column names are displayed in the table.

If you want to view the structures as well as the structure names, select Show Chemistry. This option is selected by default. To view just the structure names in the table, deselect this option.

Metric option menu

Choose the metric used to calculate the similarity or distance matrix.

Formula Key button

Open a panel that displays the definitions of the terms in the Description text box.

Description text box

Noneditable text box that displays the definition of the metric.

Tversky α and β text boxes

Specify the α and β parameters for the Tversky metric.

Screen button

Run the job to screen the data set.
