Canvas Graphical Interface

The Canvas graphical interface is designed for project-oriented work and provides chemical structure storage, data analysis and visualization, and access to a wide range of applications that are also available through the command line. Key features of the Canvas graphical interface include:

  • Input/output of standard file types
  • 2D chemical spreadsheet
  • Custom views
  • Substructure searching
  • Sorting
  • Chemical & property filtering
  • Statistics
  • Charting
  • Heat maps
  • Molecular property calculations
  • Binary 2D fingerprints
  • 3D pharmacophore fingerprints
  • Similarity searching
  • Clustering
  • Diversity
  • Self-organizing maps
  • Maximum common substructure
  • Naïve Bayes classification
  • Multiple linear regression
  • Partial least-squares regression
  • Principal components analysis & regression
  • Neural networks

Canvas Projects

Canvas stores all of its data in a project, which is nothing more than a directory that contains a number of subdirectories and files. A Canvas project directory is always assigned the extension .cnv. Canvas does not have scratch projects, so you must create a project before you can import structures.

You can save Canvas projects in zipped format, with the extension .cnvzip or .cnv.zip, and open these projects in the same way as regular projects. The projects are unzipped in the current location when they are opened.

If a Canvas project directory is read-only, then the project is opened in a read-only mode. You can perform a limited set of operations on the project, but you cannot change any of the data or store filters, partitions, or views in the project.

The Canvas Main Window

The Canvas main window has a menu bar, toolbars, a spreadsheet area, a Project View panel and a Messages View panel, and a status area. The two panels are docked into the main window by default, but can be undocked. You can show or hide these views from the Project menu. The status area reports statistics on the number of rows and columns: the total, the number visible, and the number selected.

The Spreadsheet

The spreadsheet contains the 2D structures in the first column (including the structure name if there is one), followed by columns containing the properties of the structures.

You can select rows and columns with the usual shift-click and control-click, and you can select a range of rows by dragging. You can select cells by clicking in a cell or by dragging, and you can change the selection of individual cells with control-click, or extend the selection with shift-click.

You can resize individual rows and columns by dragging their borders. To resize multiple rows and columns simultaneously, select the rows or columns and drag one of the borders inside the selection. To resize the selected columns to fit their data, double-click the right border of one of the column headings.

You can move a column by dragging it to the new location, or you can move it to either end of the table by right-clicking in the heading row and choosing Move to Start or Move to End. You can sort the columns by name by right-clicking in the heading row and choosing Sort columns by name. The sort is case-insensitive.

You can scroll the spreadsheet using the mouse wheel or the scroll boxes. If you scroll rapidly in a vertical direction, structures are not drawn until you pause or release the scroll box, because Canvas retrieves structures and properties on demand from an SQLite database. This allows you to scroll through a project containing thousands of rows with no delay.

You can step through cells in the table with the arrow keys or the tab key. Stepping through cells changes the selection.

The current cell has a border drawn around it in reverse video (black on white, yellow on blue). The selected cells are highlighted with a blue background.

The spreadsheet has shortcut menus for the column headings, the selected rows and the selected structure table cell, which open when you right-click and hold.

Column heading shortcut menu items:

  • Sort by property (Ascending)—Sort the rows by the values of the property in this column, in ascending order.
  • Sort by property (Descending)—Sort the rows by the values of the property in this column, in descending order.
  • Sort Columns by Name—Rearrange all columns in alphabetical order
  • Restore Natural Column Order—Return the columns to the order in which they were added to the project.
  • Rename Column—Change the name of the column. Opens the Edit Column Name dialog box, so you can change the name and description.
  • Move to Start—Move the column to the first position (leftmost) in the spreadsheet.
  • Move to End—Move the column to the last position (rightmost) in the spreadsheet.
  • Hide—Remove the column from the current view. The data remains in the project.
  • Delete—Remove the property in this column from the project, and from all views.
  • Set Property Value—Set the value of the property in the column for the selected rows to a specified value. Opens the Set Property Value dialog box, in which you can set the value. A null value clears the property values.
  • Create Modal Fingerprint—Create a modal fingerprint row using the data in the column, which must be a fingerprint column. Opens a dialog box in which you can choose the selected or the visible rows to define the modal fingerprint, and name the fingerprint. A new row is added, with the name displayed in the Structure column. This item is only available if you right-click on a fingerprint column.

If you select multiple columns, only the Hide and Delete items are available.

Selected rows shortcut menu items:

  • Delete—Delete the selected rows.
  • Collapse to Selected—Collapse the view to the selected rows.
  • Hide Selected—Hide the selected rows.
  • Export—Export the selected rows to a file.
  • Export to Maestro—Export the selected rows to a new Maestro session.

Selected structure cell shortcut menu items:

  • Edit Structure—Edit the structure in the 2D structure editor.
  • Edit Structure Name—Change the name of the structure.
  • Copy—Copy the structure to the clipboard in SD format.
  • Copy SMILES—Copy the structure to the clipboard as a SMILES string
  • Copy Image—Copy an image of the structure to the clipboard.
  • Paste—Paste the SD-format structure from the clipboard into the current cell to replace the current structure.
  • Paste SMILES—Paste a SMILES string from the clipboard into the current structure cell to replace the current structure.

The Toolbars

There are four toolbars: Main, Edit, Select Rows, and Structure. You can show or hide the toolbars from the shortcut menu. The Main toolbar contains buttons for many of the common actions. A brief description of the Main toolbar is given below, with cross-references to the sections where these actions are described in detail.

New Project
Create a new Canvas project. The current project is closed.
Open Project
Open an existing Canvas project.
Import
Import structures into the project from a structure file.
Export
Export structures from the project to an external file.
Find
Find text in the spreadsheet. Opens the Find panel.
Collapse to Selected
Collapse the view to show only the selected structures.
Hide Selected
Hide the selected structures.
Apply to Master
Apply the current custom view to the master view. Only available in custom views.
Save View
Save the current view. In the master view, the view is saved as a new custom view, and the Save Custom View dialog box opens.
Restore Default View (master view)
Revert All Changes (custom view)
Restore the default view in the master view, or revert all changes to the view in a custom view. In either case, the view is restored to its initial (default) state.
Edit Spreadsheet
Edit cells in the spreadsheet. When you click this button, the Edit toolbar is displayed. Only available in the master view.
Add Row
Add a new empty row to the end of the spreadsheet. Only available in the master view.
Add Column
Add a new empty column to the end of the spreadsheet. Only available in the master view.
Substructure Query
Run a substructure query. Opens the Substructure Query dialog box.
Scatter Plot
Create a scatter plot. Opens the Scatter Plot panel.
Histogram
Create a histogram. Opens the Histogram panel.
Pie Chart
Create a pie chart. Opens the Pie Chart panel.

The Edit toolbar is only active when you are editing the spreadsheet. It is displayed in place of the File toolbar when you start editing. It has the following two buttons:

Save Spreadsheet Changes
Save the spreadsheet with the changes included. When you click this button, the spreadsheet is saved, and the File toolbar is redisplayed.
Exit Without Saving Spreadsheet Changes
Discard all the changes to the spreadsheet. When you click this button, the spreadsheet is reverted to its state before you started editing, and the File toolbar is redisplayed.

The Select Rows toolbar has a Rows text box, in which you can specify rows for a further action. Rows are specified as a comma-separated list, with ranges given either as n:m or n-m. The two available actions are represented by the buttons to the right of the text box:

  • Jump to—Jump to the first of the specified rows.

  • Select—Select the specified rows.

The Structure toolbar has a button for showing or hiding the 2D structure (labeled Show or Hide), a slider for changing the size of the 2D structures, and a Reset button to reset the structure sizes to their defaults. Sliding the slider all the way to the left hides the 2D structure (but does not set it as hidden).
