Model Visualization Dialog Box

This dialog box presents the structures with the atoms color-coded by the size of their contribution to the predicted property values from the KPLS regression.

Model Visualization Dialog Box Features

  • KPLS factors box
  • Sort by option menu
  • Min and Max buttons
  • Color scale slider and boxes
  • Limits button
  • Spreadsheet
KPLS factors box

Specify the number of KPLS factors for which the model is visualized, by typing in the box or using the arrow buttons.

Sort by option menu

Sort the table rows in ascending order of the chosen property. You can choose one of three properties listed in the table, Observed, Predicted, or Model Set, or you can choose Original Order to return them to the order in the project (row index).

Min and Max buttons

Select the colors used for negative (Min) and positive (Max) values of the contributions. These buttons open a standard color chooser, in which you can choose a color.

Color scale slider and boxes

Change the cutoffs for maximum color saturation of the negative and positive contributions to the property. Values below (for negative) or above (for positive) these cutoffs are represented with the maximum color saturation. Note that the maximum saturation is not 100%, but a smaller value. The maximum and minimum values of the cutoffs can be set in the Color Scale Limits dialog box, which you open by clicking Limits.

Limits button

Set the limits for the color scales. Opens the Color Scale Limits dialog box, in which you can set the lower limit and the upper limit of the color scale independently, and reset the limits to the defaults (click Defaults).

Spreadsheet

The spreadsheet displays the structures, the observed and predicted value of the property, the uncertainty in the predicted value (if calculated), and the set to which it was assigned in the building of the model. Each atom that contributed to a fingerprint used in building the model is marked with a colored disk that represents the value of the contribution to the property due to that atom. Two colors are used: one for negative values and one for positive values. The color saturation indicates the magnitude of the contribution. Atoms that did not appear in any fingerprint are not marked with a disk.

Double-clicking a structure opens the Analyze Atomic Contributions dialog box, so you can view and analyze the atomic contributions for each fingerprint for that structure. In this dialog box you can also edit the structure and analyze the predicted property value for the new structure.
