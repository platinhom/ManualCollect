Getting Help in Canvas

Canvas Help

The Canvas help facility contains information about the panels in Canvas and how to use them. For information about a panel or the tab that is displayed in a panel, click the Help button in the panel,

or press F1. The relevant help topic is displayed in the Help panel. A full list of help topics with links to each topic can be displayed by clicking the Home button.

To search for text in the help set, enter the text in the search box, and press Enter or click the Search button.

The search results are displayed in the display area, providing links to the topics found. You can use double quotes to require an exact match. You can use "and", "or", "not", and parentheses to construct complex search expressions.

To search within a topic, click the Find on page button,

then enter the words in the search text box and press Enter or click the Search button. Use the arrow keys to the right to step through hits.

    

You can go to the previous or next topic or set of search results using the Previous and Next buttons on the left of the toolbar.

    

Other Sources of Information

If you don't find the information you need in the online help, try the following sources:

  • The user manual. To open the manual, choose Help → User Manual.

  • The Knowledge Base, at http://www.schrodinger.com/kb. You can go to the Knowledge Base from the Help menu.

  • The Known Issues page, at https://www.schrodinger.com/knownissues.

All the documentation for all products, both PDF and HTML, is available from the Contents tab, to the left.

Generally, using the web form is best because you can add machine output and upload files, if necessary. Please include the following information:

  • A description of the problem and the operation being performed when it occurred
  • All relevant user input and machine output
  • Product purchaser (company, research institution, or individual)
  • Primary product user
  • Computer platform type
  • Operating system with version number
  • Product version number
  • Canvas version number
  • mmshare version number

The best way of providing the information is to open the Diagnostics panel from the Help menu and click Technical Support when the diagnostics have run. This creates a file, which you should upload to the web form. The location of the file is given in a dialog box.
