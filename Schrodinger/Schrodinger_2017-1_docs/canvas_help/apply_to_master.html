Apply to Master Dialog Box

Apply row and column selection and visibility to the master view from a custom view.

Apply to Master Dialog Box Features

  • Rows options
  • Columns options
  • Allow expansion of Master View row set option
Rows options

Specify how the rows in the current (custom) view are applied to the master view.

  • Retain current—Show only the rows in the master view that are visible in the current view.
  • Select current—Select the rows in the master view that are visible in the current view.
Columns options

Specify whether the columns in the current (custom) view are applied to the master view.

  • Retain current—Show only the columns in the master view that are visible in the current view.
  • Retain Master—Retain the visible columns in the master view. This option allows you to operate on the rows only.
Allow expansion of Master View row set option

If not all the rows in the current custom view are visible in the master view, expand the visible rows in the master view to include those in the custom view.
