Features

Calculator Dialog Box

In this dialog box you can create new properties by performing mathematical operations on existing properties.

Calculator Dialog Box Features

  • Operate on options
  • Create column text box
  • Formula display area
  • Functions button
  • Properties button
  • Keypad
  • Execute button
  • Save button
  • Load button
Operate on options

Select the option for the rows to calculate properties for. If you want to use the selected rows, you must select them before opening the dialog box.

Create column text box

Enter the name of the new column that you want to create for the calculated values.

Formula display area

This area displays the formula that you create using the buttons in the dialog box.

Functions button

Select a mathematical function to apply. Opens a dialog box in which you can select a function from a list. When the function is selected, you can then enter its argument, and terminate the argument by clicking the closing parenthesis button. For trigonometric functions, angles are specified in radians.

Properties button

Select a property column to use in the formula. Opens a dialog box in which you can select a property from a list.

Keypad

The keypad allows you to select the basic arithmetic operations, enter numbers, clear the last term or the entire formula.

Execute button

Evaluate the formula for the row range you selected, and create a new column in the spreadsheet.

Save button

Save the formula in the project. Opens a dialog box in which you can name the formula. The default name is the name you entered in the Create column text box of the Property Calculator dialog box.

Load button

Load a saved formula. Opens a dialog box in which you can select a formula from a list of saved formulae. The formula is displayed in the Formula display area, and you can then execute it or edit it.
