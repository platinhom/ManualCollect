Molecular Properties - Advanced Options Dialog Box

In this dialog box you can customize the definitions of hydrogen bond acceptors and donors and rotatable bonds, and make settings for the calculation of electrotopological states.

Molecular Properties - Advanced Options Dialog Box Features

  • Custom definitions section
  • Electrotopological states section
Custom definitions section

In this section you can customize the definitions of hydrogen bond acceptors, hydrogen bond donors, and rotatable bonds. To do so, select the option for the definition you want to customize, then click the corresponding Browse button to locate the definition file, which must have a .txt or .def extension. The file name is displayed in the text box. The file must include a list of SMARTS patterns. You can generate a template file that contains the default definitions from the command line with $SCHRODINGER/utilities/canvasMolDescriptors. More information is available in the topic canvasMolDescriptors.

Electrotopological states section

Change the settings used in the calculation of electrotopological states. You can set the Maximum path length to the number of bonds that are considered when calculating states, and set the power by which the perturbation falls off in the Fall-off power box.
