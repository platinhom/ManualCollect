Changing and Viewing Hypothesis Attributes

In this exercise we will view different aspects of a hypothesis including excluded volumes, feature tolerances, and the reference ligand and modify features including adjusting their position and orientation, changing their type, changing what chemistry is able to match a given feature and what chemistry is forbidden to be found at a feature.

  1. Choose File → Import Structures.

    The Import dialog box opens.

  2. From the Files of type option menu, ensure that Maestro is selected.

  3. If the options are not displayed, click Options.

  4. Ensure that Import all structures and Replace Workspace are selected.

  5. Select the file at1_dcph_phypo.mae.gz.

    This file contains multiple hypotheses from a develop common pharmacophore hypotheses experiment.

  6. Click Open.

    The Import dialog box closes. The first imported hypothesis, DHNRRR_1, is displayed in the Workspace.

  7. Right click the blue H button for entry DHNRRR_1 to show the list of available options for this hypothesis.

  8. Select Reference ligand.

    The reference ligand that was used to create the hypothesis is included in the Workspace. The reference ligand is a unique entry in the entry group for the hypothesis.

  9. Hide the feature tolerance markers by clearing the Feature tolerances option. Then reactivate visualization of the feature tolerances: right click on the blue H button for entry DHNRRR_1 and check the Feature tolerances option.

    Feature tolerances are applied in screening regardless of whether they are shown in the Workspace or not.

  10. Right click on the R10 donor feature in the Workspace, to open the Edit Feature dialog box.

    This dialog box allows you to change the feature to  another type, change the position and orientation of the feature, change whether it is used, required, or ignored in screening, change what features are counted as matching the feature, specify features that are not permitted within the feature tolerance and change the feature tolerance itself.

  11. ForUsage in screening choose Required.

    This forces any ligand to match this feature in order to be recognized as satisfying the hypothesis.

  12. On the Allowed to match option menu ensure that both the R and H options are checked.

    This will enable both aromatic rings and hydrophobic features to match the R10 feature.

  13. From the Forbidden to match option menu select the negative ionic (N) and positive ionic features (P)

    This ensures that no positive or negative centers on a ligand can be found within the feature tolerance radii of R10.

  14. Finally, decrease the feature tolerance from its default of 2.0 Å to 1.0 Å.

  15. Click OK to save the new attributes for this feature in hypothesis DHNRRR_1.

    While you could open the Edit Features dialog box for all features in this hypothesis to view current attributes of the hypothesis, it is far faster to right click the H button for the DHRRR_1 entry and select “Show feature properties”, to display attributes for all features simultaneously.

  16. Right click the blue H button for the DHRRR_1 entry and clear the Show feature properties option to remove feature attributes from the workspace.

What if we wanted an acceptor feature for the ring specified by R10 instead of the aromatic ring feature?

  1. Right click on the R10 feature to open the Edit Feature dialog box.

  2. For Feature type, choose Acceptor.

    An acceptor feature replaces the ring feature in the Workspace but with an ill-defined orientation. Note that when changing feature types, including reorienting A and D features, only single vector features are created. The position and orientation of the new feature can be changed.

  3. Select Manually reposition feature.

  4. Drag the new acceptor feature using the right mouse button to overlap the N that has only two attachments in the imidazole ring.

    You will see the X, Y, and Z coordinates changing as the move is made.

  5. Change the orientation of the acceptor vector by holding down the middle mouse button over the new acceptor feature and moving the mouse until the vector is aligned with the lone-pair electrons.

    This is the standard rotation operation, applied to the feature.

  6. Click OK in the Edit Features dialog box to save the new type and orientation of the feature.

    Note that the reference ligand now includes a hydrogen cyanide molecule that is used to control placement and orientation of the modified feature. The reference ligand does not have to be a real ligand.
