Modifying Default Phase Feature Definitions

In this exercise the default feature definitions will be changed to no longer recognize trifluoromethyl groups as hydrophobic. Modifying default feature definitions is an advanced pharmacophore modeling capability and requires understanding of SMARTS patterns. In Phase, feature definitions are applied to assign features to ligands and are saved with hypotheses. In screening against a hypothesis the same feature definitions that were used to create the hypothesis are employed. Use of the Feature Definitions Presets in screening as shown in Screening Ligand Files using Feature Definition Presets or modifying the feature types allowed to match a given hypothesis feature are commonly used to generalize feature matching outside changing the default feature definitions.

  1. Choose File → Import Structures.

    The Import dialog box opens.

  2. From the Files of type option menu, ensure that Maestro is selected.

  3. If the options are not displayed, click Options.

  4. Ensure that Import all structures and Replace Workspace are selected.

  5. Select the file 3el8_prepped_lig.maegz.

  6. Click Open.

    The Import dialog box closes. The imported ligand is displayed in the Workspace.

  7. Click the Tasks button, type “phase” and press Enter.

  8. Select Develop Pharmacophore Hypothesis from the results list.

    The Develop Pharmacophore Model panel opens.

  9. Select Single ligand (Workspace) from the Create pharmacophore model option menu.

  10. Click Hypothesis Settings to open the Hypothesis Settings panel.

  11. In the Features tab click Edit Features button to open the Define Features dialog box.

    The list of SMARTS patterns is matched in descending order and the first pattern to match per initial atom in the pattern is assigned to that atom. The Feature column indicates if a feature has been recognized from a given pattern which is reflected by the features shown for the molecule in the Workspace. For instance, the [N;X2]([a])([a]) pattern is noted as being matched in the molecule by the Feature column showing yes and A1, A2, and A3 in the Workspace arise from this pattern.

  12. Change Feature to Hydrophobic (H).

    We want to not recognize trifluoromethyl (H9 in the current ligand) as a hydrophobic group. In the pattern list the first feature to be recognized is [a]C(F)(F)(F).

  13. Click the Ignore checkbox for that feature.

    Now the C(F)(F)(F) pattern a bit further down in the table is being matched by the trifluoromethyl group.

  14. Click the Ignore checkbox for that feature as well.

    Now the F pattern is being matched.

  15. Click the Ignore checkbox for that feature as well.

  16. Click OK to close the Edit Features dialog box.

    In the Workspace, the trifluoromethyl group on the ligand now has no hydrophobe feature.

    The changed feature definitions will be used by the Maestro project moving forward, including after the Maestro project has been saved and reopened.

  17. To return to the default feature definitions click Edit Features in the Hypothesis Settings panel to open the Edit Features dialog box, and click Reset All. Click OK to save the feature definitions, which are now reset to the defaults.
