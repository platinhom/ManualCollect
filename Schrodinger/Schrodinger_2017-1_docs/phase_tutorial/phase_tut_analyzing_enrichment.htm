Viewing Enrichment for a Set of Hypotheses

In this exercise we will analyze enrichment performance for a set of hypotheses generated by common pharmacophore perception. This analysis can also be performed with a single hypothesis using the Validate hypothesis workflow.

  1. Choose File → Import Structures.

    The Import dialog box opens.

  2. From the Files of type option menu, ensure that Maestro is selected.

  3. If the options are not displayed, click Options.

  4. Ensure that Import all structures and Replace Workspace are selected.

  5. Select the file erb_common_phypo.mae.gz.

    This file contains multiple hypotheses from a develop common pharmacophore hypotheses experiment.

  6. Click Open.

    The Import dialog box closes. The first imported hypothesis is displayed in the Workspace.

  7. Click the green H button on the erb_common group entry and select Enrichment Viewer.

    The Enrichment Viewer panel opens, as shown.

  8. Click on the header of the Phase Hypo Score column to sort the hypotheses by the PhaseHypoScore.

    This score is a linear combination of the Survival score that measures ligand alignment for active ligands and the BEDROC(20) performance in separating the known actives from a set of presumed decoy ligands.

  9. View additional data by selecting Total actives and Ranked actives on the Show metrics option menu.

    These metrics are added as new columns on the far right of the table, as shown.

    The ROC plot or % Screen plots can be displayed from this panel for each hypothesis.

  10. Select the top-ranked hypothesis, APRR_2 in the table and click ROC Plot to open the ROC plot panel.

    Random recovery of ligands is shown by the line going through the center. The performance of the hypothesis is good, recovering around 50% of the diverse set of known actives.

  11. Close the ROC Plot panel (using window manager options).

  12. Click the Enrichment report button to open a detailed report of enrichment per hypothesis.

This report includes a large number of enrichment metrics and the ranks and titles for all recovered known actives. These reports can be very useful to identify hypothesis most likely to meet the goals of a virtual screening exercise.
