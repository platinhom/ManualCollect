Creating a Hypothesis from a Single Ligand

In this exercise we will create a pharmacophore hypothesis from a ligand in the Maestro Workspace

  1. Choose File → Import Structures.

    The Import dialog box opens.

  2. From the Files of type option menu, ensure that Maestro is selected.

  3. If the options are not displayed, click Options.

  4. Ensure that Import all structures and Replace Workspace are selected.

  5. Select the file 3el8_prepped_lig.maegz.

  6. Click Open.

    The Import dialog box closes. The imported ligand is displayed in the Workspace.

  7. Click the Tasks button, type “phase” and press Enter.

  8. Select Develop Pharmacophore Hypothesis from the results list.

    The Develop Pharmacophore Model panel opens.

  9. Select Single ligand (Workspace) from the Create pharmacophore model option menu.

  10. Select Pick feature in Workspace and pick features D7, D8, D5, A1, and R14 by clicking on them in the Maestro Workspace. You can also click on them in the panel.

  11. Add a donor feature manually that doesn’t exist in the molecule in the Workspace.

    1. Select Add [Acceptor] in workspace and change Acceptor to Donor.

    2. Click on the aromatic hydrogen atom on the R13 ring closest to the A3 feature.

    3. Right click on the newly added feature to open the Edit feature D15 dialog.

    4. Select Manually reposition feature.

    5. Change the vector orientation and location with the mouse buttons you use for rotation and translation in the Maestro Workspace.

    6. Once oriented as desired click OK.

      The orientation and location of this donor feature does not matter for further exercises in the tutorial.

  12. Add an enclosing set of excluded volumes about the molecule:

    1. Click Hypothesis Settings to open the Hypothesis Settings dialog.

    2. In the Excluded Volumes tab, select Create excluded volume shell.

    3. Click Save to close the dialog and save changes to the options.

  13. Change the Hypothesis name to 3el8_hypo and click Create to create the hypothesis and add it to the Maestro Project Table and Entry List.

  14. Hide (undisplay) the excluded volumes for the hypothesis by right clicking on the H button for the hypothesis in the entry named 3el8_hypo, and deselecting Excluded volumes.

    This gives a view of only the features and their tolerances.

  15. Hide the tolerance markers by right-clicking on the H button and deselecting Feature tolerances.
