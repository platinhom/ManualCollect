Identifying Multiple Binding Modes from a Set of Hypotheses

In this exercise we will visualize the hypotheses and ligand alignments from an in-place common pharmacophore perception experiment to identify multiple binding modes. This uses the pharmacophore hypotheses created in Creating a Common Hypothesis from Prealigned Ligands.

 

  1. Choose File → Import Structures.

    The Import dialog box opens.

  2. From the Files of type option menu, ensure that Maestro is selected.

  3. If the options are not displayed, click Options.

  4. Ensure that Import all structures and Replace Workspace are selected.

  5. Select the file factorXa_inplace_phypo.mae.gz.

    This file contains multiple hypotheses from a develop common pharmacophore hypotheses experiment.

  6. Click Open.

    The Import dialog box closes. The first imported hypothesis is displayed in the Workspace.

  7. Click the green H button on the factorXa_inplace group entry and select Detect Binding Modes.

    TheDetect Binding Modes panel opens, as shown.

    In this table, hypotheses from the common pharmacophore perception process are the rows, and ligands that matched any hypothesis are the columns. Blue cells indicate that a ligand matched a hypothesis and red indicates that it did not. Clustering is performed to group hypotheses by the ligands that they match. The dark lines indicate where the algorithm recognizes two different binding modes, but it is clear from the grouping that these are not the only two binding modes, and there are likely more modes.

  8. Change the Proposed number of binding modes to 4, to group into 4 binding modes.

    The two large groups of hits are now identified as separate binding modes.

    Clicking on a cell in the table displays the hypothesis and active ligand in the Workspace. The selected cell is highlighted in pale yellow.

  9. To see active ligands that matched HPRRR_1 click and drag over the 5 ligands as shown.

    This displays the HPRRR_1 hypothesis and alignments of the 5 ligands to that hypothesis.

  10. Repeat the process to view the two other clusters of ligands and matched hypotheses.

    Viewing the cluster in the upper right shows the ligands are all sulfone-containing lactams with a largely conserved S1 binding pocket orientation with the halogen.

    Selecting matched ligands for the cluster that includes ADHRR_1 shows most ligands have a 2-oxopyridine ring and similar S1 binding pocket groups with a halogen.
