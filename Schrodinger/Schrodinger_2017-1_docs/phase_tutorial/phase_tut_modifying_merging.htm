Merging Hypotheses

In this section we will merge features from multiple hypotheses to create a new hypothesis. One caveat is that the feature definitions used to create the hypotheses must be the same in order to merge their features into a new hypothesis.

  1. Choose File → Import Structures.

    The Import dialog box opens.

  2. From the Files of type option menu, ensure that Maestro is selected.

  3. If the options are not displayed, click Options.

  4. Ensure that Import all structures and Replace Workspace are selected.

  5. Select the file factorXa_inplace_phypo.mae.gz.

    This file contains multiple hypotheses from a develop common pharmacophore hypotheses experiment using prealigned ligands.

  6. Click Open.

    The Import dialog box closes. The first imported hypothesis, AADHHR_1, is displayed in the Workspace.

  7. View both the AADHHR_1 and ADHRR_1 hypotheses in the Workspace by clicking the right facing arrow of the ADHRR_1 group to open it and control-clicking (command-clicking) the In column for the ADHRR_1 entry.

  8. Open the Active subgroup for the ADHRR_1 group and control-click (command-click) to also include ligand 2p93_lig_ME1.

  9. Click the Tasks button, type “phase” and press Enter.

  10. Select Develop Pharmacophore Hypothesis from the results list.

    The Develop Pharmacophore Model panel opens.

  11. For Create pharmacophore model using choose Merged hypotheses (Workspace).

    Features for all hypotheses shown in the Workspace are shown in the table.

  12. By clicking in the table, select the features A3, D4, A4, D7, and H8, as shown.

    Alternatively you can select Pick feature in Workspace and pick the features in the Workspace directly instead of in the table. When selected, features are highlighted in blue.

  13. Select Add [feature] in workspace and change the choice to Hydrophobe.

  14. Click on atom 35, the aromatic carbon atom opposite the N in the terminal pyridine ring.

    A new feature, H1, is added to the Workspace and the table.

  15. Pick H1 in the table to include it in the merged hypothesis as well.

  16. Change the Hypothesis name to "factorXa_merged" and click Create to create the merged hypothesis with the features that are highlighted in blue.

    Note that the feature names have changed though the feature attributes have been retained.
