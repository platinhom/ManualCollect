MD Simulations for 1bel

In this exercise, you will run MD simulations on the 1bel systems that you prepared earlier, and examine the trajectories. The trajectories for these two model systems demonstrate the effect of optimizing the H-bond network in the protein. In the unoptimized case, the ring in Tyr 115 flips to a new orientation. In the optimized case, this does not happen. The simulations each take about 48 hours, so you should run them in parallel on a suitable multiprocessor host.

Starting here?

Import the files 1bel-_setup_1-out.cms and 1bel-_setup_2-out.cms from your tutorial directory. If you haven’t copied the tutorial files, see Copying the Desmond Tutorial Files for instructions.
