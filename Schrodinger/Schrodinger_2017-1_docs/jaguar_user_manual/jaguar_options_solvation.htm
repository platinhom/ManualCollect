Jaguar Solvation Settings

Jaguar can treat solvated molecular systems with a self-consistent reaction field method, using either a standard Poisson-Boltzmann solver [15, 216] or the Solvation Model 8 (SM8) [217] or Solvation Model 6 (SM6) approach [225]. These models are treated in the topics listed below.
