Mutate Model to Sequence Dialog Box

In this dialog box, you can mutate the model to match the sequence and start a side-chain prediction job for the mutated residues.

To open this panel, click Mutate Model to Sequence in the PrimeX panel.

Mutate Model to Sequence Dialog Box Features

  • Alignment to reference sequence text area
  • Mutate to Match Sequence button
  • Place Side Chains button
  • Delete residues which align to gaps option
Alignment to reference sequence text area

This noneditable text area displays the alignment of the model sequence to the reference sequence. Residues in the model sequence that differ are marked in red.

Mutate to Match Sequence button

Perform the residue mutation on the reference sequence. The side chains are placed in a default orientation. After the mutation is finished, Place Side Chains button is activated. A new project entry is created for the mutated structure.

Place Side Chains button

Open the Real-Space Refinement panel to perform a side-chain prediction on the residues that were mutated. These residues are preselected in the Real-Space Refinement panel.

Delete residues which align to gaps option

This option deletes the residues that align to gaps in the model, rather than filling them in, when the mutation is performed. You will then have to perform a loop prediction to fill the gaps, but this will produce superior results to the default filling of gaps by the mutation algorithm.
