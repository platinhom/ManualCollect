Protein Preparation Command-Line Tools

The command line tools used for protein preparation are described in this document. For a description of the command options, run the command with the -h option.

On Linux and Mac platforms, open a terminal window and set the SCHRODINGER environment variable to the software installation directory. The commands must be prefixed with $SCHRODINGER/utilities/. On Windows, open a Schrodinger Command Prompt window from the Start menu, and run the commands as given.
