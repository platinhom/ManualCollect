Aggregation Profiles Panel

Display bar charts of aggregation-related properties on a per-residue basis. The properties are the Aggrescan and Zyggregator scores, the Side chain accessible surface area ratio, and the three contributions to the energy score of the patch (positive, negative, and hydrophobic).

To open this panel, click Profiles in the Protein Surface Analyzer Panel.

Aggregation Profiles Panel Features

  • Bar charts
  • Show profiles per chain option
  • Export button
Bar charts

This area displays charts for each property, in a vertical stack. Each chart represents the property value for a residue as a bar; the residues are shown in sequence order. Selecting a bar in the chart highlights it, and selects the corresponding residue in the tables in the Protein Surface Analyzer panel and in the Workspace. Likewise, residues that are selected in the tables are selected and highlighted in the charts.

For antibodies, the plot background is colored for the CDR region residues.

Show profiles per chain option

Plot the properties as bar charts for each chain separately, rather than for the protein as a whole. The charts for each property are displayed for the first chain, then for the second chain, and so on.

Export button

Export the charts to a PDF file. Opens a file selector so you can navigate to a location and name the file.
