Titration Curve tab

In this tab you can change the appearance of the markers and lines used to plot the titration curves, and choose which curves to display.

Data set table

This table provides the tools for changing the appearance of the curves and selecting the curves to display.

Click in any of the Marker or Line columns to change the value. An option menu or a text box is displayed when you click. The symbol and line are displayed in the second column.

To change the title, double-click in the column.

To show or hide a curve, select or clear the check box in the In column.

Backgroundoption menu

Choose the background color of the plot from the colors listed on this option menu.

Legend option menu

Choose the location of the legend, from Automatic or one of nine locations. To hide the legend, choose None.

Charge Profile tab

In this tab you can change the appearance of the markers and lines used to plot the charge profile curves for a chosen data set, and choose which residues to display.

Data set option menu

Choose the data set (titration curve) for which you want to display the charge profile and make settings.

Ionizable residue table

This table lists all the ionizable residues for the chosen titration curve. It provides the tools for changing the appearance of the charge profile curves and selecting the residues for which to display the charge profile.

Click in any of the Marker or Line columns to change the value. An option menu or a text box is displayed when you click. The symbol and line are displayed in the second column.

To change the title, double-click in the column.

To show or hide the charge profile curve for a residue, select or clear the check box in the In column.

Background option menu

Choose the background color of the plot from the colors listed on this option menu.

Legend option menu

Turn the legend (curve labels) on or off.
