Specifying the Force Field for the Desmond Sytem

The force field is specified from the choices on the Force field option menu. The default choice is OPLS_2005, or OPLS3 if you have a license for this force field. To specify other force fields, you can run the utility viparr—see Alternate Force Field Parameters and Constraints for Desmond Simulations.
