Examining Poses for 1fjs

In these exercises, Glide results are examined with an emphasis on visual rather than numerical appraisal.

The first set of exercises uses the Entry List and the Pose Viewer panel to display the results of the SP Glide docking job, examine individual ligand poses and their contacts with the input receptor structure.

The second set of exercises uses the Glide XP Visualizer panel to display information on the terms in the Glide XP scoring function that contribute to the ligand binding.
