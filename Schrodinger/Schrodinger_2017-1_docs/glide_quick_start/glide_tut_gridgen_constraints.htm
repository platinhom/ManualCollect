Setting Up Glide Constraints to 1fjs

The Constraints tab of the Receptor Grid Generation panel is used to define Glide constraints. In this exercise, you will define two constraints: a positional constraint and an H-bond constraint. To make it easier to see the parts of the receptor close to the ligand and to see the ligand atoms, you will first change the display.

For more information on using Glide constraints, see Setting Up Receptor Constraints for Grid Generation.
