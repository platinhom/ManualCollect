Defining the 1fjs Active Site

Now that the ligand has been excluded, the volume for which grids will be calculated can be defined:

1. Click the Site tab.

2. Ensure that the Center option selected is Centroid of Workspace ligand.
3. Ensure that the Size option selected is the default, Dock ligands similar in size to the Workspace ligand.

If you have a representative ligand in the active site, the default generates an enclosing box that is large enough for most systems. Generally, you should make the enclosing box as small as is consistent with the shape and character of the protein’s active site and with the ligands you expect to dock. However, if you think that conformations of active ligands may exist that are significantly larger than the cocrystallized ligand, you should consider enlarging the enclosing box using the Dock ligands with length <= option.
