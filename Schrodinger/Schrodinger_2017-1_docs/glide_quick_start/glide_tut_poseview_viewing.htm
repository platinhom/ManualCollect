Viewing 1fjs Poses

The Project Table panel has a facility for viewing poses, using an entry group that consists of a receptor and a set of ligands. This facility is available in the Pose Viewer panel, which you open with Entry → View Poses, when you have a single entry group selected. It is also available in the main window: Tasks → Receptor-Based Virtual Screening → Pose Viewer.

1. Choose Entry → View Poses.
2. If entry properties are not displayed in the Workspace, type S.

By default entry properties appear in the upper right corner of the Workspace, and include the entry title.

Now you are ready to view the poses.

3. Press the RIGHT ARROW key.

The second pose replaces the first in the Workspace. The RIGHT ARROW and LEFT ARROW keys can be used to step through the selected poses.

4. Shift-click the entry for the first pose.

The first pose is added to the Workspace, and the entry properties are removed.

5. Press the RIGHT ARROW key.

The third pose replaces the first two, and the entry properties are displayed.

6. Reselect the first pose by clicking its In column.
