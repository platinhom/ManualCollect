Specifying the Grid To Use for Docking

In this exercise, you will select the grid that you calculated in Generating Receptor Grids for 1fjs for the ligand docking job.

  1. Choose Workspace → Clear Workspace.

  2. Choose Tasks → Receptor-Based Virtual Screening → Docking → Ligand Docking in the Task Tool.

    How do I do this?

    1. Click the Tasks button.

      A menu is displayed. To the left there should be a pane with a TASKS heading and an APPLICATIONS heading. If it is not displayed, click Browse on the menu.

    2. Under TASKS, choose Receptor-Based Virtual Screening.

      The contents of the pane are replaced with the tasks available for Receptor-Based Virtual Screening.

    3. Under Docking, click Ligand Docking.

    The Ligand Docking panel opens with the Ligands tab displayed.

  3. Click the Browse button for the Receptor grid.

    A file selector opens.

  4. Navigate to the factorXa_grid folder, choose factorXa_grid.zip, and click Open.

    The full name of the grid file is displayed in the Receptor grid text box.
