Running VSW Jobs from the Command Line

You can run VSW jobs from the command line by entering the following command in a terminal window (Linux or Mac) or a Schrodinger Command Prompt window (Windows):

vsw [options] jobname.inp

On Linux and Mac, prepend the command with $SCHRODINGER/. For a description of the options, run the command with the -h option. The standard Job Control options are supported—see Running Jobs From the Command Line. The input file is described in detail in The Python Pipeline Input File. This description covers all pipeline stages, not just those used by VSW.

It is recommended that you generate the input file by preparing it in Maestro, then choosing Write from the Settings button menu to write it out.
