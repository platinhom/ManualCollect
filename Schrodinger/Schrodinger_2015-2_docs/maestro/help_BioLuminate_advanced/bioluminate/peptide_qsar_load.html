Summary

Load peptide sequences and the observable for each sequence that is being fit or predicted.

Opening the Peptide QSAR - Load Sequences and Observables Dialog Box

To open the Peptide QSAR - Load Sequences and Observables dialog box, you can:

  • Click Load Sequences and Observables in the Peptide QSAR panel.

Peptide QSAR - Load Sequences and Observables Dialog Box Features

The features shown in this dialog box depend on the choice you make from the first option menu.

  • Load sequences from option menu
  • Use a project property for the observable option and Property option menu
  • Load the observable from a CSV file option, text box and Browse button
  • Observable column box
  • Has header option
  • Enter manually in the table / Has no observables option
  • File text box and Browse button
  • Sequence box
  • Observable box
  • Name box
Load sequences from option menu

Choose the source of the sequences. There are four choices: Project Table (selected entries), Workspace (included entries), CSV file, and Fasta file. The choice you make determines which controls are displayed in the rest of the dialog box. The sequences you choose must all be of the same length.

Use a project property for the observable option and Property option menu

Take the values of the observable from the Project Table. Choose the project property for the observable from the Property option menu. Only present when the sequences are loaded from the Project Table or the Workspace.

Load the observable from a CSV file option, text box and Browse button

Read the values of the observable from a CSV file. Enter the file name in the text box or click Browse to locate and select the file. This option is not present when sequences are loaded from a CSV file, as the observable is taken from the same CSV file.

Observable column box

Specify the column number in the CSV file that contains the observable. Not present when sequences are loaded from a CSV file.

Has header option

Select this option if the CSV file has a header row, deselect it if the CSV file does not have a header row.

Enter manually in the table / Has no observables option

Select this option to enter the observable values in the table by editing the table cells, or if the sequence has no value for the observable. Not present when sequences are loaded from a CSV file.

File text box and Browse button

Enter the file name of the CSV or Fasta file in the text box or click Browse to locate and select the file. Only present when loading sequences from a file.

Sequence column box

Specify the column number in the CSV file that contains the sequence.

Observable column box

Specify the column number in the CSV file that contains the observable.

Name column box

Specify the column number in the CSV file that contains the title for the sequence.
