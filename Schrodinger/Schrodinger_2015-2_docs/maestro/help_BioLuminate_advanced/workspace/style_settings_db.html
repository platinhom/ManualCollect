Summary

This dialog box allows you to choose settings for the Workspace style: combinations of representation, color, and display of various atoms.

Opening the Workspace Style Settings Dialog Box

To open the Workspace Style Settings dialog box, you can:

  • Choose Style → Predefined Style Settings → Style Settings in the main window.

  • Choose Settings from the Apply button on the Style toolbar

    .

Workspace Style Settings Dialog Box Features

  • General tab
  • Representation tab
  • Color tab
  • Hydrogens tab
  • Surfaces tab
  • Reset button

General tab

In this tab you can make various settings for general appearance and for atom display.

Undisplay waters option menu

Choose the condition for undisplaying waters from this option menu. The three distance options are distances from the ligand. The default is to undisplay all waters.

When Workspace changes options

These options control what is changed when changes are made in the Workspace.

  • Fit to ligand—Fit the ligand to the Workspace
  • Apply settings to options—Select this option to apply setting, and select an option for the atoms to which the style is applied, from Entire Workspace and Ligand only.

Representation tab

In this tab you set the molecular representation style for the various species. Each of the option menus has five items: Wire, CPK, Ball & Stick, Tube, and Thin Tube.

Apply representations option

Apply the representation defined in this tab when the Workspace style is applied. If this option is deselected, the representation is not changed when the Workspace style is applied.

Default representation option menu

Choose the representation for atoms that are not defined by the other option menus. The default is Wire.

Ligand representation option menu

Choose the representation for the ligand. The default is Ball & Stick.

Water representation option menu

Choose the representation for water molecules. The default is Ball & Stick.

Ion representation option menu

Choose the representation for small ions (less than 10 atoms). The default is CPK.

Display residues within N Å of the ligand option and text box

Select this option to limit the display to the residues within the specified distance of the ligand, in angstroms. Residues that have any atoms within this distance are displayed.

Display ribbons option

Select this option to display ribbons for receptor atoms when representations are applied. This option is selected by default. See the Ribbons panel topic for more information.

Display ribbons within N Å of the displayed atoms option

Display ribbons within the specified distance of the displayed atoms. The ribbons are updated with the display.

Ribbon scheme option menu

Choose the color scheme for ribbons from this option menu.

Ribbon style option menu

Choose the ribbon style from this option menu.

Color tab

In this tab you can set the color schemes for the style. There are four option menus, each of which has the same range of color schemes, which are:

  • Element
  • Element (Green Carbons)
  • Element (Light Carbons)
  • Element (Entry Carbons)
  • Atom PDB B Factor (Temperature Factor)
  • Density Fit
  • Molecule Size
  • Residue Type
  • Entry
Apply colors option

Apply the color schemes defined in this tab when the Workspace style is applied. If this option is deselected, the color schemes are not changed when the Workspace style is applied.

Default color scheme option menu

Choose the color scheme for all atoms that are not defined by the other three option menus. The default is Element.

Ligand color scheme option menu

Choose the color scheme for the ligand. The default is Element (Entry Carbons).

Water color scheme option menu

Choose the color scheme for water molecules. The default is Element.

Ion color scheme option menu

Choose the color scheme for small ions (less than 10 atoms). The default is Element.

Optimize for buttons

Optimize the choice of color schemes for a given background. For Light Background, the Element (Light Carbons) scheme is chosen for all four classes of atoms. The defaults, given above, are the schemes chosen for Dark Background.

Hydrogens tab

In this tab, you can choose which of the receptor hydrogens and the ligand hydrogens to display.

Display receptor hydrogens options

Select an option for the receptor hydrogens to display, from None, Polar only, and All. The default is Polar only.

Display ligand hydrogens options

Select an option for the ligand hydrogens to display, from None, Polar only, and All. The default is All.

Surfaces tab

In this tab, you can set options to create surfaces when the Workspace style is applied. This includes applying the style when the Workspace changes, if the relevant option is selected in the General tab. The surfaces are created according to the options set in the Create Binding Site Surfaces dialog box.

Create receptor surface option

Create a surface for the receptor when the Workspace style is applied.

Create ligand surface option

Create a surface for the ligand when the Workspace style is applied.

These options are not selected by default.

Reset button

This button resets the settings to their defaults.
