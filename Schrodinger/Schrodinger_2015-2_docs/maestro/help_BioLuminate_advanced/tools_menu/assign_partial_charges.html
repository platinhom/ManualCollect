Assign Partial Charges

The Assign Partial Charges command under Edit → Advanced Edit Options assigns partial charges to the structures in the Workspace, based on the OPLS_2005 force field. If the structure does not have explicit hydrogens, the partial charges for the implicit hydrogens are added to the partial charge for the heavy atom to which they are implicitly attached.

These partial charges can be used for many purposes, such as displaying the dipole moment of the molecule (Edit → Settings → View → Display Dipole Moment).
