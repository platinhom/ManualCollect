Enhance Depth Cues

This button is located on the View toolbar in the main window:

Clicking this button causes the program to enhance the 3D cues. This is done by moving the clipping planes in as close as possible to the displayed structures. The clipping planes are shown in the Clipping Planes window.

This will only really have any effect if fogging is enabled or if perspective is enabled (see the Appearance panel).
