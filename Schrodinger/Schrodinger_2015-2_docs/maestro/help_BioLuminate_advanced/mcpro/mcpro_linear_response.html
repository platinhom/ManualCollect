MCPRO+ Linear Response Calculations

The purpose of MCPRO+ linear response calculations is to provide data for the generation of linear response models, for example using Strike to fit the data.

You can set up and run a relative binding affinity calculation in the MCPRO+ Linear Response panel, which you open by choosing Applications → MCPRO+ → Linear Response. The general layout and common features of this panel are described in the topic MCPRO+ Panels.

The input required for a linear response calculation is a ligand file and a prepared protein model system in the same frame of reference. The structural input is the same as required for Liaison and Prime MM-GBSA calculations, but the protein must be prepared as a model system beforehand, for example by setting up the model system for an MC Sampling job for a protein only. There is no need to run the job or write the input files—the Z-matrix is written at the end of the model system creation automatically.

R

When the job is run, MCPRO+ creates model systems for each of the ligands and their complexes. The bound Monte Carlo sampling run is similar to that done for other job types. The free ligand Monte Carlo sampling includes a step in which the ligand bond lengths and angles are held fixed and only the torsions and relative orientation are varied, with a much higher temperature used in the Metropolis criterion for ligand moves. This step and the subsequent quenching in the next equilibration allows higher-energy configurations to be sampled and thus a greater probability of moving over torsional barriers.

The principal output from a linear response calculation is written to the file jobname.csv, which is compiled from data for the bound and the free systems. The results in this file can be used to generate a model of binding affinities. The file has the following properties:

Property Description
Ligand_Number Ligand number. Ligands are numbered by their position in the input file.
EXX-LJ Solute-solute van der Waals energy difference (kcal/mol)
EXX-C Solute-solute coulomb energy difference (kcal/mol)
ESX-LJ Solute-solvent van der Waals energy difference (kcal/mol)
ESX-C Solute-solvent coulomb energy difference (kcal/mol)
dElj Total van der Waals energy difference of system (kcal/mol)
dEc Total coulomb energy difference of system (kcal/mol)
dSASA Change in solvent accessible surface area of ligand (Å2)
dFOSA Change in hydrophobic solvent accessible surface area of ligand (Å2)
dFISA Change in hydrophilic solvent accessible surface area of ligand (Å2)
dARSA Change in aromatic solvent accessible surface area of ligand (Å2)
dWPSA Change in weakly polar solvent accessible surface area of ligand (Å2)
#RB Number of ligand rotatable bonds.
dHBdon_solvent Change in the number of ligand - solvent donor hydrogen bonds
dHBdon_solute Change in the number of ligand - solute donor hydrogen bonds
dHBdon Change in the total number of donor hydrogen bonds for ligand
dHBacc_solvent Change in the number of ligand - solvent acceptor hydrogen bonds
dHBacc_solute Change in the number of ligand - solute acceptor hydrogen bonds
dHBacc Change in the total number of acceptor hydrogen bonds for ligand
dHBtot Change in the total number of hydrogen bonds for ligand
dEint Change in the internal energy of the ligand in kcal/mol
#SatAmine Number of saturated amines (i.e. secondary amines)
#Nitro Number of nitro groups
#Acid Number of acid groups
