MCPRO+ Minimizations

MCPRO+ minimization calculations can be performed on a protein, a protein-ligand complex, or a single ligand.

You can set up and run a minimization calculation in the MCPRO+ Minimization panel, which you open by choosing Applications → MCPRO+ → Minimization. The general layout and common features of this panel are described in the topic MCPRO+ Panels. Information on minimizations is available in the MCPRO+ — Minimization Tab topic.
