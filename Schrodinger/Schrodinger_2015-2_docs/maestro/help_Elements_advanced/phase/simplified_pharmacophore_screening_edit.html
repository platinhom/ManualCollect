Summary

In this tab you can define the hypothesis to use for screening, either by creating it from a ligand, or importing it.

Edit Tab Features

  • Title text box
  • Ligand from Workspace button
  • Ligand from File button
  • Features table
  • Import button
  • Export button
Title text box

Specify the title of the hypothesis. This title is used as the entry title in the Maestro file that is written for the reference ligand. A default title is supplied when you create or import a hypothesis.

Ligand from Workspace button

Click this button to create a hypothesis from the ligand that is displayed in the Workspace. The ligand can be part of a ligand-receptor complex, and the ligand is extracted from the complex for the reference ligand.

Ligand from File button

Click this button to read a ligand from a file to create a hypothesis. A file selector opens, so that you can navigate to and open the file. The file must contain a single structure. The ligand can be part of a ligand-receptor complex, and the ligand is extracted from the complex for the reference ligand.

Features table

This table lists the pharmacophore features that were identified in the refererence ligand. It allows you to choose the features to include in the hypothesis and decide whether they are required or optional. The table columns are described below

Use Check boxes for whether the feature is used for the hypothesis or not. Initially all boxes are checked, and the features are displayed in the Workspace on the ligand. When you clear a check box, the feature is undisplayed, and it is not used in the hypothesis for screening.
ID Feature label, which consists of a single letter for the feature type, and an index. Noneditable.
Type Text description of the feature type. Noneditable.
Must match Check boxes for whether the feature must be matched when screening molecules. By default, the feature is not required to match. These requirements are only needed when the number of sites that must match is less than the number of sites in the hypothesis (partial matching)—see the Must match on at least N site points text box in the Find Matches tab.
Import button

Import an existing hypothesis from file. Opens a file selector, in which you can navigate to and select the hypothesis coordinate (.xyz) file. The other files that are associated with the hypothesis are read as well.

Export button

Export the hypothesis that you have created to a file. Opens a file selector, in which you can navigate to a location and save the hypothesis.
