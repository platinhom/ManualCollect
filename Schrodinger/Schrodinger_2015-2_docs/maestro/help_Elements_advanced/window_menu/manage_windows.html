Managing Maestro Windows

Maestro provides some tools for arranging its windows.

Some panels can be docked into the main window. These panels are docked into the main window by default, but you can change this behavior in the Preferences panel, under General – Panels. The alternatives are not to dock the panels, to dock them into the Workspace, or to dock them into a floating window, named Docked Panels.
