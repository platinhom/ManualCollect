Edit Menu

The Edit menu provides access to tools for editing and adjusting the structures in the Workspace. The default menu items are as follows.

  • Undo—Undo the last undoable action. The menu item changes to display the name of the last command executed. When you undo an action, the menu item changes to Redo. Undoing can also be performed with the Undo button on the Edit toolbar:

  • Copy—Copy the Workspace selection onto the clipboard, and copy an image of the Workspace. The clipboard stores the atoms in the selection in Maestro format and SMILES format, and an an image of the Workspace. The SMILES string and the image can be pasted into another application; the atoms can be pasted back into the Workspace. The resolution of the image is the current screen resolution, so if you want a higher resolution, you should use the Save Image panel. Keyboard shortcut: Ctrl+C (⌘C).

  • Paste—Paste a structure from the clipboard into the Workspace. The structure can be a SMILES string, a structure from ChemDraw or Isis Draw (Windows) or a set of atoms previously cut or copied from the Workspace. Keyboard shortcut: Ctrl+V (⌘V).

    If the structure is a SMILES string, it is converted into a 2D structure. The 2d structure is added to the project as a new entry, with the SMILES string for the structure as the title. No attempt is made at 3D conversion. You should consider running LigPrep to convert the structure to 3D.

    If you paste a set of atoms that were cut or copied from Maestro, they are placed at the coordinates at which they were cut or copied onto the clipboard. You can paste atoms from one Maestro session into another session.

  • Paste Special—Paste a structure into the Workspace, with some extra processing. The items on this submenu are:

    • Selected text—Paste a structure into the Workspace that is in the Linux paste buffer (selected text). To enable this option, you must run the Maestro command workspacepastesettings enableselect=on in the Commands text box, or add it to your maestro.cmd file to turn it on when Maestro starts.
    • By Placing—Place the structure centroid by picking a location in the Workspace. This is useful when there are other structures that you want to position the structure near, or when the coordinate origin is off-screen. When you select this option, the pointer changes to an arrow, to indicate that you are picking a location in the Workspace to past the structure, rather than an atom. Keyboard shortcut: Ctrl+Shift+V (⇧⌘V).
    • As 3D—Convert a 1D or 2D structure to 3D, using any stereochemical information, and paste it into the Workspace with its centroid at the coordinate origin. The structure is added to the project as a new entry, with the SMILES string for the structure as the title.

  • Delete—Delete objects from the Workspace. This submenu has several sections. The first two sections have a single item, for picking objects to delete and for deleting the selected atoms. The next section lists other kinds of structural objects that can be deleted: hydrogens, waters, and zero-order bonds. The item in the next section opens the Atom Selection dialog box for defining which atoms to delete. The last section allows you to delete various kinds of markers associated with the structures in the Workspace: labels, measurements, H-bonds, contacts, ribbons, adjustments, and captions.

  • Pick—Choose the structural unit to use when picking in the Workspace. This submenu allows you to select a structural unit, from Atoms, Residues, Chains, Molecules, or Entries. You can then pick these units in the Workspace to make an atom selection. You can do the same tasks with the Select button on the Edit toolbar.

  • Find—Display the Find toolbar so that you can find structural objects in the Workspace.

  • Sculpting Mode—Select this option to drag atoms in the Workspace to a new location and minimize the energy of the surrounding atoms. See Structure Sculptingfor more information

  • Pick to Transform—Select this option to pick atoms to rotate and translate ("local transformation"). This allows you to modify the structure. You can temporarily exit this mode if you want to rotate or translate the workspace view by holding down the Spacebar.

  • Build—Build or modify structures in the Workspace, using the Build and Fragments toolbars and the Build panel. This submenu has the following items:

    • Build Toolbar—Display the Build toolbar

    • Fragments Toolbar—Display the Fragments toolbar

    • Fragments—Add molecular fragments to a structure, or place them in the Workspace. Opens the Build Panel at the Fragments tab.

    • Atom Properties—Modify atom properties, such as element, atom type, partial charge. Opens the Build Panel at the Atom Properties tab.

    • Residue Properties—Modify residue properties, such as residue type and residue number. Opens the Build Panel at the Residue Properties tab.

    • Clean Up Geometry—Clean up the geometry of the Workspace structure by minimizing its energy. By default the OPLS_2005 force field is used. If this does not include the atoms you need to minimize, you can set a preference to use the UFF force field, which covers a wider range of atoms but is slower.
      Keyboard shortcut: U.

  • 2D Sketcher—Open the 2D Sketcher panel, in which you can draw and edit structures in 2D, then save them either as 2D structures, or converted to 3D with LigPrep.

  • Adjust—Submenu that allows you to perform adjustment by picking, step through adjustments, adjust various defined quantities, open the Adjust panel for adjustment of distances, angles, dihedrals, and chirality, delete adjustments, or open the Rotamers dialog box to select rotamers of the selected residues. See Adjusting Structures for more information.

  • Add Hydrogens—Submenu that allows you to add hydrogen atoms to complete the valence on atoms in the Workspace. Includes items to pick atoms for hydrogen addition, select them in the Atom Selection dialog box, add hydrogens to all atoms, and open the Add Hydrogens - Advanced panel in which you can choose a force-field treatment and apply it to selected hydrogen atoms.

  • Toolbars—Submenu that allows you to show or hide the toolbars.

  • Advanced—Submenu with items for setting preferences, changing the working directory, showing the Command Input Area to type in Maestro commands, and customizing the menus, the toolbars, the mouse, and Maestro profiles.
