Summary

In this panel you can customize the profiles that are available for starting Maestro.

Opening the Customize Profiles Panel

To open the Customize Profiles panel, you can:

  • Choose Edit → Advanced → Customize Profiles in the main window.

Using the Customize Profiles Panel

A profile consists of a collection of settings for menus, toolbars, mouse actions, and preferences. These setttings are stored in files and are read when you start Maestro.

Customize Profiles Panel Features

  • Available profiles table
  • New button
  • Delete button
  • Reset button
  • Show the Choose Profile dialog box at startup option
Available profiles table

This table lists the available profiles. Standard profiles are listed at the top, with a distinguishing color, and user-defined profiles are listed below. You can select a single table row to delete (user-defined profiles) or reset (standard profiles). To change the default profile, click in the Default column for the profile that you want to use as the default.

New button

Click this button to add a new profile to the table. Opens a dialog box in which you can specify the profile name.

Delete button

Delete the selected profile. You can only delete user-defined profiles, and you cannot delete the current profile (highlighted in the table). If you delete the default profile, the profile in the row above it in the table is set as the default.

Reset button

Discard any changes made to one of the standard profiles, after confirmation.

Show the Choose Profile dialog box at startup option

Show the Choose Profile dialog box when Maestro starts, so that you can choose which profile to use for the Maestro session. If you choose not to show this dialog box, the profile that is selected as the default is used. This dialog box is not shown if you only have one profile, or if you start Maestro with an explicit choice of profile in the command.
