Summary

In this panel you can dispay a heat map of the distances between all pairs of a selected PDB atom type in a protein, and select points in the plot for highlighting and display of the distances in the Workspace.

Opening the Residue Distance Map Panel

To open the Residue Distance Map panel, you can:

  • Choose Tasks → Scripts → Workspace Tools → Residue Distance Map in the main window.

Residue Distance Map Panel Features

  • Use structures from option menu
  • File name text box and Browse button
  • Plot options
  • Plot Distances button
  • Show on axes options
  • Distance markers option
  • Highlight residues option
  • Clear Markers button
  • Clear Highlights button
  • Toolbar
  • Plot area
  • Message area
  • Export Distance Data button
Use structures from option menu

Select the source of the structure to display the distance map. The choices are Workspace or File. If you choose File, the Input file text box and Browse button are made active.

File name text box and Browse button

Specify the file to read the protein structure from. Enter the file name in the text box, or click Browse and navigate to the file. The file can be in Maestro or PDB format. Only the first structure in the file is read.

Plot options

Specify the type of atom between which distances are calculated and plotted. The choices are C-alpha (the default), C-beta, Other PDB name (for which you specify the PDB atom name in the associated text box), and Closest atoms, for the shortest distance between any atoms in two residues.

Plot Distances button

Click this button to update the plot when you change the type of atom used for the distance measurements, or when you change the structure.

Show on axes options

Choose the numbers that are shown on the axes of the plot. The choices are Atom numbers, which displays the atom number for the chosen atom type, and Residue numbers, which displays the residue number. A warning is displayed if the protein has insertion codes, as there are duplicate residue numbers in this case.

Distance markers option

Select this option to display distance markers for the points that are selected in the plot.

Highlight residues option

Select this option to highlight the pairs of residues for the points that are selected in the plot. The residues are highlighted by changing the carbon atom colors and using Ball and Stick representation for the residues.

Clear Markers button

Clear all the distance markers in the Workspace.

Clear Highlights button

Clear all the highlighting of resides in the Workspace (return the representation to that used for the rest of the protein).

Plot area

This area displays the plot of the distance between all pairs of alpha carbons in the protein structure, as a heat map. The color scale is displayed to the right of the plot. Clicking on a point in the plot selects that point. When you do this, the pair of residues is highlighted in the Workspace if Highlight residues is selected, the distance between the alpha carbons is displayed if Distance markers is selected, and the Workspace view zooms in or out so that both residues of the pair are visible at the maximum possible size.

Toolbar

The toolbar provides tools for manipulation of the plot and for saving an image of the plot. This is a generic toolbar, and some of the actions may not be useful or relevant. The toolbar buttons are described below.

Reset
Reset the plot to the original pan and zoom settings.
Back
Display the previous view of the plot in the view history.
Forward
Display the next view of the plot in the view history.
Pan/zoom
Pan the plot by dragging with the left mouse button, zoom by dragging with the right button.
Zoom to rectangle
Drag out a rectangle on the plot to zoom in to that rectangle.
Configure subplots
Configure the margins and spacing of each plot in the panel.
Save image
Save an image of the plot to file. Opens a file selector in which you can browse to a location, select the image format, and name the image.
Edit axis and curve parameters
Make settings for the title, range, labeling, and scale of the axes; the color, style, and width of lines; and the color, style, and size of markers.
Copy image
Copy an image of the plot to the clipboard, so you can paste it into another application. This button is only available in some panels.
Message area

This area, below the plot area, displays a message identifying the data that is displayed when the pointer is not over the plot, and displays the identity of the alpha carbon atoms (residue and atom number) and the distance when the pointer is over a point in the plot.

Export Distance Data button

Export the distances used to generate the map to a CSV file. Opens a file selector to name and save the file.
