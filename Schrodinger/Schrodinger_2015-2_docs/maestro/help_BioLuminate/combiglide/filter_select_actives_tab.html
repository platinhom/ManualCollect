Filter and Select Dialog Box — Actives Tab

Summary

In this tab you can designate compounds from the library as active compounds so that enrichment factors can be calculated. To define a compound, select each row in the Compound definition table in turn, select a reagent from the Reagents at position N list for each row, then click Add. The compound is added to the Active compounds table.

Note: Activity data is not required for the enrichment calculations.

Actives Tab Features

  • Compound definition table
  • Add and Clear buttons
  • Reagents at position N list
  • Filter text box
  • Active compounds table
  • Delete, Import, and Export buttons
Compound definition table

This table is used to set up the definition of a compound. The Position column lists the attachment positions, and is noneditable. The Reagent column lists the reagent selected for the given position. This column is editable, but the text in the column must be a valid reagent name for the given position, otherwise the text is not accepted. Entering text sets the filter in the Filter text box. Selecting a row in the table displays the reagents for that position in the Reagents at position N list. To add a reagent to the row, select the desired reagent in the list.

Add and Clear buttons

Click Add to add the definition in the Compound definition table to the Active compounds table. This button is only available when all three rows of the table have a reagent defined.

Click Clear to clear the Reagent column in the Compound definition table

Reagents at position N list

List of reagents at the position selected in the Compound definition table. Selecting a reagent adds it to the Reagent column for the corresponding row of the Compound definition table.

Filter text box

Specify a filter to apply to the reagent list. You can use "*" to represent an arbitrary string and "?" to represent an arbitrary single character. You can also specify alternatives by listing them in braces { }, separated by commas. For example, 3-{F,Cl,Br}* would match any reagent that began with 3-F, 3-Cl, or 3-Br. All reagents whose titles contain the specified string are listed in the Reagents at position N list when you press ENTER.

Active compounds table

This table lists the compounds that have been defined as active. There is one column for each position, which lists the reagent used at this position to define the active compound.

Delete, Import, and Export buttons

Click Delete to delete the selected rows from the Active compounds table.

Click Import to import active compound definitions into the table. A file chooser opens, in which you can navigate to the desired file.

Click Export to export all the active compound definitions in the table to a file. A file chooser opens, in which you can navigate to or specify the desired file.
