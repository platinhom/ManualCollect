Summary

Compose labels from bond properties, and pick bonds to apply the labels.

Opening the Bond Labels Panel

To open the Bond Labels panel, you can:

  • Choose Style → Bond Labels in the main window.

This panel is dockable, so it opens by default in the Workspace. You can change this behavior in the Preferences panel, under General – Panels.

Using the Bond Labels Panel

To create a label you need to choose the fields (properties) that will be displayed in the label. The available fields are listed in the Fields list. The fields are displayed in the order that they appear in the list. You can drag the fields to the desired position in the list. You can hide fields in the label by clearing the check box for the field.

Labels can be added to selected bonds by picking, and they can also be removed by picking. Existing labels can either be replaced or appended to. You can choose what effect picking bonds will have from the Pick to option menu. Once you have decided on this action, you can use the standard picking tools in the Label bonds section to pick the bonds.

You can update all existing labels with the new composition by clicking Update Existing Labels. This is a shortcut for picking all bonds that have labels.

To remove individual labels, choose Clear labels from the Pick to option menu, and pick the labels in the Workspace, or right-click on the labels and choose Delete. To remove all the labels, click Clear All Labels, or choose Labels from the Delete button menu on the Edit toolbar.

The formatting of the bond labels is set up in the Preferences panel, under Atom/Bond labels. The formatting includes the field separator, names of the fields, font, color, and default position. You can change the position of a bond label by dragging the label to the new position. To restore the label to its default position, right-click on it and choose Reset Position.

Bond Labels Panel Features

  • Composition section
  • Pick to option menu
  • Label bonds picking controls
  • Update Existing Labels button
  • Clear All Labels button
  • Label Preferences button
Composition section

In this section, you can choose the content of the bond labels. The label can be composed from the available bond properties.

Fields list

Lists the fields to include in the bond labels, in the order shown. You can select multiple fields in the list, and drag them to reposition them in the list. Each field has a check box that indicates whether it is included in the label or not.

Pick to option menu

This option menu controls the action taken when picking bonds for labeling. The options are:

  • Add labels (replace)—add labels to the picked bonds, replacing any existing labels.
  • Add labels (append)—add labels to the picked bonds, appending the new label to any existing labels.
  • Clear labels—remove labels from the picked bonds.
Label bonds picking controls

Use the standard picking tools to select the bonds to be labeled or the bonds from which to remove labels. The action taken depends on the option selected from the Pick to option menu.

Update Existing Labels button

This button applies new label content and settings to all currently labeled bonds, including settings made in the Preferences panel such as color and position.

Clear All Labels button

This button removes all bond labels. You can also remove labels from the Label All, Pick to Label, and Delete toolbar button menus.

Label Preferences button

This button opens the Preferences panel at the Atom/Bond labels tab, so you can set the label font, field separator, color, position, and set labeling options.
