Undoing an Action

The Undo command returns the structure in the Maestro Workspace to the state it was in prior to the last undoable command.

To undo an action, click the Undo/Redo button on the Edit toolbar, or choose Edit → Undo alias action in the main window, where action is the last action for which Maestro supports an undo.

Maestro provides a single level undo for any operation that results in a chemical change in a displayed structure. In addition, several other types of actions are undoable. The last operation for which the undo option is available appears on the Edit menu. Note that there is no panel associated with this menu item. An undo takes place immediately.

Each Undo command has a corresponding Redo command. Once an Undo has been executed, the Redo option appears in the Undo command's place, and the direction of the arrow on the Undo toolbar button is reversed.
