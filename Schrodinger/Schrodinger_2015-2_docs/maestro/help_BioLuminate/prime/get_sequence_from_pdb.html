Summary

This dialog box allows you to obtain a protein sequence from either the PDB web site or a local copy of the PDB database and import it into Maestro. The PDB file is downloaded first, and the sequence obtained from the PDB file.

Opening the Get Sequence from PDB ID Dialog Box

To open the Get Sequence from PDB ID dialog box, you can:

  • Click PDB ID in the Input Sequence step of the Structure Prediction panel.

Get Sequence from PDB ID Dialog Box Features

  • PDB ID text box
  • Chain name text box
  • Auto option
  • Retrieve from local installation option
  • Download from Web option
  • Download button
PDB ID text box

Enter the 4-character PDB code into this text box for the desired protein. You can use upper case or lower case for the code.

Chain name text box

If you only want to import a single chain, enter the single-letter chain name in this text box. To extract the chain with a blank name, use " "

Auto option

Retrieve the protein sequence from a local copy of the PDB database, and if that fails, download the protein sequence from the RCSB web site.

Retrieve from local installation option

Retrieve the protein sequence from a local copy of the PDB database. The possible locations for this copy are specified by environment variables. The sequence is retrieved from the first location in the following list that has a valid PDB database:

  • $SCHRODINGER_PDB
  • $SCHRODINGER_THIRDPARTY/database/pdb
  • $SCHRODINGER/thirdparty/database/pdb
Download from Web option

Download the protein sequence from the RCSB web site.

Download button

Click this button to download the PDB file and extract the sequence from it.
