Summary

This panel provides tools for designing a ligand in the context of a receptor, by modifying the structure and scoring it with Glide. It provides display tools and access to Glide scoring for the interaction energy of the ligand with the receptor.

Opening the Glide Ligand Designer Panel

To open the Glide Ligand Designer panel, you can:

  • Choose Tasks → Design and Scoring → Ligand Designer: Rigid Binding Site (uses Glide) in the main window.

Glide Ligand Designer Panel Features

  • Use receptor from Workspace option
  • Use receptor from pre-generated grid file option
  • Grid file text box and Browse button
  • Display box option
  • Pick reference ligand option
  • Move ligand only (manual docking mode) option
  • Ligand 2D structure display area
  • Score Workspace Ligand button
  • Method option menu
  • Server option
  • Results table
  • Display button
  • Delete button
  • Export button
  • Host to run Glide on option menu
  • Status area
  • Reset button
Use receptor from Workspace option

Create a receptor from the Workspace structure. To do this, a receptor with a docked ligand must be included in the Workspace. When you have picked the reference ligand, a Glide receptor grid is created based on the Workspace receptor, and is used for scoring the Workspace ligand in the binding site.

Use receptor from pre-generated grid file option

Use the receptor for a grid file that was generated previously. You must include in the Workspace the receptor and ligand that were used to generate the grid. The grid must be specified in the Grid file text box. When you have picked the reference ligand, the grid that you specified is used for scoring the Workspace ligand in the binding site.

Grid file text box and Browse button

If you selectedUse receptor from pre-generated grid file, you can type the name of the grid in the text box, or click Browse and navigate to the grid file.

Display box option

Display the grid box in the Workspace when the grid has been generated or loaded. This option is helpful to check that the ligand to be scored is in the receptor binding site.

Pick reference ligand option

Select this option and pick the ligand in the Workspace. You must pick a ligand before proceeding. If you selected Use receptor from Workspace, picking the ligand starts the Glide grid generation job, then the ligand scoring job. If you selected Use receptor from pre-generated grid file, picking the ligand starts the ligand scoring job. The picked ligand should be already docked in the receptor binding site. If it is not, a warning is displayed.

Move ligand only (manual docking mode) option

Allow the ligand to be translated or rotated. You can use this feature to move the ligand when an added functional group clashes with the receptor, for example.

Ligand 2D structure display area

This area displays the 2D structure of the ligand that is in the Workspace. The 2D structure is updated when you modify the ligand.

Score Workspace Ligand button

Run a scoring job for the current workspace ligand and incorporate the structure and score into the Results table. You can edit the workspace ligand and score it as many times as you want.

Method option menu

Choose the method for Glide ligand scoring. Refine and score does some slight structural changes to improve the score. Score in place scores the ligand as it is, without making any changes to the structure.

Server option

Use a Glide server to dock the ligand. The docking normally takes about 10 seconds, much of which is overhead for the docking job. If you use a Glide server, the overhead is incurred on the first time you dock a ligand, to start the server, and subsequently docking takes about a second. The server is automatically shut down if you close the panel, or more than 10 minutes has elapsed since a ligand was docked.

Results table

This table displays the results of the ligand scoring. You can select a single row in the table for further action. The Structure column contains a name for the structure. The ligand that you displayed and picked is labeled Reference, and subsequent edited ligands are labeled Ligand_n, where n is a sequential index starting with 1.

Display button

Display the structure for the row that is selected in the Results table.

Delete button

Delete the row that is selected in the Results table.

Export button

Export all the structures and scores in the Results table to the Project Table. The project entry titles are taken from the Structures column.

Host to run Glide on option menu

Select the host you want to use to run the Glide scoring jobs.

Status area

Displays information on the current task.

Reset button

Clear all information from the panel, including the grid. After clicking this button, you have to start again with supplying a grid, and so on.
