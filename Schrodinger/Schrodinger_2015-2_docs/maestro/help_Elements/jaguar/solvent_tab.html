Summary

In this tab you choose a solvent from a predetermined set for pKa calculations. The solvent is included in the calculation as a dielectric continuum with a cavity for the molecule.

For information about the controls at the top and bottom of the panel, see the Jaguar Panel topic.

Solvent Tab Features

  • Solvent option menu
  • Molecular weight text box
  • Density text box
  • Probe radius
Solvent option menu

Choose the solvent from this option menu. The Molecular weight, and Density text boxes are filled in, and the Probe radius value is displayed. There are only two choices, Water and DMSO, as these are the only solvents for which pKa parameters are available. The default is Water.

Molecular weight text box

Displays the molecular weight for the chosen solvent. Non-editable.

Density text box

Displays the density for the chosen solvent. Non-editable.

Probe radius

Calculated from the molecular weight and density. Non-editable.
