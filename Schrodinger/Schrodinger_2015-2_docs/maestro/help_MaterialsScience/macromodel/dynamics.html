Summary

Dynamics panel can be used to set up a molecular dynamics simulation.

Opening the Dynamics Panel

To open the Dynamics panel, you can:

  • Choose Applications → MacroModel → Dynamics in the main window.

  • Choose Applications → More Applications → MacroModel → Dynamics in the main window.

Using the Dynamics Panel

MacroModel panels allow you to choose structures from either the Workspace or the Project Table (see Energy Overview for details). The Dynamics panel contains six tabs. For descriptions of the controls in the first five tabs, see Potential, Constraints, Substructure, Mini, and Monitor. The sixth tab, Dynamics, contains the controls that pertain specifically to dynamics simulations. These are described below. Once you have finished configuring the settings on the panel, you can click Run to run the job with the current job settings, or you can choose Job Settings from the Settings button menu to open the Job Settings dialog box to make job settings, or choose Write to write the files without starting the job (see Energy Overview for details).

Method

Choose a method for conducting dynamics simulations:

  • Stochastic Dynamics: (default) The stochastic dynamics procedure is the most commonly used method because it includes random forces that simulate the buffering of a system by solvent molecules. The random forces also assist in crossing barriers during the simulation.

    REFERENCE: Gunsteren, W. F.; Berendsen, H. J. C. Molecular Simulation 1988, 1, 173.

  • Molecule Dynamics: The molecule dynamics method uses a standard constant temperature velocity-Verlet algorithm.

SHAKE

Choose an option to constrain bond lengths to maintain their original values. This allows the use of larger time-steps than unconstrained simulations. Most simulations use the Bonds to Hydrogens option. This allows time-steps up to 2 fs.

REFERENCE: Ryckaert, J.-P.; Ciccotti, G.; Berendsen, H. J. C. J. Comput. Phys.1977, 23, 327.

Simulation temperature (K)

Specify the temperature at which a simulation is carried out. The default value is 300.0  Kelvin.

Timestep

Specify how often a measurement is taken. By default, the value is set to 1.5 fs.

Equilibration time

Specify the length of the "settling down" period at the start of the simulation. The equilibration period is needed to allow initial velocities to stabilize before any monitoring data or sampled structures are written to output files.

Simulation time

Specify the total time allowed for a given simulation. The default is 10 ps. However, if converged results are desired, much longer simulations should be used, even for small to medium-sized systems.
