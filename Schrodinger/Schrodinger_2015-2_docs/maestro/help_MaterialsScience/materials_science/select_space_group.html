Summary

In this dialog box you can choose the space group to use for your crystal structure.

Opening the Select Space Group Dialog Box

To open the Select Space Group dialog box, you can:

  • Click Choose in the New Crystal Structure panel.

Using the Select Space Group Dialog Box

To choose the desired space group, select it in the list of groups. You can filter the list by choosing a crystal system from the System option menu and a lattice type from the Lattice type option menu.

Select Space Group Dialog Box Features

  • System option menu
  • Lattice type option menu
  • Space group list
System option menu

Choose the crystal system from this option menu, to restrict the list of space groups to those that are valid for the crystal system.

Lattice type option menu

Choose the lattice type from this option menu to restrict the list of space groups to the selected lattice type. If a crystal system was chosen, only the lattice types that are valid for this crystal system are shown.

Space group list

List of space groups, filtered according to any choice of system and lattice type. Select the desired space group in the list, and click OK, or double-click the space group to select it and dismiss the dialog box.
