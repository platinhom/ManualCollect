Summary

Build a set of organometallic complexes from a set of ligands. Each complex is composed of a metal, an optional "sentinel" ligand that remains the same in each complex, and enough copies of one other ligand to fill the coordination sites for the chosen complex geometry.

Opening the Build Multiple Complexes Panel

To open the Build Multiple Complexes panel, you can:

  • Choose Tools → Build Multiple Complexes in the main window.

  • Choose Applications → Materials Science → Build Organometallic Complexes → Build Multiple Complexes in the main window.

Using the Build Multiple Complexes Panel

This panel is intended for building a set of organometallic complexes from a set of ligands that are variations on a common core. To build such a set of ligands, you can use the Interactive Enumeration panel. This panel allows you to choose a core structure, which is the one that is bound to the metal, select side chains on this core as attachment points for new functional groups, and replace the side chains with a set of fragments, to build a set of structures. The important feature for building a set of complexes is that the core is the same in each ligand, as you have to designate one or more points on the core that are attached to the metal.

The coordinating (ligating) atoms are specified by their atom number. To find out what the atom numbering is in your ligands, you can label one of the ligands in the Workspace. Include a ligand, display the Labels toolbar if it is not displayed (click Labels on the Manager toolbar or choose Edit → Settings → Toolbars → Select Toolbars Displayed → Labels), then click the Label All button on the toolbar and choose Atom Number.

You can then use the numbers displayed on the atom to specify the coordinating atom (monodentate ligand) or atoms (bidentate ligand).

When specifying the atom that is ligated, you can request the removal of atoms that are attached to the ligating atom: for example, if you have a secondary amine in a ring (such as in pyrrole), you might want to remove the attached hydrogen. If there are no extra atoms to remove (as for example for cyano), select Dative bond. If there is a single extra hydrogen (as for example in pyrrole), select Remove lone hydrogen. If some other functional group is attached to the coordinating atom, select Remove atom, and choose the atom from the list, which shows the atomic symbol and the atom number. The attached atom and all atoms in the side chain originating at the attached atom are removed when the complex is built. For example, if the atom to remove is the C in a CONH2 group, the entire CONH2 group is removed. Likewise, if the atom is part of a phenyl group, the entire phenyl group is removed. If the atom to remove and the coordinating atom are part of the same ring system, no additional atoms are removed beyond the specified atom to remove.

To specify the complex geometry and isomer, choose from the Geometry and Isomer option menus. The Isomer option menu is available if you have bidentate ligands and an octahedral or square planar complex. For octahedral complexes, you can choose to build a meridional or facial isomer. Both isomers are chiral, but only one of the stereoisomers is built. The facial stereoisomer is in the Λ form, the meridional stereoisomer is in the Δ form. (To convert to the other stereoisomer, display the Build toolbar, and double-click the R/S button.) If the ligands are symmetric, there are only the two stereoisomers, which are built by choosing the facial or meridional isomer. For square planar complexes with asymmetric bidentate ligands, you can choose to build the cis or the trans isomer.

If you want to construct complexes with a common ligand, while enumerating other ligands, you can select Sentinel ligand, then specify the ligand to use as the common ("sentinel") ligand and the number of copies of this ligand to use. The ligand can be monodentate or bidentate. The remaining sites in the complex are filled with the "enumeration" ligands.

You can choose the sentinel ligand from the Ligand option menu, or sketch a new ligand by clicking Sketch and drawing it in the Ligand Sketcher. You must indicate which atoms are the ligating atoms, by adding a bond to them and labeling the terminal atom as R1 for the first ligating atom, and R2 for the second. Right-click on the atom and choose Set R Group → R1 or R2 to perform the labeling. When you have finished sketching the ligand, click Use This Structure to use it for this run only, or click Save New Template, to save it as a "template" ligand, which is added to the list of ligands on the Ligand option menu.

When you have built the complexes, you should consider cleaning them up with the Ligand Cleanup panel, and then running a geometry optimization with Jaguar before proceeding with any calculations on the complexes.

A video on optoelectronics screening, which uses this panel, is available at http://www.schrodinger.com/videos/optoelectronics_workflow_matsci/ (in Korean at http://www.schrodinger.com/videos/optoelectronics_workflow_matsci_kor/).

Use structures from option menu

Choose the source of the structural input for the job:

  • Project Table (selected entries)—Use the entries that are currently selected in the Project Table.
  • Workspace (included entries)—Use the entries that are currently included in the Workspace.
  • File—Use the structures from the specified file. When this option is selected, the Browse button is activated. Click Browse and navigate to the file you want to use. The file name is displayed in the text box when you click Open in the file selector.
File name text box and Browse button

If you chose File from the Use structures from option menu, click Browse and navigate to the structure file. The name of the file you selected is displayed in the text box.

Complex section

In this section you specify details of the complex: the metal atom and the geometry.

Metal atom text box

Enter the atomic symbol for the metal atom in this text box. Only valid atomic symbols are accepted.

Geometry option menu

Specify the coordination geometry. The choices are Octahedral, Tetrahedral, and Square planar. When you choose Octahedral, the Isomer option menu is activated.

Isomer option menu

With asymmetric bidentate ligands, two geometric isomers are possible for octahedral complexes and square planar complexes.

For octahedral complexes, you can choose whether to build the facial isomer or the meridional isomer. Both isomers are chiral, but only one of the stereoisomers is built. The facial stereoisomer is in the Λ form, the meridional stereoisomer is in the Δ form. (If the ligands are symmetric, there are only the two stereoisomers, which are built by choosing the facial or meridional isomer.)

For square planar complexes, you can choose to build the cis isomer or the trans isomer.

Sentinel ligand option and section

Select this option if you want to specify a ligand that is common to all complexes (the "sentinel" ligand), and specify the ligand with the controls in this section.

Sketch button

Click this button to sketch the sentinel ligand. The Ligand Sketcher panel opens, with tools to sketch and save the ligand, either for this run only (Use This Structure) or as a template ligand (Save New Template) to be added to the Ligand option menu for future runs as well. For more information on sketching the ligand, see Using the above and the 2D Workspace topic.

Ligand option menu

Choose a sentinel ligand from this option menu. The empirical formula of the ligand is displayed to the right of the menu, and the name given to the ligand is displayed in the Name text box. If you choose Custom, click Sketch to sketch a custom ligand.

The list of ligands is obtaned from a resource file, to which you can add using the Ligand Sketcher panel (click Sketch to open the panel; click Save New Template to save the sketched structure as a new template ligand).

Name text box

Specify the name of the ligand. This text box is automatically filled in if you choose a template ligand from the Ligand option menu, but is blank for a custom ligand.

Copies text box

Specify the number of copies of the sentinel ligand to be included in all complexes. You can use the arrow buttons or type in the number.

Enumeration ligands section

In this section you specify the common characteristics of the ligands that are used to fill the coordination sites. The ligands must have a common substructure that is coordinated to the metal.

Coordination sites option menu

Specify whether the coordination sites are to be filled with monodentate or bidentate ligands. If you choose Bidentate, the second row of controls below this option menu is activated, so you can select the second coordinating atom and specify its coordination behavior.

Coordinating atom text box and options

Specify which of the atoms in each ligand is the coordinating atom in the text box, and select an option for how the ligand needs to be prepared for bonding to the metal. If you choose Monodentate from the Ligand is option menu, only the first row of controls is available; if you choose Bidentate, both rows are available.

The coordinating atom number is the atom number in the Maestro file. If you have a ligand structure in the Workspace, you can label it with the atom numbers by choosing Atom Number from the Label All button menu on the Labels toolbar. The coordinating atom or atoms are marked in the Workspace with green markers.

The ligands that you choose must all have the same atom numbers for the part of the structure that coordinates to the metal. The choice of ligands should therefore be a series of related ligands, such as generated in the Interactive Enumeration panel.

The options for coordination behavior allow you to specify how the ligand is prepared for coordination. If the coordinating atom on the ligand donates to the metal, select Dative bond. If the ligand has a hydrogen attached to the coordinating atom that is removed (i.e. it is an acid of some kind), select Remove lone hydrogen. If there is some other atom attached, select Remove atom and choose the atom number from the option menu. This option requires a structure in the Workspace, so that the atoms connected to the coordinating atom can be determined. This atom is removed along with all atoms in the side chain that originates with this atom. The atom to be removed is marked with a red marker on the structure in the Workspace (if present).

Build Complexes button

Create the complexes. When the complexes have been created, they are added to the Project Table as single entries, named M(L)n, where M is the atomic symbol of the metal, L is the title of the ligand, and n is the number of ligands.

Reset Panel button

Clear all data from the panel and reset the options to the defaults.
