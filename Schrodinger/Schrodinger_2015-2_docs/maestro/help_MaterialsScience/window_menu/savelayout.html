Saving the Panel Layout

The Save Layout command on the Window → Advanced Options menu saves the positions and locations of Maestro panels, so that they are opened at the same location and size in the current or subsequent Maestro sessions.

This command allows you to position Maestro panels in the location that you prefer them, resize them to the size that you prefer, and save this layout for future use. The layout is saved in the layout resource file layout.res, which is written to the profiles/profile subdirectory in your Schrödinger user resource directory, ($HOME/.schrodinger/maestronn on Linux and Mac, %APPDATA%\Schrodinger\maestronn on Windows), and is read when Maestro starts. When you choose Save Layout, the layout for all panels that have been opened in the current Maestro session (whether or not they are currently visible) is added to the list read from the resource file and written out again to the resource file.

  • If you want to reset all panel layouts to the defaults, delete the resource file.
  • If you want to reset the layout for just a few panels, you can edit the resource file.
  • If you have made changes and want to revert to the last saved layout, choose Window → Advanced Options → Restore Layout.
