Summary

Use this panel to set up and run a Hybrid Monte Carlo (HMC) simulation on the Workspace structure or on a selected entry in the Project Table. HMC simulations achieve relatively efficient sampling by interleaving Monte Carlo moves with a short sequence of molecular-dynamics steps. Because HMC is used mainly as a sampling method, for example, in Liaison binding affinity calculations, the MD steps can use a somewhat larger time step than would normally be advisable. The Metropolis algorithm determines which MD moves should be accepted or rejected. This ensures that the simulation does not go far astray, even if the MD time step would normally lead to a failure of energy conservation. This is why HMC is sometimes called "bad MD but good MC".

Note: Only one entry from the Maestro Project Table can be displayed and used for each HMC job. If you need to merge two structures or entries, use the Project Table's Merge option to create a combination entry, and use that as the basis for your simulation.

Opening the Impact Hybrid Monte Carlo Panel

To open the Impact Hybrid Monte Carlo panel, you can:

  • Choose Applications → More Applications → Impact → Hybrid MC in the main window.

Impact Hybrid Monte Carlo Panel Features

Along with features common to Impact panels (see the Impact Panels Overview topic for more information), the panel includes the following four tabs, which are described in separate topics:

  • Potential
  • Constraints
  • MD Parameters
  • Hybrid MC
