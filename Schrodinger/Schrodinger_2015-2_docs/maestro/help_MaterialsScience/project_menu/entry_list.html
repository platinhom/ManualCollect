Summary

This panel contains a simplified view of the Project Table that allows you to perform basic operations such as selection and inclusion without having the full Project Table present.

Opening the Entry List Panel

To open the Entry List panel, you can:

  • Choose Edit → Settings → Entry List in the main window.

  • Choose Window → Show → Entry List in the main window.

  • Press Ctrl+E (⌘E) in the main window.

This panel is dockable, so it opens by default in the Workspace. You can change this behavior in the Preferences panel, under General – Panels.

Using the Entry List Panel

This panel is intended to provide access to the basic features of the Project Table in a lightweight panel, without having the full Project Table panel open. It can be docked into the Workspace, so you can easily switch focus from the panel to the Workspace.

You can step through the inclusion of entries in the Workspace with the Left Arrow and Right Arrow keys. Selection and inclusion is done in the same way as in the Project Table. You can perform other operations from the shortcut menus.

To filter the list of entries shown, use the text box or the Filter button. Filtering the list merely hides rows: it does not change the entry selection.

Entry List Panel Features

  • Search text box and Filter button
  • Settings button
  • Entry list
  • Project information
  • Tips text area
Search text box and Filter button

The search text box and Filter button allow you to filter the list of entries that is shown in the entry list. Filtering this list has no effect on the selection or the inclusion of the entries, or what is visible in the Project Table.

To filter the list by title, enter the text to match in the text box. The list is filtered as you type. The matching is case-insensitive. To clear the text filter, click the Clear (X) button.

You can also apply a global filter, by clicking the Filter button. The button is indented, to indicate that the global filter is applied. To remove the global filter without removing the text filter, click the button again. There are three ways of defining the global filter, which you can set by clicking and holding on the button, then choosing from the menu that is displayed.

  • Included—Show only the entries that are included in the Workspace in the list.
  • Selected—Show only the entries that are selected in the Project Table in the list.
  • Filter—Open the Entry Selection dialog box to define the filter in terms of entry properties or previously defined filters. If you want to change the filter, you can select this item again, even if it is already selected.

Filters are reapplied whenever the Project Table changes. For example, if the global filter is Selected, and you deselect an entry, it is removed from the list.

Settings button

This button menu allows you to select a property to show, show the 2D Structure column, and fit the column widths to the data, the header, or the larger of the two. Choosing Show Property opens a property selector in which you can select a single property to show in the entry list.

Entry list

Lists the project entries, with the Row column, the In column and the Title column displayed by default. For entry groups, the number of entries is shown to the right of the title. The entry list is kept synchronized with the Project Table.

You can select multiple entries in the list with the usual shift-click and control-click. If the list is filtered and you select with shift-click, only the entries that are shown in the list are added to the selection. For entries that are in between these entries but are hidden, the selection is not changed, so selected entries remain selected, unselected entries are not selected. The same applies to entry inclusion. If the list is filtered, only the entries that are shown in the list are included in the Workspace.

Shortcut menus are displayed when you right-click on a row, so that you can perform common operations, like Fix and Unfix, Duplicate, and so on. These menus are the same as the menus that appear in the Project Table, with a few exceptions. For an entry row, the Entry shortcut menu is shown, without the Copy Properties and Paste Properties items; for an entry group row, the Entry group shortcut menu is shown, without the Edit Title item.

The heading row has a reduced shortcut menu that allows you to show or hide property columns, show the 2D structure, sort by a property value, add or remove properties from the Workspace feedback, and set the column width.

Project information

This area shows information on the entries and entry groups that are present in the project, which filter (if any) has been applied to the entry list, and how many entries are shown in the entry list after filtering.

Tips text area

This area displays a list of suggestions for using the panel.
