Summary

In this tab you can filter out compounds based on conditions on a given set of properties that are important for drug candidates. Some of these properties are generated by QikProp. The property ranges in the filters represent the undesirable property ranges: molecules whose properties fall inside these ranges will be filtered out.

Filtering is unavailable if you do not have combinatorial docking results.

To use a predefined filter:

  1. Choose a filter from the Filter set option menu.
  2. Enter the number of conditions that must be met for the molecule to be discarded in the Discard structure if N or more conditions are met text box.

To use a custom filter:

  1. Choose a filter set from the Filter set option menu.

    You can choose Custom for a blank filter form, or choose one of the predefined filter sets to alter.

  2. Enter the number of conditions that must be met for the molecule to be discarded in the Discard structure if N or more conditions are met text box.
  3. Select the properties that you want to include in the filter set.
  4. Enter the values in the text box that define the property range.

Filter Tab Features

  • Filter set option menu
  • Discard structure if N or more conditions are met text box
  • Property condition options
  • Import and Export buttons
Filter set option menu

This menu provides three preset filters (Druglike, Leadlike, and Coarse), a customizable filter (Custom). and None (do not filter). If you modify one of the preset filters, the menu item switches to Custom.

Discard structure if N or more conditions are met text box

Defines the number of conditions on the properties that must be met for the structure to be discarded.

Property condition options

A property condition is included in the filter if it is selected. The property range is specified in the text boxes.

Import and Export buttons

To import a filter, click Import, and navigate to the filter in the file chooser that opens. The file name is displayed below the buttons. To export a filter to a file, click Export, and enter the file name in the file chooser that opens. The default file extension for the filters is -qfilter.txt.
