Summary

In this dialog box you can make settings that control the calculation of solvent-accessible surface areas, which are used to filter out hydrophobic sites that have too small an area.

Opening the SAS Options Dialog Box

To open the SAS Options dialog box, you can:

  • Click SAS Options in the Edit Features dialog box.

SAS Options Dialog Box Features

  • Minimum required surface area per site text box
  • Solvent radius text box
  • Surface area resolution element
Minimum required surface area per site text box

Enter the minimum surface area in Å2 that a site must have in order to be considered as a hydrophobic site.

Solvent radius text box

Enter the solvent radius. The default of 1.4 Å corresponds to water.

Surface area resolution element

Enter the width in angstroms of the smallest element to be used in the calculation of the surface area.
