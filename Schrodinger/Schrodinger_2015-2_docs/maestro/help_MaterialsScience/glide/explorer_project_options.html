Summary

This panel contains information about the project used and allows you to change the receptor and poses, and add properties.

Opening the Advanced Project Options Panel

To open the Advanced Project Options panel, you can:

  • Click Advanced in the Pose Explorer panel.

Advanced Project Options Panel Features

  • Change Receptor button
  • Change Poses button
  • Advanced Property Preparation button
Change Receptor button

Change the receptor. Opens the Change Receptor dialog box, which lists the available entries in the project by title, so you can select a different receptor. By default, the first entry in the project is selected as the receptor. You can select a single entry from the list and click Set Receptor, or you can choose not to have a receptor, by clicking No Receptor.

Change Poses button

Select the poses that you want to cluster. Opens the Select Poses dialog box, which lists the entries in the project by title. You can select all entries except the receptor by clicking Select All But Target, clear the selection by clicking Select None, or select the entries that are selected in the Project Table by clicking Use PT Selection. You can also select entries in the list with shift-click and control-click. When you have the desired entries, click Set Poses to confirm the pose selection and dismiss the dialog box.

Advanced Property Preparation button

Prepare properties for use in clustering and mapping. Opens the Property Preparation panel, in which you can calculate, import, and delete properties.
