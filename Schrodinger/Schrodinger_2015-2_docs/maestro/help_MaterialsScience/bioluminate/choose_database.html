Summary

In this dialog box you can choose the databases to use for antibody prediction or CDR grafting.

Opening the Choose Database Dialog Box

To open the Choose Database dialog box, you can:

  • Choose Applications → More Applications → BioLuminate → Antibody Modeling → Humanization → CDR Grafting in the main window.

  • Choose Applications → More Applications → BioLuminate → Antibody Modeling → Prediction in the main window.

  • Click Choose Databases in the Framework Search tab of the Antibody Prediction panel.

  • Click Choose Databases in the Antibody Humanization: CDR Grafting panel.

Using the Choose Database Dialog Box

  • To add a database to the table, click Add Database.

  • To make a database active or inactive for the current search, use the check box in the Active column.

  • To remove a database from the table, click the red − button in the Actions column.

Choose Database Dialog Box Features

  • Add Database button
  • Database table
Add Database button

Click this button to add a database to the table of available databases. Opens a file selector, in which you can select the database file, which has the extension .db, and must be a database that is generated or managed by the Antibody Database Management panel.

Database table

This table lists the available databases. You can enable or disable the use of a database with the check box in the Active column. All active databases are used in the search in the parent panel. To delete a database from the table, click the red − button in the Actions column.
