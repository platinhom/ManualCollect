Summary

The Conformational Search panel is used to set up a conformational search starting from the structure in the Workspace or from selected entries in the Project Table.

Opening the Conformational Search Panel

To open the Conformational Search panel, you can:

  • Choose Applications → MacroModel → Conformational Search in the main window.

Using the Conformational Search Panel

MacroModel panels allow you to choose structures from either the Workspace or the Project Table (see Energy Overview for details). The Conformational Search panel contains five tabs. For descriptions of the controls in the first four tabs, see Potential, Constraints, Substructure, and Mini. The fifth tab, CSearch, contains the controls that are directly related to conformational searches. Once you have finished configuring the settings on the panel, you can click Run to run the job with the current job settings, or you can choose Job Settings from the Settings button menu to open the Job Settings dialog box to make job settings, or choose Write to write the files without starting the job (see Energy Overview for details).

Method option menu

This option menu allows you to choose one of six methods for performing a conformational search. MCMM and SPMC are Monte Carlo methods that require a number of variables to be defined (see Search variable section). For pure low-mode conformational searches, setting these variables is optional.

Multi-ligand option

Select this option if the input consists of multiple structures that are not conformers. This option is only available for methods that support "serial" searches: Torsional sampling (MCMM), Mixed torsional/Low-mode sampling, and Low-mode sampling.

Perform automatic setup during calculation option

This option allows automatic setup to be performed during the calculation rather than prior to the calculation, with a set of defaults for the search variables. The automatic setup is performed for each structure processed. This option inserts an AUTO opcode into the MacroModel command file. Selecting this option disables the other controls in the CSearch tab for setting up variables.

Perform Automatic Setup button

Before starting a conformational search, variables must be set that determine which torsions are to be rotated, which rings are to be opened, etc. You can either set these variable automatically or manually:

  • To automatically generate all the necessary variables for the search, click Perform Automatic Setup.
  • To set the variables manually, choose an item from the Search variables option menu, then click the Edit button, to open the panel in which you can define and edit the conformational search variables. These panels are described in separate help topics. See the related topics list below.

If you choose to set the variables automatically, you can always review and edit the variables by using the Search variables option menu and Edit button.

Reset All Variables button

To reset (i.e. delete all instances of) the conformational search variables, click this button.

Torsion sampling options option menu

This option menu allows you to control the automatic setup of torsional sampling around some bonds that generally have high barriers. These bonds are the C-N and C-O single bonds in groups like amides and esters, and C=N and N=N double bonds. The four levels of sampling are as follows:

Restricted   Do not sample any of these bonds
Intermediate   Sample C-N and C-O single bonds other than in standard amides and esters
Enhanced   Sample all C-N and C-O single bonds
Extended   Sample all C-N and C-O single bonds and C=N and N=N double bonds
Retain mirror-image conformations option

By default, a structure and its mirror image are both considered when comparing with the other structures. If one of the two matches, the structure is eliminated. Select this option to keep both the structure and its mirror image. This option is useful when stereochemistry is critical for the biological function of the molecule, for example.

Search variables option menu and Edit button

From this option menu you can select a search variable type for editing, and click Edit to open the appropriate panel to edit the search variables. If you set up the search using the Perform Automatic Setup button, these panels can be used to view and edit the automated selections. If you choose to specify the search variables manually, select the variable type from the option menu and click the Edit button, then set options for the variable in the panel that opens. Each panel has its own help topic. For more information, see the list of related topics below.

Marker display buttons

The Display All Markers and Undisplay All Markers buttons control the display of all markers for the various conformational search variables. Markers for a specific variable can be turned on or off with the Markers or Show markers option in the appropriate panel.

Maximum number of steps text box

This option specifies the number of steps performed in any search. When the number of generated trial structures matches the value in field, the conformational search is terminated.

Use N steps per rotatable bond option and text box

This option restricts the number of steps to a multiple of the number of rotatable bonds. This capability is useful in serial searches, where the number of rotatable bonds can vary between structures. The number of steps is also limited by the value in the Maximum number of steps text box.

Number of structures to save for each search text box

This option specifies the number of structures to save for each search, counting from the lowest in energy. A zero value means "save all structures".

Energy window for saving structures text box

This is the threshold value for comparison of trial structures. New structures generated and minimized are kept only if their energy is less than this value above the current global minimum. Lowering this value results in fewer structures saved. The default value is 50 kJ/mol.

Eliminate redundant conformers using options and Cutoff text boxes

Select the distance mesasure for determining whether structures should be considered equivalent:

  • Maximum atom deviation—consider structures to be different if the maximum atom deviation for any pair of corresponding atoms exceeds the threshold given in the Cutoff text box.

  • RMSD—consider structures to be different if the RMS deviation for all compared atoms exceeds the threshold given in the Cutoff text box.

The default cutoff is 0.5 Å for both options. Unless there are fixed or frozen atoms, the structures are superimposed prior to calculating the distances.

Low-Mode Options

In the lower portion of the panel, there are three options that are only relevant to the "low-mode" searches, and these are only active when any of the six methods involving low-mode conformational searching is selected.

Probability of a torsion rotation/molecule translation

This option allows the setting of a probability that any defined torsion rotations and molecule translations will be made during the search. This should be a number from 0.0 to 1.0. This option is used with:

  • Mixed torsional/Low-mode sampling
  • Serial torsional/Low-mode sampling
  • Mixed torsional/Large scale low-mode sampling
Minimum distance for low-mode move
Maximum distance for low-mode move

These options are used for setting the minimum and maximum distance for a "low-mode" move. During a search, the fastest moving atom is moved randomly generated distances that are between these two limits. This option is used with all the low-mode methods.
