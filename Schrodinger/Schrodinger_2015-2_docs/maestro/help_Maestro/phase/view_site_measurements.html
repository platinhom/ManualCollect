Summary

This panel allows you to select distance and angle measurements between sites in a hypothesis for display in the Workspace.

Opening the View Site Measurements Panel

To open the View Site Measurements panel, you can:

  • Choose Display → View Site Measurements in the Develop Common Pharmacophore Hypotheses panel.

  • Click the View Site Measurements toolbar button in the Manage Pharmacophore Hypotheses panel or the Develop Common Pharmacophore Hypotheses panel.

Using the View Site Measurements Panel

To display individual distances or angles, select them in the tables. You can select multiple rows with shift-click and control-click.

To display all distances or all angles, click the All button below the corresponding table.

To export the selected measurements to a plain text file, click Export, and navigate to the desired location in the file selector that is displayed.

When you have finished viewing measurements, click None for each table to clear the measurements from the Workspace before closing the panel.

This panel allows you to interact with the Workspace while it is open, but not to perform any other actions in Maestro.

View Site Measurements Panel Features

  • Distances table
  • Distances Select buttons
  • Angles table
  • Angles Select buttons
  • Export button
Distances table

This table lists the intersite distances in the selected hypotheses. The hypothesis ID is listed, along with the site labels and the distance. Multiple rows can be selected with shift-click and control-click. When a row is selected, the distance is displayed in the Workspace.

Distances Select buttons

Click All to select all rows, or None to clear the selection for all rows.

Angles table

This table lists the intersite angles in the selected hypotheses. For each angle, the hypothesis ID is listed, along with the site labels and the angle. Multiple rows can be selected with shift-click and control-click. When a row is selected, the angle is displayed in the Workspace.

Angles Select buttons

Click All to select all rows, or None to clear the selection for all rows.

Export button

Export the measurements that are selected in the tables to a plain text file. Opens the Export Displayed Site Measurements file selector, in which you can navigate to the desired directory and name the file.
