Summary

In this dialog box you can set the weight of the survival score of the inactive ligands in the adjusted score, and start the scoring job for the inactives. The survival scoring function is defined in the Score Actives dialog box.

Opening the Score Inactives Dialog Box

To open the Score Inactives dialog box, you can:

  • Click Score Inactives in the Score Hypotheses step of the Build Common Pharmacophore Hypotheses panel.
