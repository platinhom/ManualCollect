Summary

In this panel you can generate a surface that indicates regions with a propensity for aggregation.

Opening the Aggregation Surface Panel

To open the Aggregation Surface panel, you can:

  • Choose Applications → BioLuminate → Aggregation Surface in the main window.

  • Choose Tasks → Biologics → Aggregation Surface in the main window.

Using the Aggregation Surface Panel

Aggregation regions are defined by identifying clusters of exposed hydrophobic residues. A molecular surface is created for a large probe molecule, representing a protein. The surface is colored red in the regions near these residues, which are the regions where aggregation might occur. The side chains of the hydrophobic atoms are colored red and displayed in ball-and-stick. The alpha carbon is also colored red, and the other backbone atoms are colored gray. The backbone is displayed as lines. Residues that are in contact with the hydrophobic side chains are also displayed; all other residues are hidden. The residues involved are colored red in the Workspace sequence viewer.

Once you have located aggregation regions, you might want to mutate residues in this region to reduce the size of the region.

To create an aggregation surface (in the Create tab):

  1. Select atoms to define the structure for the surface, if you want to limit the calculation to part of the protein.

  2. Choose an option for the part of the Workspace structure that you want to create the surface for.

  3. Enter a name in the Surface name text box, if you want to change the default name.

  4. Click Create Surface.

The surface is stored persisently with the project entry, and it can be managed in the Manage Surfaces panel.

To analyze the surface to identify residues and residue groups that contribute to aggregation regions (in the Analyze tab):

  1. Choose the surface from the Surface option menu.

  2. Click Analyze.

To select residues in a group, choose the group from the Select group option menu. You can then use the selection for other purposes.

Once you have a selection, you can run a job to perform mutations on these residues, as follows:

  1. Choose Applications → BioLuminate → Residue Scanning / Affinity Maturation → Perform Calculations to open the Residue Scanning panel.

  2. Select Analyze only selected Workspace residues.

  3. Click Analyze Workspace.

The residues that were found to contribute to the aggregation region are listed in the table in the Residues tab. You can then select any of them for mutation, define the mutations, and run the job.

If you want to export the data from the table in CSV format, click Export and save the file in the file selector that opens.

Aggregation Surface Panel Features

  • Create tab
  • Analyze tab

Create tab

In this tab you set options for creating the surface.

  • Compute aggregation surface for options
  • Surface name text box
  • Options button
  • Create Surface button
Compute aggregation surface for options

Choose an option to define the structure for which the aggregation surface is computed.

  • Workspace—Compute the surface for the structure in the Workspace. You must have only a single entry in the Workspace.
  • Entire entry containing selected atoms—Compute the surface for the entry that contains the selected atoms. This is useful if you want to view how the aggregation surface on one protein matches residues in another protein.
  • Entire chain containing selected atoms—Compute the surface for the entire chains that contain the selected atoms. This is useful for computing a surface for one chain of a multi-chain protein, for example.
  • Selected atoms and attached hydrogen atoms only—Compute the surface for the selected atoms with their attached hydrogens only. This option allows you to calculate the surface for only part of a protein chain, for example.
Surface name text box

Enter the name you want to associate with the surface in this text box. This name is used in the Manage Surfaces panel to identify the surface.

Options button

Open the Aggregation Surface - Options dialog box, where you can set parameters that control the quality and appearance of the surface, and parameters for locating aggregation regions.

Create Surface button

Click this button to create the surface. A progress bar is displayed above the buttons while the surface is being created. Surface creation should take less than a minute.

When the surface is created, it is displayed in the Workspace and other surfaces are hidden. To display more than one surface, use the Manage Surfaces panel (Workspace → Surface → Manage Surfaces).

Analyze tab

In this tab you can analyze an aggregation surface.

  • Surface option menu
  • Analyze button
  • Export button
  • Aggregation residues table
  • Select group option menu
  • Color selected green option
  • Reset Aggregation Colors button
  • Display surface option
  • Zoom to selection option
Surface option menu

Choose an aggregation surface for analysis from this option menu.

Analyze button

Analyze the chosen surface to locate residues and residue groups that contribute to aggregation regions.

Export button

Export the data in the table to a CSV file. Opens a file selector so you can save the file in the desired location.

Aggregation residues table

This table displays the residues that contribute to the aggregation region for the analyzed surface. Each residue is listed with its contribution to the surface (a count of surface elements, roughly equivalent to an area), and a group index. A group is a set of residues that contribute to the same aggregation region, defined by the Radius to find neighbors setting in the Aggregation Surface - Options dialog box.

Selecting a row in the table selects that residue in the workspace. If Color selected green is selected, the residue and the red surface patch associated with that residue is colored green.

Select group option menu

Select all residues in the table that belong to the group chosen from this menu. The total contribution for the group is shown on the menu as well as the group index.

Color selected green option

Color the residues that are selected in the table and the associated surface patches green. This allow you to see which patches or residues are selected.

Reset Aggregation Colors button

Reset the colors of the atoms and the surfaces to the original aggregation color scheme (red and gray).

Display surface option

Select or deselect this option to display or hide the surface. The coloring is still applied to the residues even when the surface is hidden.

Zoom to selection option

Zoom the view in the Workspace to the selected residues.
