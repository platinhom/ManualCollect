Summary

The Install KNIME Workflows panel allows you to install wrapped KNIME workflows so that they appear on the KNIME submenu of the Workflows menu.

Opening the Install KNIME Workflows Panel

To open the Install KNIME Workflows panel, you can:

  • Choose Workflows → Install in the main window.

Using the Install KNIME Workflows Panel

To install a workflow:

  1. If necessary, click Browse to find a directory containing KNIME Workflow startup scripts. The default directory is $HOME on Linux and %USERPROFILE% on Windows.

    Note: The startup scripts for the workflows are Python scripts, which should have a .py suffix.

  2. Select a workflow by clicking its box in the Install column.

    The Name is what appears in the Workflows → KNIME submenu.

  3. Click OK to add the workflow to the menu.

    If the workflow already exists in the menu, you are asked if you want to overwrite it. Clicking Yes overwrites the workflow; clicking No aborts the workflow installation.

Install KNIME Workflows Panel Features

  • Download link
  • Directory text box and Browse button
  • Workflow table
  • Description text area
Download link

This link takes you to the Workflow Center on the Schrödinger web site, where you can download workflows to install.

Directory text box and Browse button

The text box displays the directory for selecting workflows to install. The default directory is $HOME on Linux and %USERPROFILE% on Windows.

To choose another directory, click Browse and navigate to the directory containing the workflows you want to install.

Workflow table

This table contains all the workflows available for installation in the current directory. To choose a workflow to install, click its box in the Install field. The name of the workflow, which will appear on the Workflows menu, is listed in the Name column.

Description text area

Displays a description (if available) of the currently selected workflow.
