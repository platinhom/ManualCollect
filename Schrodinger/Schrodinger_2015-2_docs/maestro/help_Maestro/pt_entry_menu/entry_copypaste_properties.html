Copying and Pasting Entry Properties

You can copy properties from a selected entry in the Project Table panel to the clipboard, and paste them into a selection of other entries, either in the current project or another project.

To copy properties, use Entry → Copy Properties, and choose All to copy all properties, or Shown to copy the visible properties.

To paste properties, select the entries and choose Entry → Paste Properties. The copied properties overwrite any existing properties of the same (internal) name. If a property does not exist (or is of a different type) a new property is created and values added for the selected entries. If a property value is blank (null) in the copied property, it still overwrites values in the target entries.
