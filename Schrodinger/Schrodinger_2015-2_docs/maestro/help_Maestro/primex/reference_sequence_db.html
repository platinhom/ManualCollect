Summary

The Reference Sequence panel displays the reference sequence. You can use the Find text box and the Previous and Next buttons to find a particular sequence of residues within the reference sequence.

The reference sequence is not displayed in the sequence viewer because it does not correspond to any structure in the Workspace.

Opening the Reference Sequence Panel

To open the Reference Sequence panel, you can:

  • Click Reference Sequence in the PrimeX panel.
