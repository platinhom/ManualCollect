Conformational Searches with MacroModel and Jaguar

Jaguar and MacroModel can be combined into a very powerful tool for finding the preferred conformations of molecules. MacroModel is used to perform the search through conformational space to identify the lowest-energy conformations. Jaguar is then used to refine the geometries of the lowest-energy conformers. Redundant conformers can then be removed from the data set.

A sample procedure is as follows:

  1. Run the MacroModel conformational search:

    1. Choose Applications → MacroModel → Conformational Search.
    2. In the CSearch tab, choose Mixed torsional/Low-mode sampling from the Method option menu.
    3. If you are restricting the conformer set by energy, set the energy cutoff to about twice the size of the desired final cutoff, to allow for differences between the MM and QM potential energy surface.
    4. Make any other settings, then run the job.

  2. (Optional) To ensure that the search is sufficiently complete, run a second conformational search, using the results of the first search as input.

  3. Examine the results in the Project Table, and select a set of conformers for Jaguar refinement. If you have too many conformations, consider clustering the conformations with XCluster, and choosing a representative set from the clusters.

  4. Run the Jaguar optimization:

    1. Choose Applications → Jaguar → Optimization.
    2. Choose Project Table (selected entries) from the Use structures from option menu
    3. Make any other settings, then run the job. Jaguar's default settings, B3LYP/6-31G**, are appropriate for refining the structures of conformers.

      If you have sufficient tokens, you can run the optimizations in parallel (distributed) by choosing as many CPUs as there are conformers to refine. You could also run the optimizations in batches until they are all done.

  5. Run a redundant conformer elimination:

    1. Choose Applications → MacroModel → Redundant Conformer Elimination.
    2. From the Source of energy option menu, choose Jaguar
    3. Make the settings to define comparison atoms.
    4. Make any other settings, then run the job.

    This step ensures that only one copy of any structures that appeared to be different at the MM level but turned out to be the same at the QM level are kept.
