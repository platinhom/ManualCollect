Prime Structure Prediction Toolbar

The Prime Structure Prediction toolbar provides convenient access to commonly used structure prediction tasks. The names of the tasks are given in their tooltips. The toolbar tasks can also be found under the Edit menu. The buttons are described below.

Select
Select a region of the sequence to see the corresponding residues highlighted in the 3D Workspace.
Crop
This is the cropping (or truncation) tool, which can be used to truncate the query sequence at either end. You can move the truncation point by clicking on the desired residue.
Edit SSP
Edit secondary structure predictions. Highlight a region of the SSP, and then type either H for alpha-helix, E for beta-strand, or - for loop.
Edit Query Sequence
Edit the query sequence. Opens a dialog box, in which you can edit the sequence as a text string. This button is only available in the Input Sequence step.
Slide Freely
Use this button to edit the alignment between query and template. Selecting a residue and sliding to the right creates N-terminal gaps; selecting a residue and sliding to the left creates C-terminal gaps.
Slide As Block
Use this button to edit the alignment between query and template. Selecting a residue and sliding to the right creates N-terminal gaps. Selecting a residue and sliding to the left will slide all residues as a block to the left.
Add/Remove Anchors
Set alignment constraints by setting anchors at residue positions. To remove anchors, click again on the same residue.
Lock Gaps
In order to prevent gaps from being collapsed during manual editing, lock the gaps using this button. Locked gaps are indicated with a -.
Unlock Gaps
In order to allow previously locked gaps to collapse during manual editing, unlock the gaps using this button. Unlocked gaps are indicated with a ~.
The following toolbar tasks can also be selected from the Prime Display menu:
Color Scheme
Select a color scheme for coloring sequences in the sequence viewer. If there is a structure in the Workspace associated with the sequence, it is also colored.
View Template Structure
Display the template structures in the Workspace. In the Find Homologs and Edit Alignment step, it is the selected structures that are displayed. In the Build Structure step, all templates carried into this step are displayed.
View SSA
Use this button to show or hide the secondary structure assignment (SSA) of the template in the Sequence Viewer.
View SSA
Use this button to show or hide the secondary structure assignment (SSA) of the template in the Sequence Viewer.
Wrap Sequences
Use this button to wrap the sequences in the Sequence Viewer or to display them as a single scrolling line.
