Project Table — Group Menu

This menu provides actions for selecting and showing entry groups.

Deselect All   Deselect all groups.
Invert Selection   Invert the selection of the groups: the selected groups are deselected and the unselected groups are selected.
Collapse   Collapse all groups (All) or the selected groups (Selected).
Expand   Expand all groups (All), the selected groups (Selected), or expand only the selected groups and collapse all other groups (Only Selected).
Delete All Empty Groups   Delete all empty groups.
