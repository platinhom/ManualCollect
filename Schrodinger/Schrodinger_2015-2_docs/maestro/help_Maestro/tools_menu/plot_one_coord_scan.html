Summary

The plot 1D panel can be used to display data for variation of a parameter with a single variable, such as the energy as a function of a dihedral angle.

Opening the Plot of One-Coordinate Scan Panel

To open the Plot of One-Coordinate Scan panel, you can:

  • Choose Tools → Plot Coordinate Scan → One Coordinate in the main window.

  • Choose Applications → Jaguar → Plot Coordinate Scan → One Coordinate in the main window.

  • Choose Applications → MacroModel → Plot Coordinate Scan → One Coordinate in the main window.

  • Choose Tasks → Coordinate Scan → Plot → One Coordinate in the main window.

  • Click Plot in the Rapid Torsion Scan panel.

Using the Plot of One-Coordinate Scan Panel

After a coordinate scan calculation has run, the resulting .grd file can be displayed graphically using this panel. To open a .grd file, click Open, and navigate to and select the file. The data from the .grd file is be displayed in the plot area.

The appearance of a displayed plot can be changed using the controls on the left side of the panel. The range of the axes can be changed by entering values the Minimum Coordinate, Max Coordinate, Minimum Energy and Maximum Energy boxes. Any number of .grd files can be displayed in the 1D plot window. The attributes of each plot can be altered using the 1D Data panel, which is opened using the Plot Settings button. Plots can also be deleted from this panel.

Toolbar

The toolbar provides tools for manipulation of the plot and for saving an image of the plot. This is a generic toolbar, and some of the actions may not be useful or relevant. The toolbar buttons are described below.

Reset
Reset the plot to the original pan and zoom settings.
Back
Display the previous view of the plot in the view history.
Forward
Display the next view of the plot in the view history.
Pan/zoom
Pan the plot by dragging with the left mouse button, zoom by dragging with the right button.
Zoom to rectangle
Drag out a rectangle on the plot to zoom in to that rectangle.
Configure subplots
Configure the margins and spacing of each plot in the panel.
Save image
Save an image of the plot to file. Opens a file selector in which you can browse to a location, select the image format, and name the image.
Edit axis and curve parameters
Make settings for the title, range, labeling, and scale of the axes; the color, style, and width of lines; and the color, style, and size of markers.
Copy image
Copy an image of the plot to the clipboard, so you can paste it into another application. This button is only available in some panels.
Minimum Coordinate text box

The value entered in this text box determines the lower limit of the coordinate data, displayed on the x axis.

Maximum Coordinate text box

The value entered in this text box sets the upper x axis (coordinate) value.

Minimum Energy text box

The energy value entered in this text box is the lower limit for the y axis.

Maximum Energy text box

The energy value entered in this text box is the upper limit for the y axis of the data plot.

Full Scale button

To restore a plot whose range has been altered using the coordinate and energy text boxes to its full scale, click Full Scale.

Energy Units options

The two options available for energy units are kJ/mol and kcal/mol. Energies are displayed using the selected option.

Energy Scale options

The energy used in a 1D plot can be displayed with either an absolute or relative value. The Relative option is useful for estimating rotational barriers.

Decimal places text boxes

In these text boxes, you can set the number of decimal places used in the values displayed on the x and y axes.

Show legend option

If this option is selected, a legend showing the plot names and the symbols used is displayed in the plot area.

Labels text boxes

In these text boxes, you can set the labels that are displayed on the x and y axes and the title of the plot.

Plot Settings button

Clicking Plot Settings opens the 1D Data panel, from which the appearance of each plot can be altered, and plots can be deleted.

Open button

Open a .grd file for plotting. This button opens a file chooser from which a .grd file can be opened. The data in the file is added to the plot.

Export to CSV button

Click this button to write the data for all displayed datasets to a comma-delimited CSV file that can be read into a spreadsheet or plotting program for further analysis.
