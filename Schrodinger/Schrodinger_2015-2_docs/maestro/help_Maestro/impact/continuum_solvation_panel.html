Summary

The effects of solvent can be implicitly simulated using one of three available continuum-solvent methods: the Surface Generalized Born (SGB) Model, the Analytic Generalized Born (AGB) Model, and the finite-element Poisson Boltzmann Solver (PBF).

Opening the Continuum Solvation Panel

To open the Continuum Solvation panel, you can:

  • Select the Use continuum solvation option in the Potential tab of the QSite, Impact Energy Minimization, Impact Dynamics, or Impact Hybrid Monte Carlo panel, then click the adjacent Settings button.

Continuum Solvation Panel Features

  • Solvation method option menu
  • SGB displacement threshold text box
  • PBF resolution option menu
  • PBF displacement threshold text box
Solvation method

There are three continuum solvation models available within Impact: the Surface Generalized Born (SGB) Model, the finite-element Poisson Boltzmann Solver (PBF), and the Analytic Generalized Born (AGB) Model. If you use SGB solvation, you might want to select the OPLS_2005 force field in the Potential tab; the improved parameterized nonpolar model (input file keyword npsolv) will then be used instead of the default SGB terms. For more information on these models, see the Impact Command Reference Manual.

The SGB model is the only model available with the truncated Newton minimization algorithm. The other two methods are available with the steepest descent and conjugate gradient algorithms. The PBF model is the only model available with the fast multipole method.

SGB displacement threshold

This text box specifies how far (in Å) any atom may move from the coordinates used in the previous SGB calculation before a new SGB calculation must be performed. If no atom has moved this distance, the previously calculated SGB energy and forces are used.

PBF resolution

The Poisson-Boltzmann solver performs a finite-element calculation on a grid. The grid spacing controls both the accuracy of and time required for the PBF calculation. The default is to use a low-resolution grid, which should suffice for most protein work. If needed, greater accuracy can be achieved by choosing the Medium or High options.

PBF displacement threshold

This text box specifies how far (in Å) any atom may move from the coordinates used in the previous PBF calculation before a new PBF calculation must be performed. If no atom has moved this distance, the previously calculated PBF energy and forces are used.
