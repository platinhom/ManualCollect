Main Window — Display Atoms Toolbar

This toolbar contains tools for displaying atoms, undisplaying atoms, and adding atoms to the display. It does not have menu equivalents. Some of the features are available in the Display/Undisplay Atoms panel, but others are only available from this toolbar.

Display Only
This button has a menu from which you can select a category of atoms to display (as defined above), or open the Atom Selection dialog box to select atoms to display.
Also Display
This button has a menu from which you can select a category of atoms to add to the display (as defined above), or open the Atom Selection dialog box to select atoms to add to the display.
Undisplay
This button has a menu from which you can select a category of atoms to undisplay (as defined above), or open the Atom Selection dialog box to select atoms to undisplay.
