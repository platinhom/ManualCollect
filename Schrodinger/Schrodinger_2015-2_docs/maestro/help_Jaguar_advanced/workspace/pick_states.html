Picking Atoms

Maestro has powerful tools for selecting atoms in the Workspace. Many of the toolbar buttons and most of the panels allow you to choose what is selected when you click on an atom. You can select just the atom itself, or the bond, residue, chain, molecule, or entry to which it belongs. This is referred to as choosing the "structural unit".

For example, to label atoms in a residue, choose Residues from the Pick to Label button menu on the Labels toolbar.

Then click on an atom in the residue. All the atoms in the selected residue are labeled.

In the panels, the Pick option menu allows you to choose the structural unit. Depending on the operation to be performed, the Pick option menu contains one or more of the following structural units:

  • Atoms
  • Bonds
  • Residues
  • Chains
  • Molecules
  • Entries

In general, if a specified number of atoms needs to be selected (i.e. to measure an angle) or if atoms need to be selected in a specific order (i.e. to define atom pairs for superposition) then only the Atoms or Bonds structural units are available.

If, however, an arbitrary number of atoms can be specified for input and the order in which they are specified is not important, then all the structural units are available.

You can pick more than one atom at a time by dragging over the atoms, or over the ribbons assocated with the atoms if the atoms are not displayed. Dragging draws a rectangular region, which might not contain all the atoms you want, or might contain atoms that you don't want in the selection. To add atoms to the selection, hold down SHIFT when you click on an atom or drag over a number of atoms. Likewise, to invert the selection of atoms (select the unselected atoms, deselect the selected atoms), hold down CTRL while you click or drag.
