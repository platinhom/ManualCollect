Workspace

The Workspace is the 3D display area in the center of the main window, where molecular structures are displayed.

All 3D display is done in this area. This includes displaying the molecular structure as well as other information, including labels, measurements, ribbons, markers, surfaces.

You can use the full screen to display the Workspace by pressing Ctrl+= (⌘=). All operations that can be done in the Workspace are available in full-screen mode. To exit full-screen mode, press Ctrl+= (⌘=) again.

You can perform various operations on the contents of the Workspace using the Workspace shortcut menus.
