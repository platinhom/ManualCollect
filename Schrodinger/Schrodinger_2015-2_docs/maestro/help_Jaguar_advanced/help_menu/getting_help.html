Getting Help

Maestro has several types of help: automatic, context-sensitive help, tooltips, and a help facility:

  • The Auto-Help text box automatically displays a single line of information about the task you are performing, if this information is available. This text box is located at the foot of the main window.

  • For many GUI elements, such as buttons and options, tooltips (balloon help) are available. To display the tooltip, pause the pointer over the GUI element. The tooltip should appear within a few seconds near the pointer location. If it doesn't appear, check the Appearance tab of the Preferences panel to ensure that Show tooltips is selected.

  • The help facility contains more detailed information. For information about a panel or the tab that is displayed in a panel, click the Help button in the panel,

    or press F1. The relevant help topic is displayed in this panel. A full list of help topics with links to each topic can be displayed by clicking the Home button.

    You can search the help set for words in various combinations using the tool bar tools. Enter a word or phrase in the search text box and press Enter or click the Search button.

    The results are displayed in a page that has links to the topics in which the words were found. You can use double quotes to require an exact match. You can use "and", "or", "not", and parentheses to construct complex search expressions, such as '(ligand not protein) and preparation'.

    To search within a topic, click the Find on page button,

    then enter the words in the search text box and press Enter or click the Search button. Use the arrow keys to the right to step through hits.

    You can go to the previous or next topic or set of search results using the Previous and Next buttons on the left of the toolbar.

  • The Help menu provides access to the help facility and a range of other types of documentation.

    • To work through tutorial exercises, you can use the Tutorials panel to copy the tutorial files and open the tutorial documentation.

    • For tips on using Maestro, choose Help → Tips.

    • For information on keyboard shortucts, choose Help → Keyboard Shortcuts.

If you don't find the information you need in any of these places, try the following sources:

  • The manuals. To open a manual, choose Help → Documentation Index to open the Documentation Index, and follow the links. To search the manuals, choose Help → Search Manuals.

  • The Knowledge Base. The Help menu has a link to the Knowledge Base on the web site.

  • The New Features web page. The Help menu has a link to this page.

  • The Known Issues pages. The Help menu has a link to the Known Issues pages.
