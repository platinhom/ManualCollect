Summary

This panel contains information about the project used and allows you to change the structures, and add properties.

Opening the Advanced Project Options Panel

To open the Advanced Project Options panel, you can:

  • Click Advanced in the Materials Explorer panel.

Advanced Project Options Panel Features

  • Change Structures button
  • Advanced Property Preparation button
Change Structures button

Select the structures that you want to cluster. Opens the Select Structures dialog box, which lists the entries in the project by title. You can select all entries by clicking Select All, clear the selection by clicking Select None, or select the entries that are selected in the Project Table by clicking Use PT Selection. You can also select entries in the list with shift-click and control-click. When you have the desired entries, click Set Structures to confirm the structure selection and dismiss the dialog box.

Advanced Property Preparation button

Prepare properties for use in clustering and mapping. Opens the Property Preparation panel, in which you can calculate, import, and delete properties.
