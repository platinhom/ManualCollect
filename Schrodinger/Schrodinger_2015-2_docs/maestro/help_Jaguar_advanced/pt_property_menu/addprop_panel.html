Summary

The Add Property dialog box allows you to add a property to the table and set a value for the new property for the selected entries.

Opening the Add Property Dialog Box

To open the Add Property dialog box, you can:

  • Choose Property → Add in the Project Table panel.

  • Choose Property → Add in the Project Table panel.

Using the Add Property Dialog Box

To add a new property column to the Project Table:

  1. Enter a property name in the Name text box.

  2. Choose a property type from the Type option menu.

  3. (Optional) If the type is Real, set the number of decimal digits to be displayed for the property in the Precision text box. The default number of decimals is set in the Project - Project table tab of the Preferences panel.

  4. (Optional) Enter an initial value for the property in the Initial value text box.

    If a value is not given, the property column is created, but the property values remain unset. If it is given, the value is set for all selected entries.

  5. Click Add to create the property.
