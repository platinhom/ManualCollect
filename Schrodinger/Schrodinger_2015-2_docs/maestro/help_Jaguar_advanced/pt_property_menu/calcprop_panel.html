Summary

The Calculate Property panel allows you to generate one or more new properties from a fixed list, based on the attributes of the structures.

Opening the Calculate Properties Panel

To open the Calculate Properties panel, you can:

  • Choose Property → Calculate in the Project Table panel.

Using the Calculate Properties Panel

To calculate one or more properties, select them from the list, then click either All Entries or Selected Entries. If you chose properties that require options to be set, the Property Options dialog box opens, and you can make the required settings, then click Calculate to continue to the calculation.

If the property calculation takes some time, a progress dialog box is displayed. You can stop the property calculation; if you do, the properties already calculated are kept.

To recalculate a property, select Recalculate existing values before you click the button to calculate the properties.

Calculate Properties Panel Features

  • Properties list
  • Recalculate existing values option
  • Calculate N properties for buttons

Properties list

The properties you can generate are listed in this section. To calculate one or more properties, select them from the list. The number of properties selected is displayed below the list.

The available properties are:

  • Chiral centers—calculates the number of chiral centers for ligand-sized molecules and adds the chirality properties used by LigPrep (single molecule entries only)
  • Molecular charge (single molecule entries only)
  • Molecular formula (single molecule entries only)
  • Molecular weight (single molecule entries only)
  • Number of atoms
  • Number of heavy atoms
  • Number of molecules
  • Number of residues
  • Spin multiplicity
  • Secondary structure content—generates three properties: percentage of helices, percentage of beta strands, and percentage of random coils
  • AlogP
  • Hydrogen bond acceptors
  • Hydrogen bond donors
  • Number of rotatable bonds
  • Polar surface area
  • Molar refractivity
  • Polarizability
  • Substructure—calculates the number of occurrences in each entry of the substructure that you provide
  • Total surface area—calculates solvent-accessible surface area (SASA) and partitions it into hydrophobic and buried portions. Generates up to four properties.
  • Sequential index
Recalculate existing values

If this option is selected, calculating a property that already exists replaces the property values with new values. If it is not selected and the property exists, the property will be calculated only for the entries that do not already have values.

Calculate N properties for buttons

Click one of these buttons, All Entries or Selected Entries, to calculate the selected properties. The number of properties selected, N, is updated as you select properties.
