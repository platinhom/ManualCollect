Deleting Project Entries

To delete the selected entries, removing all the associated data, choose Entry → Delete in the Project Table panel or press the Delete key.

By default, you are prompted to confirm the deletion. You can choose not to show the prompt by selecting the option in the confirmation dialog box.

This command can be undone using the Undo/Redo button on the Edit toolbar in the main window

or by choosing Edit → Undo alias in the main window.
