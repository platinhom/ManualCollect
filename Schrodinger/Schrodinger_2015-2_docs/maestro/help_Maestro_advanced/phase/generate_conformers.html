Summary

The Generate Conformers dialog box generates conformers for each structure in the ligand list. The conformers are generated using a rapid torsional search, which can be followed by a minimization.

Opening the Generate Conformers Dialog Box

To open the Generate Conformers dialog box, you can:

  • Click Generate Conformers in the Prepare Ligands step of the Develop Common Pharmacophore Hypotheses panel.

Using the Generate Conformers Dialog Box

For the development of a pharmacophore model, it is important to generate a range of conformations for each ligand.

Once the conformers are generated, you have several options for selecting conformers. The energy of the conformers is evaluated in aqueous solution (with a continuum solvation model), and you can eliminate conformers based on their relative energy. If there is a large number of conformers for a given ligand, you can set a maximum number of conformers to retain. The set of generated conformers is sampled to enforce the limit.

Some options have a greater impact than others on the outcome of pharmacophore model development. Options with the greatest impact include the maximum number of conformations, maximum relative energy difference, minimum atomic deviation and number of post-minimization iterations. The default settings (rapid search, distance-dependent dielectric solvation model, and no post-minimization iterations) are likely to be adequate for many purposes.

When you have made all the settings, click Start. The Start dialog box opens, in which you can choose a host to run the job, specify the number of subjobs (which should not be greater than the number of ligands), and specify the number of processors to use. Clicking Start in this dialog box then starts the job.

Generate Conformers Dialog Box Features

For database searching, only a selection of features are available for as-needed conformer generation. Unavailable features are noted below.

  • Current conformers options
  • Limits on numbers of conformers
  • MacroModel search method option menu and related options
  • Amide bonds option menu
  • Preprocess and Postprocess options and text boxes
  • MacroModel options section

Current conformers Options

When you generate conformers, you can discard the current conformer set or you can keep it. If you keep the existing set, the new conformers are appended to the set. The set might therefore contain redundant conformers, but it will contain the original structure.

Limits on Numbers of Conformers

Two text boxes provide limits on the number of conformers generated. The limit actually used is the minimum of the two limits.

Number of conformers per rotatable bond text box

This text box specifies the maximum number of conformers to generate in the conformational search for each rotatable bond in the structure. The number given in the text box is multiplied by the number of rotatable bonds in the molecule to arrive at the maximum number of conformers. The default for this value is 10.

Maximum number of conformers per structure text box

This value limits the number of conformers returned from the generation process. If the number of conformers generated is higher than this value, a sample of all the conformers generated is returned. The default is 100.

MacroModel Search Method Option Menu and Related Options

The MacroModel search method option menu supplies two choices. Each choice has a separate set of controls, which are displayed below the option menu.

ConfGen

In this method, the ligand is divided into a core and a periphery. The peripheral groups have only one rotatable bond between the terminal groups and the rest of the molecule. All the nonperipheral rotatable bonds are assigned to the core. The conformational search generates all core configurations and then varies the peripheral configurations, either one-by-one or in a complete search. During the search, hydrogen-bonding interactions are suppressed, because conformations in which the ligand bonds to the receptor are needed in the model, not just conformations with internal hydrogen bonding.

The extent of the search depends on the sampling method as follows:

  • Rapid: All the core conformations are generated, and the conformations of the peripheral (rotamer) groups are sampled one by one.

  • Thorough: A complete set of conformations is generated for both the core and the peripheral groups.

When you choose this option, the Amide bonds option menu and the Preprocess and Postprocess options and associated controls are displayed below the MacroModel search method option menu.

There are three options for treatment of amide bonds in the search:

  • Vary conformation—This option allows the amide dihedral angle to be varied. It can take any value, not just cis or trans.
  • Retain original conformation—Retain the conformation that was in the input structure.
  • Set to trans conformation—Set the conformation to trans for all amide bonds.

Experience to date suggests that the final pharmacophore model is not usually significantly improved by a thorough search.

The Preprocess and Postprocess options perform other MacroModel tasks, including an optional minimization. Preprocessing is done on the input structure; postprocessing is done on the set of conformers generated by the conformational search. The same set of minimization options, from the MacroModel options section, are used in both cases.

Three options are available for postprocessing:

  • Minimize, then eliminate high-energy and redundant conformers—Minimize the conformers after eliminating high-energy and redundant conformers. This option has a dependent Steps text box, in which you can enter the number of minimization steps. This is the most expensive option.
  • Eliminate high-energy and redundant conformers—Eliminate redundant conformers, evaluate the energy and eliminate any conformers that exceed the energy threshold given in the Macromodel options section.
  • Eliminate redundant conformers—Perform only the redundant conformer elimination step. This is the least expensive option, as it does not require any energy evaluation. It is only available if Preprocess is deselected.
Mixed MCMM/LMOD

This method is a combined Monte-Carlo Multiple Minimum/Low Mode search, and is more accurate than the torsional sampling (ConfGen) method, but as a consequence takes longer. The difference between Rapid and Thorough sampling is in the number of steps taken per rotatable bond, which is much larger for thorough sampling. For more information on this method, see Chapter 10 of the Macromodel User Manual.

When you choose this option, the Minimization steps text box is displayed. Minimization of the conformers generated by a mixed MCMM/LMOD search is recommended. The conformer set includes the input structure.

MacroModel Options Section

In this section you can select the force field and solvation treatment for the minimization steps of the conformational search, and set options to filter out redundant conformers and conformers that are high in energy.

  • Force Field option menu
  • Customize button
  • Use customized version option
  • Solvation treatment options
  • Maximum relative energy difference text box
  • Eliminate redundant conformers using options and text boxes

Force Field option menu

The force fields available are OPLS_2005, MMFFs, and OPLS3. For details on these force fields, see OPLS_2005 Force Field and MMFF & MMFFs Force Fields.

Customize button

Customize the torsions in the OPLS3 force field for a set of molecules by refitting to quantum-mechanical calculations. Opens the Force Field Builder panel. Only available when you choose OPLS3 from the Force field option menu.

Use customized version option

Use your customized version of the OPLS3 force field, rather than the standard version in the distribution. Only available when you choose OPLS3 from the Force field option menu. This option is set by default to the value of the Use custom parameters by default option in the Preferences panel, under Jobs - Force field.

Solvation treatment options

Two continuum solvation treatments for water are provided.

  • Distance-dependent dielectric
  • GB/SA water

The distance-dependent dielectric model is somewhat faster that the GB/SA model, and usually produces similar results. For details, see MacroModel — Potential Tab and Chapter 4 of the MacroModel User Manual.

Solvation treatments are not available for as-needed conformer generation during database searching.

Maximum relative energy difference text box

This value sets an energy threshold relative to the lowest-energy conformer. Conformers that are higher in energy than this threshold are discarded. The energy is evaluated with MacroModel using the selected force field, or with an energy estimate if MacroModel processing is not used.

Eliminate redundant conformers using options and text boxes

Two cutoff criteria are available for eliminating redundant conformers:

  • Maximum atom deviation— All distances between pairs of corresponding heavy atoms must be below this cutoff for two conformers to be considered identical. This cutoff is only applied after the energy difference threshold, and only if the two conformers are within 1 kcal/mol of each other.

  • RMSD—The root-mean-square deviation of all pairs of corresponding heavy atoms must be below this cutoff for two conformers to be considered identical.

In addition to the threshold, a threshold of 60 degrees is used for torsion angle differences for polar hydrogens. This threshold cannot be changed.

The threshold is not used in as-needed conformer generation during database searching.
