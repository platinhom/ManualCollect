Summary

In this dialog box you can choose how to show hypothesis clusters in the Hypotheses table, and select the clustering level for defining the clusters.

Opening the View Clusters Dialog Box

To open the View Clusters dialog box, you can:

  • Click View Clusters in the Score Hypotheses step of the Develop Common Pharmacophore Hypotheses panel.

View Clusters Dialog Box Features

  • Show options
  • Intra-cluster similarity level text box
Show options

Select an option for the hypotheses to show for each cluster:

  • All hypotheses—Show all hypotheses in the cluster.
  • Cluster representatives: highest average similarity—Show the hypothesis that has the highest average similarity to other hypotheses in the cluster. All other hypotheses in the cluster are hidden.
  • Cluster representatives: highest survival score—Show the hypothesis that has the highest survival score in the cluster. All other hypotheses in the cluster are hidden.
Intra-cluster similarity level text box

Enter the similarity value that defines the clustering level. The similarity measure for each hypothesis in a cluster must be at least this value.
