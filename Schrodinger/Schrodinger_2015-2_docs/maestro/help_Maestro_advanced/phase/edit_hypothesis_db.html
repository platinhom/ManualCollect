Summary

This dialog box allows you to add and remove pharmacophore sites from a hypothesis, move sites, and edit the pharmacophore feature definitions for the hypothesis.

Opening the Edit Hypothesis Dialog Box

To open the Edit Hypothesis dialog box, you can:

  • Click Edit in the Manage Pharmacophore Hypotheses panel.

Using the Edit Hypothesis Dialog Box

There are two editing modes for hypothesis editing: ligand-based and freestyle, with tools in the corresponding tabs.

Ligand-Based Editing

In ligand-based editing, you can select sites that are associated with a reference ligand. The locations of these sites are fixed, and you can add them to the hypothesis or remove them from the hypothesis.

To add sites to the hypothesis, select the sites in the Ligand sites list and click Add.

To remove sites from the hypothesis, select the sites in the Hypothesis sites list and click Remove.

You can select multiple sites in either list by shift-clicking and control-clicking.

Freestyle Editing

In freestyle editing, you can place sites wherever you want and move them around, but they have no association with a ligand (even if you start with a ligand). This also means that you cannot have any directionality associated with the site: acceptors, donors, and aromatic rings can be oriented in any direction, as there is no ligand to provide an orientation for these features. You can start with a ligand-based hypothesis, but if you perform any freestyle editing, the ligand is removed from the hypothesis (upon confirmation) when you click OK. A dialog box is displayed, giving you the choice to overwrite the existing entry in the project and remove the reference ligand (Overwrite), save the hypothesis as a new entry without a reference ligand (Save), or cancel and return to editing. The ligand itself remains in the Workspace, however, until you exclude the entry.

To create a new site:

  1. Choose the feature type from the Site type option menu.
  2. Select Click to create or select sites in the Workspace.
  3. Click in the Workspace at the desired location.

The site is placed in the xy plane where you clicked. This is the "screen" plane of the current view.

To move a site:

  1. Select Click to create or select sites in the Workspace.
  2. Click on the desired site.
  3. Drag the site with the right mouse button to the desired location.

The site moves in the current xy plane. If you want to change the plane, rotate the structure using the middle mouse button, or use the Rotate X and Rotate Y buttons on the View toolbar in the main window.

  

To change the site type:

  1. Choose the feature type from the Site type option menu.
  2. Select the site (or sites) in the Hypothesis sites list.
  3. Click Change Type.

To delete sites:

  1. Select the site (or sites) in the Hypothesis sites list.
  2. Click Delete.

Changing Feature Definitions

To modify the feature definitions, click Edit Features, and make the changes in the Edit Features dialog box.

Edit Hypothesis Dialog Box Features

  • Edit Features button
  • Ligand-based tab
  • Freestyle tab

Edit Features Button

The Edit Features button opens the Edit Features dialog box so that you can edit the pharmacophore feature definitions.

Ligand-based Tab

In this tab you can edit ligand-based hypotheses. This tab is unavailable if the hypothesis has no reference ligand.

Mark sites options

Mark pharmacophore sites in the Workspace. All marks all available sites for the reference ligand; Hypothesis only marks the sites in the hypothesis. The sites are displayed as described in the Create Sites topic, and are labeled by the feature letter and index. The index used here is the index of all possible sites in the ligand. When All is selected, the markers for sites that are not used in the hypothesis are displayed in a dimmer color than for the hypothesis sites.

Ligand sites list

Lists all possible pharmacophore sites in the reference ligand based on the pharmacophore feature definitions.

Hypothesis sites list

Lists the sites that define the hypothesis.

Add button

Click to add the sites selected in the Ligand sites list to the hypothesis. Only available when ligand sites have been selected.

Remove button

Click to remove the sites selected in the Hypothesis sites list from the hypothesis. Only available when hypothesis sites have been selected.

Freestyle Tab

In this tab you can edit hypotheses by free addition and adjustment of sites. If the hypothesis has a reference ligand, it is removed from the hypothesis (upon confirmation) when you click OK.

Click to create or select sites in the Workspace option

This option enables you to click in the Workspace to add a site at that point, or to select an existing site to move it, by dragging with the left mouse button. Selected sites are marked in red. New sites are displayed as spheres colored according to feature type.

When you click to add a site, it is placed in the xy plane, which is the plane of the screen in the current view. (Clicking on an atom in a displayed structure does not place the feature on that atom.) Similarly, when you click on a site to move it, the site is moved in the xy plane. To place sites in a different plane, or to drag them out of their current plane, do a rotation (middle-click and drag) and then place or move the site. You can use the Rotate X and Rotate Y buttons on the View toolbar in the main window to switch planes to one of the orthogonal planes.

  

Click to attach site to the picked atom option

This option enables you to pick an atom in the Workspace to add a site at the atom coordinates. The site type is determined by the choice made from the Site type option menu.

Site type option menu

This option menu lists the feature types. The type that is chosen from this option menu becomes the default for creating new sites or for changing the type of the sites.

Change Type button

Changes the feature type of the selected sites to the default type, as chosen from the Site type option menu.

Delete button

Deletes the sites that are selected in the Hypothesis sites table.

Hypothesis sites list

Lists the sites in the hypothesis. You can select multiple sites with shift-click and control-click. The selected sites are marked in red in the Workspace.

The sites are always listed in alphabetical order, and the site index (the number in the site label) is updated each time you add or remove a site.
