Summary

This panel allows you to open panels on the Tasks, Tools, and Scripts menus, designate favorite menu items to show on an abbreviated menu, and search for panels whose menu path or description contain text that you enter.

Opening the Task Tree Panel

To open the Task Tree panel_type, you can:

  • Choose Tasks → Show Tree in the main window.

If the Task Tree panel is already open, it is raised.

Using the Task Tree Panel

  • To expand or collapse a submenu (node) in the tree, click on the + or − button.

    You can also expand and collapse the submenu by right-clicking on the name and choosing Selected from the Expand or Collapse submenu, or expand a submenu and collapse all others by choosing Expand → Only Selected.

  • To expand or collapse all submenus, right-click on any submenu and choose All from the Expand or Collapse submenu, or click the Settings button and choose Expand All or Collapse All.

  • To display a description of a task or panel (leaf) in the lower section of the panel, click on the task or panel.

  • To find a task or panel by searching the text of the menu, the name of the task or panel, or its description, enter text in the search box at the top of the panel. Only the items that contain matching text are shown in the tree. The tree is updated as you type. Click the clear (X) button in the search box to clear the search text and redisplay the full tree.

  • To open a panel, double click the task or panel name, or right-click on it and choose Open.

  • To designate a menu item as a "favorite", right click on it and choose Favorite. You can restrict the tree to show only the favorites by selecting Show Favorites Only on the Settings button menu.

    You can also restrict the Tools and Scripts menus to show only the designated favorites by selecting Show Favorites Only on the menu.

Task Tree Panel Features

  • Filter text box and clear button
  • Settings button
  • Menu tree
  • Description display area
Filter text box and clear button

Filter the items in the tree by the text in the text box. Items are shown in the tree if the text in the text box matches any part of the menu path, the status help message for the item (text that is shown in the status bar in the main window when you pause the pointer on a menu item), and the description text. For example, typing "ligand" in the text box shows items that have the word "ligand" in the description text, such as ConfGen → Standard, as well as items that have the word "ligand" in the menu item, such as Core Hopping → Ligand-based. The search is done as you type.

You can use wildcard characters in the search text: * for zero or more characters of any kind, and ? for any single character.

To clear the text and return to the full tree, click the clear button (X) in the text box.

Settings button

Make settings for what is displayed in the tree.

  • Task View—Show the tree for the Tasks menu.
  • Applications View—Show the tree for the Applications menu.
  • Show Favorites Only—Show only the designated favorites in the tree. The panel title displays "Favorites" to indicate that the panel is restricted to the favorites.
  • Collapse All—Close all nodes in the tree, so that only the top-level menu items are visible.
  • Expand All—Open all nodes in the tree, so that all items are visible.
Menu tree

Displays the contents of the Applications menu or the Tasks menu, as well as the Tools and Scripts menus, as a tree.

  • Right-clicking on a submenu name displays a shortcut menu that allows you to expand or collapse the tree. You can expand all nodes (All), expand the current node (Selected), or expand the current node and collapse all others (Only Selected). You can collapse all nodes or the current node.

  • Right-clicking on a menu item (leaf node) also opens a shortcut menu. In addition to the items for expanding and collapsing the tree, there is an item to designate the menu item as a favorite, and an item to open the panel for that menu item.

  • Clicking on a menu item selects it and displays its description in the display area below.

  • Double-clicking on a menu item opens its panel.

Description display area

When you click on a menu item (rather than a submenu), a description of the task performed by that menu item is displayed in the text box, with the menu item as a heading.
