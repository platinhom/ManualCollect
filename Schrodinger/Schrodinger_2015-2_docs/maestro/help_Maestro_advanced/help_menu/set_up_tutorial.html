Summary

In this panel, you can choose a tutorial, copy the tutorial input files to your choice of working directory, and view the tutorial instructions.

Opening the Tutorials Panel

To open the Tutorials panel, you can:

  • Choose Help → Tutorials in the main window.

Using the Tutorials Panel

This panel is intended to facilitate running the tutorials that are supplied with the software and documented in the manuals. To enable you to choose a particular kind of exercise or tutorial set, the tutorials are classified into various classes that belong to one of several different categories. You can choose the category from the Show tutorials by option menu (e.g. Modeling method), and the option menu below it is populated with the classes that belong to this category. When you make a choice from these option menus, a list of tutorials is displaed in the table. The tutorials listed can include entire Quick Start Guides or tutorial chapters in User Manuals, or individual exercises within one of these tutorials. You can then choose the tutorial that you are interested in from the table.

Once you have chosen the tutorial, you can copy the input files to your working directory. You can specify the working directory in the Copy to text box, or click Browse to navigate to the desired directory. After specifying the directory, click Copy to copy the tutorial input files to this directory.

To view the instructions for the tutorial, click View. If you have Adobe Reader installed, the PDF document opens at the tutorial exercise that you have chosen (which for entire manuals is the beginning of the tutorial exercises). Otherwise, the PDF document opens at the beginning, and you can use the bookmarks or the Contents to find the tutorial exercise.

Tutorials Panel Features

  • Show tutorials by option menu
  • Tutorial category option menu
  • Tutorial table
  • Copy to text box and Browse button
  • Copy button
  • View button
Show tutorials by option menu

Select a tutorial category from this option menu to use as a filter for the tutorials. This category is used as a label for the option menu below, which is populated with the classes that are defined for this category.

Tutorial category option menu

Select a tutorial class within the category defined by the Show tutorials by option menu. The table is updated with the tutorials that are in this class.

Tutorial table

This table lists the tutorials available for the choices made in the option menus above. You can select a single row in this table as the tutorial of interest, for which files can be copied and the tutorial document opened.

Copy to text box and Browse button

Enter the working directory that you want to use for the tutorial, to which the tutorial files are copied. Click Browse to navigate to the desired directory.

Copy button

Copy the tutorial files to the directory specified in the Copy to text box.

View button

Open the PDF file that contains the instructions for the tutorial. With Adobe Reader, the file should be opened at the beginning of the chosen tutorial exercise; if Adobe Reader is not available, the tutorial is opened at the beginning of the document.

Due to a known issue with Adobe Reader, if you open the same document a second time, it does not jump to the new location. To work around this problem, you can close Adobe Reader before clicking View.
