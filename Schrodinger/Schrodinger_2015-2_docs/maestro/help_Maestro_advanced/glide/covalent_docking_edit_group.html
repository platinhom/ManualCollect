Summary

Edit the SMARTS pattern that defines a reactive group, or add a new SMARTS pattern. The dialog box is named New Reactive Group if you are adding a new pattern.

Opening the Edit Reactive Group Dialog Box

To open the Edit Reactive Group dialog box, you can:

  • Click Edit in the Covalent Docking panel.

  • Click New in the Covalent Docking panel.

Using the Edit Reactive Group Dialog Box

To edit or add a SMARTS pattern for a reactive group, it can be useful to have a representative ligand displayed in the Workspace, and then you can select the atoms on the ligand that define the reactive group. When you have made a selection, click Get From Selection, and the SMARTS pattern for the selected atoms is displayed in the SMARTS text box. You can also edit the SMARTS pattern directly.

Next, use the Position option menu to select the atom in the SMARTS pattern that will form a bond to the receptor.

Edit Reactive Group Dialog Box Features

  • Smarts text box
  • Get From Selection button
  • Position option menu
Smarts text box

Lists the SMARTS pattern for the reactive group. This text box can be edited directly. The text is replaced when you click Get From Selection.

Get From Selection button

Create a SMARTS pattern from the atoms that are selected in the Workspace, and display it in the SMARTS text box.

Position option menu

Lists the available atom positions in the SMARTS pattern that can be selected for the atom that forms a bond to the receptor. Choose the desired position to use from this option menu.
