Workspace Menu

The Workspace menu controls what is displayed in the main window. Several of the items on the menu control the appearance of the atoms in the Workspace, the display of extra properties of the Workspace structure such as surfaces and dipole moments, and the selection of objects to display in the Workspace.

  • Clear—Clear the Workspace. This operation removes scratch entries as well as project entries from the Workspace. It does not remove any data from project entries.

  • Create Project Entry—Create a new project entry from the Workspace structure. A dialog box is displayed, in which you can enter a title for the entry, and choose whether to create the entry from all atoms, the selected atoms, or the displayed atoms.

  • Save Image—Save an image of the Workspace to a file in TIFF, JPEG, or PNG format, in the Save Image panel.

  • Scenes—Create and manage "scenes", which are snapshots of part or all of the project, including the Workspace content and display settings. This submenu offers the choices New, for creating a new scene, View, for viewing scenes, and Manage, for managing scenes; and commands for navigation or exiting when viewing scenes. The navigation and exit commands are also available from the Scenes toolbar, which is displayed below the Workspace when you view scenes. See the Scenes topic for details.

  • Full Screen—Enter full screen mode, in which the Workspace occupies the entire screen.

  • Style—Apply a defined visual style to the Workspace contents. This submenu contains commands for applying and reapplying the style, making style settings, creating binding site surfaces, and fixing the receptor in the Workspace. The commands are also available from the Style toolbar.

  • Fonts—Change the Workspace font settings. This submenu has commands for increasing or decreasing the font size by one point and for opening the Preferences panel to make more extensive font settings.

  • Atom Labels—Set up and display atom labels, in the Atom Labels panel.

  • Atom and Bond Coloring—Change the coloring of atoms and bonds, in the Atom and Bond Coloring panel.

  • Molecular Representation—Change the representation of atoms and bonds. Opens the Molecular Representation panel.

  • Ribbons—Change ribbon representations. Opens the Ribbons panel.

  • Display/Undisplay Atoms—Select the atoms to display or undisplay in the Workspace, in the Display/Undisplay Atoms panel.

  • Surface—Generate, import, and display surfaces. Opens a submenu for selection of the task.

  • Single-Entry Feedback—Display or undisplay entry information in the Workspace, subject to the related preference.

  • Add Caption—Add a caption to the Workspace. Opens the New Caption dialog box, in which you can type in the text, and set the caption background, font, and text color.

  • Highlights—Create, modify, delete, or apply highlights to atoms in the Workspace. Opens the Manage Highlights panel.

  • Alternate Positions— For protein structures with partial occupancies, display the atoms for all occupancies. The alternate positions are displayed as ghost atoms.

  • Dipole Moment— Display the dipole moment of the Workspace structure, if one is available. The dipole moment is displayed as an arrow with the base at the geometric centroid of the molecule. If a dipole moment calculated by Jaguar is available, it is displayed. Otherwise, the dipole moment is calculated from the partial charges, if these are available. If you calculated ESP charges with Jaguar but not the dipole moment, these charges replace the MM partial charges and can be used for the dipole moment. The appearance of the dipole moment marker is controlled by preferences.

  • Coordinate Axes— Display a set of coordinate axes in the lower left corner of the Workspace. These axes show the orientation of the Workspace view, and rotate along with the structures when you perform a rotation.

  • Crystal Mates— Display all or part of the copies of the Workspace structure that are related to the Workspace structure by crystallographic symmetry and are within van der Waals contact distance. The display can be limited to residues within a given distance of the Workspace structure. The Crystal Mates submenu offers four distances, in addition to All (display all atoms for each copy of the Workspace structure) and None (undisplay crystal mates).

    The atoms in the crystal mates are included in the Workspace as scratch entries. If you want to make use of them for a calculation, you can create an entry from the Workspace, and it will include the crystal mate atoms.

    The chain names for the crystal mates can be made unique by setting a preference in the Preferences panel.

  • Appearance— Set options for the Workspace appearance: background color, lighting, and materals effects, in the Appearance panel.
