Minimizing A Structure

If you want to perform a quick minimization of a structure or part of a structure in the Workspace, you can:

  • Choose Tools → Minimize → Selected Atoms, All Atoms, or Ligand.

  • Press Ctrl+M (⌘M) to minimize the selected atoms (or all atoms if there is no selection).

  • Press Ctrl+Shift+M (⇧⌘M) to minimize the ligand.

This action performs a simple force-field minimization of the energy with the OPLS_2005 force field on the whole structure, the selected atoms, or the ligand. The minimization stops when the RMSD between iterations is less than 0.1 Å or 1000 iterations have been performed. If you select atoms for minimization, the atoms bonded to the selection are included but constrained (150 kJ/mol force constant).

There are other options for minimizing a structure, listed below.

  • If you want more control over the parameters of a force-field minimization, including use of solvation, you can use MacroModel: choose Applications → MacroModel → Minimization.

  • If you want to minimize a set of ligand-receptor complexes with optional constraints, you can use the MacroModel Embrace Minimization method: choose Applications → MacroModel → Embrace Minimization.

  • To minimize a system for molecular dynamcis that can include solute, solvent, protein, membrane, with periodic boundary conditions, you can use the Desmond Minimization panel, after setting up a model system in the System Builder panel: choose Applications → Desmond → Minimization.

  • To perform a quantum-mechanical minimization, choose Applications → Jaguar → Optimization, Tasks → Minimization → Ab Initio QM, or Tasks → Quantum Mechanics → Optimization.

  • For a QM/MM minimization of a large system such as a protein-ligand complex, choose Applications → QSite or Tasks → Minimization → QM-MM and make MM minimization or QM optimization settings, or both.

  • For a semiempirical minimization, choose Applications → Semiempirical NDDO → Semiempirical NDDO and ensure that Optimize geometry is selected.
