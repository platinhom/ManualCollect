Merging Projects

The Merge feature merges the entries from another project (the incoming project) into the current project.

To merge a project into the current project, choose Project → Merge in the main window. The Merge From Project project selector opens.

In the project selector, navigate to the desired incoming project in the Directories list and select it in the Projects list, or enter the path to the project in the Project text box. If you want any extra files in the incoming project that were not generated by Maestro included in the merged project, select Include user-created files. When you click Merge, Maestro appends the entries of the incoming project to the current project and copies any files, after resolving any group name conflicts (by appending a numerical suffix to the incoming group names). The incoming entries are selected in the Project Table.
