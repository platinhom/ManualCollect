Summary

Clean up the structure of one or more complexes by performing a force-field minimization. The metal is treated generically, with the current positions as the equilibrium geometry.

Opening the Complex Cleanup Panel

To open the Complex Cleanup panel, you can:

  • Choose Applications → Materials Science → Build Organometallic Complexes → Clean Up Complexes in the main window.

  • Choose Tasks → Materials → Build Organometallic Complexes → Clean Up Complexes in the main window.

Using the Complex Cleanup Panel

This panel is useful for cleaning up the geometries of complexes after building, for example in the Build Single Complex panel or the Build Multiple Complexes panel.

The atoms that are coordinated to the metal can be specified in several ways. You can include or exclude hydrogens, keep existing bonds to the metal, and specify a shell of coordinated atoms by distance or by number of closest atoms. Atoms with existing bonds are added to the coordinated set specified by other means if they are not already in the coordinated set.

The metal atom is treated as though the metal-ligand bonds are at their equilibrium distances, with a generic force field whose minimum is at the current distance, and likewise with the bending parameters. The geometry around the metal is therefore not likely to change much, so the minimization will largely affect the ligand geometries.

The minimization is performed with the OPLS_2005 force field, to a convergence of 0.05 kcal mol-1 Å-1. Bonds to the metal from the coordinating atoms are replaced with single bonds or dative bonds (which appear as zero-order bonds in the Workspace).

Structures from the Workspace or the Project Table are replaced by the cleaned-up structures. If you choose to clean up structures from a file, the minimized structures are written to a file named filestem-minimized.maegz.

It is recommended to perform a QM (Jaguar) optimization on the complexes after they have been cleaned up. The cleanup should provide a much better starting geometry for the optimization than the raw structure. Choose Applications → Jaguar → Optimization to set up the geometry optimization and run the job.

Complex Cleanup Panel Features

  • Use structures from option menu
  • File name text box and Browse button
  • Atoms within d Å of the metal atom are considered coordinated option and text box
  • The N closest atoms are considered coordinated option and text box
  • Hydrogen atoms can be considered for coordination option
  • Preserve existing metal connections option
  • Clean Up button
Use structures from option menu

Choose the source of the structures for the current task.

  • Project Table (selected entries)—Use the entries that are currently selected in the Project Table.
  • Workspace (included entries)—Use the entries that are currently included in the Workspace, treated as separate structures. The Workspace should not contain a scratch entry.
  • File—Use the structures from the specified file. When this option is selected, the File name text box and Browse button are activated. Click Browse and navigate to the file you want to use. The file name is displayed in the text box when you click Open in the file selector. You can also enter the file name in the text box.
File name text box and Browse button

If you chose File from the Use structures from option menu, enter the file name in this text box, or click Browse and navigate to the structure file. The name of the file you selected is displayed in the text box.

Atoms within d Å of the metal atom are considered coordinated option and text box

Use this option to specify ligand atoms that are coordinated to the metal by their distance from the metal. Specify the distance cutoff for detecting atoms that are coordinated to the metal in the text box. If Preserve existing metal connections is selected, there may be coordinated atoms that lie outside this distance, as the union of the two sets (those inside the distance and those with existing bonds) is taken.

The N closest atoms are considered coordinated option and text box

Use this option to specify ligand atoms that are coordinated to the metal by order of their proximity to the metal. Specify the number of atoms to be coordinated in the text box. If Preserve existing metal connections is selected, the final number of coordinated atoms may be greater than the number you specify in the text box, as the union of the two sets (the closest N and those with existing bonds) is taken.

Hydrogen atoms can be considered for coordination option

Consider hydrogen atoms to be bonded to the metal if they are within bonding distance of the metal.

Preserve existing metal connections option

Keep existing bonds to the metal, regardless of whether they satisfy the criteria for coordination. These atoms are considered coordination in addition to the ones specified by number or distance. The final number of coordinated atoms depends on how many atoms with existing bonds to the metal also satisfy the other criteria. The bond type of the existing bonds might change from single to dative to satisfy the valence requirements on the coordinating atom.

Clean Up button

Clean up the complexes. The cleanup job is run locally. A progress bar is displayed to the right if the cleanup takes longer than a few seconds.
