Summary

The Monitor tab of the Dynamics and MC/SD panels can be used to designate geometrical parameters to be output to the .mmo file when a job is run.

Using the Monitor Tab

During a molecular dynamics simulation, it is possible to monitor a number of geometrical parameters. These are angles, inter-atomic distances, dihedral angles, the surface areas of individual atoms, and the population of hydrogen-bonds. The Monitor tab has a series of buttons that allow access to the individual monitor panels. Each of these panels has its own help topic. See Related Topics below. For all the dynamics monitors, the results of the monitoring appear in the .mmo file.

Monitor Tab Features

  • Number of Structures to Sample text box
  • Minimize sampled structures option
  • Surface Areas button
  • Distances button
  • Angles button
  • Dihedrals button
  • H-Bonds button
  • Reset All button
Number of Structures to Sample text box

It is possible to sample structures during a simulation. To do this, simply enter the total number of structures you wish to sample in this text box. This number of structures is selected at regular intervals throughout the simulation and written to the output (.out) file.

Minimize sampled structures option

Select this option to minimize the sampled structures. When you do so, the buttons for monitoring coordinates are disabled.

Surface Areas button

This button opens the Monitored Atom Surface Areas panel, which can be used to select atoms, either by picking or using an ASL string, whose surface area will be monitored during a dynamics simulation.

Distances button

This button opens the Monitored Distances panel, from which you can select inter-atom distances for monitoring.

Angles button

This button opens the Monitored Angles panel, in which you can select angles for monitoring.

Dihedrals button

This button opens the Monitored Torsions panel, which can be used to define dihedral angles (torsions) that will be monitored in a dynamics simulation.

H-Bonds button

This button opens the Monitored H-Bonds panel, which allows H-bonds to be selected for monitoring.

Reset All

This button removes all entries from all dynamics monitors.
