Summary

Analyze the interactions at the interface of two proteins, defined as two sets of chains. The analysis locates residues in the second set of chains that are within a given distance of residues in the first set of chains, and presents counts of hydrogen bonds, salt bridges, disulfide bonds, pi-pi stacking interactions, and van der Waals clashes, and reports the van der Waals surface complementarity and buried solvent-accessible surface area.

Opening the Protein Interaction Analysis Panel

To open the Protein Interaction Analysis panel, you can:

  • Choose Applications → BioLuminate → Protein Interaction Analysis in the main window.

  • Choose Tasks → Biologics → Protein Interaction Analysis in the main window.

Using the Protein Interaction Analysis Panel

The protein that you analyze must be properly prepared, for example in the Protein Preparation Wizard panel.

The first step is to define the two groups of chains between which the interactions will be analyzed. Initially all the chains are in the Unassigned chains list.

  1. Select chains for the first group in the Unassigned chains list.

    You can select multiple chains with the usual shift-click and control-click actions.

  2. Click the right arrow button (→) for Group 1.

    The chains are listed in the Group 1 list and removed from the Unassigned chains list.

  3. Repeat the above steps for the second group.

If you want to move chains between the groups:

  1. Select the chains in the Group 1 list or the Group 2 list.

  2. Click the left arrow button (←) for the group to unassign them.

  3. Click the right arrow button (→) for the other group to reassign them.

Once you have defined the two groups, click Determine Protein Interactions to perform the analysis. This may take a minute or two. When the analysis is done, the results are displayed in the table.

Define interacting groups section

Define the two sets of protein chains between which you want to analyze the interactions. These are called "Group 1" and "Group 2". There must be at least one chain in each group before the analysis can be performed. Chains that remain unassigned are not included in the analysis.

Unassigned chains list

This list contains the chains that have not been assigned to a group. You can select multiple chains in this list to move to the Group 1 list or the Group 2 list with the arrow buttons.

Group 1 and Group 2 lists

These lists define the two parts of the protein between which the interactions are analyzed. You can select multiple chains in either of the lists and remove them from the list with the arrow keys. The removed chains are returned to the Unassigned chains list.

Arrow buttons

These buttons transfer chains between the Unassigned chains list and the Group 1 and Group 2 lists.

Determine Protein Interactions button

Click this button to perform the analysis. This may take a minute or two. When the analysis is done, the results are displayed in the table.

Advanced Options button

Click this button if you want to change the settings that are used to analyze the protein-protein surface interactions. These settings affect the values that are listed in the table after the analysis is done. Opens the Advanced Settings dialog box.

Interactions table

This table lists the results of the analysis. The table columns are described below. You can click on a column heading to sort the table by the values in that column (except for the Specific Interactions column). When you select table rows, the residues for those rows are selected in the Workspace, and the view zooms in to those residues.

Residue This column lists residues in either group that have contact-type interactions with residues in the other group.
Closest This column lists residues from the other group that are within a specified distance of the residue listed in the Residue column. The default distance is 4.0 Å. This value can be changed in the Advanced Settings dialog box, prior to the analysis.
Specific Interactions Text list of specific interactions between the residue listed in the Residue column and residues in the other group. The list covers hydrogen bonds, salt bridges, pi-pi interactions, disulfides, and van der Waals clashes.
# HB Number of hydrogen bonds between the residue listed in the Residue column and residues in the other group. The criteria for detecting hydrogen bonds can be changed in the Advanced Settings dialog box, prior to the analysis.
# Salt Bridges Number of salt bridges between the residue listed in the Residue column and residues in the other group. The criteria for detecting salt bridges can be changed in the Advanced Settings dialog box, prior to the analysis.
# Pi Stacking Number of pi-pi stacking interactions between the residue listed in the Residue column and residues in the other group. The criteria for detecting pi-pi stacking interactions can be changed in the Advanced Settings dialog box, prior to the analysis.
# Disulfides Number of disulfide bonds between the residue listed in the Residue column and residues in the other group.
# vdW Clash Number of van der Waals clashes between the residue listed in the Residue column and residues in the other group. A clash is defined as an overlap of the van der Waals radii of two atoms by more than a specified cutoff. The cutoff can be changed in the Advanced Settings dialog box, prior to the analysis.
vdW Complementarity Van der Waals shape complementarity between the residue listed in the Residue column and residues in the other group, as defined by Lawrence, M.C. and Colman, P. M., J. Mol. Biol. 1993, 234, 946–950.
Buried SASA Fraction of the solvent-accessible surface area of the residue listed in the Residue column that is buried by the interaction with residues in the other group.
Export Table button

Export the contents of the interactions table to a CSV file. The file is comma-separated with a heading row. Table cells that have multiple rows are exported as double-quoted text with line breaks embedded.

Color residue by interaction properties section

Color residues in the Workspace by their interaction properties, using a color ramp between the minimum and maximum values.

Color by option menu

Choose the type of interaction to color residues by. The choices are:

  • Total number of specific interactions—Total number of interactions described in the Specific Interactions column (sum of the values in the # HB, # Salt Bridges, # Pi Stacking, # Disulfides, and # vdW Clash columns).
  • vdW Complementarity—van der Waals shape complementarity, as listed in the table.
  • SASA—Buried solvent-accessible surface area, as listed in the table.

When you choose an option from this menu, the text boxes for the minimum and maximum values are updated to reflect the possible range of values.

Minimum color and Maximum color option menus

Choose the colors to represent the chosen minimum value and maximum value of the interaction. The minimum and maximum values for the color display are set with the value options, described below. The color for the minimum is applied to any value below the chosen minimum for display; likewise the color for the maximum is applied to any value above the chosen maximum for display. The colors between the two limits are obtained by a linear interpolation of the RGB color values.

Minimum value and Maximum value options

Set the minimum and maximum value of the interaction property that are used to color the residues. There are two options for each value:

  • Minimum/Maximum in table—Set the values to use for coloring to the minimum value and maximum value found in the table. The color scheme so defined is then relative to the range observed, rather than fixed.
  • Given value—Set the values to use for coloring to the values given in the boxes. The default minimum value is zero; the default maximum is the observed maximum for the total number of specific interactions, 1.0 for the vdW complementarity, and 100% for the buried SASA.
Only color selected rows option

Apply the coloring only to the residues defined by the selected rows in the interactions table.

Color Residues button

Apply the coloring for the chosen property to the residues in the Workspace that are listed in the interactions table. If Only color selected rows is selected, only the residues that are selected in the table are colored, otherwise all residues in the table are colored.

Reset button

Reset all the panel settings to their defaults, and clear all data.
