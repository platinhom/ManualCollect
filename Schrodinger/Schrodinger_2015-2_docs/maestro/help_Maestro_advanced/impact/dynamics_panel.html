Summary

The Impact Dynamics panel is used to set up and run molecular dynamics simulations on a structure in the Workspace or on an entry in the Project Table.

Note: Only one entry from the Maestro Project Table can be displayed and used for each dynamics job. If you need to merge two structures or entries, use the Project Table's Merge command to create a combination entry, and use that as the basis for your simulation.

Opening the Impact Dynamics Panel

To open the Impact Dynamics panel, you can:

  • Choose Applications → Impact → Dynamics in the main window.

Using the Impact Dynamics Panel

Impact molecular dynamics simulations examine stable, ground state molecules by applying Newton's equations of motion. The constant volume and temperature (NVT) ensemble is the default for MD simulations. The constant volume and energy (NVE), and constant pressure and temperature (NPT) ensembles are also supported. NPT simulations require the use of periodic boundary conditions. Such calculations often, but not always, also use explicit solvent. See the help discussion for the Impact Soak panel to learn how to add explicit waters to your solute system.

Trajectories from an Impact molecular dynamics simulation can be viewed in Maestro. When the job is incorporated into the project, the entry for the final structure has a T button in the Title column. Clicking this button opens the Trajectory panel, which you can use to control the display of the trajectory.

Impact Dynamics Panel Features

Along with features common to Impact application panels (see the Impact Panels Overview topic for more information), the panel includes the following tabs, which are described in separate topics:

  • Potential
  • Constraints
  • MD Parameters
  • Dynamics
  • Replica Exchange
