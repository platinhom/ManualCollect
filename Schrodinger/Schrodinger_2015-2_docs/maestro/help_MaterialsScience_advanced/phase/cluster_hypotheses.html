Summary

In this panel, you choose a linkage method and submit a job to cluster the hypotheses in the Hypotheses table. Hierarchical agglomerative clustering is used based on site and vector scores for pairs of hypotheses to determine similarity. Click Start to start the job.

Opening the Cluster Hypotheses Panel

To open the Cluster Hypotheses panel, you can:

  • Click Cluster in the Score Hypotheses step of the Develop Common Pharmacophore Hypotheses panel.

Cluster Hypotheses Panel Features

There are three linkage method options, which select the method used to compute distances between clusters:

Complete

The distance between clusters is the largest distance between any pair of objects (one object from each cluster). This option produces compact, spherical clusters.

Average

The distance between clusters is the average distance between all pairs of objects in the two clusters.

Single

The distance between clusters is the smallest distance between any pair of objects (one object from each cluster). This option produces diffuse, elongated clusters.
