Summary

In this panel you can add, delete, and export scenes, which are subsets of the project with particular settings.

Opening the Manage Scenes Panel

To open the Manage Scenes panel, you can:

  • Choose File → Advanced File Options → Save/Restore/Export Full Workspace (Scenes) → Manage in the main window.

Manage Scenes Panel Features

  • Scenes table
  • New button
  • Delete button
  • Export Presentation button
  • Warn before closing modified scenes option
Scenes table

This table lists all the scenes available in the project. The following operations are supported in the table:

  • Multiple selection of scenes with shift-click and control-click
  • Reordering rows by selecting and dragging
  • Table scrolling with Page Up, Page Down, Home, and End
  • Deletion of selected scenes with Delete

The table has a shortcut menu with the following items:

  • Delete—Delete the selected scenes, after confirmation.

  • Replace—If the scene you click on is open (selected in the View column), replace it with the current data, including a thumbnail of the current Workspace.

  • Export as Project—If the scene you click on is open (selected in the View column), create a new project with this scene. Opens a project selector so you can navigate to a location and name the project.

The table columns are described below.

Row Table row index. This index is always sequential: it is not an index of specific scenes.
View Check box for scene that is currently displayed. Clicking an unchecked box opens the scene; clicking a checked box closes the scene. Only one scene can be open at a time, so opening a new scene closes the previous scene (if any).
Scene Thumbnail image of the Workspace when the scene was created. The thumbnail is not necessarily related to the contents of the scene: it depends entirely on what is in the Workspace when you create the scene.
Description Text description that you provide for the scene. The description can be edited by typing in the table cell.
New button

Create a new scene. Opens the New Scene dialog box, in which you can provide a description of the scene, and choose the entries that are included in the scene, from Included, Selected, and All. The entries used are taken from the current Project Table, whether it is the Project Table for a scene or for the project itself.

When you click OK, the scene is added to the Scenes table with a thumbnail of the Workspace. The scene is selected, and other scenes are deselected.

Delete button

Delete the selected scenes after confirmation.

Export Presentation button

Export the selected rows of the Scenes table as a presentation, which is a project that contains nothing but the exported scenes. Opens a project selector, in which you can navigate to a location and name the project. You can store the project as a regular project (.prj) or as a zipped project (.prjzip). The zipped format allows you to easily copy it to another location, and ensures that it is not changed by accident. The default is to save it as a zipped project.

Warn before closing modified scenes option

Display a warning if you close a scene that has been modified.
