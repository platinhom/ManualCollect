Zooming

To zoom in or out on the Workspace structures, do one of the following:

  • Hold down the middle and right (2nd and 3rd) mouse buttons and move the mouse to the right (zoom in) or left (zoom out).
  • Hold down the SHIFT and CTRL keys and the middle mouse button and move the mouse to the right (zoom in) or left (zoom out).
  • Rotate the mouse wheel.
  • Choose View → Zoom In or View → Zoom Out.
  • Press the K or J (zoom out) key.
  • On a trackpad, use the pinch gesture.
If you prefer to use the left mouse button (or if you only have one mouse button), choose View → Transform Mode → Zoom before zooming. Choose it again or press Esc to revert to the previous action for the left mouse button.

The Fit button on the Manager toolbar can also be used to zoom. Clicking on this button fits all atoms in the Workspace into the confines of the drawing area, or the selected atoms if there is a selection.

Fitting to the Workspace can also be done from the View menu. The View → Fit to Workspace submenu has three items: Selected Atoms, All Atoms, and Ligand, which fit these objects to the Workspace. Choosing All Atoms is useful if you want to fit the full atom set to the Workspace, but don't want to clear your atom selection. By default, there are keyboard shortcuts for Selected Atoms (Z) and Ligand (L).

When a project is closed, all viewing transformations (rotation, translation, and zoom) are saved with the project and restored when it is opened again.

The Preferences panel has a number of preferences that affect zooming and fitting to the Workspace, under Workspace - Fitting. You can customize the mouse actions for zoom in the Customize Mouse Actions panel, which you open with Edit → Settings → Mouse Actions.
