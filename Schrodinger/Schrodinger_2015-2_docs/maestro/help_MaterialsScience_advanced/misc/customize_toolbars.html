Summary

This dialog box allows you to create and delete custom toolbars and select the icons that are displayed on any of the toolbars for the panel you opened it from. You cannot add toolbar items with their actions, only select from existing items.

Opening the Customize Toolbars Dialog Box

To open the Customize Toolbars dialog box, you can:

  • Choose Edit → Settings → Toolbars → Select Buttons for Each Toolbar in the main window.

  • Choose Table → Customize → Toolbars in the Project Table panel.

  • Right-click on the toolbar and choose Customize from the shortcut menu.

Using the Customize Toolbars Dialog Box

This dialog box allows you to modify, add, or delete toolbars in the panel from which the dialog box was opened. Currently, these panels include only the Project Table panel and the main window. All of the buttons that are available in the panel can be placed on any of the toolbars, and you can have a given button on more than one toolbar. You can choose the style of the toolbar buttons, change the name of the toolbar, and add separators to the toolbar.

When you click OK, the toolbar customizations are written to a file in your Schrödinger user profile. This file is read when Maestro starts.

To change the buttons that are on a toolbar:

  1. Choose the toolbar from the Toolbar option menu.

  2. Select the buttons that you want to add to the toolbar from the Available buttons list, and click the right arrow button to add them to the toolbar.

  3. Select the buttons you want to remove from the toolbar in the Toolbar items list, and click the left arrow toolbar to remove them from the toolbar.

To create a new toolbar:

  1. Click New.

  2. Enter the name in the Name text box.

  3. Select the buttons that you want to add to the toolbar from the Available buttons list, and click the right arrow button to add them to the toolbar.

Customize Toolbars Dialog Box Features

  • Toolbar option menu
  • New button
  • Delete button
  • Name text box
  • Style option menu
  • Available buttons list
  • Arrow buttons
  • Toolbar items list
  • Add Separator button
  • Reset All button
Toolbar option menu

Lists the toolbars that are available in the panel. When you choose a toolbar, the Name text box, Style option menu, Available buttons list, and Toolbar items list are updated to reflect the current state of the toolbar that you chose.

New button

Create a new custom toolbar. When you click this button, a default name is displayed in the Name text box and on the Toolbar option menu, the Available buttons list is updated to show all buttons, and the Toolbar items list is cleared.

Delete button

Delete the toolbar that was chosen from the Toolbar option menu, on confirmation. You can delete the standard toolbars as well as custom toolbars.

Name text box

Displays the name of the toolbar. You can edit the name.

Style option menu

Lists the available style for the buttons on the toolbar. The style that you choose from this option menu is applied to the icons on the current toolbar.

Available buttons list

Lists the available buttons in a tree view, categorized by function. These categories also correspond to the default toolbars. You can select multiple items in the list with shift-click and control-click, and then click the right arrow button to transfer them to the Toolbar items list. This list is updated each time you select a toolbar.

Arrow buttons

Click the right arrow button to move the selected items from the Available buttons list to the Toolbar items list. Click the left arrow button to move the selected items from the Toolbar items list to the Available buttons list.

Toolbar items list

Lists the items on the current toolbar (buttons and separators). You can select multiple buttons in the list with shift-click and control-click, and then click the right arrow button to transfer them to the Available buttons list. You can also drag the selected items to a new location in the list.

Add Separator button

Add a separator to the toolbar. A line is placed in the toolbar at the point where you place the separator.

Reset All button

Delete all customizations and restore the default toolbars.
