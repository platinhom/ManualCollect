Summary

In this panel you can identify reactive sequences in the Workspace structure, by matching them to extended PROSITE patterns that define the reactive sites.

Opening the Reactive Protein Residues Panel

To open the Reactive Protein Residues panel, you can:

  • Choose Applications → More Applications → BioLuminate → Reactive Residues in the main window.

Using the Reactive Protein Residues Panel

The main workflow in this panel is:

  1. Display the protein you want to analyze in the Workspace
  2. Select options for the reactive residues to identify
  3. (Optional) Define your own reactive residues
  4. Click Analyze Workspace to run the analysis
  5. Examine the results in the table or the Workspace, or both.

Reactive Protein Residues Panel Features

  • Sequences to identify options
  • Analyze Workspace button
  • Reactive residue table
  • Color legend
  • Show option menu
  • Show only residues with solvent exposure >= N text box
  • Show only residues with B-factor >= X text box
  • Fit on selection option
  • Reset button
Sequences to identify options

Select an option for the sequences to identify as reactive in the Workspace structure. You can select either option or both options.

Predefined option

Identify sequences for four predefined reactive groups, which are listed.

User-defined option and menu

Identify sequences for reactive groups that you define. The menu lists all the reactive groups that you have defined, and you can select specific groups from this list. The selected groups are listed in the text box. To define these groups, click Edit.

Edit button

Add, edit or delete patterns that define reactive sites. Opens the Edit Patterns dialog box, in which you can change the definition of the patterns that define a reactive protein site. The patterns are specified in extended PROSITE syntax, a description of which is given in the panel. A pattern is defined by a name, a definition, and an optional hotspot. The hotspot is the residue in the pattern that is to be selected, instead of the entire pattern, and is defined by the index of the residue in the pattern (starting from 1 for the first residue).

Analyze Workspace button

Click this button to analyze the Workspace structure for its reactive groups. The results are presented in the table below, and the reactive sites are marked with spheres in the Workspace that are colored according to the type of reactive site.

Reactive residue table

This table lists the results of the analysis. You can sort the table by clicking in the heading of the column you want to sort by.

Type Type of reactive site
Residues Reactive residue identity (chain, residue number and insertion code, 3-letter residue name)
SASA Solvent-accessible surface area of the residue
Exposure Solvent exposure of the residue (percentage of the surface that is solvent-exposed)
B Factor PDB temperature factor for the residue, if available. The value given is the average over all atoms in the residue.
Color legend

The reactive groups are listed here, colored with the colors used in the Workspace to highlight the reactive residues.

Show option menu

Show sites of a particular type in the table and the Workspace. The menu lists the types of sites that were found and are listed in the table.

Show only residues with solvent exposure >= N text box

Show in the table and the Workspace only the sites that have a solvent exposure that is greater than the specified percentage.

Show only residues with B-factor >= X text box

Show in the table and the Workspace only the sites that have a PDB temperature factor (B-factor) that is greater than the specified value.

Fit on selection option

When you select a reactive residue in the table, zoom in on that residue in the Workspace.

Reset button

Clear the panel of all settings and data.
