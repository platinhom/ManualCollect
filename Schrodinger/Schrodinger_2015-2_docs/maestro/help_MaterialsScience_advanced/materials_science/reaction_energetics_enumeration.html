Summary

Calculate reaction barriers or reaction enthalpies for a series of related compounds, based on a prototype unimolecular rearrangement reaction. The structures for the reactions are enumerated by addition of functional groups to the structures for the prototype reaction.

Opening the Reaction Energetics Enumeration Panel

To open the Reaction Energetics Enumeration panel, you can:

  • Choose Tasks → Reaction Energetics Enumeration in the main window.

  • Choose Applications → Materials Science → Reaction Energetics Enumeration in the main window.

Using the Reaction Energetics Enumeration Panel

This panel allows you to evaluate either the reaction barrier or the reaction enthlapy for a series of related unimolecular reactions. The reactions must be based on a prototype reaction, in which substitutions are made on the structures in the prototype reaction. For useful results, the reaction pathway and products should not depend very much on the substituents: the transition states and the products must have similar geometries to those of the prototype structures for the common atoms.

Prior to using this panel, you should optimize the geometry of the prototype reactant molecule, and do a transition state search or optimize the prototype product geometry. All three structures must be unimolecular: reactant, transition state, and product. The geometries of these structures are held fixed during the remainder of the calculations. When the structures are enumerated, the geometry of the additional functional groups is optimized in each structure before the energies are calculated.

The atom numbering in the prototype reactant and the prototype transition state or product must be the same. This is necessary because the attachment points for the substituents must be on the corresponding atoms in these structures. For a transition state, the atom numbering must be the same as that of the reactant used in the search. For a product that is not used in a transition state search, the numbering is not required to be the same, so you must make sure that it is. One way to do this is to build the product from the reactant in the Workspace with the build tools, breaking and forming bonds without deleting any atoms. To ensure that no automatic deletion of hydrogens is done, deselect Adjust number of hydrogens following additive build operations under Builder – Behavior in the Preferences panel.

  1. Choose the type of calculation.

    The choices are Reaction barrier or Reaction enthalpy. The choice determines whether you need a transition state structure or a product structure in the second enumeration.

  2. Enumerate the reactant structures.

    When you click Enumerate Reactant, the Interactive Enumeration panel opens. In this panel you should import the reactant structure that you optimized, specify the bonds at which you want to substitute other functional groups, and define the set of functional groups to substitute. The structures are enumerated combinatorially when you click Enumerate: all possible substitutions are made, to create a combinatorial library. This set of structures is used for the reactants.

  3. Enumerate the transition state structures or the product structures.

    First choose the source of the structure from the Structure option menu. If you choose File, click Import to import the structure, which is then added to the project. When you click Enumerate Transition State or Enumerate Product, the same sets of functional groups that you chose for the reactant are added to the same bonds in the transition state or the product, to create a combinatorial library.

  4. (Optional) Pick atoms to freeze during geometry optimization.

    Select Pick in Workspace to select atoms to freeze during optimization. The prototype reactant structure is placed in the Workspace, and you can pick the atoms to freeze. These atoms are marked in the Workspace and listed in the Frozen atoms text box. If you want to clear the frozen atom list, click the X button in this text box. The atoms are frozen in all reactant and product structures. This feature is only available for reaction enthalpies.

  5. Run the job.

    When the reactant and transition state or product structures have been enumerated, you can start the calculation. Choose a calculation quality, and make any keyword settings that you want to add. Before you click Run to run the job, click the Settings button (gear icon) to make settings for the job. You can distribute the job over multiple processors to reduce the turnaround time, as well as selecting a host on which to run the job. For each structure, a partial geometry optimization is run, so the calculations can take some time.

When the job finishes, the reactant structures and either the transition state structures or the product structures are added to the Project Table, if you chose the output option to append new entries in the job settings dialog box. The reaction enthalpy or the reaction barrier in kcal/mol is added to the reactant structures as a property.

Reaction Energetics Enumeration Panel Features

  • Calculation options
  • Reactant enumeration section
  • Transition state enumeration section
  • Product enumeration section
  • Pick in Workspace to select atoms to freeze during optimization option
  • Frozen atoms text box
  • Calculation quality options
  • Keywords text box
Calculation options

Choose the type of calculation you want to do.

  • Reaction barrier—Calculate the barrier to the reaction. This option requires a transition state structure for the prototype reaction.
  • Reaction enthalpy—Calculate the enthalpy for the reaction. This option requires a product structure for the prototype reaction.
Reactant enumeration section

Using the prototype reactant structure, generate a set of structures with functional group substitutions at one or more sites on the reactant structure. The selection of the prototype reactant structure, the definition of the substitution points and substituents, and the enumeration of the structures, is done in the Interactive Enumeration panel, which you open by clicking Enumerate Reactant. The enumerated structures are added to the Project Table.

Transition state enumeration section
Product enumeration section

This section has the same tools for transition state enumeration and product enumeration. The label depends on whether you chose to calculate reaction barriers or reaction enthalpies.

Structure option menu

Choose the source of the transition state structure or the product structure. The structure can be the one that is displayed in the Workspace, or selected in the Project Table, or in a file.

Import button
Browse button

Click this button to identify the transition state and display the 2D structure in the text area. If you choose to read the structure from a file, the button is labeled Browse, and opens a file selector so you can locate the file.

Enumerate Transition State button
Enumerate Product button

Enumerate the structures for the transition state or the product, using the substituents defined in the reactant enumeration. The reactant enumeration must finish before you can run this enumeration.

Pick in Workspace to select atoms to freeze during optimization option

Select this option and pick atoms in the prototype reactant structure that you want to freeze during the geometry optimization. The prototype reactant is displayed in the Workspace when you select this option. As you pick atoms, they are marked in the Workspace and listed in the Frozen atoms text box. The atoms are frozen in both the reactant and the product structure. This option is only available if you select Reaction enthalpy for the calculation type.

Frozen atoms text box

This text box lists the atoms that were picked for freezing. You can clear the list of frozen atoms by clicking the X button in this text box.

Calculation quality options

Select an option for the quality of the results. The three options represent different levels of theory and basis set, which are shown in the Keywords text box.

Keywords text box

Specify keywords to be added to the gen section of the Jaguar input file. The default keywords are determined by the Calculation quality option that you selected. For information on the keywords, see the Jaguar User Manual.
