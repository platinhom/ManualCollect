Tips for Using Maestro

To center any atom in the Workspace, just right-click on that atom. For PrimeX users, right-clicking also centers the electron density on that atom.

The yellow highlighted region at the bottom of the main Maestro window has helpful instructions, such as the action to be applied when picking atoms, or a description of menu items. This region ("Autohelp") is updated as various windows are opened or toolbar buttons are used to select an action.

Molecular rendering can be controlled from the Molecular representation – Atoms and bonds section of the Preferences panel. Here, you can modify molecular rendering options such as line width thickness, resolution, etc.

Use the Molecule Number or Molecule Number (Carbons) color scheme to rapidly identify a noncovalently bound ligand in a protein as well as any chain breaks that might be present.

Holding down the right mouse button over an atom in the Workspace provides quick access to many useful operations, such as changing the molecular representation, coloring atoms, and even mutating amino acids. See Main Window Shortcut Menus for more information.

Surfaces can be clipped separately from atoms in the Workspace. First, display the clipping planes window by choosing Window → Clipping Planes. Next, right-click in the clipping planes window and choose Surfaces from the shortcut menu. Drag the orange lines to adjust clipping of the surfaces. See Clipping Planes Window for more information.

There are many keyboard shortcuts in Maestro. For example, Ctrl+G (⌘G) switches between local and global transformations for a molecule, and Ctrl+L (⌘L) tiles the Workspace . For a full list of shortcuts, see Using the Keyboard in Maestro.

Function keys can be customized to your needs.

  • Choose Maestro → Function Key Macros to open the Function Key Macros panel. F2 through F12 can now be programmed to perform a set of desired operations. F1 is reserved for opening the help.

    The Protein Preparation Wizard provides an interactive way to set up your protein system. Many operations can be automatically performed, such as assigning bond orders, generating a set of reasonable ionization and tautomeric states for ligands, predicting rotameric states for Asn, Gln, and His, optimizing hydroxyl and thiol torsions, and performing a constrained minimization on the system. Use the Prep Wiz toolbar button to open it.

    Jobs can be monitored and controlled through our Job Control facility, which is available in Maestro by choosing Applications → Monitor Jobs or Tasks → Monitor Jobs, or from the command line by typing $SCHRODINGER/jobcontrol.

    Saving a project allows you to keep a running "lab book" of your work. Once a project is saved the first time, the work you do in that project is automatically saved. It can also be shared with your colleagues. The same Maestro project can be opened on all platforms.

    To select one or more rows in the Project Table (highlighted yellow), use control- and shift-click. Operations can be performed on this entire set by right-clicking. The set of selected entries can also be dragged up or down to move these rows as a block in the Project Table.

    Properties in the Project Table can easily be hidden and program-specific subsets can be selected by choosing Property → Show menu and making your selection. For example, to show only the properties that have values for at least one of the selected entries, choose For Selected Entries.

    Use the ePlayer to perform repetitive operations on entries in the Project Table. In the Project Table, choose Options from the ePlayer menu to specify a command file to run on each selected entry.

    To copy a property value to other rows in the same column, first select the rows, then edit the property value and press Ctrl+Enter (⌘Enter).

    Right-click on the blue header cells of the Project Table for useful options, such as sorting, setting displayed decimal places, clearing values, and adjusting column width.

    The Build toolbar provides an easy way to build molecules in the Workspace. The pencil, usually the first button on the Build toolbar, can be used to sketch molecules or connect existing molecules with bonds.

    Global preferences can be set in the Preferences panel (Maestro menu). To set additional Maestro startup preferences, create or edit your maestro.cmd file (in the maestronn directory of your Schrödinger user resources directory). For example, to customize the display settings and change some default Glide options:

    repall tuberadius=0.18
prefer workspacemenushow=true
ribbon display=both
glidesettings maxatom=200
glidesettings maxrotbonds=35

    A complete list of commands can be found in the Maestro Command Reference Manual.

    Commands corresponding to the tasks performed in Maestro are logged in the Command Script Editor (found on the Edit menu). Scripts can be created with these commands to be run in the Command Script Editor. The same scripts can be loaded into function keys, the ePlayer, or even loaded into Maestro upon starting.

    The Command Input Area (View menu) at the bottom of the Workspace allows you to enter commands using the Maestro command language. You can scroll through commands for recent operations using the UP ARROW and DOWN ARROW keys, and edit the commands. The commands can also be found in the Command Script Editor (Edit menu). One useful command is cd, which changes the Maestro current working directory. A complete list of commands can be found in the Maestro Command Reference Manual.

    The Pose Viewer functionality allows quick viewing of H-bonds and clashes from Glide docking results. Just import your Glide Pose Viewer file into Maestro. If the Pose Viewer panel doesn't open automatically, right-click and hold on the group row for the poses in the Project Table and choose View Poses. You can then use Ctrl+RIGHT ARROW (⌘RIGHT ARROW) to step through the poses. (Ctrl+LEFT ARROW (⌘LEFT ARROW) steps back up the list).
