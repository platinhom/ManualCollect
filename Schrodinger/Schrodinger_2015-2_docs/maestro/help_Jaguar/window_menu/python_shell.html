Summary

This panel allows you to type in and run Python code in the context of Maestro.

Opening the Python Shell Panel

To open the Python Shell panel:

  • Choose Window → Python Shell in the main window.

Using the Python Shell Panel

The Python shell uses the version of Python that is distributed with the software. A number of modules are imported by default: 

from codeop import CommandCompiler, compile_command
from PyQt4 import QtCore, QtGui
from PyQt4.QtCore import Qt
import pythongui_ui
import sys, os, traceback, string
import pyhelp
from schrodinger.maestro import maestro

These modules should make it relatively easy to interact with Maestro You can also import many other modules from the Schrödinger Python API.

Python Shell Panel Features

  • Output display area
  • Input area
  • Run button
  • Clear button
Output display area

This text box displays the output from the Python commands, both standard output (stdout) and standard error (stderr). The text is color-coded by its source: blue for user input, red for standard error, and black for standard output. The text box is read-only.

Input area

This text box, below the output display area, can be used to enter Python code for execution.

Run button

Run the code entered into the input area.

Clear button

Clear the output display area.
