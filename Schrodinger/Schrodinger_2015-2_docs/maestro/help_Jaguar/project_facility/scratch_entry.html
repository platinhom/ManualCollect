Scratch Entry

A scratch entry consists of all of the atoms in the Workspace that are NOT part of any entry in the Project Table. For example, if you begin drawing structures in an empty Workspace these structures are considered part of the scratch entry.

The scratch entry reserves the name "Scratch". In fact, no entries in a project can begin with "Scratch". Also, there is NO row in the Project Table for the scratch entry, as it is not part of the project.

When a molecule that is part of the scratch entry is connected to an entry in the project, it is merged into that entry. That is, it is removed from the scratch entry and placed in the entry to which it is connected.

To convert a scratch entry into a project entry, do one of the following:

  • Choose Workspace → Create Project Entry in the main window.

  • Press Ctrl+Shift+N (⇧⌘N) in the main window.

  • Click the Create Entry button on the Project toolbar in the main window.

  • Click the Create Entry button on the Workspace toolbar in the main window.

  • Right-click in the Workspace and choose Create Project Entry.
