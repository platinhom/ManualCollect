Summary

The measurements panel is used to add and remove measurement labels for distances (defined by two atoms), angles (defined by three atoms), dihedrals (defined by four atoms), and NMR spin-spin coupling constants for vicinial hydrogens (defined by two atoms). The measurements can be stored for later use.

Opening the Measurements Panel

To open the Measurements panel, you can:

  • Choose Tools → Measurements → measurement-type.

This panel is dockable, so it opens by default in the Workspace. You can change this behavior in the Preferences panel, under General – Panels.

Using the Measurements Panel

To display measurements of distances, angles, dihedral angles, and NMR spin-spin coupling constants, define each entity by picking the appropriate number of atoms. To add a new measurement, choose an object from the Pick option menu in the Define atoms for parameter measurement section, and select atoms or bonds in the Workspace. If you select the wrong atom, click it again to deselect it. When the last atom is selected, the measurement is listed in the table and displayed in the Workspace as a set of lines defining the measured entity and a label that gives the value.

To delete an existing measurement, select the measurement in the list and click Delete. To delete all measurements of that type, click Delete All. The measurements are removed from the Workspace and the table.

All the defined measurements are deleted when Maestro is closed. They are not saved in the structure file unless you choose to create a property for the measurements.

The number of decimals in the measurements, the font for the measurement markers, and the border around the markers can be set in the Preferences panel.

To move a measurement marker, drag it in the Workspace.

Measurements Panel Features

The measurements panel has four tabs. Each tab has the following features:

  • Define atoms for parameter measurement section
  • Table
  • Delete and Delete All buttons
  • Write button
  • Preferences button
  • Create property for selected entries option

Define atoms for parameter  measurement section

Each of the tabs (Distances, Angles, and Dihedrals) contains a set of picking controls that you use to define the measurement. You can select either atoms or bonds. If you choose atoms, you must select two atoms for a distance or a coupling constant, three for an angle, or four for a dihedral. If you choose bonds, you must select one bond for a distance, two for an angle, or three for a dihedral. When an atom is selected, it is marked with a purple cube.

When selection is complete, the measurement is marked in the Workspace if you have Markers selected, and the measurement is added to the table. By default, distances are marked by a dashed purple line, angles are marked by a solid green line on the bonds and a dashed green line connecting the two bonds, dihedrals are marked by a solid red line on the bonds and a dashed red line connecting the 1-2 bond with the 3-4 bond, and coupling constants are marked by a dashed turquoise line. The measurement that is selected in the table is drawn with thicker lines than the unselected measurements. You can show or hide the markers with the Markers option, and you can change the color of the markers in the Preferences panel, which you can open by clicking Preferences.

Table

The table in the center of the panel contains the atoms that define the measurements and the results of the measurement. Each row in the table displays information for a single measurement item. Only one row in a table can be selected at a time. The selected row is the current measurement item that is highlighted with two parallel solid lines, as described in the chart above.

Delete and Delete All buttons

Click Delete to delete the selected measurement item. Click Delete All to delete all measurement items in the table.

Write button

Opens the Write Measurements panel, in which you can write all measurement items to a plain text file. The file contains one measurement per line. Each line contains the atom numbers, measurement value, and related information, separated by a comma, a tab character, or a user-defined delimiter.

Preferences button

Opens the Preferences panel with the Measurements tab displayed, so you can set the display precision, the text font, the background for the text, and the marker color.

Create property for selected entries option

Select Create property for selected entries to add a property to the Project Table for each subsequent measurement that you make. If the measurement property already exists, its value is updated. You can create a property for multiple entries by selecting them in the Project Table, and defining the measurement within the included entry. The included entry must be one of the selected entries.

For distance, angle, dihedral, and coupling constant measurements, the property name is constructed from the property type and the atom numbers: for example, Distance 12-24, Angle 32-48-69, Dihedral 12-34-56-78, Coupling 15-21. In the property name, the sequence of picked atom numbers is always shown with the smallest number first. Atom names can be used instead of atom numbers in the property name if you create the property using a Maestro command. For example, the command distance atomname1 atomname2 generates a property named "Distance atomname1-atomname2" in the Project Table.

There can be any number of distance, angle, or dihedral properties in the Project Table. These property values are real numbers.

Restrictions on Measurements

Measurements are allowed between "background" entries (those that are included in the Workspace, but not selected in the Project Table) and one specified entry (one that is both included and selected). In this way, it is possible to generate properties such as distances between a series of ligands and a given receptor as a background.

In the processing of a measurement, selected atoms may come from different entries (picked entries). You must follow some rules when you apply a measurement as a property to selected entries:

  • A measurement is allowed within a single entry, but this entry must be both selected and included.
  • One and only one of the picked entries may be selected. It is an error if a measurement is made by specifying atoms from more than a single selected entry.
  • The scratch entry can be picked. However, if any picked atoms come from the scratch entry, then all picked atoms should come from that entry.
  • If none of the picked entries is selected, the measurement is not applied to selected entries.
