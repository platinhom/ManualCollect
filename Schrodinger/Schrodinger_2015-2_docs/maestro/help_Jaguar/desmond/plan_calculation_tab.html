Summary

In this tab you choose the FEP protocol for each type, and set some options for the simulation.

Plan Calculation Tab Features

  • In pure solvent option
  • FEP protocol option menu
  • Buffer size text box
  • Production simulation time text box
  • Temperature text box
  • Force field option menu
In pure solvent option

This option should remain selected, as no calculation is performed if you deselect it.

FEP protocol option menu

The FEP protocol option menu offers five choices:

  • Desmond NPT—Use the NPT ensemble for the calculation
  • Desmond NVT—Use the NVT ensemble for the calculation
  • Desmond NPT (quick relaxation)—Use the NPT ensemble with a short relaxation time for the calculation
  • Desmond NVT (quick relaxation)—Use the NVT ensemble with a short relaxation time for the calculation
  • User defined—This option displays a text box and Browse button, which allow you to navigate to a protocol file (.msj). You can set up a protocol using the Desmond panel.
Buffer size text box

Specify the size of the buffer zone in angstroms between the solute and the box boundary. The buffer size is ignored if you use an existing model system as input.

Production simulation time text box

Specify the simulation time in ns for the production simulation (after any initial simulations) for each lambda value. The default value is 5 ns for relative free energies (mutation FEP calculations), and 2 ns for calculations in vacuum. The longer time for total free energies is recommended for accurate results.

Temperature text box

Specify the temperature for the simulation.

Force field option menu

Choose the force field to use in the simulations. The default is OPLS_2005, unless you set the default in the Preferences panel.
