Summary

The Embrace Minimization panel is used to minimize the energy of protein-ligand complexes for a series of pre-positioned ligand structures and return a molecular mechanics estimate for the binding free energy for each ligand.

Opening the Embrace Minimization Panel

To open the Embrace Minimization panel, you can:

  • Choose Applications → MacroModel → Embrace Minimization in the main window.

Using the Embrace Minimization Panel

The Embrace Minimization panel contains four tabs. For descriptions of the controls in the first three tabs, see Potential, Substructure, and Mini. The fourth tab, Embrace, contains controls for minimizing the energy of protein-ligand complexes for a series of pre-positioned ligand structures. These are described below. Once you have finished configuring the settings on the panel, you can click Run to run the job with the current job settings, or you can choose Job Settings from the Settings button menu to open the Job Settings dialog box to make job settings, or choose Write to write the files without starting the job (see Energy Overview for details).

To run an Embrace minimization calculation, first set up the potential and the minimization options as for any MacroModel minimization calculation. See Potential and Mini for more information. You must also specify the receptor, ligands, and output options in the Embrace tab, described below.

Because the minimization includes the entire receptor, it is advisable to define substructures and fix or freeze atoms in these substructures to reduce the time spent minimizing the structure, and to eliminate optimizations that have little influence on the binding. See the Substructure tab topic for information on defining substructures.

Embrace Minimization Panel Features

  • Source of ligands
  • Receptor
  • Association energy mode
  • Structures saved
Source of ligands

The ligand structures can either come from a file or from the selected entries in the Project Table. If you choose Input file, you can specify a file for the ligand structures by entering the path to the file in the Input file text box or by clicking Browse to navigate to the file. In either case, the structures are written to the file jobname.mae in Maestro's current working directory.

Receptor

To run an Embrace calculation you must specify the receptor.

  • If you are using an input file, you can do one of the following:
    • Select the First structure in file option
    • Enter an entry name or click Choose to select an entry
  • If you are using selected entries, you can do one of the following:
    • Select the First selected entry option
    • Enter an entry name or click Choose to select an entry
Association energy mode

There are two modes in which the Embrace calculation can function:

  • Energy difference mode: The calculation is performed on the receptor first, then on the ligand, and finally on the complex. The energy difference is: Delta E = Ecomplex - Eligand - Eprotein. This mode includes relaxation of the ligand and receptor on binding. If GB/SA solvation is in use, the solvation free energy is decomposed in a meaningful manner.

  • Interaction energy mode: The interaction energy between the receptor and each ligand is calculated. This energy is calculated from interactions between ligand atoms and receptor atoms, but excludes the contributions coming from the ligand only and the receptor only. It does not take account of the relaxation when the ligand is removed from the receptor. Also, if GB/SA solvation is in use, the solvation free energy is not decomposed in a rigorous manner.

Structures saved

There are three options for how the output structures can be written to the output file:

  • Complexes only (Ligand-receptor complexes only)
  • Ligands only
  • Receptor, ligands and complexes
