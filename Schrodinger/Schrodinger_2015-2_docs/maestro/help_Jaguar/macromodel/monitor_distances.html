Summary

This panel is used to set up distances to be monitored during a dynamics simulation. Results are saved in the .mmo file.

Opening the Monitored Distances Panel

To open the Monitored Distances panel, you can:

  • Click Distances in the Monitor tab of the Dynamics panel.

  • Click Distances in the Monitor tab of the MC/SD panel.

Using the Monitored Distances Panel

Distances to be monitored can be specified by picking the atoms that define the distances, using the standard picking controls in the Define monitored distances section, and clicking on two atoms. When the first atom is picked, Maestro places a purple box around it. After the second atom is picked, the atoms appear in the list at the top of the panel. Maestro traces the distance with a purple dotted line and places a purple "eye" icon beside the dotted line. To distinguish the currently selected distance, the lines are drawn thicker.

To delete a distance from the list, select it in the list and click Delete. To delete all monitored distances, click Delete All.

Monitored Distances Panel Features

  • Monitored distances list
  • Define monitored distances section
  • Delete button
  • Delete All button
Monitored distances list

Lists the distances to be monitored. After the third atom is picked, a new entry appears in the list.

Define monitored distances section

The Define monitored distances section contains the standard picking controls, which are used to select atoms that define a distance. It also contains a Markers option, which you can use to show markers for the monitored distances in the Workspace.

Delete button

To delete a monitored distance, select it in the Monitored Distances List, then click Delete.

Delete All button

Click this button to delete all monitored distances.
