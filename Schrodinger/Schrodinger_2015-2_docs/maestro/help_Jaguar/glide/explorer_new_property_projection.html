Summary

In this dialog box you can project a property onto a particular clustering of the poses.

Opening the New Property Projection Dialog Box

To open the New Property Projection dialog box, you can:

  • Click New Property Projection in the Pose Explorer panel.

New Property Projection Dialog Box Features

  • Map option menu
  • Property list
  • Property filter text box
  • Statistic option menu
  • Calculate Projection button
Map option menu

Choose the self-organizing map (pose clustering) that you want to project a property onto.

Property list

Choose a property to project onto the self-organizing map.

Property filter text box

Enter a text string to filter the properties by. The property list is restricted to the properties that have a match to the string anywhere in the property name.

Statistic option menu

Choose a statistic to calculate for the poses in each cell, as a representation of the property.

Calculate Projection button

Calculate the projection of the property onto the map, and add it to the projections in the Pose Explorer panel. After the projection is generated, the dialog box closes.
