MAE-to-SD

Convert Maestro to SD format

Backend implementation

utilities/sdconvert
sdconvert is used to convert Maestro format into SD output

Dialog Options

Include Input
Includes all input columns in the output
Replace Column
All of the input columns are included in the output except the newly computed SD column replaces the Maestro column that was converted. If this option is checked, then the "Include Input" option is also checked and disabled.

Ports

Input Ports
0 Molecules in Maestro format
Output Ports
0 Molecules in SD format
