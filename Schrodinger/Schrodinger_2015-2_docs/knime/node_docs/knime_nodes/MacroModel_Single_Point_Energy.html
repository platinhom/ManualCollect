MacroModel Single Point Energy

Calculate the energy of a molecule using MacroModel

Backend implementation

scripts/mmodCLI.py
The script mmodCLI.py is used to implement this node.

Dialog Options

Include Log in output
Include the Macromodel log in the output data table
High-throughput mode
High-throughput mode runs all structures at once (as opposed to running the script for each row). This is typically faster and is set on by default.
Force Field
Available force fields: AMBER, AMBER94, MM2, MM3, MMFF, MMFFs, OPLS, OPLS-AA-2001, OPLS-AA-2003, OPLS-AA-2005
Solvent
Available Solvents: None (i.e. vacuum), Water
Electrostatic treatment
Force field defined (default), distance dependent, constant dielectric
Dielectric constant
defaults to 1.0

Ports

Input Ports
0 Input data table containing molecules in Maestro format
Output Ports
0 Output data table containing molecules in Maestro format
