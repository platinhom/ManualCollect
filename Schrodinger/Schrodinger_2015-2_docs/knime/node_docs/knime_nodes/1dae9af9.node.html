Phase DB Creation

Creates a Phase 3D Database and conformations from a set of Molecules.

Backend implementation

$SCHRODINGER/phase_database
An import task and revise task of phase_database are used in sequence to implement this node.

Dialog Options

Column containing structure
Select column in the data input table containing the structures
Phase DB Prefix
The full path name and file prefix of the destination Phase DB. (i.e., {absolute-directory-name}/{prefix})
Input contains multiple conformations per molecule
Flag indicating whether or not the supplied structure file contains multiple conformations per molecule. If checked, consecutive structures with identical titles and connectivities will be treated as conformations of a single molecule. If unchecked, each structure will be treated as a separate molecule.
