Get PDB

Retrieves the PDB for the structure(s) with PDB code(s).

Backend implementation

utilities/getpdb
The command line tool getpdb is used to implement this node.

Dialog Options

[PDB code]{:Chain ID}
PDB code and (optional Chain ID) specified. For multiple structures, use comma ',' to separate the structures' PDB code(s).

Ports

Output Ports
0 Molecule with TITLE, ATOM, HETATM, MODEL, ENDMDL and TER records (for Chain if specified)

Views

Std output/error of Get PDB
Std output/error of Get PDB
