Extract MAE Properties

Extract Maestro properties from Maestro Cells. Note that the list of available properties can only be determined if the data provided by the upstream node is already available.

The names of data items in a Maestro format file follow a convention which allows the type of the data item and the “owner” of the data to be identified. Names have the form 't_o_d', where t is the type descriptor, o is the “owner” and d is the actual data name.

More information on these property names can be found in the Maestro User Manual in the sub-section A.4. Data Item Names under the section The Maestro File Format.

Dialog Options

Input Column Include and Replacement Strategy

Input plus Output - The input columns are included in the output. If there are duplicates, then a "_input" is added to the end of the input columns.

Output replaces Input - The input columns are included in the output, but the duplicate input columns are removed.

Output only - Only Output columns are included.

Short Column Names in Output
Removes the "r_user_"-like tags at the beginning of the column name, if they exist, so that the output columns are shorter and easier to read.
Rescan input
The column names listed in the Exclude and Include list boxes are set based on the node's input. If the input has changed (i.e., the list of MAE properties is different than what is shown), then this button is enabled. Pressing it reloads all of the column names and places them in the Exclude list box.
Exclude
Properties that should be excluded from the output
Include
Properties that should be included in the output
New Property
Add a new user-defined property. This new property needs to have a type that is either a String, Integer, or Double. The added property has the necessary prefix ("s_", "i_", or "r_" respectively) plus what string is entered into the text field. All user-defined properties are removed if the Rescan input button is selected.
