Chemistry External Tool 0:1

This node allows users to run a user-specified external program that has no inputs and outputs one column. The output can be of any of these types: Maestro, Sequence, Alignment, SD, Mol2, Smiles, String, Double, Integer, Text, FingerPrint, Canvas Matrix, Hypothesis or Glide Grid. Return code for the external process needs to be zero or else the node will fail.

Note: for the Hypothesis and Glide Grid types, since each cell represents a group of files, the output variable (i.e., %output_1%) represents a text file that has a file prefix on each line. Therefore, the underlying script will generate a text file with a list of prefixes on each line to send the output into Knime.

Dialog Options

Command line(s)
The command line(s) specified for the external tool. Environment variables can be specified by using a dollar sign ('$') before it.

A '%flow_n%' string, where n is 1,2, or 3, can be used to specify the flow variables flow_n.
Note: If the flow variable has a newline character in it, the node writes it to a file and the string gets converted to the file name. Otherwise, it gets converted to the value of the flow variable.

To specify the output file, use the string '%output_1%'. Note: The output file, which is specified in the command line(s), needs to be generated by the external tool or else this node will fail.
Output Type
The type of output for this node. This is dependent on what type of file the external program generates.
Output Column Name
If selected, the text field allows the user to specify the output column name

Ports

Output Ports
0 Output data from the external process (can also include the input, if specified)
