Add Hydrogens

Adds hydrogen atoms to produce an all-atom structure.

Backend implementation

utilities/applyhtreat
The command line tool applyhtreat is used to implement this node.

Dialog Options

Include Input
Includes all input columns in the output
Replace Column
All of the input columns are included in the output except the new column replaces the selected input column. If this option is checked, then the "Include Input" option is also checked and disabled.

Ports

Input Ports
0 Molecules in Maestro format
Output Ports
0 Molecules with hydrogens added in Maestro format

Views

Std output/error of Apply H Treatment
Std output/error of Apply H Treatment
