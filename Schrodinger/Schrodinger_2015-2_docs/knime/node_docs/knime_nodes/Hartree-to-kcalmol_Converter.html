Hartree-to-kcal/mol Converter

Convert column values from Hartree to kcal/mol or from kcal/mol to Hartree

Dialog Options

Convert
Select conversion, either Hartree -> kcal/mol or kcal/mol -> Hartree
Column to Convert
Select which column to convert.

Ports

Input Ports
0 Any data with at least one column that has Double precision
Output Ports
0 Input data with one or more columns converted
