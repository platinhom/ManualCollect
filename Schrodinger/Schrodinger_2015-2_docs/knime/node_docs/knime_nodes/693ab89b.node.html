Setup Diagnosis

Diagnostic tool for the currently running KNIME instance.
This node verifies the following and reports the result in the View.
1. Required environment
2. Software versions
3. External library 'knimeUtils'
4. KNIME_MAIN license
5. MMSHARE_EXEC and temporary directory paths
6. VM arguments passed
7. $SCHRODINGER/machid output
