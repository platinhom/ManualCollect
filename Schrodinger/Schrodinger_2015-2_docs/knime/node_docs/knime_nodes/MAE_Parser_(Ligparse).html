MAE Parser (Ligparse)

Ligparse is a tool for examining the structural composition of a molecular dataset represented within a Maestro formatted file. Additionally, ligparse can use this structural information to select subsets of compounds from the original dataset that meet some user defined criteria.

Backend implementation

utilities/ligparse
The command line tool ligparse is used to implement this node.
