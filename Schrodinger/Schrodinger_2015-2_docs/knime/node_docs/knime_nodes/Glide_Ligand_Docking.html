Glide Ligand Docking

Ligand Docking uses Glide to dock ligands into existing grids. The docking calculations can be carried out in either Standard Precision (SP), Extra Precision (XP), or High-throughput Virtual Screening (HTVS) mode.

Backend implementation

glide
This command line tool glide is used to implement this node.

Ports

Input Ports
0 Glide Grid
1 Ligands in Maestro or SD format
Output Ports
0 Pose Viewer (i.e Receptor and docked ligands)
1 Receptor in Maestro format
2 Docked ligands in Maestro format
