Alignment Writer

Write alignments to a file in FASTA and Maestro format. The output format is determined by the file extension.

Backend implementation

utilities/seqconvert
seqconvert is used to convert Maestro Alignment format into FASTA output (if necessary)

Dialog Options

Column containing structure
Select the column in the input data table that contains the alignment data. This only applies to input data tables that contain more than one alignment column.
Overwrite existing files
If any output file exists, this node fails.
Write all alignments
Whether all input rows are written
Start
The start row specified if all alignments are not written.
End
Write to the last row if all input rows are not written.
Total
The number of rows written if all alignments are not written and the "End" option is not selected.

Ports

Input Ports
0 Alignments in Maestro format

Views

Std out/err of Alignment Writer
Std out/err of Alignment Writer
