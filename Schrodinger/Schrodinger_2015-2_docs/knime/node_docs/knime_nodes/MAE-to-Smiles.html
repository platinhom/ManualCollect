MAE-to-Smiles

Convert Maestro format to SMILES

Backend implementation

utilities/canvasConvert
canvasConvert is used to convert Maestro format into SMILES output

Dialog Options

Include Input
Includes all input columns in the output
Replace Column
All of the input columns are included in the output except the newly computed Smiles column replaces the Maestro column that was converted. If this option is checked, then the "Include Input" option is also checked and disabled.
Omit Title
Whether the molecule titles should be included in the output (by default, the titles are included in the column after the smiles molecules)

Ports

Input Ports
0 Input data table containing Molecules in Maestro format
Output Ports
0 Output data table containing input molecules in SMILES format (and input columns if 'Include input' is selected)
