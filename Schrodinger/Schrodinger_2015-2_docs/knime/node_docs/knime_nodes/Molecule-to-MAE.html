Molecule-to-MAE

Convert Mol2, SD or PDB to Maestro

Backend implementation

utilities/mol2convert
mol2convert is used to convert mol2 input to Maestro format output (if necessary)
utilities/sdconvert
sdconvert is used to convert sd input to Maestro format output (if necessary)
utilities/canvasConvert
canvasConvert is used to convert SMILES input to Maestro format output (if necessary)
utilities/pdbconvert
pdbconvert is used to convert PDB input to Maestro format output (if necessary)

Dialog Options

Column containing structure
Column in the input data table containing the structure to be converted.
Include Input
Includes all input columns in the output

Ports

Input Ports
0 Input molecule in Smiles, Mol2, SD, or PDB format
Output Ports
0 Output molecule in Maestro format

Views

Std output/error of Molecule2MAE
Std output/error of Molecule2MAE
