Set Molecule Title

Sets the title for a molecule from a selected column. If there is more than one molecule column, then the molecule column also needs to be selected.

Dialog Options

Column containing input
Select target molecule column in the data input table to set the title
Molecule title column
Select column in the data input table containing the titles to set for the target molecule

Ports

Input Ports
0 The input needs at least one molecule (SD, MOL2, or Maestro format) and at least one String column that is the molecule title that is set.
Output Ports
0 The output is the same as the input except the molecule has its title set to the input column selected.
