Desmond Trajectory Extract Frames

The trajectory_extract_frame.py script extracts selected frames from a Desmond trajectory into a table of output structures.

Platform Limitations: Linux only

Backend implementation

$SCHRODINGER/run -FROM desmond trajectory_extract_frame.py

Dialog Options

Column containing Desmond Trajectory
Select column in the data input table containing the Desmond Trajectory
ASL
Enter the ASL expression to specify the part of the structure to be extracted. The default is all. Not valid with pdb or cms output types.
Frames to extract
Inclusive range expressions to specify frames to extract from the trajectory. The frame selection format is "from:to:skip", empty field suggests from=0; to=last; skip=1 Comma separated list of ranges are allowed. Leaving this field blank will allow extracting of all the frames.
Frame frequency
Specify frequency of frames to extract. For example, a value of 2 will extract every other frame in the specified range. Please note that the frequency speficied in this field will be applied only if the "Frames to extract" has a single range(i.e, without any commas) like, 'from:to' or ':to' or 'from:'
Output type
Select the output structure type: mae, pdb or cms
Location of output CMS files
This is where the output CMS files will be stored if output type chosen is CMS.

Ports

Input Ports
0 Input data table containing Desmond Trajectory
Output Ports
0 Structures in Maestro, PDB or CMS format
