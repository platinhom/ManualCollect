Alignment Reader

Read alignment files in FASTA and Maestro format. The input format is determined by the file extension. Note that alignments are converted to Maestro format by default.

Backend implementation

utilities/seqconvert
seqconvert is used to convert FASTA input to Maestro Alignment format (if necessary)

Dialog Options

import all aligments
Read all alignments found in the input file; uncheck to read only a subset of the aligments; the subset to be read in can be controlled via the Start and Total fields.
Load into one cell
Read aligments into one table row/cell; the default is to store at most one aligment per row

Ports

Output Ports
0 Alignments in Maestro format
