XLS Reader

This node reads the first sheet from an Excel workbook file (.xls). The maximum number of columns read in from a spreadsheet is 256 (all additional columns are truncated). If the input sheet has more than 65536 rows it truncates it as well.

Dialog Options

Selected File
Enter a valid input file name. You can also choose a previously selected file from the drop-down list, or select a file from the "Browse..." dialog.
Read column header
If checked, the column names are written out in the first row of the spreadsheet.
Read row ID
If checked, the row IDs are added to the output, in the first colum of the spreadsheet. Note: row IDs need to be unique to be read into Knime, otherwise it will fail.
skip blank rows
If checked, any rows that don't have any values will be skipped. (on by default)
skip blank columns
If checked, any columns that don't have any values will be skipped. (on by default)
skip hidden columns
If checked, any columns that are hidden will be skipped. (on by default)

Ports

Output Ports
0 The XLS file in Knime datatable format.
