Combine Fingerprints

Combines fingerprints for distinct or overlapping sets of molecules. When a molecule appears more than once, the bit sets are combined using a logical OR. If fingerprints were created using the -scaling flag, real-valued data for duplicate molecules are averaged.

Backend implementation
