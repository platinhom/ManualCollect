MAE-to-Pdb

Convert Maestro to PDB format

Backend implementation

utilities/pdbconvert
pdbconvert is used to convert Maestro format into PDB output

Dialog Options

Include Input
Includes all input columns in the output
Replace Column
All of the input columns are included in the output except the newly computed PDB column replaces the Maestro column that was converted. If this option is checked, then the "Include Input" option is also checked and disabled.

Ports

Input Ports
0 Molecules in Maestro format
Output Ports
0 Molecules in PDB format

Views

Console output of MAE-to-PDB
