BLAST

Search a protein sequence database using BLAST
Note: To enable web-based searching, please ensure the option "Allow use of web server for BLAST searches" found in the Jobs panel of Maestro preferences, has been checked.

Backend implementation

scripts/blastCLI.py run by cmdRunner.py
The script blastCLI.py is passed to cmdRunner.py to implement this node.

Dialog Options

Include Log in output
Include the BLAST log file in the output
Database
The database that will be searched (PDB or NR from NCBI)
Filter query sequence
Filter query sequence to remove low-complexity regions
Expand non-redundant hits
Expand non-redundant BLAST hits by showing additional PDB sequences that are identical to the original hit

Ports

Input Ports
0 Sequences in Maestro format
Output Ports
0 BLAST alignments in Maestro format
