Edit Menu

The items on the Edit menu are listed below. Only Copy and Find are on the Edit menu in the custom view windows.

Copy

Copy the table selection to the clipboard. The selection can consist of entire rows or columns, or a rectangular subset of rows and columns. You can then paste them into a spreadsheet application, such as OpenOffice Calc (Linux), Microsoft Excel (Windows), or iWork Numbers (Mac).
Keyboard shortcut: CTRL+C (⌘C)

Clear

Clear the content of the selected cells.

Delete

Delete the selected rows or columns. You are prompted to confirm the action, which cannot be undone.

Select All

Select all cells in the table.
Keyboard shortcut: CTRL+A (⌘A)

Clear Selection

Clear the selection of table cells.
Keyboard shortcut: CTRL+U (⌘U)

Invert Selection

Select the cells that are not selected, and deselect the cells that are selected.

Expand Selection To Entire Rows

Expand the selection so that the entire row is selected for any row that has one or more cells selected.

Expand Selection To Entire Columns

Expand the selection so that the entire column is selected for any column that has one or more cells selected.

Find

Find text in the table, with the option of selecting the cells in which the text is found. Opens the Find panel.
Keyboard shortcut: CTRL+F (⌘F)

Add Row

Append a new empty row to the spreadsheet. When the row is created, the Structure cell is selected so that you can add a structure.

Add Column

Append a new empty column to the spreadsheet. Opens the New Column dialog box, in which you can name the column. The name must be unique.
