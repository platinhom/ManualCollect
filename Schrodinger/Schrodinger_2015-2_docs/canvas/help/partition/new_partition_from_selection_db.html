Summary

In this dialog box, you can create a new partition of the structures (rows) into classes. The partition can be random, based on property values, or empty when it is created. You can create classes later, if you wish. The partition is displayed as a column in the spreadsheet, colored by class.

Opening the New Partition from Selection Dialog Box

To open the New Partition from Selection dialog box:

  • Choose Partition > New > From Selection in the main window.

New Partition from Selection Dialog Box Features

  • Name text box
  • Create classes for selected and unselected rows option
  • Description text box
Name text box

Enter a name for the partition. This name is displayed as the column name in the spreadsheet. If you assign classes when you create the partition, the name is replaced with a standard name, which you can edit.

Description text box

Enter a description of the partition. A description is not required, but it can be used to record the details of the partitioning, for example.

Create classes for selected and unselected rows option

Create a class for the selected rows and another class for the unselected rows. This is an easy way to make a Boolean partition. If this option is not selected, a class is created only for the selected rows, and the unselected rows remain unclassified.
