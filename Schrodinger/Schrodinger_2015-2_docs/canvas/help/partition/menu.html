Partition Menu

The Partition menu provides tools for creating, storing, modifying, applying, and deleting partitions of the spreadsheet rows into classes. The items on the menu are listed below. Most of the items are only available in the Master View. From custom views, you can open and close partitions, but you cannot modify them. Changes to partitions are saved automatically.

Partitions are displayed in the same way as a property, with a color scheme used to identify the class members. The class values can be used in the same way as property values for fitting, clustering, querying, and so on. The column heading begins with Class:: so that the name of the originating property can be included in the partition name.

New

Create a new partition. Submenu with the following items:

  • From Selection—Create a partition from the selected rows.
    Keyboard shortcut: ALT+N
  • From Property—Create a partition from the values of a property.
  • Random Split—Create a partition by random assignment to classes.
Each item opens a dialog box, in which you can name the partition, provide a description, and assign the classes in the partition.

Open

Open a partition. Opens the Open Partition dialog box, in which you can choose the partition, and view its description. If a partition is already open, it is closed first. Available in custom views.
Keyboard shortcut: ALT+O

Save As

Save the current partition with a new name. Opens the Save Partition dialog box, in which you can enter the name and description.

Edit

Edit the current partition. Opens the Edit Partition dialog box, in which you can change the name, description, class values, and colors used.

Add/Move to

Add or move the rows that have any cells selected to the class chosen from this submenu. The submenu lists the existing classes in the current partition, and provides a New Class item for creating a new class. You can use the shortcut ALT+n to move rows to class n, where n is the class index given in parentheses on the submenu, and you can use the shortcut ALT+L to create a new class.

Unclassify

Remove the rows that have any cells selected from their classes, so that they are no longer in any class. A class that has no rows in it after this operation continues to exist as a class: it is not removed.
Keyboard shortcut: ALT+0 (zero).

Hide Color and Show Color

Hide or show the colors in the current partition. Available in custom views.

Delete

Delete the current partition after confirmation.

Filter

Filter the view by selection of classes from the current partition. Opens the Partition Filter dialog box.

Close

Close the current partition. Available in custom views.
Keyboard shortcut: ALT+W
