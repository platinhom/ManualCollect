Summary

In this panel you can filter the rows in the current view by selection of classes from the current partition. The panel allows you to interact with the spreadsheet, so you can choose classes and apply the filter multiple times without closing the panel.

Opening the Partition Filter Panel

To open the Partition Filter panel:

  • Choose Partition → Filter in the main window.

Using the Partition Filter Panel

To filter the rows to include only those in selected classes, select the classes in the table, ensure that Hide rows for selected classes is cleared, and click Apply or OK.

To filter the rows to hide rows that are in selected classes, select the classes in the table, ensure that Hide rows for selected classes is selected, and click Apply or OK.

To filter the rows to show only those that are not in a class, ensure that no rows are selected in the table, clear Hide unclassified rows, and click Apply or OK.

Clicking Apply allows you to view one or more classes in the spreadsheet and interact with them, then choose other classes in the panel, and view them, without closing and reopening the panel.

Partition Filter Panel Features

  • Name text box
  • Description text area
  • Classes table
  • Hide rows for selected classes option
  • Hide unclassified rows option
  • Apply button
Name text box

Displays the name of the current partition. Noneditable.

Description text area

Displays the description of the current partition. Noneditable.

Classes table

Displays the classes in the current partition. The table cells are noneditable. You can select multiple rows in the table. The selected rows define the classes that you want to show or hide.

Hide rows for selected classes option

Hide the rows for the classes that are selected in the table. When this option is not selected (the default), the rows for the selected classes are shown and the rows for the unselected classes are hidden.

Hide unclassified rows option

Hide the rows that are not in any class. This option is selected by default. When it is not selected, the unclassified rows are included (shown).

Apply button

Apply the filtering to the spreadsheet without closing the panel. You can interact with the spreadsheet, and make and apply multiple selections of classes.
