Scripts Menu

The Scripts menu appears on the menu bar in the main window (master view). The menu contains two fixed items, and below that are items for scripts, when scripts are installed.

Manage

Manage the scripts on the scripts menu. You can delete or reload scripts, or view their descriptions. Opens the Manage Scripts dialog box.
Keyboard shortcut: CTRL+M (⌘M)

Install

Install Python scripts on the scripts menu. Opens the Install Scripts dialog box.
Keyboard shortcut: CTRL+I (⌘I)
