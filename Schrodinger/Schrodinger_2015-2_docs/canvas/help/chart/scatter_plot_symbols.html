Summary

In this dialog box you can change the style, color, and size of the plot symbols for the current scatter plot. The color and the size can each be made dependent on the value of a property, which allows you to visually represent the values of two more properties.

Opening the Plot Symbols Dialog Box

To open the Plot Symbols dialog box:

  • Choose Settings → Plot Symbols in the Scatter Plot panel.

Plot Symbols Dialog Box Features

  • Style option menu
  • Color section
  • Size section
Style option menu

Choose the shape to be used for the plot symbols

Color section

Set the color to be used for the plot symbols. There are two main options:

  • Fixed—Use a fixed color for the symbols. The color is displayed next to the option. Click Select to open a color chooser in which you can select the color.

  • By property—Color the symbols according to the values of the property chosen from the option menu. The property need not be the same as that used for either of the axes.

    If you choose a continuous (real or integer) property, the color scheme is a linear interpolation between the colors for the limiting values. You can set the colors by clicking the Min and Max buttons and choosing a color from the color chooser that opens. The colors for the limits are displayed below these two buttons. The default limiting values are the maximum and minimum value, but you can change the limiting values in the Limits text boxes. Property values that are larger than the maximum limiting value or smaller than the minimum limiting value are colored according to the color for the maximum or minimum. To return to the default limits, click Reset.

    For categorical properties or an open partition, a cycle of distinct colors is used for each category or class, and the controls for the limiting values and the colors at the limiting values are not available.

Size section

Set the size to be used for the plot symbols. There are two main options:

  • Fixed—Use a fixed size for the symbols. Set the point size of the symbols in the box.

  • By property—Set the size of the symbols according to the values of the property chosen from the option menu. The property need not be the same as that used for either of the axes. The size is interpolated linearly between the sizes set for the limiting values of the properties . You can set the minimum and maximum point size in the Min and Max boxes. The default limiting values are the maximum and minimum value of the property, but you can change the limiting values in the Limits text boxes. Property values that are larger than the maximum limiting value or smaller than the minimum limiting value are assigned the point size for the maximum or minimum. To return to the default limits, click Reset.
