Summary

In this dialog box you can set up the y axis and its labels.

Opening the Y Axis Dialog Box

To open the Y Axis dialog box:

  • Choose Settings → Y Axis in the Histogram panel.

Y Axis Dialog Box Features

  • Axis limits section
  • Axis line section
  • Tick marks section
  • Tick mark labels section
  • Axis title section
  • Color buttons
Axis limits section

Set the minimum and maximum values on the axis, and choose an option for displaying the bars as counts or as a percentage of the maximum.

Axis line section

Set the color and weight of the axis line, in points.

Tick marks section

Set the spacing of the tick marks and the y value at which the marks should start; and set the color and weight of the marks.

Tick mark labels section

Set the spacing and starting value for the tick mark labels, the font size in points, the color, and the number of decimal places displayed. The spacing should be a multiple of the tick mark spacing.

Axis title section

Set the axis title, font size in points, and color of the text

Color buttons

The Color buttons open a color chooser, in which you can select the desired color. The selected color is displayed to the right of the button.
