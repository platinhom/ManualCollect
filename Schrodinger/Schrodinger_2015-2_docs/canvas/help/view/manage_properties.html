Summary

In this dialog box you can select the properties that are displayed in the current view.

Opening the Manage Properties Dialog Box

To open the Manage Properties dialog box:

  • Choose View → Manage Properties.

Using the Manage Properties Dialog Box

  • To add properties to the list that is shown in the view, select the properties in the Available properties list and click Add.

  • To remove properties from the list that is shown in the view, select the properties in the Visible properties list and click Remove.

  • To restrict the list of properties shown in the Available properties list to those containing a text string, deselect Search visible properties and type the text in the search text box.

  • To restrict the list of properties shown in the Visible properties list to those containing a text string, select Search visible properties and type the text in the search text box.

Manage Properties Dialog Box Features

  • Available properties list
  • Visible properties list
  • Add, Add All, Remove Remove All buttons
  • Search text box
  • Search visible properties option
Available properties list

This list contains properties that are available to show but are not visible. You can select multiple rows with shift-click and control-click, and use the Add button to transfer them to the Visible properties list.

Visible properties list

This list contains the properties that have been chosen to show. You can select multiple rows with shift-click and control-click, and use the Remove button to remove them from the list and transfer them to the Available properties list.

Add, Add All, Remove Remove All buttons

Use these buttons to transfer properties between the Available properties list and the Visible properties list. The Add and Remove buttons operate on the selected properties.

Search text box

Search for text in one of the two property lists, determined by the Search visible properties option. The list is restricted to property names that contain the search text, and is done as you type. To show the full list, clear the text box.

Search visible properties option

Select this option to search the Visible properties list; deselect it to search the Available properties list.
