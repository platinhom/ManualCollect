View Menu

The View menu appears on the menu bar in both the main window and the custom view windows. The available items depend on whether the menu is in the main window or a custom view window.

Refresh

Refresh the master view when external data is incorporated into the project, such as molecular descriptors or fingerprints, so that the new properties are displayed in the view. Not available in custom view windows.
Keyboard shortcut: F5

Open

Open an existing custom view. Opens a dialog box from which you can select a view, then opens the view in a new window. Not available in custom view windows.

Save

Save the current view. Not available in the main window.

Save As

Save a view as a custom view. Opens the Save Custom View dialog box, in which you can choose the content, name and describe the view, and open it in a new window.

Close

Close the current view. Only available from a custom view.

Details

Display details about the current view. Opens the Details dialog box, which allows you to change the view description. Only available from a custom view.

Logic

Apply a logical operation on the current view and another view to modify the current view. Opens the Apply Logic to View dialog box.

Manage Properties

Select the properties that are displayed in the view. Opens the Manage Properties dialog box.

Random Subset

Choose a random subset of the rows from the current view. Opens the Choose Random Subset dialog box, in which you can make settings for the number of rows, the seed, and where to store the results.

Apply to Master

Apply the current view settings to the Master View. Opens the Apply to Master dialog box, in which you can make settings for the rows, columns, and visibility in the master view. The settings in the Master View are temporary, and are not stored. Only available from a custom view.

Delete

Delete the selected rows or columns from the view. The entire rows or columns must be selected; partially selected rows or columns are not deleted. Not available in the main window.

Export Row IDs

Export all row IDs from the view to a binary file. These row IDs can be used to select structures in a Canvas project for processing by a Canvas application when run from the command line.

Collapse To Selected

Display only the rows and columns that contain selected cells. The rows and columns that do not have selected cells are not deleted from the view, only hidden. You can redisplay them by choosing Undo from the View menu.

Select Rows with Missing Values

Select the rows for which one or more columns do not have values.

Select Columns with Missing Values

Select the columns for which one or more rows do not have values.

Select Rows with Invalid Chemistry

Select the rows for which a structure is not available. This can happen on import if the structure block is badly formatted or the SMILES string is invalid; in this case a string noting the error is displayed in the structure cell.

Hide Selected

Hide the rows and columns that contain selected cells. The rows and columns are not deleted from the view, only hidden. You can redisplay them by choosing Undo from the View menu.

Undo View Change

Undo the last change to the view. Can be used to undo multiple changes. Only available in the main window as Undo Manage Properties.

Restore Default View

Restore the default view in the main window. Not available in a custom view.

Revert All Changes

Undo all changes to the view since the last save. Not available in the main window.
