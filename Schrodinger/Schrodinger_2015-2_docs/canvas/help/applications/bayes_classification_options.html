Summary

Set advanced options for Bayes classification, including Kullback-Leibler acceptance criteria for fingerprint data, and the smoothing coefficient.

Opening the Bayes Classification - Advanced Options Dialog Box

To open the Bayes Classification - Advanced Options dialog box:

  • Click Advanced Options in the Bayes Classification panel.

Bayes Classification - Advanced Options Dialog Box Features

  • Smoothing coefficient text box
  • Keep bits in top box
  • Apply distance cutoffs option
  • Positive cutoff option and text box
  • Negative cutoff option and text box
Smoothing coefficient text box

Specify the smoothing coefficient. Values are normally given in the range 0 to 1.

Keep bits in top box

Specify the percentage of the most significant bits to keep during training, as ranked by Kullback-Leibler significance.

Apply distance cutoffs option

Select this option to apply distance cutoffs during training.

Positive cutoff option and text box

Select this option to apply a distance cutoff to positively correlated features, and specify the cutoff in the text box.

Negative cutoff option and text box

Select this option to apply a distance cutoff to negatively correlated features, and specify the cutoff in the text box.
