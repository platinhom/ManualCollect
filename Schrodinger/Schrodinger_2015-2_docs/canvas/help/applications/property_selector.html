Property Selection Tools

Property selection tools are used in a number of panels to select one or more properties. This tool set consists of the following features:

Available properties list

This list contains properties that are available for use but are not in the set selected for use. You can select multiple rows with shift-click and control-click, and use the Add button to transfer them to the Selected properties list.

Add, Add All, Remove, and Remove All buttons

Use these buttons to transfer properties between the Available properties list and the Selected properties list. The Add and Remove buttons operate on the properties that are selected in the relevant list.

Selected properties list

This list contains the properties that have been selected for use. You can select multiple rows with shift-click and control-click, and use the Remove button to transfer them to the Selected properties list.

Search text box

Search for text in one of the two property lists, determined by the Search selected properties option. The list is restricted to property names that contain the search text, and is done as you type. To show the full list, clear the text box.

Search selected properties option

Select this option to search the Selected properties list; deselect it to search the Available properties list.
