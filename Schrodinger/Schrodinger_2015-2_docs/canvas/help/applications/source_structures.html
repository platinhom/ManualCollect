Summary

In this panel, you can view the source structures that contain the scaffolds selected in the Scaffold Decomposition of jobname panel, and run an R-group analysis of the structures.

Opening the Source Structures Panel

To open the Source Structures panel:

  • Click Show Source Structures in the Scaffold Decomposition of jobname panel.

Source Structures Panel Features

  • Sync with curently selected scaffolds option
  • Structures table
  • Analyze R-groups on buttons
  • Export structures buttons
  • Cell size slider
Sync with curently selected scaffolds option

Update the structures table to display the structures for the scaffolds that are currently selected in the Scaffold Decomposition of jobname panel.

Structures table

This table displays the structures that contain the selected scaffolds, along with the property value in the same graphical form as for the scaffolds. The row numbering also matches the scaffold row numbering.

Analyze R-groups on buttons

Open the R-Group Analysis panel to perform an R-group analysis on the structures that are selected in the structures table (Selected), or on all structures in the structures table (All). If the structures have more than one scaffold in common, you are prompted to choose the scaffold you want to use for R-group analysis.

Export structures buttons

Export the structures that are selected in the structures table (Selected), or all structures in the structures table (All) to a file. These buttons open a file selector, in which you can navigate to a location, choose the file type, and name the file.
