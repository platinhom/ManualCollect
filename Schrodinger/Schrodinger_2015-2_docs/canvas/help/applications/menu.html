Applications Menu

The Applications menu provides access to the Canvas computational programs. When you start a job, the Start Application dialog box opens, in which you can choose a host and the number of CPUs (when relevant). When you run a job, messages about the job are displayed in the Messages View window, and a new item is added in the Project View window.

An alternative view of the applications is provided in the Tasks menu, which classifies the applications by the kind of task performed.

3D Minimization

Convert 2D structures to 3D or minimize 3D structures, using LigPrep. Opens the 3D Minimization dialog box.

Molecular Properties

Calculate various molecular properties. Opens the Molecular Properties dialog box.

Feature Selection

Identify subsets of properties that are representative of a larger set, using hierarchical clustering. Opens the Feature Selection dialog box.

Binary Fingerprints

Calculate fingerprints from structures. Opens the Binary Fingerprints from Structures dialog box.

3D Pharmacophore Fingerprints

Calculate fingerprints from pharmacophore representations of 3d structures. Opens the 3D Pharmacophore Fingerprints dialog box.

Similarity/Distance Matrix

Calculate similarity or distance matrices from fingerprints or from properties. This submenu has two items: Create from Fingerprints, which opens the Similarity/Distance Matrix from Fingerprints dialog box, and Create from Properties, which opens the Similarity/Distance Matrix from Properties dialog box.

Similarity/Distance Screen

Screen structures using the similarity or distance matrix. Opens the Similarity/Distance Screen dialog box.

Shape Screen

Screen structures by shape similarity to a query structure. Opens the Shape Screen dialog box

Diversity-Based Selection

Choose compounds for optimum diversity. Opens the Diversity-Based Selection dialog box.

Hole-Filling/Library Optimization

Select a diverse subset of structures that fills holes in a library and optionally satisfy criteria on properties. Opens the Hole-Filling and Library Optimization dialog box.

Library Comparison

Calculate nearest-neighbor similarities between two libraries of compounds. Opens the Library Comparison dialog box.

Hierarchical Clustering

Cluster structures according to a similarity or distance matrix. Opens the Hierarchical Clustering dialog box.

Leader-Follower Clustering

Cluster structures using the leader-follower method. Opens the Leader-Follower Clustering dialog box.

K-Means Clustering

Cluster structures using the k-means method. Opens the K-Means Clustering dialog box.

Self-Organizing Map

Create and apply a Kohonen self-organizing map for a set of properties or fingerprints. Opens the Self-Organizing Map dialog box.

Principal Components Analysis

Perform a principal components analysis for selected properties. Opens the Principal Components Analysis dialog box.

Maximum Common Substructure

Find the maximum common substructure among the source structures, based on a particular definition of atom and bond types. Opens the Maximum Common Substructure dialog box.

Bayes Classification

Build and apply a Bayes classification model for selected properties. Opens the Bayes Classification dialog box.

Multiple Linear Regression

Build and apply a multiple linear regression model for selected properties. Opens the Multiple Linear Regression dialog box.

Partial Least Squares Regression

Build and apply a partial least squares regression model for selected properties. Opens the Partial Least Squares Regression dialog box.

Kernel-Based PLS Regression

Build and apply a kernel-based partial least squares regression model for selected properties. Opens the Kernel-Based PL Regression dialog box.

Principal Components Regression

Build and apply a regression model based on principal components for selected properties. Opens the Principal Components Regression dialog box.

Neural Networks

Build and apply a neural network model for selected properties. Opens the Neural Networks dialog box.

Recursive Partitioning

Build a model for property ranges or categories using recursive partitioning. Opens the Recursive Partitioning dialog box.

Scaffold Decomposition

Locate all possible scaffolds (ring systems with connecting linkers) in a set of molecules. Opens the Scaffold Decomposition dialog box.

R-Group Analysis

Analyze a set of compounds into a common core and a set of groups attached to the core, then analyze properties related to the sets of groups at each attachment position. Opens the R-Group Analysis dialog box.

External Application

Run an external application with structures from Canvas. Opens the External Application dialog box.
