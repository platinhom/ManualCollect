Summary

In this dialog box you can filter the fingerprints used in the KPLS regression model either by ON/OFF frequency or by information content.

Opening the KPLS - Advanced Fingerprints Options Dialog Box

To open the KPLS - Advanced Fingerprints Options dialog box:

  • Click Advanced Options for Fingerprints in the Kernel-Based Partial Least-Squares Regression dialog box.

KPLS - Advanced Fingerprints Options Dialog Box Features

There is one section in this dialog box, labeled Filter bits by.

  • ON/OFF frequency option
  • Information content option
ON/OFF frequency option

Select this option to filter out bits that do not vary much between molecules.

Discard bits that are ON in fewer than N % of molecules option and box

Select this option to discard bits that are set by fewer than the specified percentage of molecules. Default: 5%

Discard bits that are ON in more than N % of molecules option and box

Select this option to discard bits that are set by more than the specified percentage of molecules. Default: 95%.

Information content option

Select this option to filter bits by their information content.

Retain the N most informative bits option and box

Select this option to keep the most informative bits, and enter the number of bits to be kept in the text box. (Sets the -mostSig command option.)
