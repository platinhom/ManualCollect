Summary

Perform hierarchical agglomerative clustering on a set of structures using a similarity or distance matrix.

Opening the Hierarchical Clustering Dialog Box

To open the Hierarchical Clustering dialog box:

  • Choose Applications → Hierarchical Clustering in the main window.

Data set option menu

Select the source of the data. The data source can involve both row and column selection. The choices are:

  • Selected rows—Use the rows that are selected in the master view. You must select the rows before opening the dialog box.

  • Visible rows—Use the rows that are displayed (visible) in the master view. You must ensure that the required rows are visible before opening the dialog box.

  • Saved view—Use the rows and columns in a chosen saved view. When you select this item, an option menu that lists the saved views in the project is displayed to the right. You can choose the desired view from this option menu. A second option menu, labeled Domain, allows you to choose whether to restrict the rows or the columns, or both, to those of the saved view. Otherwise the visible rows or columns of the Master View are used.

  • Original data set for jobname—Use the rows and columns that were used for the specified job. This item is only available when you are cloning a job or viewing the results of a job.

Save results as text box

Assign a name to the results. This name is listed under Hierarchical Clustering in the Project View window.

From fingerprints option

Select this option to generate a similarity or distance matrix from pregenerated fingerprints.

Fingerprint column option menu

Choose the fingerprint column to use for the matrix. You must generate fingerprints beforehand and incorporate them into the project. The available fingerprints are those that are listed under Binary Fingerprints in the Project View window and marked as incorporated.

Metric option menu

Choose the metric used to calculate the similarity or distance matrix.

Formula Key button

Open a panel that displays the definitions of the terms in the Description text box.

Description text box

Noneditable text box that displays the definition of the metric.

Tversky α and β text boxes

Specify the α and β parameters for the Tversky metric.

From file option

Select this option to read the similarity or distance matrix from a file.

File name text box and Browse button

Enter the name of the similarity or distance matrix file in the text box, or click Browse and navigate to the file.

Matrix treatment section

Specify how to use the matrix if it is asymmetric. Select Matrix is asymmetric, and choose an option for selecting the value to use for the off-diagonal elements.

Cluster linkage method option menu

Select a method for linking the clusters. The method is described in the Description text box. The default is Average.

Single Closest inter-cluster pair
Complete Farthest inter-cluster pair
Average Average distance between all inter-cluster pairs
Centroid Euclidean distance between cluster centroids
McQuitty Average distance to the two clusters merged in forming a given cluster
Ward Sum of squared distances to merged cluster centroid (minimum variance)
Weighted Centroid Weighted center of mass distance, also known as median
Flexible beta Weighted average intra-cluster and inter-cluster distances (Lance-Williams) with beta=0.25.
Schrödinger Closest distance between terminal (right-to-left) points in 1D cluster orderings.
Description text box

Noneditable text box that displays a description of the cluster linkage method.

Create Clusters button

Run the job to perform the clustering.
