Summary

Calculate a similarity or a distance matrix for selected structures from binary fingerprints using a chosen metric.

Opening the Similarity/Distance Matrix from Fingerprints Dialog Box

To open the Similarity/Distance Matrix from Fingerprints dialog box:

  • Choose Applications → Similarity/Distance Matrix > Create from Fingerprints in the main window.

Similarity/Distance Matrix from Fingerprints Dialog Box Features

  • Data set option menu
  • Fingerprint column option menu
  • Metric option menu
  • Formula Key button
  • Description text box
  • Tversky α and β text boxes
  • Structure tooltips option
  • Color by value option
  • Results area
  • Compute button
  • Export button
Data set option menu

Select the source of the data. The data source can involve both row and column selection. The choices are:

  • Selected rows—Use the rows that are selected in the master view. You must select the rows before opening the dialog box.

  • Visible rows—Use the rows that are displayed (visible) in the master view. You must ensure that the required rows are visible before opening the dialog box.

  • Saved view—Use the rows and columns in a chosen saved view. When you select this item, an option menu that lists the saved views in the project is displayed to the right. You can choose the desired view from this option menu. A second option menu, labeled Domain, allows you to choose whether to restrict the rows or the columns, or both, to those of the saved view. Otherwise the visible rows or columns of the Master View are used.

  • Original data set for jobname—Use the rows and columns that were used for the specified job. This item is only available when you are cloning a job or viewing the results of a job.

Fingerprint column option menu

Choose the fingerprint column to use for the matrix. You must generate fingerprints prior to calculating the matrix, and incorporate them into the project. The available fingerprints are those that are listed under Binary Fingerprints in the Project View window and marked as incorporated.

Metric option menu

Choose the metric used to calculate the matrix.

Formula Key button

Open a panel that displays the definitions of the terms in the Description text box.

Description text box

Noneditable text box that displays the definition of the metric.

Tversky α and β text boxes

Specify the α and β parameters for the Tversky metric.

Structure tooltips option

Show the 2D structures for a table cell in a tool tip when the pointer pauses over the table cell.

Color by value option

Color the results table cells by value, using a linear interpolation between red (highest) and blue (lowest) in RGB color space.

Results area

Displays the matrix as a table, labeled with the structure names.

Compute button

Calculate the matrix. A dialog box opens, in which you can choose to display the matrix as a table in the results area or export the matrix directly to a CSV file. If you choose to export to file, click Browse to navigate to the location and name the file, or enter the file name in the Matrix file text box.

Export button

Export the distance matrix to a CSV file. Opens a file selector, in which you can navigate to a location and name the file.
