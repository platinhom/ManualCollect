Structure Menu

The Structure menu provides access to tasks that can be performed on the structures.

Show Structures in PyMOL

Open PyMOL to view the structures in 3D. Adds a check box to each structure cell, which can be used to send structures to PyMOL for viewing. This menu item is replaced with Close PyMOL when PyMOL opens.

Close PyMOL

Close PyMOL. This item replaces Show Structures in PyMOL when PyMOL is opened.

Hide

Hide the 2D structure, and display only the structure name in the Structure column. (Replaces Show.)

Show

Display the 2D structure and the structure name in the Structure column. (Replaces Hide.)

Structure Filter

Filter table rows by chemical content of the structures, using SMARTS patterns. Opens the Structure Filter dialog box.

Autoscale

Automatically scale the 2D structure so that it fits in the table cell.

Highlight Matches

When doing a query on a substructure, highlight the matches in each matching structure. Applies for Query → Substructure Query and Applications → Maximum Common Substructure.

Align Matches

When doing a query on a substructure, align the 2D display of the matches in each matching structure to a reference structure, which must be selected from among the matches. The default references structure is the first match. Applies for Query → Substructure Query and Applications → Maximum Common Substructure. To change the reference structure, you must deselect this option, choose the new reference, then select this option to align to the new reference.

This option is automatically deselected if you delete the reference row, modify the reference structure so that it no longer matches the query, hide the reference structure row in a custom view then save the view, or undo the substructure query filter.

Choose Highlight Color

Choose the highlight color for highlighting matches to a substructure query. Opens a color chooser.

Color Heteroatoms

Apply a color scheme to heteroatoms (non-carbon atoms). If this option is off, structures are drawn entirely in black.

Show All Hydrogens

Show all hydrogen atoms in 2D structures.

Edit

Edit the structure in the current cell. Opens the Edit Structure dialog box, or ChemDraw on Windows if the preference to use ChemDraw is set in the Preferences panel. Only available from the master view. You can also edit the structure by double-clicking it in the spreadsheet.

Edit Name

Edit the name of the structure in the Structure column. Opens the Edit Structure Name dialog box, in which you can enter the desired name.

Change Structure Name Property

Change all structure names to the value of a chosen property. Opens the Change Structure Name Property dialog box, in which you can choose the property (such as a company ID), decide whether to keep the chosen property in the spreadsheet (other than as the structure name), and decide whether to store the current structure name in a new property. Properties with real number values are not permitted as structure names, so are not available.

Copy Structure

Copy the structure onto the Canvas clipboard in MDL (SD) format. You can then paste the structure into another structure cell.

Copy SMILES

Copy the structure onto the clipboard as a SMILES string. You can then paste the string into another location, such as a structure cell, a text box that requires a SMILES string, or another application.

Copy Image

Copy an image of the structure onto the clipboard as a bitmap. You can then paste the image into another application.

Paste

Paste the structure on the Canvas clipboard into the current structure cell. The existing structure in the cell is replaced.

Paste SMILES

Paste the SMILES string on the clipboard into the current structure cell. The existing structure in the cell is replaced.

Detect Duplicates

Check for duplicate structures in the database, and create two columns, Has duplicates and Duplicate of, that record information on the duplication. These columns contain the UID of the first (lowest ID) of a set of identical structures: Has duplicates is set to this UID for the first of the set; Duplicate of is set to this ID for the rest of the set.
