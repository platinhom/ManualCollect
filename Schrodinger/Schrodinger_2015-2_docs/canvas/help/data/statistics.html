Summary

In this dialog box you can compute various univariate and bivariate statistics for the current view. The results are displayed in a table.

Opening the Statistics Dialog Box

To open the Statistics dialog box:

  • Choose Data > Statistics in the current view.

Statistics Dialog Box Features

  • Scope options
  • Color by value option
  • Ignore properties with missing values option
  • Univariate tab
  • Bivariate tab
  • Compute button
  • Export button
Scope options

Select Selected data to compute statistics for the rows and columns that are selected in the view. Select All data to compute statistics for all rows and columns in the view.

Color by value option

Color the table cells by value, using a blue-white-red color map. Only available for bivariate statistics.

Ignore properties with missing values option

Select this option to exclude from the calculation of the statistics any properties that have missing values.

Univariate tab

In this tab, you can select the univariate statistics you want to calculate.

Univariate statistics list

Lists the various statistics that can be computed. You can select multiple statistics from the list for calculation.

Results table

Results of the calculation are displayed in a table to the right of the list. In addition to the various statistics, a Count column lists the number of rows in the spreadsheet for which the property has a value.

Bivariate tab

This tab provides a choice of bivariate statistics, and displays the values of the statistic in a table.

Bivariate statistics list

Lists the various statistics that can be computed. You can select a single statistic from the list for calculation.

Results table

Results are displayed in a table to the right of the list. The results are colored by the statistic value if Color by value is selected. Clicking in a table cell displays a scatter plot of the two variables (properties) used to compute the statistic.

Compute button

Click this button to compute the selected statistics.

Export button

Export the results table in the current tab as a CSV file. Opens a file selector, in which you can navigate to the location and name the file.
