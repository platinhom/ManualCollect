Summary

Sort the rows in the current view by the values of one or more properties.

Opening the Sort Dialog Box

To open the Sort dialog box:

  • Choose Data > Sort in any view window.

Sort Dialog Box Features

  • New sort option
  • Reverse current order option
  • Restore natural order option
  • Sort by option menus
  • Add button
  • Remove button
  • Sort list
  • Store results in options
New sort option

Select this option to start a new sort.

Reverse current order option

Reverse the order of the rows in the current view. When this option is selected, all other controls are unavailable, except for the Store results in options.

Restore natural order option

Restore the order in which the rows are stored in the database.

Sort by option menus

Choose a property whose values you want to sort by and select the direction of the sort from these option menus.

Ascending and Descending options

Select one of these options to determine the direction in which the values of the property selected from the Sort by option menu are ordered.

Add button

Add the property and sort direction to the end of the sort list.

Remove button

Remove the selected property from the sort list.

Sort list

List of properties by which the rows are sorted. The rows are sorted by the listed properties in the order they are given in the list. Each row indicates the direction of the sort as well as the property. You can select a single row in the list to delete the property from the list. You can drag a row to move it up or down the list.

Store results in options

Choose the view in which to store the results of the sort. If you choose Current View, the current view is updated with the new ordering of the rows. If you choose choose New View, the Save Custom View dialog box opens after the sort is done, and you can name, describe, and save the sort results as a new view.
