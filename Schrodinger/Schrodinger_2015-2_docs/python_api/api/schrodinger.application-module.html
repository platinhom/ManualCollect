Package application


A package for application specific modules.

desmond:
    A package containing modules for the Desmond project

epik:
    A package exposing the Epik functionality.

glide:
    A package for creating Glide grid generation and docking jobs

inputconfig:
    Module for writing and reading simplified input files

jaguar:
    A package for creating Jaguar QM related tasks

ligprep:
    A package containing LigPrep-related modules.

macromodel:
    A package for creating and running MacroModel and Cluster jobs.

mcpro:
    A package containing modules for the MCPRO+ product

phase:
    A package containing Phase-related modules.

prepwizard:
    Module that is used by PrepWizard

report2d:
    Module that is used by 2D Viewer and generate_2d_report to produce
    PDF and HTML files with 2D structure images.

ska:
    Module for running SKA backend (protein structure alignment)

vsw:
    A package containing the VSW modules and stages

watermap:
    Module with WaterMap-related functions.
