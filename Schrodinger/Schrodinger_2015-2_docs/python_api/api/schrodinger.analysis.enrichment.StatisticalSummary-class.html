Class StatisticalSummary


Container class to store summary metrics describing an array of
numerical values.

API example:
------------
values = [x/1000 for x in range(1, 1001)]
stat_summary = StatisticalSummary(
    'test',
    numpy.array(values)
)
print stat_summary.mean 
>> 0.50
print stat_summary.std
>> 0.288
